# 电子书管理与工具

## 一、电子书格式对比

| 格式 | 全称 | 特点 | 适用设备 | DRM 常见性 |
|------|------|------|----------|-----------|
| EPUB | Electronic Publication | 开放标准，可重排，支持 CSS 样式 | 全平台（除 Kindle 原生）| 常见 |
| PDF | Portable Document Format | 固定版面，排版精确 | 全平台 | 常见 |
| MOBI | Mobipocket | 旧版 Kindle 格式，支持笔记 | Kindle 设备 | 常见 |
| AZW3 | Amazon Word 3 | Kindle 新格式（KF8），支持 HTML/CSS | Kindle 设备 | 常见 |
| AZW | Amazon Word | Kindle 旧专有格式 | 旧款 Kindle | 常见 |
| KFX | Kindle Format X | Kindle 最新格式，增强排版 | 新款 Kindle | 是 |
| DJVU | DjVu | 高压缩率扫描文档 | 全平台 | 否 |
| CBZ/CBR | Comic Book Archive | 漫画压缩包格式 | 漫画阅读器 | 否 |
| DOCX | Office Open XML | Word 原生格式 | 全平台 | 否 |
| TXT | Plain Text | 纯文本，无格式 | 全平台 | 否 |

EPUB 是目前最推荐的通用格式，支持可重排、可缩放，适合各种屏幕尺寸。
## 二、DRM（数字版权管理）

DRM 限制电子书在不同设备和应用间的转移。主要 DRM 方案：
1. Adobe Digital Editions (ADE) DRM：
   - 最广泛使用的 EPUB/PDF DRM
   - 最多授权 6 台设备
   - 通过 Adobe ID 管理

2. Kindle DRM：
   - Amazon 专有 DRM
   - 仅限 Kindle 设备和应用
   - 通过亚马逊账户管理
3. Apple FairPlay：
   - 苹果 iBooks 使用
   - 仅限苹果生态
4. 第三方 DRM（如 Kobo DRM、Nook DRM）
重要提示：移除 DRM 可能违反版权法和服务条款，请仅对自己合法购买的电子书并在当地法律允许范围内操作。
## 三、Calibre：全能电子书管理工具

Calibre 是功能最强大的开源电子书管理软件，核心功能包括：

### 安装与界面
- 官方网站 calibre-ebook.com
- 支持 Windows / macOS / Linux
- 中文界面支持

### 核心功能

1. 书库管理：
   - 元数据编辑（标题、作者、封面、标签、评论）
   - 搜索与筛选（按标签、系列、评分等）
   - 虚拟书架（根据规则自动分组）
   - 批量操作（批量修改元数据、批量转换）

2. 格式转换：
   - 支持 EPUB / MOBI / AZW3 / PDF / DOCX / TXT 等 20+ 格式互转
   - 转换时可定制字体、边距、行距、输出配置
   - 使用方法：选中书籍 -> 转换书籍 -> 选择输出格式

3. 内容服务器：
   - 在局域网内通过浏览器访问书库
   - 支持无线推送至 Kindle 等设备
4. 新闻抓取：
   - 内置数百个新闻源
   - 定期抓取并转换为电子书格式
### 常用快捷键
| 操作 | Windows/Linux | macOS |
|------|-------------|-------|
| 添加书籍 | Ctrl+O | Cmd+O |
| 编辑元数据 | Ctrl+D | Cmd+D |
| 转换书籍 | Ctrl+T | Cmd+T |
| 发送到设备 | Ctrl+Shift+S | Cmd+Shift+S |
| 打开书籍 | Enter | Enter |
| 搜索 | Ctrl+F | Cmd+F |

## 五、个人数字图书馆建设

### 文件组织策略

推荐目录结构：
```
/Digital_Library/
  /Books/                  # 图书
    /Fiction/
    /NonFiction/
    /Reference/
  /Papers/                 # 学术论文
    /ByYear/
    /ByTopic/
  /Documents/              # 技术文档、手册
  /Metadata_Backup/        # Calibre 数据库备份
```

### 元数据管理最佳实践
- 使用 Calibre 统一管理元数据
- 为每本书添加标签（至少 2-3 个）
- 为系列书籍设置系列名和序号
- 定期更新封面
- 导出书库到 CSV 做备份
### 备份策略

1. 主备份：导出 Calibre 数据库（.db 文件）
2. 文件备份：备份完整书库文件夹（含电子书文件）
3. 云同步：使用 OneDrive / iCloud / NAS 同步书库文件夹
4. 频率：每次新增 10 本以上时触发备份

## 六、格式转换实用场景
| 场景 | 推荐操作 |
|------|----------|
| 从 Kindle 迁移到其他设备 | MOBI/AZW3 -> EPUB |
| 在电脑上批注 | 任意格式 -> PDF |
| 在手机上阅读 | PDF -> EPUB（可重排）|
| 存档/打印 | EPUB -> PDF（固定排版）|
| 导入笔记工具 | EPUB -> DOCX/Markdown |
| 仅保留文字信息 | 任意格式 -> TXT |

## 七、常见问题
| 问题 | 解决方法 |
|------|----------|
| EPUB 显示乱码 | Calibre 转换时设置字体编码为 UTF-8 |
| Kindle 不认识 EPUB | 使用 Calibre 转换为 AZW3 或 MOBI |
| PDF 在 E Ink 上字太小 | 使用 KOReader 或 Briss 裁剪白边 |
| 元数据丢失 | 使用 Calibre 插件的元数据下载功能 |
| 书库太大加载慢 | 分割为多个 Calibre 书库 |
