# 跨体系对照表

## 概述

本文档建立 UNESCO 基础科学分类、中国学科门类体系以及其他国际通用分类体系之间的映射关系，便于跨体系检索和学术交流。

## UNESCO 基础科学与中国理学门类的对应关系

| UNESCO 学科 | 中国理学一级学科 | 说明 |
|------------|-----------------|------|
| 数学 | 基础数学、计算数学、概率论与数理统计、运筹学与控制论 | 完全对应 |
| 物理学 | 理论物理、粒子物理与原子核物理、凝聚态物理、光学等 | 完全对应 |
| 化学 | 无机化学、分析化学、有机化学、物理化学、高分子化学与物理 | 完全对应 |
| 生物学 | 植物学、动物学、生理学、微生物学、遗传学、生态学等 | 完全对应 |
| 地球科学 | 自然地理学、大气科学、海洋科学、地质学 | 完全对应 |
| 天文学 | 天体物理、天体测量与天体力学 | 部分对应 |
| 逻辑学 | 基础数学（数理逻辑方向） | 部分对应 |

## 使用建议

1. **跨体系检索时**：先确定源体系中的学科，通过本对照表找到目标体系中的对应学科
2. **交叉学科研究**：注意一个交叉学科可能对应多个传统学科
3. **国际交流**：根据交流对象选择合适的分类体系
4. **文献检索**：同时使用多个分类体系进行检索以提高查全率
