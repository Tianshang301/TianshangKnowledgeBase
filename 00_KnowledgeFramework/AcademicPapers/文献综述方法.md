# 文献综述方法

## 一、文献综述的定义与目的
文献综述是对特定研究领域现有文献的系统性梳理、评估和综合，其核心目的包括：
- 确认已有研究的贡献与局限
- 识别研究空白和争议点
- 为自身研究提供理论基础和方法借鉴
- 避免重复已有的研究成果

## 二、综述类型对比
| 类型 | 方法 | 目标 | 适用场景 |
|------|------|------|----------|
| 系统性综述 | 严格预设方法，全面检索，可重复 | 回答特定研究问题 | 医学、公共卫生、循证实践 |
| 叙述性综述 | 选择性回顾，非正式方法 | 概述领域进展 | 人文学科、理论性论文 |
| 范围综述 | 广泛检索，映射关键概念 | 界定研究领域的范围 | 新研究领域、概念厘清 |
| 元分析 | 系统性综述 + 统计合并 | 定量综合效应量 | 有多项同质定量研究时 |
| 整合性综述 | 结合理论与实证 | 提出新概念框架 | 跨学科研究、理论建构 |
| 快速综述 | 简化系统综述流程 | 快速为决策提供证据 | 政策制定有时间限制 |

## 三、PRISMA 框架（系统性综述）

PRISMA 是系统性综述和元分析的首选报告规范，核心流程为四阶段：
1. 识别阶段：
   - 在多个数据库中执行预定义的检索策略
   - 记录检索日期、数据库名称、检索词组合
   - 导出所有检索结果至文献管理工具

2. 筛选阶段：
   - 去除重复文献
   - 根据标题和摘要进行初筛
   - 记录排除原因

3. 合格性评估：
   - 获取初筛通过文献的全文
   - 依据预设的纳入/排除标准逐一评估
   - 记录每篇被排除文献的具体原因

4. 纳入阶段：
   - 确定最终纳入的文献数量
   - 使用 PRISMA 流程图展示各阶段数量变化

## 四、文献检索策略
构建检索式时使用布尔运算符：
- AND：缩小范围，要求同时满足多个条件
- OR：扩大范围，满足任一条件即可
- NOT：排除特定内容（慎用，可能遗漏相关文献）

检索词构建技巧：
1. 将研究问题分解为 PICO（Population, Intervention, Comparison, Outcome）
2. 为每个概念列出同义词和近义词
3. 使用截词符（*）匹配词形变化
4. 使用短语检索（""）精确匹配
5. 利用数据库的主题词表（如 MeSH）

常用学术数据库：

| 学科 | 推荐数据库 |
|------|-----------|
| 综合 | Web of Science, Scopus, Google Scholar |
| 医学 | PubMed, Embase, Cochrane Library |
| 工程 | IEEE Xplore, Engineering Village |
| 计算机 | ACM Digital Library, DBLP, arXiv |
| 社会科学 | SSCI, PsycINFO, ERIC |
| 中文 | CNKI 知网、万方、维普、CSD |
| 预印本 | arXiv, bioRxiv, SSRN, ResearchGate |

## 六、文献阅读与笔记方法

1. 三遍阅读法：
   - 第一遍：浏览标题、摘要、结论，判断相关度（5-10 分钟）
   - 第二遍：仔细阅读全文，标注关键方法、结果和论断（30-60 分钟）
   - 第三遍：批判性评估，记录与自身研究的联系和差异
2. 笔记记录建议：
   - 用自己的语言复述而非摘抄原文
   - 记录引用信息和页码
   - 为每篇文献标注贡献类型（理论、方法、数据、工具）

## 七、综合与写作

文献综述写作结构（叙述性综述）：
1. 引言：提出研究问题和综述目的
2. 主体段落：按主题/时间/方法论组织
   - 主题式：将文献按照核心主题分组讨论
   - 时间线式：按发展历程组织，展示领域演进
   - 方法论式：按研究方法分类，对比不同方法的发现
3. 讨论与结论：识别研究空白，指出方法论问题，建议未来方向
4. 参考文献：使用一致的引用格式

写作技巧：
- 避免"流水账式"列举：不要逐篇总结"作者 A 发现 X，作者 B 发现 Y"
- 使用对比和综合：将多篇文献放在一起讨论它们的共识与分歧
- 引入批判性评价：指出研究设计的局限、样本问题、结论的有效性
