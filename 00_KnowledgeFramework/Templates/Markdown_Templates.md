# Markdown 模板

## 基础语法

### 文档结构
```markdown
# 标题
## 二级标题
### 三级标题

正文内容

- 列表项
- 列表项

**粗体** *斜体* `代码`
```

### 扩展语法
- 表格
- 任务列表
- 脚注
- 数学公式（LaTeX）
- 代码块
## 常用模板

### 读书笔记模板
```markdown
# 书名

## 基本信息
- 作者：
- 出版社：
- 阅读时间：
## 核心内容
### 主要观点
### 关键概念

## 个人感悟
```

### 学习笔记模板
```markdown
# 课程名称

## 学习目标
## 知识要点
## 例题与练习
## 总结与反思
```
