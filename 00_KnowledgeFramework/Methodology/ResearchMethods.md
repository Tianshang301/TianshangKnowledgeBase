# 研究方法概览

## 概述

研究方法是进行科学研究的系统化方法论。本文档介绍定量研究、定性研究、混合方法研究、文献综述方法以及研究伦理。

## 定量研究（Quantitative Research）

### 核心特点
- 基于数字和统计分析
- 追求客观性和普遍性
- 可重复性和可验证性
- 大样本研究

### 主要方法

#### 1. 实验研究（Experimental Research）
- **特点**：控制变量，建立因果关系
- **类型**：
  - 真实验设计
  - 准实验设计
  - 前实验设计
- **应用**：自然科学、心理学、教育学

#### 2. 调查研究（Survey Research）
- **特点**：收集大量数据，了解现状
- **工具**：问卷、访谈
- **抽样**：随机抽样、分层抽样
- **应用**：社会科学、市场研究

#### 3. 相关研究（Correlational Research）
- **特点**：分析变量间关系
- **方法**：相关分析、回归分析
- **局限**：不能确定因果关系
- **应用**：心理学、教育学

### 统计分析
- **描述统计**：均值、中位数、标准差
- **推断统计**：t 检验、方差分析、卡方检验
- **多元分析**：因子分析、聚类分析
- **软件工具**：SPSS、R、Python

## 定性研究（Qualitative Research）

### 核心特点
- 基于文字、图像等非数字数据
- 强调主观意义和情境
- 探索性和解释性
- 小样本深度研究

### 主要方法

#### 1. 访谈研究（Interview Research）
- **类型**：
  - 结构化访谈
  - 半结构化访谈
  - 非结构化访谈
- **技巧**：倾听、追问、记录
- **应用**：心理学、社会学

#### 2. 观察研究（Observation Research）
- **类型**：
  - 参与观察
  - 非参与观察
- **记录**：田野笔记、录像
- **应用**：人类学、社会学

#### 3. 案例研究（Case Study）
- **特点**：深入分析个别案例
- **类型**：单案例、多案例
- **应用**：商业、教育、医学

### 数据分析
- **主题分析**：识别主题和模式
- **内容分析**：系统化分析文本
- **叙事分析**：分析故事和叙述
- **软件工具**：NVivo、ATLAS.ti

## 混合方法研究（Mixed Methods Research）

### 核心特点
- 结合定量和定性方法
- 三角验证提高效度
- 更全面理解研究问题
- 灵活的研究设计

### 主要设计

#### 1. 解释性序列设计
```
定量研究 → 定性研究（解释定量结果）
```

#### 2. 探索性序列设计
```
定性研究 → 定性研究（探索问题）
```

#### 3. 并行设计
```
定量研究 + 定性研究（同时进行）
```

### 优势与挑战
| 优势 | 挑战 |
|------|------|
| 更全面的理解 | 方法整合复杂 |
| 三角验证 | 时间成本高 |
| 灵活性 | 需要多种技能 |
| 互补性 | 结果解释困难 |

## 文献综述方法

### 系统综述（Systematic Review）
- **特点**：系统化、可重复的文献综述
- **步骤**：
  1. 制定研究问题
  2. 制定检索策略
  3. 文献筛选
  4. 质量评价
  5. 数据综合
- **应用**：循证医学、政策研究

### 元分析（Meta-Analysis）
- **特点**：定量综合多个研究结果
- **方法**：效应量计算、异质性检验
- **优势**：
  - 提高统计功效
  - 解决争议问题
  - 发现调节变量
- **应用**：医学、心理学、教育学

### 批判性综述
- **特点**：批判性评价现有研究
- **重点**：研究质量、理论贡献
- **应用**：理论发展、研究规划

## 研究伦理

### 基本原则

#### 1. 知情同意（Informed Consent）
- 告知研究目的和程序
- 说明风险和收益
- 保证自愿参与
- 允许随时退出

#### 2. 数据保护（Data Protection）
- 匿名化处理
- 数据保密
- 限制访问
- 合理存储

#### 3. 保密性（Confidentiality）
- 保护参与者身份
- 匿名报告
- 限制数据共享

#### 4. 公正性（Justice）
- 公平选择参与者
- 平衡风险和收益
- 特殊群体保护

### 伦理审查
- **机构审查委员会（IRB）**：审查研究方案
- **知情同意书**：必须获得批准
- **伦理问题报告**：及时报告问题
- **持续监督**：定期审查研究进展
