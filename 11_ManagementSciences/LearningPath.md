# 学习路径：管理科学

## 基础阶段
1. **管理学原理**
2. **微观经济学与宏观经济学**
3. **会计学基础**
4. **统计学**
5. **组织行为学**

## 专业方向
| 方向 | 核心课程 |
|------|----------|
| 工商管理 | 营销、财务、人力资源、战略 |
| 管理科学与工程 | 运筹学、供应链、项目管理 |
| 公共管理 | 公共政策、公共财政、电子政务 |
| 图书情报 | 信息组织、知识管理、档案学 |

- 管理世界（中文期刊）
