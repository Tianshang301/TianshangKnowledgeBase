# 运筹学

## 线性规划

### 标准形式
```
目标函数：min Z = c^T x
约束条件：Ax = b
          x ≥ 0
```
- c：价值系数向量
- A：约束矩阵
- b：右端项向量
- x：决策变量向量

### 单纯形法
- **基本思想**：从可行域的一个顶点出发，沿可行域的边移动到使目标函数值更优的相邻顶点
- **步骤**：
  1. 引入松弛变量，将不等式约束转化为等式
  2. 建立初始单纯形表
  3. 检验最优性（检验数σj ≥ 0）
  4. 确定入基变量（σj < 0对应的变量）
  5. 确定出基变量（最小比值原则）
  6. 进行基变换运算
  7. 重复步骤3-6直至最优

### 对偶理论
- **原问题与对偶问题的关系**：
  - 原问题求最小值 ↔ 对偶问题求最大值
  - 原问题约束条件 ≤ ↔ 对偶变量 ≥ 0
  - 原问题变量 ≥ 0 ↔ 对偶约束条件 ≥
- **强对偶定理**：若原问题有最优解，则对偶问题也有最优解，且目标函数值相等
- **互补松弛定理**

## 整数规划

### 整数规划类型
- **纯整数规划**：所有变量为整数
- **混合整数规划**：部分变量为整数
- **0-1整数规划**：变量取值为0或1

### 求解方法
- **分支定界法**：
  1. 求解松弛问题
  2. 对整数约束进行分支
  3. 通过定界剪枝减少搜索
- **割平面法**：通过添加割平面约束逼近整数最优解
- **隐枚举法**：用于0-1规划

## 动态规划

### 基本概念
- **阶段（Stage）**：决策过程的时间段
- **状态（State）**：每个阶段的初始条件
- **决策（Decision）**：每个阶段的选择
- **状态转移方程**：S(k+1) = T(S(k), D(k))
- **指标函数**：衡量策略优劣的指标
- **最优值函数**：最优策略下的指标值

### 最优性原理
- 一个多阶段决策过程的最优策略具有这样的性质：无论初始状态和初始决策如何，其后的决策必须相对于初始决策产生的状态构成最优策略。

### 应用领域
- 最短路径问题
- 资源分配问题
- 生产计划问题
- 库存管理问题
- 设备更新问题

## 排队论

### 基本概念
- **顾客到达过程**：泊松过程
- **服务时间分布**：指数分布
- **排队规则**：先到先服务（FCFS）、后到先服务（LCFS）、优先级服务

### M/M/1模型
- **假设**：泊松到达、指数服务、单服务台、无限容量、先到先服务
- **基本公式**：
  - 系统利用率：ρ = λ/μ
  - 平均队列长度：Lq = ρ²/(1-ρ)
  - 平均顾客数：L = ρ/(1-ρ)
  - 平均等待时间：Wq = ρ/(μ-λ)
  - 平均逗留时间：W = 1/(μ-λ)

### M/M/c 模型
- **多服务台模型**
- **关键参数**：服务台数 c、到达率λ、服务率μ

### 排队系统优化
- **经济分析**：服务成本与等待成本的平衡
- **系统设计**：确定最优服务台数量

## 图与网络

### 最短路径问题
- **Dijkstra 算法**：求解从源点到所有其他节点的最短路径
- **Floyd 算法**：求解所有节点对之间的最短路径

### 最小生成树
- **Prim 算法**：从某一顶点开始逐步扩展
- **Kruskal 算法**：按边权从小到大选择

### 最大流问题
- **Ford-Fulkerson 方法**：通过增广路径逐步增加流量
- **容量限制**：0 ≤ f(e) ≤ c(e)
- **流量守恒**：流入量 = 流出量

### 最小费用最大流
- 在保证最大流的前提下，使总费用最小

## 博弈论基础

### 博弈的要素
- **参与人（Players）**：决策主体
- **策略（Strategies）**：参与人的行动方案
- **支付（Payoffs）**：各策略组合下的收益
- **信息（Information）**：参与人对博弈的了解

### 博弈类型
| 分类标准 | 类型 |
|---------|------|
| 按信息 | 完全信息/不完全信息、完美信息/不完美信息 |
| 按行动顺序 | 同时博弈/序贯博弈 |
| 按合作性 | 合作博弈/非合作博弈 |

### 纳什均衡
- **定义**：每个参与人的策略都是对其他参与人策略的最优反应
- **存在性**：每个有限博弈至少存在一个纳什均衡（可能混合策略）

## 决策分析

### 决策树
- **组成**：决策节点（方形）、机会节点（圆形）、结果节点（三角形）
- **分析步骤**：
  1. 绘制决策树
  2. 估算各分支概率
  3. 估算各结果收益
  4. 从右向左计算期望值
  5. 选择最优决策

### 不确定性决策准则
| 准则 | 思路 | 态度 |
|------|------|------|
| 最大最大准则 | 乐观准则 | 乐观 |
| 最大最小准则 | 悲观准则 | 保守 |
| 最小最大后悔值准则 | 最小化最大后悔 | 折中 |
| 等概率准则 | 各状态等概率 | 中立 |
