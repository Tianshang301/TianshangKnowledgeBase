# InterdisciplinaryMethodologies

本索引汇集了 InterdisciplinaryMethodologies 相关的核心知识文件，涵盖 SystemsThinking - 系统思维：以系统视角理解整体与部分关系的思维方法，涵盖反馈回路、涌现、层级结构等核心概念、ComplexityScience - 复杂性科学：研究复杂系统中的涌现、自组织、适应等现象的跨学科理论、TransdisciplinaryResearch - 跨学科研究：跨越学科边界整合知识与方法论的研究范式，涵盖研究框架设计、团队协作等 等内容。
