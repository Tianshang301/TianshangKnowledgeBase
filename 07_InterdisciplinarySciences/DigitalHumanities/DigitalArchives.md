# 数字档案

## 概述

数字档案是利用数字化抢术将传统档案资源转化为数字形式，并过网络提供利用的档案管理系统它是数字人文基硢设施的重要组成部分，为文化遗产保护和学术研究提供支撑?

**古籍OCR?*
```python
# 使用PaddleOCR识别古籍文字
from paddleocr import PaddleOCR

ocr = PaddleOCR(use_angle_cls=True, lang='ch')
result = ocr.ocr('ancient_text.jpg')

for line in result[0]:
    print(line[1][0])  # 识别结果
```
