# 学习路径：交叉学科

## 学习顺序
1. **基础**：了解交叉学科方法论
2. **数据科学**：统计学 + 编程 + 领域知识
3. **认知科学**：心理学 + 神经科学 + AI
4. **生物信息学**：生物学 + 计算机科学
5. **量子信息**：物理学 + 信息科学
6. **数字人文**：人文 + 计算
