# TianshangKnowledgeBase（天殇知识库）

欢迎来到 **TianshangKnowledgeBase** – 个人综合知识库。  
本知识库依据 **联合国教科文组织（UNESCO）基础学科分类** 与 **中华人民共和国教育部学科门类体系** 构建，覆盖从小学到大学本科及以上所有主要学科领域，目标总容量约 **1 GB**（纯文本约 600 MB + 多媒体辅助 400 MB）。

> 让知识有序生长，日有所进。

## 🚀 使用建议

1. **克隆与初始化**  
   ```bash
   git clone https://github.com/your-username/TianshangKnowledgeBase.git
   cd TianshangKnowledgeBase
   ```

2. **推荐工具**  
   - 本地编辑： Obsidian（免费） + 插件（Copilot、Smart Connections）  
   - 云端备份： GitHub + Git LFS（用于大文件）  
   - 全文检索： `ripgrep` 或 Obsidian 内置搜索

3. **内容填充策略**  
   - 文本内容尽量采用 Markdown + LaTeX，保持可移植性。  
   - 每个学科文件夹内可创建一个 `_README.md` 说明本学科的参考教材、学习顺序。  
   - 图片、PDF 等放至 `00_KnowledgeFramework/` 对应子目录，通过相对路径引用。

4. **与 AI 结合**  
   - 可使用 **Obsidian Copilot** 插件（支持 DeepSeek API 免费额度）进行语义检索、内容生成。  
   - 若需完全离线，用 **Ollama + Qwen2.5** 搭配 `smart-connections` 插件实现本地 RAG。
