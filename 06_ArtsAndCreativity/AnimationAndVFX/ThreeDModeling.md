# 3D建模

3D建模是创建三维数字对象的过程，广泛应用于游戏、影视、建筑可视化和工业设计等领域。从概念设计到最终渲染输出，3D建模涉及几何造型、材质纹理、光照渲染等多个技术环节。

## 主流3D建模软件

### Blender

Blender 是一款开源免费的全能3D创作套件，涵盖建模、雕刻、动画、渲染、视频编辑和合成等完整工作流程。其 Cycles 和 Eevee 渲染器分别提供离线物理渲染和实时预览渲染能力。Blender 的 Python API 允许深度自定义和自动化，社区插件生态丰富。

### 其他专业工具

- **Maya**：影视动画行业标准，强大的动画和绑定系统
- **3ds Max**：建筑可视化和游戏建模常用，Modifier Stack 工作流高效
- **ZBrush**：数字雕刻行业标准，支持数十亿面的超高精度雕刻
- **Substance Painter/Designer**：PBR 材质制作的专业工具
- **Houdini**：程序化建模和特效领域的领导者

## 多边形建模（Polygon Modeling）

多边形建模是最基础也是最常用的建模方法，通过操纵顶点（Vertex）、边（Edge）和面（Face）来构建三维形状。

### 核心操作

- **挤出（Extrude）**：从现有面或边拉伸出新的几何体
- **环切（Loop Cut）**：在模型表面添加新的边循环，增加细节
- **倒角（Bevel）**：将尖锐边角柔化为斜面或圆角
- **布尔运算（Boolean）**：通过并集、差集、交集操作组合几何体
- **细分曲面（Subdivision Surface）**：通过算法平滑低多边形网格

### 拓扑学（Topology）

良好的拓扑结构是高质量模型的基础。动画角色需要**边循环（Edge Loops）**沿着肌肉走向排列，以确保变形时网格自然弯曲。四边面（Quad）是最佳的面类型，三角面（Triangle）应尽量避免出现在变形区域，N-gon（超过四条边的面）在游戏引擎中可能产生渲染问题。

### 布线原则

1. 边循环应跟随解剖学结构和肌肉走向
2. 关节区域需要足够的细分以支持平滑变形
3. 避免极点（Poles）聚集——即超过5条边汇聚的顶点
4. 保持网格密度均匀，避免局部过于稀疏或密集

## 数字雕刻（Digital Sculpting）

数字雕刻模拟传统雕塑的工作方式，使用虚拟工具在高密度网格上进行自由形体塑造。

### 雕刻工作流程

1. **基础形状（Base Mesh）**：从简单几何体开始建立大形
2. **大型雕刻**：使用 Clay Buildup、Move 等工具塑造整体比例
3. **中级细节**：使用 Dam Standard、Trim Dynamic 等工具雕刻肌肉和结构
4. **精细细节**：使用 Alpha 纹理和精细笔刷添加皮肤纹理、褶皱等

### Retopology（重拓扑）

雕刻产生的高密度网格不适合直接用于动画和游戏，需要通过重拓扑创建干净的低多边形版本。常用方法包括手动重拓扑（使用 Shrinkwrap 修改器投射到高模表面）和自动重拓扑工具（如 InstantMeshes、QuadriFlow）。

## UV展开（UV Unwrapping）

UV展开是将三维模型表面"展开"为二维平面的过程，这是纹理贴图的必要步骤。

### UV展开策略

- **接缝切割（Seam Marking）**：在模型的隐蔽位置标记切割线
- **岛状排列（Island Packing）**：将展开的UV岛高效排列在0-1空间内
- **Texel Density**：保持纹理像素密度一致，避免某些区域清晰度明显不同
- **UDIM 工作流**：支持多UV瓦片，适用于影视级高精度纹理

### 常见问题

- **拉伸（Stretching）**：UV变形导致纹理扭曲，需要检查并修正
- **接缝可见**：纹理在接缝处不连续，需要在绘制时注意对齐
- **空间浪费**：UV岛排列不紧凑导致纹理分辨率浪费

## 材质与贴图（Material & Texturing）

### PBR 工作流程

基于物理的渲染（Physically Based Rendering）是现代游戏和影视的标准材质系统。核心贴图包括：

| 贴图类型 | 英文名称 | 功能描述 |
|---------|---------|---------|
| 基础色贴图 | Albedo/Base Color | 物体的固有颜色，不含光照信息 |
| 法线贴图 | Normal Map | 模拟表面凹凸细节，不增加几何复杂度 |
| 粗糙度贴图 | Roughness Map | 控制表面反射的模糊程度 |
| 金属度贴图 | Metallic Map | 区分金属和非金属区域 |
| 环境光遮蔽 | Ambient Occlusion | 模拟缝隙和凹陷处的阴影 |
| 高度贴图 | Height/Displacement Map | 真正改变几何形状的凹凸信息 |

### 纹理绘制工具

- **Substance Painter**：支持图层式 PBR 纹理绘制，智能材质和蒙版功能强大
- **Substance Designer**：基于节点的程序化纹理生成
- **Quixel Mixer**：混合扫描纹理的快速材质制作工具
- **Mari**：影视级高分辨率纹理绘制

## 实践建议

1. 从简单的硬表面模型（如家具、武器）开始练习多边形建模
2. 学习解剖学基础知识，提升角色建模的准确性
3. 理解 PBR 材质的物理原理，而非死记硬背参数
4. 建立自己的材质库和模型资产库，提高工作效率
5. 关注游戏引擎（Unity/Unreal）中的实时渲染优化技巧
