# 环境设计

## 1. 室内设计（Interior Design）

### 1.1 空间规划（Spatial Planning）

| 规划步骤 | 内容 | 工具 |
|----------|------|------|
| 1. 需求分析 | 使用者数量、功能需求、行为模式 | 访谈、问卷 |
| 2. 节目列表（Program） | 空间类型清单及面积要求 | 表格列表 |
| 3. 分区（Zoning） | 公共/私密/服务区域划分 | 气泡图 |
| 4. 动线规划（Circulation） | 主要/次要动线，避免交叉 | 箭头流向图 |
| 5. 家具布局 | 按功能和动线布置家具 | 平面图、1:50比例 |

**空间分区类型**：

| 分区 | 功能 | 隐私等级 | 示例 |
|------|------|----------|------|
| 公共区 | 接待/社交/活动 | 开放 | 客厅、大堂、餐厅 |
| 私密区 | 休息/个人生活 | 封闭 | 卧室、浴室 |
| 服务区 | 储存/准备/作业 | 半开放 | 厨房、洗衣房、储藏室 |

### 1.2 地面材料

| 材料 | 特性 | 适用空间 | 维护 |
|------|------|----------|------|
| 实木地板 | 温暖、自然、有等级感 | 客厅、卧室 | 需定期打蜡 |
| 工程木地板 | 稳定、价格适中 | 全屋 | 较易维护 |
| 瓷砖 | 防水、耐磨、多样 | 卫生间、厨房、玄关 | 易清洁 |
| 天然石材 | 奢华、耐久 | 大堂、高档住宅 | 需密封处理 |
| 地毯 | 柔软、隔音、温暖 | 卧室、办公室 | 需专业清洁 |
| 混凝土 | 工业感、耐久 | 商业空间、地下室 | 需密封 |
| LVT（豪华乙烯基片） | 仿木/石、防水 | 卫生间、商业 | 极易维护 |

### 1.3 墙面材料

| 墙面处理 | 效果 | 适用 |
|----------|------|------|
| 乳胶漆 | 经济、色彩选择多 | 通用 |
| 壁纸/壁布 | 花纹/肌理丰富 | 卧室/客厅 |
| 木饰面板 | 温暖高级 | 背景墙/办公 |
| 石材/砖 | 质感/工业 | 特色墙/室外 |
| 软包/硬包 | 吸音+豪华 | 卧室床头/影音室 |
| 微水泥 | 无缝/极简 | 现代风格 |

### 1.4 天花设计

| 类型 | 特征 | 适用 |
|------|------|------|
| 平顶 | 最简单，造价低 | 现代/极简 |
| 跌级吊顶 | 分层，可藏灯带 | 客厅/办公 |
| 井格式 | 仿古梁格结构 | 传统/欧式 |
| 暴露式 | 裸露管道/结构 | 工业/LOFT |
| 格栅吊顶 | 透气/序列感 | 商业空间 |
| 弧形/波浪 | 柔化空间 | 前卫设计 |

### 1.5 照明设计（Lighting Design）

**照明层次**：

| 层次 | 功能 | 照度参考 | 示例 |
|------|------|----------|------|
| 环境光（Ambient） | 整体照明 | 150-300 lux | 天花板筒灯/灯带 |
| 任务光（Task） | 特定作业照明 | 300-750 lux | 台灯、橱柜下灯 |
| 重点光（Accent） | 突出物品/装饰 | 3-5倍环境光照度 | 射灯、轨道灯 |
| 装饰光（Decorative） | 装饰/氛围 | 低照度 | 吊灯、壁灯 |

**色温选择**：

| 色温区间 | 光色感觉 | 适用空间 |
|----------|----------|----------|
| 2700-3000K | 暖黄光 | 卧室、餐厅、客厅（温馨） |
| 3500-4000K | 中性白光 | 厨房、卫生间、书房 |
| 5000-6500K | 冷白光 | 办公室、医院、商业 |

### 1.6 色彩心理学在室内设计中的应用

| 色彩 | 心理效果 | 室内应用 |
|------|----------|----------|
| 蓝色 | 冷静、降低心率 | 卧室/浴室（放松） |
| 绿色 | 自然、平衡 | 书房/客厅（舒适） |
| 黄色 | 愉悦、刺激 | 厨房/餐厅（增进食欲） |
| 红色 | 激情、提高心率 | 餐厅/健身房（小面积） |
| 紫色 | 奢华、创造 | 卧室/创意工作室 |
| 灰色 | 中性、现代 | 任何空间（基底色） |
| 白色 | 洁净、放大空间 | 小空间/简约风格 |

## 2. 景观设计（Landscape Architecture）

### 2.1 场地分析要素

| 要素 | 分析内容 | 影响设计 |
|------|----------|----------|
| 地形（Topography） | 高程变化、坡度 | 排水、阶梯、观景位置 |
| 土壤 | 类型、排水性、PH值 | 植物选择 |
| 气候 | 日照、风向、降水 | 建筑朝向、植物适应性 |
| 植被 | 现有树木、植物群落 | 保留/移除决策 |
| 水文 | 水源、地表径流 | 排水设计、水景 |
| 视线（Views） | 好的视野/不良遮挡 | 开口/遮蔽位置 |
| 动线 | 周边道路、人行流线 | 入口/路径连接 |

### 2.2 软质景观（Softscape）

| 植物类型 | 高度 | 功能 | 设计应用 |
|----------|------|------|----------|
| 乔木 | >6m | 遮阴、框景、结构 | 主景树、行道树 |
| 小乔木/大灌木 | 2-6m | 空间分隔、背景 | 边界、视线引导 |
| 灌木 | 0.5-2m | 界定空间、遮挡 | 绿篱、基础种植 |
| 地被植物 | <0.5m | 覆盖地面、防草 | 替代草皮 |
| 草皮/草坪 | — | 活动空间、视觉开阔 | 游乐区、休憩区 |
| 攀缘植物 | — | 垂直绿化、覆盖 | 墙面、棚架 |

### 2.3 硬质景观（Hardscape）

| 要素 | 材料选择 | 注意事项 |
|------|----------|----------|
| 铺装 | 透水砖/石材/混凝土/木板 | 防滑、排水坡度2% |
| 路径 | 宽度≥1.2m（两人并行） | 无障碍坡道 |
| 围墙/护栏 | 砖/石材/木材/金属 | 隐私与安全性 |
| 台阶 | 高15cm×深30cm为舒适 | 每10-12级设休息平台 |
| 水景 | 喷泉/跌水/静水池/溪流 | 循环过滤系统 |
| 露台/平台 | 木材/复合木/石材 | 与室内地面高度差≤20cm |

### 2.4 可持续景观设计

| 策略 | 方法 | 优势 |
|------|------|------|
| 雨水花园（Rain Garden） | 低洼集水区+耐湿植物 | 减少径流，过滤污染 |
| 原生植物 | 适应当地气候的本地植物 | 低维护，节水 |
| 旱生景观（Xeriscaping） | 耐旱植物+覆盖物 | 节水50-80% |
| 绿色屋顶 | 屋顶覆土种植 | 保温、雨水管理 |
| 透水铺装 | 雨水渗透地下 | 补充地下水 |
| 垂直绿化 | 墙面攀缘/模块式绿化 | 降温、美化 |

## 3. 展陈设计（Exhibition Design）

### 3.1 展示类型

| 展示类型 | 空间特征 | 设计重点 | 案例 |
|----------|----------|----------|------|
| 博物馆陈列 | 长期、恒温恒湿、保护性 | 叙事流线、展柜照明 | 故宫博物院展厅 |
| 画廊展览 | 周期性更换、灵活 | 墙面布局、灯光色温 | 白立方画廊 |
| 商业展厅 | 品牌推广、产品体验 | 沉浸式体验、互动 | 苹果旗舰店 |
| 展会/博览会 | 短期、模块化搭建 | 快速搭建、视觉冲击 | 世博会国家馆 |
| 临时装置 | 快闪、创意展示 | 新媒体技术 | 艺术装置 |

### 3.2 展陈设计要素

| 要素 | 设计考量 |
|------|----------|
| 参观动线 | 顺时针方向（自然习惯），避免折返/交叉 |
| 叙事序列 | 起→承→转→合，每个展区有明确的叙事段落 |
| 照明设计 | 展品表面照度200-500 lux，紫外线过滤 |
| 展柜设计 | 微环境控制（温湿度）、防反光玻璃、安全锁 |
| 文字与图形 | 各级标题清晰、多语言、无障碍高度（视线高度） |
| 多媒体 | 触控屏、投影、VR/AR、声音导览 |
| 互动性 | 动手操作/参与体验——提升参与感与记忆度 |

### 3.3 人类工效学在展陈中的应用

| 标准 | 参数 |
|------|------|
| 视线高度 | 站立：145-155cm；坐姿：115-125cm |
| 最佳视域 | 视线上下30°以内 |
| 阅读距离 | 文字：40-60cm；大型展品：1-3m |
| 标签高度 | 距地面120-140cm（站立阅读）|
| 过道宽度 | 主通道≥1.8m；次通道≥1.2m |

## 4. 空间与人的关系

### 4.1 空间尺度体验

| 尺度类型 | 比例 | 心理感受 |
|----------|------|----------|
| 亲切尺度 | H/W < 1 | 亲密、安全 |
| 舒适尺度 | H/W ≈ 1 | 均衡、宜人 |
| 宏伟尺度 | H/W > 2 | 崇高、压制 |
| 超尺度 | H/W >> 2 | 神圣、震撼 |

### 4.2 空间的感知要素

| 要素 | 设计方法 | 效果 |
|------|----------|------|
| 围合度 | 墙/顶/地界定 | 开放感vs封闭感 |
| 光线 | 方向/强度/色温 | 明快vs昏暗 |
| 材质 | 光滑/粗糙/软硬 | 触觉暗示 |
| 声音 | 吸音/反射 | 安静vs热闹 |
| 气味 | 自然/人工 | 记忆唤起 |
| 温度 | 空调/自然通风 | 舒适度 |

### 4.3 亲生物设计（Biophilic Design）

将自然元素引入建筑环境以促进人的健康与幸福感：

| 原则 | 设计方法 |
|------|----------|
| 直接自然体验 | 自然光、植物、水景、自然通风 |
| 间接自然体验 | 天然材料（木/石/竹）、自然色彩、有机形态 |
| 空间与场所 | 避难感（凹室/角落）、远景（开阔视野）|
| 自然形态 | 拱形/曲线/不对称——模仿自然有机形态 |

## 5. 可持续环境设计

| 认证体系 | 国家/地区 | 重点 |
|----------|----------|------|
| LEED | 美国 | 绿色建筑设计+施工+运营 |
| BREEAM | 英国 | 建筑可持续性评估 |
| WELL | 美国 | 健康建筑标准——空气/水/光/舒适 |
| 绿色建筑评价标识 | 中国 | 节能/节地/节水/节材/室内环境 |

**被动式设计策略**：
1. 建筑朝向：坐北朝南（北半球）获取冬季日照
2. 自然通风：风压与热压驱动——促进空气流通
3. 遮阳设计：遮阳板/百叶/悬挑控制夏季日晒
4. 蓄热体：混凝土/砖石蓄热白天+夜间释放
5. 保温隔热：外墙/屋顶/窗户的保温层
6. 自然采光：天窗/高窗/光导管减少人工照明
