# 服装设计

## 1. 服装设计流程

### 1.1 前期研究（Research）

| 步骤 | 内容 | 工具/方法 |
|------|------|----------|
| 趋势调研 | 分析流行趋势、秀场、街拍 | WGSN、Vogue Runway、Pinterest |
| 灵感收集 | 从艺术/自然/建筑/文化中提取元素 | 灵感板（Mood Board） |
| 色彩方案 | 确定季度色彩方向 | Pantone色卡、趋势报告 |
| 面料调研 | 接触面料样品，了解季节面料 | 面料展（Première Vision） |
| 主题确定 | 整合以上为统一的设计主题 | 主题板（Theme Board） |

### 1.2 设计草图阶段

| 草图类型 | 用途 | 特点 |
|----------|------|------|
| 服装效果图（Croquis） | 展示服装穿在身上的外观 | 9-10头身比例，风格化 |
| 平铺图（Flat Drawing） | 展示服装的平面结构 | 精确比例，标示缝线/细节 |
| 技术款式图（Technical Flat） | 标注细节/尺寸/工艺 | 制版师和样衣工使用 |
| 细节放大图 | 放大特殊工艺/装饰 | 刺绣/褶皱/五金件 |

### 1.3 制版与样衣（Pattern Making & Sampling）

| 阶段 | 操作 | 输出 |
|------|------|------|
| 原型制版 | 按标准尺寸（如M码）制作基础纸样 | 基础样版 |
| 立裁（Draping） | 在人台上直接裁剪面料 | 立体造型确认 |
| 第一版样衣 | 白胚布或代用料制作 | 版型/尺寸修正 |
| 修改 | 根据试穿调整版型 | 修正版 |
| 面料版样衣 | 正式面料制作 | 最终确认样衣 |
| 放码（Grading） | 将基础版缩放为S/M/L/XL | 各尺码纸样 |
| 排料（Marker Making） | 在面料上排列纸样以最小浪费 | 排料图 |

## 2. 服装结构工艺

### 2.1 缝型（Seams）

| 缝型 | 特征 | 适用 |
|------|------|------|
| 平缝（Plain Seam） | 面料正面相对缝合+锁边 | 最基础，通用 |
| 来去缝（French Seam） | 两次缝合包裹毛边 | 薄料/高级成衣 |
| 包边缝（Flat-felled Seam） | 一条缝份包裹另一条 | 牛仔裤/衬衫/户外装 |
| 滚边缝（Bound Seam） | 用滚边条包裹缝份 | 无里衬外套 |
| 搭缝（Lapped Seam） | 两片重叠车缝 | 牛仔/运动装 |

### 2.2 省道与褶皱

| 结构 | 功能 | 位置 |
|------|------|------|
| 胸省（Bust Dart） | 收胸凸量 | 前后衣片 |
| 腰省（Waist Dart） | 收腰量 | 前后衣片 |
| 肩省（Shoulder Dart） | 收肩胛骨凸量 | 后衣片 |
| 普利特褶（Pleat） | 刀褶/盒褶/风琴褶 | 裙/裤/袖 |
| 碎褶（Gather） | 面料缩缝形成碎褶 | 腰部/袖口/领口 |
| 塔克褶（Tuck） | 装饰性折叠缝死 | 衬衫前胸/袖口 |

### 2.3 领型、袖型与门襟

| 分类 | 类型 | 特征 |
|------|------|------|
| 领型 | 立领/翻领/西装领/古巴领/彼得潘领/青果领 | 核心款式变化点 |
| 袖型 | 装袖/插肩袖/和服袖/蝙蝠袖/泡泡袖/灯笼袖 | 影响肩部轮廓 |
| 门襟 | 拉链门襟/纽扣门襟/暗门襟/法式门襟 | 闭合方式与装饰 |

## 3. 面料知识

### 3.1 天然纤维

| 纤维 | 来源 | 特性 | 护理 | 用途 |
|------|------|------|------|------|
| 棉（Cotton） | 棉籽纤维 | 透气、吸湿、柔软、易皱 | 可机洗 | T恤、衬衫、牛仔裤 |
| 亚麻（Linen） | 亚麻茎纤维 | 凉爽、吸湿、易皱、硬挺 | 手洗/干洗 | 夏季衬衫/裤装 |
| 羊毛（Wool） | 绵羊/美利奴 | 保暖、弹性、抗皱 | 干洗 | 西装、大衣、毛衣 |
| 丝绸（Silk） | 蚕茧 | 光滑、悬垂好、吸湿 | 干洗 | 礼服、衬衫、丝巾 |
| 羊绒（Cashmere） | 克什米尔山羊绒毛 | 极软、轻、暖 | 干洗 | 高端毛衣、大衣 |

### 3.2 化学纤维

| 纤维 | 原料 | 特性 | 用途 |
|------|------|------|------|
| 涤纶（Polyester） | PET石油基 | 挺括、抗皱、耐磨、快干 | 运动服、套装、外套 |
| 锦纶（Nylon） | 聚酰胺 | 弹性好、强韧、轻 | 丝袜、运动服、泳装 |
| 腈纶（Acrylic） | 聚丙烯腈 | 蓬松、像羊毛 | 毛衣、毯子 |
| 氨纶（Spandex/Elastane） | 聚氨酯 | 极高弹性（可拉伸500%） | 紧身衣、运动内衣 |
| 粘胶（Viscose/Rayon） | 木浆再生纤维 | 柔软、悬垂好 | 连衣裙、衬衫 |
| 莫代尔（Modal） | 山毛榉木浆 | 比粘胶更强、更柔软 | 内衣、床品 |

### 3.3 织物组织

| 组织类型 | 结构 | 特征 | 示例 |
|----------|------|------|------|
| 平纹（Plain Weave） | 一上一下交织 | 平整、最基础 | 棉布、府绸、雪纺 |
| 斜纹（Twill Weave） | 斜向纹路 | 更密实、悬垂好 | 牛仔布、卡其 |
| 缎纹（Satin Weave） | 浮长线形成光泽面 | 光滑、有光泽 | 绸缎 |
| 针织（Knitting） | 线圈串套结构 | 弹性好 | T恤针织布、毛衣 |
| 无纺布（Non-woven） | 纤维直接粘合 | 一次性 | 口罩、购物袋 |

**针织 vs 梭织**：

| 对比 | 针织 | 梭织（机织） |
|------|------|-------------|
| 结构 | 线圈串套 | 经纬线交织 |
| 弹性 | 有弹性 | 几乎没有弹性（斜裁除外）|
| 脱散性 | 易脱散 | 不脱散 |
| 应用 | T恤、毛衣、运动服 | 衬衫、西装、牛仔裤 |

## 4. 时装史概览

| 年代 | 风格特征 | 代表设计师/事件 |
|------|----------|----------------|
| 1900s-1910s | S形紧身胸衣（S-curve corset），爱德华时代 | Paul Poiret 解放女性腰部 |
| 1920s | 直筒低腰（Flapper），短发，装饰艺术 | Coco Chanel 小黑裙、香奈儿套装 |
| 1930s | 斜裁（Bias cut），修长优雅，好莱坞风格 | Madeleine Vionnet、Elsa Schiaparelli |
| 1940s | 战争——实用套装、短裙、阔肩 | Dior New Look 1947（蜂腰+大裙摆）|
| 1950s | 女性化回归——A字裙、紧身毛衣 | Balenciaga、Givenchy |
| 1960s | 迷你裙、太空风格、波普图案 | Mary Quant（迷你裙）、André Courrèges |
| 1970s | 喇叭裤、迪斯科风格、嬉皮士 | YSL（吸烟装）、Halston |
| 1980s | 权力套装（Power Suit）、大垫肩、亮色 | Giorgio Armani、Versace、Thierry Mugler |
| 1990s | 极简主义、垃圾摇滚（Grunge） | Calvin Klein、Martin Margiela、川久保玲 |
| 2000s | 低腰裤、波西米亚风、快速时尚 | Alexander McQueen、Zara/H&M崛起 |

## 6. 制衣工艺与细节

### 6.1 衬里与辅料

| 辅料类型 | 功能 | 材料 |
|----------|------|------|
| 粘合衬（Interfacing） | 提供挺度和形状支撑 | 有纺/无纺粘合衬 |
| 里衬（Lining） | 遮盖内部结构，方便穿脱 | 铜氨丝、涤纶、粘胶 |
| 垫肩（Shoulder Pad） | 塑造肩部轮廓 | 海绵、羊毛毡 |
| 拉链（Zipper） | 开合 | 金属/尼龙/隐形 |
| 纽扣（Button） | 开合+装饰 | 贝壳/树脂/金属/包布 |
| 钩扣/按扣 | 隐蔽开合 | 金属/塑料 |

### 6.2 后整理工艺

| 工艺 | 方法 | 效果 |
|------|------|------|
| 水洗（Washing） | 工业水洗/酵磨 | 柔软、做旧 |
| 石洗（Stone Wash） | 加浮石洗涤 | 牛仔布做旧 |
| 砂洗（Sand Wash） | 加砂洗涤 | 丝绸/雪纺柔化 |
| 植绒（Flocking） | 静电植绒 | 天鹅绒质感 |
| 刺绣（Embroidery） | 机绣/手绣 | 装饰 |
| 热压印花 | 热转印/硅胶印花 | 图案转印 |
| 激光切割 | 激光裁切 | 精确+边缘封边 |

### 6.3 尺码系统对比

| 尺码体系 | 依据 | 地区 | 特点 |
|----------|------|------|------|
| 欧码（EU） | 胸围cm | 欧洲 | 34-52，最通用 |
| 英码（UK） | 英寸 | 英国 | 6-24，比US码小2号 |
| 美码（US） | 英寸 | 美国 | 0-22，比UK码大2号 |
| 日码（JP） | 胸围cm | 日本 | S/M/L/LL+数字 |
| 中码（CN） | 胸围/身高 | 中国 | 155/80A 三位加体型 |

## 7. 时尚造型基础

### 7.1 体型与服装

| 体型 | 特征 | 搭配建议 |
|------|------|----------|
| A型（梨形） | 肩窄臀宽 | 强调肩部（垫肩/泡泡袖），A字裙放开 |
| H型（直筒） | 肩腰臀等宽 | 腰带制造腰线，结构感外套 |
| X型（沙漏） | 肩宽≈臀宽，腰细 | 收腰剪裁最佳——凸显曲线 |
| O型（苹果） | 腹部突出 | 垂感面料+V领，避免腰部紧束 |
| Y型（倒三角） | 肩宽臀窄 | 简约上装+膨胀感下装（阔腿裤/伞裙）|

### 7.2 经典款式清单

| 单品 | 起源 | 风格特征 |
|------|------|----------|
| 小黑裙（LBD） | Chanel 1926 | 百搭、优雅、简洁 |
| 白色衬衫 | 19世纪男性衬衣 | 经典通勤必备 |
| 牛仔裤 | Levi's 1873 | 自由、百搭、耐穿 |
| 风衣 | Burberry 一战军装 | 经典外套 |
| 西装外套 | 男性西装演变 | 职业/正式 |
| 针织开衫 | Coco Chanel 引入女装 | 休闲层搭 |
| A字裙 | Dior New Look 演变 | 女性化+方便 |

### 7.3 面料护理标识

| 标识 | 含义 | 操作建议 |
|------|------|----------|
| 水洗槽 | 最高洗涤温度 | 40°C以下为安全水温 |
| 三角形 | 漂白许可 | 含氯/不含氯漂白剂 |
| 圆圈 | 干洗 | P=全氯乙烯，F=碳氢溶剂 |
| 正方形+圆 | 烘干 | 点=低温，线=悬挂晾干 |
| 熨斗 | 熨烫温度 | 点=低温（合成），两/三点=中/高温 |
| 圆圈+X | 不可干洗 | 仅可水洗 |
