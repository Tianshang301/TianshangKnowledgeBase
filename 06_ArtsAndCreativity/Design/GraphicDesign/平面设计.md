# 平面设计

## 1. 字体排印（Typography）

### 1.1 字体分类

| 类别 | 特征 | 代表字体 | 适用场景 |
|------|------|----------|----------|
| 衬线体（Serif） | 笔画末端有装饰性衬线 | Garamond、Times New Roman、Bodoni | 正文印刷、正式出版物、书籍 |
| 无衬线体（Sans-serif） | 笔画均匀，无衬线 | Helvetica、Futura、Roboto、Inter | 屏幕显示、标题、现代风格 |
| 手写体（Script） | 连笔/手书风格 | Brush Script、Snell Roundhand | 邀请函、logo、装饰性标题 |
| 展示体（Display） | 装饰性强/个性突出 | Impact、各式艺术字体 | 海报标题、logo、仅限大字 |
| 等宽体（Monospace） | 每个字符宽度相等 | Courier New、Consolas、Fira Code | 代码编辑、打字机效果 |
| 黑体（Heiti/简体中文） | 无衬线，中宫宽松 | 思源黑体、微软雅黑 | 屏幕显示，中文正文 |
| 宋体（Songti/简体中文） | 横细竖粗，有装饰角 | 思源宋体、方正宋体 | 中文书籍正文 |

### 1.2 字体解剖（Type Anatomy）

| 部件 | 英文 | 定义 |
|------|------|------|
| 基线 | Baseline | 字母底部对齐的参考线 |
| x高度 | x-height | 小写字母x的高度（不含上伸/下延） |
| 上伸部 | Ascender | 小写字母超出x高度的部分（b/d/f/h/k/l） |
| 下延部 | Descender | 小写字母低于基线的部分（g/j/p/q/y） |
| 大写高度 | Cap Height | 大写字母的高度 |
| 字怀 | Counter | 字母内部的封闭或半封闭空白（o/b/p的内部） |
| 笔干 | Stem | 字母主要的垂直/斜向笔画 |
| 衬线 | Serif | 笔画末端的装饰线脚 |
| 合字 | Ligature | 两个/多个字母合并为一个字形（fi/fl/ffi） |
| 字偶 | Kerning pair | 特定字母组合的间距调整（AV/To等） |

### 1.3 字体层次（Type Hierarchy）

| 层级 | 用途 | 字号比例 | 字重 | 行距 |
|------|------|----------|------|------|
| 标题（Headline） | 吸引注意，概括内容 | 2-4倍正文 | Bold/ExtraBold | 紧密 |
| 副标题（Subhead） | 分层信息 | 1.5-2倍正文 | SemiBold | 适中 |
| 正文（Body） | 主要内容阅读 | 基数（9-16pt） | Regular | 1.4-1.6倍 |
| 说明（Caption） | 注释/图注/次要信息 | 0.7-0.85倍正文 | Regular/Light | 1.2-1.4倍 |

### 1.4 字距与行距

| 概念 | 定义 | 调整方法 | 印刷参考值 |
|------|------|----------|-----------|
| 字偶距（Kerning） | 特定字母对的间距 | 手动微调 | 按视觉均匀 |
| 字间距（Tracking） | 整体字母间距 | 统一增减 | 正文正常，标题稍松 |
| 行距（Leading） | 基线与基线距离 | 按比例 | 正文1.4-1.6×字号 |
| 段落间距 | 段落间空距 | 行距的1-2倍 | 避免孤立行 |

### 1.5 对齐方式

| 对齐 | 特征 | 适用 |
|------|------|------|
| 左对齐（Flush Left） | 左端整齐，右端不规则 | 正文/阅读文本首选 |
| 右对齐（Flush Right） | 右端整齐，左端不规则 | 特殊艺术效果/表格数字 |
| 居中对齐（Centered） | 两端皆不规则 | 标题/邀请函/少量文本 |
| 两端对齐（Justified） | 两端皆整齐 | 书籍/报纸/杂志——需注意字距不均 |
| 强制对齐 | 末行也两端对齐 | 仅用于特殊设计 |

## 2. 版式布局（Layout）

### 2.1 网格系统（Grid Systems）

| 网格类型 | 结构 | 适用 |
|----------|------|------|
| 手稿网格（Manuscript Grid） | 单一矩形内容区 | 书籍正文、长篇阅读 |
| 分栏网格（Column Grid） | 垂直分为多栏 | 杂志、报纸、宣传册 |
| 模块网格（Modular Grid） | 水平和垂直方向均匀分割 | 画册、目录、数据展示 |
| 层次网格（Hierarchical Grid） | 按内容优先级不规则分割 | 网页、海报、信息图 |

**网格设计原则**：
1. 设置页面边距（内外/上下比例通常2:3或黄金比）
2. 确定栏数（通常2-12栏）
3. 栏间距（gutter）一致
4. 模块高度统一
5. 图文跨栏时保持视觉对齐

### 2.2 留白（White Space）

- 宏观留白：页边距、栏间距、大块空白区域
- 微观留白：字间距、行距、段落间距
- 留白不是浪费，而是设计元素——创造层次、引导视线、提升高级感

### 2.3 视觉层级（Visual Hierarchy）

| 层级手段 | 操作方法 |
|----------|----------|
| 大小差异 | 最大元素最先看到 |
| 色彩对比 | 鲜艳/深色最先吸引 |
| 粗细对比 | 粗体/细体的区分 |
| 空间位置 | 左上角优先（Z型浏览模式） |
| 隔离 | 独立的元素更突出 |
| 方向 | 水平vs倾斜/垂直 |

### 2.4 Z型与F型浏览模式

| 模式 | 描述 | 适用 |
|------|------|------|
| Z型（Z-Pattern） | 左上→右上→左下→右下 | 视觉均衡的页面、海报 |
| F型（F-Pattern） | 顶部横读→下移→再横读→竖扫 | 文字密集型网页 |

## 3. 品牌视觉设计

### 3.1 标志类型（Logo Types）

| 类型 | 说明 | 示例 |
|------|------|------|
| 文字标志（Wordmark） | 公司全名设计为字体标志 | Coca-Cola、Google、FedEx |
| 字母标志（Lettermark） | 公司首字母缩写 | IBM、NASA、CNN |
| 图形标志（Pictorial Mark） | 具象/抽象图形代表品牌 | Apple、Twitter、Target |
| 抽象标志（Abstract Mark） | 几何抽象符号 | Nike Swoosh、Adidas三叶草 |
| 吉祥物（Mascot） | 人物/动物/角色形象 | KFC、Pringles、米其林 |
| 徽章（Emblem） | 文字图形结合在封闭形状内 | Starbucks、Harley-Davidson |
| 组合标志（Combination Mark） | 文字+图形组合 | McDonald's、Burger King |

### 3.2 品牌识别系统

| 元素 | 内容 | 交付物 |
|------|------|--------|
| 标志 | 主标志+变体（横/竖/灰） | AI/EPS矢量文件 |
| 色彩 | 主色+辅助色+中性色 | CMYK/Pantone/HEX值 |
| 字体 | 标题字体+正文字体+字重 | 字体使用规范 |
| 辅助图形 | 图形/纹理/图案 | 品牌纹样 |
| 应用规范 | 名片/信纸/包装/数字媒体 | 品牌手册（Brand Guidelines） |

### 3.3 色彩系统

| 色彩模式 | 应用场景 | 特点 |
|----------|----------|------|
| CMYK | 印刷品 | 青色(C)+洋红(M)+黄(Y)+黑(K)—减色法 |
| Pantone（潘通） | 专色印刷 | 编号系统，精确色差控制 |
| RGB | 屏幕显示 | 红(R)+绿(G)+蓝(B)—加色法 |
| HEX | 网页设计 | 6位十六进制（#FF0000）|

## 4. 印刷与数字输出

### 4.1 分辨率

| 媒介 | 分辨率标准 | 注意 |
|------|-----------|------|
| 印刷 | 300 DPI | 1英寸含300个像素点 |
| 大幅印刷（海报/灯箱） | 150-200 DPI | 观看距离远可降低 |
| 屏幕（网页/APP） | 72 PPI | 1英寸含72个像素点 |
| Retina屏 | 144-300 PPI | 2x/3x倍图 |

### 4.2 印前设置

| 项目 | 标准值 | 说明 |
|------|--------|------|
| 出血（Bleed） | 3mm（各边） | 裁切误差预留 |
| 安全区（Safe Area） | 出血向内5mm | 重要内容不靠近裁切线 |
| 色彩模式 | CMYK | 印刷品必须CMYK |
| 叠印/镂空 | 黑色文字设叠印 | 防止套印不准露白边 |

### 4.3 文件格式

| 格式 | 特性 | 用途 |
|------|------|------|
| PDF | 跨平台、可嵌入字体/图片 | 印刷交付/电子分发 |
| AI（Illustrator） | 矢量、可编辑 | 标志/插图原始文件 |
| EPS | 矢量（旧格式） | 印刷行业兼容格式 |
| PNG | 无损、支持透明 | 网页UI/图标 |
| JPEG/JPG | 有损压缩、文件小 | 照片/图片展示 |
| SVG | 矢量、代码描述、可缩放 | 网页图标/插画 |
| WebP | Google新格式、支持透明+压缩 | 网页（替代PNG/JPEG） |

## 5. 信息设计

### 5.1 数据可视化

| 图表类型 | 适用数据 | 设计要点 |
|----------|----------|----------|
| 柱状图 | 分类对比 | 基线从0开始、等宽、间距<柱宽 |
| 折线图 | 连续趋势/时间序列 | 平滑或转折清晰、标注关键点 |
| 饼图 | 占比关系 | 不超过5个扇区，按大小顺时针排列 |
| 散点图 | 两变量相关性 | 趋势线、标注异常点 |
| 雷达图 | 多维度对比 | 维度统一量纲、不超过6维 |
| 热力图 | 矩阵/密度分布 | 色阶清晰、标注数值 |

### 5.2 信息图（Infographic）

信息图是数据+叙事+视觉设计的结合。设计原则：
1. 明确定义叙事线——数据应支撑故事而非堆砌
2. 视觉编码准确（面积/长度/色彩映射正确）
3. 层级清晰——大标题→副标题→图表→解释文字
4. 格式塔原理指导布局——接近/相似/闭合原则
5. 避免图表垃圾（chartjunk）——3D伪立体/过度装饰

### 5.3 图标设计原则

| 原则 | 说明 |
|------|------|
| 识别性 | 用户一看即知其含义（不需要文字解释）|
| 一致性 | 同一系统内线宽/圆角/风格统一 |
| 简洁性 | 去除多余细节，保留最本质的形状 |
| 可缩放 | 小尺寸下仍然可识别（关键细节不丢失）|
| 风格化 | 线型/面型/双色——保持系统统一 |

## 6. 印刷工艺详解

### 6.1 印刷方式对比

| 工艺 | 原理 | 起印量 | 单件成本 | 适用 |
|------|------|--------|----------|------|
| 胶版印刷（Offset） | 油水相斥→橡皮布转印 | ≥500 | 极低（量大） | 画册/杂志/包装 |
| 数字印刷 | 静电/喷墨直接成像 | 1起 | 固定/较高 | 名片/样张/小批量 |
| 丝网印刷 | 刮刀推油墨过网版孔 | ≥50 | 中 | T恤/特殊材质/海报 |
| 凹版印刷 | 雕刻凹版→填墨→压印 | ≥5000 | 极低 | 邮票/钞票/高质画册 |
| 柔版印刷 | 凸版+液体油墨 | ≥1000 | 低 | 标签/薄膜/瓦楞纸 |
| 烫金/UV | 热压金箔/UV光固 | ≥100 | 较高 | 高端包装/封面 |

### 6.2 纸张类型

| 纸张 | 特性 | 适用 |
|------|------|------|
| 铜版纸（光面/哑面） | 表面涂布，平滑细腻 | 画册/海报/杂志 |
| 胶版纸 | 不涂布，吸墨好 | 书籍正文/信纸 |
| 特种纸 | 纹理/触感/色彩特殊 | 封面/名片/高端印刷 |
| 新闻纸 | 便宜、易发黄 | 报纸 |
| 卡纸/灰板 | 厚、硬 | 包装盒/书封 |
