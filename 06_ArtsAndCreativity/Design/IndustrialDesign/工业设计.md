# 工业设计

## 1. 设计流程（Design Process）

### 1.1 五阶段流程

| 阶段 | 活动内容 | 输出物 | 工具 |
|------|----------|--------|------|
| 调研 | 用户研究、市场分析、竞品分析、技术调研 | 调研报告、用户画像 | 访谈、问卷、数据分析 |
| 构思 | 头脑风暴、思维导图、概念发散 | 概念草图、思维导图 | 手绘、白板、便利贴 |
| 概念发展 | 方案筛选、深化设计、CMF确定 | 效果图、3D模型、功能模型 | Rhino、SolidWorks、渲染 |
| 原型测试 | 快速原型、用户测试、功能验证 | 3D打印原型、功能样机 | 3D打印机、CNC |
| 量产 | 模具制造、生产流程、QC | 量产产品、BOM表 | CAD/CAM、注塑机 |

### 1.2 迭代特性

工业设计不是线性的——每个阶段都可能返回上一阶段。原型测试发现问题→回到概念发展修改→重新原型。迭代次数取决于预算和开发周期。

## 2. 产品草绘与渲染

### 2.1 草绘类型

| 草绘类型 | 目的 | 精细度 | 工具 |
|----------|------|--------|------|
| 概念草绘（Ideation Sketch） | 快速捕捉想法 | 粗略、多角度 | 铅笔/钢笔 |
| 透视草绘（Perspective Sketch） | 展示三维形态 | 完整透视结构 | 马克笔+彩铅 |
| 爆炸图（Exploded View） | 展示部件关系 | 分解结构 | 针管笔+数字工具 |
| 效果图（Rendering） | 展示最终外观 | 高真实感 | 马克笔/Keyshot |
| 工程草图（Technical Drawing） | 标注尺寸/细节 | 精确三视图 | CAD软件 |

### 2.2 产品草绘基础

| 透视类型 | 消失点数量 | 适用 |
|----------|-----------|------|
| 一点透视 | 1个VP | 正面展示、手持产品 |
| 两点透视 | 2个VP | 标准产品展示 |
| 三点透视 | 3个VP | 大型产品/仰视效果 |
| 等角透视（Isometric） | 无（平行线） | 工程图示 |

**产品草绘步骤**：
1. 快速几何概括（方块/圆柱/球体组合）
2. 透视辅助线
3. 细化轮廓与分型线
4. 马克笔铺色（先浅后深）
5. 高光与阴影（白色彩铅/修正液提亮）

### 2.3 数字渲染

| 软件 | 建模能力 | 渲染引擎 | 主要领域 |
|------|----------|----------|----------|
| SolidWorks | 参数化实体建模 | PhotoView 360 | 机械/工程产品 |
| Rhino 3D | 自由曲面（NURBS） | V-Ray/Flamingo | 消费电子/汽车/造型 |
| Fusion 360 | 参数化+自由曲面 | 内置渲染 | 综合性产品设计 |
| Alias | A级曲面 | 内置/VRED | 汽车/交通工具 |
| Blender | 多边形/细分 | Cycles/Eevee | 开源通用 |
| Keyshot | 非建模—仅渲染 | CPU+GPU实时 | 快速高品质渲染 |

## 3. 人机工程学（Ergonomics）

### 3.1 人体测量学（Anthropometry）

| 百分位 | 含义 | 设计应用 |
|--------|------|----------|
| 5th | 只有5%的人小于此值 | 最小可达距离/操作范围 |
| 50th | 中位数 | 平均尺寸参考 |
| 95th | 只有5%的人大于此值 | 最大尺寸/间隙设计 |
| 调节范围 | 5th至95th | 可调节设计的包容范围 |

**设计原则**：
- 可调节的，按5th-95th范围设计
- 不可调节的，按目标用户中位数或特定百分位
- 安全相关的，按照最不利情况（如踏板力按95th男性）
- 间隙/通过尺寸，按95th
- 可达/够取，按5th

### 3.2 有效使用范围（Reach Envelopes）

| 区域 | 水平范围 | 说明 |
|------|----------|------|
| 最佳区 | 肩宽范围内 | 最舒适操作区 |
| 可达区 | 手臂自然伸展范围 | 需弯腰/伸臂 |
| 极限区 | 身体前倾+手臂全伸 | 不宜频繁操作 |

### 3.3 认知人机工程学

**Don Norman 设计心理学核心概念**：

| 概念 | 定义 | 设计应用 |
|------|------|----------|
| 预设用途（Affordance） | 物品的使用可能性 | 门推/拉的把手暗示 |
| 能指（Signifier） | 指示预设用途的视觉线索 | 按钮上的"按下"文字 |
| 映射（Mapping） | 控制器与功能的空间关系 | 炉灶旋钮对应炉头位置 |
| 反馈（Feedback） | 操作后的即时回应 | 按钮点击声、震动、亮灯 |
| 概念模型（Conceptual Model） | 使用者对系统的理解 | 可视化操作逻辑 |

**Norman 的七大设计原则**：
1. 利用知识和外界信息（减少记忆负担）
2. 简化任务结构（分步引导）
3. 可见性（操作功能一目了然）
4. 正确映射（自然对应关系）
5. 利用约束（物理/逻辑/文化）
6. 容错设计（可撤销/恢复）
7. 标准化（统一操作逻辑）

## 4. 材料与制造工艺

### 4.1 塑料

| 塑料类型 | 特性 | 加工方式 | 产品应用 |
|----------|------|----------|----------|
| ABS | 强韧、耐冲击、可电镀 | 注塑、3D打印 | 外壳、玩具、家电 |
| 聚丙烯（PP） | 轻、耐化学、可弯折 | 注塑、吹塑 | 容器、日用品 |
| 聚碳酸酯（PC） | 透明、超高强度 | 注塑、挤出 | 镜片、防护罩、电器 |
| 亚克力（PMMA） | 透明、光学好 | 浇铸、热弯 | 展示架、鱼缸、灯具 |
| 尼龙（PA） | 耐磨、自润滑 | 注塑、挤出 | 齿轮、轴承、拉链 |
| PET/PETG | 透明、韧性 | 吹塑、3D打印 | 饮料瓶、包装 |

**塑料加工方式**：

| 工艺 | 原理 | 适合 | 成本 |
|------|------|------|------|
| 注塑（Injection Molding） | 熔融塑料高压注入模具 | 大批量、复杂形状 | 模具贵/单品便宜 |
| 吹塑（Blow Molding） | 塑料管充气膨胀至模具 | 中空容器 | 中等 |
| 热成型（Thermoforming） | 塑料片加热后真空吸附 | 托盘、罩壳 | 模具成本低 |
| 3D打印 | 逐层堆积（FDM/SLA/SLS） | 原型、小批量 | 无模具/单品贵 |

### 4.2 金属

| 金属 | 特性 | 加工 | 产品应用 |
|------|------|------|----------|
| 铝合金 | 轻、耐腐蚀 | 压铸、挤压、CNC | 笔记本外壳、手机框架 |
| 不锈钢 | 强度高、防锈 | 冲压、焊接、CNC | 厨具、医疗设备 |
| 碳钢 | 强度极高、价廉 | 冲压、焊接、锻造 | 结构件、工具 |
| 钛合金 | 极轻、强度高、贵金属 | CNC、3D打印 | 医疗植入、高端手表 |

**金属加工方式**：
- 冲压（Stamping）：金属板在模具中冲裁成形
- 铸造（Casting）：熔融金属倒入模具
- 挤压（Extrusion）：金属通过模具挤出连续截面
- CNC加工：计算机控制铣/车/钻，精度可达0.01mm
- 压铸（Die Casting）：熔融金属高压注入金属模具（常用于铝合金/锌合金）

### 4.3 表面处理

| 处理方式 | 效果 | 应用材料 |
|----------|------|----------|
| 喷漆（Painting） | 色彩丰富，多种质感 | 金属/塑料 |
| 电镀（Electroplating） | 金属光泽 | ABS/金属 |
| 阳极氧化（Anodizing） | 耐磨+可着色 | 铝合金 |
| 喷砂（Sandblasting） | 磨砂质感 | 金属/玻璃/塑料 |
| 拉丝（Brushing） | 金属纹理 | 不锈钢/铝 |
| IMD（模内装饰） | 图案嵌入塑料内 | 注塑件 |
| 皮革/织物包覆 | 高级触感 | 电子产品 |

## 5. 面向X的设计（Design for X）

### 5.1 DFM（面向制造的设计）

| 原则 | 说明 |
|------|------|
| 简化形状 | 避免倒扣、复杂曲面（增加模具成本） |
| 统一壁厚 | 防止缩水/翘曲，保持均匀冷却 |
| 拔模角度 | 1-3°斜角便于脱模 |
| 圆角过渡 | 避免尖角引起应力集中 |
| 合理分型线 | 分型线位置影响外观 |
| 减少装配零件 | 零件合并/一体化设计 |

### 5.2 DFA（面向装配的设计）

| 原则 | 说明 |
|------|------|
| 减少零件数量 | 零件数越少，装配时间越短 |
| 模块化装配 | 按功能分区模块 |
| 自定位设计 | 零件有导向/卡扣便于对齐 |
| 对称性/防错 | 设计成只能正确安装 |
| 减少紧固件 | 卡扣替代螺丝 |
| 从上到下装配 | 重力辅助 |

### 5.3 可持续设计（Design for Sustainability）

| 策略 | 方法 |
|------|------|
| 材料减量 | 薄壁设计、中空结构 |
| 材料替代 | 可回收塑料、生物基材料 |
| 模块化 | 便于升级/维修，延长产品寿命 |
| 易拆解 | 零件可分离回收 |
| 延长寿命 | 经典设计不过时 |
| 减少包装 | 可回收/可降解包装 |

### 5.4 生命周期评估（LCA）

| 阶段 | 环境影响 | 设计优化方向 |
|------|----------|-------------|
| 原料获取 | 采矿/伐木/石油开采 | 使用回收材料 |
| 生产制造 | 加工能耗+废弃物排放 | 低能耗工艺+废料回收 |
| 运输分销 | 运输燃油/包装 | 轻量化+紧凑包装 |
| 使用阶段 | 使用能耗/耗材 | 节能设计+长效设计 |
| 报废处理 | 填埋/焚烧/回收 | 易拆解+材料标识 |

## 6. 著名工业设计师与代表作

| 设计师 | 国籍 | 代表作 | 设计理念 |
|--------|------|--------|----------|
| Dieter Rams | 德国 | Braun 收音机/计算器 | 好设计十大原则——简约/创新/实用/美学/诚实/持久/细节/环保/极致 |
| Philippe Starck | 法国 | 外星人榨汁机（Juicy Salif）| 民主设计——"设计是人人都能享受的"|
| Jonathan Ive | 英国 | iMac/iPod/iPhone | 极简主义——材料工艺与形式的统一 |
| Ettore Sottsass | 意大利 | 红色打字机Valentine | 孟菲斯集团——色彩鲜艳/反功能主义 |
| James Dyson | 英国 | Dual Cyclone 吸尘器/Airblade干手器 | 发明驱动、空气动力学 |
| Charles & Ray Eames | 美国 | Eames Lounge Chair | 模压胶合板技术、民主设计 |
| Jasper Morrison | 英国 | 日常物品设计 | 超平常（Super Normal）设计哲学 |
| Karim Rashid | 加拿大 | KF系列产品 | 有机曲线、荧光色感、感性极简 |

**Rams 好设计十大原则**：
1. 创新 —— 好的设计是创新的
2. 实用 —— 好的设计使产品有用
3. 美学 —— 好的设计是美的
4. 易懂 —— 好的设计让产品易懂
5. 谦虚 —— 好的设计是不张扬的
6. 诚实 —— 好的设计是诚实的
7. 耐久 —— 好的设计是耐久的
8. 细节 —— 好的设计贯穿每个细节
9. 环保 —— 好的设计是环保的
10. 极少 —— 好的设计是越少越好（Weniger, aber besser）
