# 艺术学习路线图

## 总览

```
              艺术学习路径
              ┌────┬────┐
              │    │    │
            绘画  摄影  设计
              │    │    │
              ▼    ▼    ▼
           初级阶段 → 中级 → 高级 → 专业/创作
```

## 一、绘画 (Drawing & Painting)

### 初级阶段 (0-6个月)

| 主题 | 学习内容 | 练习建议 |
|------|---------|---------|
| 基础形状 | 立方体、球体、圆锥、圆柱 | 每天画20个几何体 |
| 透视基础 | 一点/两点/三点透视 | 临摹建筑照片 |
| 线条控制 | 直线、曲线、排线 | 控制手腕和手臂 |
| 光影/调子 | 5大调子 (高光/亮部/明暗交界/暗部/反光) | 石膏几何体素描 |

**推荐资源**:
- 《素描的诀窍》 (Keys to Drawing)
- Drawabox.com (免费系统课程)
- Proko (YouTube)
- 素描本 + 2B/4B/6B 铅笔

### 中级阶段 (6个月-2年)

| 主题 | 学习内容 | 说明 |
|------|---------|------|
| 色彩理论 | 色轮、互补色、冷暖、色调 | 理解颜色关系 |
| 人体结构 | 比例 (8头身)、骨骼、肌肉 | 伯里曼人体解剖 |
| 风景画 | 天空、树木、水、山 | 室外写生 |
| 构图 | 三分法、黄金比例、引导线 | 设计画面结构 |

**练习计划**:
- 每天 30 分钟速写 (Gesture Drawing)
- 每周 1 幅完整素描
- 每月 1 幅色彩作品

**推荐资源**:

| 资源 | 类型 | 内容 |
|------|------|------|
| Proko | YouTube | 人体素描教程 |
| Ctrl+Paint | 网站 | 数字绘画基础 |
| Schoolism | 付费课程 | 专业绘画培训 |
| 《色彩与光线》 | 书籍 | 色彩理论 |
| 伯里曼人体结构 | 书籍 | 人体解剖 |

### 高级阶段 (2年+)

| 方向 | 内容 | 职业方向 |
|------|------|---------|
| 写实绘画 | 超写实、肖像 | 插画、艺术创作 |
| 概念设计 | 角色/场景/道具设计 | 游戏/影视概念 |
| 漫画/插画 | 分镜、叙事 | 漫画、绘本 |
| 数字绘画 | Photoshop/Procreate | 商业插画 |

**数字绘画工具**:

| 工具 | 平台 | 价格 |
|------|------|------|
| Procreate | iPad | ¥98 |
| Photoshop | Win/Mac | 订阅 |
| Clip Studio Paint | Win/Mac | ¥500+ |
| Krita | 全平台 | 免费 |
| Paint Tool SAI | Win | ¥500 |

### 绘画学习资源汇总

| 资源 | 类型 | 适合 |
|------|------|------|

| Proko | YouTube | 人体 |

## 二、摄影 (Photography)

### 初级阶段 (0-3个月)

| 主题 | 内容 | 实践 |
|------|------|------|
| 曝光三要素 | 光圈/快门/ISO | M档拍摄练习 |
| 景深控制 | 大光圈虚化、小光圈全清晰 | 人像/风景拍摄 |
| 对焦模式 | 单次对焦/连续对焦/手动 | 静物/运动拍摄 |
| 白平衡 | 色温 (K值)、不同光源 | 室内外对比 |

**曝光三角形**:
```
              ┌──────────┐
              │  ISO      │
              │ (感光度)   │
              └────┬─────┘
                   │
     ┌──────┐     ┌┴┐     ┌──────┐
     │ 光圈  │◄────┤ ├────►│  快门  │
     │ (F值) │     │ │     │ (速度) │
     └──────┘     └─┘     └──────┘
    (进光量)             (曝光时间)
```

**推荐器材**:
- 入门: 手机 + 后期 (Lightroom)
- 进阶: 微单 (Sony α6000, Fuji X-T30)
- 专业: 全画幅 (Sony A7M4, Canon R6)

### 中级阶段 (3-12个月)

| 主题 | 内容 |
|------|------|
| 构图法则 | 三分法、对称、引导线、框架构图 |
| 光线运用 | 黄金时刻、硬光/柔光、逆光 |
| 色彩后期 | 色调曲线、HSL、色彩分级 |
| 人像摄影 | 布光 (蝴蝶光/伦勃朗光)、摆姿 |
| 风景摄影 | 前景/中景/背景、长曝光 |

**后期软件**:

| 软件 | 平台 | 价格 | 用途 |
|------|------|------|------|
| Lightroom | Win/Mac/iOS | 订阅 | 调色管理 |
| Photoshop | Win/Mac | 订阅 | 精修 |
| Capture One | Win/Mac | ¥1500+ | 专业联机 |
| Snapseed | iOS/安卓 | 免费 | 手机后期 |
| VSCO | iOS/安卓 | 免费+内购 | 滤镜 |

### 高级阶段 (1年+)

| 方向 | 内容 | 职业 |
|------|------|------|
| 肖像摄影 | 影棚布光、时尚摄影 | 商业摄影师 |
| 风光摄影 | 长曝光、全景、航拍 | 风光摄影师 |
| 街拍 | 决定性瞬间、纪实 | 纪实摄影师 |
| 后期精修 | 皮肤、合成、调色 | 修图师 |

**摄影学习资源**:

| 资源 | 类型 | 内容 |
|------|------|------|
| Phlearn | 网站 | 后期教程 |
| The Art of Photography | YouTube | 摄影理论 |
| Sean Tucker | YouTube | 纪实/街头 |

## 三、设计 (Design)

### 初级阶段 (0-6个月)

| 主题 | 内容 | 练习 |
|------|------|------|
| 设计原则 | 对比、重复、对齐、亲密性 | 分析优秀海报 |
| 色彩 | 色轮、配色方案 (单色/互补/类比) | 创建调色板 |
| 字体 | 衬线/无衬线、字距、行距 | 字体搭配练习 |
| 布局 | 网格系统、留白 | 杂志版式模仿 |

**推荐资源**:
- 《写给大家看的设计书》 (Robin Williams)
- 《设计中的设计》 (原研哉)
- Canva (入门设计工具)

### 中级阶段 (6个月-1年)

| 方向 | 工具 | 学习内容 |
|------|------|---------|
| UI/UX | Figma | 用户界面、交互设计、原型 |
| 品牌设计 | Illustrator | Logo设计、VI系统 |
| 排版设计 | InDesign | 杂志、书籍、海报 |

**Figma 学习路径**:
```
1. 基本操作 (形状/图层/颜色)
2. 组件和变体
3. Auto Layout
4. 交互原型
5. 设计系统
6. 协作工作流
```

### 高级阶段 (1年+)

| 方向 | 职业路径 |
|------|---------|
| UI/UX 设计师 | Web/App 产品设计 |
| 品牌设计师 | 企业VI、包装设计 |
| 动效设计师 | After Effects、Lottie |
| 产品设计师 | 全流程产品设计 |

### 设计资源推荐

## 通用建议

### 作品集 (Portfolio)

```
高质量作品集 = 3-5个完整项目
每个项目包含:
  1. 项目背景/目标
  2. 过程/草图/迭代
  3. 最终成果
  4. 你的角色/贡献
```

### 每日练习建议

| 领域 | 每天15分钟 | 每周 |
|------|-----------|------|
| 绘画 | 5分钟速写 + 10分钟临摹 | 1幅原创 |
| 摄影 | 拍10张照片 | 修3张图 |
| 设计 | 分析1个设计 | 做1个小项目 |

### 社区与学习平台

| 平台 | 用途 |
|------|------|
| Bilibili / YouTube | 免费教程 |
| Coursera / Udemy | 付费系统课程 |
| 小红书 | 国内作品分享 |
| ArtStation / Behance | 国际作品集 |
| 站酷 (Zcool) | 国内设计社区 |
