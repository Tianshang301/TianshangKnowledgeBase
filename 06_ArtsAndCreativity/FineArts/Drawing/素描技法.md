# 素描技法

## 1. 线的类型与运用

### 1.1 线的基本分类

| 线型 | 特点 | 用途 |
|------|------|------|
| 轮廓线（Contour Line） | 沿物体外缘描绘 | 定义形状与边界 |
| 结构线（Structural Line） | 沿内部骨架/结构走向 | 表现体积与透视 |
| 动态线（Gesture Line） | 快速、流畅、有韵律 | 捕捉动态与姿态 |
| 书法线（Calligraphic Line） | 有粗细/轻重变化 | 增加表现力与节奏 |
| 辅助线（Construction Line） | 轻、可擦 | 起稿、定位 |
| 消失线（Vanishing Line） | 汇聚焦点 | 透视辅助 |

### 1.2 线的表现力

| 线条特征 | 心理效果 | 运用场景 |
|----------|----------|----------|
| 直线 | 刚硬、稳定、理性 | 建筑、机械 |
| 曲线 | 柔和、优雅、流动 | 人体、自然 |
| 粗线 | 有力、突出、厚重 | 前景、阴影 |
| 细线 | 精致、轻盈、后退 | 背景、亮部 |
| 虚线/断续 | 松动、透气 | 辅助、织物 |
| 重叠线 | 动感、寻找 | 动态速写 |

## 2. 明暗与调子

### 2.1 五大调子系统

| 编号 | 调子名称 | 位置 | 表现方法 |
|------|----------|------|----------|
| 1 | 高光（Highlight） | 光源直射最近点 | 留白/白笔/硬橡皮提亮 |
| 2 | 亮部（Light） | 受光面 | 浅排线/轻擦抹 |
| 3 | 中间调（Midtone） | 介于亮暗之间的过渡区 | 中等密度排线 |
| 4 | 明暗交界线（Terminator） | 亮部转为暗部最深处 | 最密集排线/最用力 |
| 5 | 暗部（Shadow） | 背光面 | 次密排线 |
| — | 反光（Reflected Light） | 环境光反弹入暗部 | 比暗部略亮 |

五调子规律：高光→亮部→中间调→明暗交界线（最暗）→暗部→反光（略亮）

### 2.2 排线技法

#### 单排线（Hatching）

- 平行线排列，线距决定灰度
- 线距均匀→平滑渐变
- 线距渐变→体积感

#### 交叉排线（Cross-hatching）

| 层数 | 方向 | 灰度 | 效果 |
|------|------|------|------|
| 第一层 | 任意方向 | 浅灰 | 基础调子 |
| 第二层 | 45°交叉 | 中灰 | 丰富质感 |
| 第三层 | 90°交叉 | 深灰 | 阴影 |
| 第四层 | 任意角多层 | 最深 | 最暗部 |

#### 点画（Stippling）

- 点密度控制灰度
- 耗时但细腻自然
- 适合微距级精细素描
- 常用工具：针管笔（0.05-0.5mm）

#### 擦抹（Blending）

| 工具 | 效果 | 适用 |
|------|------|------|
| 纸笔（Tortillon/Stump） | 均匀平滑 | 皮肤、天空、过渡区 |
| 纸巾/棉签 | 大面积柔和 | 背景、阴影 |
| 手指 | 可控性差但温暖自然 | 小范围过渡 |
| 软橡皮（Kneaded Eraser） | 提亮/修正 | 高光、调整错误 |

#### 薄擦（Scumbling）

软性材料（炭条/炭精棒）侧锋轻擦纸面，制造斑驳的松散中间调，再用橡皮提出亮部。

### 2.3 五值灰度系统

将画面简化为5个明度层级：

| 值 | 名称 | 灰度（0=白, 10=黑） | 铅笔硬度参考 |
|----|------|-------------------|--------------|
| 1 | 白/高光 | 0-1 | 纸白/HB轻 |
| 2 | 浅灰/亮部 | 2-4 | 2H-HB |
| 3 | 中灰/中间调 | 4-6 | HB-2B |
| 4 | 深灰/暗部 | 6-8 | 4B-6B |
| 5 | 黑/最暗部 | 8-10 | 8B-9B |

## 3. 透视原理

### 3.1 一点透视（平行透视）

| 要素 | 说明 |
|------|------|
| 消失点（Vanishing Point） | 1个VP，在视平线上 |
| 视平线（Horizon Line） | 与眼睛等高 |
| 应用场景 | 走廊、铁路轨道、室内空间、街道纵深 |
| 关键 | 所有平行线向VP汇聚 |
| 经典示例 | 铁路轨道延伸至远方一点 |

### 3.2 两点透视（成角透视）

| 要素 | 说明 |
|------|------|
| 消失点 | 2个VP，在视平线左右两侧 |
| 适用范围 | 建筑转角、立方体、室内一角的两个墙面 |
| 应用 | 建筑外观表现最常用 |
| 关键 | 垂直线保持垂直，水平线斜向两个VP |
| 示例 | 房间的角落——两面墙分别向左右消失 |

### 3.3 三点透视（俯视/仰视透视）

| 类型 | 第三个消失点位置 | 效果 | 应用 |
|------|-----------------|------|------|
| 虫瞰（仰视） | 上方（天点） | 建筑高耸宏伟 | 摩天楼从地面仰望 |
| 鸟瞰（俯视） | 下方（地点） | 俯瞰城市 | 高层建筑俯视街景 |
| 应用 | 大型场景构图 | 强烈的空间张力 | 漫画/插画城市场景 |

### 3.4 大气透视（Air / Atmospheric Perspective）

通过明度与细节的变化表现远近距离——非线条透视而是色调透视。

| 要素 | 近景 | 远景 |
|------|------|------|
| 明度对比 | 强（亮-暗分明） | 弱（趋于均匀中灰） |
| 细节 | 清晰丰富 | 模糊简化 |
| 饱和度 | 高 | 低（趋于蓝灰） |
| 边缘 | 锐利 | 柔和模糊 |
| 色彩温度 | 暖色（棕/红/黄） | 冷色（蓝/紫/灰） |

Leonardo da Vinci 首次系统阐述大气透视理论。

## 4. 构图基础

### 4.1 三分法（Rule of Thirds）

画面用两条水平线和两条垂直线均分九格。主体/地平线/视觉焦点放在四条线交叉的四个"强点"上。

### 4.2 黄金比例（Golden Ratio）

| 比例 | 数值 | 应用 |
|------|------|------|
| 黄金比例 φ | 1:1.618 | 画面分割、主体大小 |
| 黄金螺旋 | 斐波那契数列螺旋 | 引导视线到焦点 |
| 黄金矩形 | 长边:短边=φ | 画幅比例选择 |

### 4.3 引导线（Leading Lines）

| 线的走向 | 视觉引导效果 |
|----------|--------------|
| 水平线 | 平静、稳定、宽广 |
| 垂直线 | 崇高、正式、力量 |
| 对角线 | 动感、紧张、冲突 |
| S形曲线 | 优雅、流动、舒缓 |
| 汇聚线 | 纵深感、焦点明确 |

### 4.4 负空间（Negative Space）

主体周围/之间的空白区域，与主体（正空间）形成图底关系。善用负空间可以：
- 平衡画面
- 强调主体形状
- 增加呼吸感
- 创造想象空间

### 4.5 平衡类型

| 平衡类型 | 原理 | 效果 | 适用 |
|----------|------|------|------|
| 对称平衡 | 左右镜像 | 稳定、庄严 | 宗教画、肖像、建筑 |
| 不对称平衡 | 视觉重量左右相等但元素不同 | 动态、活泼 | 大部分现代/写生作品 |
| 辐射平衡 | 以中心向外扩散 | 聚焦、秩序 | 圆形构图、曼陀罗 |

## 5. 人体比例与速写

### 5.1 人体比例（成人标准）

| 部位 | 比例关系 | 说明 |
|------|----------|------|
| 全身 | 约7.5-8头高 | 8头高为理想化比例 |
| 躯干 | 约3.5头高 | 下巴→乳头→肚脐→会阴 |
| 腿 | 约4头高 | 大转子→膝盖→脚踝 |
| 上肢 | 约3头高 | 肩→肘→腕→指尖 |
| 肩宽 | 约2头宽 | 男性稍宽 |
| 手 | 约1头长 | 指尖到手腕 |
| 脚 | 约1头长 | 脚跟到脚趾 |

### 5.2 面部比例

| 规律 | 比例关系 |
|------|----------|
| 眼睛位置 | 整个头部高度的中点 |
| 三庭 | 发际线→眉毛=眉毛→鼻底=鼻底→下巴 |
| 五眼 | 脸宽度≈5个眼睛宽度 |
| 鼻宽 | ≈眼睛宽度 |
| 嘴宽 | ≈虹膜内侧垂线之间 |
| 耳 | 上端≈眉毛高度，下端≈鼻底 |

### 5.3 Loomis 头像画法

Andrew Loomis 的头部画法步骤：
1. 画一个圆（颅骨部分）
2. 十字线标出面部朝向
3. 下巴用楔形连接
4. 侧面线（ear placement）标出耳朵位置
5. 按三庭五眼定位五官

### 5.4 动态速写（Gesture Drawing）

| 时间限制 | 重点 | 工具/方法 |
|----------|------|----------|
| 30秒 | 动态线（line of action）+ 整体姿态 | 软炭条，大胆流畅 |
| 1分钟 | 动态线 + 主要体块 | 加入头/胸腔/盆腔三大块 |
| 2分钟 | 动态线 + 体块 + 基本比例 | 可加入简单光影 |
| 5分钟 | 动态+比例+轮廓 | 可加入细节与明暗 |

**动态线（Line of Action）**：贯穿人体从头部到支撑脚的一条弧线，是速写的灵魂。捕捉的是人物最本质的力与动感。

## 6. 素描材料与工具

### 6.1 铅笔

| 硬度等级 | 特点 | 用途 |
|----------|------|------|
| H-9H（硬铅） | 浅、硬、锐 | 起稿、浅调子、细节 |
| HB（中性） | 中等 | 通用 |
| B-9B（软铅） | 深、软、粗 | 阴影、深调、大面积 |
| 2B-4B | 最常用范围 | 写生、人像 |

### 6.2 炭笔

| 类型 | 硬度 | 特点 |
|------|------|------|
| 炭条（Vine Charcoal） | 软/中/硬 | 松软，适合大面积+起稿 |
| 炭精棒（Compressed Charcoal） | 软至硬 | 色深，适合深暗调子 |
| 炭铅笔（Charcoal Pencil） | H/B系列 | 精准 + 深色 |

### 6.3 其他工具

| 工具 | 用途 |
|------|------|
| 色粉笔（Pastel）/ 索斯（Sanguine）/ 白垩（White Chalk） | 有色素描（如文艺复兴三色画法）|
| 钢笔/针管笔 | 线描、点画、精细排线 |
| 马克笔 | 快速设计速写 |
| 纸笔（Tortillon） | 擦抹柔化 |
| 橡皮（硬/软/电动） | 提亮、修正、纹理 |
