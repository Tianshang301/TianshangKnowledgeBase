# 绘画技法

## 1. 油画技法

### 1.1 厚涂法（Impasto）

| 项目 | 说明 |
|------|------|
| 定义 | 大量颜料堆砌于画布，保留笔触与刀痕的立体肌理 |
| 工具 | 调色刀、猪鬃笔 |
| 效果 | 强烈的触觉质感与光影折射 |
| 代表画家 | Van Gogh《星空》《向日葵》——漩涡状厚笔触；Rembrandt 晚期自画像 |
| 注意 | 干燥缓慢，需遵守肥盖瘦原则；厚涂层易开裂，需用弹性底材 |

### 1.2 罩染与薄擦（Glazing / Scumbling）

| 技法 | 操作方式 | 颜料特征 | 效果 |
|------|----------|----------|------|
| 罩染（Glazing） | 透明颜料薄层叠加于干燥底层 | 透明/半透明 | 光学混合，色彩深邃发光，如同彩色玻璃 |
| 薄擦（Scumbling） | 不透明颜料干笔薄擦于底层 | 不透明 | 模糊边缘，柔和过渡，营造气氛 |
| 应用场景 | 罩染调整整体色调 | 薄擦处理远景/雾化效果 |
| 步骤 | 底层完全干燥→稀释颜料→均匀薄涂→逐层叠加 |
| 经典范例 | Titian、Rembrandt 多层罩染技法 | Turner 薄擦营造光雾 |

### 1.3 直接画法（Alla Prima）

直接在画布上一次性完成（wet-on-wet），不在调色板上预先混合。色彩鲜活、笔触自由。适用于户外写生（plein air）。

| 优点 | 缺点 | 代表画家 |
|------|------|----------|
| 色彩鲜艳直接 | 难以大幅修改 | Sargent、Velázquez |
| 完成快速 | 无法达到罩染的深邃 | Zorn、Fechin |
| 笔触生动奔放 | 需丰富的调色经验 | 印象派画家 |

### 1.4 晕涂法（Sfumato）

达芬奇创立的柔和过渡技法——色彩与明暗之间无可见边界，如烟雾般融合。

| 特征 | 说明 |
|------|------|
| 原理 | 数十层极薄半透明罩染叠加 |
| 效果 | 边缘消失，柔和朦胧 |
| 代表作品 | Da Vinci《蒙娜丽莎》——微笑容色与眼角 |
| 操作要点 | 每层极薄，等待完全干燥，中性色调过渡 |

### 1.5 明暗对比法（Chiaroscuro）

强烈的明暗对比塑造戏剧性和立体感。暗部大面积淹没在阴影中，亮部集中于主体。

| 时期 | 特征 | 代表画家 |
|------|------|----------|
| 文艺复兴 | 柔和渐变 | Da Vinci、Raphael |
| 巴洛克 | 强烈戏剧性 | Caravaggio——酒窖光线法 |
| 荷兰黄金时代 | 三角光 | Rembrandt——自画像系列 |
| 法国古典 | 烛光场景 | Georges de La Tour《忏悔的抹大拉》|

## 2. 水彩技法

### 2.1 湿画法（Wet-on-Wet）

| 步骤 | 操作要点 | 效果 |
|------|----------|------|
| 1. 打湿纸面 | 清水均匀润湿，表面光泽无积水 | 颜色自然扩散 |
| 2. 上色 | 颜料点入湿纸面 | 柔和的羽化边缘 |
| 3. 控制扩散 | 纸面湿度控制扩散范围 | 可控制造渐云层 |
| 适用 | 天空、云层、水面、远景、背景虚化 |

### 2.2 干画法（Wet-on-Dry）

干纸上着色，边缘清晰锐利。层层叠加（layering）时每层需完全干燥——水彩由浅入深，浅层不可被深层覆盖。

| 特征 | 叠加效果 |
|------|----------|
| 第一层 | 浅色薄涂，确定基本形状 |
| 第二层 | 中间色调，塑造体积 |
| 第三层 | 深色/阴影，增加深度 |
| 第四层 | 最暗部与细节 |

### 2.3 干擦法（Dry Brush）

笔头蘸少量浓颜料，在粗糙纸面快速擦过——留下斑驳飞白效果。

| 适用场景 | 技法变化 |
|----------|----------|
| 木纹/树皮 | 干笔顺纹理擦过 |
| 岩石纹理 | 旋转化用笔 |
| 毛发/草地 | 笔尖快速挑画 |
| 光照效果 | 亮部用干擦制造闪烁感 |

### 2.4 洗涤法（Wash）

| 类型 | 操作方法 | 适用 |
|------|----------|------|
| 平涂（Flat Wash） | 画板倾斜，笔蘸足颜料均匀横扫 | 大面积平坦色块 |
| 渐变（Graded Wash） | 逐步加水稀释，由浓到淡 | 天空渐变、水面倒影 |
| 多重（Variegated Wash） | 多种颜色趁湿接合 | 丰富过渡的云层、日落 |

### 2.5 吸色法（Lifting）

趁湿或半干时用干净湿笔吸走颜料——制造亮部、云朵、高光。工具包括：干净毛笔、纸巾、海绵。

| 时机 | 效果 | 操作 |
|------|------|------|
| 湿时吸水 | 柔和亮部 | 纸巾轻按 |
| 半干时吸水 | 较清晰亮部 | 湿笔吸色后擦干笔头 |
| 干后洗色 | 需要刷洗+吸干 | 适合修正 |

### 2.6 特殊技法

| 技法 | 操作 | 效果 |
|------|------|------|
| 撒盐法 | 湿画时撒盐，干后刷去 | 雪花/星空/肌理 |
| 留白胶 | 涂覆遮蔽→上色→干后擦除 | 精确留白 |
| 喷溅法 | 牙刷蘸颜料弹拨 | 星空/浪花/纹理 |
| 刮擦法 | 笔杆/刀片半干时刻画 | 细线/高光 |
| 蜡笔阻色 | 蜡笔/蜡烛涂画后上色 | 留白纹理 |
| 酒精滴洒 | 湿画时滴酒精 | 圆形斑点 |

## 3. 丙烯技法

### 3.1 分层叠加（Layering）

丙烯速干（10-30分钟），可持续高效叠加。

| 层序 | 操作 | 媒介 |
|------|------|------|
| 底料层 | 丙烯底料（gesso）涂刷 | 均匀吸收性 |
| 底色层 | 稀释丙烯铺大色调 | 水稀释 |
| 塑造层 | 不透明丙烯塑造体积 | 厚涂/直接画 |
| 罩染层 | 透明丙烯调色 | 罩染媒介 |
| 细节层 | 精细节画 | 缓干媒介 |

### 3.2 浇注法（Pouring）

丙烯颜料 + 浇注媒介（pouring medium）调匀，倾倒在画布上，倾斜画布引导流动。多层颜料叠加产生大理石纹、细胞效果。需消泡剂/热风枪消除气泡。

### 3.3 混合媒介（Mixed Media）

| 材料 | 用法 | 效果 |
|------|------|------|
| 拼贴（Collage） | 纸、布料嵌入底层 | 丰富肌理与层次 |
| 肌理膏 | 刮刀涂抹制造浮雕塑效果 | 厚重质感 |
| 沙/玻璃珠 | 混入颜料或肌理膏 | 粗糙表面 |
| 油画棒/色粉 | 丙烯表层叠加 | 线条与色彩 |
| 模板/喷漆 | 模板遮蔽+喷涂 | 街头艺术风格 |

### 3.4 丙烯绘画优势

- **水性**：水稀释即可，无毒无味
- **速干**：高效分层，适合教学和快速创作
- **防水**：干后防水不褪色
- **多表面**：纸、布、木板、玻璃、金属均可
- **模拟性**：可模拟水彩（稀薄）/油画（厚涂）

## 4. 其他画种技法

### 4.1 水粉画（Gouache）

| 特征 | 说明 |
|------|------|
| 颜料组成 | 色粉 + 水溶性胶 + 白粉 → 不透明 |
| 与水彩区别 | 不透明，可覆盖深色 |
| 干燥特性 | 干燥后颜料变浅（干色变浅现象） |
| 技法范围 | 薄涂如透明水彩，厚涂似丙烯 |
| 应用领域 | 插画、设计稿、动画背景、户外写生 |

### 4.2 蛋彩画（Tempera）

| 特征 | 说明 |
|------|------|
| 粘合剂 | 蛋黄 + 水 + 颜料色粉 |
| 特性 | 速干、持久不变色、哑光质感 |
| 技法 | 细密交叉排线，多层薄涂，不可厚涂 |
| 干燥速度 | 极快，需迅速果断用笔 |
| 调色板 | 通常浅底（石膏底料）上绘制 |
| 代表作品 | Botticelli《春》《维纳斯的诞生》|

### 4.3 湿壁画（Fresco）

| 类型 | 方法 | 特点 | 耐久性 |
|------|------|------|--------|
| 湿壁画（Buon Fresco）| 颜料涂于新鲜湿石灰浆 | 颜料与墙面化学结合 | 极为耐久，数百年不褪 |
| 干壁画（Fresco Secco）| 干墙上用胶彩绘制 | 颜色更鲜艳 | 易剥落褪色 |
| 混合技法 | 湿画为主+干画补色 | 兼顾耐久与鲜艳 | 常见于文艺复兴 |

| 步骤 | 操作 |
|------|------|
| 1. 准备墙面 | 多层石灰砂浆抹平 |
| 2. 勾稿 | 碳粉透图（spolvero）转印 |
| 3. 分区抹灰 | 每日只抹当天可完成部分的石灰 |
| 4. 上色 | 颜料+水在湿灰上绘制 |
| 5. 修饰 | 干后补色（secco）修整 |

代表：Michelangelo《创世纪》西斯廷天顶画、Giotto 斯克洛维尼礼拜堂壁画。

## 5. 色彩理论

### 5.1 色轮（Color Wheel）

| 色轮层级 | 颜色 | 混合方式 |
|----------|------|----------|
| 三原色（Primary）| 红、黄、蓝（RYB体系）| 不可由其他颜料混合 |
| 间色（Secondary）| 橙、绿、紫 | 原色两两混合 |
| 复色（Tertiary）| 红橙、黄橙、黄绿、蓝绿、蓝紫、红紫 | 原色+间色混合 |

### 5.2 配色关系与色彩和谐

| 配色类型 | 色轮位置 | 效果 | 应用建议 |
|----------|----------|------|----------|
| 互补色 | 相对（红/绿、蓝/橙、黄/紫）| 强烈对比，相互增强 | 主体与背景分离 |
| 类似色 | 相邻（蓝/蓝绿/绿）| 和谐统一，平静 | 风景画中自然色 |
| 三色调和 | 等距120° | 丰富而平衡 | 装饰性/现代画 |
| 分裂互补 | 一色+其补色两侧邻近色 | 对比柔和 | 入门友好配色 |
| 单色 | 同色相不同明度/饱和度 | 极简优雅 | 强调光影与构图 |

### 5.3 色彩三属性

| 属性 | 定义 | 调整方式 | 示例 |
|------|------|----------|------|
| 色相（Hue）| 颜色的名称/波长 | 混合不同颜料 | 红色相、蓝色相 |
| 明度（Value）| 颜色的明暗程度 | 加白=明调（Tint）/ 加黑=暗调（Shade）|
| 饱和度（Saturation）| 颜色的纯度/鲜艳度 | 加灰/加补色=弱化（Tone）|

- **Tint**：纯色 + 白色 → 高明度、低饱和度
- **Shade**：纯色 + 黑色 → 低明度、原色相
- **Tone**：纯色 + 灰色 → 降低饱和度

### 5.4 色彩温度

| 色温 | 颜色 | 视觉心理 | 空间作用 |
|------|------|----------|----------|
| 暖色 | 红、橙、黄、棕 | 前进、热情、活泼 | 视觉上向前推进 |
| 冷色 | 蓝、绿、紫 | 后退、冷静、宁静 | 视觉上向后退缩 |
| 中性 | 黑白灰、棕 | 平衡、过渡 | 连接冷暖区域 |

### 5.5 饱和度控制

- 降低纯度：加补色（最自然）、加灰色（均匀降低）、加白（降低且变浅）
- 提高纯度：使用纯色颜料、避免过度混合
- 有限调色板：用6-8种颜料混合出丰富色阶

## 6. 素描技法

### 6.1 排线法

| 技法 | 方法 | 灰度控制 | 效果 |
|------|------|----------|------|
| 单排线（Hatching）| 平行线排列 | 线间距越密越暗 | 均匀细腻 |
| 交叉排线（Cross-hatching）| 多层不同方向叠加 | 层数越多越暗 | 丰富厚重 |
| 素描排线 | 沿形体方向排线 | 疏密渐变 | 表现体积 |

### 6.2 点画法（Stippling）

用密集的点构成明暗——点越密越暗。耗时但极具表现力，可达照片级真实感。工具：细尖笔（0.05-0.3mm针管笔）。

### 6.3 擦抹法（Blending）

用纸笔（tortillon/blending stump）、纸巾、手指或橡皮擦涂抹炭笔/铅笔粉，制造平滑无笔触的渐变——适用于皮肤、天空、金属反光面。

### 6.4 薄擦法（Scumbling in Drawing）

用软性材料（炭条/色粉条）侧锋轻擦纸面，制造松散斑驳的中间调子。可用橡皮提亮。

### 6.5 渲染（Rendering）

综合运用上述技法，通过对光影、质感、空间的精细节画，使二维画面呈现三维真实感。

| 渲染要素 | 技法组合 | 目标 |
|----------|----------|------|
| 体积感 | 五大调子 + 明暗交界线强化 | 物体立体感 |
| 质感 | 不同笔触模拟材质 | 光滑/粗糙/柔软/坚硬 |
| 空间感 | 近实远虚、前景细节/背景简化 | 画面深度 |

### 6.6 五大调子系统

1. **高光**（Highlight）——光源直接照射的最亮一点
2. **亮部**（Light）——受光面的中间调子
3. **明暗交界线**（Terminator）——明暗转折最暗处
4. **暗部**（Shadow）——背光面
5. **反光**（Reflected Light）——环境光反射入暗部
