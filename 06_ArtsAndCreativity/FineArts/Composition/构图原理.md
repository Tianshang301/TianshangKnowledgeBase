# 构图原理

## 1. 设计元素（Elements of Design）

### 1.1 点（Point）

最小的视觉单位——单个点吸引注意，三个点形成三角，多个点构成形状、肌理或节奏。应用：焦点、强调、纹理基础。

### 1.2 线（Line）

| 线的类型 | 视觉心理 | 构图作用 |
|----------|----------|----------|
| 水平线 | 平静、稳定、安宁 | 地平线、背景分割 |
| 垂直线 | 崇高、正式、强大 | 支撑结构、庄严感 |
| 对角线 | 动感、紧张、激情 | 制造冲突与能量 |
| 曲线 | 优雅、流动、柔和 | 引导视线漫游 |
| 折线 | 尖锐、不安、变化 | 表现张力 |

### 1.3 形状与形体（Shape & Form）

| 类型 | 定义 | 区别 |
|------|------|------|
| 形状（Shape） | 二维——只有长和宽 | 轮廓、剪影 |
| 形体（Form） | 三维——有长宽深 | 立体感需要光影 |
| 几何形 | 正方形、圆、三角形 | 理性、秩序、人工感 |
| 有机形 | 自然形态 | 感性、自由、生命力 |

### 1.4 空间（Space）

| 空间类型 | 定义 | 作用 |
|----------|------|------|
| 正空间（Positive Space） | 主体所占区域 | 视觉焦点 |
| 负空间（Negative Space） | 主体周围的空白 | 对比、呼吸感 |
| 深空间 | 强烈透视具有纵深 | 沉浸感 |
| 浅空间 | 平面化压缩 | 装饰性 |

### 1.5 肌理（Texture）

| 肌理类型 | 感知方式 |
|----------|----------|
| 真实肌理（Actual Texture） | 触觉可感——浮雕、厚涂 |
| 视觉肌理（Visual Texture） | 视觉模拟——绘画中的布纹、木纹 |
| 模拟肌理 | 描绘出的质感 |
| 创造肌理 | 用材料/媒介创造抽象纹理 |

### 1.6 明度与色彩（Value & Color）

明度（Value）是构图的骨架——即使去掉颜色，好的黑白构图依然成立。明度结构决定了画面的权重分布、视觉流动和情感基调。

## 2. 设计原则（Principles of Design）

### 2.1 平衡（Balance）

| 类型 | 原理 | 效果 |
|------|------|------|
| 对称平衡 | 中线左右镜像 | 稳定、庄严、正式 |
| 非对称平衡 | 视觉权重左右相等但元素不同 | 动态、活泼、现代 |
| 辐射平衡 | 从中心向外扩散 | 聚焦、神秘、秩序 |

视觉重量因素：大小、色彩饱和度、明度对比、孤立程度（孤立的物体显得更重）、纹理复杂度。

### 2.2 对比（Contrast）

| 对比类型 | 操作方法 | 效果 |
|----------|----------|------|
| 明度对比 | 深色 vs 浅色 | 最强烈的视觉冲击 |
| 色彩对比 | 互补色/冷暖对比 | 色彩活力 |
| 大小对比 | 大 vs 小 | 主次关系 |
| 形状对比 | 几何 vs 有机 | 丰富多样性 |
| 肌理对比 | 光滑 vs 粗糙 | 触觉趣味 |
| 方向对比 | 水平 vs 垂直 | 打破单调 |

### 2.3 强调与主次（Emphasis / Dominance）

| 强调手段 | 方法 | 效果 |
|----------|------|------|
| 孤立 | 主体单独出现，大量留白 | 聚焦 |
| 位置 | 中心或焦点位置 | 最自然的强调 |
| 对比 | 与周围形成强烈差异 | 最有效的强调 |
| 汇聚线 | 视线被引导至某点 | 定向引导 |
| 框架 | 自然/人工框架框住主体 | 突出主体 |

### 2.4 动势与节奏（Movement / Rhythm）

| 节奏类型 | 特征 | 效果 |
|----------|------|------|
| 重复节奏 | 相同元素等距重复 | 秩序、统一 |
| 渐变节奏 | 元素大小/间距渐变 | 动感、纵深感 |
| 交替节奏 | 两种/多种元素轮流出现 | 丰富、变化 |
| 流动节奏 | 曲线引导视线自然移动 | 优雅、舒适 |

**视觉流动**：好的构图引导观者的视线按照设计路径浏览画面——通常是从左下→右上（Z 型浏览），或沿 S 形曲线推进。

### 2.5 比例（Proportion）

| 比例系统 | 数值 | 应用 |
|----------|------|------|
| 黄金比例（Golden Ratio） | φ = 1.618 | 画幅、主体大小、分割线 |
| 三分法（Rule of Thirds） | 1:1:1 三等分 | 网格构图 |
| 斐波那契数列 | 1,1,2,3,5,8,13... | 自然比例、黄金螺旋 |
| √2矩形（DIN 标准） | 1:1.414 | 纸张比例、画面分割 |
| 模数（Modulor） | Le Corbusier 的人体模度 | 建筑与设计比例 |

**黄金分割作图**：将一条线段分为两段，使长段与短段之比等于全长与长段之比。符合此比例的形状（黄金矩形）被认为视觉上最和谐。

### 2.6 统一与变化（Unity / Variety）

| 统一方法 | 说明 |
|----------|------|
| 接近（Proximity） | 相近的元素被认为是一组 |
| 相似（Similarity） | 形状/色彩/大小相似的元素统一 |
| 连续（Continuation） | 线/轮廓延伸使观者视线追随 |
| 闭合（Closure） | 不完整形状被大脑自动补全 |
| 图底（Figure-Ground） | 主体与背景清晰分离或可转换 |

## 3. 格式塔原理（Gestalt Principles）

| 原理 | 定义 | 构图应用 |
|------|------|----------|
| 接近性 | 空间上靠近的元素被视为一组 | 分组信息，建立视觉单元 |
| 相似性 | 视觉特征相似的元素被视为一组 | 色彩/形状/大小的群组 |
| 连续性 | 视线沿排列方向自然流动 | 引导线、路径设计 |
| 闭合性 | 大脑自动补全不完整的形状 | 留白、隐含的线 |
| 图底关系 | 视觉自动区分主体和背景 | 明确的层次/可切换的双重画面 |
| 对称性 | 对称元素被视为整体 | 平衡、稳定感 |
| 共同命运 | 同方向运动的元素被视为一组 | 动势统一 |

## 4. 色彩构图

### 4.1 配色方案

| 配色类型 | 构成 | 效果 |
|----------|------|------|
| 单色（Monochromatic） | 同一色相不同明度/饱和度 | 极简、优雅、纯粹 |
| 类似色（Analogous） | 色轮相邻的2-3色 | 和谐、统一、平静 |
| 互补色（Complementary） | 色轮相对的2色 | 强烈对比、活力 |
| 分裂互补（Split-Complementary） | 一色+其补色两侧的2色 | 对比而不尖锐 |
| 三色调和（Triadic） | 色轮等距120°的3色 | 丰富而平衡 |
| 四色调和（Tetradic） | 色轮上两组互补色 | 最为丰富、复杂 |

### 4.2 同时对比（Simultaneous Contrast）

Josef Albers 在《色彩构成》中提出的现象：一种颜色会因其相邻颜色的不同而看起来不同。

| 现象 | 说明 | 应用 |
|------|------|------|
| 同色异感 | 同一灰色在不同底色上看起来深浅不同 | 调整背景可改变颜色感知 |
| 互补残像 | 盯着红色后看白色→看到绿色残像 | 利用残像增强效果 |
| 冷暖对比 | 冷色旁边的暖色更暖 | 增强画面温度 |

### 4.3 色彩与情感

| 色彩 | 情感联想 | 构图运用 |
|------|----------|----------|
| 红 | 激情、警告、爱 | 焦点色，不宜大面积 |
| 蓝 | 冷静、信任、忧郁 | 大面积背景、天空 |
| 黄 | 快乐、希望 | 提亮、点缀色 |
| 绿 | 自然、平和 | 风景/平衡色 |
| 紫 | 奢华、神秘 | 装饰性/强调色 |
| 黑 | 力量、优雅 | 极简/重量感 |
| 白 | 纯净、空无 | 负空间/呼吸感 |

## 5. 动态对称与格线系统

### 5.1 动态对称（Dynamic Symmetry）

源于古希腊/文艺复兴的构图体系，基于无理数比例（√2, √3, √5, φ）分割矩形：

| 矩形类型 | 比例 | 对角线系统 | 应用 |
|----------|------|------------|------|
| √2矩形 | 1:1.414 | 主对角线+垂线 | 标准构图 |
| √3矩形 | 1:1.732 | 主对角线+从线 | 风景画 |
| 黄金矩形 | 1:1.618 | φ分割 | 经典构图 |

### 5.2 矩形的骨架（Armature of the Rectangle）

将矩形四角连线（对角线），再从另一角向对角线作垂线、水平线与垂直线，形成天然的引导线和比例点网。这些交点对应三分法强点、黄金分割点。

### 5.3 动态对称的运用

- 人物的主要姿态沿对角线或根号矩形对角线布局
- 主要交界（如地平线/海平线）沿三等分水平线放置
- 视觉焦点放置在格线交点
- 引导线沿格线方向延伸

## 6. 构图中的空间表现

### 6.1 透视与空间深度

| 空间表现手段 | 方法 | 效果 |
|-------------|------|------|
| 线性透视 | 一点/两点/三点透视 | 精确的纵深感 |
| 大气透视 | 远淡近浓、远灰近艳 | 自然深度 |
| 重叠 | 前物遮后物 | 最基础的深度暗示 |
| 大小衰减 | 远小近大 | 空间尺度感 |
| 细节递减 | 远模糊近清晰 | 增强景深感 |

### 6.2 前景-中景-背景

| 层次 | 位置 | 特征 | 作用 |
|------|------|------|------|
| 前景 | 最靠近观众 | 暗、暖、细节、大 | 进入画面、框景 |
| 中景 | 中间区域 | 主体位置 | 叙事核心 |
| 背景 | 最远处 | 亮/冷、模糊、小 | 氛围、延伸空间 |
