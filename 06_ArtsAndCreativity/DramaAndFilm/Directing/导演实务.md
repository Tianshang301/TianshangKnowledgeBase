# 导演实务

## 一、导演的创作准备

### 1.1 导演阐述

导演阐述是导演在筹备阶段向全体剧组陈述创作理念的核心文件：

| 组成部分 | 内容 | 目的 |
|----------|------|------|
| 剧本理解 | 对剧本主题、核心冲突、人物关系的解读 | 让各部门理解"我们要拍什么" |
| 风格定位 | 视觉风格、叙事节奏、色彩基调、声音设计方向 | 统一全剧的审美调性 |
| 人物分析 | 每个主要角色的核心欲望、弧线、表演方向 | 帮助演员建立角色框架 |
| 参考影片 | 列出风格相近的影片、画作、音乐供参考 | 用具体作品沟通抽象的"感觉" |
| 工作方法 | 导演的排练风格、拍摄习惯、沟通方式 | 让剧组了解"导演怎么工作" |

### 1.2 导演的案头工作
| 工作内容 | 方法 | 产出物 |
|----------|------|--------|
| 剧本围读 | 通读剧本——不表演，纯阅读，理解整体节奏 | 第一次印象记录 |
| 逐场分析 | 每场戏的核心是什么？从哪里来到哪里去？ | 场次功能标注剧本 |
| 角色档案 | 每个角色的核心欲望、恐惧、秘密、弧线 | 角色卡片 |
| 序列划分 | 将全片划分为8-12个功能序列 | 序列结构图 |
| 情绪曲线 | 画出全片观众情绪从开场到结局的起伏图 | 情绪曲线图（为每个序列标记情绪目标） |

### 1.3 与摄影指导的沟通
| 讨论问题 | 导演的决策点 | 摄影指导的执行点 |
|----------|-------------|-----------------|
| 镜头语言 | 什么情况下用特写？什么情况下用长镜头？ | 选择镜头焦距、光圈、摄影机格式 |
| 摄影机运动 | 何时手持？何时稳定器？何时固定机位？ | 制定运镜方案、选择设备 |
| 照明风格 | 高调还是低调？自然光还是戏剧光？ | 设计灯光方案、确定光源方向 |
| 色彩调性 | 全片的色彩基调——冷、暖、太色、饱和？ | 前期测试、后期调色协调 |
| 参考画面 | 提供画作、摄影作品、电影截图 | 理解导演的视觉参考 |

## 二、排练阶段
### 2.1 排练阶段划分

| 阶段 | 时长占比 | 主要工作 | 目标 |
|------|----------|----------|------|
| 围读 | 约10% | 全剧通读——不表演，分析剧本，统一理解 | 确保所有演员对剧本的理解一致 |
| 走位排练 | 约20% | 演员在空间中走动，确定基本走位和调度 | 建立物理空间意识 |
| 细致打磨 | 约40% | 逐场逐段调整表演细节、对白节奏、情感层次 | 提升表演质量和戏剧张力 |
| 联排 | 约25% | 全剧串联——按时间顺序完整走一遍 | 检验全剧节奏和衔接 |
| 技术合成 | 约5% | 灯光、音效、换景与表演配合 | 技术要素与表演融合 |

### 2.2 排练中的导演技巧
| 技巧 | 说明 | 适用场景 |
|------|------|----------|
| 提问法 | 不问"你这里感觉怎么样？"而问"你这里想要什么？" | 演员卡在情感中无法推进时 |
| 即兴探索 | 让演员做剧本没有的即兴——探索角色的背景和关系 | 角色关系不够明确时 |
| 换位练习 | 让演员互换角色——演对手的角色 | 需要理解对方动机时 |
| 停顿练习 | 让演员在关键词前有意识地停顿3秒 | 需要加强台词的分量时 |
| 声音变化 | 让演员用不同的语速、音量、音高说同一句台词 | 寻找对白的最佳表达方式 |
| 身体优先 | 先不念台词，用身体动作把场景的情绪"跳出来" | 演员过度依赖文字时 |
| 破坏练习 | 故意打乱演员的习惯性表演方式 | 陷入套路化表演时 |

### 2.3 常见演员问题及解决方法
| 问题 | 表现 | 可能原因 | 解决方法 |
|------|------|----------|----------|
| 过火表演 | 情感夸张、动作过度 | 不信任剧本和导演，想"证明自己在演" | 要求演员"什么也别做"——做最少的动作，说最少的话 |
| 平淡乏味 | 每句台词都一样 | 缺乏具体目标、不理解角色的欲望 | 给每个场景一个明确的可执行任务 |
| 机械背词 | 只背台词不走心 | 没有建立内在动机 | 用"自己的话"重述台词的意图 |
| 节奏不应 | 笑点/高潮没到点上 | 对剧本节奏理解有偏差 | 一起分析这场的节奏——哪里快、哪里慢、为什么 |
| 过度自我 | 只演自己、不能变形 | 缺乏想象力或害怕冒险 | 用"如果我是……"练习帮助构建角色 |

## 三、现场拍摄管理
### 3.1 拍摄日流程
| 时间段 | 工作内容 | 导演的注意重点 |
|--------|----------|---------------|
| 上午7:00 | 到现场 | 查看各部门准备情况、与执行导演对日程 |
| 上午7:30 | 与摄影师看光 | 确认主镜头的光线效果 |
| 上午8:00 | 演员到场、化妆 | 与演员简要过一遍当天要拍的戏 |
| 上午8:30 | 第一镜走位 | 与演员走戏，调整走位和表演 |
| 上午9:00 | 正式拍摄 | 保持专注——每个镜头拍多少条取决于是否"达到了" |
| 中午12:30 | 午饭 | 回顾上午、调整下午计划 |
| 下午1:30 | 继续拍摄 | 注意演员的体能和情绪——下午易疲惫 |
| 下午5:00 | 最后一场（黄金一小时） | 利用最好的自然光拍日戏外景 |
| 下午6:00 | 收工 | 看一遍当天的素材（Dailies）、确认次日日程 |

### 3.2 拍摄中的沟通原则
| 原则 | 说明 | 实践方法 |
|------|------|----------|
| 公开表扬、私下批评 | 演员在全剧组面前被批评会失去安全感 | 有调整的话走到演员身边低声说 |
| 少说多做 | 用展示而非描述来沟通 | "试试这样讲……"（示范）> "这里要更悲伤一点" |
| 一条意见一个方向 | 不要同时给多个意见 | "这一条我们关注节奏"——明确单次目标 |
| 肯定且具体 | 不模糊地说"再来一条" | "刚才很好，但我想试一个更安静的版本——你前面慢半拍会更有力量" |
| 保持正面能量 | 你的情绪会传给剧组 | 疲惫时更要保持礼貌和明确 |

### 3.3 现场决策——时间vs质量

导演在现场面对的最核心矛盾：**时间（进度）vs 质量（完美）**

| 场景类型 | 妥协策略 | 不可妥协的事项 |
|----------|----------|---------------|
| 情感重场戏 | 多拍——这是观众记住的地方 | 表演的真实性 |
| 过渡场景 | 少拍——完成功能即可 | 衔接的连续性 |
| 动作场面 | 安全第一、素材够用即可 | 演员安全 |
| 对话场景 | 覆盖够用即可——正反打+全景 | 视线匹配和轴线规则 |
| 特写 | 关键情感节点必须拍 | 不做无理由的特写 |

## 四、导演风格与类型

### 4.1 导演风格分类

| 风格类型 | 特征 | 代表导演 | 适合题材 |
|----------|------|----------|----------|
| 作者型 | 强烈的个人标记、控制所有元素 | 库布里克、韦斯·安德森 | 任何——个人表达至上 |
| 演员型 | 以演员表演为核心、给予演员极大自由 | 卡索维茨、约翰·卡萨维蒂 | 角色驱动的故事 |
| 技术型 | 以视觉观感和摄影风格为中心 | 扎克·施奈德、迈克尔·贝 | 动作/科幻/大片 |
| 纪实型 | 强调真实感、自然光、非职业演员 | 达内兄弟、贾樟柯 | 社会现实题材 |
| 即兴型 | 无固定剧本、从排练和即兴中提炼 | 迈克·李、阿巴多斯 | 日常生活/关系剧 |
| 合作型 | 与编剧/演员紧密协作、集体创作 | 塔伦蒂诺、科恩兄弟 | 类型片、风格化开发 |

### 4.2 名导工作方法

| 导演 | 工作方式 | 特点 | 可借鉴的方法 |
|------|----------|------|-------------|
| 希区柯克 | 前期极度精细——故事板画到每一帧，拍摄时严格执行 | "电影在拍摄之前已经完成了" | 前期准备越充分，现场越自由 |
| 库布里克 | 多拍（10条不罕见）——追求极限完美 | 每个细节都经过数十次尝试 | 重要场景不要轻易满足"第一条好的" |
| 费里尼 | 拍摄中不断修改剧本 | 灵感和直觉优先 | 允许自己推翻之前的计划 |
| 沃纳·赫尔佐格 | 极低预算、极端环境拍摄 | 全力以趋、不设限 | 资源和方式不是限制创意的理由 |
| 王家卫 | 无剧本拍摄——在拍摄中找故事 | 依赖即兴和剪辑台上的发现 | 信任演员和偶然性 |
| 大卫·芬奇 | 极多遍拍摄（50-100条）——追求完美主义 | 近乎强迫症的控制 | 表演的精确可以反复打磨到毫米级 |

### 4.3 导演的自我修养
| 修养维度 | 具体行动 |
|----------|----------|
| 观影量 | 大量观看各类影片——不只只看"自己喜欢的"类型 |
| 阅读 | 读剧本（不止自己拍的）、读小说、读理论 |
| 观察 | 在生活中训练"导演的眼睛"——光怎么打、人怎么动、空间怎么布置 |
| 合作 | 学会倾听和信任——好导演不是"什么都知道的人"，而是"能调动身边所有人的才华的人" |
| 韧性 | 处理意外、应对压力、在有限条件下做到最好——是导演的基本功 |

## 五、导演的职业发展路径

### 5.1 从入门到成熟

| 阶段 | 特征 | 建议行动 |
|------|------|----------|
| 入门期 | 拍短片、拍习作、做助理 | 多拍多练——每部短片解决一个具体问题 |
| 磨练期 | 拍广告/MV/小制作长片 | 积累与不同团队合作的经验 |
| 突破期 | 第一部长片 | 找一个好的制片+好好选材——处女作很重要 |
| 成熟期 | 形成个人风格 | 保持创作欲望——避免商业消耗 |
| 大师期 | 风格成熟、能完成任何题材 | 培养下一代、回馈行业 |

### 5.2 导演常用工具

| 工具类别 | 工具 | 用途 |
|----------|------|------|
| 剧本写作 | Final Draft / Fade In | 标准剧本格式软件 |
| 故事板 | Storyboard Pro / Boords | 数字故事板、镜头规划 |
| 预视化 | Blender / Maya / ShotPro | 3D预视化场景 |
| 拍摄日程 | Movie Magic / Gorilla | 拍摄日程管理和预算 |
| 剪辑 | Avid / Premiere Pro / DaVinci Resolve | 后期剪辑和调色 |
| 现场监控 | 小监（5寸监视器）+ 导演监看系统 | 现场查看和回放 |
| 沟通 | 对讲机系统 / 蓝牙通话系统 | 现场各部门通讯 |
