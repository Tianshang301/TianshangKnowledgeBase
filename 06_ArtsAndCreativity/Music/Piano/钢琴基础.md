# 钢琴基础

## 1. 坐姿与手型

### 1.1 正确坐姿

| 要素 | 要求 | 常见错误 |
|:----:|:----:|:--------:|
| **凳高** | 前臂与键盘平行，肘部略高于键盘 | 凳子太低→手腕上翘；太高→耸肩 |
| **距离** | 手放在键盘上时肘部呈 90-100° | 太近→手臂蜷缩；太远→够着弹 |
| **位置** | 坐在琴凳前1/3，身体微前倾 | 坐太深→腰部无力支撑 |
| **双脚** | 左脚略前，右脚踩踏板位置 | 翘二郎腿、脚悬空 |
| **肩膀** | 自然放松下沉 | 耸肩紧张 |

### 1.2 标准手型

| 要点 | 描述 | 口诀 |
|:----:|:----:|:----:|
| **手指弯曲** | 指关节突出，像握球状 | "手里握个鸡蛋" |
| **手腕水平** | 手腕与手背平齐，不弯不翘 | "手腕不翻不压" |
| **手臂重量** | 力量从肩膀经手臂自然落到指尖 | "坠落的电梯" |
| **虎口圆撑** | 拇指与食指之间保持圆形 | "不要夹紧" |
| **指尖触键** | 用指腹前端（不是指肚平躺）触键 | "用指尖尖" |

正确的坐姿和手型是钢琴演奏的基础。坐姿不正确会导致手臂紧张、发力不畅。标准手型类似于手里握着一个球，掌关节撑起呈拱形，指尖垂直触键。

### 2.2 指法基本原则

| 原则 | 说明 | 示例 |
|:----:|:----:|:----:|
| **拇指转位 (Thumb Under)** | 拇指从下方穿过3/4指 | C 大调音阶 3→1 |
| **跨越 (Cross Over)** | 3/4指从拇指上方跨过 | 右手下行 1→3 |
| **同音换指** | 同一键换不同指 | 重复音用不同指法 |
| **五指平均** | 避免过度依赖某指 | 弱指(4,5)需加强训练 |
| **指法固定** | 确定后尽量统一 | 同一乐句始终用相同指法 |

## 4. 大小调音阶

### 4.1 大调音阶

大调音阶结构：全-全-半-全-全-全-半
| 调号 | 大调 | 小调 (关系小调) | 升/降号 |
|:----:|:----:|:--------------:|:--------:|
| 0#/b | C | a | 无 |
| 1# | G | e | F# |
| 2# | D | b | F#, C# |
| 3# | A | f# | F#, C#, G# |
| 4# | E | c# | F#, C#, G#, D# |
| 5# | B | g# | F#, C#, G#, D#, A# |
| 6# | F# | d# | F#, C#, G#, D#, A#, E# |
| 1b | F | d | Bb |
| 2b | Bb | g | Bb, Eb |
| 3b | Eb | c | Bb, Eb, Ab |
| 4b | Ab | f | Bb, Eb, Ab, Db |
| 5b | Db | bb | Bb, Eb, Ab, Db, Gb |
| 6b | Gb | eb | Bb, Eb, Ab, Db, Gb, Cb |

### 4.2 小调音阶类型

| 类型 | 构成规则 | 特征音 | 听感效果 |
|:----:|:--------:|:------:|:--------:|
| **自然小调** | 全-半-全-全-半-全-全 | b3, b6, b7 | 古朴、朴素 |
| **和声小调** | 自然小调升7级 | b3, b6, #7 | 中东色彩、引向强烈 |
| **旋律小调 (上行)** | 自然小调升6、7级 | b3, #6, #7 | 柔和、流畅 |
| **旋律小调 (下行)** | 同自然小调 | 还原6、7 | 自然回落 |

### 4.3 音阶练习要点

1. **慢速开始**：♩=60，确保每个音均匀
2. **渐快原则**：每次提升5-10 bpm，稳定后再加速
3. **节奏变体**：附点节奏、三连音节奏变化练习
4. **力度变化**：上行渐强、下行渐弱
5. **触键变化**：legato / staccato / non legato 三种触键

## 5. 和弦与转位

### 5.1 三和弦构成
| 和弦种类 | 结构 | 标记 |
|:--------:|:----:|:----:|
| **大三和弦** | 大三度 + 小三度 | C, F, G 等 |
| **小三和弦** | 小三度 + 大三度 | Cm, Dm, Em 等 |
| **减三和弦** | 小三度 + 小三度 | Cdim, Bdim 等 |
| **增三和弦** | 大三度 + 大三度 | Caug, C+ 等 |

### 5.2 转位形态
| 类型 | 低音 | 构成 | 标记 |
|:----:|:----:|:----:|:----:|
| **原位** | 根音 | 根·三·五 | C |
| **第一转位** | 三音 | 三·五·根 | C/E |
| **第二转位** | 五音 | 五·根·三 | C/G |

### 5.3 七和弦类型
| 类型 | 构成 | 标记 | 听觉色彩 |
|:----:|:----:|:----:|:--------:|
| **大七** | 大三+大七 | Cmaj7, CM7 | 梦幻、柔和 |
| **属七** | 大三+小七 | C7 | 蓝调、张カ |
| **小七** | 小三+小七 | Cm7 | 忧郁、酷 |
| **半减七** | 减三+小七 | Cm7b5, Cø7 | 悬疑 |
| **减七** | 减三+减七 | Cdim7, Co7 | 极紧张 |

## 6. 视奏策略

| 步骤 | 视奏前操作 | 用时 |
|:----:|:----------:|:----:|
| **1. 看调号** | 确定调性，预想该调音阶 | 2s |
| **2. 看拍号** | 确定律动感 | 1s |
| **3. 扫视节奏** | 找最密集/最复杂的节奏型 | 3s |
| **4. 找模式** | 音阶片段、琶音、和弦重复 | 3s |
| **5. 看难点** | 大跳、变化音、指法挑战处 | 3s |
| **6. 开始弹** | 保持稳定速度，不停顿 | — |

### 6.1 视奏基本原则

| 原则 | 说明 |
|:----:|:------|
| **眼比手快** | 眼睛始终超前 1-2 小节 |
| **节拍为王** | 宁可弹错音，不可断节拍 |
| **简化技巧** | 和弦太多时弹根音+五音 |
| **保持流动** | 从不回头看已弹过的部分 |
| **每日练习** | 每天视奏 1 首新曲（哪怕极其简单） |

## 7. 踏板使用

| 踏板 | 名称 | 效果 | 使用场景 |
|:----:|:----:|:----:|:--------:|
| **右踏板** | 延音踏板 (Damper) | 抬起全部制音器，声音延续 | 连奏、和声连接、丰富泛音 |
| **左踏板** | 柔音踏板 (Soft) | 琴槌移位，音量减小音色变柔 | 弱奏段落、音色变化 |
| **中踏板** | 持续音踏板 (Sostenuto) | 只保持弹下时的音 | 现代作品、特定音保持 |

### 7.1 右踏板技巧
| 踏板法 | 操作 | 效果 |
|:------:|:----:|:----:|
| **切分踏板 (Syncopated)** | 弹和声后快速抬-踩 | 和声连贯无混响 |
| **直踏板 (Direct)** | 与音符同时踩下 | 增加共鸣 |
| **半踏板 (Half-pedal)** | 踩一半深度 | 部分泛音保留 |
| **抖动踏板 (Flutter)** | 快速碎踩 | 特殊效果 |
