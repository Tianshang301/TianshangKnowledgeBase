# 音乐学习路线图

## 总览

```
初级阶段 (0-1年)      中级阶段 (1-3年)      高级阶段 (3年+)
    │                      │                    │
    ▼                      ▼                    ▼
乐理基础              进阶乐理              作曲/编曲
乐器入门              视唱练耳              音乐制作
简单曲目              复杂曲目              分析大师作品
                   即兴入门              演奏/创作
```

## 第一阶段：入门 (0-6个月)

### 乐理基础

| 主题 | 学习内容 | 参考资源 |
|------|---------|---------|
| 音程 | 所有自然音程识别 | 《乐理基础》李重光 |
| 音阶 | 大调/小调音阶、五声音阶 | MusicTheory.net |
| 和弦 | 大三/小三和弦、属七和弦 | Hooktheory |
| 节奏 | 全分/二分/四分/八分/十六分音符 | Rhythm Trainer |

### 乐器选择

| 乐器 | 入门难度 | 初期投入 | 推荐理由 |
|------|---------|---------|---------|
| 钢琴 | ★★☆ | ¥2000-5000 (电钢) | 直观、全面、适合学乐理 |
| 吉他 | ★★★ | ¥500-2000 | 便携、流行、和弦丰富 |
| 尤克里里 | ★☆☆ | ¥200-500 | 最简单入门 |
| 小提琴 | ★★★★ | ¥500-3000 | 较难但表现力强 |

**建议**: 从钢琴开始，键盘布局直观，易于理解乐理。

### 推荐 App

| App | 功能 | 价格 |
|-----|------|------|
| Simply Piano | 钢琴入门 | 付费订阅 |
| Yousician | 吉他/钢琴 | 付费订阅 |
| Synthesia | 钢琴自学 | 付费 |
| Perfect Ear | 耳训 | 免费+内购 |
| Complete Music Reading Trainer | 视奏 | 免费 |

## 第二阶段：基础巩固 (6个月-1年)

### 技术训练

```
钢琴:
  - 哈农练习曲 (手指独立)
  - 车尔尼599 (初级练习曲)
  - 音阶/琶音 (所有大调)

吉他:
  - 开放和弦 (C, G, D, Am, Em)
  - 横按和弦 (F, Bm)
  - 基础扫弦 + 分解和弦
```

### 简单曲目

| 曲目 | 乐器 | 难度 |
|------|------|------|
| 《小星星》 | 任意 | ★ |
| 《致爱丽丝》第一段 | 钢琴 | ★★ |
| 《生日快乐》 | 任意 | ★ |
| 《绿袖子》 | 吉他/钢琴 | ★★ |
| 《月亮代表我的心》 | 钢琴/吉他 | ★★ |

### 耳训入门

- 音程辨认 (先练纯五、纯八、大三)
- 节奏模仿
- 简单旋律听写

## 第三阶段：中级 (1-2年)

### 进阶乐理

| 主题 | 内容 |
|------|------|
| 和声学 | 功能和声、终止式、副属和弦 |
| 曲式 | 二部曲式、三部曲式、回旋曲 |
| 调式 | 中古七调式、布鲁斯音阶 |
| 即兴 | 基于和弦音阶的即兴演奏 |

### 视唱练耳

| 训练项目 | 目标 | 方法 |
|---------|------|------|
| 视唱 | 看到任何旋律能唱准 | 每天10分钟视唱练习 |
| 听辨和弦 | 识别三和弦/七和弦性质 | EarMaster |
| 节奏听写 | 听节奏并记谱 | Rhythm Trainer |
| 旋律听写 | 听旋律并在乐器上复现 | 从2小节开始 |

### 推荐教材

| 教材 | 适合 | 内容 |
|------|------|------|
| 《和声学教程》斯波索宾 | 古典和声 | 功能和声系统 |
| 《现代和声》 | 现代/爵士 | 和弦替代、Voicing |
| 《伯克利现代吉他》 | 吉他 | 爵士吉他 |
| 《爵士钢琴》 | 钢琴 | 爵士钢琴技巧 |

### YouTube 频道推荐

| 频道 | 语言 | 内容 |
|------|------|------|
| HuwJee | 英文 | 音乐理论 |
| 12tone | 英文 | 音乐分析 |
| Andrew Huang | 英文 | 音乐制作 |
| 好和弦 (NiceChord) | 中文 | 音乐理论 |
| David Bennett | 英文 | 音乐分析 |

## 第四阶段：中高级 (2-3年)

### 即兴演奏

```
钢琴即兴: 
  - 五声音阶即兴
  - 布鲁斯即兴
  - 基于 ii-V-I 的爵士即兴

吉他即兴:
  - 五声盒子 (Pentatonic Box)
  - 调式即兴
  - 和弦音即兴
```

### 音乐制作入门

| DAW | 平台 | 价格 | 特点 |
|-----|------|------|------|
| Ableton Live | Win/Mac | ¥2000+ | 电子音乐首选 |
| Logic Pro | Mac | ¥1800 | 全面、好用 |
| FL Studio | Win/Mac | ¥1000+ | 上手快 |
| Cubase | Win/Mac | ¥2000+ | 传统编曲 |
| GarageBand | Mac/iOS | 免费 | 入门最佳 |

### 录音基础

- 音频接口 (Audio Interface)
- 麦克风类型 (动圈 vs 电容)
- MIDI 键盘
- 监听耳机/音箱

## 第五阶段：高级 (3年+)

### 作曲与编曲

```
学习内容:
  - 配器法 (管弦乐编配)
  - 对位法 (复调音乐)
  - 电子音乐制作
  - 影视配乐
```

### 音乐分析

分析大师作品：

| 作曲家 | 分析曲目 | 分析重点 |
|--------|---------|---------|
| 巴赫 | 赋格的艺术 | 对位法 |
| 莫扎特 | 交响曲40号 | 古典曲式 |
| 贝多芬 | 悲怆奏鸣曲 | 和声戏剧性 |
| 德彪西 | 月光 | 印象派和声 |
| 爵士 | Giant Steps (Coltrane) | 和声替代 |

### 演出与创作

- 加入乐队/社区
- 公开演出 (Open Mic)
- 录制原创作品
- 发布到音乐平台 (网易云、Spotify)

## 每日练习建议

| 级别 | 每天练习 | 每周 |
|------|---------|------|
| 入门 | 20分钟: 音阶 + 简单曲 | 2首新曲 |
| 中级 | 45分钟: 技巧 + 曲目 + 即兴 | 1首新曲 |
| 高级 | 60分钟+: 视奏 + 即兴 + 创作 | 分析1首作品 |
