## 2. 录音技术

### 2.1 麦克风类型

| 类型 | 工作原理 | 特点 | 经典型号 | 适用场景 |
|:----:|:--------:|:----:|:--------:|:--------:|
| **动圈 (Dynamic)** | 电磁感应 | 耐用、低灵敏度、抗高声压 | SM58、SM57、Beta 52A | 现场人声、吉他音箱、鼓 |
| **电容 (Condenser)** | 静电变化 | 高灵敏度、细节丰富、需幻象电源 | U87、AKG C414、NT1-A | 录音棚人声、原声乐器 |
| **铝带 (Ribbon)** | 铝带振动切割磁感线 | 温暖自然、脆弱、八字指向 | R121、SF-1 | 吉他音箱、铜管、Room Mic |

### 2.2 指向性模式 (Polar Patterns)

| 模式 | 拾音范围 | 特点 | 适用场景 |
|:----:|:--------:|:----:|:--------:|
| **心形 (Cardioid)** | 正面 180° | 抗背面噪声 | 标准人声、乐器近拾 |
| **超心形 (Supercardioid)** | 正面 120° | 更窄、侧面拾音 | 舞台人声、乐器 |
| **全指向 (Omnidirectional)** | 360° | 无近讲效应、自然空间感 | 环境声、合唱 |
| **八字形 (Figure-8)** | 正背面 180° | 侧面抵消 | 双人对唱、MS 立体声 |

### 2.3 立体声拾音制式

| 制式 | 摆位 | 特点 |
|:----:|:----:|:------|
| **XY** | 两支心形话筒 90-135°交叉，振膜重合 | 单声道兼容性好，定位准确 |
| **ORTF** | 两支心形话筒相距 17cm，夹角 110° | 空间感自然，立体声宽度好 |
| **AB / Spaced Pair** | 两支全指向话筒相距 1-3m | 空间感宽广，但单声道兼容性差 |
| **MS (Mid-Side)** | 心形(M)+八字形(S)，需解码 | 后期可调立体声宽度 |

### 2.4 常见声源录音技巧

| 声源 | 推荐话筒 | 摆位建议 |
|:----:|:--------:|:--------:|
| **人声** | 大振膜电容 | 距嘴 15-25cm，略偏轴减少喷麦，加防喷罩 |
| **原声吉他** | 小振膜电容 | 指向琴颈与琴体连接处（12-14 品），距 15-30cm |
| **电吉他音箱** | SM57 | 对准喇叭纸盆边缘，距网罩 0-5cm，可加一只 Room mic |
| **底鼓** | Beta 52 / D112 | 放入鼓内近后鼓皮，或置于鼓外贴紧鼓皮 |
| **军鼓** | SM57 | 对准上方鼓边 2-5cm，与鼓面成 45°角 |
| **Overheads** | 小振膜电容 | XY 或 ORTF 制式，置于鼓组上方 1-1.5m |
| **钢琴** | 两支小振膜 | 拾音孔对着琴弦，低音区+高音区各一支 |

## 3. 音频编辑

### 3.1 基本编辑操作

| 操作 | 功能 | 快捷键 (常用) |
|:----:|:----:|:------------:|
| **Comping** | 多次录音取最佳片段合成 | 快捷键切换取段 |
| **Time-Stretching** | 不改变音高调整时长 | Warp / Elastic Audio |
| **Pitch Correction** | 修正音高偏差 | Melodyne / Auto-Tune / Flex Pitch |
| **量化 (Quantize)** | 将音频瞬态对齐到网格 | 根据瞬态检测对齐 |
| **Crossfade** | 交叉淡入淡出消除剪辑咔哒声 | 自动或手动 |

### 3.2 音高修正软件

| 软件 | 方式 | 特点 |
|:----:|:----:|:------|
| **Melodyne** | 直接编辑音符（音高、时值、颤音、音量） | 图形化编辑，自然，DNA 技术处理和弦 |
| **Auto-Tune** | 实时修正选定调性音阶 | 经典效果（T-Pain 声），可调速度(Retune Speed) |
| **Waves Tune** | 类似 Melodyne 的图形编辑 | 更经济的选择 |

## 4. MIDI 与虚拟乐器

### 4.1 MIDI 协议基础

| MIDI 消息 | 范围 | 功能 |
|:---------:|:----:|:----:|
| **Note On** | 0-127 音高, 0-127 力度 | 触发音符开始 |
| **Note Off** | 0-127 音高 | 音符结束 |
| **Velocity** | 1-127 | 力度（1=ppp, 64=mf, 127=fff）|
| **Pitch Bend** | -8192 ~ +8191 | 弯音轮控制 |
| **Control Change (CC)** | 0-127 控制器#, 0-127 值 | 表情/调制/音量/声像等 |
| **Program Change** | 0-127 | 切换音色 |
| **Aftertouch** | 0-127 | 键压后持续控制 |

### 4.2 常用 CC 控制器

| CC# | 名称 | 用途 |
|:---:|:----:|:----:|
| 1 | Modulation Wheel | 颤音、表情 |
| 7 | Volume | 通道音量 |
| 10 | Pan | 声像 |
| 11 | Expression | 表情（叠加于 Volume）|
| 64 | Sustain Pedal | 延音踏板 |
| 71 | Resonance | 滤波器共鸣 |
| 74 | Filter Cutoff | 滤波器截止频率 |

### 4.3 合成器类型

| 合成方式 | 原理 | 经典 | 代表音色 |
|:--------:|:----:|:----:|:--------:|
| **减法 (Subtractive)** | 振荡器→滤波器→放大器 | Moog Minimoog、Prophet-5 | Bass、Lead、Pad |
| **FM (Frequency Modulation)** | 正弦波互相调频产生谐波 | Yamaha DX7 | 电钢琴、钟声、金属质感 |
| **波表 (Wavetable)** | 扫描波形表序列 | Serum、Massive | Dubstep Bass、复杂动态 |
| **物理建模 (Physical Modeling)** | 模拟真实乐器物理特性 | Pianoteq、Modelonia | 逼真原声乐器 |

### 4.4 采样器

| 采样器 | 特点 |
|:------:|:------|
| **Kontakt** | 行业标准，海量音色库 |
| **EXS24 / Sampler (Logic)** | Logic 原生采样器 |
| **Simpler / Sampler (Live)** | Ableton 内建采样器 |
| **Battery** | 鼓采样器之王 |

## 5. 效果器

### 5.1 EQ (均衡器)

| 类型 | 可调参数 | 适用 |
|:----:|:--------:|:----:|
| **参数均衡 (Parametric EQ)** | 频率、增益、Q 值 | 精确调整 |
| **图形均衡 (Graphic EQ)** | 固定频段的推子 | 快速整体调整 |
| **搁架均衡 (Shelving EQ)** | 频率、增益 | 高/低频整体提升衰减 |
| **高通/低通滤波** | 截止频率 | 切除不需要的频段 |

### 5.2 压缩 (Compressor)

| 参数 | 作用 | 典型设置 |
|:----:|:------|:--------:|
| **Threshold** | 压缩启动电平阈值 | -20dB ~ -10dB |
| **Ratio** | 压缩比 | 2:1 (轻) ~ 4:1 (中) ~ 10:1 (重) |
| **Attack** | 超过阈值到开始压缩的时间 | 1-10ms (打击乐) / 10-50ms (人声) |
| **Release** | 低于阈值到停止压缩的时间 | 50-100ms (快) / 200-500ms (慢) |
| **Knee** | 压缩启动的平滑度 | Hard (0dB) / Soft (6-12dB) |
| **Makeup Gain** | 压缩后补偿增益 | 与被压缩量匹配 |
| **Sidechain** | 外部信号触发压缩 | 侧链压缩 (Pumping 效果) |

### 5.3 压缩类型

| 类型 | 特点 | 经典插件 |
|:----:|:----:|:--------:|
| **VCA** | 精确、快速、干净 | SSL Bus Compressor |
| **FET** | 快速、激进、染色 | CLA-76 (1176) |
| **Opto** | 平滑、慢速、温暖 | LA-2A |
| **Vari-Mu** | 电子管、音乐性 | Fairchild 670 |

### 5.4 混响 (Reverb)

| 类型 | 模拟空间 | 衰减时间 | 使用场景 |
|:----:|:--------:|:--------:|:--------:|
| **Room** | 小房间 | 0.3-1.0s | 增加存在感、不显山露水 |
| **Hall** | 音乐厅 | 1.5-3.0s | 古典、管弦乐、空间感 |
| **Plate** | 金属板机械混响 | 1.0-3.0s | 人声、军鼓 |
| **Spring** | 弹簧机械混响 | 1.0-2.5s | 吉他音箱、复古摇滚 |
| **Convolution** | 真实空间 IR 采样 | 任意 | 真实空间模拟 |

### 5.5 延迟 (Delay)

| 类型 | 延迟时间 | 反馈 | 特点 |
|:----:|:--------:|:----:|:----:|
| **Slapback** | 70-120ms | 无或低 | 复古摇滚、乡村人声 |
| **Ping-Pong** | 节奏同步 | 中高 | 左右交替回荡 |
| **Dotted 8th** | 附点八分音符 | 低中 | U2 标志性回声 |
| **Tape Echo** | 可变 | 可变 | 温暖、渐趋自激 |
| **Multi-Tap** | 多延迟头 | 独立控制 | 复杂节奏回声 |

### 5.6 调制效果 (Modulation)

| 效果 | 原理 | 听觉特点 | 使用场景 |
|:----:|:----:|:--------:|:--------:|
| **Chorus** | 短延迟+音高微变 | 厚实、拓宽 | 吉他、合成器 Pad、人声加倍 |
| **Flanger** | 极短延迟+反馈 | 喷气机声、梳状滤波 | 过渡效果、吉他 |
| **Phaser** | 全通滤波器相位偏移 | 扫频、旋转感 | 吉他、键盘、电子音乐 |
| **Vibrato** | 音高周期性变化 | 摇摆、颤音 | 吉他、人声 |
| **Tremolo** | 音量周期性变化 | 断断续续 | 复古吉他、电子 |

## 6. 混音

### 6.1 混音流程

| 步骤 | 操作 | 目标 |
|:----:|:----:|:----:|
| **1. 增益级 (Gain Staging)** | 调整各轨道输入电平 | 统一电平，留足余量 |
| **2. 平衡 (Balance)** | 推子调整相对音量 | 基础混音框架 |
| **3. 声像 (Panning)** | 分配立体声位置 | 构建声场宽度 |
| **4. EQ** | 频率均衡 | 去除冲突，增加清晰度 |
| **5. 动态处理** | 压缩/门限 | 控制动态范围 |
| **6. 空间效果** | 混响/延迟 | 创造深度和空间 |
| **7. 自动化 (Automation)** | 参数随时间变化 | 动态变化 |
| **8. 参考对比** | 与商业作品对比 | 检查平衡 |

### 6.2 Bus 与编组

| 编组 | 内容 | 处理方式 |
|:----:|:----:|:--------:|
| **鼓组 Bus** | 底鼓、军鼓、Hihat、Overheads | 鼓组压缩、整体 EQ |
| **人声 Bus** | 主唱、伴唱 | 人声压缩、混响 Send |
| **乐器 Bus** | 吉他、键盘等 | 乐器编组处理 |
| **Master Bus** | 总输出 | 限制器、轻微压缩 |

### 6.3 自动化类型

| 自动化 | 用途 | 示例 |
|:------:|:----:|:----:|
| **音量** | 段落音量变化 | 副歌音量提升 |
| **声像** | 移动声场位置 | 吉他左右摆动 |
| **效果参数** | 效果量变化 | 混响渐入、滤波器扫频 |
| **EQ 频率** | 频段扫频 | 过渡段 Filter Sweep |

## 7. 母带 (Mastering)

### 7.1 母带流程

| 步骤 | 处理 | 目标 |
|:----:|:----:|:----:|
| **1. 参考监听** | 分析参考曲目标准 | 确定响度、频率目标 |
| **2. EQ** | 整体频率平衡微调 | 修正频率缺陷 |
| **3. 立体声增强** | Mid/Side 处理 | 控制宽度和中心 |
| **4. 压缩/多段压缩** | 控制动态一致性 | 粘合混音 |
| **5. 限制器 (Limiter)** | 最大化响度 | 无削波的峰值控制 |
| **6. 抖动 (Dithering)** | 位深降低处理 | 减少量化噪声 |
| **7. 输出** | 多格式导出 | WAV/MP3/FLAC |

### 7.2 响度标准

| 平台 | 目标 Integrated LUFS | 真峰值 (True Peak) |
|:----:|:-------------------:|:-----------------:|
| **Spotify** | -14 LUFS | -1 dBTP |
| **Apple Music** | -16 LUFS | -1 dBTP |
| **YouTube** | -14 LUFS | -1 dBTP |
| **Tidal** | -14 LUFS | -1 dBTP |
| **CD** | -9 ~ -12 LUFS | -0.1 dBTP |
| **广播** | -23 LUFS (EBU R128) | -1 dBTP |
