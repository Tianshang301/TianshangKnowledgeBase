# 音乐发展史 (History of Music)

## 概述 (Overview)

音乐发展史记录了人类从远古时代到当代的音乐演进过程。这一历程不仅反映了审美观念的变迁，更折射出社会结构、宗教信仰与技术革新的深层互动。

### 时间线概览

| 时期 (Period) | 时间范围 (Time Range) | 主要特征 (Key Features) |
| --- | --- | --- |
| 中世纪 (Medieval) | 500–1400 | 格里高利圣咏、宗教主导 |
| 文艺复兴 (Renaissance) | 1400–1600 | 复调音乐、人文主义 |
| 巴洛克 (Baroque) | 1600–1750 | 通奏低音、装饰音、情感论 |
| 古典主义 (Classical) | 1750–1820 | 奏鸣曲式、平衡结构 |
| 浪漫主义 (Romantic) | 1820–1900 | 个人情感、标题音乐 |
| 20世纪至今 (Contemporary) | 1900–至今 | 多元风格、电子音乐 |

## 巴洛克时期 (Baroque: 1600–1750)

### 核心特征

巴洛克 (Baroque) 一词原指"形状不规则的珍珠"，音乐上表现为华丽、精致、富有装饰性。

| 特征 | 说明 |
|------|------|
| 通奏低音 (Basso Continuo) | 持续的低音线条即兴伴奏 |
| 情感论 (Doctrine of Affections) | 特定音乐手法对应特定情绪 |
| 对比与戏剧性 | 强弱、快慢、音色的强烈对比 |

### 重要作曲家 (Key Composers)

- **巴赫 (J.S. Bach, 1685–1750)** — 赋格艺术巅峰，代表《马太受难曲》《勃兰登堡协奏曲》
- **亨德尔 (G.F. Handel, 1685–1759)** — 清唱剧大师，代表《弥赛亚》
- **维瓦尔第 (A. Vivaldi, 1678–1741)** — 协奏曲奠基人，代表《四季》

### 主要体裁

- **赋格 (Fugue)** — 复调对位的最高形式
- **协奏曲 (Concerto)** — 独奏与乐队的对话
- **组曲 (Suite)** — 舞曲的集合
- **清唱剧 (Oratorio)** — 宗教题材的大型声乐作品

## 古典主义时期 (Classical: 1750–1820)

### 核心特征

"古典" (Classical) 意味着清晰、平衡、形式完美。音乐追求理性的优雅。

### 奏鸣曲式 (Sonata Form)

奏鸣曲式是古典时期最重要的音乐结构：

| 部分 | 功能 |
|------|------|
| 呈示部 (Exposition) | 展示主部与副部两个主题 |
| 展开部 (Development) | 主题动机的变形与冲突 |
| 再现部 (Recapitulation) | 主题回归与统一 |

### 重要作曲家

- **海顿 (Haydn, 1732–1809)** — 交响曲之父，确立四乐章结构
- **莫扎特 (Mozart, 1756–1791)** — 旋律天才，歌剧与协奏曲的完美结合
- **贝多芬 (Beethoven, 1770–1827)** — 古典与浪漫的桥梁，九部交响曲的不朽遗产

## 浪漫主义时期 (Romantic: 1820–1900)

### 核心特征

浪漫主义 (Romanticism) 强调个人情感的表达、对自然的崇拜与民族意识的觉醒。

### 标题音乐 (Program Music)

与纯音乐 (Absolute Music) 相对，标题音乐通过音乐讲述故事或描绘场景。

| 作曲家 | 代表作品 | 创作理念 |
|--------|----------|----------|
| 柏辽兹 (Berlioz) | 《幻想交响曲》 | 固定乐思 (Idée Fixe) |
| 李斯特 (Liszt) | 《前奏曲》 | 交响诗 (Symphonic Poem) |
| 理查·施特劳斯 (R. Strauss) | 《查拉图斯特拉如是说》 | 极致描绘性 |

### 民族乐派

- **俄罗斯** — 柴可夫斯基、穆索尔斯基
- **北欧** — 格里格、西贝柳斯
- **东欧** — 德沃夏克、斯美塔那
- **西班牙** — 阿尔贝尼兹、格拉纳多斯

### 歌剧的繁荣

威尔第 (Verdi) 的意大利歌剧与瓦格纳 (Wagner) 的乐剧 (Music Drama) 代表了歌剧艺术的巅峰。

## 现代与当代 (Modern & Contemporary: 1900–至今)

### 20世纪音乐的主要流派

| 流派 | 代表人物 | 特点 |
|------|---------|------|
| 印象主义 (Impressionism) | 德彪西 (Debussy) | 色彩性、模糊调性 |
| 表现主义 (Expressionism) | 勋伯格 (Schoenberg) | 无调性、十二音序列 |
| 新古典主义 (Neoclassicism) | 斯特拉文斯基 (Stravinsky) | 回归古典形式 |
| 简约主义 (Minimalism) | 菲利普·格拉斯 (P. Glass) | 重复音型、渐进变化 |
| 电子音乐 (Electronic) | 斯托克豪森 (Stockhausen) | 合成器与电子声效 |

### 爵士、流行与融合

20世纪也是流行音乐崛起的时代：爵士 (Jazz)、摇滚 (Rock)、嘻哈 (Hip-Hop) 等风格深刻影响了当代音乐生态。

## 中国传统音乐与世界

| 时期 | 代表性音乐形式 |
|------|---------------|
| 先秦 | 礼乐制度、钟磬祭祀音乐 |
| 唐宋 | 大曲、词调、说唱 |
| 明清 | 戏曲四大声腔 (昆腔、弋阳腔、梆子腔、皮黄腔) |
| 近现代 | 中西结合 (如《黄河大合唱》) |
