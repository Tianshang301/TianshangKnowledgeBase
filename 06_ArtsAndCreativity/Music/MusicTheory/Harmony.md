# 和声学基础

## 自然音和声 (Diatonic Harmony)

### 和弦功能分组

在和声学中，和弦按功能分为三组：

| 功能组 | 和弦 | 特征 | 感觉 |
|--------|------|------|------|
| 主功能 (T) | I, vi, iii | 稳定 | 回家的感觉 |
| 下属功能 (SD) | IV, ii | 离开主音 | 离开的感觉 |
| 属功能 (D) | V, vii° | 强烈导向主音 | 紧张，需要解决 |

和声功能理论是调性音乐的基石。主功能提供稳定感，下属功能引入不稳定，属功能则强烈要求解决回主功能，形成 T-S-D-T 的功能循环。

### 功能进行原则

```
T → SD → D → T
I → IV → V → I (最经典进行)
```

禁止进行: SD → D → SD (功能倒置)

### 和弦进行公式

#### I-IV-V-I (最基础)

```
C 大调: C → F → G → C
听感: 完整、终止感强
```

主-下属-属-主的进行是和声功能逻辑最直接的体现，几乎适用于所有音乐风格。

#### ii-V-I (爵士核心)

```
C 大调: Dm → G → C
听感: 流畅、爵士味
```

ii-V-I 是爵士乐最核心的和声进行，每个音级的根音关系为上行四度，构成最自然的和声运动。

#### I-vi-IV-V (50年代进行)

```
C 大调: C → Am → F → G
听感: 怀旧、流行 (Do-Wop)
```

#### I-V-vi-IV (流行和弦)

```
C 大调: C → G → Am → F
听感: 能量感、乐观
示例: "Let It Be" - The Beatles
```

#### 卡农进行

```
C 大调: C → G → Am → Em → F → C → F → G
示例: "Canon in D" - Pachelbel
```

## 终止式 (Cadences)

| 类型 | 和弦进行 | 感觉 | 使用场景 |
|------|---------|------|---------|
| 正格终止 (PAC) | V → I | 结束感最强 | 段落的束 |
| 不完全正格终止 (IAC) | V → I (非根音旋律) | 结束感较弱 | 乐句结束 |
| 变格终止 | IV → I | 神圣、阿门 | 宗教音乐 |
| 半终止 | I → V 或 IV → V | 暂停、未结束 | 乐句中间 |
| 伪终止 ( deceptive ) | V → vi | 意外、欺骗 | 制造悬念 |

## 和弦进行模式

### 和声节奏 (Harmonic Rhythm)

和弦更换的频率。常见的模式：
```
慢节奏: 每 2-4 小节换一次和弦 (古典慢板)
中节奏: 每 1-2 小节换一次 (流行歌曲)
快节奏: 每拍换和弦 (爵士 bebop)
```

### 全和弦进行 (Circle Progression)

按五度下行方向连接和弦：

```
C → F → B° → E → Am → Dm → G → C
```

这是最自然的和声运动方式。

## 副属和弦 (Secondary Dominants)

属和弦的属和弦，临时借用其他调的属和弦：

| 标记 | 含义 | C 大调示例 |
|------|------|----------|
| V/V | 属和弦的属和弦 | D7 (→ G7) |
| V/ii | ii 的属和弦 | A7 (→ Dm) |
| V/vi | vi 的属和弦 | E7 (→ Am) |
| V/iii | iii 的属和弦 | B7 (→ Em) |
| V/IV | IV 的属和弦 | C7 (→ F) |

```
// 和声进行示例（钢琴谱）：
// C → E7 → Am → D7 → G7 → C
//     V/vi    V/ii   V/V
```

副属和弦通过引入临时变化音，丰富和声色彩，增强向目标和弦的倾向性。

## 转调 (Modulation)

### 近关系转调
转到相差一个调号以内的调：

| 原调 | 近关系调 |
|------|---------|
| C 大调 | G 大调 (1♯)、F 大调 (1♭)、A 小调、D 小调、E 小调 |

### 共同和弦转调 (Pivot Chord Modulation)

通过两个调共有的和弦作为桥梁：
```
C 大调: C - F - C - G7 - C
                     ↑ (G7 = D 小调的 V7)
D 小调:                 Dm - Am - Dm
```

### 直接转调 (Direct/Phrase Modulation)

没有过渡和弦，直接进入新调。

### 其他转调方式

- **半音转调**: 通过半音变化进入新调
- **渐调 (Gradual Modulation)**: 通过一系列近关系调逐步转远
- **等和弦转调**: 利用等音变化 (如减七和弦)

## 变和弦 (Chromatic Harmony)

### 那不勒斯六和弦 (Neapolitan 6th)

```
标记: N6 或 ♭II6
构成: ♭3 - 4 - ♭6 (第一转位)
C 大调: D♭ - F - A♭
功能: 下属功能，通常解决到 V
```

### 增六和弦 (Augmented 6th Chords)

三种增六和弦，都包含增六度音程 (♭6-♯4)，都解决到 V：
| 类型 | 构成 | 示例 (C 大调) |
|------|------|-------------|
| 意大利增六 | ♭6-1-♯4 | A♭-C-F♯ |
| 法国增六 | ♭6-1-2-♯4 | A♭-C-D-F♯ |
| 德国增六 | ♭6-1-♭3-♯4 | A♭-C-E♭-F♯ |

## 四部和声 (SATB)

### 声部分配

| 声部 | 缩写 | 范围 |
|------|------|------|
| 女高音 | S | C4 - C6 |
| 女中音 | A | G3 - G5 |
| 男高音 | T | C3 - C5 |
| 男低音 | B | E2 - E4 |

### 和弦排列规则

1. **密集排列**: 上三声部相邻音在八度以内
2. **开放排列**: 上三声部相邻音超过八度
3. **混合排列**: 混合使用

### 重复规则

| 和弦 | 优先重复 | 其次 |
|------|---------|------|
| 大三和弦 (I) | 根音 | 五音 (少重复三音) |
| 小三和弦 (i) | 根音 | 五音 |
| 减三和弦 (vii°) | 三音 | 根音 |
| 属七和弦 (V7) | 根音、五音 | 省略五音 |

### 声部进行规则

```
禁止:
1. 平行五度 (两个声部间隔纯五度平行移动)
2. 平行八度 (两个声部间隔纯八度平行移动)
3. 反向八度/五度 (反向进行到纯八/五度)
4. 声部交叉 (女低音高于女高音等)
5. 隐伏五/八度 (同向进行到五/八度，外声部)
```

## 音级标记法

罗马数字标记法：

```
C 大调:
I    ii   iii  IV   V    vi   vii°
C    Dm   Em   F    G    Am   B°
```

爵士标记法：

```
CΔ7  Dm7  Em7  FΔ7  G7   Am7  Bø7
```

## 和声分析流程

```
1. 确定调性
2. 标记每个和弦的级数
3. 识别功能组 (T/SD/D)
4. 标记终止式
5. 识别非和弦音 (经过音、辅助音等)
6. 检查声部进行
```
