# 音阶详解

## 大调音阶 (Major Scale)

### 全半音模式
```
W - W - H - W - W - W - H
全 - 全 - 半 - 全 - 全 - 全 - 半
```

以 C 大调为例：
```
C - D - E - F - G - A - B - C
全  全  半  全  全  全  半
```

### 所有大调与调号

| 大调 | 调号 | 升降号 |
|------|------|--------|
| C 大调 | 无 | - |
| G 大调 | 1个升号 | F♯ |
| D 大调 | 2个升号 | F♯, C♯ |
| A 大调 | 3个升号 | F♯, C♯, G♯ |
| E 大调 | 4个升号 | F♯, C♯, G♯, D♯ |
| B 大调 | 5个升号 | F♯, C♯, G♯, D♯, A♯ |
| F♯大调 | 6个升号 | F♯, C♯, G♯, D♯, A♯, E♯ |
| C♯大调 | 7个升号 | F♯, C♯, G♯, D♯, A♯, E♯, B♯ |
| F 大调 | 1个降号 | B♭ |
| B♭大调 | 2个降号 | B♭, E♭ |
| E♭大调 | 3个降号 | B♭, E♭, A♭ |
| A♭大调 | 4个降号 | B♭, E♭, A♭, D♭ |
| D♭大调 | 5个降号 | B♭, E♭, A♭, D♭, G♭ |
| G♭大调 | 6个降号 | B♭, E♭, A♭, D♭, G♭, C♭ |
| C♭大调 | 7个降号 | B♭, E♭, A♭, D♭, G♭, C♭, F♭ |

### 五度圈 (Circle of Fifths)

```
            C
          /   \
        F       G
       /         \
      B♭          D
     /              \
    E♭              A
     \              /
      A♭           E
       \         /
        D♭     B
          \   /
            G♭/F♯
```

顺时针（上行五度）每步加一个升号，逆时针（下行五度）每步加一个降号。

### 关系小调 (Relative Minor)

每个大调都有一个关系小调，共享相同的调号：

```
C 大调 → A 小调 (无升降)
G 大调 → E 小调 (1♯)
D 大调 → B 小调 (2♯)
F 大调 → D 小调 (1♭)
大调的关系小调 = 大调的第六级音
```

大调音阶由七个音组成，是全音和半音按特定模式排列的序列。大调音阶的特点包含大三度和大七度，其明亮、开阔的听感使其成为西方音乐中最常用的音阶。

## 小调音阶 (Minor Scale)

### 三种小调

| 类型 | 音程模式 | 特征音 | 示例 (A 小调) |
|------|---------|--------|-------------|
| 自然小调 | W-H-W-W-H-W-W | ♭3, ♭6, ♭7 | A-B-C-D-E-F-G-A |
| 和声小调 | W-H-W-W-H-3H-H | ♭3, ♭6, ♯7 | A-B-C-D-E-F-G♯-A |
| 旋律小调 (上行) | W-H-W-W-W-W-H | ♭3, ♯6, ♯7 | A-B-C-D-E-F♯-G♯-A |

**注意**: 旋律小调下行时还原为自然小调。

### 小调与大调对应
```
C 大调: C D E F G A B C
        ↓ ↓   ↓ ↓   ↓ ↓
A 小调: A B C D E F G A
       1 2 ♭3 4 5 ♭6 ♭7
```

小调音阶分为三种类型：自然小调、和声小调和旋律小调。自然小调的特点是包含小三度和小七度，听感忧郁而古朴。和声小调通过升高第七级音产生导音，增强了向主音的倾向性。旋律小调上行时升高第六和第七级音，下行时还原，使旋律线条更加流畅。

## 教会调式 (Modes)

### 调式总览

| 调式 | 音程模式 | 从白键开始 | 特征音 | 听感 |
|------|---------|-----------|--------|------|
| Ionian (伊奥尼亚) | W-W-H-W-W-W-H | C 到 C | - | 明亮 (大调) |
| Dorian (多利亚) | W-H-W-W-W-H-W | D 到 D | ♭3 | 爵士、民谣 |
| Phrygian (弗里吉亚) | H-W-W-W-H-W-W | E 到 E | ♭2 | 西班牙、弗拉明戈 |
| Lydian (利底亚) | W-W-W-H-W-W-H | F 到 F | ♯4 | 梦幻、缥缈 |
| Mixolydian (混合利底亚) | W-W-H-W-W-H-W | G 到 G | ♭7 | 布鲁斯、摇滚 |
| Aeolian (伊奥利亚) | W-H-W-W-H-W-W | A 到 A | ♭3, ♭6, ♭7 | 忧郁 (自然小调) |
| Locrian (洛克里亚) | H-W-W-H-W-W-W | B 到 B | ♭2, ♭5 | 不稳定 |

### 调式公式（相对于大调）
```
Ionian:    1 2 3 4 5 6 7       (大调)
Dorian:    1 2 ♭3 4 5 6 ♭7    (小调 + ♯6)
Phrygian:  1 ♭2 ♭3 4 5 ♭6 ♭7  (小调 + ♭2)
Lydian:    1 2 3 ♯4 5 6 7     (大调 + ♯4)
Mixolydian:1 2 3 4 5 6 ♭7     (大调 + ♭7)
Aeolian:   1 2 ♭3 4 5 ♭6 ♭7   (自然小调)
Locrian:   1 ♭2 ♭3 4 ♭5 ♭6 ♭7 (减音阶)
```

教会调式起源于古希腊音乐理论，在中世纪和文艺复兴时期广泛使用。每个调式由全音和半音的不同排列构成，赋予其独特的听觉色彩。现代音乐中，调式被广泛用于爵士、摇滚和世界音乐创作。

## 五声音阶 (Pentatonic Scale)

### 大调五声音阶

```
公式: 1 2 3 5 6
C 大调五声: C D E G A (省略 F 和 B)
```

### 小调五声音阶

```
公式: 1 ♭3 4 5 ♭7
A 小调五声: A C D E G
```

### 应用

五声音阶广泛用于民谣、摇滚、布鲁斯和中国传统音乐。由于没有半音，五声音阶的音响和谐悦耳，容易上手，是即兴演奏的基础工具。

## 布鲁斯音阶 (Blues Scale)

```
小调布鲁斯: 1 ♭3 4 ♯4 5 ♭7
            (小调五声 + ♯4/♭5 蓝调音)

A 小调布鲁斯: A C D D♯ E G
```

布鲁斯音阶在小调五声音阶的基础上增加了降五级音（也称蓝调音），赋予音乐独特的布鲁斯韵味，是爵士、摇滚和 R&B 音乐的基石。

## 全音音阶 (Whole-Tone Scale)

```
公式: W-W-W-W-W-W (6个音)
C 全音阶: C D E F♯ G♯ A♯ C
```

全音音阶由六个全音组成，没有半音，营造出梦幻、漂浮的听感。常用于印象派音乐（德彪西）和爵士乐中，制造朦胧而神秘的效果。

## 半音阶 (Chromatic Scale)

```
C: C C♯ D D♯ E F F♯ G G♯ A A♯ B C
  (12个半音)
```

半音阶包含全部12个半音，是最完整的音阶形式。它不归属于任何调性，常用于增加音乐的紧张感和色彩变化，在古典、爵士和现代音乐中广泛使用。

## 音阶度数名称

| 级数 | 名称 | 英文 |
|------|------|------|
| 1 | 主音 | Tonic |
| 2 | 上主音 | Supertonic |
| 3 | 中音 | Mediant |
| 4 | 下属音 | Subdominant |
| 5 | 属音 | Dominant |
| 6 | 下中音 | Submediant |
| 7 | 导音 | Leading Tone |
| 8 | 主音 (八度) | Tonic |

音阶中的每个音都有特定的名称，这些名称反映了它们在调性中的功能角色。主音是调性的中心，属音和下属音分别从上方和下方支撑主音，导音则强烈倾向于解决到主音。

## 音阶练习方法

1. **唱名法**: do re mi fa sol la ti do
2. **双手对称练习**: 钢琴/其他乐器上的音阶练习
3. **模进 (Sequence)**: 按固定模式在不同音高上重复
4. **调式对比**: 同一主音不同调式对比练习
5. **节奏变体**: 附点节奏、三连音节奏变化练习
6. **力度变化**: 上行渐强、下行渐弱
