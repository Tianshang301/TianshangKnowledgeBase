# 和弦详解

## 三和弦 (Triads)

### 四种三和弦
| 类型 | 构成 | 音程结构 | 示例 (C) | 标记 |
|------|------|---------|---------|------|
| 大三和弦 | 根音 + 大三度 + 纯五度 | 大三+小三 | C-E-G | C, CΔ |
| 小三和弦 | 根音 + 小三度 + 纯五度 | 小三+大三 | C-E♭-G | Cm, C- |
| 增三和弦 | 根音 + 大三度 + 增五度 | 大三+大三 | C-E-G♯ | C+, Caug |
| 减三和弦 | 根音 + 小三度 + 减五度 | 小三+小三 | C-E♭-G♭ | C°, Cdim |

三和弦由三个音按照三度关系叠置而成，是最基本的和弦形式。大三和弦与小三和弦为协和和弦，增三和弦与减三和弦则具有不协和性，需要解决。

### 和弦和弦级数

在大调中，各级三和弦：
| 级数 | 和弦 | 标记 | 功能 |
|------|------|------|------|
| I | 大三和弦 | C | 主和弦 (Tonic) |
| ii | 小三和弦 | Dm | 上主和弦 (Supertonic) |
| iii | 小三和弦 | Em | 中和弦 (Mediant) |
| IV | 大三和弦 | F | 下属和弦 (Subdominant) |
| V | 大三和弦 | G | 属和弦 (Dominant) |
| vi | 小三和弦 | Am | 下中和弦 (Submediant) |
| vii° | 减三和弦 | B° | 导和弦 (Leading Tone) |

## 七和弦 (Seventh Chords)

### 五种常用七和弦
| 类型 | 构成 | 音程 | 示例 (C) | 标记 |
|------|------|------|---------|------|
| 大七和弦 | 大三 + 大七度 | 大三+小三+大三 | C-E-G-B | Cmaj7, CΔ7 |
| 属七和弦 | 大三 + 小七度 | 大三+小三+小三 | C-E-G-B♭ | C7 |
| 小七和弦 | 小三 + 小七度 | 小三+大三+小三 | C-E♭-G-B♭ | Cm7, C-7 |
| 半减七和弦 | 减三 + 小七度 | 小三+小三+大三 | C-E♭-G♭-B♭ | Cø7, Cm7♭5 |
| 减七和弦 | 减三 + 减七度 | 小三+小三+小三 | C-E♭-G♭-B♭♭ | C°7, Cdim7 |

### 其他七和弦
| 类型 | 构成 | 标记 | 示例 |
|------|------|------|------|
| 小大七和弦 | 小三 + 大七度 | Cm(maj7) | C-E♭-G-B |
| 增大七和弦 | 增三 + 大七度 | C+maj7 | C-E-G♯-B |
| 属七降九 | 属七 + ♭9 | C7♭9 | C-E-G-B♭-D♭ |

七和弦在三和弦的基础上叠加七度音，增加了和弦的色彩和紧张度。所有七和弦都包含不协和的七度音程，因此都是不协和和弦，需要解决到稳定的三和弦。

## 延伸和弦 (Extended Chords)

### 九和弦 (9th)

在大七或属七基础上叠加九度音：
```
C 大九: C-E-G-B-D (CΔ9)
C 属九: C-E-G-B♭-D (C9)
C 小九: C-E♭-G-B♭-D (Cm9)
```

### 十一和弦 (11th)

```
C 属十一: C-E-G-B♭-D-F (C11)
        通常省略三音
```

### 十三和弦 (13th)

```
C 属十三: C-E-G-B♭-D-F-A (C13)
        通常省略五音和十一音
```

### 和弦排列

延伸和弦中通常省略不协和的音：

```
C13 常用排列: C - B♭ - E - A (省略 G、D、F)
              根音 七音 三音 十三音
```

## 挂留和弦 (Sus Chords)

| 类型 | 构成 | 示例 (C) | 标记 |
|------|------|---------|------|
| 挂二 | 根音 + 大二度 + 纯五度 | C-D-G | Csus2 |
| 挂四 | 根音 + 纯四度 + 纯五度 | C-F-G | Csus4 |

挂留和弦没有三度，不明确是大调还是小调。

## 斜杠和弦 (Slash Chords)

表示根音不是和弦的原位根音：

```
C/E = C 大和弦，E 为低音 (第一转位)
C/G = C 大和弦，G 为低音 (第二转位)
F/G = F 大和弦，G 为低音 (复合和弦)
```

## 和弦转位

### 三和弦转位
| 转位 | 低音 | 音程结构 | 示例 (C 大三) | 标记 |
|------|------|---------|------------|------|
| 原位 | 根音 | 根·三·五 | C-E-G | C |
| 第一转位 | 三音 | 三·五·根 | E-G-C | C/E |
| 第二转位 | 五音 | 五·根·三 | G-C-E | C/G |

### 七和弦转位
| 转位 | 低音 | 音程结构 | 示例 (C7) | 标记 |
|------|------|---------|----------|------|
| 原位 | 根音 | 根·三·五·七 | C-E-G-B♭ | C7 |
| 第一转位 | 三音 | 三·五·七·根 | E-G-B♭-C | C7/E |
| 第二转位 | 五音 | 五·七·根·三 | G-B♭-C-E | C7/G |
| 第三转位 | 七音 | 七·根·三·五 | B♭-C-E-G | C7/B♭ |

## 声部进行 (Voice Leading)

基本原则：
1. **同音保持**: 共同音尽量保持在同一声部
2. **就近解决**: 各声部用最小的移动 (级进为主)
3. **避免平行五八度**: 两个声部平行移动纯五/纯八度
4. **避免反向八度**: 反向进行到八度
5. **避免声部交叉**: 声部保持上下位置不变
