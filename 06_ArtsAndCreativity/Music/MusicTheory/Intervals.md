# 音程详解

## 音程定义

音程是两个音之间的音高距离。下方音称为根音，上方音称为冠音。音程分为旋律音程（先后发声）与和声音程（同时发声）。

## 音程的度数命名

### 自然音程 (基本度数)

| 度数 | 半音数 | 名称 | 简称 |
|------|--------|------|------|
| 纯一度 | 0 | 同度 | P1 |
| 小二度 | 1 | 半音 | m2 |
| 大二度 | 2 | 全音 | M2 |
| 小三度 | 3 | - | m3 |
| 大三度 | 4 | - | M3 |
| 纯四度 | 5 | - | P4 |
| 增四度/减五度 | 6 | 三全音 | A4/d5 |
| 纯五度 | 7 | - | P5 |
| 小六度 | 8 | - | m6 |
| 大六度 | 9 | - | M6 |
| 小七度 | 10 | - | m7 |
| 大七度 | 11 | - | M7 |
| 纯八度 | 12 | 八度 | P8 |

### 音程共分五类

- **纯音程 (Perfect)**: 1、4、5、8 度
- **大音程 (Major)**: 2、3、6、7 度
- **小音程 (Minor)**: 大音程减半音
- **增音程 (Augmented)**: 纯/大音程加半音
- **减音程 (Diminished)**: 纯/小音程减半音

## 协和音程 vs 不协和音程
| 类别 | 音程 | 听觉感受 |
|------|------|---------|
| 极完全协和 | 纯一度、纯八度 | 空旷、融合 |
| 完全协和 | 纯四度、纯五度 | 稳定、和谐 |
| 不完全协和 | 大/小三度、大/小六度 | 柔和、悦耳 |
| 不协和 | 大/小二度、大/小七度、增四/减五度 | 紧张、需要解决 |

## 音程转位

将音程的根音移高八度或冠音移低八度。
| 原音程 | 转位后 | 规律 |
|--------|--------|------|
| 纯一度 | 纯八度 | 纯↔纯 |
| 大二度 | 小七度 | 大↔小 |
| 大三度 | 小六度 | 大↔小 |
| 纯四度 | 纯五度 | 纯↔纯 |
| 增四度 | 减五度 | 增↔减 |
| 小六度 | 大三度 | 小↔大 |

**规则**: 原度数 + 转位度数 = 9，大↔小，增↔减，纯↔纯

## 复音程

超过八度的音程：

```
复音程 = 单音程 + 7 (度数)
如: 十度 = 三度 + 7
   C4 → E5 是十度 (大三度 + 八度)
```

### 训练方法

1. **唱名法**: 用 do re mi fa sol la ti do 唱音阶
2. **对比法**: 同一音程在不同调上反复听
3. **构唱**: 给定一个音，唱出指定音程的上方音
4. **App 练习**: EarMaster, Perfect Ear, Tenuto

## 音程在音乐中的应用

- **旋律音程**: 构成旋律的横向进行
- **和声音程**: 两个音同时发声 (双音、和弦基础)
- **经过音/辅助音**: 填充音程之间的空隙
- **跳进 vs 级进**: 级进 (二度) 平滑，跳进 (三度以上) 有表现力
