# 计算机科学学习路线图

## 入门阶段 (Beginner)

### 1. 计算机基础
- 文件系统与路径、命令行操作（CLI）
- 开发环境搭建（IDE、包管理、版本管理）
- **资源**：《计算机科学导论》

### 2. 编程入门
- 选择一门语言：**Python**（推荐）、JavaScript、Java
- Python 学习路径：
  - 语法基础（变量、类型、控制流、函数）
  - 面向对象编程（类、继承、多态）
  - 基本数据结构和标准库
- **免费资源**：
  - CS50 (Harvard) - 优秀的入门课程
  - Python for Everybody (Coursera)
  - 《Automate the Boring Stuff with Python》

### 3. 数据结构与算法
- **核心数据结构**：数组、链表、栈、队列、哈希表、树、堆、图
- **核心算法**：排序（快排、归并）、搜索（二分）、图遍历（BFS/DFS）
- **教材推荐**：《算法导论》CLRS / 《图解算法》
- **刷题平台**：LeetCode、Codeforces

### 4. Git 版本控制
- 基本工作流：clone/add/commit/push/pull/branch/merge
- **资源**：Learn Git Branching (交互式)

## 中级阶段 (Intermediate)

### 1. 操作系统
- 进程与线程、调度算法、内存管理（虚拟内存、分页）、文件系统
- **教材推荐**：《Operating Systems: Three Easy Pieces》
- **项目**：实现一个简易 shell

### 2. 计算机网络
- OSI 七层模型、TCP/IP、HTTP/HTTPS、DNS、TLS
- **教材推荐**：《计算机网络：自顶向下方法》（Computer Networking: A Top-Down Approach）
- **项目**：实现简易 HTTP 服务器

### 3. 数据库
- 关系型：SQL、索引、事务、ACID、范式设计
- NoSQL：MongoDB、Redis 的基本使用
- **教材推荐**：《Database System Concepts》
- **项目**：设计一个博客系统的数据库

### 4. 设计模式
- 常用模式：单例、工厂、策略、观察者、装饰器
- **教材推荐**：《Head First Design Patterns》

### 5. 构建项目
- 选择一个完整的全栈项目：
  - Web 应用：Flask/Django + React + PostgreSQL
  - CLI 工具：用 Python 写实用命令行工具
  - API 服务：RESTful API 设计与实现

## 高级阶段 (Advanced)

### 1. 编译原理
- 词法分析、语法分析（LL/LR）、语义分析、中间代码生成、优化
- **教材推荐**：《编译原理》（Dragon Book）
- **项目**：实现一个简易解释器或编译器

### 2. 分布式系统
- 一致性（CAP、Paxos/Raft）、分布式存储、消息队列
- **教材推荐**：《Designing Data-Intensive Applications》（DDIA）
- **资源**：MIT 6.824 Distributed Systems

### 3. 人工智能与机器学习
- **基础**：线性回归、逻辑回归、神经网络、CNN/RNN
- **进阶**：Transformer、强化学习、GAN
- **资源**：Stanford CS229 / CS231n / CS224n

### 4. 系统设计
- 设计大型系统：URL 短链、聊天系统、推荐系统
- **资源**：System Design Interview (Alex Xu)
- **练习**：设计 Netflix / Twitter / 微信

## 书籍推荐汇总

| 阶段 | 书名 |
|------|------|
| 入门 | 《计算机科学概论》、《Python 编程从入门到实践》 |
| 数据结构 | 《算法图解》、《算法》（第4版） |
| 系统 | 《深入理解计算机系统》（CSAPP）、《TCP/IP 详解》 |
| 进阶 | 《设计数据密集型应用》（DDIA）、《现代操作系统》 |

## 学习建议

1. **动手编码**：每学一个概念都要写代码验证
2. **做项目**：项目是 best teacher，不要只刷题
3. **读源码**：阅读优秀的开源项目代码
4. **写博客**：费曼学习法，把学会的教给别人
5. **持续学习**：技术更新快，保持学习的习惯
