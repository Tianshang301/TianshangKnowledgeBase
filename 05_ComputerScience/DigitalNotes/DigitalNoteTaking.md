# 数字化笔记 (Digital Note Taking)

## 概述

数字化笔记是利用电子设备和软件进行信息记录、组织与检索的方法体系。相比传统纸笔笔记，数字化笔记具备可搜索、可同步、多媒体融合等优势。

## 核心原则

- **可检索性**: 全文搜索 + 标签体系 + 结构化元数据
- **双向链接**: 建立知识间的关联网络
- **渐进式总结**: 从原始信息到提炼洞察的分层处理
- **跨平台同步**: 多设备无缝衔接

## 主流方法论

- **Zettelkasten**: 原子化卡片 + 双向链接构建知识网络
- **PARA**: Project/Area/Resource/Archive 四层组织框架
- **GTD 收件箱**: 快速捕获 → 定期处理 → 归档
- **每日笔记 + 周回顾**: 持续记录与定期复盘
