# 数据库系统 Database Systems

数据库系统是用于存储、管理和检索数据的软件系统，是现代信息系统的核心基础设施。数据库管理系统提供数据定义、数据操纵、数据控制和数据查询等功能，确保数据的一致性、完整性和安全性。

关系型数据库（如 MySQL、PostgreSQL、Oracle）以表格形式组织数据，使用 SQL 进行查询和操作，支持 ACID 事务特性。NoSQL 数据库（如 MongoDB、Redis、Cassandra）提供灵活的数据模型，适用于大规模分布式系统和特定应用场景。NewSQL 数据库试图结合两者的优势。

数据库设计的核心概念包括实体-关系模型、范式化和索引优化。事务管理确保并发环境下的数据一致性，恢复机制在系统故障时保护数据。随着云计算的发展，云数据库服务（如 AWS RDS、Azure SQL）提供了按需扩展和高可用的解决方案。数据仓库和大数据技术进一步扩展了数据库系统的处理能力。
