## 二、结构化编程时代

1968年，Edsger Dijkstra 发表《Go To Statement Considered Harmful》，推动了结构化编程革命。

C 语言诞生于1972年，由 Dennis Ritchie 在贝尔实验室开发，用于重写 Unix 操作系统。

```c
// C 语言 - 结构化编程典范
#include <stdio.h>

int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}

int main() {
    int arr[] = {1, 2, 3, 4, 5};
    int sum = 0;

    for (int i = 0; i < 5; i++) {
        sum += arr[i];
    }

    printf("Sum: %d, Factorial(5): %d\n", sum, factorial(5));
    return 0;
}
```

同期重要语言：Simula 67（面向对象先驱）、Smalltalk（纯 OO 语言）、Prolog（逻辑式编程）。

## 三、面向对象编程浪潮

1980年代，面向对象编程（OOP）成为主流范式。

```cpp
// C++ - 面向对象 + 泛型编程
#include <iostream>
#include <vector>
#include <algorithm>

template <typename T>
class Container {
private:
    std::vector<T> items;

public:
    void add(const T& item) { items.push_back(item); }

    template <typename Predicate>
    std::vector<T> filter(Predicate pred) {
        std::vector<T> result;
        std::copy_if(items.begin(), items.end(),
                     std::back_inserter(result), pred);
        return result;
    }
};
```

## 四、脚本语言的崛起

1990年代，脚本语言以开发效率高、语法简洁迅速流行。

```python
# Python - 优雅的脚本语言
def fibonacci_generator():
    a, b = 0, 1
    while True:
        yield a
        a, b = b, a + b

fib = fibonacci_generator()
first_10 = [next(fib) for _ in range(10)]
print(f"First 10 Fibonacci: {first_10}")

# 列表推导式与高阶函数
numbers = range(20)
even_squares = [x**2 for x in numbers if x % 2 == 0]
print(f"Even squares: {even_squares}")
```

## 五、Java 与.NET 平台

James Gosling 领导的 Java 项目于1995年发布，口号是"Write Once, Run Anywhere"。

```java
// Java - 企业级应用标准
public class StreamExample {
    public static void main(String[] args) {
        List<String> names = Arrays.asList("Alice", "Bob", "Charlie", "David");

        List<String> result = names.stream()
            .filter(name -> name.length() > 3)
            .map(String::toUpperCase)
            .sorted()
            .collect(Collectors.toList());

        System.out.println(result);
    }

    // 泛型方法
    public static <T> T getFirst(List<T> list) {
        return list.isEmpty() ? null : list.get(0);
    }
}
```

C#于2000年由 Anders Hejlsberg 设计，与.NET 框架同时发布。

## 六、新时代语言生态

2000年后，计算机硬件的发展和新的应用需求催生了新一代语言。

```rust
// Rust - 内存安全且高性能
fn quicksort<T: Ord>(arr: &mut [T]) {
    if arr.len() <= 1 {
        return;
    }

    let pivot = partition(arr);
    quicksort(&mut arr[..pivot]);
    quicksort(&mut arr[pivot + 1..]);
}

fn partition<T: Ord>(arr: &mut [T]) -> usize {
    let len = arr.len();
    let pivot_idx = len / 2;
    arr.swap(pivot_idx, len - 1);

    let mut i = 0;
    for j in 0..len - 1 {
        if arr[j] <= arr[len - 1] {
            arr.swap(i, j);
            i += 1;
        }
    }
    arr.swap(i, len - 1);
    i
}
```

```go
// Go - 简洁的并发编程
func mergeSort(arr []int) []int {
    if len(arr) <= 1 {
        return arr
    }

    mid := len(arr) / 2
    left := make(chan []int)
    right := make(chan []int)

    go func() {
        left <- mergeSort(arr[:mid])
    }()
    go func() {
        right <- mergeSort(arr[mid:])
    }()

    return merge(<-left, <-right)
}
```

## 七、编程范式演进

编程范式的时间演化：

```
1950s:  命令式编程（机器码、汇编）
1960s:  结构化编程（ALGOL, Pascal）
1970s:  逻辑式编程（Prolog）、函数式编程（ML）
1980s:  面向对象编程（C++, Smalltalk）
1990s:  组件化编程（COM, JavaBeans）
2000s:  泛型编程、元编程
2010s:  函数式复兴、响应式编程
2020s:  并发编程、AI 辅助编程
```

```python
import numpy as np
import torch
import torch.nn as nn

class NeuralNetwork(nn.Module):
    def __init__(self):
        super().__init__()
        self.layers = nn.Sequential(
            nn.Linear(784, 256),
            nn.ReLU(),
            nn.Dropout(0.2),
            nn.Linear(256, 128),
            nn.ReLU(),
            nn.Linear(128, 10),
            nn.Softmax(dim=1)
        )

    def forward(self, x):
        return self.layers(x)
```
