# 正则表达式基础

## 字面字符

```regex
hello       # 匹配 "hello" 字符串
123         # 匹配 "123"
Hello World # 匹配 "Hello World"（空格也匹配）
a           # 匹配单个字符 "a"
```

## 元字符

```regex
.       # 匹配任意单个字符（除换行符）
^       # 匹配字符串开头
$       # 匹配字符串结尾
-       # 前一个字符出现 0 次或多次
-       # 前一个字符出现 1 次或多次
?       # 前一个字符出现 0 次或 1 次
{ }     # 量词，指定出现次数
[ ]     # 字符类，匹配括号内任意字符
\       # 转义字符
|       # 或运算
( )     # 分组和捕获
```

## 字符类

```regex
[abc]       # 匹配 a、b 或 c
[^abc]      # 匹配除 a、b、c 之外的任意字符
[a-z]       # 匹配 a 到 z 之间的任意小写字母
[A-Z]       # 匹配任意大写字母
[0-9]       # 匹配任意数字
[a-zA-Z0-9] # 匹配任意字母或数字
[0-9a-fA-F] # 匹配十六进制字符

# 预定义字符类
\d          # 匹配任意数字，等价于 [0-9]
\w          # 匹配单词字符（字母、数字、下划线），等价于 [a-zA-Z0-9_]
\s          # 匹配空白字符（空格、制表符、换行等）
\D          # 匹配非数字，等价于 [^0-9]
\W          # 匹配非单词字符，等价于 [^a-zA-Z0-9_]
\S          # 匹配非空白字符
```

### 字符类示例

```regex
[aeiou]             # 匹配任意元音字母
[^0-9]              # 匹配非数字字符
[\s\S]              # 匹配任意字符（包括换行）
[.\n]               # 匹配点号或换行符
[a-z&&[^aeiou]]     # 匹配辅音字母（某些实现支持 &&）

# 实际应用
\d{3}-\d{3}-\d{4}   # 匹配美国电话号码格式: 123-456-7890
[\w.]+@[\w.]+\.\w+  # 简单邮箱匹配
```

## 量词

```regex
a*          # a 出现 0 次或多次
a+          # a 出现 1 次或多次
a?          # a 出现 0 次或 1 次
a{3}        # a 恰好出现 3 次
a{3,}       # a 出现至少 3 次
a{3,5}      # a 出现 3 到 5 次

# 贪婪与懒惰
a+          # 贪婪匹配（尽可能多）
a+?         # 懒惰匹配（尽可能少）

# 示例
\d{4}               # 匹配 4 位数字
\d{2,4}             # 匹配 2 到 4 位数字
https?://           # 匹配 http:// 或 https://
".*?"               # 匹配最短的双引号字符串
```

## 锚点

```regex
^hello      # 匹配以 "hello" 开头的字符串
world$      # 匹配以 "world" 结尾的字符串
^hello$     # 精确匹配 "hello"
\bword\b    # 匹配完整的单词 "word"（单词边界）
\Bword\B    # 匹配非单词边界中的 "word"

# 锚点示例
^\d{3}$     # 匹配恰好 3 位数字的字符串
^\s*$       # 匹配空行或空白行
\b\w{3}\b   # 匹配 3 个字母的单词
\b\w+\b     # 匹配所有单词

# 多行模式
# 使用 m 标志后：
^       # 匹配每行的开头
$       # 匹配每行的结尾
```

## 分组与捕获

```regex
# 捕获分组
(abc)       # 捕获组，按左括号编号 \1, \2, ...
(\d{3})-(\d{4})  # 两个捕获组

# 反向引用
(\w)\1      # 匹配重复字符，如 "aa", "bb"
(\d{3})-\1  # 匹配 "123-123" 但不匹配 "123-456"
<(\w+)>.*?</\1> # 匹配 HTML 标签，如 <div>...</div>

# 非捕获分组
(?:abc)     # 仅分组，不捕获

# 命名捕获（部分引擎支持）
(?<name>\w+)    # Python 语法
(?P<name>\w+)   # Python 语法
(?'name'\w+)    # .NET 语法

# 分组示例
(https?)://([^/]+)(/[^\s]*)    # URL 匹配：协议、域名、路径
(\d{4})-(\d{2})-(\d{2})        # 日期匹配，组 1=年，组 2=月，组 3=日
(?:Mr|Ms|Mrs)\.?\s+(\w+)       # 称谓（不捕获）后跟名字（捕获）
```

## 转义

```regex
\.          # 匹配字面点号
\*          # 匹配字面星号
\+          # 匹配字面加号
\\          # 匹配反斜杠
\(          # 匹配左括号
\)          # 匹配右括号
\[          # 匹配左方括号
\]          # 匹配右方括号
\{          # 匹配左花括号
\}          # 匹配右花括号
\|          # 匹配竖线
\^          # 匹配脱字符
\$          # 匹配美元符号
\?          # 匹配问号

# 转义示例
\.txt$      # 匹配以 ".txt" 结尾的文件名
C:\\Users\\  # 匹配 Windows 路径 "C:\Users\"
\(\d{3}\)    # 匹配 "(123)"
```

## 综合示例

```regex
# 邮箱（简化版）
[\w.-]+@[\w.-]+\.\w+

# URL
https?://[\w./?=&%-]+

# IP 地址（IPv4）
\b(?:\d{1,3}\.){3}\d{1,3}\b

# 日期：YYYY-MM-DD
\d{4}-\d{2}-\d{2}

# 时间：HH:MM:SS
\b([01]\d|2[0-3]):[0-5]\d:[0-5]\d\b

# HTML 标签
<[^>]+>

# 中文汉字
[\u4e00-\u9fa5]

# 密码强度（至少 8 位，含大小写字母和数字）
(?=.*[a-z])(?=.*[A-Z])(?=.*\d).{8,}

# 文件名提取
[^\\/]+$

# 数字（整数和小数）
-?\d+(?:\.\d+)?

# 十六进制颜色
#?([a-fA-F0-9]{3}|[a-fA-F0-9]{6})
```
