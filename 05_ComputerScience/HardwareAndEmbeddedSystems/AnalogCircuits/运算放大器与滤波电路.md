# 运算放大器与滤波电路

## 一、运算放大器基础

运算放大器（Op-Amp）是模拟电路的核心器件，理想运放具有以下特性：

| 参数 | 理想值 | 实际典型值 |
|------|--------|-----------|
| 开环增益 A_ol | 无穷大 | 10^5 - 10^7 |
| 输入阻抗 R_in | 无穷大 | 1M - 10^12 |
| 输出阻抗 R_out | 0 | 10 - 100 |
| 带宽 BW | 无穷大 | 1MHz - 1GHz |
| 共模抑制比 CMRR | 无穷大 | 80 - 120dB |
| 转换速率 SR | 无穷大 | 0.5 - 5000 V/us |

理想运放的两个关键推论：

$$
V_+ \approx V_- \quad \text{(虚短)}
$$
$$
I_+ = I_- = 0 \quad \text{(虚断)}
$$

## 二、基本运放电路

反相放大器：

$$
V_{out} = -\frac{R_f}{R_{in}} V_{in}
$$

```python
def inverting_amplifier(rf, rin, vin):
    """计算反相放大器输出"""
    return -(rf / rin) * vin
```

同相放大器：

$$
V_{out} = \left(1 + \frac{R_f}{R_1}\right) V_{in}
$$

电压跟随器（缓冲器）：

$$
V_{out} = V_{in}, \quad A_v = 1
$$

差分放大器：

$$
V_{out} = \frac{R_f}{R_{in}} (V_2 - V_1)
$$

仪表放大器常用于精密测量：

$$
V_{out} = \left(1 + \frac{2R_1}{R_g}\right) \frac{R_3}{R_2} (V_2 - V_1)
$$

## 三、有源滤波器设计

滤波器分类：

| 类型 | 通过频率 | 阻带频率 | 应用 |
|------|----------|----------|------|
| 低通 | f < f_c | f > f_c | 音频、去噪 |
| 高通 | f > f_c | f < f_c | 隔直、AC耦合 |
| 带通 | f_c1 < f < f_c2 | 其余 | 选频通信 |
| 带阻 | f_c1 < f < f_c2 之外 | f_c1 < f < f_c2 | 陷波滤波 |
| 全通 | 全部 | 无 | 相位补偿 |

一阶低通有源滤波器：

$$
H(s) = \frac{1}{1 + sRC}
$$

$$
f_c = \frac{1}{2\pi RC}
$$

二阶Sallen-Key低通滤波器：

$$
H(s) = \frac{A_v}{1 + sRC(3 - A_v) + (sRC)^2}
$$

其中 $A_v = 1 + \frac{R_f}{R_1}$。

```python
import numpy as np

def sallen_key_lowpass(r, c, gain, f):
    """计算Sallen-Key低通滤波器的频率响应"""
    w = 2 * np.pi * f
    rc = r * c
    denom = 1 + 1j * w * rc * (3 - gain) + (1j * w * rc)**2
    return gain / denom
```

## 四、滤波器阶数与响应类型

不同阶数的衰减特性：

| 阶数 | 衰减斜率 | 元件数 | 复杂度 |
|------|----------|--------|--------|
| 一阶 | 20dB/dec | 1R + 1C | 最低 |
| 二阶 | 40dB/dec | 2R + 2C | 低 |
| 四阶 | 80dB/dec | 4R + 4C | 中 |
| 八阶 | 160dB/dec | 8R + 8C | 高 |

常用滤波器响应类型：

| 响应类型 | 通带纹波 | 滚降特性 | 群延迟 |
|----------|----------|----------|--------|
| Butterworth | 无纹波 | 平坦滚降 | 较好 |
| Chebyshev I | 通带有纹波 | 陡峭滚降 | 较差 |
| Chebyshev II | 阻带有纹波 | 陡峭滚降 | 较差 |
| Elliptic | 通带+阻带有纹波 | 最陡滚降 | 最差 |
| Bessel | 无纹波 | 最平缓 | 最好 |

$$

|H_{Butterworth}(j\omega)| = \frac{1}{\sqrt{1 + (\frac{\omega}{\omega_c})^{2n}}}

$$

$$

|H_{Chebyshev}(j\omega)| = \frac{1}{\sqrt{1 + \epsilon^2 T_n^2(\frac{\omega}{\omega_c})}}

$$

其中 $T_n$ 是n阶切比雪夫多项式，$\epsilon$ 决定通带纹波。

## 五、实际运放的非理想特性

失调电压（Input Offset Voltage）：

$$
V_{out} = A_{ol} \cdot V_{os}
$$

温度漂移：典型值 $1-10 \mu V/^{\circ}C$

偏置电流（Input Bias Current）：

$$
I_B = \frac{I_+ + I_-}{2}
$$

## 六、滤波器设计实例

设计一个通带频率100Hz、阻带频率1kHz的二阶低通滤波器：

```python
def design_butterworth_lowpass(fc, order):
    """设计Butterworth低通滤波器"""
    import math
    R = 10000  # 10k 标准电阻
    C = 1 / (2 * math.pi * fc * R)
    
    print(f"截止频率: {fc} Hz")
    print(f"电阻R: {R/1000}k")
    print(f"电容C: {C*1e6:.2f}uF")
    
    # 对于二阶滤波器
    stages = order // 2
    for i in range(stages):
        Q = 1 / (2 * math.sin(math.pi * (2*i + 1) / (2 * order)))
        print(f"第{i+1}级: Q = {Q:.3f}")
    
design_butterworth_lowpass(100, 2)
```

## 七、噪声分析与抑制

运放噪声模型：

$$
e_{n, total} = \sqrt{e_n^2 + (i_n R)^2 + 4kTR}
$$

热噪声（Johnson-Nyquist）：

$$
v_n = \sqrt{4k_B T R \Delta f}
$$

其中 $k_B = 1.38 \times 10^{-23} J/K$，T为绝对温度。

信噪比计算：

$$
SNR_{dB} = 20 \log_{10} \frac{V_{signal}}{V_{noise}}
$$

降低噪声的策略：

```
1. 选择低噪声运放（如OPA2277）
2. 减小电阻值以降低热噪声
3. 限制带宽，使用滤波器
4. 屏蔽与接地技术
5. 差分信号传输
6. 电源去耦电容
```

## 八、滤波器仿真与测试

使用Python进行滤波器仿真：

```python
import numpy as np
import matplotlib.pyplot as plt

def butterworth_filter(n, fc, fs, order=4):
    """模拟Butterworth滤波"""
    w = 2 * np.pi * fc / fs
    # 简化模型
    H = 1 / np.sqrt(1 + (w / (2 * np.pi * fc))**(2 * order))
    return H

frequencies = np.logspace(1, 4, 1000)
response = [butterworth_filter(4, 100, f) for f in frequencies]

# 计算-3dB点
fc_idx = np.argmin(np.abs(np.array(response) - 0.707))
print(f"-3dB频率: {frequencies[fc_idx]:.1f} Hz")
```

示波器和频谱分析仪是滤波器测试的核心工具。网络分析仪可精确测量幅频和相频特性。
