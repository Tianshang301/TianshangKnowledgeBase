# 硬件学习路线图

## 入门阶段

### 1. 数字电路基础
- **内容**：
  - 二进制、布尔代数、逻辑门（AND/OR/NOT/NAND/NOR/XOR）
  - 组合逻辑：加法器、多路选择器、译码器
  - 时序逻辑：触发器（D/JK）、计数器、寄存器
- **教材推荐**：《数字电子技术基础》（阎石）
- **仿真工具**：Logisim / EDA Playground

### 2. 51 单片机入门
- **学习点**：
  - GPIO（发光二极管、按键）
  - 定时器/计数器
  - 中断系统
  - UART 串口通信
  - I2C / SPI 协议
- **开发板**：STC89C52 最小系统板
- **工具链**：Keil C51 + STC-ISP
- **项目**：流水灯 → 数码管显示 → 温度传感器读取

### 3. 模拟电路基础
- **内容**：
  - 欧姆定律、基尔霍夫定律
  - 二极管、三极管、场效应管
  - 运算放大器基础
  - RC/RL 电路
- **教材推荐**：《模拟电子技术基础》（童诗白）

## 中级阶段

### 1. STM32 (ARM Cortex-M)
- **学习点**：
  - ARM Cortex-M 架构
  - 时钟系统（PLL）
  - GPIO、中断（NVIC）
  - 定时器（PWM、输入捕获）
  - ADC/DAC
  - DMA
  - FreeRTOS 实时操作系统基础
- **开发板**：STM32F103C8T6（Blue Pill）或 STM32F407
- **工具链**：STM32CubeIDE / Keil MDK
- **资源**：
  - 野火/正点原子教程
  - STM32CubeMX 图形化配置

### 2. PCB 设计
- **学习点**：
  - 原理图设计
  - 封装设计
  - 布局布线规则
  - 信号完整性基础
  - 电源/地平面处理
- **工具**：KiCad（免费开源）、Altium Designer（商业）
- **项目**：设计 STM32 最小系统板

### 3. 高级接口与协议
- **USB**：HID、CDC、Mass Storage
- **CAN**：汽车电子常用
- **Ethernet**：LWIP 协议栈
- **蓝牙/WiFi**：ESP32、HC-05

## 高级阶段

### 1. FPGA 与数字设计
- **内容**：
  - Verilog / VHDL 硬件描述语言
  - 组合逻辑与时序逻辑的 HDL 实现
  - 状态机设计
  - 仿真与验证
- **开发板**：Xilinx Spartan-6 / Artix-7 或 Intel Cyclone IV
- **工具**：Vivado / Quartus
- **资源**：
  - 《数字设计与计算机体系结构》 - Harris & Harris
  - FPGA4Fun 网站

### 2. 嵌入式 Linux
- **内容**：
  - Bootloader (U-Boot)
  - Linux 内核裁剪与编译
  - 根文件系统 (Buildroot / Yocto)
  - 设备树 (Device Tree)
  - 驱动开发（字符设备、GPIO、I2C、SPI）
- **硬件**：Raspberry Pi / BeagleBone / i.MX6
- **项目**：制作嵌入式 Linux 开发板

### 3. 高级主题
- **RTOS 深入**：FreeRTOS 源码分析
- **DSP**：数字信号处理算法实现
- **计算机体系结构**：流水线、缓存、虚拟内存
- **SoC 设计**：系统级芯片设计流程

## 项目推荐

| 阶段 | 项目名称 | 所用知识 |
|------|---------|---------|
| 入门 | 智能台灯 | 51单片机、PWM调光、光敏传感器 |
| 入门 | 电子时钟 | DS1302、数码管、按键 |
| 中级 | 四轴无人机飞控 | STM32、MPU6050、PID控制 |
| 中级 | 智能小车 | STM32、电机驱动、超声波避障 |
| 高级 | 简易逻辑分析仪 | FPGA、USB、上位机 |
| 高级 | 网络摄像头 | Linux、摄像头驱动、流媒体 |

## 工具与仪器

- **万用表**：必备基础工具
- **示波器**：调试数字信号（推荐 Siglent/Rigol 入门款）
- **逻辑分析仪**：便宜好用（Saleae 克隆版）
- **焊接工具**：恒温烙铁 + 热风枪 + 焊锡丝
- **电源**：可调直流电源

## 学习建议

1. **从做中学**：只看书不可能学会硬件
2. **先抄后改**：先复现现有项目，再自己改进
3. **调试是核心能力**：学会看波形、读手册、分析问题
4. **读芯片手册**：数据手册 (Datasheet) 是最权威的资料
5. **示波器是第三只手**：越早学会用越好
6. **注意安全**：涉及市电时务必小心
