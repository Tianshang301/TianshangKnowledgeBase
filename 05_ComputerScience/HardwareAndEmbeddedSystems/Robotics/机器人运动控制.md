# 机器人运动控制

## 一、机器人运动学基础

机器人运动学研究物体运动的几何关系，不考虑产生运动的力。

正运动学：给定关节角度求末端执行器位姿。

$$
T = T_1(q_1) \cdot T_2(q_2) \cdot ... \cdot T_n(q_n)
$$

逆运动学：给定末端执行器位姿求关节角度。

二连杆机械臂正运动学：

$$
x = L_1\cos\theta_1 + L_2\cos(\theta_1 + \theta_2)
$$

$$
y = L_1\sin\theta_1 + L_2\sin(\theta_1 + \theta_2)
$$

逆运动学解析解：

$$
\cos\theta_2 = \frac{x^2 + y^2 - L_1^2 - L_2^2}{2L_1L_2}
$$

$$
\theta_1 = \text{atan2}(y, x) - \text{atan2}(L_2\sin\theta_2, L_1 + L_2\cos\theta_2)
$$

## 二、DH 参数法

Denavit-Hartenberg 参数法用四个参数描述相邻连杆的坐标变换：

| 参数 | 符号 | 含义 |
|------|------|------|
| 连杆长度 | a_i | z_{i-1}沿 x_i 方向到 z_i 的距离 |
| 连杆扭转角 | alpha_i | z_{i-1}绕 x_i 旋转到 z_i 的角度 |
| 关节距离 | d_i | x_{i-1}沿 z_{i-1}到 x_i 的距离 |
| 关节角 | theta_i | x_{i-1}绕 z_{i-1}旋转到 x_i 的角度 |

齐次变换矩阵：

$$
T_i^{i-1} =
\begin{bmatrix}
\cos\theta_i & -\sin\theta_i\cos\alpha_i & \sin\theta_i\sin\alpha_i & a_i\cos\theta_i \\
\sin\theta_i & \cos\theta_i\cos\alpha_i & -\cos\theta_i\sin\alpha_i & a_i\sin\theta_i \\
0 & \sin\alpha_i & \cos\alpha_i & d_i \\
0 & 0 & 0 & 1
\end{bmatrix}
$$

## 三、移动机器人运动模型

差速驱动模型：

```
            v_r
      ______________
     |   [O]  [O]   |  v_l
     |     [O]      |  (万向轮)
      --------------
           ^
           v
```

运动学方程：

$$
\begin{bmatrix}
\dot{x} \\
\dot{y} \\
\dot{\theta}
\end{bmatrix}
=
\begin{bmatrix}
\frac{v_r + v_l}{2} \cos\theta \\
\frac{v_r + v_l}{2} \sin\theta \\
\frac{v_r - v_l}{W}
\end{bmatrix}
$$

其中 W 为轮距，v_r 和 v_l 为左右轮速度。

```python
import math

class DifferentialDrive:
    def __init__(self, wheel_base=0.3):
        self.wheel_base = wheel_base
        self.x = 0.0
        self.y = 0.0
        self.theta = 0.0

    def update(self, v_left, v_right, dt):
        v = (v_left + v_right) / 2.0
        omega = (v_right - v_left) / self.wheel_base

        self.x += v * math.cos(self.theta) * dt
        self.y += v * math.sin(self.theta) * dt
        self.theta += omega * dt
        self.theta = math.atan2(math.sin(self.theta), math.cos(self.theta))

        return self.x, self.y, self.theta

robot = DifferentialDrive(wheel_base=0.2)
for t in range(100):
    pose = robot.update(0.5, 0.52, 0.1)
    print(f"t={t*0.1:.1f}s: x={pose[0]:.3f}, y={pose[1]:.3f}, theta={pose[2]:.3f}")
```

## 四、PID 控制算法

PID 控制器是工业机器人最广泛使用的控制算法。

$$
u(t) = K_p e(t) + K_i \int_0^t e(\tau)d\tau + K_d \frac{de(t)}{dt}
$$

离散化形式：

$$
u_k = K_p e_k + K_i \sum_{i=0}^k e_i \Delta t + K_d \frac{e_k - e_{k-1}}{\Delta t}
$$

```python
class PIDController:
    def __init__(self, kp, ki, kd, dt=0.01, output_limits=None):
        self.kp = kp
        self.ki = ki
        self.kd = kd
        self.dt = dt
        self.output_limits = output_limits

        self.integral = 0.0
        self.prev_error = 0.0

    def compute(self, setpoint, measurement):
        error = setpoint - measurement
        self.integral += error * self.dt
        derivative = (error - self.prev_error) / self.dt
        self.prev_error = error

        output = self.kp * error + self.ki * self.integral + self.kd * derivative

        if self.output_limits:
            output = max(self.output_limits[0], min(self.output_limits[1], output))

        return output

    def reset(self):
        self.integral = 0.0
        self.prev_error = 0.0

# 位置跟踪示例
pid = PIDController(kp=2.0, ki=0.5, kd=0.1, dt=0.01, output_limits=(-1.0, 1.0))
position = 0.0
target = 1.0

for _ in range(500):
    control = pid.compute(target, position)
    position += control * 0.01  # 简单系统模型
```

## 五、轨迹规划

梯形速度规划（Trapezoidal Velocity Profile）：

```python
def generate_trapezoidal_trajectory(start, end, max_vel, accel, dt=0.01):
    distance = end - start
    accel_time = max_vel / accel
    accel_dist = 0.5 * accel * accel_time ** 2

    if 2 * accel_dist > distance:
        # 三角速度曲线
        accel_time = math.sqrt(distance / accel)
        max_vel = accel * accel_time
        constant_time = 0
    else:
        constant_dist = distance - 2 * accel_dist
        constant_time = constant_dist / max_vel

    total_time = 2 * accel_time + constant_time
    t = 0.0
    trajectory = []

    while t <= total_time:
        if t < accel_time:
            pos = start + 0.5 * accel * t ** 2
            vel = accel * t
        elif t < accel_time + constant_time:
            pos = start + accel_dist + max_vel * (t - accel_time)
            vel = max_vel
        else:
            decel_t = t - accel_time - constant_time
            pos = end - 0.5 * accel * (accel_time - decel_t) ** 2
            vel = max_vel - accel * decel_t
        trajectory.append((t, pos, vel))
        t += dt

    return trajectory
```

$S$ 形速度曲线在起止处加速度连续变化，减小冲击：

$$
j(t) = \frac{da}{dt}, \quad a(t) = \int j(t) dt
$$

## 六、ROS 机器人操作系统

ROS（Robot Operating System）是机器人开发的标准化框架。

核心概念：

| 概念 | 说明 | 类比 |
|------|------|------|
| Node | 可执行程序 | 进程 |
| Topic | 异步通信通道 | 发布/订阅 |
| Service | 同步 RPC 调用 | 请求/响应 |
| Action | 带反馈的异步任务 | 长时间任务 |
| Message | 数据结构定义 | protobuf |
| Bag | 数据记录与回放 | rosbag |

发布者示例：

```python
#!/usr/bin/env python3
import rospy
from geometry_msgs.msg import Twist

def move_robot():
    rospy.init_node('robot_controller', anonymous=True)
    pub = rospy.Publisher('/cmd_vel', Twist, queue_size=10)
    rate = rospy.Rate(10)  # 10Hz

    twist = Twist()
    twist.linear.x = 0.2
    twist.angular.z = 0.1

    while not rospy.is_shutdown():
        pub.publish(twist)
        rate.sleep()

if __name__ == '__main__':
    try:
        move_robot()
    except rospy.ROSInterruptException:
        pass
```

```
传感器数据 --> 前端里程计 --> 后端优化 --> 地图构建
```

| 控制模式 | 精度 | 复杂度 | 适用场景 |
|----------|------|--------|----------|
| 开环控制 | 低 | 低 | 步进电机、简单动作 |

| 阻抗控制 | 中 | 高 | 人机交互、力控制 |
