# 量子算法

量子算法利用量子力学特性（叠加、纠缠、干涉）在特定问题上实现指数级或多项式级加速。本文件介绍最重要的量子算法及其应用。

## 1. 早期量子算法

### 1.1 Deutsch-Jozsa算法
Deutsch-Jozsa算法是第一个展示量子计算优势的算法，解决的问题是：给定一个函数 \(f: \{0,1\}^n \rightarrow \{0,1\}\)，判断它是常数函数（对所有输入输出相同）还是平衡函数（输出0和1的数量相等）。

**经典情况**：最坏情况需要 \(2^{n-1}+1\) 次查询。
**量子情况**：只需1次查询。

**算法步骤**：
1. 初始化n个量子比特为 \(|0\rangle^{\otimes n}\)
2. 应用Hadamard门创建叠加态
3. 应用Oracle函数 \(U_f\)（量子黑盒）
4. 再次应用Hadamard门
5. 测量，若全为0则为常数函数，否则为平衡函数

### 1.2 Bernstein-Vazirani算法
Bernstein-Vazirani算法是Deutsch-Jozsa的特例，用于找到隐藏的比特串 \(s\)，其中Oracle函数为 \(f(x) = s \cdot x \mod 2\)。

**经典情况**：需要n次查询（每次确定s的一个比特）。
**量子情况**：只需1次查询。

**应用**：密码学中的线性函数识别。

### 1.3 Simon算法
Simon算法解决周期查找问题：给定函数 \(f: \{0,1\}^n \rightarrow \{0,1\}^n\)，满足 \(f(x) = f(y)\) 当且仅当 \(x \oplus y = s\)（s是隐藏的周期）。

**经典情况**：需要 \(O(2^{n/2})\) 次查询。
**量子情况**：需要 \(O(n)\) 次查询。

Simon算法是Shor算法的前身，启发了量子傅里叶变换的使用。

## 2. 重要量子算法

### 2.1 Shor算法（大数分解）
Shor算法是量子计算最著名的应用，能在多项式时间内分解大整数，威胁RSA加密。

**问题**：给定合数N，找到其非平凡因子。
**复杂度**：经典最优算法（数域筛法）需要 \(O(\exp(c(\log N)^{1/3}(\log \log N)^{2/3}))\)，Shor算法需要 \(O((\log N)^3)\)。

**算法核心**：
1. 选择随机数a < N
2. 计算a模N的阶r（量子部分）
3. 若r为偶数且 \(a^{r/2} \not\equiv -1 \mod N\)，则计算 \(\gcd(a^{r/2} \pm 1, N)\)

**量子部分**（阶查找）：
1. 创建叠加态 \(\frac{1}{\sqrt{Q}}\sum_{x=0}^{Q-1}|x\rangle\)
2. 计算 \(f(x) = a^x \mod N\)
3. 应用量子傅里叶变换
4. 测量并提取周期

**对RSA的威胁**：RSA-2048需要约4000个逻辑量子比特，但考虑纠错需要数百万物理量子比特。

### 2.2 Grover算法（无序搜索）
Grover算法在无序数据库中搜索目标项，提供平方加速。

**问题**：在N个项的数据库中找到满足条件的项。
**复杂度**：经典需要 \(O(N)\) 次查询，Grover需要 \(O(\sqrt{N})\) 次。

**算法步骤**：
1. 初始化叠加态 \(|s\rangle = H^{\otimes n}|0\rangle^{\otimes n}\)
2. 重复约 \(\frac{\pi}{4}\sqrt{N}\) 次：
   - 应用Oracle（标记目标态）
   - 应用扩散算子（放大目标态振幅）
3. 测量得到目标态

**应用**：
- 数据库搜索
- 锚定问题（SAT求解）
- 优化问题

**局限性**：需要知道解的数量以确定迭代次数，否则可能错过最优解。

### 2.3 量子傅里叶变换（QFT）
QFT是许多量子算法的核心组件，是离散傅里叶变换的量子版本。

**定义**：将计算基态转换为频率基态：
\[
|j\rangle \rightarrow \frac{1}{\sqrt{N}}\sum_{k=0}^{N-1}e^{2\pi ijk/N}|k\rangle
\]

**实现**：使用Hadamard门和受控相位门，需要 \(O(n^2)\) 个门（n为量子比特数）。

**应用**：
- Shor算法中的阶查找
- 量子相位估计
- 隐藏子群问题

### 2.4 HHL算法（线性方程组求解）
HHL算法用于求解线性方程组 \(A\vec{x} = \vec{b}\)，在特定条件下实现指数加速。

**输入**：矩阵A（稀疏、条件数好）、向量b（量子态形式）。
**输出**：解向量x的量子态 \(|x\rangle\)。

**复杂度**：经典共轭梯度法需要 \(O(N\kappa \log(1/\epsilon))\)，HHL需要 \(O(\text{poly}(\log N, \kappa, 1/\epsilon))\)。

**步骤**：
1. 量子相位估计提取A的特征值
2. 受控旋转实现 \(1/\lambda\) 操作
3. 逆相位估计清理辅助量子比特

**限制**：
- 需要高效的量子态制备
- 解向量读出困难（只能采样）
- 条件数κ影响算法效率

## 3. 变分量子算法

变分量子算法是NISQ时代的主要算法范式，结合经典优化和量子电路。

### 3.1 变分量子本征求解器（VQE）
VQE用于寻找分子基态能量，是量子化学的核心算法。

**原理**：
1. 准备参数化量子电路 \(|\psi(\theta)\rangle\)
2. 测量期望值 \(\langle\psi(\theta)|H|\psi(\theta)\rangle\)
3. 经典优化器更新参数θ
4. 迭代直至收敛

**优势**：
- 电路深度浅，适合NISQ设备
- 可处理强关联体系
- 天然适应噪声环境

**应用**：
- 分子能量计算
- 化学反应路径
- 材料性质预测

### 3.2 量子近似优化算法（QAOA）
QAOA用于组合优化问题，是Grover算法在优化问题上的推广。

**问题**：寻找图的最大割、旅行商问题等的近似解。

**结构**：
1. 初始化叠加态
2. 交替应用问题哈密顿量 \(U_C(\gamma)\) 和混合哈密顿量 \(U_B(\beta)\)
3. 测量得到近似解

**性能**：对于某些问题，QAOA在p层电路时性能随p增加而改善，但最优性能可能仍低于经典算法。

## 4. 量子模拟

量子模拟是量子计算最自然的应用，直接模拟量子系统。

### 4.1 哈密顿量模拟
模拟量子系统的时间演化： \(|\psi(t)\rangle = e^{-iHt}|\psi(0)\rangle\)。

**方法**：
- 乘积公式（Trotter分解）
- 线性组合酉操作（LCU）
- 量子信号处理

### 4.2 量子行走
量子行走是经典随机行走的量子类比，在图问题上提供二次加速。

**应用**：
- 图同构测试
- 元素唯一性问题
- 空间搜索

## 5. 量子算法复杂性

### 5.1 量子计算复杂性类
- **BQP**：量子多项式时间可解的问题
- **QMA**：量子版的NP完全问题
- **QCMA**：经典证明的QMA

### 5.2 量子优势
已知量子优势的问题：
- 大数分解（Shor）
- 无序搜索（Grover）
- 模拟量子系统
- 某些代数问题（隐藏子群）

### 5.3 无量子优势的问题
- 排序
- 图连通性
- 大多数NP完全问题（除非BQP=NP）

## 6. 总结

量子算法展示了量子计算的潜力，从早期的理论演示（Deutsch-Jozsa）到实用的密码学威胁（Shor）和优化加速（Grover）。变分量子算法（VQE、QAOA）为NISQ时代提供了实用路径。随着量子硬件的进步，更多量子算法有望在实际问题中展现优势。
