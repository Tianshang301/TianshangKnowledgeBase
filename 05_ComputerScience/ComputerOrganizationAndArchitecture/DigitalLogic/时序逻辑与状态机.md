# 时序逻辑与状态机

## 一、时序逻辑基础

与组合逻辑不同，时序逻辑的输出不仅取决于当前输入，还取决于历史状态。时序逻辑电路包含存储元件(触发器和锁存器)来保持内部状态。状态是时序电路的核心概念，表示电路在特定时刻必须记住的所有信息。

时序逻辑分为同步时序电路(所有存储元件由统一时钟驱动)和异步时序电路(无统一时钟)。同步电路是数字系统设计的主流范式，具有可预测的行为和成熟的 EDA 工具支持。

## 二、锁存器与触发器

### 2.1 基本存储元件

**SR 锁存器**是最基本的存储单元，由两个交叉耦合的或非门或与非门构成：

```
S-R 锁存器特性表：
S R | Q  Q'
0 0 | Q  Q'  保持
0 1 | 0   1  复位
1 0 | 1   0  置位
1 1 | 1   1  禁止态
```

**D 锁存器**在使能信号有效时将输入传递到输出(透明模式)，使能无效时保持。

### 2.2 边沿触发触发器

**D 触发器**只在时钟边沿采样输入，消除了锁存器的透明窗口问题：

$$
Q_{\text{new}} = \begin{cases}
D & \text{clk 上升沿} \\
Q_{\text{old}} & \text{其他}
\end{cases}
$$

其他触发器类型包括：

| 类型 | 特性 | 应用 |
|------|------|------|
| T 触发器 | 每次时钟沿翻转 | 计数器除法 |
| JK 触发器 | 全功能，$J=K=1$翻转 | 通用 |
| D 触发器 | 延迟跟随 | 寄存器、流水线 |
| SR 触发器 | 置位/复位 | 控制电路 |

### 2.3 时序参数

触发器的关键时序参数：

- **建立时间($t_{\text{su}}$)**：数据必须在时钟沿前保持稳定的最小时间
- **保持时间($t_h$)**：数据必须在时钟沿后保持稳定的最小时间
- **时钟到输出延迟($t_{cq}$)**：时钟沿到输出稳定的时间
- **最小时钟周期**：$T_{\text{min}} = t_{cq} + t_{\text{logic,max}} + t_{\text{su}}$

时序违规导致亚稳态——触发器的输出在一段不确定时间内无法稳定到0或1。

## 三、有限状态机(FSM)

### 3.1 FSM 模型

有限状态机由六元组 $(\mathcal{S}, \mathcal{I}, \mathcal{O}, \delta, \lambda, s_0)$ 定义：

- $\mathcal{S}$：有限状态集合
- $\mathcal{I}$：输入符号集合
- $\mathcal{O}$：输出符号集合
- $\delta: \mathcal{S} \times \mathcal{I} \rightarrow \mathcal{S}$：状态转移函数
- $\lambda$：输出函数
- $s_0$：初始状态

### 3.2 Moore 机与 Mealy 机

| 特性 | Moore 机 | Mealy 机 |
|------|---------|---------|
| 输出依赖 | 仅当前状态 | 当前状态+当前输入 |
| 输出变化 | 时钟沿后(慢一拍) | 输入变化立即响应 |
| 状态数 | 通常更多 | 通常更少 |
| 响应速度 | 慢(需等时钟) | 快(异步输出) |
| 安全性 | 输出稳定 | 可能产生毛刺 |

**Moore 机模型**：$\lambda: \mathcal{S} \rightarrow \mathcal{O}$，输出 $y(t) = \lambda(s(t))$

**Mealy 机模型**：$\lambda: \mathcal{S} \times \mathcal{I} \rightarrow \mathcal{O}$，输出 $y(t) = \lambda(s(t), x(t))$

## 四、状态机设计方法

### 4.1 Verilog 实现

```verilog
// Moore 状态机：序列检测器"101"
module seq_detector_moore (
    input clk, rst_n, din,
    output reg dout
);

    typedef enum reg [2:0] {S0, S1, S2, S3} state_t;
    state_t state, next_state;

    // 状态寄存器
    always_ff @(posedge clk, negedge rst_n) begin
        if (!rst_n) state <= S0;
        else state <= next_state;
    end

    // 次态逻辑
    always_comb begin
        case (state)
            S0: next_state = din ? S1 : S0;
            S1: next_state = din ? S1 : S2;
            S2: next_state = din ? S3 : S0;
            S3: next_state = din ? S1 : S2;
        endcase
    end

    // 输出逻辑(仅依赖当前状态)
    always_ff @(posedge clk, negedge rst_n) begin
        if (!rst_n) dout <= 0;
        else dout <= (state == S3);
    end
endmodule
```

```verilog
// Mealy 状态机：序列检测器"101"
module seq_detector_mealy (
    input clk, rst_n, din,
    output reg dout
);

    typedef enum reg [1:0] {S0, S1, S2} state_t;
    state_t state, next_state;

    always_ff @(posedge clk, negedge rst_n) begin
        if (!rst_n) state <= S0;
        else state <= next_state;
    end

    always_comb begin
        case (state)
            S0: begin
                next_state = din ? S1 : S0;
                dout = 0;
            end
            S1: begin
                next_state = din ? S1 : S2;
                dout = 0;
            end
            S2: begin
                next_state = din ? S0 : S0;
                dout = din ? 1 : 0;  // 检测到101时输出1
            end
        endcase
    end
endmodule
```

### 4.2 状态编码选择

| 编码方式 | 触发器数 | 组合逻辑 | 功耗 | 适用场景 |
|----------|----------|----------|------|----------|
| 二进制 | $\log_2 N$ | 复杂 | 低 | 状态数多 |
| 独热码(one-hot) | $N$ | 简单 | 中 | FPGA 实现，状态少 |
| 格雷码 | $\log_2 N$ | 中 | 最低 | 相邻转移多 |

## 五、计数器与移位寄存器

### 5.1 计数器

计数器是最常见的时序电路。n 位二进制计数器从0计数到$2^n-1$：

- **同步计数器**：所有位在同一个时钟沿更新，速度快但进位逻辑复杂
- **异步计数器(行波计数器)**：每个触发器时钟来自前一级输出，结构简单但有累积延迟

**环形计数器**和**约翰逊计数器**用于产生独特的状态序列。

### 5.2 移位寄存器

移位寄存器在时钟作用下逐位移动数据，用于串行到并行转换(如 SPI 接口)、延迟线和伪随机数生成(线性反馈移位寄存器，LFSR)。

## 六、时序分析

静态时序分析(STA)验证设计是否满足时序约束。关键路径分析找到最大延迟路径，确保所有路径满足建立时间和保持时间要求。

**建立时间检查**：$T_{\text{clk}} \geq t_{cq,\text{FF1}} + t_{\text{logic,max}} + t_{\text{su,FF2}} + t_{\text{skew}}$

**保持时间检查**：$t_{cq,\text{FF1}} + t_{\text{logic,min}} - t_{\text{hold,FF2}} - t_{\text{skew}} \geq 0$

## 七、实践设计考虑

FPGA 中的时序电路应遵循同步设计原则：单一全局时钟、避免异步复位、使用同步使能信号控制数据流。跨时钟域数据传递使用双触发器同步器或异步 FIFO。
