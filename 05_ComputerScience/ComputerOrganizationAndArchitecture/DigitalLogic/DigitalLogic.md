# 数字逻辑 (Digital Logic)

## 1. 数制与转换 (Number Systems)

### 常用数制

| 进制 | 基数 | 数字符号 | 表示后缀 | 用途 |
|------|------|---------|---------|------|
| 二进制 | 2 | 0,1 | B 或 0b | 计算机内部 |
| 八进制 | 8 | 0~7 | O 或 0o | Unix 权限 |
| 十进制 | 10 | 0~9 | D 或无 | 日常 |
| 十六进制 | 16 | 0~9, A~F | H 或 0x | 内存地址、颜色 |

### 进制转换

```
二进制 → 十进制：
  1011.101₂ = 1×2³ + 0×2² + 1×2¹ + 1×2⁰ + 1×2⁻¹ + 0×2⁻² + 1×2⁻³
            = 8 + 0 + 2 + 1 + 0.5 + 0 + 0.125 = 11.625₁₀

十进制 → 二进制：
  13₁₀ = ?
  13 ÷ 2 = 6 余 1  (LSB)
   6 ÷ 2 = 3 余 0
   3 ÷ 2 = 1 余 1
   1 ÷ 2 = 0 余 1  (MSB)
  → 1101₂

二进制 → 十六进制（每 4 位一组）：
  1011 1100 1110₂ = 0xBCE

常用 2 的幂：
  2¹⁰ = 1024 (1K)    2²⁰ = 1,048,576 (1M)
  2³⁰ = 1,073,741,824 (1G)    2⁴⁰ = 1T
```

### 补码 (Two's Complement)

```
n 位补码表示范围：-2ⁿ⁻¹ ~ 2ⁿ⁻¹ - 1

8-bit 补码：
  0    = 0000 0000
  +127 = 0111 1111
  -1   = 1111 1111
  -128 = 1000 0000

补码取负：按位取反 + 1
  +5 = 0000 0101
  -5 = 1111 1011    (取反 1111 1010 → +1)
```

## 2. 布尔代数 (Boolean Algebra)

### 基本定律

| 定律 | 表达式 |
|------|--------|
| 交换律 | A + B = B + A, A · B = B · A |
| 结合律 | (A+B)+C = A+(B+C), (A·B)·C = A·(B·C) |
| 分配律 | A·(B+C) = A·B + A·C, A + B·C = (A+B)·(A+C) |
| 同一律 | A + 0 = A, A · 1 = A |
| 零一律 | A + 1 = 1, A · 0 = 0 |
| 互补律 | A + A' = 1, A · A' = 0 |
| 重叠律 | A + A = A, A · A = A |
| 吸收律 | A + A·B = A, A·(A+B) = A |
| 对偶律 | (A+B)' = A'·B', (A·B)' = A' + B' |

布尔表达式标准形式：

$$f(A,B,C) = \sum m_i \quad (\text{最小项之和})$$

$$f(A,B,C) = \prod M_i \quad (\text{最大项之积})$$

### 德摩根定理 (De Morgan's Theorem)

```
(A + B)' = A' · B'     (与非门等价)
(A · B)' = A' + B'     (或非门等价)

推广：
  (A₁ + A₂ + ... + Aₙ)' = A₁' · A₂' · ... · Aₙ'
  (A₁ · A₂ · ... · Aₙ)' = A₁' + A₂' + ... + Aₙ'
```

### 卡诺图 (Karnaugh Map)

```
2 变量卡诺图：
      B'   B
  A' ┌────┬────┐
     │ m0 │ m1 │
  A  ├────┼────┤
     │ m2 │ m3 │
     └────┴────┘

3 变量卡诺图：
      B'C'  B'C   BC   BC'
  A' ┌────┬────┬────┬────┐
     │ m0 │ m1 │ m3 │ m2 │
  A  ├────┼────┼────┼────┤
     │ m4 │ m5 │ m7 │ m6 │
     └────┴────┴────┴────┘

化简规则：
  相邻方格（上下左右）可合并
  2ⁿ 个相邻方格合并消去 n 个变量

卡诺图化简对比：

| 圈组大小 | 消去变量数 | 对应乘积项项数 | 示例 |
|---------|-----------|--------------|------|
| 1 个方格 | 0 | 3 | $A'BC'$ |
| 2 个方格 | 1 | 2 | $AB$ |
| 4 个方格 | 2 | 1 | $C$ |
| 8 个方格 | 3 | 1（常数） | 1 |

## 3. 逻辑门 (Logic Gates)

### 基本门电路

```
AND (与):    Y = A · B
OR (或):     Y = A + B
NOT (非):    Y = A'

NAND (与非): Y = (A · B)'   ← 通用门（可构成任意电路）
NOR (或非):  Y = (A + B)'   ← 通用门
XOR (异或):  Y = A ⊕ B = A'·B + A·B'
XNOR (同或): Y = A ⊙ B = A·B + A'·B'

  AND        OR        NAND       NOR        XOR        NOT
A───&───   A───≥1──  A───&───   A───≥1──  A───=1──   A───1───
B───┤      B───┤     B───┤○     B───┤○     B───┤         │
    Y         Y        Y         Y         Y         ○───Y
```

### 三态门 (Tri-state)

```
三态门输出：0, 1, Z (高阻态)

┌───┬───┬────┐
│EN │IN │OUT │
├───┼───┼────┤
│ 0 │ X │ Z  │  (高阻，断开)
│ 1 │ 0 │ 0  │
│ 1 │ 1 │ 1  │
└───┴───┴────┘

用途：总线共享（多个输出共用一条线，同一时间只有一个使能）
```

## 4. 组合逻辑电路 (Combinational Circuits)

### 加法器

```
半加器 (Half Adder)：
  S = A ⊕ B
  C = A · B

  A ─┬─=1── S
     │
     └─&─── C
  B ─┘

全加器 (Full Adder)：
  S = A ⊕ B ⊕ Cin
  Cout = A·B + A·Cin + B·Cin

  A ──┤
      │ HA1  ┌────┐
  B ──┤─────┤ HA2 │── S
  Cin ─┤    └────┘
       │
       │ OR ── Cout
       └──────┤
```

### 超前进位加法器 (Carry Lookahead)

```
CLA 原理：并行计算进位，减少传播延迟

生成信号: Gᵢ = Aᵢ · Bᵢ
传播信号: Pᵢ = Aᵢ ⊕ Bᵢ

进位公式：
  C₁ = G₀ + P₀·C₀
  C₂ = G₁ + P₁·G₀ + P₁·P₀·C₀
  C₃ = G₂ + P₂·G₁ + P₂·P₁·G₀ + P₂·P₁·P₀·C₀

级联 CLA 块可扩展到 16/32/64 位
```

### 多路选择器 (Multiplexer)

```
2-to-1 MUX：
  Y = S'·D₀ + S·D₁

  D₀ ──┤&
       │  OR── Y
  D₁ ──┤&
       │
  S ───┤ (S 控制选择)

4-to-1 MUX：
  Y = S₁'S₀'·D₀ + S₁'S₀·D₁ + S₁S₀'·D₂ + S₁S₀·D₃
```

### 译码器与编码器

```
3-to-8 译码器：
  ┌───┬───┬───┬────────────────────┐
  │ A₂│ A₁│ A₀│ Y₇Y₆Y₅Y₄Y₃Y₂Y₁Y₀  │
  ├───┼───┼───┼────────────────────┤
  │ 0 │ 0 │ 0 │ 00000001           │
  │ 0 │ 0 │ 1 │ 00000010           │
  │...│   │   │ ...                │
  │ 1 │ 1 │ 1 │ 10000000           │
  └───┴───┴───┴────────────────────┘

优先级编码器 (8-to-3)：
  输入优先级 I₇ > I₆ > ... > I₀
  输出对应输入最高优先级的 3-bit 编码
```

## 5. 时序逻辑电路 (Sequential Circuits)

### 锁存器与触发器

| 类型 | 时钟 | 触发方式 | 特性 |
|------|------|---------|------|
| SR 锁存器 | 无 | 电平 | 基本存储单元，R=S=1 非法 |
| D 锁存器 | 有 | 电平触发 | 透明（时钟=1时跟随输入） |
| D 触发器 | 有 | 边沿触发 | 仅在时钟上升/下降沿采样 |
| JK 触发器 | 有 | 边沿触发 | J=K=1 时翻转 |
| T 触发器 | 有 | 边沿触发 | T=1 时翻转（计数器核心） |

```
D 触发器 (上升沿触发)：
         ┌────┐
  D ────┤ D  Q├─── Q
  CLK ──┤>   Q├─── Q'
         └────┘

  时序图：
    CLK:  ┌┐  ┌┐  ┌┐  ┌┐  ┌┐
    D:   ─┘  └──┘  └──────┘  └──
    Q:   ───────┘  └────────────
               ↑ 上升沿采样
```

### 寄存器与计数器

```
n 位寄存器：
  n 个 D 触发器并行连接，共用时钟

二进制计数器（4-bit 同步）：
  ┌────┐  ┌────┐  ┌────┐  ┌────┐
  │ T  │  │ T  │  │ T  │  │ T  │
  │ Q0 │  │ Q1 │  │ Q2 │  │ Q3 │
  └─┬──┘  └─┬──┘  └─┬──┘  └─┬──┘
    │       │ Q0    │ Q0·Q1│ Q0·Q1·Q2
    └───────┴───────┴───────┘
    每位 T 端接所有低位相与
```

## 6. 有限状态机 (FSM)

### Mealy vs Moore

| 特性 | Mealy | Moore |
|------|-------|-------|
| 输出依赖 | 当前状态 + 输入 | 仅当前状态 |
| 状态数 | 较少 | 可能较多 |
| 输出时序 | 输入变化立即影响 | 一个时钟后变化 |
| 同步性 | 异步输出 | 同步输出 |
| 示例 | 序列检测器 | 计数器 |

```
FSM 设计步骤：
  1. 状态定义 → 状态图 (State Diagram)
  2. 状态表 (State Table)
  3. 状态编码 (State Encoding)
  4. 下一状态逻辑 (Next State Logic)
  5. 输出逻辑 (Output Logic)

状态编码：
  二进制编码:  n 个状态需 ⌈log₂n⌉ 个触发器
  独热编码:    n 个状态需 n 个触发器（适合 FPGA）
```

## 7. 存储器元件

### SRAM vs DRAM

| 特性 | SRAM | DRAM |
|------|------|------|
| 存储单元 | 6 个晶体管 | 1 个晶体管 + 1 个电容 |
| 速度 | 快 (1-10ns) | 慢 (10-50ns) |
| 密度 | 低 | 高 |
| 功耗 | 较高 | 较低（需刷新） |
| 刷新 | 不需要 | 需要 (64ms) |
| 成本 | 高 | 低 |
| 用途 | Cache | 主存 |

```
SRAM 单元 (6T)：
      VDD
       │
   ┌───┴───┐
   │       │
  ┌┴┐     ┌┴┐
  │ │     │ │
  └┬┘     └┬┘
   │       │
   │ BL   BLB│
   └─┬──┬───┘
     │  │
   ┌─┴──┴─┐
   │ 存取管│
   └──┬───┘
      │
    Word Line
```

## 8. PLD（可编程逻辑器件）

| 类型 | 结构 | 可编程 | 特点 |
|------|------|--------|------|
| PLA | 与阵列 + 或阵列 | 两者均可编程 | 灵活，速度较慢 |
| PAL | 与阵列可编程 + 或阵列固定 | 仅与阵列 | 速度较快 |
| CPLD | 多个 PAL 块互连 | 芯片级 | 中等规模 |
| FPGA | LUT + 布线资源 | 门级 | 大规模，可重构 |

```
FPGA 基本结构：
  ┌──────────────────────────────────┐
  │  IOB    IOB    IOB    IOB        │
  ├────┬────┬────┬────┬──────────────┤
  │CLB │ CLB │ CLB │ CLB │            │
  ├────┼────┼────┼────┤  BRAM        │
  │CLB │ CLB │ CLB │ CLB │            │
  ├────┼────┼────┼────┤  DSP         │
  │CLB │ CLB │ CLB │ CLB │            │
  ├────┴────┴────┴────┴──────────────┤
  │          布线资源                  │
  └──────────────────────────────────┘

CLB (可配置逻辑块) 含 LUT + 触发器
LUT (查找表): 实现任意 n 输入组合逻辑 (如 6-LUT)
```

## 9. 时序分析 (Timing Analysis)

### 关键时序参数

```
建立时间 (Setup Time, tₛᵤ):
  数据必须在时钟沿到来之前保持稳定的最短时间

保持时间 (Hold Time, tₕ):
  数据必须在时钟沿之后保持稳定的最长时间

传播延迟 (Propagation Delay, tₚ):
  从输入变化到输出稳定的时间

时钟偏移 (Clock Skew):
  时钟到达不同触发器的时差

最大时钟频率：
  f_max = 1 / (t_clk-to-q + t_logic + t_su + t_skew)

  t_clk-to-q: 触发器时钟到输出延迟
  t_logic:     组合逻辑延迟
  t_su:        建立时间
  t_skew:      时钟偏移（负偏移可提高频率）

时序约束建立时间裕量公式：

$$t_{slack} = T_{clk} - t_{clk-to-q} - t_{logic} - t_{su} - t_{skew} \geq 0$$

保持时间裕量公式：

$$t_{hold\_slack} = t_{clk-to-q} + t_{logic} - t_{hold} - t_{skew} \geq 0$$

常见逻辑门典型传播延迟（74HC 系列，5V）：

| 逻辑门 | 典型 $t_{pd}$ | 最大 $t_{pd}$ |
|--------|-------------|--------------|
| NOT (74HC04) | 7 ns | 13 ns |
| AND (74HC08) | 7 ns | 13 ns |
| OR (74HC32) | 7 ns | 13 ns |
| XOR (74HC86) | 10 ns | 18 ns |
| D-FF (74HC74) | 14 ns | 25 ns |
