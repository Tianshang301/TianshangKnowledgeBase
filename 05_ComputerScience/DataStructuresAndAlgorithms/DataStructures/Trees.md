# 树

## 概述

树是一种分层数据结构，由节点和连接节点的边组成。树是图的一种特殊形式（无环连通图）。

## 树的基本概念

- **根节点**: 没有父节点的顶层节点
- **叶节点**: 没有子节点的节点
- **高度**: 从根到最远叶子的边数
- **深度**: 从根到当前节点的边数
- **二叉树**: 每个节点最多有两个子节点

## 常见树类型

| 类型 | 特点 | 平衡条件 |
|------|------|---------|
| 二叉搜索树 (BST) | 左 < 根 < 右 | 不保证平衡 |
| AVL 树 | 严格平衡 | 左右子树高度差 ≤ 1 |
| 红黑树 | 近似平衡 | 最长路径 ≤ 2× 最短 |
| B 树/B+ 树 | 多路搜索 | 每节点多关键字 |
| 线段树 | 区间查询 | 完全二叉树 |
| Trie | 字符串键值 | 共享前缀 |
