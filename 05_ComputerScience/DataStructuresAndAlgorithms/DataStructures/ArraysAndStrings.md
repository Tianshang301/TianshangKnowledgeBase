# 数组与字符串

## 概述

数组和字符串是最基础且最常用的线性数据结构。数组是连续内存中存储的同类型元素集合，字符串是字符的序列。

## 核心操作

- **数组**: 索引访问 O(1)、插入/删除 O(n)、遍历 O(n)
- **字符串**: 连接、子串搜索、模式匹配（KMP、Boyer-Moore、Rabin-Karp）

## 典型算法

- 双指针、滑动窗口、前缀和
- 字符串哈希、后缀数组、Trie 树
