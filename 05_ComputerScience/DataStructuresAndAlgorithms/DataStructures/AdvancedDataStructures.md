# 高级数据结构

## 概述

高级数据结构建立在基础数据结构之上，用于解决更复杂的问题，通常在竞赛编程和系统设计中广泛应用。

## 常见高级数据结构

| 结构 | 主要用途 | 时间复杂度 |
|------|---------|-----------|
| 线段树 (SegmentTree) | 区间查询与更新 | O(log n) |
| 树状数组 (Fenwick) | 前缀和与单点更新 | O(log n) |
| 平衡树 (AVL/RBTree) | 动态有序集合 | O(log n) |
| 并查集 (UnionFind) | 连通性问题 | O(α(n)) |
| Trie 树 | 字符串检索 | O(k) |
| 后缀数组 | 字符串模式匹配 | O(n log n) |
| 稀疏表 (SparseTable) | 静态区间最值 | O(1) 查询 |
