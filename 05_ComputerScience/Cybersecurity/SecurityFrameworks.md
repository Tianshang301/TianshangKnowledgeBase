# 安全框架

## 概述

安全框架（Security Framework）是组织为管理网络安全风险而采用的政策、流程和技术控制的系统化指南。框架提供标准化的方法论来评估、实施和维护安全控制。

## 主流安全框架

| 框架 | 发布机构 | 特点 |
|------|---------|------|
| NIST CSF | NIST（美国） | 五大核心功能：识别、保护、检测、响应、恢复 |
| ISO 27001 | ISO | 信息安全管理体系（ISMS）认证标准 |
| CIS Controls | CIS | 18个关键安全控制，实施优先级明确 |
| COBIT | ISACA | IT治理与风险管理框架 |
| PCI DSS | PCI SSC | 支付卡行业数据安全标准 |

## 框架选择建议

- **企业全面风险管理**：ISO 27001 + NIST CSF 结合
- **云安全**：CIS Benchmarks + CSA CCM
- **合规驱动**：根据行业监管要求选择（如金融/医疗/政务）
