# 密码学 Cryptography

密码学是研究如何在不安全信道上实现安全通信的学科，涉及加密、解密、哈希函数和数字签名等技术。密码学是信息安全的核心基础，保护数据的机密性、完整性、真实性和不可否认性。

密码学的主要分支包括对称加密（使用相同的密钥进行加密和解密，如 AES、DES）和非对称加密（使用公钥-私钥对，如 RSA、ECC）。对称加密速度快，适合大量数据加密；非对称加密解决了密钥分发问题，常用于数字签名和密钥交换。哈希函数（如 SHA-256）将任意长度数据映射为固定长度的摘要，用于数据完整性验证和密码存储。

密码学的经典应用包括安全通信（TLS/SSL 协议保护 Web 通信）、数字签名（验证身份和防止抵赖）、公钥基础设施（管理数字证书）和区块链（工作量证明和交易验证）。量子密码学（BB84 协议）利用量子力学原理实现理论上不可破解的通信。随着量子计算的发展，后量子密码学正在研究和标准化能够抵抗量子攻击的新型密码算法。
