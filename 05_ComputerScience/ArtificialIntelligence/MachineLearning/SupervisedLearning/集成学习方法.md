# 集成学习方法

## 一、集成学习概述

集成学习通过组合多个基学习器来获得比单个学习器更优的泛化性能。其理论基础在于：如果基学习器之间具有足够的多样性，则集成模型的误差可以通过投票或平均显著降低。

考虑 $T$ 个独立基学习器，每个的误差为 $\epsilon$，采用多数投票的集成误差为：

$$
P(\text{ensemble error}) = \sum_{k=\lceil T/2 \rceil}^{T} \binom{T}{k} \epsilon^k (1-\epsilon)^{T-k}
$$

当 $\epsilon < 0.5$ 时，随着 $T$ 增大，集成误差指数级下降。这一结论依赖于"独立性"假设，而实际基学习器之间难以完全独立。

## 二、Boosting方法

### 2.1 AdaBoost

AdaBoost(Adaptive Boosting)通过迭代调整样本权重训练基学习器。算法核心：

1. 初始化样本权重 $w_i^{(1)} = 1/N$
2. 每轮 $t = 1, \ldots, T$：
   - 基于当前权重训练基学习器 $h_t$
   - 计算误差率 $\epsilon_t = \sum_{i=1}^N w_i^{(t)} \cdot \mathbb{I}[h_t(x_i) \neq y_i]$
   - 计算权重 $\alpha_t = \frac{1}{2} \ln\left(\frac{1-\epsilon_t}{\epsilon_t}\right)$
   - 更新权重 $w_i^{(t+1)} \propto w_i^{(t)} \cdot \exp(-\alpha_t y_i h_t(x_i))$

最终集成：$H(x) = \text{sign}\left(\sum_{t=1}^T \alpha_t h_t(x)\right)$

AdaBoost的指数损失函数理论框架：

$$
\mathcal{L}(y, f(x)) = \exp(-y f(x))
$$

前向分步加法建模将AdaBoost解释为指数损失函数的贪心优化。

### 2.2 Gradient Boosting

Gradient Boosting Machine(GBM)通过拟合当前模型的负梯度残差来逐步优化。第 $t$ 轮的基学习器拟合残差：

$$
r_{i}^{(t)} = -\left[ \frac{\partial \mathcal{L}(y_i, F_{t-1}(x_i))}{\partial F_{t-1}(x_i)} \right]
$$

对于平方损失 $\mathcal{L}(y, F) = \frac{1}{2}(y - F)^2$，残差为 $r_i^{(t)} = y_i - F_{t-1}(x_i)$。

### 2.3 XGBoost

XGBoost对GBM进行系统优化，使用二阶泰勒展开近似损失函数：

$$
\mathcal{L}^{(t)} \approx \sum_{i=1}^N \left[ g_i f_t(x_i) + \frac{1}{2} h_i f_t^2(x_i) \right] + \Omega(f_t)
$$

其中 $g_i = \partial_{F_{t-1}} \mathcal{L}(y_i, F_{t-1})$，$h_i = \partial^2_{F_{t-1}} \mathcal{L}(y_i, F_{t-1})$，正则项 $\Omega(f) = \gamma T + \frac{1}{2}\lambda \sum_{j=1}^T w_j^2$。

| 特性 | GBM | XGBoost | LightGBM | CatBoost |
|------|-----|---------|----------|----------|
| 分裂算法 | 贪心 | 预排序 | 直方图 | 对称分裂 |
| 缺失值处理 | 无 | 自动学习 | 自动学习 | 内置处理 |
| 分类特征 | 需编码 | 需编码 | 需编码 | 原生支持 |
| GPU支持 | 有限 | 好 | 好 | 好 |
| 训练速度 | 慢 | 中 | 快 | 中 |

## 三、Bagging与随机森林

### 3.1 Bagging

Bagging(Bootstrap Aggregating)通过自助采样生成 $T$ 个训练子集，分别训练基学习器后平均/投票：

$$
H(x) = \frac{1}{T} \sum_{t=1}^T h_t(x) \quad \text{(回归)}
$$

**包外估计(OOB)**：每个样本在约 $1/e$ 的基学习器中未被使用，可作为天然验证集。

### 3.2 随机森林

随机森林在Bagging基础上引入随机特征选择：每棵决策树分裂时，从随机选择的 $m$ 个特征子集中选择最优分裂特征。引入额外随机性增加了基学习器多样性。

随机森林的泛化误差上界：

$$
P_{x,y}(H(x) \neq y) \leq \frac{\bar{\rho}(1 - s^2)}{s^2}
$$

其中 $\bar{\rho}$ 为基学习器间的平均相关性，$s$ 为基学习器的平均强度。

## 四、Stacking

Stacking通过元学习器组合基学习器的预测结果。训练过程分为两层：

**第一层**：训练 $K$ 个基学习器，使用交叉验证生成元特征

```python
def stacking_cv(base_models, meta_model, X, y, n_folds=5):
    meta_features = np.zeros((len(X), len(base_models)))
    skf = StratifiedKFold(n_splits=n_folds)

    for i, model in enumerate(base_models):
        for train_idx, val_idx in skf.split(X, y):
            model.fit(X[train_idx], y[train_idx])
            meta_features[val_idx, i] = model.predict(X[val_idx])

    meta_model.fit(meta_features, y)
    return meta_model
```

**第二层**：元学习器训练，学习如何最优组合基学习器的预测。

## 五、多样性与集成策略

### 5.1 多样性度量

基学习器的多样性是集成成功的关键。常用度量包括：

- **分歧度(disagreement)**：$\text{dis}_{ij} = \frac{1}{N} \sum_{k=1}^N \mathbb{I}[h_i(x_k) \neq h_j(x_k)]$
- **Q统计量**：$Q_{ij} = \frac{N^{11}N^{00} - N^{01}N^{10}}{N^{11}N^{00} + N^{01}N^{10}}$
- **相关系数**：$\rho_{ij} = \frac{\text{Cov}(h_i, h_j)}{\sigma_{h_i}\sigma_{h_j}}$

### 5.2 集成修剪

移除冗余基学习器可以降低存储和计算成本。排序方法按性能-多样性评分选择最优子集：

$$
\text{score}(h_i) = \text{perf}(h_i) + \lambda \cdot \text{div}(h_i, \mathcal{S})
$$

## 六、集成学习的变体

### 6.1 投票与平均策略

- **硬投票**：多数投票 $H(x) = \text{mode}\{h_1(x), \ldots, h_T(x)\}$
- **软投票**：概率平均 $H(x) = \arg\max_c \frac{1}{T} \sum_{t=1}^T p_t(c|x)$
- **加权平均**：按验证性能加权 $H(x) = \sum_{t=1}^T w_t h_t(x)$，$w_t \propto \text{acc}_t$

### 6.2 贝叶斯模型平均与组合

贝叶斯模型组合(BMA)通过后验概率加权模型：

$$
P(y|x, \mathcal{D}) = \sum_{t=1}^T P(y|x, h_t) P(h_t|\mathcal{D})
$$

## 七、实践指南

不同集成方法的最佳适用场景：

| 场景 | 推荐方法 | 原因 |
|------|----------|------|
| 高方差低偏差 | Bagging/随机森林 | 降低方差 |
| 低方差高偏差 | Boosting(XGBoost) | 降低偏差 |
| 异构模型 | Stacking | 利用互补性 |
| 快速原型 | LightGBM | 训练速度快 |
| 分类特征多 | CatBoost | 原生支持 |

```python
from sklearn.ensemble import RandomForestClassifier, GradientBoostingClassifier
from sklearn.ensemble import VotingClassifier

# 异质集成
rf = RandomForestClassifier(n_estimators=100, max_depth=10)
gb = GradientBoostingClassifier(n_estimators=100, learning_rate=0.1)
ensemble = VotingClassifier(
    estimators=[('rf', rf), ('gb', gb)],
    voting='soft'
)
```
