# 计算机视觉 Computer Vision 概述

**计算机视觉（Computer Vision, CV）** 是人工智能的重要分支，旨在让计算机从图像或视频中获取高层语义理解，模拟人类的视觉感知能力。

核心任务包括：**图像分类（Image Classification）** 将图像归入预定义类别；**目标检测（Object Detection）** 同时定位与识别图中物体（如 YOLO、Faster R-CNN）；**语义分割（Semantic Segmentation）** 对每个像素分类；**实例分割（Instance Segmentation）** 进一步区分同类的不同个体。经典模型历经 AlexNet、VGG、ResNet 等 CNN 架构，到近年 Vision Transformer（ViT）与多模态大模型的发展。代表性数据集有 ImageNet（千类图像分类基准）和 COCO（目标检测与分割标准数据集）。应用领域涵盖自动驾驶（车道检测/行人识别）、医学影像（病灶分割/辅助诊断）、安防监控（人脸识别/行为分析）以及工业质检等。
