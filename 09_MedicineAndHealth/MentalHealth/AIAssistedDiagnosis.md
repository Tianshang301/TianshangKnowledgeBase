# AI辅助精神科诊断

AI辅助精神科诊断是利用人工智能技术改进精神疾病筛查、诊断和监测的新兴领域。传统精神科诊断依赖主观评估，AI技术有望提供更客观、一致和可及的诊断支持。

## NLP语音分析

### 语音特征与精神疾病

**抑郁症语音特征：**

| 特征类别 | 具体特征 | 变化方向 | 机制 |
|---------|---------|---------|------|
| 韵律特征 | 语速 | 减慢 | 精神运动迟缓 |
| | 音高变化 | 减少 | 情感平淡 |
| | 停顿频率 | 增加 | 认知加工困难 |
| 声学特征 | 基频(F0) | 降低 | 能量不足 |
| | 振幅 | 降低 | 动力缺乏 |
| | 谐噪比 | 改变 | 声带张力变化 |
| 语言特征 | 词汇多样性 | 降低 | 思维贫乏 |
| | 句子复杂度 | 降低 | 认知功能下降 |
| | 第一人称使用 | 增加 | 自我关注增加 |

**精神分裂症语音特征：**
- 语义连贯性降低
- 词汇联想异常
- 语用障碍
- 语流障碍

**焦虑障碍语音特征：**
- 语速加快
- 停顿减少
- 音高变异性增加
- 重复和修正增加

| 指标 | 测量 | 心理健康关联 |

| 内容分享 | 转发、分享 | 兴趣变化 |

**语言指标：**

| 指标 | 测量 | 心理健康关联 |
|------|------|-------------|
| 情感词比例 | 正/负面词汇 | 负面情感：增加 |
| 第一人称代词 | 我、我的 | 自我关注：增加 |
| 绝对化词汇 | 总是、从不 | 认知歪曲：增加 |
| 死亡相关词 | 死亡、自杀 | 自杀风险：增加 |
| 社交词 | 朋友、家人 | 社交功能：减少 |

## 计算精神病学

### 理论框架

**计算精神病学（Computational Psychiatry）核心概念：**

| 概念 | 含义 | 应用 |
|------|------|------|
| 贝叶斯脑 | 大脑进行概率推断 | 妄想研究 |
| 强化学习 | 通过奖励学习决策 | 成瘾、抑郁 |
| 预测编码 | 预测误差驱动学习 | 精神分裂症 |
| 自由能原理 | 最小化意外 | 多种精神疾病 |

### 计算模型应用

**抑郁症计算模型：**

| 模型 | 机制 | 预测 |
|------|------|------|
| 习得性无助 | 控制感缺失 | 抑郁易感性 |
| 奖励敏感性 | 奖励加工异常 | 快感缺失 |
| 反刍思维 | 负性思维循环 | 症状持续 |
| 贝叶斯信念更新 | 信念更新偏差 | 认知歪曲 |

**精神分裂症计算模型：**
- 精准预测编码异常
- 多巴胺信号异常
- 信念更新偏差
- 感觉运动整合障碍

### 数字表型

**数字表型（Digital Phenotyping）定义：**
- 通过数字设备被动收集的行为数据
- 反映日常功能和症状状态
- 实时、客观、连续

**数据来源：**

| 数据类型 | 设备 | 行为指标 |
|---------|------|---------|
| 活动数据 | 智能手机加速度计 | 活动水平、节律 |
| 社交数据 | 通话、短信记录 | 社交频率、模式 |
| 位置数据 | GPS | 活动范围、规律性 |
| 睡眠数据 | 手机使用模式 | 睡眠时间、质量 |
| 认知数据 | 键盘输入 | 打字速度、错误率 |
