# 中医基础理论

## 概述
中医基础理论是研究中医学基本概念、基本原理和基本规律的学科，包括阴阳学说、五行学说、藏象学说、气血津液学说、经络学说等。

---

## 第一章 阴阳学说

### 一、阴阳的基本概念
- **阴**：凡是运动的、外向的、上升的、温热的、明亮的属阳
- **阳**：凡是静止的、内守的、下降的、寒冷的、晦暗的属阴

### 二、阴阳学说的基本内容

#### 1. 阴阳对立
- 阴和阳是相互对立的两个方面
- 如：上与下、动与静、寒与热

#### 2. 阴阳互根
- 阴和阳是相互依存、互为根本的
- 阴根于阳，阳根于阴

#### 3. 阴阳消长
- 阴和阳在一定范围内不断变化
- 阴消则阳长，阳消则阴长

#### 4. 阴阳转化
- 阴和阳在一定条件下可以相互转化
- 重阴必阳，重阳必阴

### 三、阴阳学说在中医中的应用
- 解释人体组织结构
- 解释生理功能
- 解释病理变化
- 指导疾病诊断
- 指导疾病治疗

## 第二章 五行学说

### 一、五行的基本概念
- 木、火、土、金、水五种基本物质的抽象概念

### 二、五行的特性
| 五行 | 特性 | 对应脏腑 |
|------|------|----------|
| 木 | 生长、升发、条达 | 肝、胆 |
| 火 | 温热、向上、明亮 | 心、小肠 |
| 土 | 生化、承载、受纳 | 脾、胃 |
| 金 | 清洁、收敛、肃降 | 肺、大肠 |
| 水 | 寒凉、滋润、向下 | 肾、膀胱 |

### 三、五行生克乘侮

#### 1. 相生
- 木→火→土→金→水→木
- 生我者为母，我生者为子

#### 2. 相克
- 木→土→水→火→金→木
- 克我者为所不胜，我克者为所胜

#### 3. 相乘
- 五行中某一行过于亢盛，对其所胜一行进行过度克制

#### 4. 相侮
- 五行中某一行过于亢盛，对其所不胜一行进行反向克制

## 第三章 藏象学说

### 一、五脏功能

| 脏腑 | 主要功能 | 联系 |
|------|----------|------|
| 心 | 主血脉、主神明 | 开窍于舌，其华在面 |
| 肝 | 主疏泄、主藏血 | 开窍于目，其华在爪 |
| 脾 | 主运化、主统血 | 开窍于口，其华在唇 |
| 肺 | 主气、主宣发肃降 | 开窍于鼻，其华在皮毛 |
| 肾 | 主藏精、主水、主纳气 | 开窍于耳及二阴，其华在发 |

### 二、六腑功能
| 腑 | 主要功能 |
|----|----------|
| 胆 | 储存和排泄胆汁 |
| 胃 | 受纳腐熟水谷 |
| 小肠 | 受盛化物、泌别清浊 |
| 大肠 | 传化糟粕 |
| 膀胱 | 贮存和排泄尿液 |
| 三焦 | 通行元气、运行水液 |

## 第四章 气血津液学说

### 一、气
| 气的类型 | 功能 |
|----------|------|
| 元气 | 推动人体生长发育，温煦脏腑 |
| 宗气 | 走息道以行呼吸，贯心脉以行血 |
| 营气 | 化生血液，营养全身 |
| 卫气 | 护卫肌表，温养脏腑 |

### 二、血
- **功能**：濡养全身，神志活动的物质基础
- **生成**：水谷精微化生，肾精化血

### 三、津液
- **津**：质地清稀，流动性大
- **液**：质地稠厚，流动性小
- **功能**：滋润濡养

## 第六章 病因病机

### 一、六淫
| 病因 | 特点 | 致病特点 |
|------|------|----------|
| 风 | 善行数变 | 头痛、恶寒、身痛 |
| 寒 | 凝滞收引 | 恶寒、疼痛、拘急 |
| 暑 | 升散耗气 | 高热、口渴、多汗 |
| 湿 | 重浊黏滞 | 头重、肢体困重 |
| 燥 | 干涩伤津 | 口鼻干燥、皮肤干裂 |
| 火 | 炎上急迫 | 高热、烦渴、出血 |

### 二、七情
| 情志 | 对应脏腑 | 致病特点 |
|------|----------|----------|
| 喜 | 心 | 喜则气缓 |
| 怒 | 肝 | 怒则气上 |
| 忧 | 肺 | 忧则气沉 |
| 思 | 脾 | 思则气结 |
| 恐 | 肾 | 恐则气下 |
| 惊 | 心 | 惊则气乱 |

### 三、痰饮瘀血
- **痰饮**：水液代谢障碍形成的病理产物
- **瘀血**：血行不畅或离经之血

## 复习要点

1. 阴阳学说包括对立、互根、消长、转化四个方面
2. 五行学说以生克乘侮关系解释事物之间的联系
3. 藏象学说以五脏为中心，联系六腑和形体官窍
4. 气血津液是维持生命活动的基本物质
5. 经络是运行气血、联络脏腑肢节的通路
6. 六淫和七情是常见的致病因素
