# 基因组学

基因组学（Genomics）是研究生物体完整基因组结构、功能和演化的学科，是精准医学的核心基础。随着高通量测序技术的快速发展，基因组学已从科研走向临床，为疾病诊断、治疗和预防提供了全新的分子视角。

## 全基因组测序

### 技术原理

全基因组测序（Whole Genome Sequencing, WGS）是对生物体整个基因组进行DNA序列测定的技术。与早期的Sanger测序相比，新一代测序（Next-Generation Sequencing, NGS）技术大幅提升了通量并降低了成本。

**主要测序平台：**

| 平台 | 技术特点 | 读长 | 应用场景 |
|------|---------|------|---------|
| Illumina | 边合成边测序 | 150-300bp | 临床诊断、变异检测 |
| PacBio HiFi | 单分子实时测序 | 10-25kb | 结构变异、甲基化 |
| Oxford Nanopore | 纳米孔测序 | 超长读长 | 快速诊断、现场检测 |
| MGI | DNA纳米球 | 100-400bp | 大规模人群研究 |

### 临床应用

WGS在临床中的应用日益广泛：

1. **罕见病诊断**：约30-50%的罕见病患者通过WGS获得分子诊断
2. **肿瘤基因组图谱**：全面揭示肿瘤突变负荷（TMB）和微卫星不稳定性（MSI）
3. **药物基因组学**：指导个体化用药方案
4. **产前筛查**：无创产前检测（NIPT）的扩展应用

### 数据分析流程

```
原始数据 → 质量控制 → 比对参考基因组 → 变异检测 → 变异注释 → 临床解读
```

关键分析工具包括BWA、GATK、ANNOVAR等，临床解读需遵循ACMG变异分类标准。

## 外显子组测序

### 技术特点

全外显子组测序（Whole Exome Sequencing, WES）靶向捕获并测序人类基因组中约2%的外显子区域，这些区域编码约85%的已知致病变异。

**WES与WGS的比较：**

- **成本效益**：WES成本约为WGS的1/3-1/2
- **数据分析**：WES数据量小，分析更简便
- **覆盖度**：WES在外显子区域覆盖更深
- **局限性**：WES无法检测非编码区变异和结构变异

### 临床应用价值

外显子组测序在以下领域表现突出：

- **智力发育障碍**：诊断率约30-40%
- **遗传性心脏病**：如长QT综合征、肥厚型心肌病
- **遗传性肿瘤综合征**：Lynch综合征、Li-Fraumeni综合征
- **代谢性疾病**：苯丙酮尿症、糖原累积病

## 全基因组关联研究

### 研究方法

全基因组关联研究（Genome-Wide Association Studies, GWAS）通过在全基因组范围内检测大量单核苷酸多态性（SNP），寻找与特定疾病或性状相关的遗传变异。

**研究设计要素：**

1. **样本量**：现代GWAS通常需要数万至数十万样本
2. **SNP芯片**：通常检测50万-500万个SNP位点
3. **统计阈值**：全基因组显著性阈值为P < 5×10⁻⁸
4. **重复验证**：需在独立队列中进行验证

### 主要发现

GWAS已揭示了大量疾病的遗传基础：

- **2型糖尿病**：已发现超过400个风险位点
- **精神分裂症**：超过270个风险位点
- **冠心病**：超过160个风险位点
- **自身免疫性疾病**：HLA区域的广泛关联

### 多基因风险评分

基于GWAS结果构建的多基因风险评分（Polygenic Risk Score, PRS）可综合评估个体的遗传风险：

```
PRS = Σ (βi × Gi)
```

其中βi为效应值，Gi为个体在位点i的基因型。

PRS在以下领域展现应用潜力：
- 冠心病风险分层
- 乳腺癌筛查策略优化
- 精神疾病早期预警

## 结构变异

### 变异类型

结构变异（Structural Variants, SV）是指基因组中大于50bp的DNA片段变异，包括：

| 变异类型 | 描述 | 检测方法 |
|---------|------|---------|
| 缺失（Deletion） | DNA片段丢失 | WGS、芯片 |
| 重复（Duplication） | DNA片段重复 | WGS、MLPA |
| 倒位（Inversion） | DNA片段方向颠倒 | 长读长测序 |
| 易位（Translocation） | 染色体间片段交换 | FISH、WGS |
| 插入（Insertion） | 外源DNA片段插入 | 长读长测序 |

### 临床意义

结构变异在多种疾病中发挥重要作用：

- **神经发育障碍**：22q11.2缺失综合征、Williams综合征
- **肿瘤**：基因融合（如BCR-ABL）、拷贝数变异
- **遗传性心血管病**：LDLR基因大片段缺失
- **自闭症谱系障碍**：16p11.2缺失/重复

### 检测技术进展

长读长测序技术（PacBio、Oxford Nanopore）在结构变异检测中具有独特优势：

1. **分辨率提升**：可检测复杂结构变异和重复序列
2. **单倍型解析**：实现染色体级别的单倍型定相
3. **表观遗传学**：直接检测DNA甲基化修饰

## 基因组学的伦理与挑战

### 数据隐私

基因组数据的隐私保护面临严峻挑战：
- 基因组数据的唯一性和不可更改性
- 家族成员隐私的连带影响
- 基因歧视的潜在风险

### 临床转化障碍

- 变异解读的复杂性和不确定性
- 偶发性发现（Incidental Findings）的处理
- 遗传咨询资源的不足
- 不同种族群体遗传数据的公平性
