# 伴随诊断

伴随诊断（Companion Diagnostics, CDx）是一种与特定治疗药物配对使用的体外诊断技术，用于评估患者是否适合接受该药物治疗。伴随诊断是精准医学实施的关键环节，确保"对的药物给对的患者"。

## 生物标志物

### 标志物类型

**基因突变标志物：**
- 点突变：EGFR L858R、T790M
- 插入缺失：EGFR exon19缺失
- 基因融合：ALK融合、ROS1融合
- 基因扩增：HER2扩增

**蛋白表达标志物：**
- 免疫组化检测：PD-L1表达
- 蛋白过表达：HER2蛋白（IHC 3+）
- 蛋白缺失：MMR蛋白（MLH1、MSH2等）

**基因组特征标志物：**
- 微卫星不稳定性（MSI-H）
- 肿瘤突变负荷（TMB-H）
- 同源重组缺陷（HRD）
- 基因组瘢痕评分

**液体活检标志物：**
- ctDNA突变：EGFR T790M
- 循环肿瘤细胞（CTC）
- 外泌体标志物

### 标志物发现与验证

生物标志物的开发遵循严格的验证流程：

1. **发现阶段**：高通量筛选候选标志物
2. **分析验证**：评估检测方法的准确性、精密度、稳定性
3. **临床验证**：在临床试验中证明标志物与疗效的关联
4. **临床效用**：证明基于标志物的治疗决策改善患者结局

## 靶向治疗选择

### 肺癌伴随诊断

**EGFR靶向治疗：**

| 检测方法 | 目标 | 适用药物 | 灵敏度 |
|---------|------|---------|--------|
| PCR | 已知突变 | 一/二/三代TKI | 1%-5% |
| NGS Panel | 广谱突变 | 多种TKI | 0.1%-1% |
| FISH | 扩增 | TKI | - |
| IHC | 蛋白表达 | 参考 | - |

**ALK/ROS1融合检测：**
- FISH：金标准，但操作复杂
- IHC：筛查方法，需FISH确认
- NGS：可同时检测多种融合类型
- RT-PCR：快速检测已知融合

### 乳腺癌分子分型

乳腺癌的分子分型指导治疗策略：

| 分子分型 | ER/PR | HER2 | Ki-67 | 治疗策略 |
|---------|-------|------|-------|---------|
| Luminal A | + | - | 低 | 内分泌治疗 |
| Luminal B | + | +/- | 高 | 内分泌+化疗 |
| HER2阳性 | - | + | - | 抗HER2治疗 |
| 三阴性 | - | - | - | 化疗/免疫治疗 |

### 结直肠癌标志物

**RAS/BRAF突变检测：**
- KRAS/NRAS exon 2-4突变：抗EGFR单抗禁忌
- BRAF V600E突变：预后不良标志
- 检测方法：NGS或PCR

**MSI/MMR检测：**
- MSI-H/dMMR：免疫治疗适应证
- IHC筛查+PCR确认
- 预测Lynch综合征风险

### 其他肿瘤伴随诊断

**泛肿瘤标志物：**
- **TMB-H**：帕博利珠单抗（Pembrolizumab）泛癌种适应证
- **MSI-H/dMMR**：多款免疫检查点抑制剂适应证
- **NTRK融合**：拉罗替尼（Larotrectinib）泛癌种适应证
- **RET融合**：塞尔帕替尼（Selpercatinib）泛癌种适应证

## 审批流程

### 美国FDA审批

**共同开发模式（Co-development）：**
- 药物与伴随诊断同步开发
- 在同一临床试验中验证
- 获得联合批准

**审批途径：**
1. **PMA（上市前批准）**：最高级别审批
2. **510(k)**：基于实质等同性
3. **De Novo**：新型低中风险器械

**FDA指导原则：**
- 2014年《伴随诊断指南》
- 2020年《肿瘤生物标志物检测指南》
- "或同等"（or equivalent）政策

### 欧盟CE认证

**IVDR法规（2022年实施）：**
- 风险分类：Class A-D
- 伴随诊断通常为Class C
- 需要公告机构审核
- 性能评估要求更严格

### 中国NMPA审批

**注册分类：**
- 第三类体外诊断试剂
- 需要临床试验数据
- 与药品审评中心联合审评

**审批流程：**
1. 产品注册检验
2. 临床试验申请
3. 临床试验实施
4. 注册申请与技术审评
5. 行政审批

## 检测平台与技术

### 主要检测技术比较

| 技术 | 优势 | 局限 | 适用场景 |
|------|------|------|---------|
| IHC | 成本低、普及广 | 主观判读 | 蛋白表达筛查 |
| FISH | 灵敏度高 | 操作复杂 | 基因扩增/融合 |
| PCR | 快速、灵敏 | 检测范围有限 | 已知突变 |
| NGS | 信息量大 | 成本高、周期长 | 多基因检测 |
| 数字PCR | 超高灵敏度 | 通量低 | 微小残留病灶 |

### NGS Panel设计

肿瘤NGS Panel的设计考虑：

- **基因数量**：从几十个到数百个基因不等
- **变异类型**：SNV、InDel、CNV、融合
- **测序深度**：通常要求500-1000x以上
- **样本类型**：组织样本和/或液体活检样本

### 质量控制

伴随诊断的质量控制要素：

1. **分析前**：样本采集、运输、储存标准化
2. **分析中**：试剂批间一致性、仪器校准
3. **分析后**：报告标准化、结果解读一致性
4. **室间质评**：定期参加外部质量评估

## 发展趋势

### 泛肿瘤标志物

从组织特异性向泛肿瘤标志物发展的趋势：

- MSI-H/dMMR已在多癌种获批
- TMB-H的泛癌种应用
- NTRK融合的"组织不可知"治疗

### 人工智能辅助

AI在伴随诊断中的应用：

- 数字病理学辅助IHC判读
- 机器学习优化变异解读
- 影像基因组学整合分析

### 实时监测

从静态诊断向动态监测发展：

- 液体活检指导治疗调整
- ctDNA动态监测疗效
- MRD检测指导辅助治疗
