# 精准肿瘤学

精准肿瘤学（Precision Oncology）是将分子诊断和基因组分析整合到肿瘤诊疗全过程的医学模式，旨在为每位患者提供基于其肿瘤分子特征的个体化治疗方案。通过分子分型、生物标志物指导用药和免疫治疗评估，实现肿瘤治疗的精准化。

## 分子分型

### 分子分型概念

分子分型（Molecular Subtyping）是根据肿瘤的分子特征将患者分为不同亚型的过程，每种亚型具有独特的生物学行为、预后特征和治疗敏感性。

**分子分型的意义：**
- 揭示肿瘤异质性
- 指导治疗策略选择
- 预测预后
- 发现新的治疗靶点

### 主要肿瘤分子分型

**肺癌分子分型：**

| 分子亚型 | 频率 | 靶向药物 | 预后 |
|---------|------|---------|------|
| EGFR突变 | 15-50% | 奥希替尼 | 良好 |
| ALK融合 | 3-7% | 阿来替尼 | 良好 |
| ROS1融合 | 1-2% | 克唑替尼 | 良好 |
| KRAS G12C | 13% | Sotorasib | 中等 |
| BRAF V600E | 2-4% | 达拉非尼+曲美替尼 | 中等 |
| MET扩增 | 3-4% | 赛沃替尼 | 中等 |
| RET融合 | 1-2% | 塞尔帕替尼 | 良好 |
| NTRK融合 | <1% | 拉罗替尼 | 良好 |

**乳腺癌分子分型：**

基于PAM50基因表达谱的内在分型：
- **Luminal A**：ER+/HER2-，低增殖，预后最佳
- **Luminal B**：ER+，高增殖，需要化疗
- **HER2富集**：HER2扩增，抗HER2治疗敏感
- **基底样型**：三阴性表型，化疗为主

**结直肠癌分子分型：**

共识分子亚型（Consensus Molecular Subtypes, CMS）：
- **CMS1（MSI免疫型）**：MSI-H，免疫治疗敏感
- **CMS2（经典型）**：WNT/MYC激活，预后良好
- **CMS3（代谢型）**：代谢失调，混合特征
- **CMS4（间质型）**：EMT激活，预后最差

### 分子肿瘤委员会

分子肿瘤委员会（Molecular Tumor Board, MTB）是精准肿瘤学实施的核心机制：

**组成成员：**
- 肿瘤内科医生
- 分子病理学家
- 遗传咨询师
- 生物信息学家
- 临床药师

**讨论内容：**
- 分子检测结果解读
- 治疗方案推荐
- 临床试验匹配
- 争议病例讨论

## NGS指导用药

### NGS检测范围

肿瘤NGS检测可覆盖多种基因变异类型：

**标准Panel检测：**
- 单核苷酸变异（SNV）
- 插入缺失（InDel）
- 基因拷贝数变异（CNV）
- 基因融合/重排

**扩展检测内容：**
- 肿瘤突变负荷（TMB）
- 微卫星不稳定性（MSI）
- 同源重组缺陷（HRD）
- 特定基因突变特征

### 检测策略

**组织样本检测：**
- 金标准，信息量最完整
- 需要足够的肿瘤组织
- 存在肿瘤异质性问题

**液体活检检测：**
- 非侵入性，可重复取样
- 反映肿瘤整体异质性
- 灵敏度受ctDNA含量影响

**整合检测策略：**
- 组织+液体活检互补
- 提高变异检出率
- 动态监测疾病演变

### 可用药靶点分级

ESCAT（ESMO Scale for Clinical Actionability of molecular Targets）分级：

| 等级 | 定义 | 示例 |
|------|------|------|
| I级 | 已获批适应证 | EGFR突变-奥希替尼 |
| II级 | 临床试验证据 | KRAS G12C-Sotorasib |
| III级 | 临床有效性 | BRAF非V600E突变 |
| IV级 | 临床前证据 | 新发现的驱动突变 |
| V级 | 仅生物学意义 | 意义未明变异 |

### 生物信息学分析

NGS数据分析流程：

```
原始数据 → 质控 → 比对 → 变异检测 → 注释 → 解读 → 报告
```

**关键分析步骤：**
1. 变异体识别与过滤
2. 变异致病性评估
3. 可用药靶点匹配
4. 临床证据等级评估

## 免疫治疗标志物

### PD-L1表达

PD-L1（Programmed Death-Ligand 1）是最常用的免疫治疗标志物：

**检测方法：**
| 抗体克隆 | 平台 | 适用药物 |
|---------|------|---------|
| 22C3 | Dako | 帕博利珠单抗 |
| 28-3 | Ventana | 阿替利珠单抗 |
| SP263 | Ventana | 度伐利尤单抗 |
| SP142 | Ventana | 阿替利珠单抗（尿路上皮癌） |

**判读标准：**
- **TPS（Tumor Proportion Score）**：PD-L1阳性肿瘤细胞占比
- **CPS（Combined Positive Score）**：PD-L1阳性细胞（肿瘤+免疫）占比
- **IC（Immune Cell Score）**：PD-L1阳性免疫细胞占比

### 微卫星不稳定性

MSI（Microsatellite Instability）是DNA错配修复缺陷的标志：

**检测方法：**
- **IHC**：检测MMR蛋白（MLH1、MSH2、MSH6、PMS2）表达缺失
- **PCR**：检测微卫星位点长度变化
- **NGS**：计算MSI状态，可同时检测其他变异

**临床意义：**
- MSI-H/dMMR肿瘤对免疫治疗高度敏感
- 帕博利珠单抗获批MSI-H泛癌种适应证
- 预测Lynch综合征风险

### 肿瘤突变负荷

TMB（Tumor Mutational Burden）是评估免疫治疗反应的另一重要标志物：

**定义与计算：**
- 每兆碱基（Mb）体细胞非同义突变数量
- TMB-H通常定义为≥10 mut/Mb（各癌种阈值不同）

**影响因素：**
- 癌种类型（黑色素瘤、肺癌TMB较高）
- 既往治疗史（化疗/放疗可增加TMB）
- DNA修复缺陷

### 新兴免疫标志物

**肿瘤微环境特征：**
- 肿瘤浸润淋巴细胞（TIL）密度
- 免疫评分（Immunoscore）
- T细胞炎症基因表达谱（GEP）

**基因组特征：**
- 特定基因变异（如STK11、KEAP1突变与免疫治疗耐药相关）
- 染色体不稳定性
- 基因组倍体

## 临床应用案例

### 非小细胞肺癌精准治疗

**一线治疗决策流程：**
1. 组织获取与分子检测
2. 检测EGFR、ALK、ROS1等驱动基因
3. 阳性者：对应靶向治疗
4. 阴性者：检测PD-L1表达
5. PD-L1高表达：免疫治疗±化疗
6. PD-L1低表达：化疗±免疫治疗

### 泛肿瘤精准治疗

**组织不可知（Tissue-Agnostic）治疗：**
- MSI-H/dMMR：帕博利珠单抗
- NTRK融合：拉罗替尼/恩曲替尼
- TMB-H：帕博利珠单抗
- RET融合：塞尔帕替尼

## 挑战与展望

### 当前挑战

- 肿瘤异质性与时空演化
- 可用药靶点的有限性
- 耐药机制的复杂性
- 检测标准化与质量控制
- 卫生经济学与可及性

### 未来方向

- 单细胞多组学分析
- 空间转录组学
- AI辅助药物靶点发现
- 液体活检动态监测
- 新型免疫治疗靶点开发
