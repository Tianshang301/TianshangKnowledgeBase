# 学习路径：医学与健康

## 基础阶段（第1-2年）
1. **人体解剖学**
2. **组织学与胚胎学**
3. **生理学**
4. **生物化学与分子生物学**
5. **病理学**
6. **药理学**
7. **免疫学**
8. **医学微生物学与寄生虫学**

## 临床阶段（第3-5年）
| 科目 | 核心内容 |
|------|----------|
| 内科学 | 各系统疾病的诊断与治疗 |
| 外科学 | 手术学、麻醉学 |
| 妇产科学 | 妇科、产科 |
| 儿科学 | 儿童疾病 |
| 神经病学 | 神经系统疾病 |
| 精神病学 | 精神障碍 |

## 推荐资料
- 人民卫生出版社统编教材
- UpToDate 临床决策支持
- NEJM、The Lancet 期刊
