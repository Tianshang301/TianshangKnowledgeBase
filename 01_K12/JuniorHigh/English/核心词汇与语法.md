## 第二部分：初中语法基础

### 一、词法
#### 1. 名词
- **可数名词**：单数（a/an + n.），复数（加 s/es）  - 一般加 -s：book → books
  - s/x/sh/ch 加 -es：box → boxes
  - 辅音+y → 变 y 为 i 加 es：city → cities
  - f/fe → 变 ve 加 s：knife → knives
  - 不规则：man→men, woman→women, child→children, tooth→teeth, foot→feet, mouse→mice
- **不可数名词**：water, milk, bread, rice, news, information, advice, homework
- **名词所有格**：  - 's：Tom's book, children's toys
  - of：the name of the book

#### 2. 代词

| 类别 | 词语 |
|------|------|
| **人称代词（主格）** | I, you, he, she, it, we, they |
| **人称代词（宾格）** | me, you, him, her, it, us, them |
| **形容词性物主代词** | my, your, his, her, its, our, their |
| **名词性物主代词** | mine, yours, his, hers, its, ours, theirs |
| **反身代词** | myself, yourself, himself, herself, itself, ourselves, themselves |
| **指示代词** | this, that, these, those |
| **不定代词** | some, any, no, every, someone, anyone, everyone, nobody, something, anything, everything, nothing, either, neither, both, all, none, each, many, much, few, little |
| **疑问代词** | who, whom, whose, which, what |

#### 3. 形容词与副词
**比较级和最高级**
- 单音节加 -er/-est（tall → taller → tallest）
- 以 y 结尾：变 y 为 i 加 -er/-est（happy → happier → happiest）
- 多音节：加 more/most（beautiful → more beautiful → most beautiful）
- 不规则：good/well → better → best；bad/badly → worse → worst；many/much → more → most；little → less → least；far → farther/further → farthest/furthest

#### 4. 动词的时态
| 时态 | 构成 | 用法 | 时间状语 |
|------|------|------|--------|
| **一般现在时** | do/does | 习惯/事实/状态 | always, usually, often, sometimes, every day |
| **一般过去时** | did | 过去的动作/状态 | yesterday, last..., ago, in 2010 |
| **一般将来时** | will do / be going to do | 将来的动作/计划 | tomorrow, next..., in the future |
| **现在进行时** | am/is/are doing | 现在正在发生 | now, at the moment, Listen! |
| **过去进行时** | was/were doing | 过去某时正在发生 | at that time, at 8 last night |
| **现在完成时** | have/has done | 已完成/持续至今 | already, yet, ever, never, since, for |

#### 5. 情态动词
- **can/could**（能，可以）
- **may/might**（可以，可能）
- **must**（必须）/ **mustn't**（禁止）
- **need**（需要）/ **needn't**（不必）
- **should**（应该）
- **had better**（最好）
- **would like**（想要）

#### 6. 介词

**时间介词**
- at（at 6 o'clock, at night, at Christmas）
- on（on Monday, on May 1st, on the morning of）
- in（in the morning, in May, in 2020, in summer）
- since（since 2019），for（for two hours）
**地点介词**
- at（at school, at home），in（in Beijing, in the room）
- on（on the desk, on the street），under，behind，next to，between，among，in front of

**方式介词**：by（by bus），with（with a pen），in（in English）

### 二、句法
#### 1. 五大基本句型

| 句型 | 结构 | 例句 |
|------|------|------|
| **主 + 谓** | S + V | The sun rises. |
| **主 + 谓 + 宾** | S + V + O | I like apples. |
| **主 + 系 + 表** | S + V + P | She is a teacher. |
| **主 + 谓 + 间宾 + 直宾** | S + V + IO + DO | He gave me a book. |
| **主 + 谓 + 宾 + 宾补** | S + V + O + C | She made me happy. |

**系动词**：be, become, look, sound, smell, taste, feel, seem, keep, stay, turn, get, grow

#### 2. 宾语从句
- 陈述句语序（主谓顺序不变）
- 引导词：
  - 陈述句用 that（可省略）  - 一般疑问句用 if/whether
  - 特殊疑问句保留疑问词
- 时态：主句过去→从句过去；主句现在→从句任意；真理用一般现在
#### 3. 状语从句
| 类型 | 引导词 |
|------|--------|
| **时间** | when, while, as, before, after, until, as soon as, since |
| **条件** | if, unless, as long as |
| **原因** | because, since, as |
| **结果** | so...that, such...that |
| **目的** | so that, in order that |
| **让步** | although, though, even though |
| **比较** | than, as...as, not as/so...as |

#### 4. 定语从句
- 先行词（修饰的名词/代词）+ 关系词 + 从句
- 关系代词：who（人，主格），whom（人，宾格），whose（人/物，所有格），which（物），that（人/物）
- 关系副词：when（时间），where（地点），why（原因）

#### 5. 被动语态
- be + 过去分词
- 时态通过 be 动词体现
  - 一般现在：am/is/are + done
  - 一般过去：was/were + done
  - 一般将来：will be + done
  - 现在完成：have/has been + done
  - 含情态动词：can/must be + done

#### 6. 非谓语动词
**to do**（不定式）：
- 作主语：To learn English is important.
- 作宾语：I want to go.
- 作目的状语：I came to see you.

**doing**（动名词）：
- 作主语：Swimming is fun.
- 作宾语：I enjoy reading.
- 介词后：Thank you for helping me.

**do/doing/done**（分词）
- 现在分词表示主动/进行：a sleeping baby
- 过去分词表示被动/完成：a broken cup

#### 7. There be 句型
- There is/are + n. + 地点
- 就近原则：There is a book and two pens.
- 时态变化：There was/were, There will be, There have/has been

#### 8. 祈使句与感叹句
- **祈使句**：动词原形开头（Read the book.），否定句前加 Don't（Don't run.）
- **感叹句**：  - What + a/an + adj + n + (主+谓)!
  - How + adj/adv + (主+谓)!
