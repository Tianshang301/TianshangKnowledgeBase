## 答案解析

### 1
（1）核外电子数 = 质子数 = 8
（2）相对原子质量 ≈ 质子数 + 中子数 = 8+8 = 16
（3）一个氧原子有8个电子，一个氧分子有2个氧原子，电子总数 = 8×2 = 16

### 2
（1）H：氢元素 / 一个氢原子
（2）2H：两个氢原子
（3）H₂：氢气 / 一个氢分子（由两个氢原子构成）
（4）2H₂：两个氢分子
### 3
（1）H₂O
（2）CO₂（3）NaCl
（4）NaOH
（5）H₂SO₄（6）CaCO₃
### 4
（1）KMnO₄：K⁺¹Mn⁺⁷O₄⁻²
（2）H₂SO₄：H₂⁺¹S⁺⁶O₄⁻²
（3）NH₄NO₃：N⁻³H₄⁺¹N⁺⁵O₃⁻²
（4）KClO₃：K⁺¹Cl⁺⁵O₃⁻²

### 5
（1）3Fe + 2O₂ → Fe₃O₄（2）2C₂H₂ + 5O₂ → 4CO₂ + 2H₂O
（3）2Al + 3H₂SO₄ → Al₂(SO₄)₃ + 3H₂↑

### 6
（1）C + O₂ →点燃 CO₂（2）Fe + CuSO4 → FeSO4 + Cu
（3）HCl + NaOH → NaCl + H₂O
（4）2KMnO₄ →△ K₂MnO₄ + MnO₂ + O₂↑

### 7
（1）H₂O：1×2+16 = 18
（2）CaCO₃：40+12+16×3 = 100
（3）C₂H₅OH：12×2+1×6+16 = 46

### 8
（1）SO₂相对分子质量 = 32+32 = 64
S% = 32/64×100% = 50%
（2）设需要SO₂质量为x，x×50%=16g，x=32g

### 9
H₂O₂质量 = 68×6% = 4.08g
2H₂O₂ →MnO₂ 2H₂O + O₂↑
2×34 = 68 —— 32
4.08 —— x
x = 4.08×32/68 = 1.92g

### 10
Zn + H₂SO₄ → ZnSO4 + H₂↑
65 —— 2
6.5 —— x
x = 6.5×2/65 = 0.2g

### 11
氮气（N₂），约占空气体积的 78%

### 12
（1）红磷量不足（氧气未耗尽）（2）装置漏气（3）未冷却到室温就打开止水夹
### 13
用燃着的木条分别伸入三瓶气体中。
- 木条燃烧更旺 → 氧气
- 木条熄灭 → 二氧化碳
- 木条正常燃烧 → 空气

### 14
金刚石和石墨的碳原子排列方式（原子结构）不同。金刚石中碳原子呈空间网状结构，硬度大；石墨中碳原子呈层状结构，层间容易滑动，所以质软。
### 15
药品：大理石（或石灰石）和稀盐酸
CaCO₃ + 2HCl → CaCl₂ + CO₂↑ + H₂O

### 16
- CO：可燃性（能燃烧）或还原性（能还原金属氧化物）
- CO₂：能使澄清石灰水变浑浊 / 不燃烧也不支持燃烧
### 17
溶质质量分数 = 20/(20+80)×100% = 20%

### 18
NaOH质量为：100×10% = 10g
水的质量为：100-10 = 90g

### 19
稀释前后溶质质量不变：
100×20% = (100+x)×5%
20 = 5+0.05x
x = 300g

### 20
pH=3 < 7，呈酸性。紫色石蕊试液遇酸变红。
### 21
（1）Fe₂O₃ + 6HCl → 2FeCl₃ + 3H₂O
（2）Ca(OH)₂ + Na₂CO₃ → CaCO₃↓ + 2NaOH
（3）AgNO₃ + NaCl → AgCl↓ + NaNO₃
### 22
各取少量溶液于试管中，滴加稀盐酸（或稀硫酸）。
- 有气泡产生 → Na₂CO₃（CO₃²⁻ + 2H⁺ → CO₂↑ + H₂O）
- 无气泡 → NaCl

### 23
铁钉表面覆盖一层红色物质（铜），溶液由蓝色变为浅绿色。Fe + CuSO4 → FeSO4 + Cu

### 24
Cu 不能与稀盐酸反应。因为 Cu 在金属活动性顺序中位于 H 之后。
### 25
Fe₂O₃ + 3CO →高温 2Fe + 3CO₂
### 26
（1）防止冷凝水回流到热的试管底部，炸裂试管
（2）防止高锰酸钾粉末进入导管（3）防止水倒吸进入试管，使试管炸裂

### 27
常见错误：
- 没有用玻璃棒引流
- 漏斗下端没有紧靠烧杯壁
- 液面高于滤纸边缘
- 滤纸边缘高于漏斗边缘

### 28
左码右物时，氯化钠实际质量 = 砝码质量 - 游码质量
实际称得的氯化钠少于所需质量，导致溶液质量分数偏小。

### 29
石油、不可再生
### 30
C2H5OH + 3O2 →点燃 2CO2 + 3H2O
### 31
A：稀盐酸（使石蕊变红），B：氢氧化钠溶液（使石蕊变蓝），C：碳酸钠溶液（与盐酸反应产生气泡），D：氯化钠溶液（中性，与盐酸不反应）
### 32
一定含有CaCO3和BaCl2（产生不溶于水的沉淀且部分溶解产生气泡），一定不含有Na2SO4（若有则沉淀全部不溶），可能含有NaCl。
### 33
Zn + H2SO4 → ZnSO4 + H2↑
65 — 2
13g — x
x = 13×2/65 = 0.4g
### 34
CaCO3 + 2HCl → CaCl2 + CO2↑ + H2O
100 — 44
x — 4.4g
x = 4.4×100/44 = 10g
质量分数 = 10/12.5×100% = 80%
### 35
分子在不断运动。酒精分子扩散到空气中，达到一定浓度就能被人闻到。距离越远，扩散所需时间越长。
### 36
铁生锈是铁与空气中的氧气和水共同作用的结果。防锈方法：①涂油漆；②镀锌等保护层；③保持干燥；④制成不锈钢合金。
### 37
将浓硫酸沿烧杯壁缓慢注入水中，同时用玻璃棒不断搅拌。因为浓硫酸稀释时放出大量热，且密度比水大，若将水注入浓硫酸，水浮在表面沸腾飞溅，可能造成危险。
### 38
洗洁精具有乳化作用，能将油污乳化成小液滴，随水流冲走，而不是溶解油污。

### 39
Mr(H2O) = 1×2 + 16 = 18
### 40
（1）核电荷数 = 质子数 = 12
（2）核外电子数 = 质子数 = 12
（3）相对原子质量 ≈ 质子数 + 中子数 = 12 + 12 = 24
（4）该元素是镁，元素符号为Mg
### 41
正确操作：左物右码，物体质量 = 砝码 + 游码
左码右物时：砝码 = 物体 + 游码
物体 = 砝码 - 游码 = 5 - 0.6 = 4.4g
实际称得食盐4.4g
### 42
当导管口有气泡连续均匀地冒出时开始收集。因为刚开始排出的是试管中的空气，等空气排净后再收集才能得到较纯净的氧气。
### 43
Zn + H2SO4 → ZnSO4 + H2↑
98 — 2
x — 0.1g
x = 0.1×98/2 = 4.9g
稀硫酸中H2SO4质量分数 = 4.9/20×100% = 24.5%
### 44
方法一（计算法）：
Mg + 2HCl → MgCl2 + H2↑  24—2，10g Mg产生H2质量 = 10×2/24 ≈ 0.833g
2Al + 6HCl → 2AlCl3 + 3H2↑  54—6，10g Al产生H2质量 = 10×6/54 ≈ 1.111g
Fe + 2HCl → FeCl2 + H2↑  56—2，10g Fe产生H2质量 = 10×2/56 ≈ 0.357g
∴铝产生的氢气最多。
方法二（等质量金属产氢比较）：每产生1g H2所需金属质量：Mg需12g，Al需9g，Fe需28g。同样质量下，Al产氢最多。
