# 化学键
## 概述

化学键是原子之间相互结合的强烈作用力。本文档介绍离子键、共价键、化学式的书写规则、化合价的概念以及化学方程式的书写与配平。
## 离子键
### 定义
阴、阳离子之间通过静电作用形成的化学键。
### 形成条件
- 通常形成于金属元素和非金属元素之间
- 活泼金属容易失去电子形成阳离子
- 活泼非金属容易得到电子形成阴离子

### 示例
```
氯化钠（NaCl）的形成：
Na → Na⁺ + e⁻（失去电子）
Cl + e⁻ → Cl⁻（得到电子）
Na⁺ + Cl⁻ → NaCl（静电吸引）
```

### 离子化合物
- 由离子键构成的化合物
- 通常含有金属元素和非金属元素
- 熔融状态或水溶液能导电

## 共价键
### 定义
原子之间通过共用电子对形成的化学键。
### 形成条件
- 通常形成于非金属元素之间
- 原子之间通过共用电子达到稳定结构

### 示例
```
水分子（H₂O）的形成：
H· + ·O· + ·H → H:O:H
两个H原子各提供一个电子，与O原子的两个电子形成共用电子对
```

### 共价化合物
- 由共价键构成的化合物
- 通常只含有非金属元素
- 熔融状态不导电

## 化学式的书写规则

### 离子化合物化学式
1. 正价元素（金属）在前，负价元素（非金属）在后
2. 原子个数比为最简整数比3. 示例：   - 氯化钠：NaCl
   - 氧化镁：MgO
   - 氯化钙：CaCl₂
### 共价化合物化学式
1. 一般正价元素在前，负价元素在后
2. 原子个数符合实际组成
3. 示例：   - 水：H₂O
   - 二氧化碳：CO₂   - 氨气：NH₃
## 化合价的概念与应用
### 定义
元素的原子在形成化合物时表现出的一种性质，表示原子间相互化合的数目。
### 常见化合价
| 元素 | 化合价 | 元素 | 化合价 |
|------|--------|------|--------|
| H | +1 | O | -2 |
| Na | +1 | Cl | -1 |
| Ca | +2 | S | -2 |
| Al | +3 | N | -3 |

### 化合价规则1. 化合物中各元素正负化合价代数和为02. 单质中元素化合价为零

### 应用
```
已知Al为+3价，O为-2价，求氧化铝的化学式

解：
设化学式为Al₂O₃
(+3)×2 + (-2)×3 = 0
6 - 6 = 0 √
答：氧化铝的化学式为Al₂O₃
```

## 化学方程式的书写与配平
### 书写步骤
1. **写**：写出反应物和生成物的化学式
2. **配**：配平化学方程式
3. **注**：注明反应条件和生成物状态
### 配平方法

#### 最小公倍数法
```
配平：P + O₂ → P₂O₅
步骤：1. 找出O原子个数的最小公倍数：10
2. 在O₂前配5，在P₂O₅前配2
3. 配平P原子：P前配4

结果：4P + 5O₂ → 2P₂O₅
```

#### 观察法
```
配平：Fe₂O₃ + CO → Fe + CO₂
步骤：1. 观察Fe：Fe₂O₃前配1，Fe前配2
2. 观察O：Fe₂O₃中有3个O，CO需3个O
3. 观察C：CO前配3，CO₂前配3

结果：Fe₂O₃ + 3CO → 2Fe + 3CO₂
```

### 反应条件
| 条件 | 符号 |
|------|------|
| 点燃 | 点燃 |
| 加热 | △ |
| 催化 | 催化 |
| 高温 | 高温 |

### 生成物状态
- **气体**：用↑表示
- **沉淀**：用↓表示
- **液体和固体反应物**：反应物不用标状态
### 示例
```
铁在氧气中燃烧：
3Fe + 2O₂ → Fe₃O₄
碳酸钙高温分解：
CaCO₃ → CaO + CO₂↑
```
