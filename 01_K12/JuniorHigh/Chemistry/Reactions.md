# 化学反应

## 概述

化学反应是物质发生变化的过程。本文档介绍化学反应类型、质量守恒定律、化学方程式的计算、金属活动性顺序以及酸碱盐的性质。
## 化学反应类型

### 化合反应
- **定义**：由两种或两种以上物质生成一种物质的反应
- **特点**：多变一
- **示例**：
  ```
  2H₂ + O₂ → 2H₂O
  C + O₂ → CO₂
  ```

### 分解反应
- **定义**：由一种物质生成两种或两种以上物质的反应
- **特点**：一变多
- **示例**：
  ```
  2H₂O → 2H₂↑ + O₂↑
  CaCO₃ → CaO + CO₂↑
  ```

### 置换反应
- **定义**：一种单质和一种化合物反应，生成另一种单质和另一种化合物的反应
- **特点**：单质 + 化合物 → 新单质 + 新化合物
- **示例**：
  ```
  Fe + CuSO4 → FeSO4 + Cu
  Zn + H₂SO₄ → ZnSO4 + H₂↑
  ```

### 复分解反应
- **定义**：两种化合物互相交换成分，生成另外两种化合物的反应
- **特点**：化合物 + 化合物 → 新化合物 + 新化合物
- **发生条件**：生成物中有水、气体或沉淀
- **示例**：
  ```
  HCl + NaOH → NaCl + H₂O
  Na₂CO₃ + CaCl₂ → CaCO₃↓ + 2NaCl
  ```

## 质量守恒定律

### 内容
参加化学反应的各物质的质量总和，等于反应后生成的各物质的质量总和。
### 微观解释
- 原子种类不变
- 原子数目不变
- 原子质量不变

### 宏观解释
- 元素种类不变
- 元素质量不变
- 物质总质量不变
### 应用
```
镁在空气中燃烧，生成氧化镁，参加反应的镁的质量为6g，生成氧化镁的质量为10g，求参加反应的氧气的质量。
解：
根据质量守恒定律：参加反应的镁的质量 + 参加反应的氧气的质量 = 生成氧化镁的质量
6g + 氧气的质量 = 10g
氧气的质量 = 4g

答：参加反应的氧气的质量为4g。
```

## 化学方程式的计算

### 计算步骤
1. **设**：写出配平的化学方程式
2. **找**：找出已知量和未知量
3. **设**：设未知量为 x
4. **列**：根据化学方程式列出比例式
5. **解**：求解
6. **答**：写出答案
### 示例
```
实验室用过氧化氢制取氧气，如果要制取3.2g 氧气，需要过氧化氢的质量是多少？

解：
2H₂O₂ → 2H₂O + O₂↑
68                    32
x                     3.2g

68/32 = x/3.2g
x = 6.8g

答：需要过氧化氢6.8g。
```

## 金属活动性顺序表

### 顺序
K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au

### 规律
1. 位于前面的金属能把位于后面的金属从其盐溶液中置换出来
2. 位于氢前面的金属能置换出酸中的氢

### 应用
```
判断下列反应能否发生：
1. Fe + CuSO₄ → ?  （能，Fe 在 Cu 前面）
2. Cu + FeSO₄ → ?  （不能，Cu 在 Fe 后面）
3. Zn + HCl → ?    （能，Zn 在 H 前面）
4. Cu + HCl → ?    （不能，Cu 在 H 后面）
```

## 酸碱盐的性质

### 酸的性质
- **通性**：  1. 与指示剂反应（使紫色石蕊变红）  2. 与活泼金属反应生成盐和氢气  3. 与金属氧化物反应生成盐和水  4. 与碱反应生成盐和水  5. 与某些盐反应

- **常见酸**：  - 盐酸（HCl）  - 硫酸（H₂SO₄）
  - 硝酸（HNO₃）

### 碱的性质
- **通性**：  1. 与指示剂反应（使紫色石蕊变蓝）  2. 与非金属氧化物反应生成盐和水
  3. 与酸反应生成盐和水  4. 与某些盐反应

- **常见碱**：  - 氢氧化钠（NaOH）  - 氢氧化钙（Ca(OH)₂）

### 盐的性质
- **通性**：  1. 与金属反应  2. 与酸反应
  3. 与碱反应
  4. 与另一种盐反应

- **常见盐**：  - 氯化钠（NaCl）  - 碳酸钠（Na₂CO₃）
  - 碳酸钙（CaCO₃）
