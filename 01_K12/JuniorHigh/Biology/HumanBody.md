# 人体

## 概述

人体是由多个系统组成的复杂有机整体。本文档介绍消化系统、呼吸系统、循环系统、泌尿系统、神经系统和内分泌系统。
## 消化系统

### 消化器官
| 器官 | 功能 |
|------|------|
| 口腔 | 物理性消化（咀嚼）和化学性消化（唾液淀粉酶） |
| 咽 | 食物和空气的共同通道 |
| 食道 | 输送食物 |
| 胃 | 暂存食物，初步消化蛋白质 |
| 小肠 | 消化和吸收的主要场所 |
| 大肠 | 吸收水分和无机盐，形成粪便 |
| 肛门 | 排出粪便 |

### 消化腺
| 消化腺 | 分泌液 | 功能 |
|--------|--------|------|
| 唾液腺 | 唾液 | 含唾液淀粉酶 |
| 胃腺 | 胃液 | 含胃蛋白酶 |
| 肠腺 | 肠液 | 含多种消化酶 |
| 肝脏 | 胆汁 | 乳化脂肪（不含酶） |
| 胰腺 | 胰液 | 含多种消化酶 |

## 呼吸系统

### 呼吸器官
| 器官 | 功能 |
|------|------|
| 鼻 | 温暖、湿润、清洁空气 |
| 咽 | 气体通道 |
| 喉 | 气体通道，发声 |
| 气管 | 气体通道 |
| 支气管 | 气体通道 |

### 肺
- **位置**：胸腔内
- **结构**：由大量肺泡组成
- **功能**：气体交换的场所

### 肺泡的特点
- 数量多（约3亿个）
- 肺泡壁薄（一层细胞）
- 肺泡外有毛细血管
- 肺泡外有弹性纤维
## 循环系统

### 心脏
- **位置**：胸腔中部偏左
- **结构**：四个腔（左心房、左心室、右心房、右心室）
- **功能**：血液循环的动力器官

### 血管
| 类型 | 结构特点 | 功能 |
|------|----------|------|
| 动脉 | 管壁厚，弹性大 | 将血液从心脏输送到全身 |
| 静脉 | 管壁薄，弹性小 | 将血液从全身送回心脏 |
| 毛细血管 | 管壁薄，管径小 | 进行物质交换 |

### 血液
| 成分 | 功能 |
|------|------|
| 红细胞 | 运输氧气和部分二氧化碳 |
| 白细胞 | 防御和保护 |
| 血小板 | 促进止血和加速凝血 |
| 血浆 | 运载血细胞，运输养料和废物 |

### 血液循环
- **体循环**：左心室→主动脉→全身毛细血管→上下腔静脉→右心房
- **肺循环**：右心室→肺动脉→肺部毛细血管→肺静脉→左心房
## 泌尿系统

### 结构
| 器官 | 功能 |
|------|------|
| 肾脏 | 形成尿液 |
| 输尿管 | 输送尿液 |
| 膀胱 | 暂存尿液 |
| 尿道 | 排出尿液 |

### 尿液的形成1. **肾小球的滤过作用**：血液流经肾小球时，除血细胞和大分子蛋白质外，其他物质滤过到肾小囊腔形成原尿
2. **肾小管的重吸收作用**：原尿流经肾小管时，全部葡萄糖、大部分水和部分无机盐被重吸收回血液
## 神经系统

### 中枢神经
| 器官 | 功能 |
|------|------|
| 脑 | 调节人体生理活动的最高级中枢 |
| 脊髓 | 反射和传导功能 |

### 周围神经
| 类型 | 功能 |
|------|------|
| 脑神经 | 感觉和运动功能 |
| 脊神经 | 感觉和运动功能 |

### 反射
- **定义**：通过神经系统对外界或内部刺激作出的规律性反应
- **反射弧**：感受器→传入神经→神经中枢→传出神经→效应器
### 反射类型
| 类型 | 特点 | 示例 |
|------|------|------|
| 非条件反射 | 先天就有，不需学习 | 缩手反射、膝跳反射 |
| 条件反射 | 后天学习形成 | 谈虎色变、望梅止渴 |

## 内分泌系统
### 主要激素
| 激素 | 分泌腺 | 功能 |
|------|--------|------|
| 生长激素 | 垂体 | 促进生长发育 |
| 甲状腺激素 | 甲状腺 | 促进新陈代谢和生长发育 |
| 胰岛素 | 胰岛 | 调节糖代谢 |
| 肾上腺素 | 肾上腺 | 加快心跳，升高血糖 |

### 激素调节特点1. 微量高效
2. 通过体液运输
3. 作用于靶器官、靶细胞
