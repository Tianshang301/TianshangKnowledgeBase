# 细胞生物学基础

## 概述

细胞是生物体结构和功能的基本单位。本文档介绍细胞的基本结构、动物细胞与植物细胞的区别、细胞的分裂、细胞的分化与生长以及显微镜的使用方法。
## 细胞的基本结构
### 细胞膜
- **位置**：细胞的最外层（植物细胞在细胞壁内侧）
- **功能**：  - 保护细胞
  - 控制物质进出
  - 进行细胞间的信息交流

### 细胞质
- **位置**：细胞膜以内、细胞核以外的部分
- **组成**：细胞质基质和各种细胞器
- **功能**：细胞新陈代谢的主要场所

### 细胞核
- **位置**：细胞中央或一侧
- **结构**：  - 核膜：双层膜，有核孔
  - 核仁：与核糖体形成有关  - 染色质：由DNA和蛋白质组成
- **功能**：遗传信息库，细胞代谢和遗传的控制中心
### 其他细胞器
| 细胞器 | 功能 |
|--------|------|
| 线粒体 | 有氧呼吸的主要场所，提供能量 |
| 叶绿体 | 光合作用的场所（植物细胞特有） |
| 核糖体 | 蛋白质合成的场所 |
| 内质网 | 蛋白质加工和运输 |
| 高尔基体 | 蛋白质加工、包装和分泌 |

## 动物细胞与植物细胞的区别

### 相同点
- 都有细胞膜、细胞质、细胞核
- 都有线粒体、核糖体等细胞器

### 不同点
| 结构 | 动物细胞 | 植物细胞 |
|------|----------|----------|
| 细胞壁 | 无 | 有（支持和保护） |
| 叶绿体 | 无 | 有（进行光合作用） |
| 液泡 | 无或小 | 有大液泡（储存物质） |
| 形状 | 不规则 | 较规则 |

## 细胞的分裂
### 有丝分裂
- **特点**：分裂产生的两个子细胞与母细胞染色体数目相同
- **过程**：  1. **间期**：DNA复制，蛋白质合成
  2. **前期**：染色体出现，核膜核仁消失  3. **中期**：染色体排列在细胞中央  4. **后期**：着丝点分裂，姐妹染色单体分开
  5. **末期**：染色体解螺旋，核膜核仁重新出现

### 无丝分裂
- **特点**：分裂过程简单，不出现纺锤丝和染色体
- **示例**：蛙的红细胞

### 分裂的意义
- 使生物体生长
- 使生物体繁殖
- 使生物体更新细胞

## 细胞的分化与生长

### 细胞生长
- **定义**：细胞体积由小变大的过程
- **特点**：细胞不能无限长大
### 细胞分化
- **定义**：在个体发育中，由一个或一种细胞增殖产生的后代，在形态、结构和生理功能上发生稳定性差异的过程
- **特点**：  - 分化后的细胞遗传物质不变
  - 分化是持久的、不可逆的
- **结果**：形成不同的组织

## 显微镜的使用方法

### 显微镜结构
| 部件 | 功能 |
|------|------|
| 目镜 | 放大物像 |
| 物镜 | 放大物像 |
| 载物台 | 放置玻片标本 |
| 遮光器 | 调节光线强弱 |
| 反光镜 | 反射光线 |
| 粗准焦螺旋 | 较大幅度升降镜筒 |
| 细准焦螺旋 | 小幅度升降镜筒 |

### 使用步骤
1. **取镜安放**：右手握镜臂，左手托镜座
2. **对光**：转动转换器，使低倍物镜对准通光孔，调节反光镜3. **观察**：放置玻片标本，先用粗准焦螺旋找到物像，再用细准焦螺旋调清晰

### 注意事项
- 先用低倍镜，再用高倍镜
- 调节粗准焦螺旋时眼睛要看着物镜
- 显微镜成倒像，移动玻片与物像移动方向相反

## 细胞的物质交换

### 细胞膜的选择透过性
- **自由扩散**：O₂、CO₂、水等小分子直接通过细胞膜
- **协助扩散**：需要载体蛋白，但不消耗能量（如葡萄糖进入红细胞）
- **主动运输**：需要载体蛋白，消耗能量，可逆浓度梯度运输（如K⁺、Na⁺）
- **胞吞胞吐**：大分子物质进出细胞的方式，需要消耗能量

### 物质运输方式对比
| 运输方式 | 方向 | 载体 | 能量 | 示例 |
|----------|------|------|------|------|
| 自由扩散 | 高→低浓度 | 不需要 | 不消耗 | O₂、CO₂、水 |
| 协助扩散 | 高→低浓度 | 需要 | 不消耗 | 葡萄糖进红细胞 |
| 主动运输 | 低→高浓度 | 需要 | 消耗ATP | Na⁺、K⁺、Ca²⁺ |
| 胞吞 | 细胞外→内 | 不需要 | 消耗ATP | 白细胞吞噬细菌 |
| 胞吐 | 细胞内→外 | 不需要 | 消耗ATP | 分泌蛋白释放 |

## 细胞的能量转换

### 线粒体——能量工厂
- **结构**：双层膜，内膜向内折叠形成嵴
- **功能**：有氧呼吸的主要场所，将有机物中的化学能转变为ATP中的活跃化学能
- **分布**：代谢旺盛的细胞中线粒体数量多
- **有氧呼吸过程**：
  1. 第一阶段（细胞质基质）：葡萄糖→丙酮酸+少量ATP
  2. 第二阶段（线粒体基质）：丙酮酸→CO₂+[H]+少量ATP
  3. 第三阶段（线粒体内膜）：[H]+O₂→H₂O+大量ATP

### 叶绿体——能量转换站
- **结构**：双层膜，内有基粒（由类囊体堆叠而成）
- **功能**：光合作用的场所，将光能转变为化学能
- **光合作用过程**：
  1. 光反应阶段（类囊体薄膜）：光能→ATP+[H]，释放O₂
  2. 暗反应阶段（叶绿体基质）：CO₂→有机物（卡尔文循环）

## 细胞的类型

### 原核细胞与真核细胞的区别
| 特征 | 原核细胞 | 真核细胞 |
|------|----------|----------|
| 细胞核 | 无成形细胞核（拟核） | 有核膜包被的细胞核 |
| 细胞器 | 只有核糖体 | 有各种细胞器 |
| DNA | 环状，裸露 | 线状，与蛋白质结合形成染色体 |
| 大小 | 较小（1-10μm） | 较大（10-100μm） |
| 代表生物 | 细菌、蓝藻、支原体 | 动物、植物、真菌、原生动物 |

### 重要原核生物
- **大肠杆菌**：肠道内的常见细菌
- **乳酸菌**：用于制作酸奶和泡菜
- **蓝藻（蓝细菌）**：能进行光合作用的原核生物
- **硝化细菌**：能进行化能合成作用

## 细胞的衰老与凋亡

### 细胞衰老的特征
1. 水分减少，体积变小，代谢减慢
2. 多种酶活性降低
3. 色素积累（如老年斑）
4. 呼吸减慢，细胞核增大
5. 细胞膜通透性改变

### 细胞凋亡与坏死的区别
| 特征 | 细胞凋亡 | 细胞坏死 |
|------|----------|----------|
| 性质 | 程序性死亡，主动过程 | 非程序性死亡，被动过程 |
| 原因 | 基因控制的正常生理过程 | 外界不利因素（损伤、缺氧等） |
| 过程 | 细胞皱缩，形成凋亡小体 | 细胞肿胀，破裂 |
| 对机体的影响 | 有利，维持稳态 | 有害，引发炎症 |
| 举例 | 蝌蚪尾巴消失、手指形成 | 烫伤、冻伤 |

## 显微镜使用进阶技巧

### 高倍显微镜使用方法
1. 先用低倍镜找到观察对象（视野范围大、易定位）
2. 将观察对象移至视野中央
3. 转动转换器换高倍镜（不要转动粗准焦螺旋）
4. 调节细准焦螺旋使物像清晰
5. 调节光圈和反光镜，使视野亮度合适

### 临时装片制作
- **擦**：用纱布擦净载玻片和盖玻片
- **滴**：滴一滴清水（或生理盐水）在载玻片中央
- **取**：用镊子撕取或挑取材料
- **展**：将材料放入水滴中展开
- **盖**：用镊子夹起盖玻片，一侧先接触水滴，缓缓放下
- **染**：在盖玻片一侧滴加染液，另一侧用吸水纸吸引
