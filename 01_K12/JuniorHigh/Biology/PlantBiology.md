# 植物生物学
## 概述

植物学是研究植物的科学。本文档介绍植物的分类、种子的结构与萌发、根茎叶的结构与功能、光合作用、蒸腾作用以及植物的生殖方式。
## 植物的分类
### 藻类植物
- **特点**：结构简单，无根茎叶分化
- **生活环境**：大多生活在水中
- **示例**：水绵、海带、紫菜
### 苔藓植物
- **特点**：有茎叶分化，无真正的根（假根）
- **生活环境**：生活在阴湿环境
- **示例**：葫芦藓、地钱
### 蕨类植物
- **特点**：有根茎叶分化，有输导组织
- **生活环境**：生活在阴湿环境
- **繁殖方式**：用孢子繁殖
- **示例**：肾蕨、满江红

### 种子植物
- **特点**：用种子繁殖
- **分类**：  - 裸子植物：种子裸露，无果皮包被（松、柏、杉）  - 被子植物：种子有果皮包被（开花植物）

## 种子的结构与萌发

### 种子的结构
| 结构 | 功能 |
|------|------|
| 种皮 | 保护种子内部结构 |
| 胚 | 新植物的幼体（胚芽、胚轴、胚根、子叶） |
| 胚乳 | 储存营养物质（有些种子没有） |

### 胚的结构
- **胚芽**：发育成茎和叶
- **胚轴**：发育成连接根和茎的部分
- **胚根**：发育成根
- **子叶**：储存或转运营养物质

### 种子萌发条件
- **自身条件**：胚完整、有活力、不处于休眠期
- **外界条件**：适宜的温度、一定的水分、充足的空气

## 根茎叶的结构与功能
### 根的结构
| 部位 | 功能 |
|------|------|
| 根冠 | 保护分生区 |
| 分生区 | 产生新细胞 |
| 伸长区 | 细胞伸长，使根伸长 |
| 成熟区 | 吸收水分和无机盐（有根毛） |

### 茎的结构
| 部位 | 功能 |
|------|------|
| 表皮 | 保护作用 |
| 木质部 | 输导水分和无机盐（向上运输） |
| 韧皮部 | 输导有机物（向下运输） |
| 形成层 | 使茎加粗（双子叶植物特有） |

### 叶的结构
| 部位 | 功能 |
|------|------|
| 表皮 | 保护作用（有气孔） |
| 叶肉 | 进行光合作用（有叶绿体） |
| 叶脉 | 输导和支持作用 |

## 光合作用

### 公式
```
二氧化碳 + 水 → 有机物（储存能量）+ 氧气
        光能
        叶绿体
```

### 条件
- **原料**：二氧化碳、水
- **产物**：有机物（主要是淀粉）、氧气
- **场所**：叶绿体
- **条件**：光

### 影响因素
- 光照强度
- 二氧化碳浓度
- 温度
- 水分

### 应用
- 合理密植
- 延长光照时间
- 增加二氧化碳浓度

## 蒸腾作用

### 定义
水分以气体状态从植物体内散失到体外的过程。
### 主要器官
- 叶（主要通过气孔进行）
### 意义
1. 降低叶片温度
2. 促进水分和无机盐的运输3. 促进根对水分的吸收
### 影响因素
- 光照强度
- 环境温度
- 空气湿度
- 空气流动状况

## 植物的有性生殖与无性生殖
### 有性生殖
- **定义**：由两性生殖细胞结合形成受精卵，再由受精卵发育成新个体
- **特点**：具有双亲的遗传特性，生活力强
- **过程**：开花 → 传粉 → 受精 → 结果

### 无性生殖
- **定义**：不经过两性生殖细胞的结合，由母体直接产生新个体
- **类型**：  - 分裂生殖（藻类、苔藓）
  - 出芽生殖（酵母菌）  - 孢子生殖（蕨类）
  - 营养生殖（马铃薯、草莓）
- **特点**：保持母体的优良性状，繁殖速度快
