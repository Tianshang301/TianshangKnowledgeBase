# 统计与概率
## 初中统计与概率概述
统计与概率是初中数学的重要内容，培养数据分析和随机思维的能力。中考占比约10%-15%。

---

## 第一部分：统计
### 一、数据的收集

**调查方式**
- **普查**：对所有对象进行调查（如人口普查）
  - 优点：结果准确  - 缺点：耗费人力物力，有时无法进行
- **抽样调查**：从总体中抽取样本进行调查  - 优点：省时省力  - 缺点：结果有误差

**抽样方法**
- **简单随机抽样**：每个个体被抽中的概率相等
- **分层抽样**：按某种特征分成几层，每层中抽样
- **系统抽样**：按一定间隔抽样
**概念**
| 概念 | 含义 |
|------|------|
| 总体 | 所要考察对象的全体 |
| 个体 | 总体中的每一个考察对象 |
| 样本 | 从总体中抽取的一部分个体 |
| 样本容量 | 样本中个体的数目 |

### 二、数据的整理与描述
**统计图**
| 类型 | 特点 | 适用场景 |
|------|------|---------|
| **条形统计图** | 直观显示各组数量 | 比较不同类别的数量 |
| **折线统计图** | 显示变化趋势 | 随时间变化的数据 |
| **扇形统计图** | 显示各部分占比例 | 比例分配关系 |
| **直方图** | 显示数据分布 | 连续数据的分布情况 |

**制作扇形统计图步骤**：1. 计算各部分占总体的百分比
2. 计算各部分对应圆心角度数（百分比×360°）3. 画扇形，标注名称和百分比

### 三、数据的分析

**平均数**
- 算术平均数：x̄ = (x₁ + x₂ + ... + xₙ)/n
- 加权平均数：x̄ = (x₁f₁ + x₂f₂ + ... + xₖfₖ)/n

**中位数**
- 将数据从小到大排序
- n为奇数：最中间的数
- n为偶数：中间两个数的平均数
**众数**：出现次数最多的数据（可能没有，也可能不止一个）

**极差** = 最大值 - 最小值
**方差**（衡量数据波动大小）：
```
s² = [(x₁-x̄)² + (x₂-x̄)² + ... + (xₙ-x̄)²]/n
```
- 方差越大，数据波动越大、越不稳定
- 方差越小，数据波动越小、越稳定

**标准差**：方差的算术平方根（与原数据单位一致）

**用样本估计总体**
- 用样本平均数估计总体平均数
- 用样本方差估计总体方差

**示例**：
```
甲、乙两人射击成绩如下（环数）：
甲：7, 8, 8, 9, 8
乙：6, 10, 8, 7, 9

甲的平均数 x̄甲 = (7+8+8+9+8)/5 = 8
乙的平均数 x̄乙 = (6+10+8+7+9)/5 = 8

甲的方差 s₁² = [(7-8)²+(8-8)²+(8-8)²+(9-8)²+(8-8)²]/5 = 0.4
乙的方差 s₂² = [(6-8)²+(10-8)²+(8-8)²+(7-8)²+(9-8)²]/5 = 2

∵s₁² < s₂²，∴甲的成绩更稳定
```

## 第二部分：概率
### 一、事件分类
| 事件类型 | 定义 | 概率 |
|---------|------|------|
| **必然事件** | 一定发生的事件 | P = 1 |
| **不可能事件** | 一定不发生的事件 | P = 0 |
| **随机事件** | 可能发生也可能不发生的事件 | 0 < P < 1 |

### 二、概率的计算

**等可能事件的概率**：
```
P(A) = m/n
```
其中，n为所有等可能结果总数，m为事件A包含的结果数

**示例**：
```
掷一颗均匀骰子，求掷出偶数的概率。
所有结果：1,2,3,4,5,6 共n=6
偶数结果：2,4,6 共m=3
P(偶数) = 3/6 = 1/2
```

### 三、列表法与树状图法
**列表法**（适用于两步试验）：
```
同时掷两枚硬币，求一正一反的概率。
        正      反
正    (正,正) (正,反)
反    (反,正) (反,反)

所有等可能结果：4种
一正一反的结果：(正,反)(反,正) 共2种
P = 2/4 = 1/2
```

**树状图法**（适用于两步及以上试验）：
```
从3个男2个女生中选2人，求恰好一男一女的概率。
第一次   第二次   结果
男₁ → 男₂(男₁男₂)/男₃(男₁男₃)/女₁(男₁女₁)/女₂(男₁女₂)
男₂ → 男₁(男₂男₁)/男₃(男₂男₃)/女₁(男₂女₁)/女₂(男₂女₂)
男₃ → 男₁/男₂/女₁/女₂
女₁ → 男₁/男₂/男₃/女₂
女₂ → 男₁/男₂/男₃/女₁

共有5×4=20种等可能结果
一男一女的结果数：3×2+2×3=12种
P = 12/20 = 3/5
```

### 四、频率与概率的关系
- 在大量重复试验中，事件A发生的频率稳定于某个常数
- 这个常数就是事件A的概率
- 用频率估计概率：试验次数越大，估计越准确

### 五、概率的应用

**游戏公平性**
- 各方获胜概率相等 → 公平
- 各方获胜概率不等 → 不公平
**示例**：
```
甲乙玩转盘游戏，转盘分为面积相等的三个扇形（红、黄、蓝）。
甲说："指针指向红色甲胜，指向蓝色乙胜，指向黄色重转。"
这种游戏规则公平吗？

P(甲胜) = 1/3，P(乙胜) = 1/3
两人获胜概率相等，公平。
```
