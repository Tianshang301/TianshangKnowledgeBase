# 代数入门

## 概述

代数是数学的重要分支，用字母表示数并进行运算。本文档覆盖初中代数核心内容：有理数、实数、整式运算、因式分解、方程与不等式，以及函数初步。
## 一、有理数

### 正数与负数
- **正数**：大于0的数（如 3, 0.5, 2/3）
- **负数**：小于0的数，前面加负号（如 -3, -0.5）
- **0既不是正数也不是负数**

### 数轴
- 规定了原点、正方向和单位长度的直线
- 数轴上右边的数总比左边的数大
### 绝对值
- **定义**：数轴上表示数a的点到原点的距离，记作|a|
- **性质**：  - |a| ≥ 0
  - |a| = a（a≥0时）；|a| = -a（a<0时）
- **几何意义**：|a-b|表示数轴上a与b两点间的距离

### 有理数运算
- **加法法则**：  - 同号相加，取相同符号，绝对值相加  - 异号相加，取绝对值大的符号，大绝对值减小绝对值
- **减法法则**：减去一个数等于加上它的相反数
- **乘法法则**：同号得正，异号得负，绝对值相乘
- **除法法则**：除以一个数等于乘以它的倒数

### 乘方
- aⁿ表示n个a相乘，a为底数，n为指数
- 负数的奇次幂为负，偶次幂为正
- 科学记数法：a×10ⁿ（1≤|a|<10）
## 二、实数
### 平方根
- **定义**：若x²=a（a≥0），x叫a的平方根
- 正数有两个平方根，互为相反数
- 0的平方根是0
- 负数没有平方根
- **算术平方根**：非负的平方根，记作√a

### 立方根
- **定义**：若x³=a，x叫a的立方根，记作³√a
- 正数的立方根为正，负数的立方根为负，0的立方根是0

### 无理数
- 无限不循环小数叫无理数
- 常见无理数：π、√2、√3、e 等
- **实数分类**：  - 实数 = 有理数 + 无理数  - 有理数 = 整数 + 分数

### 实数运算
- 实数的加、减、乘、除、乘方运算与有理数相同
- 实数的混合运算顺序：先乘方、再乘除、最后加减，有括号先算括号内

## 三、整式
### 代数式
- **单项式**：数字与字母的积（如 3x, -2a²b）  - 系数：单项式中的数字因数
  - 次数：所有字母的指数和
- **多项式**：多个单项式的和（如 2x+3, a²-ab+1）  - 项：每个单项式叫多项式的一项  - 次数：次数最高项的次数
- **整式**：单项式和多项式的统称
### 整式运算

#### 整式加减
- **同类项**：所含字母相同且相同字母指数也相同的项
- **合并同类项**：系数相加减，字母和指数不变
- **去括号法则**：  - 括号前是"+"，去括号后各项符号不变  - 括号前是"-"，去括号后各项符号改变
#### 整式乘法
- **同底数幂相乘**：aᵐ·aⁿ = aᵐ⁺ⁿ
- **幂的乘方**：(aᵐ)ⁿ = aᵐⁿ
- **积的乘方**：(ab)ⁿ = aⁿbⁿ
- **同底数幂相除**：aᵐ÷aⁿ = aᵐ⁻ⁿ（a≠0，m>n）
- **零指数幂**：a⁰ = 1（a≠0）
- **负整数指数幂**：a⁻ⁿ = 1/aⁿ（a≠0）
#### 乘法公式
- **平方差公式**：(a+b)(a-b) = a²-b²
- **完全平方公式**：(a±b)² = a²±2ab+b²
- 示例：
  ```
  (x+3)(x-3) = x²-9
  (2x+1)² = 4x²+4x+1
  ```

#### 整式除法
- 单项式相除：系数相除，同底数幂相除
- 多项式除以单项式：逐项相除再相加
### 因式分解

因式分解是把多项式化为几个整式乘积的形式。
**提取公因式法**
```
ma + mb + mc = m(a + b + c)
示例：6x² + 9x = 3x(2x + 3)
```

**公式法**
```
平方差：a² - b² = (a+b)(a-b)
完全平方：a² ± 2ab + b² = (a±b)²
示例：x² - 9 = (x+3)(x-3)
       x² + 6x + 9 = (x+3)²
```

**十字相乘法**
```
x² + (p+q)x + pq = (x+p)(x+q)
示例：x² + 5x + 6 = (x+2)(x+3)
```

**分组分解法**
```
示例：ax + ay + bx + by = a(x+y) + b(x+y) = (x+y)(a+b)
```

## 四、分式
### 基本概念
- 形如 A/B（B≠0）的式子叫分式，A为分子，B为分母
- **最简分式**：分子分母没有公因式
- **分式有意义**：分母≠0；**分式值为0**：分子=0且分母≠0

### 分式运算
- **加减**：同分母相加减，分母不变分子相加减；异分母先通分
- **乘除**：分子乘分子，分母乘分母；除以一个分式等于乘以它的倒数
- **乘方**：(A/B)ⁿ = Aⁿ/Bⁿ
### 分式方程
- **解法**：  1. 去分母（方程两边同乘最简公分母）
  2. 解整式方程  3. 验根（代入最简公分母，看是否为0）
- **增根**：使最简公分母为0的根，要舍去

## 五、一元一次方程
### 标准形式
- ax + b = 0（a≠0）
### 解法步骤
1. 去分母（两边同乘分母的最小公倍数）2. 去括号3. 移项（含未知数移到左边，常数移到右边）4. 合并同类项5. 系数化1（两边同除以未知数系数）

**示例**：
```
解：2(x-1) = 3x + 4
2x - 2 = 3x + 4
2x - 3x = 4 + 2
-x = 6
x = -6
```

## 六、一元二次方程
### 标准形式
- ax² + bx + c = 0（a≠0）
### 解法

**直接开平方法**
```
x² = 9 → x = ±3
(x-1)² = 4 → x-1 = ±2 → x = 3 或 x = -1
```

**配方法**
```
x² - 4x + 1 = 0
x² - 4x = -1
(x² - 4x + 4) = -1 + 4
(x-2)² = 3
x-2 = ±√3
x = 2 ± √3
```

**公式法**
```
x = [-b ± √(b²-4ac)] / 2a
其中 Δ = b² - 4ac 为判别式
```

**因式分解法**
```
x² - 5x + 6 = 0
(x-2)(x-3) = 0
x = 2 或 x = 3
```

### 判别式
- Δ > 0：两个不等实根
- Δ = 0：两个相等实根
- Δ < 0：无实根（有两个共轭复根）
### 根与系数关系（韦达定理）
- x₁ + x₂ = -b/a
- x₁·x₂ = c/a

## 七、不等式与不等式组
### 基本性质
1. 若a>b，则a±c > b±c
2. 若a>b且c>0，则ac > bc
3. 若a>b且c<0，则ac < bc

### 一元一次不等式
- 解法与方程类似，注意乘除负数时要改变不等号方向
**示例**：
```
2x - 1 > 5
2x > 6
x > 3
```

### 一元一次不等式组
- 解为各不等式解集的交集
- 口诀：同大取大，同小取小，大小小大中间找，大大小小找不到

```
{x > 1
{x < 4
解：1 < x < 4
```

### 一元二次不等式（用二次函数图像解）
```
x² - 5x + 6 > 0
对应方程 x²-5x+6=0 的根：x=2, x=3
二次项系数a=1>0，开口向上，解：x < 2 或 x > 3
```

## 八、二元一次方程组

### 代入法1. 从一方程解出一个未知数
2. 代入另一方程
3. 解一元一次方程4. 回代求另一未知数
### 加减法1. 适当变形使某未知数系数相同或相反
2. 两方程相加或相减消元
3. 解一元一次方程4. 回代求另一未知数
**示例**：
```
2x + 3y = 7  ①
3x - 2y = 4  ②
①×2：4x + 6y = 14  ③
②×3：9x - 6y = 12  ④
③+④：13x = 26
x = 2

代入①：4 + 3y = 7
y = 1
解：x=2, y=1
```

## 九、应用题模型

### 行程问题
```
相遇：甲路程 + 乙路程 = 总路程
追及：速度差 × 时间 = 路程差
```

### 工程问题
```
工作效率 × 工作时间 = 工作总量
通常把总工作量看作1
```

### 利润问题
```
利润 = 售价 - 进价
利润率 = 利润 / 进价 × 100%
```

### 增长率问题
```
增长(下降)率：a(1±x)ⁿ = b
其中a为基数，x为增长率，n为次数，b为结果
```
