# 函数

## 概述

函数是描述变量之间依赖关系的核心数学概念。本文档覆盖初中函数知识：函数的概念与表示、正比例函数、一次函数、反比例函数、二次函数，以及函数的实际应用。
## 一、函数的概念

### 定义
在某个变化过程中，如果有两个变量 x 和 y，并且对于 x 的每一个确定的值，y 都有唯一确定的值与其对应，那么就说**y 是 x 的函数**，**x 是自变量**，**y 是因变量**。
### 三要素1. **定义域**：自变量 x 的取值范围2. **值域**：函数值 y 的取值范围3. **对应法则**：x 与 y 之间的对应关系（即函数表达式）
### 表示方法
- **解析法**：用数学表达式表示，如 y = 2x + 1
- **列表法**：用表格列出 x 与 y 的对应值
- **图像法**：在坐标系中画出函数图像

### 判断是否为函数
- 对于每一个 x，是否有唯一确定的 y 与之对应
- 在图像上：垂直于 x 轴的直线与图像最多只有一个交点（垂线检验法）
## 二、正比例函数

### 定义
- 形如 y = kx（k≠0）的函数叫正比例函数
- k 为比例系数
### 图像与性质
- **图像**：过原点(0,0)的一条直线
- **k>0**：图像在一、三象限，y 随 x 增大而增大
- **k<0**：图像在二、四象限，y 随 x 增大而减小
- |k|越大，直线越陡
**示例**：
```
y = 2x
当 x=0时 y=0；当 x=1时 y=2；当 x=-1时 y=-2
图像是一条过原点且向右上倾斜的直线
```

## 三、一次函数
### 定义
- 形如 y = kx + b（k≠0）的函数叫一次函数
- 当 b=0时为正比例函数（一次函数的特例）
### 图像与性质
- **图像**：一条直线（可由正比例函数图像平移得到）
- **k（斜率）**：  - k>0：直线上升，y 随 x 增大而增大  - k<0：直线下降，y 随 x 增大而减小  - |k|越大，直线越陡
- **b（截距）**：直线与 y 轴交点的纵坐标为 b
- 过点(0,b)和点(-b/k,0)

### 求一次函数解析式（待定系数法）1. 设一般式 y = kx + b
2. 代入两个已知点坐标3. 解方程组求 k、b

**示例**：
```
一次函数图像过 A(1,3)和 B(-1,-1)，求解析式。
设 y = kx + b
代入 A：3 = k + b
代入 B：-1 = -k + b
两式相加：2 = 2b ∴b = 1
代入得：3 = k + 1 ∴k = 2
解析式为：y = 2x + 1
```

## 四、反比例函数

### 定义
- 形如 y = k/x（k≠0）的函数叫反比例函数
- 也可写作 y = k·x⁻¹ 或 xy = k
- **定义域**：x≠0

### 图像与性质
- **图像**：双曲线（两支）
- **k>0**：  - 图像在一、三象限
  - 在每个象限内，y 随 x 增大而减小
- **k<0**：  - 图像在二、四象限
  - 在每个象限内，y 随 x 增大而增大
- 双曲线无限接近坐标轴但永不与坐标轴相交
- 图像关于原点中心对称

### k 的几何意义
- 过双曲线上任意一点 P 作 x 轴、y 轴的垂线，垂足分别为 A、B
- 矩形 OAPB 的面积 = |k|

## 五、二次函数
### 定义
- 形如 y = ax² + bx + c（a≠0）的函数叫二次函数
### 三种表达式
- **一般式**：y = ax² + bx + c
- **顶点式**：y = a(x-h)² + k，顶点(h,k)
- **交点式**：y = a(x-x₁)(x-x₂)，与 x 轴交于(x₁,0)和(x₂,0)

### 图像与性质
- **图像**：抛物线
- **开口方向**：  - a>0：开口向上（有最小值）
  - a<0：开口向下（有最大值）
- **对称轴**：x = -b/(2a)
- **顶点坐标**：(-b/(2a), (4ac-b²)/(4a))

### 配方法（化为顶点式）
```
y = ax² + bx + c
  = a[x² + (b/a)x] + c
  = a[x² + (b/a)x + (b/(2a))²] + c - a·(b/(2a))²
  = a[x + b/(2a)]² + (4ac-b²)/(4a)
```

### 与坐标轴的交点
- 与 y 轴交点：(0, c)
- 与 x 轴交点：解方程 ax²+bx+c=0
  - Δ>0：两个不同交点  - Δ=0：一个交点（顶点在 x 轴上）  - Δ<0：没有交点（图像在 x 轴上方或下方）
### 平移规律
- y = ax² → y = a(x-h)² → y = a(x-h)²+k
- 左加右减（对 x），上加下减（对整体）
**示例**：
```
求 y = x² - 4x + 3 的顶点和与 x 轴交点。
配方法：y = (x-2)² - 1
顶点(2,-1)，对称轴 x=2
a=1>0，开口向上，最小值为-1

与 x 轴交点：x²-4x+3=0
(x-1)(x-3)=0
x=1或 x=3
与 x 轴交于(1,0)和(3,0)
```

## 六、分段函数
### 定义
- 在不同自变量范围内，函数表达式不同
**示例**：
```
y = |x| = 
  { x  (x≥0)
  { -x (x<0)

y =
  { 2x+1  (x<0)
  { x²   (x≥0)
```
分段函数在初中数学中常见于水费、电费、出租车费等实际问题。
## 七、函数图像变换
### 平移
- y = f(x) → y = f(x-h)：向右平移 h 个单位
- y = f(x) → y = f(x)+k：向上平移 k 个单位
### 对称
- y = f(x) → y = -f(x)：关于 x 轴对称
- y = f(x) → y = f(-x)：关于 y 轴对称
- y = f(x) → y = -f(-x)：关于原点对称
## 八、函数的实际应用

### 行程问题
```
题目：汽车以60km/h 速度行驶，写出路程 s(km)与时间 t(h)的函数关系式。
解：s = 60t（t≥0）——正比例函数
```

### 利润最值问题
```
题目：某商品进价40元，售价60元，每天可售100件。每降价1元多卖10件。设降价 x 元，求利润 y 与 x 的函数及最大利润。
解：
单件利润 = (60-x) - 40 = 20-x
销量 = 100 + 10x
y = (20-x)(100+10x) = -10x² + 100x + 2000

对称轴 x = -100/[2×(-10)] = 5
当 x=5时，y 最大 = -10×25 + 100×5 + 2000 = 2250
最大利润2250元，此时售价55元
```

### 面积问题
```
题目：用20m 篱笆围矩形花园，一边靠墙，设靠墙边长为 x m，求面积 S 与 x 的函数及最大面积。
解：设靠墙边长为 x m，另一边为(20-x)/2 m
S = x·(20-x)/2 = -0.5x² + 10x（0<x<20）
对称轴 x = -10/[2×(-0.5)] = 10
当 x=10时，S 最大 = -0.5×100 + 10×10 = 50
最大面积为50m²
```

### 储蓄问题
```
本金 P 元，年利率 r%，存 n 年后本息和：
单利：S = P(1 + nr)
复利：S = P(1+r)ⁿ
```
