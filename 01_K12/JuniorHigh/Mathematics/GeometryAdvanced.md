# 几何进阶

## 概述

几何是研究图形性质的科学。本文档覆盖初中几何核心内容：基本图形与概念、三角形全等与相似、勾股定理、四边形、圆、锐角三角函数、坐标几何与几何变换。
## 一、基本几何概念
### 线与角
- **直线**：没有端点，向两端无限延伸
- **射线**：有一个端点，向一端无限延伸
- **线段**：有两个端点，有限长
- **角**：由两条有公共端点的射线组成

### 相交线与平行线
- **对顶角**相等
- **邻补角**互补（和为180°）
- **平行线判定**：  - 同位角相等，两直线平行  - 内错角相等，两直线平行  - 同旁内角互补，两直线平行
- **平行线性质**：  - 两直线平行，同位角相等  - 两直线平行，内错角相等  - 两直线平行，同旁内角互补

### 垂线
- 两条相交直线成90°时，互相垂直
- 垂线段最短
- 点到直线的距离：垂线段的长度

## 二、三角形

### 三角形基本性质
- 内角和 = 180°
- 外角 = 不相邻的两个内角之和
- 三角形任意两边之和 > 第三边
- 三角形任意两边之差 < 第三边
### 三角形分类
```
按角分：锐角三角形、直角三角形、钝角三角形
按边分：不等边三角形、等腰三角形、等边三角形
```

### 等腰三角形- **性质**：  - 两腰相等
  - 两底角相等  - 三线合一：顶角平分线、底边中线、底边高重合
- **判定**：  - 两边相等
  - 两角相等
  - 三线合一

### 等边三角形- **性质**：三边相等，三角均为60°
- **判定**：  - 三边相等
  - 三角相等
  - 有一个角为60°的等腰三角形

### 直角三角形- **性质**：  - 两锐角互余（和为90°）  - 斜边中线等于斜边一半  - 30°角所对的直角边等于斜边一半
### 三角形的四心
| 名称 | 定义 | 性质 |
|------|------|------|
| 内心 | 三条角平分线交点 | 到三边距离相等（内切圆圆心） |
| 外心 | 三边垂直平分线交点 | 到三个顶点距离相等（外接圆圆心） |
| 重心 | 三条中线交点 | 重心分中线为2:1两部分 |
| 垂心 | 三条高线交点 | 锐角三角形垂心在内部 |

## 三、全等三角形

### 判定方法
| 判定 | 条件 | 说明 |
|------|------|------|
| SSS | 三边对应相等 | 边边边 |
| SAS | 两边及其夹角对应相等 | 边角边 |
| ASA | 两角及其夹边对应相等 | 角边角 |
| AAS | 两角及其中一角的对边对应相等 | 角角边 |
| HL | 直角三角形斜边和一直角边对应相等 | 只在直角三角形中 |

### 性质
- 对应边相等
- 对应角相等
- 周长相等
- 面积相等

**示例**：
```
已知：AB=AC，AD 是 BC 边上的中线，求证：△ABD≌△ACD

证明：∵AD 是 BC 边上的中线，∴BD=CD
在△ABD 和△ACD 中：
  AB=AC（已知）
  BD=CD（已证）
  AD=AD（公共边）
∴△ABD≌△ACD（SSS）
```

## 四、相似三角形

### 判定方法
| 判定 | 条件 |
|------|------|
| AA | 两角对应相等 |
| SAS | 两边对应成比例且夹角相等 |
| SSS | 三边对应成比例 |

### 性质
- 对应边成比例
- 对应角相等
- 周长比 = 相似比
- 面积比 = 相似比的平方
- 对应高、中线、角平分线比 = 相似比
**示例**：
```
△ABC 中，DE∥BC，AD:DB=2:1
求 DE:BC

证明：∵DE∥BC
∴∠ADE=∠ABC，∠AED=∠ACB
∴△ADE∽△ABC（AA）
∴AD:AB = DE:BC
AD:AB = 2:(2+1)
DE:BC = 2:3
```

## 五、勾股定理
### 定理
- 直角三角形两直角边的平方和等于斜边的平方
- a² + b² = c²（c 为斜边）

### 逆定理
- 若三角形三边满足 a² + b² = c²，则该三角形为直角三角形

### 勾股数
- 常见的勾股数：3,4,5；5,12,13；7,24,25；8,15,17；9,40,41
- 勾股数的整数倍也是勾股数：如6,8,10（3,4,5的2倍）

**示例**：
```
直角三角形直角边为3和4，求斜边。c² = 3² + 4² = 9 + 16 = 25
c = √25 = 5
```

### 勾股定理的应用
```
题目：一棵树折断，树尖触地点距树根6m，折断处距地面4m，求树高。
解：设折断处以上部分长 x m
由勾股定理：x² = 4² + 6² = 52
x = √52 = 2√13
树高 = 4 + 2√13 ≈ 11.2m
```

## 六、四边形

### 平行四边形
- **性质**：  - 两组对边分别平行且相等  - 两组对角分别相等
  - 对角线互相平分  - 邻角互补
- **判定**：  - 两组对边分别平行
  - 两组对边分别相等
  - 一组对边平行且相等
  - 对角线互相平分  - 两组对角分别相等

### 矩形
- **性质**：平行四边形所有性质 + 四个角都是直角 + 对角线相等
- **判定**：  - 有一个角是直角的平行四边形  - 对角线相等的平行四边形  - 有三个角是直角的四边形
### 菱形
- **性质**：平行四边形所有性质 + 四条边相等 + 对角线互相垂直 + 每条对角线平分一组对角
- **判定**：  - 有一组邻边相等的平行四边形  - 对角线互相垂直的平行四边形  - 四条边都相等的四边形

### 正方形- **性质**：矩形 + 菱形的所有性质
  - 四条边相等，四个角是直角
  - 对角线相等且互相垂直平分
  - 每条对角线平分一组对角
- **判定**：  - 有一组邻边相等的矩形
  - 有一个角是直角的菱形

### 梯形
- **定义**：只有一组对边平行的四边形
- **等腰梯形**：两腰相等  - 同一底上的两个角相等
  - 对角线相等
- **直角梯形**：有一个角是直角
### 中位线
- **三角形中位线**：平行于第三边且等于第三边一半
- **梯形中位线**：平行于两底且等于两底和的一半
## 七、圆

### 基本概念
- **圆心**、**半径**、**直径**、**弦**、**弧**、**圆心角**、**圆周角**、**扇形**
- 同圆或等圆中，直径是半径的2倍
- 不在同一直线上的三点确定一个圆

### 圆心角与圆周角
- **圆心角**：顶点在圆心的角，等于所对弧的度数
- **圆周角**：顶点在圆上的角，等于所对弧度数的一半
- 同弧所对的圆周角相等
- 直径所对的圆周角是直角

### 垂径定理
- 垂直于弦的直径平分这条弦，并且平分弦所对的两条弧
### 切线
- **判定**：过半径外端且垂直于半径的直线是切线
- **性质**：切线垂直于过切点的半径
- 从圆外一点引圆的两条切线，切线长相等
- 切线长定理：PA² = PC·PD（P 为圆外一点，PA 为切线，PCD 为割线）

### 圆的有关计算
- **周长**：C = 2πr
- **面积**：S = πr²
- **弧长**：l = nπr/180（n 为圆心角度数）
- **扇形面积**：S = nπr²/360 或 S = lr/2

**示例**：
```
⊙O 半径6cm，圆心角60°，求弧长和扇形面积。
弧长 l = 60π×6/180 = 2π ≈ 6.28cm
扇形面积 S = 60π×36/360 = 6π ≈ 18.84cm²
```

### 圆与圆的位置关系
| 位置 | 圆心距 d 与半径 R,r 的关系|
|------|----------------------|
| 外离 | d > R+r |
| 外切 | d = R+r |
| 相交 | |R-r| < d < R+r |
| 内切 | d = |R-r| |
| 内含 | d < |R-r| |

## 八、锐角三角函数
### 定义（直角三角形中）
- sinA = 对边/斜边
- cosA = 邻边/斜边
- tanA = 对边/邻边

### 特殊角的三角函数
| 角度 | sin | cos | tan |
|------|-----|-----|-----|
| 30° | 1/2 | √3/2 | √3/3 |
| 45° | √2/2 | √2/2 | 1 |
| 60° | √3/2 | 1/2 | √3 |

### 解直角三角形
- 已知两边 → 用勾股定理求第三边，用三角函数求角
- 已知一边一角 → 用三角函数求其他边
**示例**：
```
Rt△ABC 中，∠C=90°，∠A=30°，AB=10，求 BC、AC。
BC = AB·sin30° = 10×1/2 = 5
AC = AB·cos30° = 10×√3/2 = 5√3
```

## 九、坐标几何
### 平面直角坐标系
- x 轴（横轴），y 轴（纵轴），原点 O(0,0)
- 象限：第一象限(+,+)、第二象限(-,+)、第三象限(-,-)、第四象限(+,-)

### 距离公式
- 两点距离：d = √[(x₂-x₁)² + (y₂-y₁)²]

### 中点坐标
- 中点坐标：((x₁+x₂)/2, (y₁+y₂)/2)

### 直线方程
- **斜截式**：y = kx + b（k 为斜率，b 为截距）
- **点斜式**：y - y₁ = k(x - x₁)
- **斜率公式**：k = (y₂-y₁)/(x₂-x₁)

## 十、几何变换
### 平移
- 图形沿某方向移动一定距离
- 图形形状大小不变，位置改变
- 对应点连线平行且相等

### 旋转
- 图形绕某点（旋转中心）转动一定角度
- 旋转中心、旋转方向、旋转角三要素
- 性质：对应点到旋转中心距离相等，对应点与中心连线所成角等于旋转角
### 轴对称（反射）
- 图形关于某条直线（对称轴）对称
- 对应点连线被对称轴垂直平分
- 常见对称图形：等腰三角形（1条）、等边三角形（3条）、矩形（2条）、正方形（4条）、圆（无数条）
### 位似
- 图形按一定比例放大或缩小
- 位似中心、位似比（相似比）
- 性质：对应点连线交于位似中心，对应边平行
- 位似是特殊的相似：相似比 = 位似比
### 中心对称
- 旋转180°后与自身重合
- 平行四边形是中心对称图形（对称中心为对角线交点）
