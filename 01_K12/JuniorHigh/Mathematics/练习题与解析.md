## 答案解析

### 1
(-15) + (+9) - (-3) × (-2)
= (-15) + 9 - 6
= -6 - 6
= -12

### 2
(-2)² × 3 - (-4) ÷ 2
= 4 × 3 - (-4) ÷ 2
= 12 - (-2)
= 12 + 2
= 14

### 3
|-7| + (-3)² - √16
= 7 + 9 - 4
= 12

### 4
(-1/2) × (-4) + (-2)³ ÷ 4
= 2 + (-8) ÷ 4
= 2 + (-2)
= 0

### 5
3a²b - 2ab² + a²b + 4ab²
= (3a²b + a²b) + (-2ab² + 4ab²)
= 4a²b + 2ab²

### 6
(2x + 3)(x - 4)
= 2x·x + 2x·(-4) + 3·x + 3·(-4)
= 2x² - 8x + 3x - 12
= 2x² - 5x - 12

### 7
x² - 9 = (x + 3)(x - 3)
（平方差公式：a² - b² = (a + b)(a - b)）
### 8
x² + 6x + 9 = (x + 3)²
（完全平方公式：a² + 2ab + b² = (a + b)²）
### 9
2x² - 8 = 2(x² - 4) = 2(x + 2)(x - 2)
（先提取公因式2，再用平方差公式）
### 10
x² - 5x + 6 = (x - 2)(x - 3)
（十字相乘法：找两数之积为6、之和为-5，即-2和-3）
### 11
\frac{x² - 1}{x + 1} = \frac{(x + 1)(x - 1)}{x + 1} = x - 1 \quad (x \neq -1)

### 12
\frac{2}{x} + \frac{3}{2x} = \frac{4}{2x} + \frac{3}{2x} = \frac{7}{2x} \quad (x \neq 0)

### 13
3x - 7 = 2x + 5
3x - 2x = 5 + 7
x = 12

### 14
\frac{x}{2} - 3 = \frac{x}{3} + 1
两边乘以6：3x - 18 = 2x + 6
3x - 2x = 6 + 18
x = 24

### 15
\begin{cases}
2x + y = 7 \quad ①\\
x - y = 2 \quad ②\end{cases}
①+②得：3x = 9，x = 3
代入②：3 - y = 2，y = 1
∴\begin{cases}x = 3\\y = 1\end{cases}

### 16
x² - 5x + 6 = 0
(x - 2)(x - 3) = 0
x₁ = 2，x₂ = 3

### 17
2x² - 4x - 6 = 0
两边除以2：x² - 2x - 3 = 0
(x - 3)(x + 1) = 0
x₁ = 3，x₂ = -1

### 18
3x - 5 > 2x + 1
3x - 2x > 1 + 5
x > 6

### 19
\begin{cases}
2x - 1 > 3 \quad ①\\
x + 2 \leq 7 \quad ②\end{cases}
由①得：2x > 4，x > 2
由②：x ≤ 5
∴解集为2 < x ≤ 5

### 20
y = 2x - 3
（1）与x轴交点：令y=0，2x-3=0，x=1.5，交点为(1.5, 0)
　　与y轴交点：令x=0，y=-3，交点为(0, -3)
（2）图像：过点(0,-3)和(1.5,0)的直线，斜率为2
（3）y>0时，2x-3>0，x>1.5

### 21
y = \frac{6}{x}
（1）x=2时，y=6/2=3
（2）代入x=3：y=6/3=2，而A点y=2，所以A在函数图像上
（3）当x>0时，y随x的增大而减小（反比例函数性质）
### 22
y = x² - 4x + 3
（1）顶点公式：x = -b/(2a) = 4/2 = 2
　　y = 2² - 4×2 + 3 = 4 - 8 + 3 = -1
　　顶点(2, -1)，对称轴 x = 2
（2）令y=0：x² - 4x + 3 = 0，(x-1)(x-3)=0
　　x₁=1，x₂=3，与x轴交点为(1,0)和(3,0)
（3）令x=0：y=3，与y轴交点为(0,3)

### 23
设涨价x元，售价=(50+x)元，月销量=200-10x件，月利润=(售价-进价)×销量= [(50+x)-40]×(200-10x)
= (10+x)(200-10x)
= 2000 - 100x + 200x - 10x²
= -10x² + 100x + 2000
= -10(x² - 10x) + 2000
= -10(x - 5)² + 2250

当x=5时，利润最大为2250元，即售价定为55元时，月利润最大，为2250元。
### 24
∠C = 180° - 50° - 65° = 65°
∵∠B = ∠C = 65°，∴ 等腰三角形
### 25
斜边 = √(3²+4²) = √(9+16) = √25 = 5cm
面积 = (1/2)×3×4 = 6cm²

### 26
证明：过A作直线DE∥BC
∵DE∥BC，∴ ∠DAB = ∠B（内错角相等），∠EAC = ∠C（内错角相等），又∠DAB + ∠BAC + ∠EAC = 180°（平角定义）
∴∠B + ∠BAC + ∠C = 180°
即三角形内角和为180°

### 27
∵D、E是AB、AC的中点，∴DE是△ABC的中位线
∴DE = (1/2)BC = (1/2)×10 = 5cm

### 28
在△ABC和△DEF中：
∠A = ∠D（已知）
AB = DE（已知）
∠B = ∠E（已知）
∴△ABC ≌ △DEF（ASA全等判定定理）
### 29
∵DE∥BC
∴AD/AB = AE/AC
AD/AB = 2/(2+3) = 2/5
AE = 4cm
∴4/AC = 2/5
AC = 4×5/2 = 10cm

### 30
平行四边形对角相等，邻角互补
∠C = ∠A = 70°
∠B = ∠D = 180° - 70° = 110°

### 31
矩形对角线勾股定理：
AC² = AB² + BC² = 6² + 8² = 36 + 64 = 100
AC = √100 = 10cm

### 32
周长 = 2πr = 2π×5 = 10π ≈ 31.42cm
面积 = πr² = π×5² = 25π ≈ 78.54cm²

### 33
∠ACB = 90°（直径所对的圆周角是直角）
理由：连接OC，OA=OB=OC，则∠ACB = ∠OCA + ∠OCB = (180°-∠AOC)/2 + (180°-∠BOC)/2
= 180° - (∠AOC+∠BOC)/2 = 180° - 180°/2 = 90°

### 34
∠A=30°，AB=10cm
BC = AB×sin30° = 10×(1/2) = 5cm
AC = AB×cos30° = 10×(√3/2) = 5√3 ≈ 8.66cm

### 35
设树高h，tan30° = h/12
h = 12×tan30° = 12×(√3/3) = 4√3 ≈ 4×1.732 = 6.928 ≈ 6.9米
### 36
排序：3, 5, 5, 5, 6, 7, 8, 9
（1）众数：5（出现3次）
（2）中位数：(5+6)/2 = 5.5
（3）平均数：(3+7+5+9+5+8+5+6)/8 = 48/8 = 6

### 37
（1）平均分：
甲 = (85+90+88+92+86+89+91+87+90+88)/10 = 886/10 = 88.6
乙 = (75+95+80+90+85+95+78+92+88+82)/10 = 860/10 = 86

（2）方差（S²）：
甲：各数与88.6的偏差平方和 = (85-88.6)²+(90-88.6)²+...+(88-88.6)²
= 12.96+1.96+0.36+11.56+6.76+0.16+5.76+2.56+1.96+0.36 = 44.4
S²甲 = 44.4/10 = 4.44

乙：各数与86的偏差平方和 = (75-86)²+(95-86)²+...+(82-86)²
= 121+81+36+16+1+81+64+36+4+16 = 456
S²乙 = 456/10 = 45.6

∵S²甲 < S²乙，∴甲组成绩更稳定
### 38
总球数 = 3+2 = 5
（1）P(红球) = 3/5
（2）P(白球) = 2/5

### 39
所有可能结果：(正正), (正反), (反正), (反反)，共4种

（2）一正一反：(正反)(反正)2种，P = 2/4 = 1/2

### 41
(3x²y - 2xy² + xy) - (2x²y - 3xy² + 2xy)
= 3x²y - 2xy² + xy - 2x²y + 3xy² - 2xy
= (3x²y - 2x²y) + (-2xy² + 3xy²) + (xy - 2xy)

### 42

### 43

### 44

### 45

②×2得：4x - 2y = 2  ③
①+③得：7x = 14，x = 2

∴{x = 2, y = 3}

### 46

### 47

= -10x² + 1300x - 30000
= -10(x² - 130x) - 30000

即售价65元时，最大月利润12250元。

### 48

36 - 4k ≥ 0，k ≤ 9

### 49
y = -x² + 2x + 3 = -(x² - 2x) + 3 = -(x - 1)² + 4
（1）顶点坐标(1, 4)
（2）令y=0：-x²+2x+3=0，x²-2x-3=0，(x-3)(x+1)=0
　　x₁=3，x₂=-1，与x轴交点为(3,0)和(-1,0)
（3）a=-1<0，开口向下，当x>1时，y随x增大而减小

### 50
代入A(1,2)：k + b = 2  ①
代入B(-1,0)：-k + b = 0  ②
①+②得：2b = 2，b = 1
代入①得：k = 1
∴解析式为y = x + 1

### 51

同时打开效率 = 1/4 - 1/6 = 3/12 - 2/12 = 1/12

### 59

### 60
设甲的速度为x km/h，则乙的速度为2.5x km/h
甲用时：60/x，乙用时：60/(2.5x) = 24/x
60/x - 24/x = 1
36/x = 1
x = 36
∴甲的速度为36km/h，乙的速度为90km/h
