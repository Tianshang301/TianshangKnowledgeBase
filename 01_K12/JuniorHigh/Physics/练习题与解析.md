# 初中物理练习题与解析

---

## 一、机械运动
**1** 一辆汽车以 15m/s 的速度匀速行驶了 2 分钟，求汽车行驶的路程。
**2** 甲、乙两地相距 120km，一辆汽车从甲地到乙地的平均速度为 60km/h，返回时的平均速度为 40km/h，求全程的平均速度。
**3** 一个物体做匀速直线运动，在 5s 内通过了 20m 的路程，则它在前 3s 内的速度是多少？

---

## 二、声现象

**4** 声音在空气中的传播速度约为 340m/s，某人看到闪电后 5s 才听到雷声，求雷电发生处距该人的距离。
**5** 向一口枯井中喊话，0.6s 后听到回声，求枯井的深度。（声速取 340m/s）

---

## 三、光学
**6** 一束光线与镜面成 30° 角射到平面镜上，求反射角的大小。
**7** 某同学身高 1.7m，站在平面镜前 2m 处，求：
（1）像到镜面的距离
（2）像的高度（3）若他向镜面走近 0.5m，像到他的距离是多少。
**8** 一束光从空气斜射入水中，入射角为 45°，折射角为 32°，则光在水中的传播速度比在空气中快还是慢？

---

## 四、质量与密度

**9** 一块金属的质量为 54g，体积为 20cm³，求该金属的密度，并判断可能是什么金属。（ρ铝=2.7g/cm³，ρ铁=7.9g/cm³，ρ铜=8.9g/cm³）
**10** 一个空瓶子的质量为 200g，装满水后的总质量为 700g，求瓶子的容积。

---

## 五、力与运动
**11** 一个质量为 50kg 的物体静止在水平地面上，求它所受的重力。（g=10N/kg）
**12** 用 20N 的水平力推一个重 100N 的箱子，箱子未被推动，此时摩擦力是多大？用 30N 的水平力推时，箱子做匀速直线运动，摩擦力是多大。
**13** 一物体在月球上的质量是 6kg，求：（1）在地球上受到的重力
（2）在月球上受到的重力（g 月 = g 地/6，g 地=10N/kg）

---

## 六、压强
**14** 一个质量为 60kg 的人，每只脚与地面的接触面积为 200cm²，求他站立时对地面的压强。（g=10N/kg）
**15** 水坝修建为上窄下宽的形状，请用液体压强的知识解释原因。
**16** 潜水员在水下 20m 处，求他受到的水的压强。（ρ水=1×10³kg/m³，g=10N/kg）

---

## 七、浮力
**17** 一个体积为 50cm³ 的物体浸没在水中，求它受到的浮力。（ρ水=1×10³kg/m³，g=10N/kg）
**18** 一个质量为 270g 的铝块（ρ铝=2.7g/cm³），一半体积浸入水中，求所受浮力。（g=10N/kg）

---

## 八、简单机械
**19** 用一根撬棍撬起一块石头，动力臂为 1.2m，阻力臂为 0.3m，若石头重 600N，求至少需要多大的力？

**20** 用动滑轮提升一个重为 200N 的物体（不计滑轮重和摩擦），求所需的拉力。

---

## 九、功和能

**21** 用 50N 的水平推力使重 200N 的物体沿水平地面前进 10m，求推力做的功和重力做的功。
**22** 一台起重机将重 5000N 的货物匀速提升 4m，用了 10s，求起重机的功率。
**23** 一个物体从高 5m 的光滑斜面顶端滑下，求到达斜面底端时的速度。（g=10N/kg，不计摩擦）

---

## 十、电路
**24** 三个电阻 R₁=5Ω，R₂=10Ω，R₃=15Ω 串联在电路中，总电压为 6V，求：（1）电路中的总电阻（2）电路中的电流（3）R₁两端的电压
**25** R₁=10Ω 和 R₂=15Ω 并联在电路中，干路电流为 0.5A，求：（1）总电阻（2）电源电压（3）通过 R₁和 R₂的电流

---

## 十一、机械运动（续）

**26** 一列火车长200m，以72km/h 的速度通过一座长1000m 的大桥，求火车完全通过大桥所需的时间。

**27** 甲、乙两物体做匀速直线运动，速度之比为3:2，时间之比为4:5，求路程之比。

---

## 十二、声现象（续）

**28** 某人站在两座山之间大喊一声，0.4s 后听到第一次回声，1.2s 后听到第二次回声，求两座山之间的距离。（声速取340m/s）

---

## 十三、光学（续）

**29** 一束光从空气射向某透明介质，入射角为60°，折射角为35°，求光在这种介质中的传播速度。（空气中光速 c=3×10⁸m/s，sin60°≈0.866，sin35°≈0.574）

**30** 在"探究凸透镜成像规律"实验中，凸透镜焦距为10cm。当物体距凸透镜25cm 时，成什么性质的像？当物体距凸透镜8cm 时，成什么性质的像？

---

## 十四、质量与密度（续）

**31** 一个铁球的质量为158g，体积为30cm³，请判断该铁球是实心还是空心。（ρ铁=7.9g/cm³）

---

## 十五、力与运动（续）

**32** 一物体受到两个力的作用：F₁=8N 方向向东，F₂=6N 方向向北，求这两个力的合力大小和方向。

**33** 用弹簧测力计水平拉动一个重20N 的物体在水平桌面上做匀速直线运动，测力计示数为5N。求物体受到的滑动摩擦力大小。若拉力增大到8N，滑动摩擦力是否改变？

---

## 十六、压强（续）

**34** 一个底面积为200cm²的圆柱形容器中装有水，水深30cm，求：（1）水对容器底的压强 （2）水对容器底的压力

**35** 用吸管喝饮料时，饮料为什么会被吸上来？请用大气压的知识解释。

---

## 十七、浮力（续）

**36** 一艘轮船满载时排水量为5000t，求：（1）轮船受到的浮力 （2）轮船排开水的体积（g=10N/kg，ρ海水=1.0×10³kg/m³）

---

## 十八、简单机械（续）

**37** 用滑轮组提升重物，每个滑轮重20N，物体重300N。若不计绳重和摩擦，采用最省力绕法，求绳端的拉力。

---

## 十九、功和能（续）

**38** 一辆汽车发动机的功率为60kW，以72km/h 的速度匀速行驶，求汽车的牵引力。

**39** 一个物体从高为10m 的光滑斜面顶端由静止滑下，求到达斜面底端时的速度。（g=10N/kg，不计摩擦）

---

## 二十、电路（续）

**40** 在电路中，R₁=6Ω，R₂=4Ω串联，电源电压为12V。求：（1）电路的总电阻和电流 （2）R₁和 R₂两端的电压

**41** 标有"6V 3W"的小灯泡，求：（1）小灯泡正常发光时的电流和电阻 （2）如果把它接在9V 的电源上，需要串联一个多大的电阻才能让它正常发光？

---

## 二十一、热学

**42** 质量为2kg 的水从20℃升高到100℃，吸收了多少热量？[c 水=4.2×10³J/(kg·℃)]

**43** 将0.5kg 温度为-10℃的冰加热到0℃的水，需要吸收多少热量？[c 冰=2.1×10³J/(kg·℃)，λ冰=3.36×10⁵J/kg]

---

## 二十二、磁学

**44** 如何用安培定则判断通电螺线管的 N、S 极？请简述方法。

## 答案解析

### 1
v = 15m/s，t = 2min = 120s
s = vt = 15×120 = 1800m

### 2
去程时间：t₁ = 120/60 = 2h
回程时间：t₂ = 120/40 = 3h
总路程：s = 120×2 = 240km
总时间：t = 2+3 = 5h
平均速度：v = s/t = 240/5 = 48km/h

### 3
匀速直线运动速度：v = s/t = 20/5 = 4m/s
前3s 速度也是4m/s（匀速直线运动速度不变）
### 4
s = vt = 340×5 = 1700m

### 5
声音传播总路程：s = vt = 340×0.6 = 204m
井深 = s/2 = 102m（声音下去再上来）
### 6
入射光线与镜面成30°，则入射角 = 90°-30° = 60°
反射角 = 入射角 = 60°

### 7
（1）像到镜面距离 = 人到镜面距离 = 2m
（2）像的高度 = 人的高度 = 1.7m
（3）走近0.5m 后，人到镜面1.5m，像到镜面1.5m，像到人的距离 = 3m

### 8
光从空气斜射入水中，折射角小于入射角（32°<45°），说明光速在水中比空气中慢。
### 9
ρ = m/V = 54/20 = 2.7g/cm³
与铝的密度相同，可能是铝。
### 10
水的质量：m 水 = 700-200 = 500g
水的密度：ρ水 = 1g/cm³
容积：V = m 水/ρ水 = 500/1 = 500cm³

### 11
G = mg = 50×10 = 500N

### 12
20N 时未推动，静摩擦力与推力平衡：f = 20N
30N 匀速直线运动，滑动摩擦力与推力平衡：f = 30N

### 13
（1）G 地 = mg = 6×10 = 60N
（2）G 月 = 60/6 = 10N
（质量是物体的属性，不随位置改变，仍然为6kg）
### 14
压力 F = G = mg = 60×10 = 600N
受力面积 S = 2×200cm² = 400cm² = 0.04m²
压强 p = F/S = 600/0.04 = 15000Pa

### 15
液体压强随深度增加而增大。水坝底部受到水的压强比顶部大得多，所以底部要更宽更厚，以承受更大的压强。
### 16
p = ρgh = 1×10³×10×20 = 2×10⁵Pa

### 17
V 排 = V 物 = 50cm³ = 5×10⁻⁵m³
F 浮 = ρ水 gV 排 = 1×10³×10×5×10⁻⁵ = 0.5N

### 18
铝块密度ρ=ρ铝=2.7g/cm³
V 物 = m/ρ = 270/2.7 = 100cm³
V 排 = V 物/2 = 50cm³ = 5×10⁻⁵m³
F 浮 = ρ水 gV 排 = 1×10³×10×5×10⁻⁵ = 0.5N

### 19
F₁L₁ = F₂L₂
F₁×1.2 = 600×0.3
F₁ = 180/1.2 = 150N

### 20
动滑轮省一半力：F = G/2 = 200/2 = 100N

### 21
推力做功：W 推 = Fs = 50×10 = 500J
重力做功：因为重力方向竖直向下，物体水平移动，重力方向上无位移，W 重 = 0J

### 22
W = Fs = Gh = 5000×4 = 20000J
P = W/t = 20000/10 = 2000W = 2kW

### 23
机械能守恒：mgh = ½mv²
v = √(2gh) = √(2×10×5) = √100 = 10m/s

### 24
（1）R 总 = R₁+R₂+R₃ = 5+10+15 = 30Ω
（2）I = U/R 总 = 6/30 = 0.2A
（3）U₁ = IR₁ = 0.2×5 = 1V

### 25
（1）1/R 总 = 1/R₁+1/R₂ = 1/10+1/15 = 3/30+2/30 = 5/30
R 总 = 30/5 = 6Ω
（2）U = IR 总 = 0.5×6 = 3V
（3）I₁ = U/R₁ = 3/10 = 0.3A
I₂ = U/R₂ = 3/15 = 0.2A

### 26
v = 72km/h = 20m/s，s = 200 + 1000 = 1200m
t = s/v = 1200/20 = 60s
### 27
s₁:s₂ = v₁t₁:v₂t₂ = (3×4):(2×5) = 12:10 = 6:5
### 28
声音到近山距离：s₁ = 340×0.4/2 = 68m
声音到远山距离：s₂ = 340×1.2/2 = 204m
两山距离：s = s₁ + s₂ = 68 + 204 = 272m
### 29
折射率 n = sin60°/sin35° = 0.866/0.574 ≈ 1.51
v = c/n = 3×10⁸/1.51 ≈ 1.99×10⁸m/s
### 30
u=25cm > 2f=20cm，成倒立缩小的实像（照相机原理）
u=8cm < f=10cm，成正立放大的虚像（放大镜原理）
### 31
同质量实心铁球体积：V = m/ρ = 158/7.9 = 20cm³
实际体积30cm³ > 20cm³，所以是空心。
### 32
合力 F = √(8²+6²) = √100 = 10N
方向：tanθ = 6/8 = 0.75，θ ≈ 37°，即东偏北37°
### 33
匀速直线运动，摩擦力与拉力平衡：f = F = 5N
滑动摩擦力大小与压力、接触面粗糙程度有关，与拉力大小无关，增大拉力后摩擦力仍为5N，此时物体做加速运动。
### 34
p = ρgh = 1×10³×10×0.3 = 3000Pa
F = pS = 3000×0.02 = 60N
### 35
用吸管喝饮料时，吸管内气压减小，外界大气压将饮料压入吸管。
### 36
F 浮 = G 排 = m 排 g = 5000×10³×10 = 5×10⁷N
V 排 = m 排/ρ = 5000×10³/10³ = 5000m³
### 37
最省力绕法：n=3段绳子承担物重
F = (G 物 + G 动)/n = (300+20)/3 = 320/3 ≈ 106.7N
### 38
v = 72km/h = 20m/s，P = 60kW = 60000W
F = P/v = 60000/20 = 3000N
### 39
机械能守恒：mgh = ½mv²
v = √(2gh) = √(2×10×10) = √200 ≈ 14.14m/s
### 40
（1）R 总 = 6+4 = 10Ω，I = 12/10 = 1.2A
（2）U₁ = IR₁ = 1.2×6 = 7.2V，U₂ = IR₂ = 1.2×4 = 4.8V
### 41
（1）I = P/U = 3/6 = 0.5A，R = U²/P = 36/3 = 12Ω
（2）串联分压：UR = 9-6 = 3V，R 串 = UR/I = 3/0.5 = 6Ω
### 42
Q = cmΔt = 4.2×10³×2×(100-20) = 4.2×10³×2×80 = 6.72×10⁵J
### 43
冰升温吸热：Q₁ = c 冰 mΔt = 2.1×10³×0.5×10 = 1.05×10⁴J
冰熔解吸热：Q₂ = λm = 3.36×10⁵×0.5 = 1.68×10⁵J
总吸热：Q = Q₁+Q₂ = 1.05×10⁴+1.68×10⁵ = 1.785×10⁵J
### 44
用右手握住螺线管，四指指向电流方向，大拇指所指方向即为螺线管的 N 极。
