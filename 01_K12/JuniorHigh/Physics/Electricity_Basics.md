# 电学基础

## 概述

电学是研究电荷运动和电场的物理学分支。本文档介绍电荷与电流、电路组成、串并联电路、欧姆定律、电功与电功率、焦耳定律以及家庭电路。
## 电荷与电流
### 电荷
- **种类**：正电荷、负电荷
- **性质**：同种电荷相互排斥，异种电荷相互吸引
- **单位**：库仑（C）
### 电流
- **定义**：电荷的定向移动形成电流
- **方向**：正电荷定向移动的方向（与电子移动方向相反）
- **单位**：安培（A），1A = 1000mA

### 电压
- **定义**：使电荷定向移动形成电流的原因
- **单位**：伏特（V），1V = 1000mV

### 电阻
- **定义**：导体对电流的阻碍作用
- **单位**：欧姆（Ω），1kΩ = 1000Ω

## 电路组成

### 基本组成
| 组件 | 作用 | 符号 |
|------|------|------|
| 电源 | 提供电能 | - |
| 导线 | 连接电路 | - |
| 开关 | 控制电路通断 | - |
| 用电器 | 消耗电能 | - |

### 电路状态
| 状态 | 说明 |
|------|------|
| 通路 | 电路闭合，有电流通过 |
| 断路 | 电路断开，无电流 |
| 短路 | 电源两极直接相连，电流很大，危险 |

## 串联电路与并联电路
### 串联电路
- **定义**：用电器依次连接
- **特点**：  - 电流处处相等：I = I₁ = I₂  - 总电压等于各部分电压之和：U = U₁ + U₂  - 总电阻等于各电阻之和：R = R₁ + R₂
- **应用**：节日装饰灯、老式圣诞灯
### 并联电路
- **定义**：用电器并列连接
- **特点**：  - 各支路电压相等：U = U₁ = U₂  - 干路电流等于各支路电流之和：I = I₁ + I₂  - 总电阻的倒数等于各电阻倒数之和：1/R = 1/R₁ + 1/R₂
- **应用**：家庭电路、照明电路
### 串并联对比
| 项目 | 串联 | 并联 |
|------|------|------|
| 电流 | 处处相等 | 干路电流等于支路电流之和 |
| 电压 | 总电压等于各部分电压之和 | 各支路电压相等 |
| 电阻 | 总电阻等于各电阻之和 | 总电阻倒数等于各电阻倒数之和 |
| 开关作用 | 控制整个电路 | 只控制所在支路 |

## 欧姆定律

### 内容
导体中的电流跟导体两端的电压成正比，跟导体的电阻成反比。
### 公式
I = U/R

### 应用
- **求电流**：I = U/R
- **求电压**：U = IR
- **求电阻**：R = U/I

### 示例
```
已知：导体两端电压U = 6V，通过导体的电流I = 0.3A
求：导体的电阻R

解：R = U/I = 6V / 0.3A = 20Ω
```

## 电功与电功率

### 电功
- **定义**：电流做的功
- **公式**：W = UIt
- **单位**：焦耳（J），1kWh = 1千瓦时 = 3.6×10⁶J

### 电功率
- **定义**：单位时间内电流做的功
- **公式**：P = W/t = UI
- **单位**：瓦特（W），1kW = 1000W

### 额定功率与实际功率
- **额定功率**：用电器在额定电压下的功率
- **实际功率**：用电器在实际电压下的功率
### 示例
```
一个灯泡标有"220V 40W"
求：灯泡正常工作时的电流和电阻。
解：
额定电流 I = P/U = 40W / 220V ≈ 0.18A
电阻 R = U/I = 220V / 0.18A ≈ 1222Ω
```

## 焦耳定律
### 内容
电流通过导体产生的热量跟电流的平方成正比，跟导体的电阻成正比，跟通电时间成正比。
### 公式
Q = I²Rt

### 应用
- **电热器**：电热水器、电饭煲
- **防止电热**：散热片、风扇
### 示例
```
一个电阻R = 100Ω，通过电流I = 2A，通电时间t = 100s
求：产生的热量Q

解：Q = I²Rt = (2A)² × 100Ω × 100s = 40000J
```

## 家庭电路与安全用电
### 家庭电路组成
- **进户线**：火线和零线
- **电能表**：测量消耗的电能
- **总开关**：控制整个电路
- **保险装置**：保护电路（熔断器、空气开关）
- **用电器**：插座、灯具、电器
### 安全用电
- **原则**：不接触低压带电体，不靠近高压带电体
- **注意事项**：  - 不用湿手触摸电器
  - 不在电线上晾衣服
  - 发现有人触电，先切断电源
  - 安全电压：不高于36V

### 触电急救
1. 切断电源（拉闸、用干燥木棒挑开电线）2. 拨打120急救电话
3. 进行人工呼吸（如需要）
