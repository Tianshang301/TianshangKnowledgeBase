# 通用技术

> 通用技术是高中必修课程，旨在培养学生的技术素养、创新精神和实践能力。根据《普通高中通用技术课程标准（2017年版2020年修订）》编写。

---

## 课程结构

```
通用技术课程
├── 必修（各2学分）
│   ├── 技术与设计1（2学分）

│   │   ├── 工程基础
│   │   ├── 机器人技术
│   │   └── 工程设计基础
│   └── 技术与职业系列
```

## 必修模块

### 技术与设计1

#### 第一章 技术及其性质

**1.1 技术的起源与发展**
- 技术的定义：人类为了满足需求和愿望而对自然进行的改造
- 技术的发展阶段：石器→青铜→铁器→蒸汽→电气→信息→智能
- 技术与科学的区别与联系

**1.2 技术的性质**
- 目的性：技术源于人类的需求
- 创新性：技术的本质在于创新
- 综合性：技术是多种知识的综合
- 两面性：技术既有积极作用也有负面影响
- 专利性：知识产权保护

**1.3 技术与人类**
- 技术对人类生活方式的改变
- 技术对生产方式的影响
- 技术伦理与技术责任

#### 第二章 设计的基础

**2.1 设计的概念**
- 设计：有目的、有计划的创造性活动
- 设计的原则：创新、实用、经济、美观、安全、环保、伦理

**2.2 设计的一般过程**
```
发现与明确问题 → 制定设计方案 → 制作原型或模型 → 测试与评估 → 优化与完善 → 产品发布
```

**2.3 设计思维**
- 同理心：理解用户需求
- 定义问题：明确设计目标
- 构思方案：头脑风暴
- 原型制作：快速迭代
- 测试验证：用户反馈

**2.4 设计的交流**
- 技术语言：图样、图表、符号
- 口头交流与书面表达
- 计算机辅助表达

#### 第三章 图样的绘制与识读

**3.1 三视图**
- 正投影法原理
- 主视图、俯视图、左视图
- 三视图的对应关系（长对正、高平齐、宽相等）
- 简单形体的三视图绘制

**3.2 尺寸标注**
- 尺寸的组成：尺寸线、尺寸界线、尺寸数字
- 标注规则
- 常见的尺寸注法

**3.3 机械制图基础**
- 剖视图
- 零件图
- 装配图

**3.4 计算机辅助设计**
- CAD软件的基本操作
- 二维绘图
- 三维建模入门

#### 第四章 材料与加工

**4.1 常用材料**
- 金属材料：钢铁、铝合金、铜合金
- 非金属材料：木材、塑料、玻璃、陶瓷
- 复合材料：碳纤维、玻璃钢
- 新材料：记忆合金、纳米材料

**4.2 材料的性能**
- 力学性能：强度、硬度、韧性、弹性
- 物理性能：导电性、导热性、密度
- 化学性能：耐腐蚀性、抗氧化性
- 工艺性能：可加工性、可塑性

**4.3 加工工艺**
- 划线：划针、划规、样冲
- 锯割：手锯的使用
- 锉削：锉刀的类型与使用
- 钻孔：台钻操作
- 连接：螺纹连接、焊接、胶接
- 表面处理：打磨、喷漆、电镀

**4.4 安全操作规范**
- 工具设备的正确使用
- 防护装备的佩戴
- 应急处理常识

#### 第五章 结构设计

**5.1 结构的基本概念**
- 结构的定义：承受载荷的物体构架
- 构件：梁、柱、板、壳
- 连接方式：刚接、铰接、滑动

**5.2 受力分析**
- 拉力、压力、剪力、扭矩、弯矩
- 应力与应变
- 材料的许用应力

**5.3 结构稳定性**
- 重心与稳定性
- 支撑面积与稳定性
- 三角形结构的稳定性原理

**5.4 结构设计案例**
- 桥梁结构设计
- 简易承重结构
- 框架结构模型

#### 第六章 流程与系统

**6.1 流程**
- 流程的含义：时间上的先后顺序
- 流程图：框图和工序图
- 流程优化：并行、串联

**6.2 系统**
- 系统的定义：相互关联的要素组成的整体
- 系统的特性：整体性、相关性、目的性、环境适应性
- 系统分析：输入→处理→输出

**6.3 控制**
- 开环控制与闭环控制
- 传感器、控制器、执行器
- 反馈机制
- 简单控制系统的设计

## 技术项目实践案例

### 案例1：创意台灯设计

| 阶段 | 内容 |
|------|------|
| 需求分析 | 确定台灯的使用场景和功能需求 |
| 方案设计 | 绘制草图，选择材料和光源 |
| 原型制作 | 制作台灯模型，连接电路 |
| 测试改进 | 测试照度、稳定性，优化外观 |
| 成果展示 | 制作展板，撰写设计说明书 |

### 案例2：桥梁承重模型
- 设计目标：用规定材料制作承重最大的桥梁
- 材料：木条、胶水、棉线
- 约束条件：跨度30cm，自重不超过200g
- 评价标准：承重/自重比

### 案例3：智能浇花系统
- 传感器：土壤湿度传感器
- 控制器：Arduino/单片机
- 执行器：水泵
- 功能：自动检测土壤湿度，启动浇水
- 拓展：手机远程控制

## 技术素养培养

| 素养维度 | 具体要求 |
|---------|---------|
| 技术意识 | 理解技术与社会的关系，形成使用技术的正确态度 |
| 工程思维 | 运用系统化方法解决实际问题 |
| 创新设计 | 在设计中体现创造性，提出新颖方案 |
| 图样表达 | 运用技术语言进行设计交流 |
| 物化能力 | 将设计方案转化为实物或模型 |
