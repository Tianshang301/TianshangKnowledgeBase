# 综合实践活动

> 综合实践活动是国家义务教育课程方案和普通高中课程方案规定的必修课程，贯穿 K12全过程，包括研究性学习、社会实践和社区服务三大板块。

---

## 课程定位

| 学段 | 课时要求 | 学分要求 |
|------|---------|---------|
| 小学1-2年级 | 每学期≥6课时 | 渗透式 |
| 小学3-6年级 | 每学期≥12课时 | 渗透式 |
| 初中阶段 | 每学期≥15课时 | 渗透式 |
| 高中阶段 |  | 研究性学习学分+社会实践3学分+社区服务2学分 |

---

## 一、研究性学习
### 1.1 研究性学习概述
研究性学习是指学生在教师指导下，从自然、社会和自身生活中选择和确定研究专题，主动获取知识、应用知识、解决问题的学习活动。
#### 一般流程
```
选定课题
   →制定方案
   →开题论证   →实施研究
   →分析处理
   →撰写报告
   →成果展示
```

### 1.2 课题选择指南

#### 选题原则
- **真实性**：来自真实生活，有意义。
- **可行性**：时间、资源、能力可行。
- **探究性**：有探究空间和深度。
- **兴趣性**：学生感兴趣，愿意投入。
#### 选题方向

**自然科学**
- 校园植物多样性调查。
- 水质检测与净化方法。
- 厨余垃圾堆肥实验
- 不同光源对植物生长的影响
- 电磁辐射测量与防护。
**社会科学**
- 中学生手机使用情况调查。
- 社区老年人生活状况调查。
- 垃圾分类实施效果评估
- 本地传统文化传承现状
- 中学生消费观念调查。
**工程技术类**
- 简易净水器设计与制作。
- 智能垃圾分类装置
- 校园雨水收集系统设计
- 太阳能应用小制作
- 桥梁承重结构研究

### 1.3 研究方法

#### 文献研究
- 关键词检索。
- 文献筛选与整理
- 文献综述撰写

#### 调查研究
- 问卷设计（题型、信度、效度）
- 抽样方法（随机抽样、分层抽样）
- 访谈技巧。
- 数据整理与分析。
#### 实验研究
- 实验变量的控制（自变量、因变量、控制变量）
- 实验设计原则（对照、重复、随机）
- 实验记录与数据处理。
#### 观察研究
- 观察计划制定
- 观察记录方法
- 观察结果的分析。
### 1.4 研究报告撰写

| 部分 | 内容要求 |
|------|---------|
| 标题 | 简洁明确，反映研究内容 |
| 摘要 | 概述目的、方法、结果、结论 |
| 引言 | 研究背景、问题提出、研究意义 |
| 文献综述 | 已有研究成果梳理 |
| 研究方法 | 研究对象、方法、过程 |
| 结果与分析 | 数据呈现、图表分析 |
| 结论与建议 | 主要发现、对策建议 |
| 参考文献 | 引用规范 |
| 附录 | 问卷、原始数据等 |

### 1.5 成果展示形式
- 研究报告
- 展板展示
- 多媒体演示。
- 实物模型
- 论文答辩
- 学术小论文。

## 二、社会实践
### 2.1 社会实践概述

社会实践是学生走出校园，接触社会、了解国情、服务社会的实践性学习活动。
### 2.2 实践形式

#### 社会考察
- 参观博物馆、科技馆、纪念馆
- 考察工厂、农场、企业。
- 走访社区、乡村、城市地标。
#### 社会调查
- 主题调研（环保、交通、文化等）
- 数据分析与报告撰写。
- 提出改进建议

#### 职业体验
- 跟岗实习（跟随专业人士工作一天）
- 职业访谈（了解不同职业的特点）
- 模拟职场（角色扮演）

#### 生产劳动实践
- 学工：工厂生产体验。
- 学农：农业生产体验。
- 学商：商业经营体验。
### 2.3 高中社会实践学分要求

| 项目 | 学分 | 内容要求 |
|------|------|---------|
| 军训 | 2学分 | 不少于7天 |
| 社会考察 | 1学分 | 每学期至少1次 |
| 职业体验 | 不计入学分但须完成 | 高一高二各至少1次 |

### 2.4 社会实践报告格式

```
标题：×××社会实践报告一、实践背景与目的
二、实践时间与地点
三、实践内容与过程
四、实践结果与分析
五、感悟与收获
六、问题与建议
```

## 三、社区服务
### 3.1 社区服务概述

社区服务是学生以志愿者的身份参与社区建设和服务，培养社会责任感和奉献精神的活动。
### 3.2 服务类型

| 类型 | 内容 | 适合学段 |
|------|------|---------|
| 环保 | 垃圾分类宣传、植树绿化、河道清理 | 小学—高中 |
| 敬老类 | 敬老院慰问、陪伴老人、帮助打扫 | 小学—高中 |
| 助残 | 帮助残障人士、助残日活动 | 初中—高中 |
| 助学 | 辅导低年级学生、爱心义卖 | 初中—高中 |
| 文化 | 图书馆整理、文化讲解、传统节日活动 | 小学—高中 |
| 社区治理 | 文明宣传、安全巡查、社区议事 | 高中 |

### 3.3 高中社区服务要求

- 高中三年不少于10小时
- 每学期不少于5小时
- 需有服务记录和证明
- 撰写服务心得

### 3.4 志愿服务记录表
| 项目 | 内容 |
|------|------|
| 服务时间 | ×年×月×日×小时 |
| 服务地点 | ××社区/机构 |
| 服务内容 | 具体做了什么 |
| 服务时长 | ×小时 |
| 服务评价 | 服务对象签字/盖章 |
| 个人感悟 | 收获与反思 |

## 四、各学段综合实践主题建议

### 小学阶段

| 年级 | 主题建议 | 主要形式 |
|------|---------|---------|
| 1-2年级 | 认识校园植物、我爱我家、安全小卫士 | 观察、游戏、简单制作 |
| 3-4年级 | 垃圾分类调查、家乡特产探秘、小小气象员 | 调查、实验、参观 |
| 5-6年级 | 水资源调查、传统文化探究、校园节能方案 | 研究性学习、社区服务 |

### 初中阶段

| 学期 | 主题建议 | 主要形式 |
|------|---------|---------|
| 七年级上 | 校园安全隐患排查与改进 | 考察+建议 |
| 七年级下 | 家乡传统美食文化探究 | 研究性学习 |
| 八年级上 | 社区垃圾分类现状调查 | 调查+报告 |
| 八年级下 | 科技创新小制作 | 工程实践 |
| 九年级上 | 青春期同伴关系调查 | 研究性学习 |
| 九年级下 | 职业初体验 | 职业体验 |

### 高中阶段

| 学期 | 主题建议 | 主要形式 |
|------|---------|---------|
| 高一 | 城市交通拥堵问题研究 | 研究性学习 |
| 高一 | 企业参访与职业访谈 | 社会实践 |
| 高二 | 社区公益服务项目策划与实施 | 社区服务 |
| 高二 | 科技创新项目（课题研究） | 研究性学习 |
| 高三 | 大学专业认知与生涯规划 | 社会实践/研究性学习 |

## 五、综合实践评价
### 5.1 评价原则
- 过程性评价与结果性评价相结合
- 自评与他评相结合
- 定性与定量相结合。
### 5.2 评价维度

| 维度 | 评价内容 | 权重建议 |
|------|---------|---------|
| 参与态度 | 积极性、责任感、合作精神 | 30% |
| 过程记录 | 资料的完整性和规范性 | 20% |
| 能力发展 | 问题解决、创新思维、实践能力 | 30% |
| 成果质量 | 报告、作品或服务的质量 | 20% |

### 5.3 档案袋评价
- 收集：研究计划、活动记录、成果作品、反思日记。
- 展示：学期末举办综合实践成果展。
- 交流：学生互评、教师点评。

## 六、综合实践活动项目案例
### 案例1：校园节水方案研究
| 阶段 | 任务 | 时长 |
|------|------|------|
| 选题 | 发现校园水龙头浪费现象，确定研究主题 | 1 |
| 调查 | 记录一周内各时段用水量，采访后勤部 | 2 |
| 分析 | 计算日浪费水量，分析主要原因 | 1 |
| 方案 | 设计节水改造方案（感应龙头、节水宣传） | 1 |
| 实施 | 制作节水标语，向学校提交改造建议 | 2 |
| 总结 | 撰写研究报告，展示成果 | 1 |

### 案例2：社区老年人数字鸿沟帮扶行动
- **目标**：帮助社区老年人掌握智能手机基本操作。
- **流程**：  1. 调研：了解社区老年人使用手机的困难
  2. 备课：制作简单易懂的操作教程（大字版）  3. 教学：一对一或小组教学（微信视频、健康码、挂号）
  4. 总结：撰写服务心得和调研报告
- **服务时长**：每人不少于10小时

### 案例3：家乡非物质文化遗产调查

- **对象**：地方戏曲、手工艺、民俗活动等
- **方法**：文献检索、传承人访谈、实地考察
- **产出**：  - 调查报告
  - 非遗宣传手册
  - 短视频记录  - 校园非遗文化节。
