# 高中生物 · 分子与细胞
> 必修1 模块

---

## 一、细胞的分子组成

### 1. 组成细胞的元素
| 类别 | 元素 |
|------|------|
| **大量元素** | C, H, O, N, P, S, K, Ca, Mg |
| **微量元素** | Fe, Mn, B, Zn, Mo, Cu（铁锰硼锌钼铜） |
| **最基本元素** | C（碳是生命的核心元素）|

### 2. 无机物
| 成分 | 功能 | 备注 |
|------|------|------|
| **水** | 良好溶剂、运输、参与反应、维持温度 | 自由水/结合水比例影响代谢 |
| **无机盐** | 维持渗透压、pH、构成化合物 | Ca²⁺（骨）、Fe²⁺（血红素）、I⁻（甲状腺激素）|

### 3. 有机物
| 种类 | 基本单位 | 功能 | 检测 |
|------|---------|------|------|
| **糖类** | 单糖 | 主要能源物质 | 还原糖+斐林试剂→砖红色沉淀 |
| **脂质** | 甘油+脂肪酸 | 储能、生物膜成分、调节 | 脂肪+苏丹Ⅲ/Ⅳ→橘黄/红色 |
| **蛋白质** | 氨基酸（20种） | 结构/催化/运输/调节/免疫 | 双缩脲试剂→紫色 |
| **核酸** | 核苷酸 | 储存遗传信息 | 甲基绿（DNA绿）/吡罗红（RNA红）|

**氨基酸结构特点**：至少一个氨基（-NH₂）和一个羧基（-COOH）连在同一个碳原子上
**脱水缩合**：肽键数 = 氨基酸数 - 肽链数
**蛋白质结构多样性的原因**：
1. 氨基酸种类不同
2. 氨基酸数目不同
3. 氨基酸排列顺序不同
4. 肽链空间结构不同

**DNA 与 RNA 的区别**：
| 比较 | DNA | RNA |
|------|-----|-----|
| 五碳糖 | 脱氧核糖 | 核糖 |
| 碱基 | A, T, C, G | A, U, C, G |
| 结构 | 双螺旋 | 一般为单链 |
| 分布 | 主要细胞核 | 主要细胞质 |

## 二、细胞的结构与功能
### 1. 细胞膜
- **成分**：磷脂双分子层（基本骨架）+ 蛋白质 + 少量糖类
- **结构特点**：流动性
- **功能特点**：选择透过性
- **功能**：将细胞与外界隔开、控制物质进出、进行细胞间信息交流

### 2. 细胞器
| 细胞器 | 结构特点 | 功能 |
|--------|---------|------|
| **线粒体** | 双层膜、嵴 | 有氧呼吸主要场所（动力工厂）|
| **叶绿体** | 双层膜、基粒 | 光合作用场所（养料制造厂）|
| **内质网** | 单层膜 | 蛋白质加工、脂质合成 |
| **高尔基体** | 单层膜 | 蛋白质分选、加工、包装 |
| **核糖体** | 无膜 | 蛋白质合成（氨基酸脱水缩合）|
| **溶酶体** | 单层膜 | 消化（含水解酶）|
| **液泡** | 单层膜 | 储存物质、维持渗透压 |
| **中心体** | 无膜 | 与有丝分裂有关（动物/低等植物）|

### 3. 细胞核
- **核膜**（双层膜、核孔控制大分子进出）
- **核仁**（与核糖体形成有关）
- **染色质**（DNA + 组蛋白，与染色体是同一物质的不同形态）
- **功能**：遗传信息库、细胞代谢和遗传的控制中心
### 4. 物质跨膜运输

| 方式 | 方向 | 是否需要载体 | 是否耗能 | 举例 |
|------|------|------------|---------|------|
| **自由扩散** | 高→低 | 否 | 否 | O₂, CO₂, H₂O, 甘油 |
| **协助扩散** | 高→低 | 是 | 否 | 葡萄糖进入红细胞 |
| **主动运输** | 低→高 | 是 | 是 | K⁺, Na⁺, 氨基酸, 葡萄糖 |
| **胞吞/胞吐** |  | 否 | 是 | 大分子、颗粒 |

## 三、细胞的代谢

### 1. 酶
- **本质**：绝大多数是蛋白质，少数是 RNA
- **特点**：高效性、专一性、作用条件温和
- **影响酶活性的因素**：温度、pH
- **酶促反应速率**：受底物浓度、酶浓度、温度和 pH 影响

### 2. ATP
ATP → ADP + Pi + 能量（由 ATP 合酶/水解酶催化）
- **功能**：直接能源物质
- **来源**：光合作用（光反应）、细胞呼吸
### 3. 细胞呼吸

| 类型 | 场所 | 总反应式 | 产能 |
|------|------|---------|------|
| **有氧呼吸** | 细胞质基质+线粒体 | C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O + 能量 | 大量 |
| **无氧呼吸（乳酸）** | 细胞质基质 | C₆H₁₂O₆ → 2 乳酸 + 能量 | 少量 |
| **无氧呼吸（酒精）** | 细胞质基质 | C₆H₁₂O₆ → 2C₂H₅OH + 2CO₂ + 能量 | 少量 |

**有氧呼吸三个阶段**：
1. 细胞质基质：C₆H₁₂O₆ → 2 丙酮酸 + 4[H] + 少量 ATP
2. 线粒体基质：2 丙酮酸 → 6CO₂ + 20[H] + 少量 ATP
3. 线粒体内膜：24[H] + 6O₂ → 12H₂O + 大量 ATP

### 4. 光合作用

**总反应式**：6CO₂ + 12H₂O →光能叶绿体 C₆H₁₂O₆ + 6O₂ + 6H₂O

**光反应**（类囊体薄膜）：
- 水的光解：H₂O → 2H⁺ + 2e⁻ + ½O₂
- ATP 形成：ADP + Pi →光能 ATP
- NADPH 形成：NADP⁺ + 2e⁻ + H⁺ → NADPH

**暗反应（卡尔文循环）**（叶绿体基质）：
- CO₂ 固定：CO₂ + C₅ → 2C₃
- C₃ 还原：C₃ →ATP/NADPH (CH₂O) + C₅
**影响光合速率的因素**：光照强度、CO₂ 浓度、温度、水分、矿质元素

## 四、细胞的增殖

### 1. 有丝分裂（体细胞增殖）
**细胞周期**：分裂间期（G₁, S, G₂）+ 分裂期（M）
| 时期 | 特点 |
|------|------|
| **间期** | DNA 复制、蛋白质合成、中心体复制（动物）|
| **前期** | 染色质→染色体、核膜解体、核仁消失、纺锤体形成 |
| **中期** | 染色体排列于赤道板（观察最佳时期）|
| **后期** | 着丝粒分裂，姐妹染色单体分离（染色体数加倍）|
| **末期** | 染色体→染色质、核膜核仁重建、纺锤体消失 |

**动植物有丝分裂的区别**：
- 前期：动物由中心体发出星射线形成纺锤体，植物从两极发出纺锤丝
- 末期：动物细胞膜向内缢裂，植物在赤道板处形成细胞板
**有丝分裂的意义**：保持亲子代细胞遗传性状的稳定。
### 2. 减数分裂（生殖细胞形成）

**减数第一次分裂**：同源染色体分离（染色体数减半）
**减数第二次分裂**：姐妹染色单体分离（类似有丝分裂）
**精子的形成**：1 个精原细胞 → 4 个精子
**卵细胞的形成**：1 个卵原细胞 → 1 个卵细胞 + 3 个极体
### 3. 细胞分化、衰老和癌变
- **分化**：基因选择性表达的结果，遗传物质不变
- **全能性**：已分化细胞仍具发育成完整个体的潜能
- **衰老特征**：水分减少、酶活性降低、色素积累、呼吸减慢
- **癌细胞特征**：无限增殖、形态改变、表面糖蛋白减少（易扩散转移）
- **原癌基因与抑癌基因**：原癌基因负责调节细胞周期，抑癌基因阻止细胞异常增殖
