## 一、遗传的分子基础

### 1. DNA 是遗传物质的证据

**肺炎链球菌转化实验**（格里菲斯 + 艾弗里）：
- 活的 S 型菌 → 小鼠死
- 活的 R 型菌 → 小鼠活
- 加热杀死的 S 型菌 → 小鼠活
- R 型活菌 + 加热杀死的 S 型菌 → 小鼠死（体内有 S 型活菌）
- **结论**：加热杀死的 S 型菌含有"转化因子"

**噬菌体侵染细菌实验**（赫尔希和蔡斯）：
- ³⁵S 标记蛋白质：放射性在上清液
- ³²P 标记 DNA：放射性在沉淀物（细菌内）
- **结论**：DNA 是遗传物质
### 2. DNA 的结构
- **双螺旋结构**（沃森和克里克，1953）
- 两条链反向平行
- 碱基互补配对：A=T，G≡C
- 磷酸和脱氧核糖交替连接构成基本骨架
- 氢键：A-T 间 2 个，G-C 间 3 个
### 3. DNA 复制
- **方式**：半保留复制
- **场所**：细胞核（主要）
- **条件**：模板（DNA 双链）、原料（4 种 dNTP）、酶（解旋酶、DNA 聚合酶）、能量（ATP）
- **方向**：5'→3'
- **特点**：边解旋边复制、半保留复制

### 4. 基因的表达
**转录**（DNA → mRNA）：
- 场所：细胞核
- 模板：DNA 一条链
- 原料：4 种 NTP（核糖核苷酸）
- 酶：RNA 聚合酶
**翻译**（mRNA → 蛋白质）：
- 场所：核糖体
- 模板：mRNA
- 运载工具：tRNA（反密码子与密码子互补配对）
- 密码子：mRNA 上决定一个氨基酸的三个相邻碱基（64 个，3 个终止密码子）
- **中心法则**：DNA → RNA → 蛋白质，RNA 可逆转录为 DNA

## 二、遗传的基本规律

### 1. 基因分离定律（孟德尔第一定律）
- 一对等位基因在减数分裂时随同源染色体分开而分离
- 典型比例：Aa × Aa → 3:1（显性:隐性）
- 测交：Aa × aa → 1:1

### 2. 基因自由组合定律（孟德尔第二定律）
- 非同源染色体上的非等位基因自由组合
- 典型比例：AaBb × AaBb → 9:3:3:1
- 实质：减Ⅰ后期非同源染色体自由组合
- 测交：AaBb × aabb → 1:1:1:1

### 3. 孟德尔成功的原因
1. 选材正确（豌豆：自花传粉、相对性状明显）
2. 从简单到复杂（先研究一对再研究多对）
3. 运用统计学方法
4. 科学设计实验程序（假说-演绎法）

### 4. 伴性遗传
| 类型 | 特点 | 举例 |
|------|------|------|
| **伴 X 显** | 女性患者多于男性，父病女必病 | 抗维生素D佝偻病 |
| **伴 X 隐** | 男性患者多于女性，母病子必病 | 红绿色盲、血友病 |
| **伴 Y 遗传** | 只在男性中遗传 | 耳毛 |

### 5. 人类遗传病
| 类型 | 举例 | 遗传方式 |
|------|------|---------|
| **常染色体显性** | 多指、并指 | 父母一方患病则子代患病概率 1/2 |
| **常染色体隐性** | 白化病、镰状细胞贫血 | 父母都携带致病基因子代才患病 |
| **伴 X 隐性** | 红绿色盲、血友病 | 母病子必病 |
| **多基因遗传病** | 唇裂、原发性高血压 | 受多对基因和环境共同影响 |
| **染色体异常** | 唐氏综合征（21-三体）| 染色体数目变异 |

## 三、变异与育种

### 1. 基因突变
- **概念**：DNA 分子中碱基对的替换、增添或缺失
- **特点**：普遍性、随机性、不定向性、低频性、多害少利性
- **诱变因素**：物理（辐射）、化学（亚硝酸）、生物（病毒）
- **意义**：生物变异的根本来源，进化的原材料
### 2. 基因重组
- **类型**：自由组合（非同源染色体）、交叉互换（同源染色体）、基因工程
- **意义**：产生新性状组合，是生物变异的重要来源
### 3. 染色体变异
**结构变异**：缺失、重复、倒位、易位
**数目变异**：
- 整倍体：单倍体、二倍体、多倍体
- 非整倍体：单体、三体
**单倍体育种**（花药离体培养 + 秋水仙素加倍）：
- 优点：明显缩短育种年限
- 原理：染色体数目变异

**多倍体育种**（秋水仙素处理）：
- 原理：抑制纺锤体形成
- 实例：三倍体无子西瓜

### 4. 基因工程
- **工具**：限制性内切核酸酶、DNA 连接酶、载体
- **步骤**：目的基因获取 → 构建表达载体 → 导入受体细胞 → 筛选和鉴定
- **应用**：转基因作物、基因治疗、药物生产

## 四、进化
### 1. 现代生物进化理论
- **种群**是生物进化的基本单位
- **突变和基因重组**产生进化的原材料
- **自然选择**决定生物进化的方向
- **隔离**是物种形成的必要条件

### 2. 自然选择学说（达尔文）
过度繁殖 → 生存斗争 → 遗传变异 → 适者生存
### 3. 物种形成
地理隔离 → 种群基因频率差异累积 → 生殖隔离（新物种形成）
**基因频率计算**：p(A) + q(a) = 1；p² + 2pq + q² = 1（哈迪-温伯格定律）

### 4. 共同进化与生物多样性
- 不同物种之间、生物与无机环境之间相互影响中进化
- 生物多样性：基因多样性、物种多样性、生态系统多样性

## 五、遗传规律的应用

### 基因分离定律的应用
**显隐性的判断**：
1. 具有相对性状的纯合亲本杂交，F₁表现出的性状为显性
2. 具有相同性状的亲本杂交，后代出现性状分离，新出现的性状为隐性

**基因型的推断**：
- 表现型为隐性性状→基因型为隐性纯合子
- 表现型为显性性状→基因型可能为AA或Aa
- 测交法：待测个体×隐性纯合子，根据后代判断

**概率计算**：
- Aa × Aa → AA:Aa:aa = 1:2:1（显:隐 = 3:1）
- 生男生女概率各为1/2
- 两种遗传病同时计算时使用乘法定理和加法定理

### 基因自由组合定律的应用
**棋盘法**：
- 先分析每一对基因的分离情况
- 再将各对基因的结果相乘（乘法定理）
- 例如：AaBb × AaBb → (3:1)(3:1) = 9:3:3:1

**常见比例分析**：
| 杂交组合 | 后代比例 | 基因型 |
|----------|----------|--------|
| AaBb × AaBb | 9:3:3:1 | 双杂合自交 |
| AaBb × aabb | 1:1:1:1 | 双杂合测交 |
| AABb × aaBb | 3:1 | 单对基因杂合 |
| AaBb × Aabb | 3:3:1:1 | 一对测交一对自交 |

**特殊比例**：
- 9:7（互补作用）→ A_B_表现为一种性状，其余表现为另一种
- 9:6:1（累加作用）→ A_B_一种，A_bb和aaB_一种，aabb一种
- 15:1（重叠作用）→ 只要有显性基因就表现为一种性状
- 12:3:1（显性上位）
- 9:3:4（隐性上位）

### 伴性遗传的应用
**系谱图分析步骤**：
1. 判断显隐性（无中生有为隐性，有中生无为显性）
2. 判断致病基因位置（常染色体还是性染色体）
3. 写出相关个体基因型
4. 计算后代患病概率

**伴X隐性遗传特点**：
- 男性患者多于女性
- 隔代交叉遗传
- 母亲患病儿子一定患病，女儿患病父亲一定患病

**伴X显性遗传特点**：
- 女性患者多于男性
- 世代连续性
- 父亲患病女儿一定患病，母亲患病儿子不一定患病

## 六、基因的本质与表达精讲

### 遗传信息的传递——中心法则
**DNA→DNA（复制）**：遗传信息从DNA传递到DNA
**DNA→RNA（转录）**：遗传信息从DNA传递到RNA
**RNA→蛋白质（翻译）**：遗传信息从RNA传递到蛋白质
**RNA→DNA（逆转录）**：某些病毒中遗传信息从RNA传递到DNA
**RNA→RNA（RNA复制）**：某些病毒中遗传信息从RNA传递到RNA

### 基因表达调控
**原核生物基因表达调控**：
- 操纵子模型（乳糖操纵子）
- 调节基因→阻遏蛋白→与操纵基因结合→阻止RNA聚合酶转录

**真核生物基因表达调控**：
- 转录水平调控：启动子、增强子、沉默子
- 转录后调控：RNA加工、剪接
- 翻译水平调控：起始因子、抑制因子
- 翻译后调控：蛋白质修饰、降解

### 表观遗传学
**定义**：DNA序列不改变但基因表达发生可遗传的改变

**机制**：
- DNA甲基化：甲基化抑制基因表达
- 组蛋白修饰：乙酰化促进表达，去乙酰化抑制表达
- RNA干扰：小RNA分子调控基因表达

**实例**：
- 蜜蜂的蜂王和工蜂由相同的DNA发育而来，但蜂王浆的摄入导致表观遗传差异
- 小鼠的毛色受Agouti基因甲基化水平影响

## 七、生物进化理论精讲

### 种群基因频率的变化
**基因频率计算**：
- 基因频率 = 某种基因的数目 / 该基因及其等位基因的总数
- AA个体数×2 + Aa个体数 = A基因数
- aa个体数×2 + Aa个体数 = a基因数
- 总基因数 = 个体数 × 2

**哈迪-温伯格平衡定律**：
- 条件：种群足够大、随机交配、无突变、无迁移、无选择
- 公式：p² + 2pq + q² = 1，p + q = 1
- p = A基因频率，q = a基因频率
- p² = AA基因型频率，2pq = Aa基因型频率，q² = aa基因型频率

**影响基因频率的因素**：
1. 突变（产生新基因）
2. 基因重组（产生新基因型）
3. 自然选择（定向改变基因频率）
4. 遗传漂变（小种群中基因频率随机变化）
5. 基因流动（种群间个体迁移）

### 自然选择的类型
| 类型 | 特点 | 示例 |
|------|------|------|
| 稳定性选择 | 保留中间类型，淘汰极端 | 人类新生儿体重 |
| 定向选择 | 保留一个极端，淘汰另一极端 | 工业革命中黑色桦尺蛾 |
| 分裂选择 | 保留两个极端，淘汰中间 | 雀类喙的大小分化 |

### 物种形成的三种方式
**异域物种形成**：
- 地理隔离导致种群基因频率差异
- 生殖隔离形成新物种
- 实例：加拉帕戈斯群岛的地雀

**同域物种形成**：
- 同一地区内由于生态位分化
- 生殖隔离不依赖地理隔离
- 实例：苹果蝇的寄主分化

**邻域物种形成**：
- 相邻分布区的种群在交界处基因交流受阻
- 实例：环物种

## 八、生物技术与遗传学应用

### 遗传咨询与产前诊断
**遗传咨询步骤**：
1. 医生询问病史、绘制系谱图
2. 分析遗传方式和再发风险
3. 提出建议（产前诊断、终止妊娠等）

**产前诊断方法**：
| 方法 | 时间 | 检查内容 |
|------|------|----------|
| B超 | 孕早期/中期 | 胎儿外观畸形 |
| 羊膜穿刺 | 16-20周 | 染色体异常、基因检测 |
| 绒毛膜取样 | 10-13周 | 早期染色体、基因检测 |
| 无创产前检测 | 12周后 | 21-三体等染色体异常 |

### 基因治疗
**原理**：将正常基因导入患者细胞，替代或修复缺陷基因

**类型**：
- 体细胞基因治疗（不影响后代）
- 生殖细胞基因治疗（影响后代，伦理争议大）

**载体**：
- 病毒载体（逆转录病毒、腺病毒）
- 非病毒载体（脂质体、纳米颗粒）

**应用案例**：
- SCID（重症联合免疫缺陷症）的基因治疗
- 某些遗传性眼病的基因治疗
- CAR-T细胞治疗（免疫细胞基因改造）

### 基因检测与基因身份证
**应用**：
- 疾病易感性预测
- 个体识别（亲子鉴定、犯罪侦破）
- 药物敏感性检测（个体化用药）

**伦理争议**：
- 基因歧视问题
- 隐私保护问题
- 基因信息的社会公平性
