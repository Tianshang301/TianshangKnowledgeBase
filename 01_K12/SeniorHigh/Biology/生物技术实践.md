# 高中生物 · 生物技术实践
> 选择性必修 3 模块（现代生物科技专题）

---

## 一、传统发酵技术
| 产品 | 微生物 | 原理 | 条件 |
|------|--------|------|------|
| **果酒** | 酵母菌 | 无氧呼吸产生酒精 | 无氧、18-30℃ |
| **果醋** | 醋酸菌 | 酒精→醋酸 | 有氧、30-35℃ |
| **腐乳** | 毛霉 | 产生蛋白酶和脂肪酶 | 15-18℃ |
| **泡菜** | 乳酸菌 | 无氧发酵产生乳酸 | 无氧、室温 |

## 二、微生物培养

### 1. 培养基成分
- **碳源**：提供碳元素（葡萄糖、淀粉）
- **氮源**：提供氮元素（蛋白胨、NH₄NO₃）
- **水**：必需的溶剂
- **无机盐**：微量元素
- **特殊成分**：维生素、生长因子
### 2. 无菌技术
| 方法 | 对象 | 条件 |
|------|------|------|
| 湿热灭菌（高压蒸汽）| 培养基、器具 | 121℃、15-30min |
| 干热灭菌 | 玻璃器皿 | 160-170℃、2h |
| 灼烧灭菌 | 接种环、涂布器 | 火焰灼烧 |
| 化学消毒 | 双手、桌面 | 酒精擦拭 |

### 3. 接种方法
- **平板划线法**：分离纯化菌种
- **稀释涂布平板法**：菌落计数

---

## 三、植物组织培养
- **原理**：植物细胞的全能性
- **条件**：无菌、营养（MS 培养基）、植物激素（生长素/细胞分裂素比例）
- **过程**：外植体 → 脱分化 → 愈伤组织 → 再分化 → 完整植株

---

## 四、动物细胞工程
### 1. 动物细胞培养
- 液体培养基，含动物血清
- 原代培养 → 传代培养
- 培养条件：37℃、5% CO₂
### 2. 动物细胞融合
- 诱导方法：PEG（聚乙二醇）/ 灭活仙台病毒 / 电融合
- **单克隆抗体**：B 细胞 + 骨髓瘤细胞 → 杂交瘤细胞 → 单抗
  - 特点：特异性强、灵敏度高
  - 应用：疾病诊断、靶向治疗
### 3. 动物体细胞核移植
- 原理：动物细胞核具有全能性
- 应用：克隆动物（多莉羊）

## 五、胚胎工程

## 六、基因工程
### 工具酶
- **限制酶**：识别并切割特定 DNA 序列（产生黏性末端或平末端）
- **DNA 连接酶**：连接 DNA 片段

### 载体
- **质粒**：最常用，环状 DNA
- **噬菌体**：λ噬菌体
- **动植物病毒**

### 操作步骤
1. 目的基因的获取
2. 基因表达载体的构建（核心步骤）
3. 将目的基因导入受体细胞
4. 目的基因的检测与鉴定

### PCR 技术
- **变性**：94℃，DNA 双链解旋
- **退火**：50-60℃，引物与模板结合
- **延伸**：72℃，Taq 酶合成新链
- 循环 30-35 次，指数扩增

### 应用
- 转基因作物（抗虫棉、抗除草剂大豆）
- 基因治疗（导入正常基因）
- 基因工程药物（胰岛素、生长激素）

## 八、蛋白质工程

### 与基因工程的关系
- **基因工程**：生产自然界已有的蛋白质
- **蛋白质工程**：改造或创造新的蛋白质
- **关系**：蛋白质工程是基因工程的延伸

### 操作流程
1. 确定目标蛋白质的功能
2. 设计预期蛋白质的结构
3. 推测应有的氨基酸序列
4. 找到对应的脱氧核苷酸序列（基因）
5. 通过基因工程改造或合成基因
6. 表达产生目标蛋白质

### 应用案例
- 改造酶的活性，提高催化效率
- 设计新型药物蛋白
- 改良工业用酶的稳定性

## 十、实验技能进阶

### 微生物培养实验
**实验1：微生物的分离与纯化**
1. 制备培养基并灭菌
2. 稀释样品（梯度稀释）
3. 涂布平板或平板划线
4. 恒温培养（根据微生物选择适宜温度）
5. 观察菌落特征，挑取单菌落纯化

**实验2：土壤中分解尿素细菌的分离**
1. 取土样（富含有机质的土壤）
2. 制备选择培养基（以尿素为唯一氮源）
3. 梯度稀释样品液
4. 涂布平板培养
5. 加入酚红指示剂，观察颜色变化

**实验3：果酒与果醋的制作**
- 果酒发酵：酵母菌无氧呼吸产生酒精
- 果醋发酵：醋酸菌有氧发酵将酒精转化为醋酸
- 关键控制：温度、氧气、pH

### 植物组织培养实验
**实验材料**：胡萝卜根、MS 培养基、植物激素（生长素、细胞分裂素）

**操作步骤**：
1. 配制 MS 培养基，加入激素，调节 pH
2. 培养基分装灭菌
3. 胡萝卜根消毒处理
4. 切取形成层部位（外植体）
5. 接种到培养基上
6. 培养条件：25℃、光照12h/天
7. 观察愈伤组织形成和分化

### DNA 粗提取与鉴定
**实验原理**：
- DNA 在 NaCl 溶液中的溶解度随 NaCl 浓度变化
- DNA 不溶于冷酒精，蛋白质溶于冷酒精
- 二苯胺试剂使 DNA 呈现蓝色

**操作步骤**：
1. 取材（鸡血或菜花）
2. 破碎细胞（加洗涤剂、食盐）
3. 过滤去除杂质
4. 加入冷酒精沉淀 DNA
5. 挑取白色丝状物（DNA）
6. 二苯胺试剂鉴定（沸水浴加热5min）

## 高考实验题解题策略

### 实验设计原则
1. **对照原则**：设置对照组，排除无关变量干扰
2. **单一变量原则**：控制其他条件相同，只改变一个变量
3. **平行重复原则**：重复实验，减少偶然误差
4. **科学性原则**：实验方案具有科学依据

### 实验题答题模板
**实验目的**：探究/验证/鉴定……
**实验假设**：如果……那么……
**实验材料**：……
**实验步骤**：
1. 分组编号
2. 施加实验变量（处理组/对照组）
3. 控制无关变量（相同且适宜）
4. 观察记录实验结果
**实验结果**：……
**实验结论**：……

### 常见实验题型
1. **验证性实验**：验证已知结论
2. **探究性实验**：探究未知问题
3. **评价性实验**：评价他人的实验方案
4. **设计性实验**：独立设计实验方案
