# 高中化学 · 化学实验

> 涵盖高中化学所有核心实验操作与技术

---

## 一、常用仪器

| 仪器 | 用途 | 注意事项 |
|------|------|---------|
| 试管 | 少量反应 | 加热时管口不对人 |
| 烧杯 | 溶解、加热 | 垫石棉网加热 |
| 量筒 | 量取液体 | 不用于配製溶液、不加热 |
| 容量瓶 | 配製溶液 | 不用来反应、不加热、不贮存 |
| 滴定管 | 滴定 | 酸式/碱式、润洗 |
| 分液漏斗 | 分液 | 下层从下口放出，上层从上口倒出 |
| 冷凝管 | 冷凝 | 下口进水，上口出水 |
| 蒸馏烧瓶 | 蒸馏 | 加沸石防暴沸 |

### 仪器使用口诀
- 量筒：定量取液不加热，不用作反应容器
- 容量瓶：配液专用，不用来溶质，不贮存
- 滴定管：酸式装酸/氧化剂，碱式装碱
- 分液漏斗：使用前检漏

## 二、基本操作

### 过滤
**一贴二低三靠**：
- 一贴：滤纸紧贴漏斗壁
- 二低：滤纸边缘低于漏斗口，液面低于滤纸边缘
- 三靠：烧杯尖嘴靠玻璃棒，玻璃棒靠三层滤纸处，漏斗尖嘴靠烧杯壁

### 蒸馏
- 温度计水银球在蒸馏烧瓶支管口处
- 先通冷凝水后加热
- 加沸石防暴沸
- 馏分沸点范围：纯净物沸点恒定

### 萃取分液
检漏 → 装液 → 振荡 → 静置 → 分液

**萃取剂要求**：与原溶剂不互溶，溶质在其中溶解度大

**常见萃取剂**：CCl₄（密度>水）、苯/乙醚（密度<水）

### 溶液配製
计算 → 称量 → 溶解 → 冷却 → 转移 → 洗涤 → 定容 → 摇匀 → 装瓶

**关键点**：
- 容量瓶使用前检查是否漏水
- 转移要用玻璃棒引流
- 洗涤2-3次，洗涤液转入容量瓶
- 定容时视线平视凹液面最低处

### 中和滴定

**步骤**：检漏 → 润洗 → 装液 → 赶气泡沫 → 调液面 → 加指示剂 → 滴定 → 读数

**指示剂选择**：

| 滴定类型 | 指示剂 | 颜色变化 |
|---------|--------|---------|
| 强酸强碱 | 酚酞 | 无色→浅红 |
| 强酸强碱 | 甲基橙 | 红→黄 |
| 强酸弱碱 | 甲基橙 | — |
| 强碱弱酸 | 酚酞 | — |

**关键操作**：
- 滴定管润洗
- 赶尖嘴气泡
- 最后一滴颜色变化且半分钟不褪
- 平行滴定2-3次

## 三、物质的检验与鉴别

### 离子检验

| 离子 | 试剂 | 现象 |
|------|------|------|
| Na⁺ | 焰色反应 | 黄色火焰 |
| K⁺ | 焰色反应 | 紫色（透过蓝色钴玻璃）|
| Ca²⁺ | 焰色反应 | 砖红色火焰 |
| SO₄²⁻ | BaCl₂ + 稀 HCl | 白色沉淀 |
| NH₄⁺ | NaOH + 加热 | 湿润红色石蕊试纸变蓝 |
| Fe³⁺ | KSCN | 血红色 |
| Fe²⁺ | NaOH | 白色→灰绿→红褐 |
| Cl⁻ | AgNO₃ + 稀 HNO₃ | 白色沉淀 |
| CO₃²⁻ | 稀 HCl + 澄清石灰水 | 石灰水变浑浊 |

### 气体检验

| 气体 | 检验方法 | 现象 |
|------|---------|------|
| O₂ | 带火星木条 | 复燃 |
| H₂ | 点燃 | 淡蓝色火焰，产物为水 |
| CO₂ | 澄清石灰水 | 变浑浊 |
| NH₃ | 湿润红色石蕊试纸 | 变蓝 |
| Cl₂ | 湿润淀粉 KI 试纸 | 变蓝 |
| SO₂ | 品红溶液 | 褪色（加热恢复）|

## 四、气体的製备

### 气体发生装置选择

| 装置类型 | 适用条件 | 举例 |
|---------|---------|------|
| 固+固加热 | 反应物为固体，需加热 | O₂（KClO₃分解）、NH₃ |
| 固+液不加热 | 固体+液体，不需加热 | H₂（Zn+H₂SO₄）、CO₂、O₂（H₂O₂）|
| 固+液加热 | 固体+液体，需加热 | Cl₂（MnO₂+HCl）、SO₂ |

### 常见气体实验室製法

**O₂**：$2KClO_3 \xrightarrow{MnO_2,\ \triangle} 2KCl + 3O_2\uparrow$（排水法收集）

**H₂**：$Zn + H_2SO_4(\text{稀}) \to ZnSO_4 + H_2\uparrow$（排水法或向下排空气法）

**CO₂**：$CaCO_3 + 2HCl \to CaCl_2 + CO_2\uparrow + H_2O$（向上排空气法）

**NH₃**：$2NH_4Cl + Ca(OH)_2 \xrightarrow{\triangle} CaCl_2 + 2NH_3\uparrow + 2H_2O$（向下排空气法，用碱石灰干燥）

**Cl₂**：$MnO_2 + 4HCl(\text{浓}) \xrightarrow{\triangle} MnCl_2 + Cl_2\uparrow + 2H_2O$（向上排空气法，用饱和食盐水除 HCl）

**SO₂**：$Na_2SO_3 + H_2SO_4(\text{浓}) \to Na_2SO_4 + SO_2\uparrow + H_2O$（向上排空气法）

## 五、物质的分离与提纯

| 方法 | 原理 | 适用范围 |
|------|------|---------|
| **过滤** | 不溶性固体与液体分离 | 粗盐提纯 |
| **蒸发结晶** | 利用溶解度差 | 从溶液中得到固体 |
| **降温结晶** | 溶解度随温度变化 | 分离溶解度差异大的物质 |
| **蒸馏** | 沸点差异 | 分离互溶液体（酒精/水）|
| **分液** | 互不相溶 | 油水分离 |
| **萃取** | 溶解度差 | 从水溶液中提取溶质 |
| **洗气** | 气体溶解度差 | 除杂（饱和食盐水洗 Cl₂中 HCl）|
| **升华** | 物质可直接由固体变气体 | I₂、萘的提纯 |

### 除杂原则
- 不增：不引入新杂质
- 不减：不减少被提纯物质
- 易分离：杂质转化为易分离物质

## 六、高中化学核心实验

### 实验1：配製一定物质的量浓度的溶液

**步骤**：计算 → 称量 → 溶解 → 冷却 → 转移 → 洗涤 → 定容 → 摇匀 → 装瓶

**误差分析**：

| 操作 | 误差 |
|------|------|
| 称量时左码右物 | 偏低 |
| 溶解后未冷却转移 | 偏高 |
| 转移时液体溅出 | 偏低 |
| 未洗涤烧杯和玻璃棒 | 偏低 |
| 定容时仰视 | 偏低 |
| 定容时俯视 | 偏高 |

### 实验2：酸碱中和滴定
**关键**：指示剂选择、滴定管使用、终点判断

**误差分析**：
- 滴定管未润洗 → 偏高
- 尖嘴有气泡未赶 → 偏高
- 读数时仰视/俯视 → 有影响

### 实验3：焰色反应
**操作**：铂丝 → 稀盐酸洗净 → 酒精灯灼烧至无色 → 蘸取待测液 → 灼烧观察

### 实验4：乙酸乙酯的製备

$$ CH_3COOH + C_2H_5OH \xrightleftharpoons[\triangle]{浓 H_2SO_4} CH_3COOC_2H_5 + H_2O $$

**装置**：大试管+导管（长导管冷凝回流）

**收集**：饱和 Na₂CO₃溶液（溶解乙醇、中和乙酸、降低酯溶解度）

**现象**：液面上有透明油状液体，有果香味

### 实验5：原电池

**Cu-Zn 原电池**：
- 负极（Zn）：$Zn - 2e^- \to Zn^{2+}$
- 正极（Cu）：$2H^+ + 2e^- \to H_2\uparrow$

**现象**：Zn 片溶解，Cu 片上有气泡，电流计偏转

## 七、实验安全

| 事故 | 处理方法 |
|------|---------|
| 酸溅到皮肤 | 大量水冲洗，涂 NaHCO₃ |
| 碱溅到皮肤 | 大量水冲洗，涂硼酸 |
| 酸入眼睛 | 立即大量水冲洗5分钟 |
| 酒精灯着火 | 用湿布盖灭 |
| 金属钠着火 | 用沙土盖灭 |
| 吸入 Cl₂ | 移至通风处，吸 NaHCO₃雾化液 |
| 误食重金属盐 | 喝大量牛奶/蛋清 |

## 实验操作规范

1. **三不原则**：不闻味、不品尝、不触手
2. **节约原则**：药品按需取用，不浪费
3. **保护原则**：尾气处理（有毒气体用吸收装置）
4. **记录原则**：如实记录现象和数据，不编造
