# 高中化学 · 化学反应原理

> 选择性必修 1 模块

---

## 一、化学反应与能量

### 1. 热化学方程式
- 注明物质状态（s, l, g, aq）
- 注明 ΔH 的正负（放热为负，吸热为正）
- ΔH 的单位：kJ/mol
- 系数表示物质的量，可为分数
### 2. 盖斯定律
化学反应的热效应只与反应体系的始态和终态有关，与途径无关：ΔH = ΔH₁ + ΔH₂ + ΔH₃ + ...

### 3. 燃烧热与中和热
- **燃烧热**：1 mol 物质完全燃烧生成稳定氧化物放出的热量
  - C → CO₂(g)，H → H₂O(l)，S → SO₂(g)
- **中和热**：强酸强碱稀溶液中和生成 1 mol 水放出 57.3 kJ
  - H⁺ + OH⁻ → H₂O ΔH = -57.3 kJ/mol

## 二、化学反应速率

### 1. 表达方式
v = Δc/Δt（单位：mol/(L·s) 或 mol/(L·min)）
**注意**：化学反应速率是平均速率，均为正值。
### 2. 影响反应速率的因素
**内因**：反应物本身的性质（决定因素）

**外因**：
| 因素 | 影响 |
|------|------|
| **浓度** | 增大浓度 → 速率加快 |
| **压强** | 增大压强（缩小体积）→ 气体反应速率加快 |
| **温度** | 升高温度 → 速率加快（每升高 10℃，速率约增 2-4 倍）|
| **催化剂** | 降低活化能，同等幅度加快正逆反应速率 |
| **其他** | 固体表面积、光、电磁波 |

### 3. 有效碰撞理论
- **有效碰撞**：能发生化学反应的分子碰撞
- **条件**：足够能量 + 合适取向
- **活化能**：使普通分子变成活化分子所需最低能量
- **催化剂**：降低活化能，提高活化分子百分数

## 三、化学平衡
### 1. 化学平衡的特点
逆（可逆反应）、等（v正 = v逆）、动（动态平衡）、定（各组分浓度恒定）、变（条件改变平衡移动）

### 2. 平衡常数（K）
aA + bB ⇌ cC + dD
K = [C]ᶜ[D]ᵈ / [A]ᵃ[B]ᵇ（纯固体和纯液体不写入）
- K 越大 → 反应进行程度越大
- K 只与温度有关（温度变 K 变）
- 通过 Qc 与 K 比较判断平衡移动方向

### 3. 平衡转化率
某反应物已转化量 / 起始总量 × 100%

### 4. 勒夏特列原理
如果改变影响平衡的一个条件，平衡就向能够减弱这种改变的方向移动。
| 条件改变 | 平衡移动方向 |
|---------|-------------|
| 增大反应物浓度 | 正向移动 |
| 增大压强 | 向气体体积减小的方向移动 |
| 升高温度 | 向吸热方向移动 |

### 5. 等效平衡

| 条件 | 投料要求 | 结果 |
|------|---------|------|
| 恒温恒容（气体分子数不等）| 投料等比同数 | 完全等效 |
| 恒温恒容（气体分子数相等）| 投料等比 | 等效 |
| 恒温恒压 | 投料等比 | 等效 |

### 6. 化学反应进行的方向
- ΔH - TΔS < 0 → 自发
- ΔH - TΔS > 0 → 非自发
- ΔH - TΔS = 0 → 平衡

## 四、水溶液中的离子平衡

### 1. 弱电解质的电离
- 强电解质：完全电离（强酸、强碱、大部分盐）
- 弱电解质：部分电离（弱酸、弱碱、水）
- 电离平衡常数：Kₐ（弱酸）、K_b（弱碱），Kₐ 越大 → 酸性越强
### 2. 水的离子积
K_w = [H⁺][OH⁻] = 1.0×10⁻¹⁴（25℃），升温 → K_w 增大

### 3. pH 计算
pH = -lg[H⁺]，pOH = -lg[OH⁻]
pH + pOH = pK_w = 14（25℃）

| 溶液 | pH（25℃）|
|------|----------|
| 中性 | pH = 7 |
| 酸性 | pH < 7 |
| 碱性 | pH > 7 |

### 4. 盐类水解
**规律**：谁弱谁水解，谁强显谁性
- 强酸弱碱盐 → 酸性（NH₄Cl、FeCl₃）
- 强碱弱酸盐 → 碱性（CH₃COONa、Na₂CO₃）
- 强酸强碱盐 → 中性（NaCl）
**影响水解的因素**：温度↑→水解程度↑，浓度↓→水解程度↑

### 5. 沉淀溶解平衡
- 溶度积：K_sp = [Mⁿ⁺]ᵐ[Aᵐ⁻]ⁿ
- Q > K_sp → 沉淀生成；Q = K_sp → 饱和；Q < K_sp → 沉淀溶解
- 沉淀转化：溶解度大的沉淀 → 溶解度小的沉淀

## 五、电化学

### 1. 原电池
**构成条件**：
1. 两个活动性不同的电极
2. 电解质溶液
3. 形成闭合回路
4. 自发氧化还原反应

**电极判断**：
- 负极（较活泼金属）：失电子，发生氧化反应
- 正极（较不活泼金属/石墨）：得电子，发生还原反应

**化学电源**：
- 锌锰干电池：Zn | NH₄Cl | MnO₂(C)
- 铅蓄电池：Pb | H₂SO₄ | PbO₂（可充电）
- 氢氧燃料电池：H₂ | KOH | O₂
### 2. 电解池
**电极判断**：
- 阳极：接电源正极 → 发生氧化反应
- 阴极：接电源负极 → 发生还原反应

**放电顺序**：
- 阳极（氧化）：活性电极 > S²⁻ > I⁻ > Br⁻ > Cl⁻ > OH⁻ > 含氧酸根
- 阴极（还原）：Ag⁺ > Cu²⁺ > H⁺（酸）> Fe²⁺ > Zn²⁺ > H⁺（水）> Al³⁺
**电解的应用**：
- 电解饱和食盐水（氯碱工业）
- 电镀：镀层金属作阳极，镀件作阴极
- 电冶金：电解熔融 NaCl/Al₂O₃
### 3. 金属腐蚀与防护
- **化学腐蚀**：直接与氧化剂反应
- **电化学腐蚀**：吸氧腐蚀（主要）和析氢腐蚀（酸性环境）
- **防护**：覆盖保护层、牺牲阳极保护法、外加电流阴极保护法
