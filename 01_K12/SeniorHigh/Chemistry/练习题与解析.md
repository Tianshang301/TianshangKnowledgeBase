## 一、选择题
**1** 下列物质中，属于电解质的是（ ）
A. 铜  B. 蔗糖  C. NaCl溶液  D. 熔融的KNO₃
**2** 在周期表中，第三周期第ⅦA族的元素是（ ）
A. Cl  B. Br  C. I  D. F

**3** 下列反应中，属于氧化还原反应的是（ ）
A. CaO + H₂O → Ca(OH)₂  B. HCl + NaOH → NaCl + H₂O
C. Fe + CuSO₄ → FeSO₄ + Cu  D. AgNO₃ + NaCl → AgCl↓ + NaNO₃
**4** 相同浓度的下列溶液，pH最小的是（ ）
A. NaCl  B. CH₃COONa  C. NH₄Cl  D. Na₂CO₃
**5** 用惰性电极电解CuCl₂溶液，阳极产生的物质是（ ）
A. Cu  B. Cl₂  C. O₂  D. H₂
**6** 下列有机物中，能使酸性KMnO₄溶液褪色的是（ ）
A. 甲烷  B. 苯  C. 乙烯  D. 乙烷

**7** 实验室制取CO₂气体，最好选用的试剂是（ ）
A. 碳酸钠和稀硫酸  B. 大理石和稀盐酸
C. 碳酸钙和稀硫酸  D. 碳酸钠和稀盐酸

**8** 在 2A + B ⇌ 3C的反应中，下列能说明达到平衡的是（ ）
A. v(A) = v(B)  B. 各物质浓度不再变化  C. 反应停止  D. 各物质浓度相等

## 五、化学反应速率与化学平衡

**16** 在容积为 $2\text{L}$ 的密闭容器中发生反应：$\text{N}_2 + 3\text{H}_2 \rightleftharpoons 2\text{NH}_3$。开始时加入 $1\text{mol}$ $\text{N}_2$ 和 $3\text{mol}$ $\text{H}_2$，$5$ 分钟后达到平衡，测得 $\text{NH}_3$ 的浓度为 $0.4\text{mol/L}$。求：
（1）$\text{N}_2$ 和 $\text{H}_2$ 的转化率；
（2）该温度下的平衡常数 $K$（用浓度表示）。

**17** 已知反应 $2\text{NO}_2(g) \rightleftharpoons \text{N}_2\text{O}_4(g)$ 的 $\Delta H = -57.2\text{kJ/mol}$。
（1）升高温度，平衡向哪个方向移动？
（2）增大压强，平衡向哪个方向移动？
（3）若在恒温恒容条件下，通入一定量的 $\text{NO}_2$，平衡如何移动？$\text{NO}_2$ 的转化率如何变化？

## 六、电化学提高

**18** 现有锌-铜原电池，电解质溶液为 $0.1\text{mol/L}$ 的 $\text{CuSO}_4$ 溶液。
（1）写出正负极的反应式；
（2）若反应一段时间后，$\text{CuSO}_4$ 溶液浓度降为 $0.05\text{mol/L}$（假设溶液体积 $1\text{L}$），求转移的电子数和锌片减少的质量。

**19** 用惰性电极电解 $500\text{mL}$ $0.2\text{mol/L}$ 的 $\text{AgNO}_3$ 溶液，当电路中通过 $0.02\text{mol}$ 电子时：
（1）写出电极反应式；
（2）计算阳极产生的气体在标准状况下的体积。

## 七、有机化学提高

**20** 化合物 A 的分子式为 $\text{C}_4\text{H}_{10}\text{O}$，A 能与金属钠反应产生氢气，能被酸性 $\text{KMnO}_4$ 氧化生成 B（$\text{C}_4\text{H}_8\text{O}_2$）。A 在浓硫酸加热下发生消去反应生成 C（$\text{C}_4\text{H}_8$），C 与 $\text{Br}_2$ 的 $\text{CCl}_4$ 溶液反应生成 D（$\text{C}_4\text{H}_8\text{Br}_2$）。
（1）推断 A、B、C、D 的结构简式；
（2）写出 A → B 和 A → C 的化学方程式。

**21** 某烃 A 含碳 $85.7\%$，含氢 $14.3\%$（质量分数），标准状况下该烃的密度为 $1.875\text{g/L}$。
（1）求 A 的分子式；
（2）若 A 能使酸性 $\text{KMnO}_4$ 溶液褪色，且与 $\text{HBr}$ 加成产物只有一种，写出 A 的结构简式。

## 八、综合计算

**22** 将 $2.3\text{g}$ 钠放入一定量的水中，反应后得到 $100\text{mL}$ 溶液。
（1）生成氢气在标准状况下的体积；
（2）所得 $\text{NaOH}$ 溶液的物质的量浓度；
（3）若将所得溶液加水稀释至 $500\text{mL}$，求稀释后溶液的 pH 值。

**23** 向 $100\text{mL}$ $0.1\text{mol/L}$ 的 $\text{AlCl}_3$ 溶液中逐滴加入 $0.5\text{mol/L}$ 的 $\text{NaOH}$ 溶液：
（1）当生成沉淀量最大时，需要 $\text{NaOH}$ 溶液的体积；
（2）当沉淀恰好完全溶解时，需要 $\text{NaOH}$ 溶液的体积。

**24** 将 $6.4\text{g}$ 铜与足量浓硝酸反应：$\text{Cu} + 4\text{HNO}_3(\text{浓}) \rightarrow \text{Cu(NO}_3)_2 + 2\text{NO}_2 \uparrow + 2\text{H}_2\text{O}$
（1）生成 $\text{NO}_2$ 在标准状况下的体积；
（2）若将生成的 $\text{NO}_2$ 全部溶于水配成 $500\text{mL}$ 溶液，求所得 $\text{HNO}_3$ 的物质的量浓度。

## 答案解析

### 1
**D**。熔融的KNO₃能导电，是电解质。铜是单质，蔗糖是非电解质，NaCl溶液是混合物。
### 2
**A**。第三周期第ⅦA族是氯（Cl）。
### 3
**C**。Fe + CuSO₄ → FeSO₄ + Cu，铁和铜的化合价发生变化。
### 4
**C**。NH₄Cl是强酸弱碱盐，水解显酸性，pH最小。
### 5
**B**。电解CuCl₂溶液，阳极Cl⁻放电产生Cl₂。
### 6
**C**。乙烯含有C=C双键，能使酸性KMnO₄褪色。
### 7
**B**。大理石和稀盐酸反应速率适中，便于控制。
### 8
**B**。各物质浓度不再变化是达到平衡的标志。
### 9
Cl、三、ⅦA

### 10
Al（1mol Al产生1.5mol H₂，最多）

### 11
减弱、增强
### 12
（1）2Fe + 3Cl₂ →点燃 2FeCl₃
（2）2Na + 2H₂O → 2NaOH + H₂↑
（3）2Al + 2NaOH + 2H₂O → 2NaAlO₂ + 3H₂↑
（4）MnO₂ + 4HCl(浓) →△ MnCl₂ + Cl₂↑ + 2H₂O

### 13
Na的物质的量 = 4.6/23 = 0.2mol
2Na + 2H₂O → 2NaOH + H₂↑
0.2mol Na产生H₂为0.1mol
V(H₂) = 0.1×22.4 = 2.24L

### 14
n(NaOH) = 0.5×0.1 = 0.05mol
m(NaOH) = 0.05×40 = 2.0g

### 15
n(CO₂) = 4.48/22.4 = 0.2mol
CaCO₃ + 2HCl → CaCl₂ + CO₂↑ + H₂O
n(CaCO₃) = n(CO₂) = 0.2mol
m(CaCO₃) = 0.2×100 = 20g

### 16
（1）平衡时 $n(\text{NH}_3) = 0.4 \times 2 = 0.8\text{mol}$
$\text{N}_2 + 3\text{H}_2 \rightleftharpoons 2\text{NH}_3$
起始(mol)：$1$，$3$，$0$
转化(mol)：$0.4$，$1.2$，$0.8$
平衡(mol)：$0.6$，$1.8$，$0.8$
$\text{N}_2$ 转化率 $= \frac{0.4}{1} \times 100\% = 40\%$
$\text{H}_2$ 转化率 $= \frac{1.2}{3} \times 100\% = 40\%$
（2）平衡浓度：$[\text{N}_2] = \frac{0.6}{2} = 0.3\text{mol/L}$，$[\text{H}_2] = \frac{1.8}{2} = 0.9\text{mol/L}$，$[\text{NH}_3] = 0.4\text{mol/L}$
$K = \frac{[\text{NH}_3]^2}{[\text{N}_2][\text{H}_2]^3} = \frac{0.4^2}{0.3 \times 0.9^3} = \frac{0.16}{0.2187} \approx 0.732$

### 17
（1）正反应放热，升高温度平衡向逆反应方向（吸热方向）移动
（2）正反应气体分子数减少，增大压强平衡向正反应方向移动
（3）通入 $\text{NO}_2$，反应物浓度增大，平衡正向移动。但 $\text{NO}_2$ 自身转化率降低。

### 18
（1）负极（Zn）：$\text{Zn} - 2e^- \rightarrow \text{Zn}^{2+}$
正极（Cu）：$\text{Cu}^{2+} + 2e^- \rightarrow \text{Cu}$
（2）$\Delta n(\text{Cu}^{2+}) = (0.1 - 0.05) \times 1 = 0.05\text{mol}$
转移电子 $n(e^-) = 2 \times 0.05 = 0.1\text{mol}$
$\Delta n(\text{Zn}) = 0.05\text{mol}$，$\Delta m(\text{Zn}) = 0.05 \times 65 = 3.25\text{g}$

### 19
（1）阳极：$2\text{H}_2\text{O} - 4e^- \rightarrow \text{O}_2 \uparrow + 4\text{H}^+$（或 $4\text{OH}^- - 4e^- \rightarrow \text{O}_2 \uparrow + 2\text{H}_2\text{O}$）
阴极：$\text{Ag}^+ + e^- \rightarrow \text{Ag}$
（2）$n(\text{O}_2) = \frac{0.02}{4} = 0.005\text{mol}$
$V(\text{O}_2) = 0.005 \times 22.4 = 0.112\text{L} = 112\text{mL}$

### 20
（1）A 为伯醇，可被氧化为羧酸：
A：$\text{CH}_3\text{CH}_2\text{CH}_2\text{CH}_2\text{OH}$（1-丁醇）
B：$\text{CH}_3\text{CH}_2\text{CH}_2\text{COOH}$（丁酸）
C：$\text{CH}_3\text{CH}_2\text{CH}=\text{CH}_2$（1-丁烯）
D：$\text{CH}_3\text{CH}_2\text{CHBrCH}_2\text{Br}$（1,2-二溴丁烷）
（2）$2\text{CH}_3\text{CH}_2\text{CH}_2\text{CH}_2\text{OH} + \text{O}_2 \xrightarrow{\text{Cu}/\Delta} 2\text{CH}_3\text{CH}_2\text{CH}_2\text{CHO} + 2\text{H}_2\text{O}$
$\text{CH}_3\text{CH}_2\text{CH}_2\text{CHO} \xrightarrow[\text{氧化}]{\text{KMnO}_4/\text{H}^+} \text{CH}_3\text{CH}_2\text{CH}_2\text{COOH}$
$\text{CH}_3\text{CH}_2\text{CH}_2\text{CH}_2\text{OH} \xrightarrow{\text{浓}\text{H}_2\text{SO}_4/\Delta} \text{CH}_3\text{CH}_2\text{CH}=\text{CH}_2 + \text{H}_2\text{O}$

### 21
（1）$M = 1.875 \times 22.4 = 42\text{g/mol}$
$n(\text{C}):n(\text{H}) = \frac{85.7}{12}:\frac{14.3}{1} = 1:2$，最简式 $\text{CH}_2$，式量 $14$
$42/14 = 3$，分子式 $\text{C}_3\text{H}_6$
（2）能使 $\text{KMnO}_4$ 褪色含不饱和键，与 $\text{HBr}$ 加成产物只有一种说明结构对称，A 为丙烯：$\text{CH}_3\text{CH}=\text{CH}_2$

### 22
（1）$n(\text{Na}) = \frac{2.3}{23} = 0.1\text{mol}$
$2\text{Na} + 2\text{H}_2\text{O} \rightarrow 2\text{NaOH} + \text{H}_2\uparrow$
$n(\text{H}_2) = 0.05\text{mol}$，$V(\text{H}_2) = 0.05 \times 22.4 = 1.12\text{L}$
（2）$n(\text{NaOH}) = 0.1\text{mol}$，$c = \frac{0.1}{0.1} = 1\text{mol/L}$
（3）稀释后 $c(\text{NaOH}) = \frac{0.1}{0.5} = 0.2\text{mol/L}$
$c(\text{H}^+) = \frac{10^{-14}}{0.2} = 5 \times 10^{-14}\text{mol/L}$，$\text{pH} = -\lg(5 \times 10^{-14}) = 13.3$

### 23
$n(\text{Al}^{3+}) = 0.1 \times 0.1 = 0.01\text{mol}$
（1）$\text{Al}^{3+} + 3\text{OH}^- \rightarrow \text{Al(OH)}_3\downarrow$
$n(\text{OH}^-) = 0.03\text{mol}$，$V(\text{NaOH}) = \frac{0.03}{0.5} = 0.06\text{L} = 60\text{mL}$
（2）$\text{Al(OH)}_3 + \text{OH}^- \rightarrow \text{AlO}_2^- + 2\text{H}_2\text{O}$
$n(\text{OH}^-)_{\text{总}} = 0.03 + 0.01 = 0.04\text{mol}$
$V(\text{NaOH})_{\text{总}} = \frac{0.04}{0.5} = 0.08\text{L} = 80\text{mL}$

### 24
$n(\text{Cu}) = \frac{6.4}{64} = 0.1\text{mol}$
（1）$\text{Cu} + 4\text{HNO}_3(\text{浓}) \rightarrow \text{Cu(NO}_3)_2 + 2\text{NO}_2 \uparrow + 2\text{H}_2\text{O}$
$n(\text{NO}_2) = 0.2\text{mol}$，$V(\text{NO}_2) = 0.2 \times 22.4 = 4.48\text{L}$
（2）$3\text{NO}_2 + \text{H}_2\text{O} \rightarrow 2\text{HNO}_3 + \text{NO}$
$n(\text{HNO}_3) = \frac{2}{3} \times 0.2 = \frac{0.4}{3}\text{mol}$
$c(\text{HNO}_3) = \frac{0.4/3}{0.5} \approx 0.267\text{mol/L}$
