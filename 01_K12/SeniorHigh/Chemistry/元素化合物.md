# 高中化学 · 元素化合物
> 涵盖：钠、氯、铁、铝、硫、氮、硅及其他重要元素及其化合物

---

## 一、钠及其化合物
### 1. 钠（Na）**物理性质**：银白色、质软、密度比水小、比煤油大，熔点低。
**化学性质**：
- 与 O₂ 反应：
  - 常温：4Na + O₂ → 2Na₂O（白色）
  - 加热：2Na + O₂ →△ Na₂O₂（淡黄色）
- 与水反应：2Na + 2H₂O → 2NaOH + H₂↑
  - 现象：浮（水面）、熔（成小球）、游（四处游动）、响（嘶嘶声）、红（酚酞变红）
- 与酸反应：2Na + 2HCl → 2NaCl + H₂↑
- 与乙醇反应：2Na + 2C₂H₅OH → 2C₂H₅ONa + H₂↑

**保存**：煤油中（隔绝空气和水）

### 2. 氧化钠（Na₂O）与过氧化钠（Na₂O₂）

| 性质 | Na₂O | Na₂O₂ |
|------|------|--------|
| 颜色 | 白色 | 淡黄 |
| 氧元素价 | -2 | -1 |
| 与水反应 | Na₂O + H₂O → 2NaOH | 2Na₂O₂ + 2H₂O → 4NaOH + O₂↑ |
| 与 CO₂ 反应 | Na₂O + CO₂ → Na₂CO₃ | 2Na₂O₂ + 2CO₂ → 2Na₂CO₃ + O₂↑ |

**Na₂O₂ 用途**：供氧剂、漂白剂、氧化剂

### 3. 碳酸钠（Na₂CO₃）与碳酸氢钠（NaHCO₃）

| 性质 | Na₂CO₃ | NaHCO₃ |
|------|---------|---------|
| 俗称 | 纯碱、苏打 | 小苏打 |
| 溶解 | 易溶 | 可溶（溶解度较小）|
| 热稳定 | 稳定 | 2NaHCO₃ →△ Na₂CO₃ + H₂O + CO₂↑ |
| 与盐酸 | 剧烈：Na₂CO₃ + 2HCl → 2NaCl + H₂O + CO₂↑ | 更剧烈（立即产生气体）|
| 与 NaOH | 不反应 | NaHCO₃ + NaOH → Na₂CO₃ + H₂O |
| 用途 | 玻璃、造纸、肥料 | 发酵粉、灭火器 |

**鉴别方法**：
- 加热：NaHCO₃ 放出使澄清石灰水变浑浊的气体
- 加入 CaCl₂：Na₂CO₃ 产生白色沉淀，NaHCO₃ 不反应。

## 二、氯及其化合物
### 1. 氯气（Cl₂）
**物理性质**：黄绿色、刺激性气味、有毒、密度比空气大。
**化学性质**：
- 与金属反应：
  - 2Na + Cl₂ →点燃 2NaCl（白烟）
  - Cu + Cl₂ →点燃 CuCl₂（棕黄色烟）
  - 2Fe + 3Cl₂ →点燃 2FeCl₃（棕褐色烟）
- 与非金属反应：H₂ + Cl₂ →点燃 2HCl（苍白色火焰，工业制盐酸）
- 与水反应：Cl₂ + H₂O → HCl + HClO（次氯酸，漂白性）
- 与碱反应：
  - Cl₂ + 2NaOH → NaCl + NaClO + H₂O（尾气吸收）
  - 2Cl₂ + 2Ca(OH)₂ → CaCl₂ + Ca(ClO)₂ + 2H₂O（制漂白粉）

**实验室制法**：MnO₂ + 4HCl(浓) →△ MnCl₂ + Cl₂↑ + 2H₂O

**氯水的成分**：Cl₂、HClO、H⁺、Cl⁻、ClO⁻、OH⁻、H₂O

### 2. 次氯酸（HClO）
- 弱酸（比 H₂CO₃ 还弱）
- 强氧化性：漂白、杀菌消毒
- 不稳定：2HClO →光照 2HCl + O₂↑

### 3. 氯离子的检验：Ag⁺ + Cl⁻ → AgCl↓（白色沉淀，不溶于稀硝酸）

## 三、铁及其化合物
### 1. 铁（Fe）**物理性质**：银白色、有磁性（可被磁铁吸引）
**化学性质**：
- 与 O₂：3Fe + 2O₂ →点燃 Fe₃O₄（火星四射）
- 与 S：Fe + S →△ FeS（黑色）
- 与 Cl₂：2Fe + 3Cl₂ →点燃 2FeCl₃
- 与水蒸气：3Fe + 4H₂O(g) →高温 Fe₃O₄ + 4H₂
- 与酸：
  - 稀 HCl/H₂SO₄：Fe + 2H⁺ → Fe²⁺ + H₂↑
  - 浓 H₂SO₄/HNO₃：常温钝化

### 2. 铁的氧化物
| 氧化物 | 颜色 | 铁元素价 | 性质 |
|--------|------|-----------|------|
| **FeO** | 黑色 | +2 | 碱性氧化物 |
| **Fe₂O₃** | 红棕 | +3 | 碱性氧化物，炼铁原料 |
| **Fe₃O₄** | 黑色 | +2,+3 | 磁性氧化物，有磁性 |

### 3. 铁的氢氧化物

| 氢氧化物 | 颜色 | 稳定性 | 与酸反应 |
|---------|------|--------|---------|
| **Fe(OH)₂** | 白色 | 4Fe(OH)₂ + O₂ + 2H₂O → 4Fe(OH)₃ | Fe(OH)₂ + 2H⁺ → Fe²⁺ + 2H₂O |
| **Fe(OH)₃** | 红褐 | 2Fe(OH)₃ →△ Fe₂O₃ + 3H₂O | Fe(OH)₃ + 3H⁺ → Fe³⁺ + 3H₂O |

**Fe(OH)₂ 制备**：需隔绝氧气（用煮沸过的水，液封，长滴管插入液面下）

### 4. Fe²⁺ 和 Fe³⁺ 的检验
| 方法 | Fe²⁺ | Fe³⁺ |
|------|-------|-------|
| 观察颜色 | 浅绿 | 棕黄 |
| NaOH 溶液 | 白色→灰绿→红褐 | 红褐色沉淀 |
| KSCN 溶液 | 无现象（加氧化剂后变红） | 血红色 |
| 先加 KSCN 再加氯水 | 无→血红色 |  |
| K₃[Fe(CN)₆]（铁氰化钾） | 蓝色沉淀 | 无现象 |

**Fe²⁺ 和 Fe³⁺ 的转化**：
- Fe²⁺ → Fe³⁺：加氧化剂（Cl₂、O₂、HNO₃、H₂O₂、KMnO₄）
- Fe³⁺ → Fe²⁺：加还原剂（Fe、Cu、I⁻、SO₂）

## 四、铝及其化合物
### 1. 铝（Al）**物理性质**：银白色、密度小、导电导热性好

**化学性质**：
- 与 O₂：4Al + 3O₂ → 2Al₂O₃（表面形成致密氧化膜）
- 与非氧化性酸：2Al + 6H⁺ → 2Al³⁺ + 3H₂↑
- 与强碱：2Al + 2NaOH + 2H₂O → 2NaAlO₂ + 3H₂↑
- 铝热反应：2Al + Fe₂O₃ →高温 Al₂O₃ + 2Fe（焊接铁轨）

### 2. 氧化铝（Al₂O₃）
- 白色固体，熔点很高
- **两性氧化物**：
  - 与酸：Al₂O₃ + 6H⁺ → 2Al³⁺ + 3H₂O
  - 与碱：Al₂O₃ + 2OH⁻ → 2AlO₂⁻ + H₂O

### 3. 氢氧化铝（Al(OH)₃）
- 白色胶状沉淀
- **两性氢氧化物**：
  - 与酸：Al(OH)₃ + 3H⁺ → Al³⁺ + 3H₂O
  - 与碱：Al(OH)₃ + OH⁻ → AlO₂⁻ + 2H₂O
- 不稳定：2Al(OH)₃ →△ Al₂O₃ + 3H₂O
- 制备：Al³⁺ + 3NH₃·H₂O → Al(OH)₃↓ + 3NH₄⁺（不用强碱）

### 4. 铝盐与偏铝酸盐
**Al³⁺ 和 AlO₂⁻ 的双水解**：Al³⁺ + 3AlO₂⁻ + 6H₂O → 4Al(OH)₃↓

**Al³⁺ → Al(OH)₃ → AlO₂⁻ 的转化**（两性特征）：
- Al³⁺ + 3OH⁻ → Al(OH)₃↓
- Al(OH)₃ + OH⁻ → AlO₂⁻ + 2H₂O

## 五、硫及其化合物
### 1. 硫（S）**物理性质**：黄色固体，不溶于水，微溶于乙醇，易溶于 CS₂。
**化学性质**：
- 与金属：2Na + S → Na₂S（研磨或加热）；Fe + S →△ FeS
- 与 O₂：S + O₂ →点燃 SO₂（蓝紫色火焰）
- 与 H₂：H₂ + S →△ H₂S

### 2. 二氧化硫（SO₂）
**物理性质**：无色、刺激性气味、密度比空气大、易液化

**化学性质**：
- **酸性氧化物**：SO₂ + H₂O → H₂SO₃（亚硫酸）
- **还原性**：2SO₂ + O₂ ⇌催化剂 2SO₃（工业制硫酸第一步）
- **漂白性**：使品红溶液褪色（可逆，加热恢复）
- **氧化性**：SO₂ + 2H₂S → 3S↓ + 2H₂O

**SO₂ 和 CO₂ 的鉴别**：
- 用品红溶液：SO₂ 使品红褪色
- 用溴水/酸性 KMnO₄：SO₂ 使溴水/KMnO₄ 褪色

### 3. 浓硫酸（H₂SO₄）
**三大特性**：
- **吸水性**：作干燥剂（不能干燥 NH₃、H₂S、HI）
- **脱水性**：蔗糖炭化（C₁₂H₂₂O₁₁ →浓H₂SO₄ 12C + 11H₂O）
- **强氧化性**：
  - Cu + 2H₂SO₄(浓) →△ CuSO₄ + SO₂↑ + 2H₂O
  - C + 2H₂SO₄(浓) →△ CO₂↑ + 2SO₂↑ + 2H₂O

**常温钝化**：Fe、Al 在浓 H₂SO₄ 中钝化（致密氧化膜）

## 六、氮及其化合物
### 1. 氮气（N₂）
- 无色无味气体，占空气 78%
- 结构稳定（N≡N 三键），化学性质稳定
- N₂ + 3H₂ ⇌高温高压催化剂 2NH₃（工业合成氨）

### 2. 氨气（NH₃）
**物理性质**：无色、刺激性气味、比空气轻、极易溶于水（1:700）。
**化学性质**：
- **与水反应**：NH₃ + H₂O → NH₃·H₂O → NH₄⁺ + OH⁻（碱性）
- **与酸反应**：NH₃ + HCl → NH₄Cl（白烟，检验 NH₃）
- **还原性**：
  - 4NH₃ + 5O₂ →催化剂△ 4NO + 6H₂O（催化氧化，工业制硝酸第一步）
  - 2NH₃ + 3CuO →△ N₂ + 3Cu + 3H₂O

**实验室制法**：2NH₄Cl + Ca(OH)₂ →△ CaCl₂ + 2NH₃↑ + 2H₂O

**喷泉实验**：利用 NH₃ 极易溶于水，形成压强差。
### 3. 硝酸（HNO₃）
**物理性质**：无色液体，易挥发，有刺激性气味。
**化学性质**：
- **不稳定性**：4HNO₃ →光照△ 4NO₂↑ + O₂↑ + 2H₂O（避光保存）
- **强氧化性**：
  - Cu + 4HNO₃(浓) → Cu(NO₃)₂ + 2NO₂↑ + 2H₂O
  - 3Cu + 8HNO₃(稀) → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O
  - C + 4HNO₃(浓) →△ CO₂↑ + 4NO₂↑ + 2H₂O
- **钝化**：Fe、Al 在浓 HNO₃ 中钝化
**王水**：浓 HNO₃ : 浓 HCl = 1:3，可溶解金、铂

### 4. 铵盐
- 都易溶于水
- 受热分解：NH₄Cl →△ NH₃↑ + HCl↑
- 与碱反应：NH₄⁺ + OH⁻ →△ NH₃↑ + H₂O（检验 NH₄⁺）
### 5. 氮的氧化物
- NO：无色气体，有毒
- NO₂：红棕色气体，有毒
- 3NO₂ + H₂O → 2HNO₃ + NO（工业制硝酸第三步）

## 七、硅及其化合物
### 1. 硅（Si）
- 灰黑色固体，类金属
- 导电性介于导体与绝缘体之间（半导体）
- 用途：芯片、太阳能电池（高纯硅）
### 2. 二氧化硅（SiO₂）
- 不溶于水（与 CO₂ 不同）
- 与 HF 反应：SiO₂ + 4HF → SiF₄↑ + 2H₂O（雕刻玻璃）
- 与强碱：SiO₂ + 2NaOH → Na₂SiO₃ + H₂O（因此不能用磨口玻璃塞装碱液）
### 3. 硅酸钠（Na₂SiO₃）
- 俗称泡花碱，水溶液为水玻璃
- 防火材料、黏合剂

## 八、常见元素及其化合物转化关系

### 钠的转化：Na → Na₂O → Na₂O₂ → NaOH → Na₂CO₃ → NaHCO₃
### 铁的转化：Fe → FeO/Fe₂O₃/Fe₃O₄ → Fe²⁺ →氧化→还原→ Fe³⁺
### 铝的转化：Al → Al₂O₃ → Al(OH)₃ → NaAlO₂ → Al³⁺ →OH⁻/H⁺ → Al(OH)₃ →OH⁻/H⁺ → AlO₂⁻

### 硫的转化：H₂S → S → SO₂ → SO₃ → H₂SO₄ → CuSO₄
### 氮的转化：N₂ → NH₃ → NO → NO₂ → HNO₃ → Cu(NO₃)₂
