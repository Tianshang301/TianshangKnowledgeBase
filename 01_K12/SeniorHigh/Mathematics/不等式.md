# 高中数学 · 不等式
> 涵盖：不等关系与性质、比较大小、一元二次不等式、分式不等式、绝对值不等式、基本不等式（均值不等式）及其推广、柯西不等式、排序不等式、不等式的证明方法、线性规划

## 一、不等关系与不等式的性质

### 1. 实数比较大小原理

$a - b > 0 \Leftrightarrow a > b$
$a - b = 0 \Leftrightarrow a = b$
$a - b < 0 \Leftrightarrow a < b$

### 2. 不等式的基本性质

| 性质 | 内容 | 说明 |
|------|------|------|
| **对称性** | $a > b \Leftrightarrow b < a$ | 方向可互换 |
| **传递性** | $a > b$ 且 $b > c \Rightarrow a > c$ | 同向可传递 |
| **可加性** | $a > b \Rightarrow a + c > b + c$ | 同加一个数不变 |
| **可乘性** | $a > b, c > 0 \Rightarrow ac > bc$；$a > b, c < 0 \Rightarrow ac < bc$ | 乘负数要变号 |
| **同向可加** | $a > b, c > d \Rightarrow a + c > b + d$ | 同向不等式相加 |
| **同向同正可乘** | $a > b > 0, c > d > 0 \Rightarrow ac > bd$ | 正数同向可乘 |
| **可乘方性** | $a > b > 0 \Rightarrow a^n > b^n\ (n \in \mathbb{N}^*)$ | 正数可乘方 |
| **可开方性** | $a > b > 0 \Rightarrow \sqrt[n]{a} > \sqrt[n]{b}\ (n \in \mathbb{N}^*)$ | 正数可开方 |
| **倒数性质** | $a > b, ab > 0 \Rightarrow \frac{1}{a} < \frac{1}{b}$ | 同号取倒要变号 |

### 3. 比较大小的方法
| 方法 | 操作 | 适用 |
|------|------|------|
| **作差法** | $a - b$ 与 $0$ 比较 | 一切实数 |
| **作商法** | $\frac{a}{b}$ 与 $1$ 比较（需 $b > 0$） | 正数比较 |
| **中间量法** | 通过中间值（$0, 1$ 等）传递比较 | 间接比较 |
| **函数法** | 构造函数，利用单调性 | 含变量表达式 |
| **放缩法** | 适当放大或缩小后比较 | 不等式证明 |

## 二、一元二次不等式

### 1. 标准形式
$ax^2 + bx + c > 0$（$\ge 0$，$< 0$，$\le 0$），$a \neq 0$

### 2. 解法（数形结合）

二次函数 $y = ax^2 + bx + c$ 的图像与 $x$ 轴的交点决定解集。
| 判别式 $\Delta = b^2 - 4ac$ | $a > 0$ 的图像 | $ax^2 + bx + c > 0$ 解集 | $ax^2 + bx + c < 0$ 解集 |
|-----------------------------|---------------|--------------------------|--------------------------|
| $\Delta > 0$ | 与 $x$ 轴有两个交点 $x_1, x_2\ (x_1 < x_2)$ | $\{x \mid x < x_1 \text{ 或 } x > x_2\}$ | $\{x \mid x_1 < x < x_2\}$ |
| $\Delta = 0$ | 与 $x$ 轴相切于 $x_0 = -\frac{b}{2a}$ | $\{x \mid x \neq x_0\}$ | $\varnothing$ |
| $\Delta < 0$ | 与 $x$ 轴无交点 | $\mathbb{R}$ | $\varnothing$ |

**注意**：若 $a < 0$，不等式两边同乘 $-1$ 转化为 $a > 0$ 情形

### 3. 含参一元二次不等式

**分类讨论标准**：
1. 讨论二次项系数 $a$：$a = 0$（退化为一次）、$a > 0$、$a < 0$
2. 讨论判别式 $\Delta$：$\Delta > 0$、$\Delta = 0$、$\Delta < 0$
3. 讨论两根大小（当两根含参时）：$x_1$ 与 $x_2$ 的大小比较
**常见题型**
- 解含参数的不等式：$x^2 - (a+1)x + a > 0$
- 恒成立求参：$\forall x \in \mathbb{R}$，$ax^2 + bx + c > 0$ 恒成立
### 4. 一元二次不等式恒成立条件
| 条件 | $ax^2 + bx + c > 0$ 恒成立 | $ax^2 + bx + c < 0$ 恒成立 |
|------|----------------------------|----------------------------|
| $a = 0$ | $b = 0, c > 0$ | $b = 0, c < 0$ |
| $a \neq 0$ | $a > 0$ 且 $\Delta < 0$ | $a < 0$ 且 $\Delta < 0$ |

## 三、分式不等式

### 1. 标准形式
$\frac{f(x)}{g(x)} > 0$（$\ge 0$，$< 0$，$\le 0$）
### 2. 解法
分式不等式化为整式不等式求解，注意分母不为零。
- $\frac{f(x)}{g(x)} > 0 \Leftrightarrow f(x) \cdot g(x) > 0$
- $\frac{f(x)}{g(x)} \ge 0 \Leftrightarrow f(x) \cdot g(x) \ge 0$ 且 $g(x) \neq 0$
- $\frac{f(x)}{g(x)} < 0 \Leftrightarrow f(x) \cdot g(x) < 0$
- $\frac{f(x)}{g(x)} \le 0 \Leftrightarrow f(x) \cdot g(x) \le 0$ 且 $g(x) \neq 0$

### 3. 常见陷阱
- 去分母时不要直接乘以分母，因为分母符号不确定
- 一定要排除使分母为零的 $x$ 值

## 四、绝对值不等式

### 1. 基本形式

| 不等式 | 解集 |
|--------|------|
| $|x| < a\ (a > 0)$ | $-a < x < a$ |
| $|x| > a\ (a > 0)$ | $x < -a$ 或 $x > a$ |
| $|x| < a\ (a \le 0)$ | $\varnothing$ |
| $|x| > a\ (a < 0)$ | $\mathbb{R}$ |

### 2. 一般形式的解法

| 类型 | 方法 |
|------|------|
| $|f(x)| < g(x)$ | $-g(x) < f(x) < g(x)$ |
| $|f(x)| > g(x)$ | $f(x) < -g(x)$ 或 $f(x) > g(x)$ |
| $|f(x)| < |g(x)|$ | 两边平方：$f^2(x) < g^2(x)$ |
| $|f(x)| > |g(x)|$ | 两边平方：$f^2(x) > g^2(x)$ |

### 3. 含多个绝对值
**零点分段法**：
1. 找零点（每个绝对值为零的点）
2. 分段讨论（去掉绝对值符号）
3. 各段解集取并集
**几何意义**
- $|x - a|$ 表示数轴上 $x$ 到 $a$ 的距离
- $|x - a| + |x - b|$ 表示 $x$ 到 $a$ 和 $b$ 的距离之和
### 4. 三角不等式
$||a| - |b|| \le |a \pm b| \le |a| + |b|$

## 五、基本不等式（均值不等式）
### 1. 基本形式
若 $a, b > 0$，则 $\frac{a + b}{2} \ge \sqrt{ab}$，当且仅当 $a = b$ 时取等。
**推论**
- $a + b \ge 2\sqrt{ab}$（和积关系，和为定值积最大，积为定值和最小）
- $ab \le \left[\frac{a + b}{2}\right]^2$

### 2. 三个正数的均值不等式
若 $a, b, c > 0$，则 $\frac{a + b + c}{3} \ge \sqrt[3]{abc}$，当且仅当 $a = b = c$ 时取等。
### 3. 推广：$n$ 个正数的均值不等式
若 $a_1, a_2, \dots, a_n > 0$，则 $\frac{a_1 + a_2 + \dots + a_n}{n} \ge \sqrt[n]{a_1 a_2 \dots a_n}$

### 4. 常用变形

| 变形 | 公式 |
|------|------|
| 平方平均 $\ge$ 算术平均 | $\sqrt{\frac{a^2 + b^2}{2}} \ge \frac{a + b}{2}$ |
| 算术平均 $\ge$ 几何平均 | $\frac{a + b}{2} \ge \sqrt{ab}$ |
| 几何平均 $\ge$ 调和平均 | $\sqrt{ab} \ge \frac{2}{\frac{1}{a} + \frac{1}{b}}$ |
| 链式 | $\sqrt{\frac{a^2 + b^2}{2}} \ge \frac{a+b}{2} \ge \sqrt{ab} \ge \frac{2}{\frac{1}{a}+\frac{1}{b}}$ |

**记忆**：平方 $\ge$ 算术 $\ge$ 几何 $\ge$ 调和

### 5. 应用技巧
**一正二定三相等**：
1. **一正**：各项均为正数
2. **二定**：和或积为定值
3. **三相等**：验证等号成立条件
**常见技巧**
- 拆项：$a + \frac{1}{a}\ (a > 0) \ge 2$
- 凑项：将表达式配成能够使用均值不等式的形式
- $1$ 的代换：已知 $ax + by = 1$，求 $\frac{1}{x} + \frac{1}{y}$ 最小值
- 换元法：引入新变量简化问题
- 构造不等式：利用已知条件构造均值不等式

**典型例题**
- 已知 $x > 0$，求 $x + \frac{1}{x}$ 的最小值 $\to 2$
- 已知 $x > 0$，$y > 0$，且 $x + y = 1$，求 $\frac{1}{x} + \frac{1}{y}$ 的最小值 $\to 4$
- 已知 $x > 2$，求 $x + \frac{1}{x-2}$ 的最小值 $\to 4$

## 六、柯西不等式

### 1. 二维形式
$(a^2 + b^2)(c^2 + d^2) \ge (ac + bd)^2$

当且仅当 $\frac{a}{c} = \frac{b}{d}$ 时取等。
### 2. 三维形式
$(a^2 + b^2 + c^2)(x^2 + y^2 + z^2) \ge (ax + by + cz)^2$

### 3. 一般形式
$(\sum a_i^2)(\sum b_i^2) \ge (\sum a_i b_i)^2$

### 4. 向量形式
$|\alpha \cdot \beta| \le |\alpha| \cdot |\beta|$

### 5. 应用
- 求最值：已知 $a^2 + b^2 = 1$，求 $a + 2b$ 最大值 $\to \sqrt{1^2+2^2} = \sqrt{5}$
- 证明不等式

## 七、排序不等式

**顺序和 $\ge$ 乱序和 $\ge$ 逆序和**

设 $a_1 \le a_2 \le \dots \le a_n$，$b_1 \le b_2 \le \dots \le b_n$（均为实数）：
- 顺序和：$a_1 b_1 + a_2 b_2 + \dots + a_n b_n$
- 乱序和：$a_1 b_{k_1} + a_2 b_{k_2} + \dots + a_n b_{k_n}$
- 逆序和：$a_1 b_n + a_2 b_{n-1} + \dots + a_n b_1$
有：逆序和 $\le$ 乱序和 $\le$ 顺序和

## 八、不等式的证明方法
### 1. 综合法
从已知条件出发，利用不等式的性质逐步推导到结论。思路：由因导果。
### 2. 分析法
从结论出发，逐步寻找使结论成立的充分条件，直到得到一个显然成立的条件。思路：执果索因。
### 3. 反证法
假设结论不成立，推出与已知条件矛盾的结论，从而证明原结论成立。
**步骤**：
1. 假设结论的反面成立
2. 从假设出发推导矛盾
3. 说明原结论成立
### 4. 放缩法
适当放大或缩小不等式的一边，从而证明不等式。
**常用放缩**
- 舍去或添加正（负）项
- 利用不等式性质：$a > b > 0 \Rightarrow \frac{1}{a} < \frac{1}{b}$
- 利用基本不等式：$\sqrt{ab} \le \frac{a + b}{2}$
- 利用函数单调性
- 常见放缩：$\frac{1}{n^2} < \frac{1}{n-1} - \frac{1}{n}$

### 5. 数学归纳法
用于证明与自然数 $n$ 有关的不等式。
**步骤**：
1. 验证 $n = 1$（或起始值）成立
2. 假设 $n = k$ 成立，证明 $n = k + 1$ 成立
3. 由数学归纳法原理，命题对一切自然数成立

### 6. 换元法
通过变量替换简化不等式，常用：
- 三角换元：$x = \sin \theta$，$x = \tan \theta$ 等
- 整体换元

### 7. 构造函数法
构造辅助函数，利用函数单调性、极值等证明不等式。

## 九、线性规划
### 1. 基本概念

| 概念 | 定义 |
|------|------|
| **二元一次不等式** | $ax + by + c > 0$（或 $< 0$，$\ge 0$，$\le 0$） |
| **可行解** | 满足所有约束条件的 $(x, y)$ |
| **可行域** | 所有可行解构成的平面区域 |
| **目标函数** | 要求最值的函数，通常为 $z = ax + by$ |
| **最优解** | 使目标函数取得最值的可行解 |

### 2. 二元一次不等式表示的区域
$Ax + By + C > 0$ 表示直线 $Ax + By + C = 0$ 的某一侧平面区域。
**判断方法**：
1. 直线定界（画直线，注意虚实）
2. 特殊点定域（通常用原点 $(0,0)$ 测试）
3. 若原点代入使不等式成立，则含原点的一侧为可行域
### 3. 线性规划的解法步骤

1. **画**：画出约束条件对应的可行域
2. **移**：平移目标函数的等值线 $z = ax + by$
3. **求**：在边界交点处求最优解
4. **答**：写出答案
### 4. 目标函数的常见类型
| 类型 | 形式 | 几何意义 |
|------|------|----------|
| **截距型** | $z = ax + by$ | 直线的纵截距 |
| **距离型** | $z = (x-a)^2 + (y-b)^2$ | 点到点的距离平方 |
| **斜率型** | $z = \frac{y-b}{x-a}$ | 点与点连线的斜率 |
| **绝对值型** | $z = |ax + by + c|$ | 点到直线的距离 |

### 5. 线性规划常见题型
- 求目标函数的最大（小）值（截距型）
- 求可行域的面积
- 含参线性规划（目标函数含参或约束条件含参）
- 非线性目标函数（距离、斜率）
- 整数规划（最优解为整数点）
- 实际应用（生产安排、资源配置）

### 6. 实际应用

**步骤**：
1. 设变量
2. 列约束条件（线性不等式组）
3. 写目标函数（线性函数）
4. 求解线性规划问题
5. 结合实际回答问题

**经典问题**：产品配比、材料切割、运输调度、营养搭配
