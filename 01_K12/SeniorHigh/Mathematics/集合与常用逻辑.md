# 高中数学 · 集合与常用逻辑

## 一、集合
### 1. 集合的概念
**定义**：指定的某些对象的全体称为集合，集合中的每个对象称为元素。
**元素的三大特性**

| 特性 | 说明 |
|------|------|
| **确定性** | 给定集合，元素必须确定，不能模棱两可 |
| **互异性** | 同一集合中的元素互不相同，不能重复 |
| **无序性** | 集合中的元素没有顺序之分 |

**元素与集合的关系**
- $a \in A$：$a$ 是集合 $A$ 的元素
- $a \notin A$：$a$ 不是集合 $A$ 的元素
**常见的数集及其符号**

| 数集 | 符号 | 含义 |
|------|------|------|
| 自然数集 | $\mathbb{N}$ | $0, 1, 2, 3, \dots$ |
| 正整数集 | $\mathbb{N}^*$ 或 $\mathbb{N}^+$ | $1, 2, 3, \dots$ |
| 整数集 | $\mathbb{Z}$ | $\dots, -2, -1, 0, 1, 2, \dots$ |
| 有理数集 | $\mathbb{Q}$ | 分数、整数、有限小数、无限循环小数 |
| 实数集 | $\mathbb{R}$ | 有理数 $+$ 无理数 |
| 复数集 | $\mathbb{C}$ | $a + bi\ (a,b \in \mathbb{R})$ |
| 空集 | $\varnothing$ | 不含任何元素的集合 |

### 2. 集合的表示法

| 方法 | 说明 | 示例 |
|------|------|------|
| **列举法** | 将元素一一列举，用 $\{\}$ 括起 | $\{1, 2, 3, 4, 5\}$ |
| **描述法** | $\{\text{代表元素} \mid \text{元素满足的条件}\}$ | $\{x \mid x > 0, x \in \mathbb{R}\}$ |
| **图示法** | Venn 图（韦恩图） | 用封闭曲线表示集合 |
| **区间法** | 数轴上的区间 | $(a, b)$, $[a, b]$, $(a, +\infty)$ |

**描述法注意**
- 竖线前是代表元素，竖线后是元素满足的条件
- 代表元素的取值范围一般要说明（如 $x \in \mathbb{R}$）
- 条件要准确、无歧义

### 3. 集合的关系
| 关系 | 定义 | 记法 | Venn 图 |
|------|------|------|---------|
| **子集** | $A$ 的所有元素都属于 $B$ | $A \subseteq B$ | $A$ 在 $B$ 内部 |
| **真子集** | $A \subseteq B$ 且 $A \neq B$ | $A \subset B$ | $A$ 在 $B$ 内部且不相等 |
| **相等** | 互相为子集（$A \subseteq B$ 且 $B \subseteq A$） | $A = B$ | 两圆重合 |
| **不包含** | $A$ 不是 $B$ 的子集 | $A \not\subseteq B$ | |

**重要结论**
- $\varnothing$ 是任何集合的子集：$\varnothing \subseteq A$
- $\varnothing$ 是任何非空集合的真子集
- $A \subseteq A$（任何集合都是自身的子集）
- 若 $A \subseteq B$ 且 $B \subseteq C$，则 $A \subseteq C$（传递性）
- 有 $n$ 个元素的集合，其子集个数为 $2^n$，真子集 $2^n-1$，非空真子集 $2^n-2$

### 4. 集合的运算
#### (1) 交集
$A \cap B = \{x \mid x \in A \text{ 且 } x \in B\}$

性质
- $A \cap A = A$
- $A \cap \varnothing = \varnothing$
- $A \cap B = B \cap A$
- $A \cap B \subseteq A$，$A \cap B \subseteq B$
- $(A \cap B) \cap C = A \cap (B \cap C)$

#### (2) 并集
$A \cup B = \{x \mid x \in A \text{ 或 } x \in B\}$

性质
- $A \cup A = A$
- $A \cup \varnothing = A$
- $A \cup B = B \cup A$
- $A \subseteq A \cup B$，$B \subseteq A \cup B$
- $(A \cup B) \cup C = A \cup (B \cup C)$
- $A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$
- $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$

#### (3) 补集
设全集为 $U$，则 $A$ 的补集：$\complement_U A = \{x \mid x \in U \text{ 且 } x \notin A\}$

性质
- $A \cup (\complement_U A) = U$
- $A \cap (\complement_U A) = \varnothing$
- $\complement_U (\complement_U A) = A$
- $\complement_U (A \cup B) = (\complement_U A) \cap (\complement_U B)$
- $\complement_U (A \cap B) = (\complement_U A) \cup (\complement_U B)$

#### (4) 全集
包含所有研究元素的集合，记作 $U$。
### 5. 集合的运算律

| 运算 | 公式 |
|------|------|
| **交换律** | $A \cap B = B \cap A$；$A \cup B = B \cup A$ |
| **结合律** | $(A \cap B) \cap C = A \cap (B \cap C)$；$(A \cup B) \cup C = A \cup (B \cup C)$ |
| **分配律** | $A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$；$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$ |
| **吸收律** | $A \cap (A \cup B) = A$；$A \cup (A \cap B) = A$ |
| **德摩根律** | $\complement_U (A \cup B) = (\complement_U A) \cap (\complement_U B)$；$\complement_U (A \cap B) = (\complement_U A) \cup (\complement_U B)$ |
| **双重否定** | $\complement_U (\complement_U A) = A$ |
| **同一律** | $A \cap U = A$；$A \cup \varnothing = A$ |
| **零一律** | $A \cap \varnothing = \varnothing$；$A \cup U = U$ |
| **补余律** | $A \cap \complement_U A = \varnothing$；$A \cup \complement_U A = U$ |

### 6. 集合中元素的个数（容斥原理）

**有限集**：含有有限个元素的集合
**$\mathrm{Card}(A)$**：集合 $A$ 中元素的个数。
**容斥原理**
- 二集合：$\mathrm{Card}(A \cup B) = \mathrm{Card}(A) + \mathrm{Card}(B) - \mathrm{Card}(A \cap B)$
- 三集合：$\mathrm{Card}(A \cup B \cup C) = \mathrm{Card}(A) + \mathrm{Card}(B) + \mathrm{Card}(C) - \mathrm{Card}(A \cap B) - \mathrm{Card}(A \cap C) - \mathrm{Card}(B \cap C) + \mathrm{Card}(A \cap B \cap C)$

### 7. 集合常见题型与方法
#### (1) 集合运算中的含参问题
已知集合运算结果求参数值或取值范围：
- 利用数轴分析（实数集）
- 利用 Venn 图分析
- 利用方程或不等式求解
- **特别注意空集**：$A \cap B = \varnothing$ 时，要考虑 $A = \varnothing$ 或 $B = \varnothing$ 的情形
#### (2) 集合中的创新题
- 新定义集合（差集、对称差集等）
- 集合与其它知识点的综合
- 集合新运算

## 二、常用逻辑用语

### 1. 命题

**定义**：可以判断真假的陈述句称为命题。
- 真命题：判断为真的命题
- 假命题：判断为假的命题
**命题的结构**：若 $p$，则 $q$（$p$ 是条件，$q$ 是结论）

**注意**
- 疑问句、祈使句、感叹句都不是命题
- 陈述句不一定都是命题（"$x > 3$" 不是命题，因为无法判断真假）
- 含有变量的语句通常不是命题（开语句）
### 2. 四种命题

| 名称 | 形式 | 举例 |
|------|------|------|
| **原命题** | 若 $p$，则 $q$ | 若 $x > 0$，则 $x^2 > 0$ |
| **逆命题** | 若 $q$，则 $p$ | 若 $x^2 > 0$，则 $x > 0$ |
| **否命题** | 若 $\lnot p$，则 $\lnot q$ | 若 $x \le 0$，则 $x^2 \le 0$ |
| **逆否命题** | 若 $\lnot q$，则 $\lnot p$ | 若 $x^2 \le 0$，则 $x \le 0$ |

**四种命题的关系**
```
原命题 ←→ 逆命题      否命题 ←→ 逆否命题
```

箭头表示互为逆否（等价关系）

**等价关系（重要）**
- 原命题 $\Leftrightarrow$ 逆否命题（同真同假）
- 逆命题 $\Leftrightarrow$ 否命题（同真同假）
- 原命题与逆命题、否命题之间 **没有** 必然的真假关系
**利用等价命题**：当直接证明原命题困难时，可以证明其逆否命题

### 3. 充分条件与必要条件
#### (1) 定义
若 $p \Rightarrow q$，则 $p$ 是 $q$ 的 **充分条件**，$q$ 是 $p$ 的 **必要条件**。若 $p \Leftrightarrow q$，则 $p$ 是 $q$ 的 **充分必要条件**（充要条件）。
#### (2) 四种关系的判定
| 条件关系 | 逻辑含义 | 集合视角（$p$ 对应 $A$，$q$ 对应 $B$） |
|----------|----------|--------------------------------------|
| **充分不必要** | $p \Rightarrow q$ 且 $q \nRightarrow p$ | $A \subset B$（$A$ 是 $B$ 的真子集） |
| **必要不充分** | $q \Rightarrow p$ 且 $p \nRightarrow q$ | $B \subset A$（$B$ 是 $A$ 的真子集） |
| **充要条件** | $p \Leftrightarrow q$ | $A = B$ |
| **既不充分也不必要** | $p \nRightarrow q$ 且 $q \nRightarrow p$ | $A \not\subseteq B$ 且 $B \not\subseteq A$ |

#### (3) 判断方法
1. **定义法**：判断 $p \Rightarrow q$ 和 $q \Rightarrow p$ 是否成立
2. **集合法**：将 $p$、$q$ 对应为集合 $A$、$B$，看包含关系
3. **等价法**：利用等价命题转化判断
4. **传递法**：$p \Rightarrow q$ 且 $q \Rightarrow r$，则 $p \Rightarrow r$

**常用逆向思维**
- $\lnot p$ 是 $\lnot q$ 的充分条件 $\Rightarrow$ $q$ 是 $p$ 的充分条件（逆否命题）
- 问 $p$ 是 $q$ 的什么条件转化为看"$q$ 是 $p$ 的什么条件"

### 4. 逻辑联结词
| 联结词 | 符号 | 含义 | 真假判定 |
|--------|------|------|----------|
| **且** | $\land$ | $p \land q$（$p$ 和 $q$ 同时成立） | 一假即假 |
| **或** | $\lor$ | $p \lor q$（至少一个成立） | 一真即真 |
| **非** | $\lnot$ | $\lnot p$（否定 $p$） | 真假相反 |

**真值表**

| $p$ | $q$ | $p \land q$ | $p \lor q$ | $\lnot p$ |
|-----|-----|-------------|------------|-----------|
| 真 | 真 | 真 | 真 | 假 |
| 真 | 假 | 假 | 真 | 假 |
| 假 | 真 | 假 | 真 | 真 |
| 假 | 假 | 假 | 假 | 真 |

**记忆口诀**
- 且：同真才真，一假即假
- 或：同假才假，一真即真
- 非：真变假，假变真
**含有逻辑联结词的命题的真假判断步骤**：
1. 判断每个简单命题的真假
2. 根据联结词的真值表确定复合命题的真假
### 5. 全称量词与存在量词
#### (1) 全称量词
**常见词**：所有的、任意一个、一切、每一个、任何、$\forall$

**全称命题**：$\forall x \in M, p(x)$（对 $M$ 中任意 $x$，$p(x)$ 成立）
**真假判定**
- 真：对 $M$ 中每个 $x$，$p(x)$ 都成立
- 假：存在一个 $x_0 \in M$ 使 $p(x_0)$ 不成立（举反例）

#### (2) 存在量词
**常见词**：存在一个、至少有一个、有些、有的、$\exists$

**特称命题**：$\exists x_0 \in M, p(x_0)$（存在 $M$ 中的 $x_0$，使 $p(x_0)$ 成立）
**真假判定**
- 真：至少存在一个 $x_0 \in M$ 使 $p(x_0)$ 成立
- 假：对 $M$ 中所有 $x$，$p(x)$ 都不成立

#### (3) 含有一个量词的命题的否定
| 原命题 | 否定 |
|--------|------|
| $\forall x \in M, p(x)$ | $\exists x_0 \in M, \lnot p(x_0)$ |
| $\exists x_0 \in M, p(x_0)$ | $\forall x \in M, \lnot p(x)$ |

**口诀**：全称变特称，特称变全称，结论取否定

**常见错误**
- 否定时忘记改变量词（$\forall$ 和 $\exists$）
- 否定时只否定部分，未否定全部结论
- 否定"且"和"或"时没有使用德摩根律
#### (4) 含参量词问题
- 恒成立问题：$\forall x \in D, f(x) > 0$（转化为求最小值 $> 0$）
- 存在性问题：$\exists x \in D, f(x) > 0$（转化为求最大值 $> 0$）
- 注意参数与变量的区别

## 三、高频考点与典型题

### 1. 集合运算综合题
**题型**：已知集合 $A = \{x \mid x^2 - 3x + 2 \le 0\}$，$B = \{x \mid x^2 - (a+1)x + a \le 0\}$，求 $A \cup B$
**思路**：先解不等式化简集合，再按参数 $a$ 分段讨论

### 2. 集合关系中的含参讨论
**题型**：$A = \{x \mid -2 \le x \le 5\}$，$B = \{x \mid m+1 \le x \le 2m-1\}$，$B \subseteq A$，求 $m$ 范围
**思路**：分类讨论 $B = \varnothing$ 和 $B \neq \varnothing$ 两种情况

### 3. 充要条件的证明
**题型**：证明 $p$ 是 $q$ 的充要条件
**思路**：
1. 必要性：$q \Rightarrow p$
2. 充分性：$p \Rightarrow q$
3. 两者均需证明，缺一不可

### 4. 量词命题求参数范围
**题型**：$\forall x \in [1, 2]$，$x^2 - ax + 1 > 0$ 恒成立，求 $a$ 范围
**思路**：分离参数 $a < x + \frac{1}{x}$ 在 $[1,2]$ 上恒成立 $\Rightarrow a < (x + \frac{1}{x})_{\min}$
