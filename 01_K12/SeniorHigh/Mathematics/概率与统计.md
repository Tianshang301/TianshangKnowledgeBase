# 高中数学 · 概率与统计
> 涵盖：计数原理、排列组合、二项式定理、概率、随机变量及其分布、统计

---

## 一、计数原理
### 1. 分类加法计数原理
完成一件事有 $n$ 类不同方案，第 $i$ 类方案有 $m_i$ 种方法，则完成这件事共有：$N = m_1 + m_2 + \dots + m_n$ 种方法。
**关键词**："或"、"任选一类" $\to$ 用加法
### 2. 分步乘法计数原理
完成一件事需要 $n$ 个步骤，第 $i$ 个步骤有 $m_i$ 种方法，则完成这件事共有：$N = m_1 \times m_2 \times \dots \times m_n$ 种方法。
**关键词**："并且"、"先后" $\to$ 用乘法
**分类与分步的区别**
- 分类：每一类都能独立完成任务
- 分步：每一步都不能独立完成任务，需所有步骤配合

## 二、排列
### 1. 排列的定义
从 $n$ 个不同元素中取出 $m$ 个（$m \le n$），按照一定的顺序排成一列，称为从 $n$ 个不同元素中取出 $m$ 个的一个排列。
### 2. 排列数公式
$A_n^m = P_n^m = \frac{n!}{(n - m)!} = n \times (n-1) \times (n-2) \times \dots \times (n-m+1)$

**特殊情形**
- $A_n^n = n!$（全排列）
- $A_n^1 = n$
- $A_n^0 = 1$

### 3. 阶乘
$n! = n \times (n-1) \times (n-2) \times \dots \times 2 \times 1$
$0! = 1$（规定）

### 4. 排列问题的常见约束类型
| 约束类型 | 解题策略 |
|----------|----------|
| **相邻问题** | 捆绑法（将相邻元素视为一个整体，再考虑内部排列） |
| **不相邻问题** | 插空法（先排其他元素，再将不相邻元素插入空位） |
| **定序问题** | 总排列数除以定序元素的全排列 |
| **含特殊元素** | 优先考虑特殊元素的位置 |
| **含特殊位置** | 优先满足特殊位置的限制条件 |
| **数字排列** | 注意 $0$ 不能作首位 |

## 三、组合
### 1. 组合的定义
从 $n$ 个不同元素中取出 $m$ 个（$m \le n$），不考虑顺序，称为从 $n$ 个不同元素中取出 $m$ 个的一个组合。
### 2. 组合数公式
$C_n^m = \frac{A_n^m}{m!} = \frac{n!}{m!(n - m)!}$

**特殊情形**
- $C_n^0 = C_n^n = 1$
- $C_n^1 = n$
- $C_n^{n-1} = n$

### 3. 组合数的性质

| 性质 | 公式 |
|------|------|
| **对称性** | $C_n^m = C_n^{n-m}$ |
| **递推公式** | $C_n^m = C_{n-1}^{m-1} + C_{n-1}^m$ |
| **多个组合和** | $C_n^0 + C_n^1 + \dots + C_n^n = 2^n$ |
| **奇偶和差** | $C_n^0 - C_n^1 + C_n^2 - \dots = 0$ |
| **求和** | $C_m^m + C_{m+1}^m + \dots + C_n^m = C_{n+1}^{m+1}$ |

### 4. 排列与组合的区别
- 排列：与顺序有关（交换顺序为不同排法）
- 组合：与顺序无关（交换顺序为同一种选法）
### 5. 常见组合问题
- **平均分组**：均分后要除以组数的阶乘（消除顺序）
- **不平均分组**：直接按组分
- **相同元素分配**：隔板法
- **不同元素分配**：先选后排
- **至少问题**：正难则反（用总数减去不符合条件数）

## 四、二项式定理

### 1. 二项式展开公式
$(a + b)^n = C_n^0 a^n + C_n^1 a^{n-1} b + C_n^2 a^{n-2} b^2 + \dots + C_n^k a^{n-k} b^k + \dots + C_n^n b^n$
### 2. 通项公式
$T_{k+1} = C_n^k \cdot a^{n-k} \cdot b^k$（$k = 0, 1, 2, \dots, n$）
### 3. 二项式系数的性质

| 性质 | 内容 |
|------|------|
| **对称性** | $C_n^k = C_n^{n-k}$ |
| **最大值** | $n$ 为偶数：中间一项 $C_n^{n/2}$ 最大；$n$ 为奇数：中间两项 $C_n^{(n-1)/2} = C_n^{(n+1)/2}$ 最大 |
| **各系数和** | $C_n^0 + C_n^1 + \dots + C_n^n = 2^n$ |
| **奇偶项和** | $C_n^0 + C_n^2 + C_n^4 + \dots = 2^{n-1}$；$C_n^1 + C_n^3 + C_n^5 + \dots = 2^{n-1}$ |

### 4. 二项式系数与系数的区别
- 二项式系数：$C_n^k$（仅与 $n, k$ 有关）
- 项的系数：$C_n^k \cdot a^{n-k} \cdot b^k$（还取决于 $a, b$ 的系数）

### 5. 常用方法与题型
| 题型 | 方法 |
|------|------|
| 求特定项 | 利用通项公式，令 $x$ 的指数为特定值 |
| 求系数和 | 赋值法（令 $x = 1, -1, 0$ 等） |
| 求系数最大项 | 利用不等式组 $T_{k+1} \ge T_k$ 且 $T_{k+1} \ge T_{k+2}$ |
| 证明整除 | 将数拆成两数和，利用展开式 |

## 五、概率
### 1. 随机事件

**基本概念**
- **随机试验**：可以重复进行、结果不确定的试验
- **样本空间 $\Omega$**：随机试验所有可能结果构成的集合
- **随机事件**：样本空间的子集
- **基本事件**：样本空间的单点集
**事件的关系与运算**
| 关系/运算 | 含义 |
|-----------|------|
| 事件 $A$ 包含于 $B$ | $A$ 发生 $\Rightarrow B$ 发生 |
| $A = B$ | $A \subseteq B$ 且 $B \subseteq A$ |
| $A \cup B$（和事件） | $A$ 或 $B$ 至少一个发生 |
| $A \cap B$（积事件） | $A$ 和 $B$ 同时发生 |
| $A - B$（差事件） | $A$ 发生且 $B$ 不发生 |
| $\bar{A}$（对立事件） | $A$ 不发生，记作 $\bar{A}$ |
| $A \cap B = \varnothing$（互斥事件） | $A$ 和 $B$ 不能同时发生 |

### 2. 概率的定义与性质

**概率的公理化定义**：函数 $P$ 满足：
1. 非负性：$P(A) \ge 0$
2. 规范性：$P(\Omega) = 1$
3. 可列可加性：对互不相容的事件 $A_1, A_2, \dots$

**概率的性质**
- $P(\varnothing) = 0$
- $P(\bar{A}) = 1 - P(A)$
- 若 $A \subseteq B$，则 $P(A) \le P(B)$
- $P(A \cup B) = P(A) + P(B) - P(A \cap B)$

### 3. 古典概型

**满足条件**：
1. 样本空间有限
2. 每个基本事件等可能
**计算公式**：$P(A) = \frac{|A|}{|\Omega|} = \frac{A \text{ 包含的基本事件数}}{\text{总基本事件数}}$

### 4. 几何概型

**满足条件**：
1. 样本空间是无穷不可数集合
2. 每个样本点等可能

**计算公式**：$P(A) = \frac{A \text{ 的测度（长度/面积/体积）}}{\Omega \text{ 的测度}}$
### 5. 条件概率

**定义**：在事件 $B$ 已发生的条件下，事件 $A$ 发生的概率：
$P(A \mid B) = \frac{P(AB)}{P(B)} \ (P(B) > 0)$
**性质**
- $0 \le P(A \mid B) \le 1$
- $P(\Omega \mid B) = 1$
- $P(A_1 \cup A_2 \mid B) = P(A_1 \mid B) + P(A_2 \mid B) - P(A_1 A_2 \mid B)$

### 6. 事件的独立性
**定义**：$P(AB) = P(A) \cdot P(B) \Leftrightarrow A, B$ 相互独立

**性质**
- 若 $A, B$ 独立，则 $A$ 与 $\bar{B}$、$\bar{A}$ 与 $B$、$\bar{A}$ 与 $\bar{B}$ 也独立
- 多个事件独立：任意子集均满足独立性
**注意**：互斥 $\neq$ 独立
- 互斥：$A \cap B = \varnothing$，$P(AB) = 0$
- 独立：$P(AB) = P(A)P(B)$
- 若 $P(A) > 0, P(B) > 0$，则互斥一定不独立

### 7. 全概率公式
设 $B_1, B_2, \dots, B_n$ 构成样本空间的一个 **完备事件组**（两两互斥，且和为 $\Omega$），则对任意事件 $A$：
$P(A) = \sum_{i=1}^n P(B_i) \cdot P(A \mid B_i)$

### 8. 贝叶斯公式
在全概率公式条件下：
$P(B_k \mid A) = \frac{P(B_k) \cdot P(A \mid B_k)}{\sum_{i=1}^n P(B_i) \cdot P(A \mid B_i)}$

**意义**：后验概率 $=$（先验概率 $\times$ 似然）/ 全概率

## 六、随机变量及其分布
### 1. 随机变量的概念
定义在样本空间 $\Omega$ 上的实值函数 $X = X(\omega)$，称为随机变量。
**分类**
- **离散型**：取值有限或可数（如骰子的点数、命中次数）
- **连续型**：取值充满一个区间（如身高、体重、时间）

### 2. 离散型随机变量的分布列
设 $X$ 的取值为 $x_1, x_2, \dots, x_n$，对应的概率为 $p_1, p_2, \dots, p_n$。
**性质**
- $p_i \ge 0$
- $\sum p_i = 1$

**常用分布**
| 分布 | 描述 | 概率分布 | 期望 $E(X)$ | 方差 $D(X)$ |
|------|------|----------|-------------|-------------|
| **两点分布（0-1分布）** | 一次试验成功 | $P(X=1) = p, P(X=0) = 1-p$ | $p$ | $p(1-p)$ |
| **二项分布 $X \sim B(n, p)$** | $n$ 次独立重复试验 | $P(X=k) = C_n^k p^k (1-p)^{n-k}$ | $np$ | $np(1-p)$ |
| **超几何分布** | 不放回抽样 | $P(X=k) = \frac{C_M^k C_{N-M}^{n-k}}{C_N^n}$ | $\frac{nM}{N}$ | |
| **泊松分布 $X \sim P(\lambda)$** | 单位时间事件数 | $P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}$ | $\lambda$ | $\lambda$ |

### 3. 离散型随机变量的数字特征

**数学期望（均值）**：$E(X) = \sum x_i \cdot p_i$
**性质**
- $E(C) = C$
- $E(aX + b) = aE(X) + b$
- $E(X + Y) = E(X) + E(Y)$
- 若 $X, Y$ 独立，则 $E(XY) = E(X) \cdot E(Y)$

**方差**：$D(X) = E[(X - E(X))^2] = \sum [x_i - E(X)]^2 \cdot p_i = E(X^2) - [E(X)]^2$

**性质**
- $D(C) = 0$
- $D(aX + b) = a^2 D(X)$
- $D(X + Y) = D(X) + D(Y) + 2\text{Cov}(X, Y)$
- 若 $X, Y$ 独立，则 $D(X + Y) = D(X) + D(Y)$

**标准差**：$\sigma(X) = \sqrt{D(X)}$

### 4. 连续型随机变量
**概率密度函数 $f(x)$**
- $f(x) \ge 0$
- $\int_{-\infty}^{+\infty} f(x) \, dx = 1$
- $P(a \le X \le b) = \int_a^b f(x) \, dx$

### 5. 正态分布
**概率密度函数**：$f(x) = \frac{1}{\sigma \sqrt{2\pi}} \cdot e^{-\frac{(x-\mu)^2}{2\sigma^2}}$

记作：$X \sim N(\mu, \sigma^2)$

**性质**
- 关于 $x = \mu$ 对称
- $x = \mu$ 处取最大值
- $\sigma$ 越大曲线越扁平，$\sigma$ 越小曲线越陡峭
**标准正态分布**：$\mu = 0, \sigma = 1$，记作 $X \sim N(0, 1)$

**$3\sigma$ 原则**
| 区间 | 概率 |
|------|------|
| $(\mu - \sigma, \mu + \sigma)$ | $68.27\%$ |
| $(\mu - 2\sigma, \mu + 2\sigma)$ | $95.45\%$ |
| $(\mu - 3\sigma, \mu + 3\sigma)$ | $99.73\%$ |

### 6. 随机变量的常见题型
- 求分布列 $\to$ 先确定取值，再算概率，最后列表
- 求期望和方差 $\to$ 直接代入公式
- 二项分布与超几何分布的辨析
- 正态分布的概率计算（查表或利用对称性）

## 七、统计
### 1. 抽样方法

| 方法 | 适用条件 | 操作 |
|------|----------|------|
| **简单随机抽样** | 总体个体数较少 | 抽签法、随机数表法 |
| **系统抽样** | 总体个体数较多 | 等距抽样（编号$\to$分段$\to$取第一段$\to$等距抽取） |
| **分层抽样** | 总体差异明显分层 | 按比例在各层中抽取 |

### 2. 用样本估计总体

#### (1) 数字特征

| 特征 | 定义 | 特点 |
|------|------|------|
| **众数** | 出现次数最多的数据 | 可能不唯一 |
| **中位数** | 排序后中间的数 | 不受极值影响 |
| **平均数** | $\bar{x} = \frac{\sum x_i}{n}$ | 受极值影响大 |
| **方差** | $s^2 = \frac{\sum (x_i - \bar{x})^2}{n}$ | 衡量波动大小 |
| **标准差** | $s = \sqrt{s^2}$ | 单位与原始数据相同 |

#### (2) 统计图表

| 图表类型 | 用途 |
|----------|------|
| **频率分布直方图** | 显示数据的分布形态 |
| **茎叶图** | 保留原始数据，适合小样本 |
| **箱线图** | 显示数据的五数概括（最小, $Q_1$, 中位, $Q_3$, 最大） |
| **散点图** | 显示两个变量的关系 |
| **折线图** | 显示数据随时间的变化趋势 |
| **条形图/饼图** | 分类数据的展示 |

### 3. 相关关系

**相关系数 $r$**：
$r = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum (x_i - \bar{x})^2 \cdot \sum (y_i - \bar{y})^2}}$

- $|r| \le 1$
- $r > 0$：正相关；$r < 0$：负相关
- $|r|$ 越接近 $1$，线性相关越强
- $|r|$ 越接近 $0$，线性相关越弱
### 4. 线性回归
**回归直线方程**：$\hat{y} = \hat{b}x + \hat{a}$

**最小二乘法**：
$\hat{b} = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sum (x_i - \bar{x})^2} = \frac{\sum x_i y_i - n \bar{x} \bar{y}}{\sum x_i^2 - n \bar{x}^2}$
$\hat{a} = \bar{y} - \hat{b} \bar{x}$

回归直线过定点 $(\bar{x}, \bar{y})$

**非线性回归**：通过对数变换、指数变换等化为线性回归
### 5. 独立性检验
**$2 \times 2$ 列联表**
| | 事件 $B$ | $\bar{B}$ | 合计 |
|---------|------|-----------|------|
| 事件 $A$ | $a$ | $b$ | $a+b$ |
| $\bar{A}$ | $c$ | $d$ | $c+d$ |
| 合计 | $a+c$ | $b+d$ | $n$ |

**卡方统计量**：$\chi^2 = \frac{n(ad - bc)^2}{(a+b)(c+d)(a+c)(b+d)}$

**临界值**（$P(\chi^2 > k) = \alpha$）：
| $\alpha$ | $0.10$ | $0.05$ | $0.01$ | $0.001$ |
|----------|--------|--------|--------|---------|
| $k_0$ | $2.706$ | $3.841$ | $6.635$ | $10.828$ |

- $\chi^2 > k_0$：拒绝原假设，认为两变量有关
- $\chi^2 \le k_0$：没有充分证据拒绝原假设

### 6. 统计推断常见题型
- 频率分布直方图 $\to$ 求平均数、中位数、众数
- 茎叶图 $\to$ 比较两组数据的集中趋势和离散程度
- 线性回归分析 $\to$ 计算回归方程、预测
- 独立性检验 $\to$ 判断两分类变量是否有关
