# 高中数学 · 解析几何

> 涵盖：直线、圆、椭圆、双曲线、抛物线、圆锥曲线综合问题

---

## 一、直线
### 1. 倾斜角与斜率

**倾斜角** $\alpha$：直线向上的方向与 $x$ 轴正方向的夹角，范围 $[0^\circ, 180^\circ)$ 即 $[0, \pi)$

**斜率** $k$（倾斜角 $\neq 90^\circ$）：
- $k = \tan \alpha$
- $k = \frac{y_2 - y_1}{x_2 - x_1}$（两点式）

**$k$ 与 $\alpha$ 的关系**
- $\alpha = 0^\circ \Rightarrow k = 0$
- $0^\circ < \alpha < 90^\circ \Rightarrow k > 0$
- $\alpha = 90^\circ \Rightarrow k$ 不存在（直线垂直于 $x$ 轴）
- $90^\circ < \alpha < 180^\circ \Rightarrow k < 0$

### 2. 直线的五种方程
| 名称 | 方程形式 | 适用条件 |
|------|----------|----------|
| **点斜式** | $y - y_0 = k(x - x_0)$ | $k$ 存在 |
| **斜截式** | $y = kx + b$ | $k$ 存在 |
| **两点式** | $\frac{y - y_1}{y_2 - y_1} = \frac{x - x_1}{x_2 - x_1}$ | 不垂直坐标轴 |
| **截距式** | $\frac{x}{a} + \frac{y}{b} = 1$ | $a \neq 0, b \neq 0$ |
| **一般式** | $Ax + By + C = 0\ (A^2 + B^2 \neq 0)$ | 一切直线 |

**注意**
- 点斜式不能表示垂直于 $x$ 轴的直线
- 截距式不能表示过原点或平行坐标轴的直线
- 一般式可以表示任何直线

### 3. 直线的平行与垂直

设直线 $l_1$：$A_1x + B_1y + C_1 = 0$，$l_2$：$A_2x + B_2y + C_2 = 0$

| 关系 | 斜率存在 | 一般式 |
|------|----------|--------|
| **平行** | $k_1 = k_2$ 且 $b_1 \neq b_2$ | $A_1B_2 = A_2B_1$ 且 $A_1C_2 \neq A_2C_1$ |
| **垂直** | $k_1 \cdot k_2 = -1$ | $A_1A_2 + B_1B_2 = 0$ |
| **重合** | $k_1 = k_2$ 且 $b_1 = b_2$ | $\frac{A_1}{A_2} = \frac{B_1}{B_2} = \frac{C_1}{C_2}$ |

### 4. 距离公式

| 类型 | 公式 |
|------|------|
| **两点间距离** | $|AB| = \sqrt{(x_1 - x_2)^2 + (y_1 - y_2)^2}$ |
| **点到直线距离** | $d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$ |
| **平行线间距离** | $d = \frac{|C_1 - C_2|}{\sqrt{A^2 + B^2}}$ |

### 5. 直线的交点
两条直线联立方程组的解即为交点坐标：
- 有唯一解 $\to$ 相交
- 无解 $\to$ 平行
- 无穷多解 $\to$ 重合

### 6. 对称问题

| 对称类型 | 方法 |
|----------|------|
| 点关于点对称 | 中点坐标公式 |
| 点关于直线对称 | ① 中点在直线上；② 连线与直线垂直 |
| 直线关于点对称 | 转化为点关于点对称 |
| 直线关于直线对称 | ① 平行线：用距离公式；② 相交线：求交点 + 角平分线 |

## 二、圆

### 1. 圆的方程

| 形式 | 方程 | 圆心 | 半径 |
|------|------|------|------|
| **标准方程** | $(x - a)^2 + (y - b)^2 = r^2$ | $(a, b)$ | $r$ |
| **一般方程** | $x^2 + y^2 + Dx + Ey + F = 0$ | $(-\frac{D}{2}, -\frac{E}{2})$ | $\frac{1}{2}\sqrt{D^2 + E^2 - 4F}$ |

**一般方程的条件**：$D^2 + E^2 - 4F > 0$（等于 $0$ 为点圆，小于 $0$ 为虚圆）

### 2. 直线与圆的位置关系

| 关系 | 代数法（联立方程 $\Delta$） | 几何法（$d$ 为圆心到直线距离） |
|------|----------------------------|-------------------------------|
| **相交** | $\Delta > 0$ | $d < r$ |
| **相切** | $\Delta = 0$ | $d = r$ |
| **相离** | $\Delta < 0$ | $d > r$ |

### 3. 弦长公式
$|AB| = 2\sqrt{r^2 - d^2}$（$d$ 为圆心到直线的距离）

### 4. 圆的切线方程
过圆上一点 $(x_0, y_0)$ 的切线方程（圆 $(x-a)^2 + (y-b)^2 = r^2$）：
$(x_0 - a)(x - a) + (y_0 - b)(y - b) = r^2$

### 5. 圆与圆的位置关系

对于圆 $O_1(r_1)$ 和 $O_2(r_2)$，$d = |O_1O_2|$：
| 关系 | 条件 |
|------|------|
| 外离 | $d > r_1 + r_2$ |
| 外切 | $d = r_1 + r_2$ |
| 相交 | $|r_1 - r_2| < d < r_1 + r_2$ |
| 内切 | $d = |r_1 - r_2|$ |
| 内含 | $d < |r_1 - r_2|$ |

## 三、椭圆

### 1. 定义
平面内与两个定点 $F_1$、$F_2$ 的距离之和为常数 $2a$（$2a > |F_1F_2|$）的点的轨迹。

### 2. 标准方程与性质

| | 焦点在 $x$ 轴 | 焦点在 $y$ 轴 |
|------|---------------|---------------|
| **标准方程** | $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1\ (a > b > 0)$ | $\frac{y^2}{a^2} + \frac{x^2}{b^2} = 1\ (a > b > 0)$ |
| **焦点** | $F_1(-c, 0), F_2(c, 0)$ | $F_1(0, -c), F_2(0, c)$ |
| **顶点** | $(\pm a, 0), (0, \pm b)$ | $(0, \pm a), (\pm b, 0)$ |
| **范围** | $|x| \le a, |y| \le b$ | $|x| \le b, |y| \le a$ |
| **长轴** | $2a$（在 $x$ 轴上） | $2a$（在 $y$ 轴上） |
| **短轴** | $2b$ | $2b$ |

**基本关系**：$c^2 = a^2 - b^2$，$a > c > 0$

**离心率**：$e = \frac{c}{a} \in (0, 1)$，$e$ 越大越扁

### 3. 点与椭圆的位置关系
点 $P(x_0, y_0)$ 与椭圆 $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$：
- $\frac{x_0^2}{a^2} + \frac{y_0^2}{b^2} < 1$：点在椭圆内
- $\frac{x_0^2}{a^2} + \frac{y_0^2}{b^2} = 1$：点在椭圆上
- $\frac{x_0^2}{a^2} + \frac{y_0^2}{b^2} > 1$：点在椭圆外

## 四、双曲线

### 1. 定义
平面内与两个定点 $F_1$、$F_2$ 的距离之差的绝对值为常数 $2a$（$0 < 2a < |F_1F_2|$）的点的轨迹。

### 2. 标准方程与性质

| | 焦点在 $x$ 轴 | 焦点在 $y$ 轴 |
|------|---------------|---------------|
| **标准方程** | $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1\ (a > 0, b > 0)$ | $\frac{y^2}{a^2} - \frac{x^2}{b^2} = 1\ (a > 0, b > 0)$ |
| **焦点** | $F_1(-c, 0), F_2(c, 0)$ | $F_1(0, -c), F_2(0, c)$ |
| **顶点** | $(\pm a, 0)$ | $(0, \pm a)$ |
| **渐近线** | $y = \pm \frac{b}{a} x$ | $y = \pm \frac{a}{b} x$ |
| **范围** | $|x| \ge a, y \in \mathbb{R}$ | $|y| \ge a, x \in \mathbb{R}$ |
| **实轴** | $2a$（在 $x$ 轴上） | $2a$（在 $y$ 轴上） |
| **虚轴** | $2b$ | $2b$ |

**基本关系**：$c^2 = a^2 + b^2$，$c > a > 0$

**离心率**：$e = \frac{c}{a} > 1$

### 3. 等轴双曲线
$a = b$，方程 $x^2 - y^2 = \pm a^2$，渐近线 $y = \pm x$

## 五、抛物线

### 1. 定义
平面内与一个定点 $F$ 和一条定直线 $l$（$F$ 不在 $l$ 上）距离相等的点的轨迹。

### 2. 标准方程与性质

| 标准方程 | $y^2 = 2px$ | $y^2 = -2px$ | $x^2 = 2py$ | $x^2 = -2py$ |
|----------|-------------|--------------|-------------|--------------|
| **开口方向** | 向右 | 向左 | 向上 | 向下 |
| **焦点** | $(\frac{p}{2}, 0)$ | $(-\frac{p}{2}, 0)$ | $(0, \frac{p}{2})$ | $(0, -\frac{p}{2})$ |
| **准线** | $x = -\frac{p}{2}$ | $x = \frac{p}{2}$ | $y = -\frac{p}{2}$ | $y = \frac{p}{2}$ |
| **对称轴** | $x$ 轴 | $x$ 轴 | $y$ 轴 | $y$ 轴 |
| **顶点** | $(0, 0)$ | $(0, 0)$ | $(0, 0)$ | $(0, 0)$ |

**焦参数** $p > 0$：焦点到准线的距离

### 3. 抛物线的重要性质

过抛物线 $y^2 = 2px$ 焦点 $F$ 的弦 $AB$：
- $|AB| = x_A + x_B + p$
- 当 $AB \perp x$ 轴时（通径），$|AB|_{\min} = 2p$

## 六、圆锥曲线综合问题

### 1. 弦长公式
设直线 $y = kx + b$ 与圆锥曲线交于两点 $A(x_1, y_1)$、$B(x_2, y_2)$：
$|AB| = \sqrt{1 + k^2} \cdot |x_1 - x_2| = \sqrt{1 + k^2} \cdot \frac{\sqrt{\Delta}}{|a|}$
（$\Delta$ 为联立方程消元后二次方程的判别式，$a$ 为二次项系数）

### 2. 中点弦问题
设圆锥曲线弦 $AB$ 的中点为 $M(x_0, y_0)$：
- **椭圆**：$k_{AB} = -\frac{b^2 x_0}{a^2 y_0}$
- **双曲线**：$k_{AB} = \frac{b^2 x_0}{a^2 y_0}$
- **抛物线** $y^2 = 2px$：$k_{AB} = \frac{p}{y_0}$

### 3. 点差法
设 $A(x_1, y_1)$、$B(x_2, y_2)$ 在圆锥曲线上，代入方程后相减，利用中点坐标公式。

### 4. 直线与圆锥曲线的位置关系
联立直线与曲线方程，消元后：
- $\Delta > 0$：相交于两点
- $\Delta = 0$：相切
- $\Delta < 0$：相离

### 5. 定值与定点问题
**定值**：证明某个量与参数无关
**定点**：找出曲线恒过的定点坐标

**常用方法**
- 设参数（直线斜率 $k$ 等）
- 联立方程用韦达定理
- 化简整理得定值或定点

### 6. 最值与范围问题
**常用方法**
- 建立目标函数，转化为函数最值
- 利用基本不等式
- 利用圆锥曲线的几何性质
- 利用判别式法（$\Delta \ge 0$）
