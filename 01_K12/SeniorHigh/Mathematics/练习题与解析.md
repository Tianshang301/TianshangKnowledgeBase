# 高中数学练习题与解析

---

## 一、集合与常用逻辑

**1** 已知集合 $A = \{x \mid x^2 - 3x + 2 = 0\}$，$B = \{x \mid 0 < x < 3, x \in \mathbb{Z}\}$，求 $A \cap B$。
**2** 命题"$\forall x > 0$，$x^2 > 0$"的否定是______。

---

## 二、函数
**3** 已知函数 $f(x) = 2x^2 - 4x + 1$。（1）求函数的对称轴和顶点坐标。（2）求函数在区间 $[0, 3]$ 上的最大值和最小值。
**4** 已知函数 $f(x) = 
\begin{cases}
x + 1, & x \le 0 \\
x^2, & x > 0
\end{cases}$，求 $f(f(-1))$。

---

## 三、数列
**5** 已知等差数列 $\{a_n\}$ 中，$a_1 = 2$，$a_3 + a_5 = 18$，求数列的通项公式和前 $n$ 项和 $S_n$。
**6** 已知等比数列 $\{a_n\}$ 中，$a_1 = 3$，$a_4 = 24$，求公比 $q$ 和 $S_5$。

---

## 四、不等式

**7** 解不等式：$x^2 - 5x + 6 > 0$。
**8** 已知 $x > 0$，$y > 0$，且 $x + y = 4$，求 $xy$ 的最大值。

---

## 五、导数
**9** 求函数 $f(x) = x^3 - 3x^2 + 2$ 的单调区间和极值。
**10** 求曲线 $y = x^2 + 2x$ 在点 $(1, 3)$ 处的切线方程。

---

## 六、三角函数
**11** 已知 $\sin\theta = \frac{3}{5}$，且 $\theta$ 是第二象限角，求 $\cos\theta$ 和 $\tan\theta$。
**12** 已知函数 $f(x) = \sin(2x + \frac{\pi}{3})$。（1）求函数的最小正周期。（2）求函数在 $[0, \pi]$ 上的单调递增区间。

---

## 七、概率与统计

**13** 一个袋中有 $3$ 个红球和 $2$ 个白球，不放回地依次取出 $2$ 个球，求：（1）第一次取到红球的概率。（2）两次都取到红球的概率。（3）第二次取到红球的概率。
**14** 随机变量 $X \sim B(5, 0.4)$，求 $P(X = 2)$ 和 $E(X)$。

---

## 八、解析几何
**15** 已知圆 $C$：$x^2 + y^2 - 2x + 4y - 4 = 0$。（1）求圆心坐标和半径。（2）判断点 $P(3, 1)$ 与圆的位置关系。
**16** 已知椭圆 $\frac{x^2}{9} + \frac{y^2}{4} = 1$，求椭圆的焦点坐标和离心率。

---

## 九、立体几何
**17** 已知正三棱柱的底面边长为 $2$，侧棱长为 $3$。（1）求侧面积。（2）求体积。
**18** 在正方体 $ABCD\text{-}A_1B_1C_1D_1$ 中，求异面直线 $A_1B$ 和 $B_1C$ 所成角的大小。

---

## 十、复数
**19** 计算：$(2 + 3i)(1 - 2i)$。
**20** 若复数 $z$ 满足 $(1 + i)z = 2 - i$，求 $z$。

---

## 十一、向量

**21** 已知向量 $\vec{a} = (2, -1, 3)$，$\vec{b} = (-1, 2, 1)$，求：
（1）$|\vec{a}|$ 和 $|\vec{b}|$；
（2）$\vec{a} \cdot \vec{b}$ 以及 $\vec{a}$ 与 $\vec{b}$ 的夹角 $\theta$（用反三角函数表示）；
（3）$\vec{a} \times \vec{b}$。

**22** 在 $\triangle ABC$ 中，$\overrightarrow{AB} = \vec{a}$，$\overrightarrow{AC} = \vec{b}$，且 $|\vec{a}| = 3$，$|\vec{b}| = 4$，$\vec{a} \cdot \vec{b} = 6$，求 $\triangle ABC$ 的面积。

---

## 十二、导数应用提高

**23** 求函数 $f(x) = x^3 - 3x$ 在区间 $[-2, 2]$ 上的最大值和最小值，并指出函数图像的拐点。

**24** 某厂商生产一种产品，固定成本为 $100$ 万元，每生产 $x$ 万件产品的可变成本为 $C(x) = 0.01x^2 + 2x$ 万元，售价为 $p(x) = 10 - 0.01x$ 万元/万件。求：
（1）利润函数 $L(x)$；
（2）产量为多少时利润最大？最大利润是多少？

---

## 十三、圆锥曲线提高

**25** 已知双曲线 $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$ 的渐近线方程为 $y = \pm \frac{3}{4}x$，且过点 $(5, 3)$，求双曲线的标准方程和离心率。

**26** 已知抛物线 $y^2 = 2px$ 的焦点为 $F$，过点 $F$ 的直线与抛物线交于 $A$、$B$ 两点，且 $|AB| = 8$，$AB$ 的中点到 $y$ 轴的距离为 $2$，求抛物线的方程。

---

## 十四、数列提高

**27** 已知数列 $\{a_n\}$ 满足 $a_1 = 1$，$a_{n+1} = 2a_n + 1$。
（1）证明：$\{a_n + 1\}$ 是等比数列；
（2）求 $\{a_n\}$ 的通项公式和前 $n$ 项和 $S_n$。

**28** 已知数列 $\{a_n\}$ 的前 $n$ 项和为 $S_n$，且 $S_n = 2n^2 - n$。
（1）求 $\{a_n\}$ 的通项公式；
（2）若 $b_n = \frac{1}{a_n a_{n+1}}$，求 $\{b_n\}$ 的前 $n$ 项和 $T_n$。

---

## 十五、概率提高

**29** 甲、乙两人进行投篮比赛，甲的命中率为 $0.7$，乙的命中率为 $0.6$，两人各投 $3$ 次。
（1）求甲恰好命中 $2$ 次的概率；
（2）求两人命中次数相等的概率。

**30** 一个盒子里有 $5$ 个红球和 $3$ 个白球，每次从中随机取出 $1$ 个球，取出后不再放回。求：
（1）第 $2$ 次取到红球的概率；
（2）已知第 $1$ 次取到红球，第 $2$ 次取到白球的概率。

---

## 十六、不等式提高

**31** 已知 $a > 0$，$b > 0$，$a + b = 2$，求 $\frac{1}{a} + \frac{4}{b}$ 的最小值。

**32** 已知 $x > 0$，$y > 0$，且 $x + 2y = 1$，求 $\frac{1}{x} + \frac{1}{y}$ 的最小值。

---

## 答案解析

### 1
$A = \{x \mid (x-1)(x-2) = 0\} = \{1, 2\}$
$B = \{x \mid 0 < x < 3, x \in \mathbb{Z}\} = \{1, 2\}$
$A \cap B = \{1, 2\}$

### 2
$\exists x > 0$，$x^2 \le 0$

### 3
（1）$f(x) = 2(x^2 - 2x) + 1 = 2(x - 1)^2 - 1$
对称轴 $x = 1$，顶点 $(1, -1)$
（2）$f(0) = 1$，$f(1) = -1$，$f(3) = 2 \times 9 - 12 + 1 = 7$
最大值 $7$，最小值 $-1$

### 4
$f(-1) = -1 + 1 = 0$
$f(0) = 0 + 1 = 1$
$f(f(-1)) = f(0) = 1$

### 5
$a_3 + a_5 = (a_1 + 2d) + (a_1 + 4d) = 2a_1 + 6d = 4 + 6d = 18$，$d = \frac{7}{3}$
$a_n = 2 + (n-1) \times \frac{7}{3} = \frac{7n - 1}{3}$
$S_n = \frac{n(a_1 + a_n)}{2} = \frac{n(2 + \frac{7n-1}{3})}{2} = \frac{n(7n+5)}{6}$

### 6
$a_4 = a_1 \cdot q^3 = 3q^3 = 24$，$q^3 = 8$，$q = 2$
$S_5 = \frac{a_1(1 - q^5)}{1 - q} = \frac{3(1 - 32)}{1 - 2} = 3 \times 31 = 93$

### 7
$x^2 - 5x + 6 = (x - 2)(x - 3) > 0$
$x < 2$ 或 $x > 3$

### 8
$x + y = 4$，$xy \le \frac{(x+y)^2}{4} = \frac{16}{4} = 4$
当 $x = y = 2$ 时，$xy$ 取最大值 $4$

### 9
$f'(x) = 3x^2 - 6x = 3x(x - 2)$
令 $f'(x) = 0$，得 $x = 0$ 或 $x = 2$
$(-\infty, 0)$：$f'(x) > 0$，增；$(0, 2)$：$f'(x) < 0$，减；$(2, +\infty)$：$f'(x) > 0$，增
极大值 $f(0) = 2$，极小值 $f(2) = 8 - 12 + 2 = -2$

### 10
$y' = 2x + 2$，$y'|_{x=1} = 4$
切线方程：$y - 3 = 4(x - 1)$，即 $y = 4x - 1$

### 11
$\cos^2\theta = 1 - \sin^2\theta = 1 - \frac{9}{25} = \frac{16}{25}$
$\theta$ 在第二象限，$\cos\theta = -\frac{4}{5}$
$\tan\theta = \frac{\sin\theta}{\cos\theta} = \frac{3/5}{-4/5} = -\frac{3}{4}$

### 12
（1）$T = \frac{2\pi}{2} = \pi$
（2）由 $2k\pi - \frac{\pi}{2} \le 2x + \frac{\pi}{3} \le 2k\pi + \frac{\pi}{2}$
$2k\pi - \frac{5\pi}{6} \le 2x \le 2k\pi + \frac{\pi}{6}$
$k\pi - \frac{5\pi}{12} \le x \le k\pi + \frac{\pi}{12}$
在 $[0, \pi]$ 上，增区间为 $[0, \frac{\pi}{12}]$ 和 $[\frac{7\pi}{12}, \pi]$

### 13
（1）$P_1 = \frac{3}{5}$
（2）$P_2 = \frac{3}{5} \times \frac{2}{4} = \frac{3}{10}$
（3）$P_3 = \frac{3}{5} \times \frac{2}{4} + \frac{2}{5} \times \frac{3}{4} = \frac{3}{10} + \frac{3}{10} = \frac{3}{5}$

### 14
$P(X=2) = C(5,2) \times 0.4^2 \times 0.6^3 = 10 \times 0.16 \times 0.216 = 0.3456$
$E(X) = np = 5 \times 0.4 = 2$

### 15
（1）$x^2 + y^2 - 2x + 4y - 4 = 0$
$(x^2 - 2x + 1) + (y^2 + 4y + 4) = 9$
$(x - 1)^2 + (y + 2)^2 = 9$
圆心 $(1, -2)$，半径 $r = 3$
（2）$|CP| = \sqrt{(3-1)^2 + (1+2)^2} = \sqrt{4 + 9} = \sqrt{13} < 3$
所以点 $P$ 在圆内

### 16
$a^2 = 9$，$b^2 = 4$，$c^2 = a^2 - b^2 = 5$
$c = \sqrt{5}$，焦点 $(\pm\sqrt{5}, 0)$
$e = \frac{c}{a} = \frac{\sqrt{5}}{3}$

### 17
（1）侧面是矩形，底边 $2$，高 $3$
侧面积 $= 3 \times 2 \times 3 = 18$
（2）底面积 $= \frac{\sqrt{3}}{4} \times 2^2 = \sqrt{3}$
体积 $= \sqrt{3} \times 3 = 3\sqrt{3}$

### 18
异面直线 $A_1B$ 与 $B_1C$ 所成角为 $60^\circ$
可用向量法：设正方体棱长为 $1$，$\overrightarrow{A_1B} = (0,1,-1)$，$\overrightarrow{B_1C} = (-1,0,-1)$
$\cos\theta = \frac{|\overrightarrow{A_1B} \cdot \overrightarrow{B_1C}|}{|\overrightarrow{A_1B}| \cdot |\overrightarrow{B_1C}|} = \frac{|0+0+1|}{\sqrt{2} \times \sqrt{2}} = \frac{1}{2}$，$\theta = 60^\circ$

### 19
$(2 + 3i)(1 - 2i) = 2 - 4i + 3i - 6i^2 = 2 - i + 6 = 8 - i$

### 20
$z = \frac{2 - i}{1 + i} = \frac{(2 - i)(1 - i)}{(1 + i)(1 - i)} = \frac{2 - 2i - i + i^2}{1 - i^2} = \frac{2 - 3i - 1}{2} = \frac{1 - 3i}{2}$

### 21
（1）$|\vec{a}| = \sqrt{2^2 + (-1)^2 + 3^2} = \sqrt{14}$，$|\vec{b}| = \sqrt{(-1)^2 + 2^2 + 1^2} = \sqrt{6}$
（2）$\vec{a} \cdot \vec{b} = 2 \times (-1) + (-1) \times 2 + 3 \times 1 = -1$
$\cos\theta = \frac{-1}{\sqrt{14} \times \sqrt{6}} = -\frac{1}{2\sqrt{21}}$，$\theta = \arccos\left(-\frac{1}{2\sqrt{21}}\right)$
（3）$\vec{a} \times \vec{b} = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ 2 & -1 & 3 \\ -1 & 2 & 1 \end{vmatrix} = -7\vec{i} - 5\vec{j} + 3\vec{k}$

### 22
$S_{\triangle} = \frac{1}{2}|\vec{a}||\vec{b}|\sin\theta$
$\sin\theta = \sqrt{1 - \left(\frac{6}{12}\right)^2} = \frac{\sqrt{3}}{2}$
$S_{\triangle} = \frac{1}{2} \times 3 \times 4 \times \frac{\sqrt{3}}{2} = 3\sqrt{3}$

### 23
$f'(x) = 3x^2 - 3 = 3(x^2 - 1)$，令 $f'(x) = 0$，得 $x = \pm 1$
$f(-2) = -2$，$f(-1) = 2$，$f(1) = -2$，$f(2) = 2$
最大值 $2$，最小值 $-2$
$f''(x) = 6x$，令 $f''(x) = 0$，$x = 0$，拐点 $(0, 0)$

### 24
（1）$L(x) = x(10 - 0.01x) - (100 + 0.01x^2 + 2x) = -0.02x^2 + 8x - 100$
（2）$L'(x) = -0.04x + 8 = 0$，$x = 200$
$L(200) = -0.02 \times 40000 + 1600 - 100 = 700$
产量 $200$ 万件时利润最大，为 $700$ 万元

### 25
$\frac{b}{a} = \frac{3}{4}$，设 $a = 4k$，$b = 3k$
代入点 $(5,3)$：$\frac{25}{16k^2} - \frac{9}{9k^2} = 1$，$\frac{9}{16k^2} = 1$，$k = \frac{3}{4}$
$a = 3$，$b = \frac{9}{4}$，双曲线方程 $\frac{x^2}{9} - \frac{16y^2}{81} = 1$
$c = \sqrt{9 + \frac{81}{16}} = \frac{15}{4}$，$e = \frac{5}{4}$

### 26
$F(\frac{p}{2}, 0)$，设直线 $x = my + \frac{p}{2}$，代入 $y^2 = 2px$
$y^2 - 2pmy - p^2 = 0$，$y_1 + y_2 = 2pm$，$y_1y_2 = -p^2$
中点横坐标 $\frac{x_1 + x_2}{2} = \frac{2pm^2 + p}{2} = 2$ ①
$|AB| = 2p(1 + m^2) = 8$ ②
由①②得 $p = 4$，抛物线 $y^2 = 8x$

### 27
（1）$a_{n+1} + 1 = 2(a_n + 1)$，$a_1 + 1 = 2$，$\{a_n + 1\}$ 是等比数列
（2）$a_n + 1 = 2^n$，$a_n = 2^n - 1$
$S_n = \frac{2(2^n - 1)}{2 - 1} - n = 2^{n+1} - n - 2$

### 28
（1）$a_n = S_n - S_{n-1} = (2n^2 - n) - [2(n-1)^2 - (n-1)] = 4n - 3$
（2）$b_n = \frac{1}{(4n-3)(4n+1)} = \frac{1}{4}\left(\frac{1}{4n-3} - \frac{1}{4n+1}\right)$
$T_n = \frac{1}{4}\left(1 - \frac{1}{4n+1}\right) = \frac{n}{4n+1}$

### 29
（1）$P = C_3^2 \times 0.7^2 \times 0.3 = 0.441$
（2）$P = C_3^0 \times 0.3^3 \times C_3^0 \times 0.4^3 + C_3^1 \times 0.7 \times 0.3^2 \times C_3^1 \times 0.6 \times 0.4^2 + C_3^2 \times 0.7^2 \times 0.3 \times C_3^2 \times 0.6^2 \times 0.4 + C_3^3 \times 0.7^3 \times C_3^3 \times 0.6^3 = 0.32076$

### 30
（1）$P = \frac{5}{8} \times \frac{4}{7} + \frac{3}{8} \times \frac{5}{7} = \frac{5}{8}$
（2）$P = \frac{3}{7}$

### 31
$\frac{1}{a} + \frac{4}{b} = \frac{1}{2}(a+b)\left(\frac{1}{a} + \frac{4}{b}\right) = \frac{1}{2}\left(5 + \frac{4a}{b} + \frac{b}{a}\right) \ge \frac{1}{2}(5 + 4) = \frac{9}{2}$
当 $a = \frac{2}{3}$，$b = \frac{4}{3}$ 时取最小值 $\frac{9}{2}$

### 32
$\frac{1}{x} + \frac{1}{y} = (x+2y)\left(\frac{1}{x} + \frac{1}{y}\right) = 3 + \frac{x}{y} + \frac{2y}{x} \ge 3 + 2\sqrt{2}$
当 $x = \sqrt{2}y$ 时取最小值 $3 + 2\sqrt{2}$
