# 高中数学 · 数列

> 涵盖：数列的概念、等差数列、等比数列、通项公式求法、数列求和、数列不等式、递推数列、数列应用题

---

## 一、数列的概念

### 1. 定义

按一定次序排列的一列数称为数列。数列中的每个数称为数列的项。
数列的一般形式：$a_1, a_2, a_3, \dots, a_n, \dots$ 简记为 $\{a_n\}$

### 2. 数列的分类
| 分类标准 | 类型 | 说明 |
|--------|------|------|
| **项数** | 有穷数列 | 项数有限 |
| | 无穷数列 | 项数无限 |
| **变化趋势** | 递增数列 | $a_{n+1} > a_n$ 恒成立 |
| | 递减数列 | $a_{n+1} < a_n$ 恒成立 |
| | 常数列 | $a_{n+1} = a_n$ 恒成立 |
| | 摆动数列 | 有时大有时小 |
| | 周期数列 | 项呈现周期性重复 |

### 3. 通项公式

数列 $\{a_n\}$ 的第 $n$ 项 $a_n$ 与 $n$ 之间的函数关系：$a_n = f(n)$

**注意**：不是所有数列都有通项公式，有些数列通项公式不唯一。
### 4. 递推公式

已知数列的首项（或前几项），以及从某项起每一项与它前一项（或前几项）的关系式。
**常见递推类型**
- $a_{n+1} = a_n + d$（等差递推）
- $a_{n+1} = a_n \cdot q$（等比递推）
- $a_{n+1} = a_n + f(n)$
- $a_{n+1} = a_n \times f(n)$
- $a_{n+1} = pa_n + q$（线性递推）
### 5. 数列的前 $n$ 项和

$S_n = a_1 + a_2 + a_3 + \dots + a_n$

**基本关系**
- $a_1 = S_1$
- $a_n = S_n - S_{n-1} \ (n \ge 2)$
- $S_n - S_{n-1} = a_n$

## 二、等差数列
### 1. 定义

从第 $2$ 项起，每一项与它的前一项的差等于同一个常数 $d$：$a_{n+1} - a_n = d$（常数）

**等差中项**：$a, A, b$ 成等差数列 $\iff 2A = a + b$

### 2. 通项公式

$a_n = a_1 + (n-1)d$

**变形**
- $a_n = a_m + (n-m)d$（推广公式）
- $a_n = dn + (a_1 - d)$（可看作关于 $n$ 的一次函数）

### 3. 前 $n$ 项和公式

$S_n = \frac{n(a_1 + a_n)}{2} = na_1 + \frac{n(n-1)d}{2}$

**变形**：$S_n = \frac{d}{2}n^2 + \left(a_1 - \frac{d}{2}\right)n$（可看作关于 $n$ 的二次函数，无常数项）
### 4. 等差数列的性质

| 性质 | 内容 |
|------|------|
| **$m+n = p+q$** | $a_m + a_n = a_p + a_q$（下标和相等则项和相等） |
| **等距性** | $a_k, a_{k+m}, a_{k+2m}, \dots$ 仍是等差数列，公差为 $md$ |
| **片段和** | $S_n, S_{2n} - S_n, S_{3n} - S_{2n}, \dots$ 成等差数列，公差为 $n^2 d$ |
| **奇偶性** | 项数 $2n$：$S_{\text{偶}} - S_{\text{奇}} = nd$；项数 $2n-1$：$S_{\text{奇}} - S_{\text{偶}} = a_n$ |
| **若 $\{a_n\}$ 等差** | ① $\{ca_n\}$ 等差；② $\{a_n + b_n\}$ 等差；③ $\{a_n + c\}$ 等差 |
| **$S_n$ 最值** | $a_1 > 0, d < 0$ 时 $S_n$ 有最大值；$a_1 < 0, d > 0$ 时 $S_n$ 有最小值 |

### 5. 等差数列的判定方法
| 方法 | 判定条件 |
|------|----------|
| **定义法** | $a_{n+1} - a_n = \text{常数}$ |
| **等差中项法** | $2a_{n+1} = a_n + a_{n+2}$ |
| **通项公式法** | $a_n = pn + q$（一次函数形式） |
| **前 $n$ 项和法** | $S_n = An^2 + Bn$（二次函数无常数项） |

## 三、等比数列
### 1. 定义

从第 $2$ 项起，每一项与它的前一项的比等于同一个常数 $q$：$\frac{a_{n+1}}{a_n} = q$（常数，$q \neq 0$）
**等比中项**：$a, G, b$ 成等比数列 $\iff G^2 = ab$

### 2. 通项公式

$a_n = a_1 \cdot q^{n-1}$

**变形**：$a_n = a_m \cdot q^{n-m}$（推广公式）

### 3. 前 $n$ 项和公式

$$
S_n = \begin{cases}
na_1, & q = 1 \\
\frac{a_1(1 - q^n)}{1 - q} = \frac{a_1 - a_n q}{1 - q}, & q \neq 1
\end{cases}
$$

### 4. 等比数列的性质

| 性质 | 内容 |
|------|------|
| **$m+n = p+q$** | $a_m \cdot a_n = a_p \cdot a_q$（下标和相等则项积相等） |
| **等距性** | $a_k, a_{k+m}, a_{k+2m}, \dots$ 仍是等比数列，公比为 $q^m$ |
| **片段和** | $S_n, S_{2n} - S_n, S_{3n} - S_{2n}, \dots$ 成等比数列（$q \neq -1$） |
| **对称性** | 若三个数成等比，可设为 $\frac{a}{q}, a, aq$ |
| **若 $\{a_n\}$ 等比** | ① $\{ca_n\}$ 等比；② $\{a_n^2\}$ 等比；③ $\{1/a_n\}$ 等比；④ 各项同号 |

### 5. 等比数列的判定方法
| 方法 | 判定条件 |
|------|----------|
| **定义法** | $\frac{a_{n+1}}{a_n} = \text{常数}$ |
| **等比中项法** | $a_{n+1}^2 = a_n \cdot a_{n+2} \ (a_n \neq 0)$ |
| **通项公式法** | $a_n = c \cdot q^n$（指数函数形式） |
| **前 $n$ 项和法** | $S_n = A(q^n - 1) \ (q \neq 1)$ |

---

## 四、等差与等比数列的对比
| 对比项 | 等差数列 | 等比数列 |
|--------|----------|----------|
| **定义** | $a_{n+1} - a_n = d$ | $\frac{a_{n+1}}{a_n} = q$ |
| **通项** | $a_n = a_1 + (n-1)d$ | $a_n = a_1 q^{n-1}$ |
| **中项** | $2A = a + b$ | $G^2 = ab$ |
| **前 $n$ 项和** | $S_n = \frac{n(a_1+a_n)}{2}$ | $S_n = \frac{a_1(1-q^n)}{1-q} \ (q \neq 1)$ |
| **递增条件** | $d > 0$ | $a_1 > 0, q > 1$ 或 $a_1 < 0, 0 < q < 1$ |
| **性质** | 和相等 | 积相等 |

## 五、通项公式的求法
### 1. 公式法
直接利用等差或等比数列的通项公式。
### 2. 累加法
适用于 $a_{n+1} - a_n = f(n)$ 型：

$$
a_n = a_1 + \sum_{i=1}^{n-1} f(i)
$$

**常见 $f(n)$**
- $f(n) = \text{常数}$ → 等差数列
- $f(n) = kn + b$ → $a_n$ 为二次型
- $f(n) = q^n$ → 用等比求和公式
### 3. 累乘法
适用于 $\frac{a_{n+1}}{a_n} = f(n)$ 型：

$$
a_n = a_1 \cdot \prod_{i=1}^{n-1} f(i)
$$

**常见 $f(n)$**
- $f(n) = \text{常数}$ → 等比数列
- $f(n) = \frac{n}{n+1}$ → 约分得 $a_n = \frac{a_1}{n}$

### 4. 构造法（待定系数法）
**类型 1**：$a_{n+1} = pa_n + q$（$p \neq 1, q \neq 0$）
- 构造：$a_{n+1} + \lambda = p(a_n + \lambda)$，其中 $\lambda = \frac{q}{p-1}$
- $\{a_n + \lambda\}$ 为等比数列
**类型 2**：$a_{n+1} = pa_n + qn + r$

- 构造：$a_{n+1} + A(n+1) + B = p(a_n + An + B)$
- 解出 $A, B$

**类型 3**：$a_{n+1} = pa_n + q \cdot r^n$

- 两边同除以 $r^{n+1}$ 或构造新数列

**类型 4**：$a_{n+1} = p \cdot a_n^m$

- 取对数：$\ln a_{n+1} = m \ln a_n + \ln p$
- 转化为线性递推

**类型 5**：$a_{n+1} = \frac{aa_n + b}{ca_n + d}$（分式递推）
- 取倒数转化
- 特征根法（不动点法）

### 5. 利用 $S_n$ 求 $a_n$

$$
a_n = \begin{cases}
S_1, & n = 1 \\
S_n - S_{n-1}, & n \ge 2
\end{cases}
$$

**注意**：$n = 1$ 时需单独验证，有时 $a_1$ 不满足 $n \ge 2$ 的通项公式，需要分段表示。
### 6. 取倒数法
$a_{n+1} = \frac{a_n}{ka_n + b}$ → 取倒数得 $\frac{1}{a_{n+1}} = b \cdot \frac{1}{a_n} + k$

### 7. 取对数法

$a_{n+1} = c \cdot a_n^d$ 型（$a_n > 0$）→ 取自然对数
### 8. 特征根法（二阶线性递推）
$a_{n+2} = p a_{n+1} + q a_n$

**步骤**
1. 写出特征方程：$x^2 = p x + q$
2. 解特征根：
   - 两根不等 $x_1 \neq x_2$：$a_n = A \cdot x_1^{n-1} + B \cdot x_2^{n-1}$
   - 两根相等 $x_1 = x_2$：$a_n = (A + Bn) \cdot x_1^{n-1}$
3. 代入 $a_1, a_2$ 确定 $A, B$

## 六、数列求和方法
### 1. 公式法
| 数列 | 前 $n$ 项和公式 |
|------|-----------------|
| 等差数列 | $S_n = \frac{n(a_1 + a_n)}{2} = na_1 + \frac{n(n-1)d}{2}$ |
| 等比数列 | $S_n = \frac{a_1(1 - q^n)}{1 - q} \ (q \neq 1)$ |
| 自然数 | $1 + 2 + \dots + n = \frac{n(n+1)}{2}$ |
| 平方和 | $1^2 + 2^2 + \dots + n^2 = \frac{n(n+1)(2n+1)}{6}$ |
| 立方和 | $1^3 + 2^3 + \dots + n^3 = \left[\frac{n(n+1)}{2}\right]^2$ |

### 2. 分组求和法
将数列分成两组或多组，每组分别求和。
**适用**：通项由多个不同类项加减而成。
**示例**：$a_n = 2^n + n$ → 分组为等比数列 $\{2^n\}$ 和等差数列 $\{n\}$

### 3. 裂项相消法
**原理**：将一项拆分成两项之差，使求和时中间项相互抵消。
**常用裂项公式**
| 公式 | 说明 |
|------|------|
| $\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$ | 分母为相邻整数积 |
| $\frac{1}{n(n+k)} = \frac{1}{k}\left(\frac{1}{n} - \frac{1}{n+k}\right)$ | 分母为相差 $k$ 的整数积 |
| $\frac{1}{(2n-1)(2n+1)} = \frac{1}{2}\left(\frac{1}{2n-1} - \frac{1}{2n+1}\right)$ | 分母为相邻奇数积 |
| $\frac{1}{n(n+1)(n+2)} = \frac{1}{2}\left[\frac{1}{n(n+1)} - \frac{1}{(n+1)(n+2)}\right]$ | 三项裂项 |
| $\frac{1}{\sqrt{n} + \sqrt{n+1}} = \sqrt{n+1} - \sqrt{n}$ | 分母含根号 |
| $\frac{1}{a\sqrt{b} + b\sqrt{a}} = \frac{1}{ab}(\sqrt{a} - \sqrt{b})$ | 根号分式 |
| $n \cdot n! = (n+1)! - n!$ | 阶乘裂项 |

### 4. 错位相减法
**适用**：等差数列 $\times$ 等比数列（通项为 $a_n \cdot b_n$，其中 $\{a_n\}$ 等差，$\{b_n\}$ 等比）
**步骤**
1. 写出 $S_n$
2. 两边同乘以等比数列的公比 $q$
3. 两式错位相减
4. 化简整理

**模板**：

$$
\begin{aligned}
S_n &= a_1b_1 + a_2b_2 + \dots + a_nb_n \\
qS_n &= a_1b_2 + a_2b_3 + \dots + a_{n-1}b_n + a_nb_{n+1} \\
(1-q)S_n &= a_1b_1 + (a_2-a_1)b_2 + \dots + (a_n-a_{n-1})b_n - a_nb_{n+1}
\end{aligned}
$$

### 5. 倒序相加法
**适用**：与首末两端等距的两项和相等（如等差数列求和）
**原理**：

$$
\begin{aligned}
S_n &= a_1 + a_2 + \dots + a_n \\
S_n &= a_n + a_{n-1} + \dots + a_1 \\
2S_n &= (a_1 + a_n) + (a_2 + a_{n-1}) + \dots + (a_n + a_1)
\end{aligned}
$$

**推广**：$f(x) + f(1/x) = \text{常数}$ 型
### 6. 并项求和法
将数列中相邻两项合并，使合并后成为等差或等比数列。
**适用**：正负交替的数列

## 七、数列不等式

### 1. 数列单调性的证明

- 作差法：$a_{n+1} - a_n$ 与 $0$ 比较
- 作商法：$\frac{a_{n+1}}{a_n}$ 与 $1$ 比较（各项正数）
- 函数法：构造函数 $f(x)$，利用导数
### 2. 数列有界性的证明

- 放缩法
- 数学归纳法
- 利用已知不等式
### 3. 数列不等式的常用证明方法

- **放缩法**：将通项放大或缩小，转化为可求和的形式
- **数学归纳法**：$n = k$ 到 $n = k+1$ 的递推
- **构造数列法**：构造新数列 $\{b_n\}$，证明 $b_n \ge 0$
- **裂项放缩**：$a_n < \frac{1}{n-1} - \frac{1}{n}$ → 求和后大部分项抵消
**常见放缩不等式**
- $\frac{1}{n^2} < \frac{1}{n-1} - \frac{1}{n} \ (n \ge 2)$
- $\frac{1}{n+1} < \ln\left(1 + \frac{1}{n}\right) < \frac{1}{n}$
- $2(\sqrt{n+1} - \sqrt{n}) < \frac{1}{\sqrt{n}} < 2(\sqrt{n} - \sqrt{n-1})$

## 八、特殊数列与递推

### 1. 周期数列

存在正整数 $T$，使 $a_{n+T} = a_n$ 对所有 $n$ 成立。
### 2. 摆动数列

正负交替或大小交替的数列。
### 3. 差分数列

记 $\Delta a_n = a_{n+1} - a_n$，研究差分数列的特征。
### 4. 求数列最大项

- $a_n \ge a_{n+1}$ 且 $a_n \ge a_{n-1}$ $\Rightarrow$ $a_n$ 是最大项
- 构造函数求极值
- 考虑数列的单调性转化

## 九、数列应用题

### 1. 单利与复利模型
- 单利：等差数列模型
- 复利：等比数列模型（$(1+r)^n$）
### 2. 分期付款模型

- 等额本息还款
- 等额本金还款

### 3. 增长率模型
- 平均增长率：$a_n = a_0(1 + r)^n$
- 递减模型：$a_n = a_0(1 - r)^n$

### 4. 堆叠问题

- 三角形数：$T_n = \frac{n(n+1)}{2}$
- 正方形数：$S_n = n^2$
- 五边形数：$P_n = \frac{n(3n-1)}{2}$

### 5. 递推应用模型

- 斐波那契数列（兔子繁殖）：$F_1 = 1, F_2 = 1, F_{n+2} = F_{n+1} + F_n$
- 上楼梯问题
- 细胞分裂

| 考点 | 题型 | 关键方法 |
|------|------|----------|

| 数列不等式的证明 | 放缩法/数学归纳法 | 灵活放缩 |
| 数列与函数综合 | 数列作为函数 | 导数、单调性 |
| 数列实际应用 | 增长率/分期付款 | 建立等差或等比模型 |
