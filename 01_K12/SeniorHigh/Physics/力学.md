## 一、运动的描述

### 1. 基本概念

| 概念 | 定义 | 说明 |
|------|------|------|
| **质点** | 有质量无大小的点 | 理想模型，物体大小远小于运动距离时可视为质点 |
| **参考系** | 描述运动的标准 | 选不同参考系运动描述不同 |
| **位移** | 初位置→末位置的有向线段 | 矢量，与路径无关 |
| **路程** | 运动轨迹的长度 | 标量，≥ 位移大小 |
| **速度** | 位移与时间的比值 | 矢量，v = Δx/Δt |
| **速率** | 路程与时间的比值 | 标量 |
| **加速度** | 速度变化与所用时间的比值 | 矢量，a = Δv/Δt |

### 2. 匀变速直线运动的公式

**三大基本公式**：
- v = v₀ + at
- x = v₀t + ½at²
- v² - v₀² = 2ax

**推论**：
- 平均速度：v̄ = (v₀ + v)/2 = v_{t/2}
- 中间位置速度：v_{x/2} = √[(v₀² + v²)/2]
- 逐差公式：Δx = aT²（纸带处理核心）
- 初速为零的匀加速直线运动：
  - 1T 末、2T 末、3T 末...速度比：v₁:v₂:v₃ = 1:2:3:...
  - 1T 内、2T 内、3T 内...位移比：x₁:x₂:x₃ = 1²:2²:3²:...
  - 第1T 内、第 2T 内、第 3T 内...位移比：x₁:x₂:x₃ = 1:3:5:...

### 3. 自由落体运动
- v₀ = 0，a = g（g ≈ 9.8 m/s²）
- v = gt
- h = ½gt²
- v² = 2gh

### 4. 竖直上抛运动
- 上升时间：t上 = v₀/g
- 上升最大高度：H = v₀²/(2g)
- 总时间：t = 2v₀/g
- 对称性：上升与下降过程对称（时间、速度大小）
### 5. s-t 图与 v-t 图
| 图表 | 斜率含义 | 面积含义 | 截距 |
|------|---------|---------|------|
| **s-t 图** | 速度 |  | 初始位置 |
| **v-t 图** | 加速度 | 位移 | 初速度 |

## 二、相互作用
### 1. 常见的力

**重力**：G = mg（方向竖直向下，作用点在重心）
**弹力**：
- 产生条件：接触 + 形变
- 弹簧：F = kx（胡克定律，k 为劲度系数）
- 方向：与形变方向相反（压力/支持力垂直于接触面）

**摩擦力**：
- **静摩擦力**：0 ≤ f ≤ fmax，方向与相对运动趋势方向相反
- **滑动摩擦力**：f = μN，方向与相对运动方向相反
- **注意**：摩擦力可以是动力也可以是阻力
### 2. 力的合成与分解
**平行四边形定则**（矢量运算通用）：
- 合力大小：F = √(F₁² + F₂² + 2F₁F₂cosθ)
- 合力范围：|F₁ - F₂| ≤ F ≤ F₁ + F₂
**正交分解法**：
1. 建立直角坐标系（让尽量多的力在轴上）
2. 将不在轴上的力分解到坐标轴上
3. 列方程：x 方向合力 = 0；y 方向合力 = 0

**力的分解**：
- 已知合力和两个分力的方向 → 唯一解
- 已知合力和一个分力的大小和方向 → 唯一解
- 已知合力和两个分力的大小 → 两解/一解/无解

### 3. 受力分析步骤
1. 确定研究对象
2. 先场力（重力/电场力/磁场力）
3. 再接触力（弹力/摩擦力）
4. 结合运动状态列方程

## 三、牛顿运动定律
### 1. 牛顿第一定律（惯性定律）
- 内容：一切物体在不受外力作用时，总保持静止或匀速直线运动状态
- 惯性：物体保持原状态的性质，只与质量有关
- 力不是维持运动的原因，而是改变运动状态的原因

### 2. 牛顿第二定律
F合 = ma（F 与 a 同向，瞬时对应关系）

**应用步骤**：
1. 确定研究对象
2. 受力分析
3. 列方程 F合 = ma
4. 解方程
**两类基本问题**：
- 已知受力 → 求加速度 → 求运动
- 已知运动 → 求加速度 → 求受力
### 3. 牛顿第三定律
F = -F'（作用力与反作用力大小相等、方向相反、同一直线、分别作用于不同物体）
**与平衡力的区别**：
| 比较 | 作用力与反作用力 | 平衡力 |
|------|-----------------|--------|
| 作用对象 | 两个不同物体 | 同一物体 |
| 性质 | 一定相同 | 不一定相同 |
| 依存关系 | 同时产生、消失 | 无关 |

### 4. 超重与失重
| 状态 | 加速度方向 | 视重与实际重力关系 |
|------|-----------|-----------------|
| **超重** | a 向上 | N > mg |
| **失重** | a 向下 | N < mg |
| **完全失重** | a = g 向下 | N = 0 |

### 5. 连接体问题
**整体法**：当各部分加速度相同时，可视为整体
**隔离法**：求内力时必须隔离分析

## 四、曲线运动
### 1. 运动的合成与分解
- **合运动** = 分运动的矢量合
- 等时性：合运动与分运动同时完成
- 独立性：各分运动互不影响

**小船过河**：
- 最短时间：船头垂直河岸，tmin = d/v船
- 最短位移：船头偏向上游，v船 > v水时可垂直到达对岸
### 2. 平抛运动
- 水平：匀速直线运动，x = v₀t，vx = v₀
- 竖直：自由落体运动，y = ½gt²，vy = gt
- 合速度：v = √(v₀² + g²t²)，tanθ = gt/v₀
- 合位移：s = √(x² + y²)
- 轨迹方程：y = (g/2v₀²)x²（抛物线）
- **推论**：速度偏转角的正切值是位移偏转角正切值的 2 倍
### 3. 圆周运动

**基本物理量**：
- 线速度：v = Δl/Δt = ωr
- 角速度：ω = Δθ/Δt = 2π/T
- 周期 T，频率 f = 1/T
- 向心加速度：a向 = v²/r = ω²r
- 向心力：F向 = mv²/r = mω²r

**水平面内的圆周运动**：
- 汽车转弯（路面倾斜/摩擦力提供向心力）
- 圆锥摆
- 转盘上的物体

**竖直面内的圆周运动**：
- **绳模型**（轻绳系小球做圆周运动）：
  - 最高点临界速度：v = √(gr)
  - v ≥ √(gr) 时能做完整圆周运动
- **杆模型**（轻杆固定小球做圆周运动）：
  - 最高点临界速度：v = 0
  - 0 < v < √(gr) 时杆提供支持力
  - v > √(gr) 时杆提供拉力

## 五、万有引力与航天

### 1. 万有引力定律
F = G·m₁m₂/r²（G = 6.67×10⁻¹¹ N·m²/kg²）
**适用条件**：质点间 / 球体间（r 为球心距）
### 2. 天体运动

**环绕天体**（卫星）：
- GMm/r² = mv²/r = mω²r = m(4π²/T²)r
- 轨道速度：v = √(GM/r)（r↑ v↓）
- 角速度：ω = √(GM/r³)
- 周期：T = 2π√(r³/GM)

**近地卫星**：r ≈ R，v = √(gR) ≈ 7.9 km/s（第一宇宙速度）
**卫星轨道速度**：
- 第一宇宙速度（环绕速度）：7.9 km/s
- 第二宇宙速度（逃逸速度）：11.2 km/s
- 第三宇宙速度：16.7 km/s

**地球同步卫星**：
- T = 24h，轨道在赤道正上方
- 轨道半径固定（约 3.6×10⁴ km）
**中心天体质量的计算**：
- 利用环绕天体：M = 4π²r³/GT²
- 利用表面重力：M = gR²/G

**黄金代换式**：GM = gR²

## 六、功与能

### 1. 功
W = Fs cosθ（θ为 F 与 s 的夹角）

**正功与负功**：
- θ < 90°：W > 0（动力做功）
- θ = 90°：W = 0（不做功）
- θ > 90°：W < 0（阻力做功）

**总功**：
- W总 = W₁ + W₂ + W₃ + ...（各力做功的代数和）
- W总 = F合·s·cosθ（合外力做功）
### 2. 功率
P = W/t = Fv（平均功率/瞬时功率）
- 额定功率：发动机长时间正常工作最大功率
- 机车启动的两种方式：
  - **恒定功率启动**：v↑→F↓→a↓→最后匀速
  - **恒定加速度启动**：先匀加速到额定功率，再变加速到匀速
### 3. 动能与动能定理
**动能**：Ek = ½mv²

**动能定理**：W合 = ΔEk = Ek₂ - Ek₁ = ½mv₂² - ½mv₁²
**适用**：无论恒力/变力、直线/曲线、单/多个过程均适用

### 4. 重力势能与弹性势能
**重力势能**：Ep = mgh（相对零势能面）

**重力做功与重力势能变化**：WG = -ΔEp = mgh₁ - mgh₂
**弹性势能**：Ep = ½kx²（弹簧）

### 5. 机械能守恒定律
**条件**：只有重力（或弹簧弹力）做功

**表达式**：
- Ek₁ + Ep₁ = Ek₂ + Ep₂
- ΔEk = -ΔEp（动能增量 = -势能增量）

## 七、动量
### 1. 动量与冲量
- 动量：p = mv（矢量，单位 kg·m/s）
- 冲量：I = Ft（矢量，单位 N·s）
### 2. 动量定理
F·Δt = Δp = mv₂ - mv₁
**应用**：缓冲（延长作用时间减小冲击力）

### 3. 动量守恒定律

**条件**：系统不受外力或合外力为零
**表达式**：m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'

**碰撞类型**：
| 类型 | 特点 | 规律 |
|------|------|------|
| **弹性碰撞** | 动能守恒 | 动量守恒 + 动能守恒 |
| **非弹性碰撞** | 有动能损失 | 动量守恒 |
| **完全非弹性碰撞** | 碰后粘合，动能损失最大 | 动量守恒，共速 |

**一维弹性碰撞结论**：
v₁' = [(m₁ - m₂)v₁ + 2m₂v₂]/(m₁ + m₂)
v₂' = [(m₂ - m₁)v₂ + 2m₁v₁]/(m₁ + m₂)

**爆炸与反冲**：动量守恒，动能增加（内能转化为动能）

## 八、振动与波
### 1. 简谐振动
- 回复力：F = -kx
- 周期公式（弹簧振子）：T = 2π√(m/k)
- 周期公式（单摆）：T = 2π√(L/g)
- 振动图像：x-t 正弦/余弦图像

### 2. 机械波
- 波速：v = λf = λ/T
- 波长、频率、波速关系：v = λf
- 波的干涉：频率相同的两列波叠加，振动加强/减弱
- 波的衍射：障碍物尺寸与波长可比时明显
- 多普勒效应：波源与观察者相对运动时频率变化
