# 高中物理 · 电磁学
> 涵盖：静电场、恒定电流、磁场、电磁感应、交变电流

---

## 一、静电场

### 1. 电荷与库仑定律
- **元电荷**：e = 1.6×10⁻¹⁹ C（元电荷）
- **电荷守恒**：孤立系统电荷代数和不变
- **库仑定律**：F = k·q₁q₂/r²（k = 9.0×10⁹ N·m²/C²）
  - 适用：真空中点电荷
  - 方向：同种电荷排斥，异种电荷吸引

### 2. 电场强度
**定义**：E = F/q（矢量，单位 N/C 或 V/m）
**点电荷的电场**：E = kQ/r²

**电场强度叠加**：平行四边形定则

**电场线**：
- 始于正电荷，终止于负电荷
- 疏密表示场强大小
- 切线方向表示场强方向
- 不闭合、不相交

**匀强电场**：E = U/d（d 为沿电场方向的距离）

### 3. 电势与电势差
- **电势差**：UAB = WAB/q（电场力做功与电荷量比值）
- **电势**：φA = UAO（O 为零势能点）
- **电场力做功**：WAB = qUAB
- **电势能**：Ep = qφ
- **电场力做功与电势能变化**：WAB = -ΔEp = EpA - EpB

**等势面**：
- 沿等势面移动电荷电场力不做功
- 等势面与电场线垂直
- 等势面密处场强大

### 4. 电容器与电容
C = Q/U = εᵣS/(4πkd)（平行板电容器）

**充电后**：
- 断开电源：Q 不变，d↑→C↓→U↑→E 不变
- 不断电源：U 不变，d↑→C↓→Q↓→E↓
**电容器的串并联**：
- 串联：1/C = 1/C₁ + 1/C₂，各板带电量相等
- 并联：C = C₁ + C₂，各板电压相等

### 5. 带电粒子在电场中的运动
- **加速**：qU = ½mv² - ½mv₀²
- **偏转**（类平抛）：
  - 加速度：a = qE/m = qU/(md)
  - 偏转位移：y = ½at² = qUL²/(2mdv₀²)
  - 偏转角：tanθ = qUL/(mdv₀²)

## 二、恒定电流
### 1. 基本概念
- **电流**：I = q/t（定向移动，方向为正电荷移动方向）
- **电阻**：R = ρL/S（电阻定律，ρ 为电阻率）
- **电阻率**：金属随温度升高而增大；半导体随温度升高而减小
### 2. 欧姆定律
**部分电路**：I = U/R

**闭合电路**：I = E/(R + r)

**路端电压**：
- U = E - Ir
- R↑→I↓→U↑
- R = ∞（断路）：U = E
- R = 0（短路）：I = E/r（很大）

### 3. 电功与电功率
- **电功**：W = UIt（普遍适用）
- **电功率**：P = UI（普遍适用）
- **焦耳热**：Q = I²Rt（纯电阻电路 Q = W）
- **电源功率**：P总 = EI
- **输出功率**：P出 = UI
- **效率**：η = P出/P总 = U/E
- **最大输出功率**：当 R = r 时，Pmax = E²/(4r)

### 4. 电阻的串并联

| 连接方式 | 总电阻 | 电流 | 电压 |
|---------|--------|------|------|
| **串联** | R = R₁ + R₂ + ... | I 处处相等 | U = U₁ + U₂ + ... |
| **并联** | 1/R = 1/R₁ + 1/R₂ + ... | I = I₁ + I₂ + ... | U 处处相等 |

### 5. 电表的改装
- **电流表扩大量程**：并联小电阻（分流）
- **电压表扩大量程**：串联大电阻（分压）
- **欧姆表**：内接电源 + 调零电阻

### 6. 电路的动态分析
**程序法**：
1. 判断总电阻变化（固定→移动滑片/开关通断）
2. 判断总电流变化（I = E/(R + r)）
3. 判断路端电压变化（U = E - Ir）
4. 由总到分，逐层分析

**串反并同**（含电源内阻时适用）：
- 与变化电阻串联的元件，其电流/电压/功率的变化与电阻变化相反
- 与变化电阻并联的元件，其电流/电压/功率的变化与电阻变化相同

## 三、磁场
### 1. 磁感应强度
B = F/(IL)（B⊥I，单位特斯拉 T）
**磁感线**：
- 外部：N→S；内部：S→N（闭合曲线）
- 疏密表强弱，切线表方向
**常见磁场**：
- 条形磁铁：两极附近最强
- 蹄形磁铁：两极间匀强
- 通电直导线：同心圆（右手螺旋定则）
- 通电螺线管：内部匀强
### 2. 安培力
F = BIL sinθ（θ为 B 与 I 的夹角）

**方向**：左手定则（B 穿手心，四指电流方向，拇指受力方向）

**安培力作用下物体的运动**：先受力分析，再列牛顿第二定律方程
### 3. 洛伦兹力
f = qvB sinθ（θ为 B 与 v 的夹角）

**特点**：
- 洛伦兹力永不做功（只改变速度方向，不改变大小）
- 方向：左手定则（正电荷四指为运动方向，负电荷相反）
**带电粒子在匀强磁场中的运动**：
- v ∥ B：匀速直线运动
- v ⊥ B：匀速圆周运动
  - 半径：r = mv/(qB)
  - 周期：T = 2πm/(qB)（与速度无关！）
  - 角速度：ω = qB/m

**回旋加速器**：
- 电场加速，磁场偏转
- 最大动能：Ekmax = (qBR)²/(2m)（与加速电压无关！）
**质谱仪**：q/m = 2U/(B²r²)

### 4. 带电粒子在复合场中的运动
- **速度选择器**：qE = qvB → v = E/B（与电荷无关）
- **霍尔效应**：U = BI/(nqd)（判断载流子类型）
- **磁流体发电机**：电动势 E = Bdv

## 四、电磁感应
### 1. 磁通量
Φ = BS cosθ（θ为 B 与 S 法线夹角，单位 Wb）
### 2. 楞次定律
感应电流的磁场总要阻碍引起感应电流的磁通量的变化。
**口诀**：增反减同，来拒去留，增缩减扩

**应用步骤**：
1. 确定原磁场方向
2. 判断磁通量变化（增加/减少）
3. 根据楞次定律判断感应电流的磁场方向
4. 右手螺旋定则判断感应电流方向

### 3. 法拉第电磁感应定律
E = n·ΔΦ/Δt（n 为线圈匝数）

**导体切割磁感线**：E = BLv sinθ（θ为 B 与 v 的夹角）

**自感现象**：
- 自感电动势：E = L·ΔI/Δt
- 通电自感：灯泡逐渐变亮
- 断电自感：灯泡闪亮后熄灭（或逐渐熄灭）
### 4. 感应电流与感应电荷量
- 感应电流 I = E/R = nΔΦ/(RΔt)
- 感应电荷量 q = I·Δt = nΔΦ/R（与时间无关！）

## 五、交变电流
### 1. 正弦式交流电的产生
**瞬时值**：
- e = E_m sin ωt（E_m = NBSω）
- u = U_m sin ωt
- i = I_m sin ωt

**有效值**（热效应定义）：
- E = E_m/√2
- U = U_m/√2
- I = I_m/√2

**周期与频率**：T = 2π/ω，f = 1/T = ω/(2π)

### 2. 变压器
**理想变压器**：
- U₁/U₂ = n₁/n₂
- I₁/I₂ = n₂/n₁（单一副线圈）
- 功率关系：P₁ = P₂
**远距离输电**：
- P输 = U输·I线
- 损失功率：P损 = I线²R线
- 损失电压：U损 = I线R线
- **高压输电**：提高电压→减小电流→减小损耗
### 3. 电感与电容对交流的影响
- 电感：通直流阻交流，通低频阻高频
- 电容：通交流隔直流，通高频阻低频
