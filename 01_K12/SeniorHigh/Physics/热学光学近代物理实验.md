# 高中物理 · 热学 / 光学 / 近代物理 / 实验

---

## 一、热学
### 1. 分子动理论
**基本观点**：
- 物质由大量分子组成
- 分子永不停息地做无规则运动（布朗运动、扩散现象）
- 分子间存在引力和斥力

**阿伏伽德罗常数**：NA = 6.02×10²³ mol⁻¹
**分子大小估算**：
- 固体/液体分子：d = ³√(V₀)（V₀ 为一个分子占据的体积）
- 气体分子间距：d = ³√(V₀)（分子大小可忽略）
**布朗运动**：
- 悬浮颗粒的无规则运动
- 原因：液体分子撞击不平衡
- 温度越高、颗粒越小，布朗运动越剧烈
### 2. 温度与分子动能
**温度**：分子热运动平均动能的标志
- 物体的内能 ≠ 机械能
**分子平均动能**：Ek̄ = (3/2)kT（k 为玻尔兹曼常数）

### 3. 分子势能与内能
- 分子势能：由分子间相对位置决定
- 内能：分子动能 + 分子势能（与温度、体积、物质的量有关）

### 4. 热力学定律
**热力学第一定律**（能量守恒）：ΔU = Q + W

| 过程 | ΔU | Q | W |
|------|----|----|-----|
| 等温膨胀 | 0 | 吸热 | 对外做功 |
| 等容升温 | 增加 | 吸热 | 0 |
| 绝热膨胀 | 减少 | 0 | 对外做功 |
| 等压膨胀 | 增加 | 吸热 | 对外做功 |

**热力学第二定律**：
- 克劳修斯表述：热量不能自发地从低温物体传向高温物体
- 开尔文表述：不可能从单一热源吸热使之完全变成功而不产生其他影响
- 实质：涉及热现象的宏观过程具有方向性（不可逆）

**热力学第三定律**：绝对零度不可达
### 5. 气体实验定律（理想气体）

**玻意耳定律**（等温）：p₁V₁ = p₂V₂
**查理定律**（等容）：p₁/T₁ = p₂/T₂
**盖·吕萨克定律**（等压）：V₁/T₁ = V₂/T₂
**理想气体状态方程**：pV/T = C（常数）→ p₁V₁/T₁ = p₂V₂/T₂
**克拉珀龙方程**：pV = nRT（n 为物质的量，R = 8.31 J/(mol·K)）

## 二、光学
### 1. 几何光学

**光的反射定律**：反射角 = 入射角（法线两侧）
**光的折射定律（斯涅耳定律）**：n = sin i / sin r = c/v

**折射率**：
- n > 1（真空→介质）
- 光密介质折射率大，光疏介质折射率小
**全反射**：
- 条件：光从光密→光疏，且 i ≥ C（临界角）
- sin C = 1/n

**棱镜**：光通过棱镜向底面偏折，发生色散

### 2. 物理光学

**光的干涉**：
- 双缝干涉：Δx = λL/d（亮暗条纹交替）
  - 光程差 δ = kλ → 亮纹
  - 光程差 δ = (2k+1)λ/2 → 暗纹
- 薄膜干涉：肥皂泡、油膜上的彩色条纹
**光的衍射**：
- 单缝衍射：中间亮纹最宽最亮，两侧依次变暗
- 条件：障碍物尺寸与波长可比
**光电效应**：
- Ek = hν - W₀（爱因斯坦光电效应方程）
- 极限频率：ν₀ = W₀/h
- 瞬时性、存在截止频率
- 光电流与光强成正比（频率一定时）
**光的波粒二象性**：
- 光既有波动性又有粒子性
- 波长长→波动性显著；波长短→粒子性显著
- 物质波（德布罗意波）：λ = h/p

## 三、近代物理
### 1. 原子结构

| 模型 | 提出者 | 要点 |
|------|--------|------|
| 枣糕模型 | 汤姆生 | 正电荷均匀分布，电子嵌在其中 |
| 核式结构 | 卢瑟福 | α粒子散射实验 → 原子核小且带正电 |
| 玻尔模型 | 玻尔 | 轨道量子化、能级跃迁 |

**玻尔假设**：
- 定态：电子在特定轨道上运动不辐射能量
- 跃迁：hν = E_m - E_n
- 轨道量子化：mvr = nh/(2π)

**能级与光谱**：
- 基态：n = 1，能量最低
- 激发态：n ≥ 2
- 从高能级跃迁到低能级 → 发光
- 从低能级跃迁到高能级 → 吸收光
- 一群氢原子从 n 能级跃迁，可发射 C(n,2) 种频率的光
### 2. 原子核
**天然放射现象**：
- α射线：⁴He 核，电离能力强，穿透能力弱
- β射线：电子，电离能力中等，穿透能力中等
- γ射线：光子，电离能力弱，穿透能力强
- 三种射线均来自原子核

**衰变**：
- α衰变：₉₂²³⁸U → ₉₀²³⁴Th + ₂⁴He（质量数减4，电荷数减2）
- β衰变：₉₀²³⁴Th → ₉₁²³⁴Pa + ₋₁⁰e（质量数不变，电荷数加1）
**半衰期**：半数原子核衰变所需时间，与外界条件无关

**核反应类型**：
| 类型 | 示例 |
|------|------|
| **衰变** | ₉₂²³⁸U → ₉₀²³⁴Th + ₂⁴He |
| **人工转变** | ₇¹⁴N + ₂⁴He → ₈¹⁷O + ₁¹H |
| **裂变** | ₉₂²³⁵U + ₀¹n → ₅⁶¹⁴¹Ba + ₃⁶⁹²Kr + 3₀¹n |
| **聚变** | ₁²H + ₁³H → ₂⁴He + ₀¹n |

**质量亏损与核能**：ΔE = Δm·c²（Δm：核反应前后质量差）

**比结合能**：结合能/核子数，越大越稳定

## 四、物理实验
### 1. 力学实验

| 实验名称 | 原理 | 关键仪器 | 数据处理 |
|---------|------|---------|---------|
| **匀变速运动** | Δx = aT² | 打点计时器、纸带 | 逐差法求 a |
| **验证牛顿第二定律** | a ∝ F，a ∝ 1/m | 打点计时器、小车、轨道 | 图像法（a-F，a-1/m）|
| **探究平抛运动** | 水平匀速、竖直自由落体 | 斜槽、白纸、重锤 | 轨迹描点 |
| **验证机械能守恒** | mgh = ½mv² | 打点计时器、重锤 | 比较 gh 与 v²/2 |

**打点计时器**：
- 电磁打点计时器：4-6V 交流电，50Hz
- 电火花计时器：220V 交流电，50Hz
- 打点间隔：0.02s

**纸带处理**：
- 瞬时速度：v_n = (x_n + xₙ₊₁)/(2T)
- 加速度（逐差法）：a = [(x₄ + x₅ + x₆) - (x₁ + x₂ + x₃)]/(9T²)

**验证牛顿第二定律实验注意**：
- 平衡摩擦力：倾斜长木板
- 钩码质量远小于小车质量（使 F ≈ mg）
### 2. 电学实验

| 实验名称 | 原理 | 电路 | 关键 |
|---------|------|------|------|
| **伏安法测电阻** | R = U/I | 内接/外接 | 电表量程选择 |
| **测电源电动势和内阻** | U = E - Ir | 伏安法/安阻法/伏阻法 | 作图处理 |
| **多用电表的使用** | 欧姆定律 | 内置电源 | 调零、选择合适的倍率 |
| **描绘小灯泡伏安特性** | I-U 图像 | 分压式接法 | 小电阻用外接 |

**电流表内外接法选择**：
- 外接：R测 << Rv（小电阻）→ 测小电阻
- 内接：R测 >> RA（大电阻）→ 测大电阻
- 临界值法：对比 R测 与 √(RA·Rv)

**滑动变阻器的接法**：
- 限流式：电路中耗能少，但调节范围有限
- 分压式：调节范围从 0 开始，电压调节范围大
**多用电表**：
- 测电阻前需调零（短接表笔调欧姆零）
- 换倍率后重新调零
- 读数在表盘中央附近最准确

### 3. 实验误差分析
- **系统误差**：仪器不精确、实验方法不完善 → 可减小但不可消除
- **偶然误差**：人为因素、环境因素 → 多次测量取平均可减小
- **有效数字**：可靠数字 + 一位估计数字
### 4. 实验设计的基本原则
1. **科学性**：原理正确
2. **可行性**：仪器和条件满足
3. **安全性**：不超过仪器量程
4. **精确性**：误差尽可能小
5. **可重复性**：实验条件和步骤明确
