# 高中物理 · 近代物理专题

> 涵盖量子力学初步、原子物理、原子核物理、相对论基础、天体物理

---

## 一、量子力学初步
### 1. 黑体辐射与能量量子化
- **黑体**：能完全吸收各种波长电磁波的理想物体
- **普朗克假设**：能量不是连续的，是一份一份的
  - E = hν（h = 6.626×10⁻³⁴ J·s，普朗克常量）
### 2. 光电效应

**实验规律**：
- 存在截止频率ν₀（极限频率），低于ν₀无论多强光都不发生光电效应
- 光电流强度与入射光强度成正比
- 瞬时性（延迟时间不超过 10⁻⁹ s）
**爱因斯坦光子说**：
- 光本身就是由一个个不可分割的能量子（光子）组成
- E = hν

**爱因斯坦光电效应方程**：Ek = hν - W₀（Ek = ½mv²，W₀为逸出功）

**【应用】光电管、光控开关、太阳能电池**

### 3. 光的波粒二象性
- 光既有波动性又有粒子性
- **德布罗意波**（物质波）：λ = h/p = h/(mv)
- **不确定性原理**（海森堡）：Δx·Δp ≥ h/(4π)
  - 位置和动量不能同时被精确测量

## 二、原子结构
### 1. 原子结构模型的演变
| 模型 | 提出者 | 要点 |
|------|--------|------|
| 实心球模型 | 道尔顿 | 原子是不可再分的实心球 |
| 行星模型 | 卢瑟福 | 核式结构：原子核很小带正电，电子绕核旋转 |
| 玻尔模型 | 玻尔 | 定态假设、跃迁假设、量子化条件 |
| 电子云模型 | 量子力学 | 电子在核周围出现的概率分布 |

### 2. 玻尔理论

**三个假设**：
1. **定态假设**：原子只能处于一系列不连续的能量状态
2. **跃迁假设**：原子从高能级跃迁到低能级时辐射光子
   - hν = E_m - Eₙ（E_m > Eₙ）
3. **轨道量子化**：电子轨道满足 mvr = n·h/(2π)

**氢原子能级图**：
- E₁ = -13.6 eV（基态）
- E₂ = -3.4 eV
- E₃ = -1.51 eV
- E∞ = 0 eV（电离）

**能级跃迁规律**：
- 吸收光子：从低能级→高能级
- 辐射光子：从高能级→低能级
- 电离：吸收能量≥|E₁|，脱离原子束缚
### 3. 光谱
- **线状谱**：原子的特征谱线（用于元素鉴定）
- **氢原子光谱**：巴耳末系（可见光区）

## 三、原子核物理

### 1. 天然放射现象
- **α射线**：⁴He 核，电离能力强，穿透能力弱
- **β射线**：电子，电离能力中等，穿透能力中等
- **γ射线**：电磁波，电离能力弱，穿透能力强

**衰变规律**：
- α衰变：ᵃᶻX → ᵃ⁻⁴ᶻ₋₂Y + ⁴₂He
- β衰变：ᵃᶻX → ᵃᶻ₊₁Y + ⁰₋₁e

**半衰期**：放射性元素的原子核有半数发生衰变所需的时间
- 半衰期由原子核内部因素决定，与物理/化学状态无关
- N = N₀(1/2)^(t/T)

### 2. 核反应
| 类型 | 举例 | 特点 |
|------|------|------|
| **衰变** | ²³⁸₉₂U → ²³⁴₉₀Th + ⁴₂He | 自发进行 |
| **人工转变** | ¹⁴₇N + ⁴₂He → ¹⁷₈O + ¹₁H | 用α粒子轰击 |
| **裂变** | ²³⁵₉₂U + ¹₀n → ¹⁴¹₅₆Ba + ⁹²₃₆Kr + 3¹₀n | 重核分裂，链式反应 |
| **聚变** | ²₁H + ³₁H → ⁴₂He + ¹₀n | 轻核结合，需高温 |

### 3. 核能
**质能方程**（爱因斯坦）：E = mc²
**质量亏损**：Δm = (反应前质量) - (反应后质量)
**核能计算**：ΔE = Δm·c²

**核电站工作原理**：核裂变 → 热能 → 机械能 → 电能

## 四、相对论基础

### 1. 狭义相对论（爱因斯坦 1905）
**两个基本假设**：
1. **相对性原理**：所有惯性参考系中物理规律相同
2. **光速不变原理**：真空中光速在任何惯性系中都是常量 c

**主要结论**：
- **时间延缓**（钟慢效应）：Δt = Δt₀/√(1-v²/c²)
- **长度收缩**（尺缩效应）：L = L₀√(1-v²/c²)
- **质速关系**：m = m₀/√(1-v²/c²)
- **质能关系**：E = mc²

### 2. 广义相对论（1915）
- 等效原理：引力场与加速参考系等效
- 引力红移：引力场中光频率变低
- 光在引力场中弯曲（光线偏折）

## 五、天体物理
### 1. 万有引力与天体运动
F = G·M₁M₂/r²（G = 6.67×10⁻¹¹ N·m²/kg²）
**天体运动规律**：
- GMm/r² = mv²/r = mω²r = m(4π²/T²)r
- 卫星运行速度：v = √(GM/r)

| 轨道 | 速度 | 周期 |
|------|------|------|
| 近地轨道 | 7.9 km/s（第一宇宙速度）| ≈ 85 min |
| 地球同步轨道 | 3.1 km/s | 24 h |

### 2. 宇宙速度
- **第一宇宙速度**（环绕速度）：7.9 km/s
- **第二宇宙速度**（逃逸速度）：11.2 km/s
- **第三宇宙速度**：16.7 km/s

### 3. 恒星的演化
星际气体 → 原恒星 → 主序星 → 红巨星/超巨星 → 白矮星/中子星/黑洞
