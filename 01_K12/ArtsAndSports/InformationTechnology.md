# K12 信息技术

> 涵盖：计算机基础、网络与安全、编程入门、人工智能、数字化学习

---

## 一、计算机基础

### 1. 计算机的发展

| 阶段 | 时间 | 元器件 | 特点 |
|------|------|--------|------|
| 第一代 | 1946-1957 | 电子管 | 体积大、功耗大、速度慢 |
| 第二代 | 1958-1964 | 晶体管 | 体积减小、可靠性提高 |
| 第三代 | 1965-1970 | 中小规模集成电路 | 性能大幅提升 |
| 第四代 | 1971至今 | 大规模/超大规模集成电路 | 微型化、高性能 |
| 第五代 | 未来 | 人工智能芯片 | 智能化、类脑计算 |

### 2. 计算机系统组成

```
计算机系统
├── 硬件系统
│   ├── 中央处理器（CPU）
│   │   ├── 运算器（ALU）
│   │   └── 控制器（CU）
│   ├── 存储器
│   │   ├── 内存储器（RAM、ROM）
│   │   └── 外存储器（硬盘、U 盘、光盘）
│   ├── 输入设备（键盘、鼠标、麦克风、扫描仪、摄像头）
│   └── 输出设备（显示器、打印机、音箱）
└── 软件系统
    ├── 系统软件（Windows/macOS/Linux 操作系统）
    └── 应用软件（Office、浏览器、游戏、设计软件）
```

**冯·诺依曼体系结构**：存储程序、程序控制

### 3. 数据存储单位

- 位（bit）：最小存储单位，0或1
- 字节（Byte）：1B = 8bit
- 1KB = 1024B, 1MB = 1024KB, 1GB = 1024MB, 1TB = 1024GB

### 4. 数据在计算机中的表示

**进制转换**：

| 进制 | 数字符号 | 示例 |
|------|---------|------|
| 二进制（B） | 0, 1 | 1010₂ = 10₁₀ |
| 八进制（O） | 0-7 | 12₈ = 10₁₀ |
| 十进制（D） | 0-9 | 10 |
| 十六进制（H） | 0-9, A-F | A₁₆ = 10₁₀ |

**二进制转十进制**：按权展开求和

$$1101_2 = 1 \times 2^3 + 1 \times 2^2 + 0 \times 2^1 + 1 \times 2^0 = 8 + 4 + 0 + 1 = 13_{10}$$

**十进制转二进制**：除2取余倒排

$$13_{10} = 1101_2$$

**字符编码**：

- ASCII 码（7位，128个字符）
- Unicode（统一编码，支持各国文字）
- UTF-8（可变长度编码，兼容 ASCII）

### 5. 计算机与操作系统基础操作

- 文件管理：新建、复制/移动/删除/重命名文件
- 文件夹结构：路径（绝对路径、相对路径）
- 快捷键：Ctrl+C（复制）、Ctrl+V（粘贴）、Ctrl+X（剪切）、Ctrl+Z（撤销）、Ctrl+S（保存）
- 桌面管理、任务管理器、控制面板

## 二、网络基础

### 1. 网络分类

| 类型 | 范围 | 示例 |
|------|------|------|
| 局域网（LAN） | 一栋楼/一个单位 | 学校机房网络 |
| 城域网（MAN） | 一座城市 | 城市宽带 |
| 广域网（WAN） | 跨地区/国家 | 互联网（Internet） |
| 个人区域网（PAN） | 个人设备之间 | 蓝牙连接 |

### 2. 网络设备

- **路由器**：连接不同网络，数据包转发
- **交换机**：连接局域网内设备
- **调制解调器（Modem）**：数字信号与模拟信号互转
- **网卡**：计算机接入网络的接口
- **无线接入点（AP）**：提供 Wi-Fi 信号

### 3. IP 地址与域名

- **IP 地址**：互联网上设备的唯一标识（IPv4 32位，如192.168.1.1；IPv6 128位）
- **域名**：便于记忆的网址（如 www.baidu.com）
- **DNS（域名系统）**：将域名解析为 IP 地址

### 4. 网络协议

- **TCP/IP**：互联网核心协议
  - TCP（传输控制协议）：可靠连接
  - IP（网际协议）：寻址和路由
- **HTTP/HTTPS**：网页传输协议（HTTPS 加密）
- **FTP**：文件传输协议
- **SMTP/POP3**：电子邮件协议

### 5. 互联网应用

- 万维网（WWW）、电子邮件、即时通讯、社交网络
- 搜索引擎、在线学习、电子商务、云计算
- 物联网（IoT）：设备互联，智能家居

## 三、信息安全与道德

### 1. 网络安全风险

| 风险类型 | 描述 | 防范措施 |
|---------|------|---------|
| 病毒 | 恶意程序，损坏文件系统 | 安装杀毒软件，不下载不明文件 |
| 木马 | 伪装成正常程序窃取信息 | 不从不明来源下载软件 |
| 蠕虫 | 自我复制传播 | 及时更新系统补丁 |
| 钓鱼 | 伪装网站骗取密码 | 核对网址，不点击不明链接 |
| 勒索软件 | 加密文件索要赎金 | 定期备份重要数据 |
| DDoS 攻击 | 大量请求使服务器瘫痪 | 防火墙、流量清洗 |

### 2. 个人信息保护

- 不随意公开：姓名、电话、住址、身份证号、银行账号
- 设置强密码（大小写字母+数字+符号，至少8位）
- 不同网站使用不同密码
- 开启双重身份验证

### 3. 信息道德

- 尊重知识产权，不使用盗版软件
- 不传播虚假信息、网络谣言
- 不侵犯他人隐私
- 不进行网络欺凌
- 合理使用网络，控制上网时间

### 4. 法律法规

- 《中华人民共和国网络安全法》
- 《中华人民共和国数据安全法》
- 《中华人民共和国个人信息保护法》
- 《未成年人网络保护条例》
- 网络不是法外之地

|------|------|-------------|
| Scratch | 图形化编程，适合少儿 | 编程启蒙、游戏制作 |

### 2. Python 基础（推荐入门语言）

**输入输出**：

```python
print("Hello, World!")
name = input("请输入你的名字：")
```

**变量与数据类型**：

- 整数（int）：1, 100, -5
- 浮点数（float）：3.14, -0.5
- 字符串（str）："Hello", 'Python'
- 布尔值（bool）：True, False
- 列表（list）：[1, 2, 3]
- 字典（dict）：{"name": "Tom", "age": 12}

**条件语句**：

```python
if score >= 60:
    print("及格")
else:
    print("不及格")
```

**循环**：

```python
for i in range(5):
    print(i)

while count < 10:
    print(count)
    count = count + 1
```

**函数**：

```python
def add(a, b):
    return a + b

print(add(3, 5))  # 输出 8
```

### 3. 计算思维

| 思维方法 | 描述 |
|---------|------|
| 分解 | 将复杂问题分解为小问题 |
| 模式识别 | 找出规律和共同点 |
| 抽象 | 关注关键信息，忽略不重要细节 |
| 算法设计 | 制定步骤化解决方案 |

### 4. 算法基础

**排序算法**：

- 冒泡排序：相邻比较交换，O(n²)
- 选择排序：每轮最小放前面，O(n²)
- 插入排序：将元素插入已排序序列，O(n²)
- 快速排序：分治策略，基准分区，O(n log n)

**搜索算法**：

- 顺序搜索：逐个查找，O(n)
- 二分搜索：要求已排序，每次排除一半，O(log n)

**算法复杂度**：

- **时间复杂度**：算法运行时间随数据规模增长的增长速率
- **空间复杂度**：算法运行时占用的内存空间
- 大 O 表示法衡量算法效率：O(1) < O(log n) < O(n) < O(n log n) < O(n²) < O(2ⁿ)

**流程图**：

```
开始 → 输入 → 判断 → 处理 → 输出 → 结束
             ↘      ↗
             └── 循环 ──┘
```

- 起止框（圆角矩形）、处理框（矩形）、判断框（菱形）、输入输出框

## 六、人工智能基础

### 1. 什么是人工智能

- **定义**：让计算机模拟人类智能行为
- **发展历程**：图灵测试 → 专家系统 → 深度学习 → 大语言模型

### 2. 机器学习的主要类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 监督学习 | 用带标签数据训练 | 分类（垃圾邮件识别）、回归（房价预测） |
| 无监督学习 | 用无标签数据发现模式 | 聚类（用户分组）、降维 |
| 强化学习 | 通过奖惩机制学习策略 | 博弈（AlphaGo）、自动驾驶 |

### 3. AI 主要领域

| 领域 | 描述 | 应用 |
|------|------|------|
| 机器学习 | 从数据中学习规律 | 推荐系统、信用评分 |
| 深度学习 | 多层神经网络学习 | 图像识别、语音识别 |
| 自然语言处理 | 理解和生成语言 | 机器翻译、智能客服、ChatGPT |
| 计算机视觉 | 让计算机"看懂"图像 | 人脸识别、自动驾驶 |
| 语音识别 | 将语音转为文字 | 语音助手（Siri/小爱） |
| 机器人 | 能感知和行动的机器 | 工业机器人、扫地机器人 |

### 4. AI 的伦理

- AI 应公平、透明、可解释
- 防止算法偏见和歧视
- 保护数据隐私
- AI 不应完全取代人类决策
