# 营养与健康

> 青少年营养健康教育 (Nutrition and Wellness Education) 培养科学的饮食观念和健康的生活方式，为终身的身体健康与心理健康奠定基础。

## 营养基础 (Nutrition Fundamentals)

### 七大营养素

人体需要七大类营养素来维持正常的生理功能：

| 营养素 | 功能 | 主要食物来源 | 每日建议占比 |
| :--- | :--- | :--- | :--- |
| **碳水化合物 (Carbohydrates)** | 提供主要能量，是大脑唯一直接能源 | 谷物、薯类、水果 | 50-65% |
| **蛋白质 (Proteins)** | 组织修复、酶和激素合成 | 肉、蛋、奶、豆制品 | 10-20% |
| **脂肪 (Fats)** | 储能、脂溶性维生素吸收、细胞膜构成 | 植物油、坚果、鱼肉 | 20-30% |
| **维生素 (Vitamins)** | 代谢调节、免疫支持 | 蔬菜、水果、肝脏 | — |
| **矿物质 (Minerals)** | 骨骼、神经传导、酶活性 | 奶制品、绿叶菜、海鲜 | — |
| **水 (Water)** | 运输、温度调节、代谢介质 | 饮水、汤、水果 | 男 3.7L / 女 2.7L |
| **膳食纤维 (Dietary Fiber)** | 肠道健康、血糖稳定、饱腹感 | 全谷物、豆类、蔬菜 | 25-35g |

### 能量平衡 (Energy Balance)

能量平衡是体重管理的基础公式：

$$ \text{能量平衡} = \text{能量摄入} - \text{能量消耗} $$

- **能量摄入**：通过饮食获取的卡路里
- **能量消耗**：基础代谢率 (BMR) + 食物热效应 (TEF) + 身体活动 (PA)
- **青少年 BMR 估算 (Mifflin-St Jeor 方程)**：

  $$ \text{男性} = 10 \times \text{体重(kg)} + 6.25 \times \text{身高(cm)} - 5 \times \text{年龄(y)} + 5 $$
  $$ \text{女性} = 10 \times \text{体重(kg)} + 6.25 \times \text{身高(cm)} - 5 \times \text{年龄(y)} - 161 $$

| 状态 | 能量平衡 | 体重变化 |
| :--- | :--- | :--- |
| 能量平衡 | 摄入 = 消耗 | 体重稳定 |
| 正能量平衡 | 摄入 > 消耗 | 体重增加 |
| 负能量平衡 | 摄入 < 消耗 | 体重减少 |

## 青少年特殊营养需求

### 青春期生长发育的营养特点

- **生长突增**：女孩 10-14 岁、男孩 12-16 岁达到生长峰值
- **骨骼发育**：钙需要量达 1000-1300 mg/日
- **铁需求**：女孩月经期铁需求增加至 15-18 mg/日
- **蛋白质需求**：较成人高约 20-30%
- **早餐重要性**：不吃早餐导致注意力下降 30-40%

### 常见营养素缺乏

| 缺乏的营养素 | 症状 | 高危人群 | 改善食物 |
| :--- | :--- | :--- | :--- |
| 铁 (Iron) | 疲劳、注意力不集中、贫血 | 青春期女生、素食者 | 红肉、菠菜、豆类 |
| 钙 (Calcium) | 骨骼发育受影响、抽筋 | 乳糖不耐受者 | 酸奶、豆腐、芝麻酱 |
| 维生素 D | 免疫力下降、情绪低落 | 户外活动不足者 | 晒太阳、蛋黄、鱼肝油 |
| 维生素 B12 | 神经症状、疲劳 | 严格素食者 | 强化谷物、补充剂 |
| 碘 (Iodine) | 甲状腺功能减退 | 内陆地区人群 | 碘盐、海带、紫菜 |

## 健康饮食实践

### 三餐分配原则

$$ \text{早餐 : 午餐 : 晚餐} \approx 3 : 4 : 3 $$

| 餐次 | 时间建议 | 核心构成 | 典型搭配 |
| :--- | :--- | :--- | :--- |
| 早餐 | 7:00-8:00 | 主食 + 蛋白质 + 果蔬 | 全麦面包 + 鸡蛋 + 牛奶 + 苹果 |
| 午餐 | 12:00-13:00 | 主食 + 肉/豆 + 蔬菜 | 米饭 + 鸡胸肉 + 西兰花 |
| 晚餐 | 18:00-19:00 | 易消化主食 + 蔬菜 | 杂粮粥 + 清蒸鱼 + 沙拉 |
| 加餐 (可选) | 10:00 / 15:00 | 水果/坚果/酸奶 | 1 个香蕉 + 一小把杏仁 |

### 健康零食 vs 限制零食

| 类别 | 推荐 | 限制 (每周 < 1 次) |
| :--- | :--- | :--- |
| 健康零食 | 水果、原味坚果、酸奶、全麦饼干 | — |
| 需限制 | 薯片、糖果、含糖饮料、油炸食品 | 奶茶、冰淇淋、辣条 |
| 阅读食品标签 | 关注糖含量 ≤ 15g/100g | 反式脂肪酸 = 0 |

## 运动与健康 (Exercise and Wellness)

### 青少年运动建议

- **每天至少 60 分钟中高强度运动 (MVPA)**：
  - 有氧运动：跑步、游泳、球类运动
  - 肌肉力量：俯卧撑、深蹲、引体向上
  - 骨骼强化：跳跃、跳绳、篮球
- **每周至少 3 次高强度运动**
- **减少久坐时间**：每 45 分钟起身活动 5 分钟

### 运动营养要点

| 运动阶段 | 营养策略 | 示例 |
| :--- | :--- | :--- |
| 运动前 (1-2 小时) | 低脂、适量碳水、少量蛋白质 | 香蕉 + 全麦面包 |
| 运动中 (> 60 分钟) | 补充水分和电解质 | 运动饮料或淡盐水 |
| 运动后 (30 分钟内) | 碳水 + 蛋白质 (3:1) | 牛奶 + 燕麦 / 巧克力奶 |

## 睡眠与心理健康

### 青少年睡眠需求

$$ \text{推荐睡眠时长} = 8\text{–}10 \text{ 小时/天} $$

| 年龄组 | 推荐睡眠 | 常见问题 |
| :--- | :--- | :--- |
| 13-15 岁 | 9-10 小时 | 学业压力导致晚睡 |
| 16-18 岁 | 8-9 小时 | 电子产品干扰睡眠 |
| 睡眠不足影响 | 免疫力下降、情绪波动、学习记忆受损 | 注意力不集中 |

### 屏幕时间管理

- **建议上限**：娱乐屏幕时间 ≤ 2 小时/天
- **睡前 1 小时**：避免使用电子设备 (蓝光抑制褪黑素分泌)
- **20-20-20 法则**：每 20 分钟看 20 英尺外的物体 20 秒

### 心理健康与饮食的联系

脂肪与碳水化合物的类型和比例对大脑神经递质的合成有直接影响：

- **Omega-3 脂肪酸**：EPA 和 DHA 对大脑发育和情绪调节有益，来源：深海鱼、亚麻籽
- **色氨酸 (Tryptophan)**：5-HT (血清素) 合成前体，来源：牛奶、香蕉、坚果
- **B 族维生素**：参与神经递质合成，缺乏与抑郁风险相关

## 常见健康问题与预防

| 健康问题 | 危险因素 | 预防措施 |
| :--- | :--- | :--- |
| 青少年肥胖 | 高热量饮食 + 缺乏运动 | 控制含糖饮料、增加活动量 |
| 近视 | 近距离用眼 + 户外活动不足 | 每天 2 小时户外活动 |
| 饮食失调 (ED) | 体重焦虑、社交媒体影响 | 健康的身体形象教育 |
| 缺铁性贫血 | 月经 + 不均衡饮食 | 增加红肉/绿叶蔬菜摄入 |
| 便秘 | 膳食纤维不足 + 饮水少 | 增加蔬菜水果和水分 |

## 食品营养标签阅读

### 营养成分表解读

中国预包装食品营养标签通则 (GB 28050-2011) 要求标示"1+4"核心营养素：

| 项目 | 每 100g/mL | 营养素参考值 % (NRV%) |
| :--- | :--- | :--- |
| **能量 (Energy)** | 1800 kJ | 21% |
| **蛋白质 (Protein)** | 6.0 g | 10% |
| **脂肪 (Fat)** | 15.0 g | 25% |
| **碳水化合物 (Carbohydrate)** | 60.0 g | 20% |
| **钠 (Sodium)** | 400 mg | 20% |

**判别标准 (每日参考值)**：
- 低脂：≤ 3g/100g (固体) 或 ≤ 1.5g/100mL (液体)
- 低糖：≤ 5g/100g (固体) 或 ≤ 5g/100mL (液体)
- 高蛋白：≥ 20% NRV
- 高膳食纤维：≥ 6g/100g

### 配料表解读

配料按含量从高到低排列。常见的"隐形"成分：

| 成分 | 在配料表中的别名 | 健康注意 |
| :--- | :--- | :--- |
| 添加糖 | 白砂糖、果葡糖浆、麦芽糖浆、蜂蜜、浓缩果汁 | 过量摄入与肥胖、龋齿相关 |
| 反式脂肪酸 | 氢化植物油、起酥油、人造奶油、代可可脂 | 增加心血管疾病风险，应 < 2g/日 |
| 钠 | 食盐、味精、小苏打、亚硝酸钠 | 高钠与高血压相关 |

## 饮食文化与社会

### 不同饮食模式的比较

| 饮食模式 | 特点 | 健康评价 | 可持续性 |
| :--- | :--- | :--- | :--- |
| 地中海饮食 (Mediterranean) | 橄榄油为主、大量蔬果、鱼类、适量红酒 | ★★★★★ | ★★★★ |
| 东方传统饮食 | 谷物为主、适量鱼肉、发酵食品 | ★★★★ | ★★★★★ |
| 西式快餐饮食 | 高脂肪、高糖、高盐、低纤维 | ★★ | ★★ |
| 素食 (Vegetarian) | 植物性食物为主、不吃肉 | ★★★★ (计划良好时) | ★★★★★ |
| 生酮饮食 (Keto) | 极低碳水、高脂肪 | ★★★ (短期有效) | ★★ |

### 食物与身份认同

饮食习惯与个人的文化背景、家庭传统、价值观念密切相关。在营养教育中应尊重文化差异，避免"一刀切"的建议。例如：
- 某些文化中的发酵食品 (如纳豆、泡菜、酸菜) 含有益生菌
- 宗教饮食禁忌 (清真 Halal、犹太 Kosher) 需要理解和尊重
- 本地当季食材既有营养优势也有文化和经济意义

### 校园午餐的营养设计

理想的校园午餐应满足以下标准：
- 能量占比：午餐占全日能量的 35-40%
- 荤素比例：1:2 (一份荤菜 + 两份素菜)
- 主食粗细搭配：全谷物占主食的 1/3
- 每周食材多样性：不少于 25 种不同食材
- 低盐低油：每餐用盐 < 2g，用油 < 10g

## 情绪化饮食 (Emotional Eating)

### 识别情绪化饮食

| 生理饥饿 (Physical Hunger) | 情绪饥饿 (Emotional Hunger) |
| :--- | :--- |
| 逐渐出现 (胃咕咕叫) | 突然出现 (特别想吃某样东西) |
| 任何食物都可以满足 | 只有特定食物 (通常是高糖高脂) 才能满足 |
| 吃饱后自然停止 | 即使吃饱了也可能继续吃 |
| 进食后感到满足和精力充沛 | 进食后可能感到内疚或羞耻 |

### 应对策略

- **暂停 10 分钟法**：感觉想暴食时，先做 10 分钟其他事情 (散步、深呼吸、喝水)
- **情绪日记**：记录"什么时间 → 什么情绪 → 吃了什么 → 吃完后的感受"
- **替代活动清单**：列出除吃以外的 10 种缓解情绪的方式 (听歌、画画、跑步、打电话)
- **正念饮食 (Mindful Eating)**：专注在当下的进食体验，不评判，不分心
