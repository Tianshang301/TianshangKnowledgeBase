# 逻辑推理

## 概述

逻辑推理是批判性思维的基础。本文档介绍命题与推理、演绎推理、归纳推理、类比推理、常见逻辑谬误以及逻辑推理练习题。
## 命题与推理
### 命题
- **定义**：能够判断真假的陈述句。
- **类型**：
  - 简单命题：不含逻辑联结词
  - 复合命题：含有逻辑联结词。
### 逻辑联结词

| 联结词 | 符号 | 说明 |
|--------|------|------|
| 且（and） | ∧ | 两者都为真时为真 |
| 或（or） | ∨ | 至少一个为真时为真 |
| 非（not） | ¬ | 真变假，假变真 |
| 如果...那么... | → | 前件真后件假时为假 |
| 当且仅当 | ↔ | 两者相同时为真 |

### 推理
- **定义**：从已知命题推出新命题的过程
- **结构**：前提→结论

## 演绎推理

### 三段论
- **结构**：
  - 大前提：一般性原则
  - 小前提：特殊情况
  - 结论：根据前提得出的判断

### 示例

```
大前提：所有人都会死
小前提：苏格拉底是人
结论：苏格拉底会死
```

### 有效推理形式
| 形式 | 结构 | 示例 |
|------|------|------|
| 肯定前件 | 如果P则Q；P；所以Q | 如果下雨则地湿；下雨了；所以地湿 |
| 否定后件 | 如果P则Q；非Q；所以非P | 如果下雨则地湿；地不湿；所以没下雨 |

### 常见谬误
| 谬误 | 说明 | 示例 |
|------|------|------|
| 肯定后件 | 如果P则Q；Q；所以P | 如果下雨则地湿；地湿了；所以下雨了（可能是洒水车） |
| 否定前件 | 如果P则Q；非P；所以非Q | 如果下雨则地湿；没下雨；所以地不湿（可能洒水） |

## 归纳推理

### 定义
从个别事实推出一般性结论的推理。
### 类型
- **完全归纳**：考察全部对象
- **不完全归纳**：只考察部分对象

### 示例

```
观察到的天鹅1只是白色，天鹅2只是白色，天鹅3只是白色，...
第100只是白色。结论：所有天鹅都是白色的（可能错误）
```

### 归纳推理的局限性
- 结论具有或然性。
- 样本可能不具有代表性。
- 可能出现反例

## 类比推理

### 定义
根据两个对象在某些属性上相同或相似，推出它们在其他属性上也相同或相似的推理。
### 示例

```
地球有大气层、水、适宜的温度，存在生命
火星有大气层、水（冰）、适宜的温度
结论：火星可能存在生命
```

### 类比推理的局限性
- 相似属性与推出属性之间需要有本质联系
- 相似属性越多，结论越可靠。
## 常见逻辑谬误

### 谬误类型
| 类型 | 说明 | 示例 |
|------|------|------|
| 人身攻击 | 攻击提出论证的人 | 他自己都做不到，凭什么教别人 |
| 稻草人 | 歪曲对方观点 | 你说要多读书，难道不工作了吗 |
| 滑坡 | 极端推论 | 允许迟到一次，以后就会天天迟到 |
| 诉诸权威 | 仅因为权威就相信 | 明星说有效，所以有效 |
| 虚假两难 | 人为制造非此即彼 | 不支持就是反对 |
| 循环论证 | 用结论证明前提 | 这本书好因为评价高，评价高因为书好 |
| 以偏概全 | 从个别推出一般 | 我认识的两个人都这样，所以所有人都这样 |

## 逻辑推理练习题
### 练习1

```
所有A都是B
所有B都是C
所以，所有A都是C
这个推理有效吗？
```

### 练习2

```
如果今天下雨，那么地面会湿
今天地面湿了
所以，今天下雨了
这个推理有效吗？
```

### 练习3

```
张三学习努力
李四学习努力
王五学习努力
他们都是学生
所以，所有学生都学习努力
这个推理有效吗？
```
