## 一、物质科学实验

### 实验1：水的三态变化

**实验目的**：观察水在不同温度下的状态变化。

**实验材料**：冰块、烧杯、酒精灯、温度计、石棉网、三脚架。

**实验步骤**：
1. 将冰块放入烧杯中，测量温度并记录。
2. 用酒精灯加热烧杯，每隔1分钟记录一次温度。
3. 观察冰融化成水的过程，记录温度变化。
4. 继续加热至水沸腾，观察水变成水蒸气的过程。
5. 在烧杯口上方放置一块冷玻璃片，观察水蒸气凝结成小水滴。

**实验现象**：
| 阶段 | 温度变化 | 状态变化 |
|------|----------|----------|
| 冰→水 | 0℃保持不变 | 固态→液态（熔化） |
| 水→水蒸气 | 100℃保持不变 | 液态→气态（沸腾） |
| 水蒸气→水 | 遇冷凝结 | 气态→液态（液化） |

**实验结论**：水可以在固态、液态、气态三种状态之间变化，温度是状态变化的关键因素。

### 实验2：浮力实验

**实验目的**：探究影响物体浮沉的因素。

**实验材料**：水槽、水、橡皮泥、鸡蛋、盐、木块、石头、乒乓球。

**实验步骤**：
1. 将木块、石头、乒乓球分别放入水中，观察浮沉情况。
2. 将橡皮泥捏成球状放入水中，观察沉浮。
3. 将同一块橡皮泥捏成船形，再次放入水中，观察变化。
4. 将鸡蛋放入清水中，观察沉浮。
5. 向水中逐渐加盐并搅拌，观察鸡蛋位置变化。

**实验现象**：
| 物体 | 清水中的状态 | 说明 |
|------|-------------|------|
| 木块 | 漂浮 | 密度小于水 |
| 石头 | 下沉 | 密度大于水 |
| 乒乓球 | 漂浮 | 密度小于水 |
| 橡皮泥球 | 下沉 | 密度大于水 |
| 橡皮泥船 | 漂浮 | 排开水的体积增大 |
| 鸡蛋（清水） | 下沉 | 密度略大于水 |
| 鸡蛋（盐水） | 漂浮 | 盐水密度增大 |

**实验结论**：物体在液体中的浮沉取决于物体密度与液体密度的关系，以及物体排开液体的体积。

### 实验3：磁铁实验

**实验目的**：认识磁铁的性质和磁极。

**实验材料**：条形磁铁、蹄形磁铁、回形针、铁屑、小铁钉、塑料片、纸片、铜片。

**实验步骤**：
1. 用磁铁靠近回形针、小铁钉、塑料片、纸片、铜片，观察哪些能被吸引。
2. 将铁屑均匀撒在纸板上，将条形磁铁放在纸板下方，观察铁屑的分布图案。
3. 用一块磁铁的N极靠近另一块磁铁的N极，观察现象。
4. 用一块磁铁的N极靠近另一块磁铁的S极，观察现象。

**实验现象**：
- 磁铁能吸引铁制物品，不能吸引塑料、纸、铜。
- 铁屑在磁铁周围形成有规律的分布图案，显示磁场的存在。
- 同名磁极（N-N或S-S）相互排斥，异名磁极（N-S）相互吸引。

**实验结论**：磁铁有N极和S极两个磁极，同名磁极相斥，异名磁极相吸。磁铁能吸引铁、钴、镍等磁性材料。

### 实验4：简单电路

**实验目的**：认识电路的基本组成和连接方式。

**实验材料**：电池（1.5V）、小灯泡、导线、开关、电池盒、灯座。

**实验步骤**：
1. 用导线将电池、小灯泡、开关连接成简单电路，观察灯泡是否发光。
2. 断开开关，观察灯泡是否熄灭。
3. 将两个小灯泡串联在电路中，观察亮度变化。
4. 将两个小灯泡并联在电路中，观察亮度变化。

**实验记录**：
| 电路连接 | 灯泡数量 | 灯泡亮度 |
|----------|----------|----------|
| 简单电路 | 1个 | 正常发光 |
| 串联电路 | 2个 | 比正常暗 |
| 并联电路 | 2个 | 正常发光 |

**实验结论**：电路由电源、导线、用电器、开关四部分组成。串联电路中灯泡变暗，并联电路中灯泡正常发光。

### 实验5：光的传播实验

**实验目的**：验证光沿直线传播。

**实验材料**：手电筒、三张带小孔的纸板、白色屏幕、尺子。

**实验步骤**：
1. 将三张带小孔的纸板排成一条直线，用手电筒照射，观察光线是否能穿过所有小孔到达屏幕。
2. 将中间一张纸板稍微移动，使三个小孔不再在一条直线上，观察屏幕上的光点变化。

**实验现象**：当三个小孔在一条直线上时，光线能穿过所有小孔到达屏幕。移动一张纸板后，光线被挡住。

**实验结论**：光在同种均匀介质中沿直线传播。

## 二、生命科学实验

### 实验6：种子发芽实验

**实验目的**：探究种子发芽所需的条件。

**实验材料**：绿豆种子、透明杯子、纸巾、水、冰箱、黑纸盒。

**实验步骤**：
1. 取4个杯子，底部放入湿纸巾，各放入5粒绿豆种子。
2. 1号杯：放在温暖处，保持纸巾湿润（充足水分、适宜温度、有空气）。
3. 2号杯：放在温暖处，保持纸巾干燥（无水条件）。
4. 3号杯：放在冰箱中，保持纸巾湿润（低温条件）。
5. 4号杯：放入黑纸盒中，保持纸巾湿润（无光条件）。
6. 每天观察记录种子的变化，持续7天。

**实验记录**：
| 天数 | 1号杯 | 2号杯 | 3号杯 | 4号杯 |
|------|-------|-------|-------|-------|
| 第1天 | 吸水膨胀 | 无变化 | 无变化 | 吸水膨胀 |
| 第3天 | 胚根长出 | 无变化 | 无变化 | 胚根长出 |
| 第5天 | 胚芽长出 | 无变化 | 无变化 | 胚芽长出 |
| 第7天 | 长出绿叶 | 无变化 | 无变化 | 幼苗发黄 |

**实验结论**：种子发芽需要水分、适宜的温度和空气。光不是种子发芽的必要条件，但幼苗生长需要光照。

### 实验7：植物的向光性

**实验目的**：观察植物生长对光照方向的反应。

**实验材料**：绿豆幼苗、纸盒、剪刀、花盆、土壤。

**实验步骤**：
1. 在花盆中种植几株绿豆幼苗，待它们长到5厘米高。
2. 用纸盒做一个遮光罩，在一侧开一个小窗。
3. 将遮光罩罩在花盆上，小窗朝向有光的一侧。
4. 每天观察幼苗的生长方向并记录。

**实验现象**：3天后，绿豆幼苗明显朝向小窗（有光的方向）生长。

**实验结论**：植物具有向光性，会朝着光源的方向生长，这有利于植物获取更多的光能进行光合作用。

### 实验8：蒸腾作用观察

**实验目的**：观察植物的蒸腾作用。

**实验材料**：带叶的植物枝条、透明塑料袋、细线、水、量筒。

**实验步骤**：
1. 取一根带叶的植物枝条，插入装有水的量筒中，记录水面高度。
2. 用透明塑料袋套住枝条，用细线扎紧袋口。
3. 将装置放在阳光下，观察塑料袋内壁的变化。
4. 24小时后，再次记录量筒中的水面高度。

**实验现象**：塑料袋内壁出现小水珠，量筒中的水面下降。

**实验结论**：植物通过叶片的气孔蒸发水分，这就是蒸腾作用。蒸腾作用是植物吸收和运输水分的重要动力。

## 三、地球与宇宙实验

### 实验9：昼夜交替模拟

**实验目的**：理解地球自转与昼夜交替的关系。

**实验材料**：地球仪、手电筒、橡皮泥小人。

**实验步骤**：
1. 将橡皮泥小人粘在地球仪上表示居住的位置。
2. 在黑暗的房间里，用手电筒照射地球仪模拟太阳光。
3. 保持手电筒不动，缓慢转动地球仪（自西向东）。
4. 观察橡皮泥小人所处位置的光照变化。

**实验现象**：当地球仪转动时，橡皮泥小人的位置交替经历被照亮（白天）和背光（黑夜）。

**实验结论**：地球自转是产生昼夜交替的原因。地球自转一周约24小时，因此昼夜交替的周期也是一天。

### 实验10：月相变化模拟

**实验目的**：理解月相变化的原理。

**实验材料**：台灯（模拟太阳）、乒乓球（模拟月球）、自己（模拟地球）。

**实验步骤**：
1. 将台灯放在房间的一侧作为太阳。
2. 手持乒乓球作为月球，站在离台灯约2米处。
3. 慢慢原地旋转，同时观察乒乓球被照亮的部分。
4. 记录不同位置时乒乓球被照亮的形状。

**实验现象**：在不同位置观察时，乒乓球被照亮的部分呈现不同的形状，从新月、上弦月、满月到下弦月的各种月相。

**实验结论**：月相变化是月球绕地球公转时，日、地、月三者的相对位置发生变化而引起的。

### 实验11：土壤成分分析

**实验目的**：了解土壤的主要成分。

**实验材料**：土壤样本、烧杯、水、勺子、白纸、放大镜。

**实验步骤**：
1. 用放大镜观察干燥土壤，记录看到的物质。
2. 将土壤放在白纸上轻轻碾压，观察是否有细小的沙粒、黏土颗粒。
3. 将土壤放入烧杯中加水搅拌，静置后观察分层现象。
4. 将少量土壤放在勺子上用酒精灯加热，闻是否有焦味。

**实验现象**：
- 土壤中有大小不同的颗粒、植物的根和枯叶。
- 土壤在水中分层：底层是沙粒，中层是粉粒，上层是黏土。
- 加热时有机物燃烧产生焦味。

**实验结论**：土壤由矿物质颗粒、有机物、水分、空气等成分组成。不同成分的比例影响土壤的肥力和性质。

## 四、工程实践

### 实验12：纸桥承重

**实验目的**：探究不同结构的桥面承重能力。

**实验材料**：A4纸若干、胶带、书本（做桥墩）、硬币（做重物）、尺子。

**实验步骤**：
1. 用两本书间隔15厘米做桥墩。
2. 用一张纸直接平铺在桥墩上做桥面，测试能放多少枚硬币。
3. 将纸折成瓦楞形（波纹折叠），再次测试承重。
4. 将纸卷成圆筒形，再次测试承重。
5. 设计自己的桥面结构，尝试获得最大的承重能力。

**实验记录**：
| 桥面结构 | 承重（硬币数） | 特点分析 |
|----------|---------------|----------|
| 平面桥 | 2-3枚 | 易弯曲，承重弱 |
| 瓦楞形 | 15-20枚 | 增加结构强度 |
| 圆筒形 | 25-30枚 | 利用拱形原理 |
| 自主设计 | - | 多种结构组合 |

**实验结论**：通过改变纸张的形状可以大大提高承重能力。瓦楞形和圆筒形利用了结构力学的原理，将力分散到整个结构上。

### 实验13：自制指南针

**实验目的**：利用磁化原理制作简易指南针。

**实验材料**：缝衣针、磁铁、软木塞片、水盆、水、小刀。

**实验步骤**：
1. 用磁铁沿同一方向反复摩擦缝衣针50次以上，使针磁化。
2. 将软木塞片切成小圆片，浮在水面上。
3. 将磁化后的缝衣针轻轻地放在软木塞片上。
4. 观察针尖指示的方向，与真正的指南针对比。
5. 改变水盆的方向，观察针是否随之转动。

**实验现象**：磁化后的缝衣针总会指向南北方向，即使改变水盆的方向，针的指向也不变。

**实验结论**：地球本身是一个大磁体，磁化后的针受到地球磁场的作用，总是指向南北方向。这就是指南针的原理。

## 安全注意事项

1. 使用酒精灯时要有成人监督，远离易燃物。
2. 使用剪刀、小刀等锐器时注意安全。
3. 实验后及时洗手，尤其是接触土壤、化学物品后。
4. 不随意品尝实验材料。
5. 使用电器实验时检查电池和电线是否完好。
6. 实验结束后整理实验器材，保持桌面整洁。
