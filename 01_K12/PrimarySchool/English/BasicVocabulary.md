# 基础词汇

## 概述

词汇是英语学习的基础。本文档介绍核心词汇分类、词汇记忆方法、常用短语以及简单对话模板。

## 词汇记忆方法

### 1. 图片联想法

- **方法**：将单词与图片联系起来。
- **示例**：
  - apple → 想象一个红红的苹果
  - dog → 想象一只可爱的小狗

### 2. 分类记忆法

- **方法**：将同类词汇放在一起记。
- **示例**：
  - 水果类：apple, banana, orange, grape
  - 颜色类：red, blue, yellow, green

### 3. 词根词缀法

- **方法**：学习常见词根词缀
- **示例**：
  - un-（不）：unhappy, unable
  - -ly（副词后缀）：quickly, slowly

### 4. 联想记忆法

- **方法**：将新词与已知词联系
- **示例**：
  - cake（蛋糕）→ bake（烘焙）
  - cat（猫）→ bat（蝙蝠）

## 简单对话模板

### 自我介绍

```
A: Hello! What's your name?
B: My name is [name]. What's your name?
A: My name is [name]. Nice to meet you.
B: Nice to meet you too.
```

### 询问年龄

```
A: How old are you?
B: I'm [age] years old. How old are you?
A: I'm [age] years old.
```

### 询问爱好

```
A: What do you like?
B: I like [hobby]. What do you like?
A: I like [hobby].
```

### 询问颜色

```
A: What color is it?
B: It's [color].
```

### 询问位置

```
A: Where is my [object]?
B: It's on/in/under the [furniture].
```

### 购物对话

```
A: Can I help you?
B: I want a [object], please.
A: Here you are.
B: Thank you.
A: You're welcome.
```

## 练习建议

1. **每天学习5个新词**：坚持积累。
2. **使用单词卡片**：方便复习。
3. **大声朗读**：提高发音。
4. **造句练习**：应用词汇。
5. **阅读简单英语**：巩固词汇。
