# Primary English — Oral Communication

> Focus on daily conversation, situational dialogues, and communicative competence for primary school students.

---

## 一、Daily Greetings（日常问候）

### 1.1 Basic Greetings

| English | Chinese |
|---------|---------|
| Good morning! | 早上好！ |
| Good afternoon! | 下午好！ |
| Good evening! | 晚上好！ |
| Hello! / Hi! | 你好！ |
| How are you? | 你好吗？ |
| I'm fine, thank you. | 我很好，谢谢你。 |
| Nice to meet you! | 很高兴认识你！ |
| Goodbye! / Bye! | 再见！ |
| See you tomorrow! | 明天见！ |

### 1.2 Dialogue: Morning Greeting

```
Alice: Good morning, Bob!
Bob: Good morning, Alice! How are you?
Alice: I'm fine, thank you. How about you?
Bob: I'm great, thanks!
Alice: See you later!
Bob: See you!
```

## 二、Self-Introduction（自我介绍）

### 2.1 Key Expressions

```
My name is ...
I am ... years old.
I am a student.
I like ...
My favourite subject is ...
I live in ...
```

### 2.2 Dialogue: First Day at School

```
Teacher: Hello! What's your name?
Student: Hello! My name is Li Ming.
Teacher: How old are you, Li Ming?
Student: I'm eight years old.
Teacher: Nice to meet you!
Student: Nice to meet you too!
```

## 三、Classroom English（课堂用语）

### 3.1 For Students

| English | Chinese |
|---------|---------|
| May I come in? | 我可以进来吗？ |
| I'm sorry I'm late. | 对不起，我迟到了。 |
| Excuse me. | 打扰一下。 |
| May I ask a question? | 我可以问个问题吗？ |
| I don't understand. | 我不明白。 |
| Could you repeat that? | 你能再说一遍吗？ |
| Can I go to the bathroom? | 我可以去洗手间吗？ |
| I have a question. | 我有一个问题。 |

### 3.2 For Teachers

| English | Chinese |
|---------|---------|
| Open your books. | 打开书。 |
| Close your books. | 合上书。 |
| Listen carefully. | 仔细听。 |
| Repeat after me. | 跟我读。 |
| Raise your hand. | 举手。 |
| Work in pairs. | 两人一组练习。 |
| Time's up. | 时间到。 |
| Good job! / Well done! | 做得好！ |

### 3.3 Dialogue: In Class

```
Teacher: Good morning, class!
Class: Good morning, Miss Wang!
Teacher: Please open your books to page 10.
Amy: Excuse me, which page?
Teacher: Page 10. Now repeat after me.
Bob: I'm sorry, I don't understand.
Teacher: No problem. Let me explain again.
```

## 四、Talking About Family（谈论家庭）

### 4.1 Key Vocabulary

| English | Chinese |
|---------|---------|
| father / dad | 爸爸 |
| mother / mom | 妈妈 |
| brother | 兄弟 |
| sister | 姐妹 |
| grandfather / grandpa | 爷爷/外公 |
| grandmother / grandma | 奶奶/外婆 |
| uncle | 叔叔/舅舅 |
| aunt | 阿姨/姑姑 |
| cousin | 堂/表兄弟姐妹 |

### 4.2 Useful Expressions

```
How many people are there in your family?
There are ... people in my family.
I have a (brother / sister).
I love my family.
My father is a (doctor / teacher / driver).
```

### 4.3 Dialogue: My Family

```
Tom: How many people are in your family?
Lily: There are four people: my dad, my mom, my brother and me.
Tom: What does your father do?
Lily: He is a doctor. What about your family?
Tom: I have a big family! I have two sisters and a brother.
```

## 五、Talking About Hobbies（谈论爱好）

### 5.1 Key Expressions

```
What do you like?
I like (reading / drawing / swimming / playing football).
Do you like ...?
Yes, I do. / No, I don't.
What's your favourite ...?
My favourite (sport / food / colour) is ...
```

### 5.2 Dialogue: After School

```
Mike: What do you like to do after school?
Sarah: I like playing basketball. How about you?
Mike: I like reading books.
Sarah: That's cool! What's your favourite book?
Mike: My favourite book is Harry Potter.
Sarah: I like that book too!
```

## 六、Shopping（购物）

### 6.1 Useful Expressions

| Shopkeeper | Customer |
|-----------|----------|
| Can I help you? | Yes, please. I want ... |
| What colour do you want? | How much is it? |
| Here you are. | That's too expensive. |
| Anything else? | No, thank you. |
| That's ... yuan. | Here's the money. |

### 6.2 Dialogue: Buying a Pen

```
Shopkeeper: Can I help you?
Customer: Yes, please. I want to buy a pen.
Shopkeeper: What colour do you like?
Customer: Blue, please.
Shopkeeper: Here you are. It's 5 yuan.
Customer: That's fine. Here's the money.
Shopkeeper: Thank you. Goodbye!
Customer: Bye!
```

## 七、Asking for Directions（问路）

### 7.1 Key Expressions

| Asking | Giving Directions |
|--------|------------------|
| Excuse me, where is ...? | Go straight. |
| How can I get to ...? | Turn left / right. |
| Is it far from here? | It's next to ... |
| Which bus should I take? | It's across from ... |
| Thank you! | You're welcome! |

### 7.2 Dialogue: Finding the Library

```
Tourist: Excuse me, where is the library?
Police: Go straight, then turn left at the traffic lights.
Tourist: Is it far?
Police: No, it's about 5 minutes' walk.
Tourist: Thank you very much!
Police: You're welcome.
```

## Communication Strategy Tips

| Strategy | Example |
|----------|---------|
| Ask for repetition | "Pardon?" / "Could you say that again?" |
| Clarify meaning | "What does ... mean?" |
| Express uncertainty | "I'm not sure." |
| Agree | "I think so." / "That's right." |
| Disagree politely | "I don't think so." / "Maybe not." |
| Change topic | "By the way ..." |
| End a conversation | "Nice talking to you!" / "I have to go now." |

## Practice Activities

1. **Role Play** — Practice dialogues with classmates
2. **Daily Report** — Give a 1-minute speech about your day
3. **Picture Description** — Describe what you see in a picture
4. **Story Telling** — Retell a simple story in your own words
5. **Interview** — Ask your partner questions and report answers
