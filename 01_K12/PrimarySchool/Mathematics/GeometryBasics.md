# 几何基础

## 概述

几何是研究图形形状、大小和位置关系的数学分支。本文档介绍平面图形、周长与面积计算、立体图形以及体积与表面积计算。
## 平面图形

### 长方形- **特征**：对边相等，四个角都是直。
- **要素**：长（a）、宽（b。
- **对角线**：相等且互相平分

### 正方形- **特征**：四边相等，四个角都是直。
- **要素**：边长（a。
- **性质**：特殊的长方。
### 三角形| 分类 | 特点 |
|------|------|
| 按角 | 锐角三角形、直角三角形、钝角三角形 |
| 按边 | 等边三角形、等腰三角形、不等边三角形|

### 梯形
- **特征**：只有一组对边平。
- **要素**：上底（a）、下底（b）、高（h。
- **分类**：等腰梯形、直角梯。
### 平行四边。
- **特征**：两组对边分别平。
- **要素**：底（a）、高（h。
- **性质**：对边相等，对角相等

## 周长计算公式

| 图形 | 周长公式 | 说明 |
|------|----------|------|
| 长方形| C = 2(a+b) | a为长，b为宽 |
| 正方形| C = 4a | a为边 |
| 三角形| C = a+b+c | 三边之和 |
| 平行四边 | C = 2(a+b) | a、b为邻 |
| 梯形 | C = a+b+c+d | 四边之和 |
|  | C = 2πr = πd | r为半径，d为直 |

## 面积计算公式

### 平面图形面积公式。
| 图形 | 面积公式 | 说明 |
|------|----------|------|
| 长方形| S = ab | a为长，b为宽 |
| 正方形| S = a² | a为边 |
| 三角形| S = ½ah | a为底，h为高 |
| 平行四边 | S = ah | a为底，h为高 |
| 梯形 | S = ½(a+b)h | a为上底，b为下底，h为高 |
|  | S = πr² | r为半 |

## 立体图形

### 正方形- **特征**：六个面都是正方形，六个面都相等
- **要素**：棱长（a。
- **顶点**：8个。
- **棱**：12条。
### 长方形- **特征**：六个面都是长方形（可能有两个相对的面是正方形）
- **要素**：长（a）、宽（b）、高（c。
- **顶点**：8个。
- **棱**：12条。
### 圆柱
- **特征**：上下两个底面是相等的圆，侧面是曲面
- **要素**：底面半径（r）、高（h。
- **展开**：侧面展开是长方形

### 圆锥
- **特征**：底面是圆，侧面是曲面，有一个顶。
- **要素**：底面半径（r）、高（h。
## 体积与表面积计算

### 体积公式。
| 立体图形 | 体积公式 | 说明 |
|----------|----------|------|
| 正方形| V = a³ | a为棱 |
| 长方形| V = abc | a为长，b为宽，c为高 |
| 圆柱 | V = πr²h | r为底面半径，h为高 |
| 圆锥 | V = ⅓πr²h | r为底面半径，h为高 |

### 表面积公式表

## 练习示例

### 平面图形计算
```
长方形：长8cm，宽5cm
周长 = 2×(8+5) = 26cm
面积 = 8×5 = 40cm²

圆：半径3cm
周长 = 2×3.14×3 = 18.84cm
面积 = 3.14×3² = 28.26cm²
```

### 立体图形计算
```
长方体：长5cm，宽4cm，高3cm
体积 = 5×4×3 = 60cm³
表面积 = 2×(5×4+4×3+5×3) = 94cm²

圆柱：底面半径2cm，高5cm
体积 = 3.14×2²×5 = 62.8cm³
表面积 = 2×3.14×2² + 2×3.14×2×5 = 87.92cm²
```
