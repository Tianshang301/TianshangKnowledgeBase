# 算术基础

## 概述

算术是数学的基础，是小学数学的核心内容。本文档介绍数的认识、四则运算、运算律、估算方法、竖式计算规则以及常见运算技巧。
## 数的认识

### 整数
| 分类 | 范围 | 示例 |
|------|------|------|
| 自然 | 0,1,2,3... | 0, 5, 100 |
| 正整 | 1,2,3... | 1, 5, 100 |
| 负整 | -1,-2,-3... | -1, -5, -100 |

### 小数
| 分类 | 说明 | 示例 |
|------|------|------|
| 一位小 | 小数点后一 | 0.5, 3.2 |
| 两位小数 | 小数点后两位 | 0.25, 3.14 |
| 循环小数 | 无限循环 | 0.333... |

### 分数
| 分类 | 说明 | 示例 |
|------|------|------|
| 真分 | 分子<分母 | 1/2, 3/4 |
| 假分 | 分子≥分 | 5/3, 7/4 |
| 带分 | 整数+真分 | 1又1/2 |

### 数的大小比较
- **整数**：位数多的大
- **小数**：先比整数部分，再比小数部分
- **分数**：通分后比较分。
## 四则运算

### 加法
- **定义**：把两个数合并成一个数
- **法则**：相同数位对齐，从个位加。
- **示例**：
  ```
    123
  + 456
  -----
    579
  ```

### 减法
- **定义**：已知两个加数的和与其中一个加数，求另一个加。
- **法则**：相同数位对齐，从个位减。
- **示例**：
  ```
    579
  - 123
  -----
    456
  ```

### 乘法
- **定义**：求几个相同加数的和的简便运。
- **法则**：
  - 用第二个因数每一位去乘第一个因数
  - 哪一位上乘得的积满几十，就向前一位进几。
- **示例**：
  ```
    123
  ×  45
  -----
    615  (123×5)
   492   (123×4)
  -----
   5535
  ```

### 除法
- **定义**：已知两个因数的积与其中一个因数，求另一个因。
- **法则**：
  - 从被除数的高位除起
  - 除到被除数的哪一位，商就写在那一位上
  - 每次除后余数要比除数小。
- **示例**：
  ```
    125
  4)500
    4
  ---
    10
     8
    ---
     20
     20
    ---
      0
  ```

## 运算。
### 交换。
- **加法交换律**：a + b = b + a
- **乘法交换律**：a × b = b × a

### 结合。
- **加法结合律**：(a + b) + c = a + (b + c)
- **乘法结合律**：(a × b) × c = a × (b × c)

### 分配。
- **乘法分配律**：a × (b + c) = a × b + a × c

### 运算律应用
```
25 × 17 × 4
= 25 × 4 × 17  (交换。
= 100 × 17
= 1700

99 × 26
= (100 - 1) × 26  (分配。
= 100 × 26 - 1 × 26
= 2600 - 26
= 2574
```

## 估算方法

### 四舍五入。
- **规则**：看要保留位数的下一位，≥5则进1，<5舍去
- **示例**：234 ≈ 230（保留到十位）。
### 进一。
- **规则**：无论下一位是多少，都。
- **示例**：装油，每瓶5kg，20kg油需要4瓶。
### 去尾。
- **规则**：无论下一位是多少，都舍去
- **示例**：做衣服，每件用2m布，5m布能做2件。
### 估算技巧
1. **加减法**：估成整十、整百数
2. **乘法**：分别估算两个因数
3. **除法**：把除数估成整十、整百数

## 竖式计算规则

### 整数加减法
1. 相同数位对齐
2. 从个位算起
3. 哪一位相加满十，向前一位进1
4. 哪一位不够减，从前一位退一当十

### 整数乘法
1. 用第二个因数每一位去乘第一个因数
2. 哪一位上乘得的积满几十，就向前一位进几
3. 最后把几次乘得的积相加

### 小数加减法
1. 小数点对齐
2. 按整数加减法计算
3. 得数的小数点与横线上的小数点对齐

### 小数乘法
1. 先按整数乘法计算
2. 再看因数中共有几位小数，就从积的右边起数出几位，点上小数。
## 常见运算技。
### 凑十法
```
8 + 7 = 8 + (2 + 5) = (8 + 2) + 5 = 10 + 5 = 15
```

### 破十法
```
15 - 8 = 15 - (5 + 3) = (15 - 5) - 3 = 10 - 3 = 7
```

### 借位法
```
23 - 8 = 23 - (3 + 5) = (23 - 3) - 5 = 20 - 5 = 15
```

### 凑整法
```
99 + 6 = 100 + 6 - 1 = 105
```

### 巧算示例
```
25 × 12 = 25 × 4 × 3 = 100 × 3 = 300
```
