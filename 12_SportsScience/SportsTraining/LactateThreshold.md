# LactateThreshold

乳酸阈（Lactate Threshold, LT）是指运动强度持续增加时，血液乳酸浓度开始以非线性的方式急剧上升的临界点。
该现象反映了有氧氧化供能向无氧糖酵解供能转换的转折区。
常将乳酸阈分为第一乳酸阈（LT1，~2 mmol/L）和第二乳酸阈/血乳酸积累起点（OBLA，~4 mmol/L）。
LT 是评估耐力运动员有氧工作能力及预测耐力表现的核心指标之一。
系统训练可使 LT 对应的运动强度右移，即在更高的运动负荷下保持较低的血乳酸水平。
乳酸阈可通过连续递增负荷测试（如跑台或功率自行车）结合指尖血采样测定。
通气阈（VT）可作为非侵入的乳酸阈替代指标，通过呼出气体分析实现。
LT 训练常用的强度区间约为 LT1 以上至 OBLA 之间的强度，通常持续 20–40分钟。
"阈值训练"（Tempo Run / Sweet Spot Training）是最经典的耐力提升区段之一。
LT 值受肌纤维类型分布、毛细血管密度、线粒体酶活性和缓冲能力等多因素影响。
