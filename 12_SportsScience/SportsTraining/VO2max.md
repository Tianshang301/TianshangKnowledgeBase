# VO2max

## 概述

VO₂max（最大摄氧量）指人体在极限运动中每分钟每公斤体重能消耗的最大氧气量，单位 ml·kg⁻¹·min⁻¹。它是衡量心肺耐力的金标准，也是预测寿命的重要指标。

## 生理意义

VO₂max 反映了以下生理系统的综合效率：
- **肺通气**: 吸入氧气的能力
- **心脏泵血**: 心输出量（每搏量 × 心率）
- **血液携氧**: 血红蛋白浓度和血容量
- **肌肉利用**: 线粒体密度和氧化酶活性

## 影响因素

| 因素 | 影响 |
|------|------|
| 遗传 | 约50%由基因决定 |
| 年龄 | 30岁后每十年下降5–10% |
| 性别 | 女性通常比男性低15–30% |
| 训练状态 | 规律训练可提升10–30% |
| 海拔 | 高海拔降低可用氧量 |

## 训练提升方法

### 1. 高强度间歇训练（HIIT）
在≥90% VO₂max 强度下进行短间歇训练，是提升 VO₂max 最有效的方法。

### 2. 乳酸阈训练
在接近乳酸阈的强度持续训练（20–40分钟），提升有氧基础。

### 3. 基础耐力训练
长时间低强度有氧运动（Zone 2），增加线粒体和毛细血管密度。

## 正常参考值

| 人群 | VO₂max (ml·kg⁻¹·min⁻¹) |
|------|------------------------|
| 未训练男性 | 35–45 |
| 未训练女性 | 30–40 |
| 耐力运动员男性 | 55–70+ |
| 耐力运动员女性 | 45–60+ |
| 优秀运动员 | 80+（男性），70+（女性） |
