# Overtraining

## 概述

过度训练综合征（OTS）是由于训练负荷长期超过身体恢复能力而导致的一种适应不良状态。表现为持续运动表现下降、疲劳累积和情绪障碍，恢复期可达数周至数月。

## 临床表现

| 系统 | 症状 |
|------|------|
| 生理 | 持续疲劳、静息心率升高、睡眠障碍、食欲下降 |
| 运动表现 | 成绩停滞或下降、恢复变慢、协调性降低 |
| 免疫 | 感染频率增加、伤口愈合缓慢 |
| 心理 | 情绪低落、动力下降、易怒、焦虑 |
| 内分泌 | 皮质醇异常、性激素水平改变 |

## 分期

- **功能性过度训练（Stage 1）**: 轻度症状，休息后快速恢复
- **交感型（Stage 2）**: 交感神经过度激活，静息心率升高
- **副交感型（Stage 3）**: 副交感神经主导，持续疲劳和抑制状态

## 处理与恢复

1. **休息**: 完全停训1周或减少50–80%训练量
2. **睡眠**: 保证7–9小时，必要时增加
3. **营养**: 充足蛋白质和总能量摄入
4. **压力管理**: 减少非训练压力源
5. **渐进恢复**: 恢复训练后从50%强度开始

## 预防

- 合理周期化训练安排
- 定期监控训练负荷（sRPE、心率变异性）
- 保证恢复日和减量周
- 倾听身体信号
