# DOMS

## 概述

延迟性肌肉酸痛（Delayed Onset Muscle Soreness, DOMS）是指运动后24–72小时出现的肌肉疼痛和僵硬感。常见于新开始运动、增加训练强度或进行大量离心运动后。

## 病理机制

DOMS 的主要机制是**运动诱导的肌肉微损伤**：
- 离心收缩（肌肉在张力下拉长）造成肌纤维 Z 线微撕裂
- 炎症反应释放前列腺素和细胞因子
- 组织液渗出引起水肿和疼痛
- 疼痛于24–72小时达到高峰，5–7天逐渐消退

乳酸堆积并非 DOMS 的主要原因——乳酸在运动后30–60分钟即恢复正常水平。

## 症状

- 弥漫性肌肉酸痛和僵硬
- 触痛（机械性痛觉过敏）
- 关节活动度降低
- 肌力暂时性下降
- 轻微肿胀

## 处理与缓解

| 方法 | 效果 | 说明 |
|------|------|------|
| 轻度运动（主动恢复） | 较高 | 促进血液循环，暂时缓解疼痛 |
| 按摩/泡沫轴 | 中等 | 改善血流，降低肌张力 |
| 冷疗（冰敷） | 有限 | 减轻炎症和肿胀 |
| 热疗 | 中等 | 促进血流，缓解僵硬 |
| NSAIDs（布洛芬等） | 中等 | 消炎镇痛（需遵医嘱） |
| 补充水分 | 辅助 | 维持代谢正常 |
| 镁补充剂 | 有限证据 | 可能辅助放松肌肉 |

## 预防

- 循序渐进增加训练负荷（10%原则）
- 充分热身和整理活动
- 离心运动逐步引入
- 保证充足的恢复时间
