# FirstAidInSports

## 概述

运动急救是体育活动中对突发伤病进行现场评估和紧急处置的措施，目标是救命、防止伤情加重和促进后续治疗。运动急救人员需掌握基本的生命支持技能和常见运动损伤的现场处理方法。

## 常见急症处理

### 软组织损伤 — POLICE 原则
- **保护**（Protect）：护具或制动防止二次损伤
- **适量负荷**（Optimal Loading）：早期功能性活动促进修复
- **冰敷**（Ice）：每 2-3 小时冰敷 15-20 分钟，急性期 48-72 小时
- **加压包扎**（Compression）：弹性绷带控制肿胀
- **抬高**（Elevation）：患肢高于心脏

### 骨折与脱臼
- 就地制动，用夹板或健肢固定伤处上下关节
- 勿尝试自行复位，脱臼也应由医师处理
- 注意远端血液循环和神经功能

### 脑震荡
- 疑有脑震荡需立即停止运动，不可当天重返赛场
- 使用 SCAT5 评估症状、认知和平衡
- 症状恶化（呕吐、意识下降、瞳孔不等大）须立即送急诊

### 热急症
- **热痉挛**：停止运动，口服补液，轻度拉伸痉挛肌肉
- **热衰竭**：移至阴凉处，平卧抬腿，补液，物理降温
- **热射病**：核心体温＞40℃伴意识障碍，立即冷水浸泡降温，呼叫急救

### 心脏骤停
- 立即启动急救系统
- 高质量胸外按压（100-120 次/分，深度 5-6 cm）
- 尽早使用自动体外除颤器（AED）

## 运动场急救箱推荐配置

| 物品 | 用途 |
|------|------|
| 弹性绷带、三角巾 | 加压包扎、悬吊 |
| 消毒纱布、胶带 | 伤口覆盖 |
| 冰袋（化学或一次性） | 冷敷 |
| 剪刀、镊子、手套 | 辅助操作 |
| 口对面罩或面罩 | CPR 通气 |
