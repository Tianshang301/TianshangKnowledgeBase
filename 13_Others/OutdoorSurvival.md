# 户外生存

户外生存是在远离文明环境的自然条件下维持生命、等待救援或安全返回的技能。掌握基本的户外生存知识不仅是户外爱好者的必备技能，也是应对突发紧急情况的重要保障。

## 生存优先级

在任何生存情境中，遵循正确的优先级至关重要：

```
1. 保持冷静（保持思维清晰）
2. 庇护所（防止体温过低/过高）
3. 水（脱水比饥饿更快致命）
4. 火（取暖、净水、信号）
5. 食物（最后考虑，人体可数周不进食）
6. 求救信号（始终持续发送）
```

## 野外取火（Fire Making）

火是户外生存的核心要素，提供热量、净水、烹饪、照明和心理安慰。

### 取火方法

**现代工具**
- 打火机：最简单可靠的方式，应始终携带
- 防水火柴：密封保存，作为备用
- 火石镁棒：不受潮湿影响，可使用数千次
- 凸透镜：利用阳光聚焦取火

**原始方法**
- **弓钻法（Bow Drill）**：用弓带动木棍在底座上快速旋转摩擦生火
- **手钻法（Hand Drill）**：直接用手掌搓动木棍，技术要求高
- **火犁法（Fire Plough）**：将木棍在底座槽内前后摩擦
- **打火石（Flint and Steel）**：用钢铁击打燧石产生火花

### 燃烧三要素

```
       [热量]
        / \
       /   \
      /     \
  [燃料] ─ [氧气]
```

缺少任一要素火都会熄灭。生火时需要同时满足三个条件。

### 燃料分级

| 级别 | 材料 | 用途 |
|------|------|------|
| 引火物 Tinder | 干草、树皮纤维、棉球、纸 | 最初的火种 |
| 引燃物 Kindling | 细树枝、木片 | 从小火发展为稳定火焰 |
| 主燃料 Fuel | 粗树枝、木块 | 维持长时间燃烧 |

### 生火技巧

1. **准备充足**：先收集足够的三种级别燃料再点火
2. **防风**：在背风处生火，或搭建挡风墙
3. **底层通风**：火堆底部留出空气流通空间
4. **逐步加粗**：从细到粗逐步添加燃料
5. **火堆结构**：锥形、井字形、星形各有适用场景

## 庇护所搭建（Shelter Building）

庇护所保护你免受风雨、极端温度和昆虫的侵害。在寒冷环境中，体温过低可能在数小时内致命。

### 庇护所类型

**自然利用型**
- **倒木庇护所**：利用倒下的大树干作为主梁
- **岩石凹处**：利用天然岩壁和突出岩石
- **大树根部**：倒木的根部形成的天然遮蔽

**人工搭建型**
- **A字型庇护所**：用树枝搭建A字形框架，覆盖树叶或树枝
- **倾斜型庇护所**：一面倾斜的简单结构，搭建快速
- **雪洞（Quinzhee）**：在雪堆中挖出的洞穴，保暖效果好
- **丛林吊床**：在热带环境中隔离地面湿气和虫蛇

### 搭建原则

- **选址优先**：避开洪水区、落石区、枯树下方
- **体积最小化**：越小越容易保暖
- **地面隔热**：铺厚厚的干草、树叶隔绝地面寒气
- **入口背风**：入口朝向背风方向
- **排水考虑**：周围挖排水沟防止积水

### 保暖材料

| 材料 | 保暖效果 | 获取方式 |
|------|---------|---------|
| 干燥树叶 | 优秀 | 收集地表落叶 |
| 干草 | 优秀 | 草地收集 |
| 松针 | 良好 | 松树下收集 |
| 树皮 | 良好 | 剥取干燥树皮 |
| 泥土 | 一般 | 最后的选择 |

## 导航（Navigation）

在没有GPS和手机信号的情况下，依靠自然和传统工具导航是关键生存技能。

### 自然导航

**太阳导航**
- 日出东方，日落西方（北半球偏南，南半球偏北）
- 正午太阳在南方（北半球）或北方（南半球）
- 手表法：时针指向太阳，时针与12点的角平分线指向南方

**星空导航**
- 北极星：北斗七星勺口延长5倍找到北极星（北半球）
- 南十字星：南半球的定位星座
- 月亮：月亮的亮面方向可判断大致方位

**自然标志**
- 树木苔藓通常在背阴面（潮湿一侧）
- 蚂蚁巢穴入口通常朝南（北半球）
- 河流流向可作为方向参考

### 导航工具

- **指南针**：最基本的导航工具，学会使用地图和指南针配合
- **GPS设备**：户外手持GPS，注意电池续航
- **地图**：纸质地形图，不依赖电源
- **计步器**：通过步数估算距离

### 路线规划原则

1. 沿着河流和道路通常能到达人类聚居地
2. 避免在夜间和恶劣天气中移动
3. 做好路标标记，防止迷路
4. 估算时间和体力消耗，避免过度疲劳

## 食物获取（Food Procurement）

在短期生存情境中（3天以内），食物不是首要考虑。但在长期生存中，获取食物变得重要。

### 可食用植物识别

**通用可食用植物（需确认识别）**
- 蒲公英：全株可食用
- 车前草：叶片可食用
- 松树：树皮内层可食用，松针可泡茶
- 橡树：橡果需浸泡去苦味后食用

**危险植物特征（避免食用）**
- 白色或黄色浆果（许多有毒）
- 有苦杏仁味的植物
- 有乳白色汁液的植物
- 三叶植物（可能有毒葛）

### 昆虫食物

昆虫是高蛋白、易获取的野外食物来源：
- **推荐食用**：蟋蟀、蚂蚱、蚂蚁、蚕蛹
- **避免食用**：颜色鲜艳的昆虫、毛虫、蜘蛛
- **烹饪建议**：煮熟或烤熟以杀死寄生虫

### 水源获取

- **流动水源**：溪流、泉水优于静止水源
- **雨水收集**：利用容器或大型叶片收集
- **植物取水**：竹子、藤蔓中可能含有水分
- **露水收集**：清晨用布料沾取草上露水
- **净水方法**：煮沸（至少1分钟）、化学净水片、便携滤水器

## 急救基础（Basic First Aid）

户外环境中的伤害需要即时处理，掌握基本急救知识可以救命。

### 常见户外伤害

| 伤害类型 | 处理方法 |
|---------|---------|
| 伤口出血 | 直接压迫止血，严重时使用止血带 |
| 骨折 | 固定患处，避免移动 |
| 扭伤 | RICE原则：休息、冰敷、加压、抬高 |
| 烫伤 | 冷水冲洗，不要涂抹油脂 |
| 蛇咬 | 保持冷静，标记咬伤位置，尽快就医 |
| 中暑 | 转移到阴凉处，降温补水 |
| 失温 | 换干衣物，热饮，体对体取暖 |

### 急救包必备物品

1. 绷带和纱布
2. 创可贴
3. 消毒湿巾
4. 止痛药
5. 抗过敏药
6. 止泻药
7. 三角巾
8. 医用胶带
9. 安全别针
10. 急救手册

## 实践建议

1. 参加专业的户外生存培训课程
2. 在安全环境中练习取火、搭建庇护所等技能
3. 每次出行携带基本的生存装备（刀、火种、水、急救包）
4. 告知他人你的出行计划和预计返回时间
5. 尊重自然，保持谦逊——安全永远是第一位的
