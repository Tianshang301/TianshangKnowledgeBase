# 健康与养生

## 一、健康决定因素

### 1.1 健康影响因素模型

| 因素类别 | 占比 | 具体内容 |
|----------|------|----------|
| **生物学因素** | ~20% | 基因遗传、年龄、性别、先天条件 |
| **环境因素** | ~20% | 空气质量、水质、居住环境、工作环境 |
| **行为因素** | ~50% | 饮食、运动、吸烟、饮酒、睡眠、压力管理 |
| **社会因素** | ~10% | 教育水平、收入、社会支持、医疗可及性 |

> 行为因素是最大可控因素，改善生活方式是提升健康最有效的手段。

## 二、营养基础

### 2.1 三大宏量营养素

| 营养素 | 功能 | 来源 | 推荐供能比 |
|--------|------|------|------------|
| **碳水化合物** | 主要能量来源、大脑供能 | 全谷物、薯类、水果、豆类 | 50%-65% |
| **蛋白质** | 组织修复、酶和激素合成 | 瘦肉、蛋、奶、豆制品、鱼 | 10%-20% |
| **脂肪** | 能量储存、脂溶性维生素吸收 | 坚果、植物油、鱼油、牛油果 | 20%-30% |

### 2.2 微量营养素

| 类别 | 重要代表 | 功能 | 缺乏后果 |
|------|----------|------|----------|
| **维生素 A** | β-胡萝卜素 | 视力、免疫功能 | 夜盲症 |
| **维生素 C** | 抗坏血酸 | 抗氧化、胶原蛋白合成 | 坏血病 |
| **维生素 D** | 钙化醇 | 钙吸收、骨骼健康 | 骨质疏松 |
| **B 族维生素** | B1、B2、B6、B12、叶酸 | 能量代谢、神经系统 | 贫血、神经病变 |
| **钙 (Ca)** |  | 骨骼牙齿、肌肉收缩 | 骨质疏松 |
| **铁 (Fe)** | 血红素铁/非血红素铁 | 血红蛋白运输氧 | 缺铁性贫血 |
| **锌 (Zn)** |  | 免疫功能、伤口愈合 | 免疫力下降 |

## 三、均衡饮食原则

| 原则 | 说明 | 实践建议 |
|------|------|----------|
| **多样性** | 每天摄入12种以上食物，每周25种以上 | 不同颜色的蔬果搭配 |
| **适量性** | 控制总能量摄入，保持能量平衡 | 七分饱，使用小餐具 |
| **全食物优先** | 减少加工食品 | 吃水果而非果汁，吃全麦而非精制 |
| **充足饮水** | 成年人每日1.5-2L | 少量多次，餐前饮水 |
| **限制添加糖** | <每日总能量10% | 少喝含糖饮料，减少甜点 |
| **限制饱和脂肪** | <每日总能量10% | 选择瘦肉，植物油替代动物油 |
| **限制钠** | <5g 盐/日 | 少用酱油和调味酱，注意隐形盐 |

## 四、身体活动指南

### 4.1 WHO 运动推荐

| 年龄组 | 有氧运动 | 力量训练 | 额外说明 |
|--------|----------|----------|----------|
| **18-64岁** | 每周≥150分钟中等强度 **或** ≥75分钟高强度 | 每周≥2天，涉及主要肌群 | 每次有氧≥10分钟才计入 |
| **65岁以上** | 同上 | 同上，加上平衡训练 | 每周3天以上平衡训练预防跌倒 |
| **儿童青少年** | 每天≥60分钟中高强度 | 每周≥3天 | 以有氧为主 |

**强度判断：**
- **中等强度**：运动时心率加快但仍能说话（快走、慢骑自行车、双打网球）
- **高强度**：运动时说不出完整句子（跑步、游泳、跳绳、单打网球）

### 4.2 运动类型搭配

| 类型 | 好处 | 频率建议 | 示例 |
|------|------|----------|------|
| **有氧运动** | 心肺功能、减脂 | 3-5次/周 | 快走、跑步、游泳、骑行 |
| **力量训练** | 增肌、骨密度、代谢 | 2-3次/周 | 深蹲、俯卧撑、举重 |
| **柔韧性训练** | 关节活动度、防损伤 | 每天 | 拉伸、瑜伽、太极 |
| **平衡训练** | 防跌倒、核心稳定 | 2-3次/周 | 单脚站立、太极、普拉提 |

## 五、睡眠健康

### 5.1 睡眠周期

| 阶段 | 特征 | 占比 | 作用 |
|------|------|------|------|
| **N1（浅睡）** | 入睡期，易唤醒 | 5% | 过渡阶段 |
| **N2** | 稳定睡眠，体温下降 | 45% | 记忆巩固 |
| **N3（深睡）** | 慢波，最难唤醒 | 25% | 身体修复、生长激素分泌 |
| **REM（快速眼动）** | 做梦期，大脑活跃 | 25% | 情绪调节、记忆整合 |

> 一个完整周期约90分钟，每晚经历4-6个周期。

### 5.2 各年龄推荐睡眠时长

| 年龄组 | 推荐时长 | 可能合适 |
|--------|----------|----------|
| 新生儿（0-3月） | 14-17小时 | 11-19小时 |
| 婴儿（4-11月） | 12-15小时 | 10-18小时 |
| 幼儿（1-2岁） | 11-14小时 | 9-16小时 |
| 学龄前（3-5岁） | 10-13小时 | 8-14小时 |
| 学龄（6-13岁） | 9-11小时 | 7-12小时 |
| 青少年（14-17岁） | 8-10小时 | 7-11小时 |
| 成人（18-64岁） | 7-9小时 | 6-10小时 |
| 老年人（65+） | 7-8小时 | 5-9小时 |

### 5.3 睡眠卫生习惯

| 推荐做法 | 避免做法 |
|----------|----------|
| 固定作息时间（包括周末） | 睡前3小时大量进食 |
| 卧室黑暗、安静、凉爽（18-22°C） | 睡前使用电子设备（蓝光抑制褪黑素） |
| 睡前放松仪式（阅读、冥想） | 咖啡因（下午2点后避免） |
| 白天接触充足自然光 | 酒精助眠（破坏睡眠结构） |
| 规律运动（但睡前3小时避免剧烈运动） | 床上工作/看手机（破坏床-睡关联） |

## 六、压力管理

### 6.1 压力生理反应

**HPA 轴（下丘脑-垂体-肾上腺）反应：**
- 感知压力 → 下丘脑释放 CRH → 垂体释放 ACTH → 肾上腺释放皮质醇
- 短期：提高警觉、能量动员、免疫功能增强
- 长期：免疫抑制、代谢紊乱、心血管负担、认知下降

### 6.2 放松技术对比

| 技术 | 原理 | 操作 | 适用场景 |
|------|------|------|----------|
| **腹式呼吸** | 激活副交感神经 | 4秒吸气→4秒屏息→6秒呼气 | 焦虑发作、入睡前 |
| **渐进式肌肉放松 (PMR)** | 紧张-放松对比感知 | 从头到脚逐组肌肉收紧5秒→放松10秒 | 入睡困难、紧张性头痛 |
| **正念冥想** | 当下觉察，非评判 | 专注呼吸，思绪漂移时温柔拉回 | 日常压力、慢性焦虑 |
| **生物反馈 (Biofeedback)** | 用设备感知生理信号 | 心率和呼吸变异性监测，自主调节 | 高血压、慢性疼痛 |
| **引导想象** | 使用积极意象放松 | 想象宁静场景（海滩、森林） | 手术前焦虑、疼痛管理 |

## 七、心理健康

### 7.1 维护心理健康

| 维度 | 实践方法 |
|------|----------|
| **正念 (Mindfulness)** | 每天10分钟正念冥想，专注于当下 |
| **社会连接** | 保持与家人朋友的联系，参加社群活动 |
| **工作生活平衡** | 设定边界，定期断网，培养爱好 |
| **感恩练习** | 每天写下3件感恩的事 |
| **自我关怀 (Self-compassion)** | 用对待朋友的方式对待自己 |
| **减少比较** | 限制社交媒体，关注自身进步 |

### 7.2 心理预警信号

| 信号 | 说明 | 建议行动 |
|------|------|----------|
| 持续情绪低落 >2周 | 可能抑郁 | 寻求心理咨询 |
| 过度焦虑影响日常 | 广泛性焦虑可能 | CBT（认知行为疗法） |
| 睡眠或食欲显著改变 | 身心失衡信号 | 检查生活方式，咨询医生 |
| 社交退缩 | 早期心理问题标志 | 逐步恢复社交，寻求支持 |
| 自我伤害念头 | 紧急情况 | **立即拨打心理危机热线** |

## 八、预防性健康

### 8.1 常规筛查建议

| 检查项目 | 频率 | 开始年龄 | 说明 |
|----------|------|----------|------|
| 血压测量 | 每2年 | 18岁 | 正常后可延长 |
| 血脂检查 | 每4-6年 | 20岁 | 有风险因素更频繁 |
| 空腹血糖 | 每3年 | 45岁 | 高危人群更早 |
| 结直肠癌筛查 | 每1-10年 | 45岁 | 结肠镜/粪便潜血 |
| 乳腺癌筛查 | 每2年 | 40-45岁 | 乳腺 X 线摄影 |
| 宫颈癌筛查 | 每3-5年 | 25岁 | Pap 涂片/HPV 检测 |
| 骨密度检查 | 根据风险 | 65岁（女）/70岁（男） | DXA 扫描 |
| 眼科检查 | 每2年 | 40岁 | 青光眼、白内障筛查 |

### 8.2 成人免疫接种

| 疫苗 | 推荐人群 | 频率 |
|------|----------|------|
| 流感疫苗 | 所有成年人 | 每年 |
| Tdap（百白破） | 所有成年人 | 每10年加强 |
| HPV 疫苗 | 9-45岁 | 2-3剂 |
| 带状疱疹疫苗 | ≥50岁 | 2剂 |
| 肺炎疫苗 | ≥65岁或高危 | 1-2剂 |
| 新冠疫苗 | 所有成年人 | 按更新推荐 |

## 十一、行为改变

### 11.1 跨理论模型 (Transtheoretical Model)

| 阶段 | 特征 | 策略 |
|------|------|------|
| **前意图期** | 未考虑改变（"我没问题"） | 提供信息，提高意识 |
| **意图期** | 考虑但未行动（"我该做了"） | 讨论利弊，增强动机 |
| **准备期** | 计划在近期行动 | 制定具体计划，设定 SMART 目标 |
| **行动期** | 开始改变行为 | 管理环境，寻求支持，应对诱惑 |
| **维持期** | 坚持>6个月 | 预防复发，建立新常态 |
| **终止期** | 改变已固化 | 无需额外努力 |

### 11.2 SMART 目标设定

| 字母 | 含义 | 示例（健身） |
|------|------|--------------|
| **S**pecific | 具体明确 | "每周跑步3次" 而非 "多运动" |
| **M**easurable | 可衡量 | "每次5公里" |
| **A**chievable | 可实现 | 从当前水平逐步增加 |
| **R**elevant | 与目标相关 | 符合总体健康目标 |
| **T**ime-bound | 有时限 | "在8周内达到" |

### 11.3 习惯养成策略

| 策略 | 说明 | 示例 |
|------|------|------|
| **习惯叠加** | 在现有习惯后加新习惯 | "刷牙后立即做10个俯卧撑" |
| **环境设计** | 让好习惯方便、坏习惯困难 | 把运动鞋放门口，手机放另一个房间 |
| **2分钟法则** | 新习惯从2分钟开始 | "穿好跑鞋" 即算完成一次运动 |
| **问责机制** | 让他人监督 | 与朋友约定每天互相报告进度 |

## 十二、数字健康工具

| 类别 | 工具示例 | 功能 | 注意事项 |
|------|----------|------|----------|
| **可穿戴设备** | Apple Watch, Fitbit, 小米手环 | 计步、心率、睡眠追踪、心电图 | 数据仅供参考，非医疗设备 |
| **健康类 App** | MyFitnessPal, Lose It, 薄荷健康 | 饮食记录、热量计算、营养分析 | 注意饮食记录偏差 |

| **远程医疗** | 好大夫在线、平安好医生、丁香医生 | 在线问诊、处方开药、健康咨询 | 急重症仍须线下就医 |
