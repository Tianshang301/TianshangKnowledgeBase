# 居家维护

## 概述

居家维护是保障居住安全和舒适的重要工作。掌握基础的维护知识和技能，能够及时发现和解决家庭问题，延长设施使用寿命，节约维修费用。本章系统介绍基础水电、木工维修、墙面地面维护等内容。

## 基础水电

### 水路系统

#### 家庭水路结构

家庭水路分为给水系统和排水系统两部分。

| 系统 | 组成 | 功能 | 常见问题 |
|-----|------|------|---------|
| 给水系统 | 进水管、阀门、水龙头 | 供水 | 漏水、水压不足 |
| 排水系统 | 排水管、地漏、马桶 | 排污 | 堵塞、异味 |
| 热水系统 | 热水器、热水管 | 提供热水 | 不热、漏水 |

#### 常见水路问题处理

| 问题 | 原因 | 处理方法 |
|-----|------|---------|
| 水龙头漏水 | 垫圈老化 | 更换垫圈或阀芯 |
| 水管漏水 | 接头松动或破裂 | 拧紧接头或更换管道 |
| 下水道堵塞 | 杂物堆积 | 皮搋子或管道疏通剂 |
| 马桶堵塞 | 异物堵塞 | 皮搋子或马桶疏通器 |
| 水压不足 | 管道堵塞或阀门问题 | 清洗滤网或调整阀门 |

#### 水龙头维修

- 关闭角阀，切断水源
- 拆卸水龙头把手
- 检查阀芯和垫圈
- 更换损坏部件
- 重新组装并测试

### 电路系统

#### 电路安全

| 安全事项 | 说明 |
|---------|------|
| 断电操作 | 维修前关闭总开关 |
| 验电测试 | 用电笔确认断电 |
| 绝缘工具 | 使用有绝缘手柄的工具 |
| 防水防潮 | 保持手和工具干燥 |
| 不超负荷 | 避免过多大功率电器 |

#### 常见电路问题

| 问题 | 可能原因 | 处理方法 |
|-----|---------|---------|
| 跳闸 | 过载或短路 | 分路检查，减少负荷 |
| 插座没电 | 跳闸或线路问题 | 检查配电箱，测试线路 |
| 开关失灵 | 接触不良 | 更换开关 |
| 灯泡闪烁 | 接触不良或电压不稳 | 检查灯座和线路 |
| 漏电 | 绝缘损坏 | 立即维修 |

## 木工维修

### 常见维修

#### 桌椅维修

| 问题 | 原因 | 处理方法 |
|-----|------|---------|
| 摇动不稳 | 螺丝松动 | 加固螺丝或榫头 |
| 破损 | 碰撞或老化 | 木工胶粘合或补木 |
| 漆面脱落 | 磨损 | 打磨后重新上漆 |
| 抽屉卡滞 | 轨道问题 | 调整轨道或上蜡 |

#### 门窗维修

- 门锁不灵活：加润滑油或调整锁舌
- 门关不严：调整合页位置
- 窗户漏风：更换密封条
- 推拉门卡滞：清理轨道，调整滑轮

#### 柜体维修

| 问题 | 原因 | 处理 |
|-----|------|------|
| 柜门下垂 | 合页松动 | 调整合页螺丝 |
| 隔板脱落 | 支撑松动 | 更换支撑件 |
| 门板变形 | 受潮 | 调整或更换 |
| 把手松动 | 螺丝松动 | 拧紧或更换 |

### 工具使用

#### 基本工具

| 工具 | 用途 | 使用要点 |
|-----|------|---------|
| 锤子 | 敲打 | 握柄末端，对准敲击 |
| 螺丝刀 | 拧螺丝 | 选择匹配的刀头 |
| 扳手 | 拧螺母 | 选择合适规格 |

## 墙面地面维护

### 墙面维护

#### 墙面类型

| 类型 | 特点 | 维护方法 |
|-----|------|---------|
| 乳胶漆 | 常见，易清洁 | 轻微污渍用湿布擦拭 |
| 壁纸 | 装饰效果好 | 避免受潮 |
| 瓷砖 | 防水耐用 | 定期勾缝 |
| 木板 | 温馨自然 | 防潮防虫 |

#### 墙面修复

- 小裂缝：补缝膏填平，打磨后刷漆
- 钉孔：补墙膏填平，打磨平整
- 发霉：除霉剂处理，保持通风
- 起皮：铲除旧漆，重新涂刷

### 地面维护

#### 地板类型

| 类型 | 优点 | 缺点 | 维护 |
|-----|------|------|------|
| 实木地板 | 质感好 | 怕水怕刮 | 定期打蜡 |
| 复合地板 | 耐用稳定 | 质感一般 | 拖把拧干后擦拭 |
| 瓷砖 | 防水耐磨 | 冷硬 | 日常清洁 |
| 石材 | 高档美观 | 需维护 | 定期密封 |

#### 地板问题处理

| 问题 | 原因 | 处理 |
|-----|------|------|
| 地板起拱 | 受潮 | 除湿或更换 |
| 地板有响声 | 松动 | 加固或填充 |
| 瓷砖空鼓 | 粘贴不牢 | 重新铺贴 |
| 缝隙变大 | 干燥收缩 | 填缝处理 |

## 家电维护

### 空调

| 维护项目 | 频率 | 方法 |
|---------|------|------|
| 清洗滤网 | 每月 | 取出滤网清水冲洗 |
| 检查制冷剂 | 每年 | 专业检查 |
| 室外机清洁 | 每半年 | 清理散热片 |
| 运行测试 | 换季前 | 试运行检查 |

### 冰箱

- 定期除霜（非无霜型）
- 清洁密封条，确保密封
- 清理冷凝器灰尘
- 检查温度设置
- 不放入过热的食物

### 洗衣机

| 维护项目 | 频率 | 方法 |
|---------|------|------|
| 清洗滤网 | 每月 | 清理棉絮 |
| 筒自洁 | 每月 | 使用筒清洁程序 |
| 检查进水管 | 每半年 | 检查破裂 |
| 清理排水泵 | 必要时 | 清理异物 |

## 防水防潮

### 常见漏水点

| 位置 | 原因 | 处理 |
|-----|------|------|
| 屋顶 | 防水层老化 | 重新做防水 |
| 卫生间 | 防水层破裂 | 重做防水 |
| 窗台 | 密封不严 | 打胶密封 |
| 管道 | 接头松动 | 更换接头 |
| 地下室 | 地下水位高 | 外墙防水 |

### 防潮措施

- 保持通风
- 使用除湿机
- 安装排气扇
- 房间放置干燥剂
- 装修时做好防潮层

## 季节性维护

| 季节 | 维护事项 |
|-----|---------|
| 春季 | 清理排水沟，检查屋顶，通风除湿 |
| 夏季 | 检查空调，防蚊虫，防晒 |
| 秋季 | 清理落叶，检查暖气，封窗保温 |
| 冬季 | 防冻措施，检查取暖设备，防滑 |
