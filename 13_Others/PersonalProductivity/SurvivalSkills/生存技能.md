# 生存技能

## 概述

生存技能是在各种环境中维持生命和健康的能力，涵盖城市生存和野外生存两个方面。掌握基本的生存技能，能够在突发情况下保护自己和家人。本章系统介绍城市生存、野外生存、应急准备和自救互救等内容。

## 城市生存

### 应急包

城市应急包是在城市环境中应对突发事件的必备物品。

| 类别 | 物品 | 数量 | 说明 |
|-----|------|------|------|
| 饮水食物 | 矿泉水 | 3天量 | 每人每天2L |
| 饮水食物 | 压缩饼干 | 3天量 | 高能量 |
| 急救用品 | 创可贴、消毒 | 各若干 | 外伤处理 |
| 照明 | 手电筒、电池 | 各2 | 备用光源 |
| 工具 | 多功能刀 | 1 | 应急使用 |
| 通讯 | 充电宝 | 2 | 保持通讯 |
| 保暖 | 应急毯 | 2 | 保温 |
| 其他 | 现金、证件复印件 | 若干 | 身份证明 |

### 城市灾害应对

#### 火灾

| 阶段 | 应对措施 |
|-----|---------|
| 发现火灾 | 立即报警119 |
| 逃生 | 低姿前进，湿毛巾捂口鼻 |
| 被困 | 关闭门窗，塞住缝隙，等待救援 |
| 跳楼 | 低楼层才考虑，用床单结绳 |

#### 地震

- 室内：躲到坚固桌下或墙角
- 室外：远离建筑物和玻璃幕墙
- 震后：检查燃气和水管
- 不要使用电梯
- 注意余震

#### 停水停电

| 准备工作 | 说明 |
|---------|------|
| 储备饮用水 | 至少3天用量 |
| 备用照明 | 手电筒、蜡烛 |
| 备用电源 | 充电宝 |
| 食物储备 | 不需冷藏的食物 |
| 保暖措施 | 备用毯子 |

### 城市避难

#### 避难场所

- 熟悉附近的避难所位置
- 了解社区应急方案
- 知道最近的医院和派出所
- 约定家人集合点

#### 紧急联络

- 设定紧急联系人
- 了解紧急电话：110、119、120
- 事先和家人约定联系方式
- 准备应急通讯卡

## 野外生存

### 基本装备

| 装备 | 用途 | 选择建议 |
|-----|------|---------|
| 刀具 | 切割、制作工具 | 折叠刀或生存刀 |
| 打火工具 | 生火 | 打火机+打火石 |
| 水壶 | 储水 | 金属水壶可烧水 |
| 绳索 | 搭建、固定 | 伞绳，承重强 |
| 指南针 | 导航 | 配合地图使用 |
| 急救包 | 医疗 | 根据行程准备 |
| 应急毯 | 保暖 | 轻便、反光 |

### 方向辨别

#### 自然导航

| 方法 | 说明 |
|-----|------|
| 太阳 | 东升西落的基本方向 |
| 树木 | 南面枝叶更茂盛 |
| 北极星 | 北方天空的恒定星星 |
| 手表 | 时针对准太阳，夹角中线是南 |
| 地形 | 南坡干燥、北坡湿润 |

#### 使用工具

- 指南针：最可靠的方向工具
- GPS：定位和导航
- 地图：地形识别
- 手机：离线地图

### 食物获取

#### 可食用植物

| 植物 | 特征 | 食用部位 |
|-----|------|---------|
| 蒲公英 | 黄色花、锯齿叶 | 叶、花 |
| 车前草 | 宽叶、穗状花序 | 嫩叶 |
| 蕨菜 | 卷曲嫩芽 | 嫩芽 |
| 荠菜 | 三角形种子荚 | 全草 |
| 马齿苋 | 肉质叶 | 茎叶 |

#### 安全规则

- 不认识的不吃
- 先少量尝试
- 观察动物是否吃
- 避免蘑菇（除非非常确定）
- 避免有白色汁液的植物

#### 狩猎和捕鱼

| 方法 | 难度 | 效率 |
|-----|------|------|
| 陷阱 | 中等 | 长期有效 |
| 钓鱼 | 中等 | 看水域 |
| 弹弓 | 较难 | 需要练习 |
| 设套 | 中等 | 小型动物 |

## 应急准备

### 家庭应急计划

#### 制定计划

- 确定家人集合地点
- 了解社区应急方案
- 准备应急联系卡
- 定期进行家庭演练

#### 应急物资储备

| 储备类别 | 物品 | 储备量 |
|---------|------|-------|
| 饮用水 | 瓶装水 | 每人每天2L，3天 |
| 食品 | 罐头、压缩饼干 | 3天量 |
| 药品 | 常用药、急救用品 | 家庭用量 |
| 卫生 | 纸巾、湿巾、垃圾袋 | 适量 |
| 工具 | 手电、收音机、电池 | 各1-2 |

### 通讯方案

- 手机是主要通讯工具
- 备用充电方式（充电宝、手摇充电器）
- 收音机接收灾害信息
- 约定替代联系方式（如短信更可靠）

## 自救互救

### 止血

| 出血类型 | 特征 | 止血方法 |
|---------|------|---------|
| 毛细血管 | 渗血 | 直接压迫 |
| 静脉出血 | 暗红色，持续流出 | 压迫+抬高 |
| 动脉出血 | 鲜红色，喷射状 | 加压包扎或止血带 |

### 骨折固定

- 使用夹板固定骨折两端
- 固定上下关节
- 检查远端血液循环
- 避免直接搬动

### 搬运伤员

| 搬运方式 | 适用情况 | 操作要点 |
|---------|---------|---------|
| 单人背负 | 轻伤员 | 固定伤员双手 |
| 双人托抱 | 昏迷伤员 | 协调步伐 |
| 担架 | 重伤员 | 平稳移动 |
| 拖行 | 紧急情况 | 保护头部 |

## 心理适应

### 孤独和恐惧

- 保持规律作息
- 设定小目标
- 自我对话鼓励
- 回忆积极经历
- 相信自己的判断

### 团队协作

- 分配角色和任务
- 保持沟通畅通
- 尊重每个人的意见
- 分享资源和信息
- 相互鼓励和支持
