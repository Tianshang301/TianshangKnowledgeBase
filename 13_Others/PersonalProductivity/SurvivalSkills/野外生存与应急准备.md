# 野外生存与应急准备

## 概述

野外生存是在资源有限、环境陌生的自然环境中维持生命的能力。应急准备则是面对突发灾害时保障安全的必要措施。本章系统介绍野外生存核心原则、生存技能、应急准备方法和求救技巧。

## 野外生存核心原则

### STOP法则

野外生存的第一原则是冷静下来思考和计划。

| 字母 | 含义 | 行动 |
|-----|------|------|
| S | Stop（停下） | 停下脚步，保持冷静 |
| T | Think（思考） | 评估现状，制定计划 |
| O | Observe（观察） | 观察环境，寻找资源 |
| P | Plan（计划） | 制定行动计划 |

### 生存优先级

| 优先级 | 需求 | 时间范围 |
|-------|------|---------|
| 1 | 保护（安全、急救） | 立即 |
| 2 | 求救（定位和联系） | 尽快 |
| 3 | 饮水（水源） | 3天 |
| 4 | 食物（能量） | 3周 |
| 5 | 庇护所（保暖） | 根据需要 |

## 水源获取

### 找水方法

#### 水源线索

| 线索 | 说明 |
|-----|------|
| 地形 | 山谷、低洼处 |
| 植被 | 茂密植物生长处 |
| 动物 | 动物足迹和路径 |
| 鸟类 | 清晨和傍晚鸟群飞向水源 |
| 昆虫 | 蜜蜂和蚂蚁需要水 |

#### 野外取水

| 方法 | 操作 | 出水量 |
|-----|------|-------|
| 雨水收集 | 使用容器或防水布 | 视降雨量 |
| 露水收集 | 清晨用布料收集 | 少量 |
| 植物取水 | 切割藤蔓和仙人掌 | 视植物 |
| 太阳能蒸馏 | 塑料布+容器 | 少量 |
| 雪融化 | 加热融化 | 大量 |

### 净水方法

#### 净化技术

| 方法 | 效果 | 操作 |
|-----|------|------|
| 煮沸 | 杀灭微生物 | 沸腾至少1分钟 |
| 净水片 | 化学消毒 | 按说明使用 |
| 过滤 | 去除悬浮物 | 布料+沙石层 |
| 紫外线 | 杀灭微生物 | 专用设备 |
| 蒸馏 | 最彻底 | 收集蒸汽 |

#### 注意事项

- 不直接饮用生水
- 选择流动的水源
- 避免下游有污染源的水
- 雪水也需要煮沸

## 庇护所搭建

### 选址要求

| 考虑因素 | 理想条件 |
|---------|---------|
| 地势 | 平坦、避风、排水良好 |
| 水源 | 靠近水源但不在低洼处 |
| 材料 | 附近有天然材料 |
| 安全 | 远离危险（落石、枯树） |
| 可见度 | 容易被救援发现 |

### 常见庇护所

#### 简易庇护所

| 类型 | 材料 | 搭建方法 |
|-----|------|---------|
| A字架 | 树枝、树叶 | 两根树枝交叉，覆盖枝叶 |
| 倾斜棚 | 树枝、防水布 | 一端架高，一端贴地 |
| 树洞庇护 | 树枝、树叶 | 利用天然树洞或大树根 |
| 雪洞 | 雪 | 挖雪成洞穴 |

#### 保温方法

- 地面铺设隔潮层（树枝、树叶）
- 身体与地面隔离
- 填充保温材料
- 封住风口

## 生火技术

### 生火方法

| 方法 | 难度 | 操作 |
|-----|------|------|
| 打火机 | 最简单 | 直接点火 |
| 火柴 | 简单 | 防潮保存 |
| 放大镜 | 中等 | 聚焦阳光 |
| 打火石 | 较难 | 刮擦火花 |
| 弓钻取火 | 很难 | 摩擦生热 |

### 火堆类型

| 类型 | 特点 | 用途 |
|-----|------|------|
| 锥形火堆 | 容易点燃 | 取暖、煮食 |
| 星形火堆 | 节省燃料 | 长时间燃烧 |
| 沟渠火堆 | 隐蔽防风 | 安全烹饪 |
| 信号火堆 | 烟大火大 | 求救信号 |

### 安全用火

- 清理周围可燃物
- 挖防火沟
- 准备灭火用水或沙
- 离开前彻底熄灭
- 注意森林防火规定

## 应急准备

### 应急背包

#### 核心物品

| 类别 | 物品 | 用途 |
|-----|------|------|
| 水 | 水壶、净水片 | 饮水 |
| 食物 | 高能量食品 | 能量补充 |
| 保暖 | 应急毯、保暖衣物 | 维持体温 |
| 照明 | 手电筒、头灯 | 照明 |
| 急救 | 急救包、药物 | 医疗 |
| 工具 | 多功能刀、打火石 | 生存工具 |
| 通讯 | 手机、充电宝 | 通讯 |
| 求救 | 哨子、镜子 | 求救 |

#### 家庭应急包

- 3天的饮用水和食品
- 急救用品和常用药
- 手电筒和备用电池
- 收音机（手摇或电池）
- 重要证件复印件
- 现金和信用卡
- 保暖衣物和雨具

### 灾害应对

#### 地震

| 场所 | 应对措施 |
|-----|---------|
| 室内 | 躲在坚固桌下，保护头部 |
| 室外 | 远离建筑物、电线杆 |
| 驾车 | 停车在安全处，留在车内 |
| 震后 | 检查燃气，准备余震 |

#### 洪水

- 收到预警时转移到高地
- 切断电源和燃气
- 不涉水行走
- 准备救生设备
- 注意饮用水安全

#### 台风

- 加固门窗和室外物品
- 储备食物和水
- 远离海边和河边
- 待在室内安全位置
- 关注官方预警信息

## 心理准备

### 生存心态

| 心理因素 | 重要性 | 培养方法 |
|---------|-------|---------|
| 冷静 | 极高 | 呼吸训练 |
| 希望 | 极高 | 设定小目标 |
| 适应 | 高 | 接受现实 |
| 坚持 | 高 | 自我激励 |
| 创造力 | 中 | 灵活变通 |

### 保持士气

- 设定可达成的目标
- 保持日常习惯
- 自我对话和鼓励
- 感恩已有的条件
- 相信救援会来
