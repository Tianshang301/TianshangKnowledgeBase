# 个人理财

## 一、预算管理

### 1.1 50/30/20法则

| 类别 | 比例 | 包括 |
|------|------|------|
| 必要支出（Needs） | 50% | 房租/房贷、饮食、交通、水电、保险、基本日用品 |
| 非必要支出（Wants） | 30% | 娱乐、餐饮、购物、旅行、订阅服务 |
| 储蓄与投资（Savings） | 20% | 紧急备用金、养老金、基金定投、债务还款 |

### 1.2 预算方法

| 方法 | 说明 | 适合人群 |
|------|------|----------|
| 信封法（Envelope System） | 每月分装现金到不同信封 | 控制支出困难者 |
| 零基预算法 | 每分钱都有用途 | 精细管理 |
| 预算App追踪 | 自动分类记录 | 所有人 |

## 二、紧急备用金

| 要素 | 建议 |
|------|------|
| 金额 | 3-6个月生活支出 |
| 自由职业者 | 6-12个月生活支出 |
| 存放方式 | 货币基金（余额宝/零钱通）或活期存款 |
| 原则 | 保本+流动性优先，不可用于投资 |

## 三、债务管理

### 3.1 好债务 vs 坏债务

| 类型 | 举例 | 策略 |
|------|------|------|
| 好债务 | 房贷（低利率） | 按时还款，合理利用杠杆 |
| 坏债务 | 信用卡循环利息（15-18%）、消费贷 | 优先偿还 |

### 3.2 还款策略

| 方法 | 原理 | 适用人群 |
|------|------|----------|
| 雪球法（Debt Snowball） | 先还最小金额的债务→获得心理成就感 | 需要激励者 |
| 雪崩法（Debt Avalanche） | 先还最高利率的债务→数学最优 | 理性分析者 |

### 3.3 信用卡管理

| 原则 | 说明 |
|------|------|
| 全额还款 | 每月全额还清，避免循环利息 |
| 控制数量 | 2-3张足够 |
| 利用免息期 | 最长50-56天 |
| 信用评分 | 按时还款，保持低使用率（<30%） |

## 四、信用评分

### 4.1 信用评分构成

| 因素 | 权重 | 说明 |
|------|------|------|
| 还款历史 | 35% | 是否按时还款 |
| 信用利用率 | 30% | 已使用额度/总信用额度（<30%最佳） |
| 信用历史长度 | 15% | 最早账户的开立时间 |
| 信用种类 | 10% | 信用卡、房贷、车贷等混合 |
| 新账户数量 | 10% | 近期新开账户数量 |

### 4.2 提高信用评分

- 按时全额还款
- 保持低信用利用率（<30%）
- 不频繁申请新信用卡
- 长期持有旧账户（不关闭最老的卡）

## 五、保险

### 5.1 保险类型

| 类型 | 作用 | 建议 |
|------|------|------|
| 寿险（Term Life） | 家人保障（有依赖者时） | 保额为年收入5-10倍 |
| 寿险（Whole Life） | 储蓄+保障 | 昂贵，不推荐 |
| 健康险/医疗险 | 覆盖医疗费用 | 基本必备 |
| 意外险 | 意外伤害保障 | 价格低，建议配置 |
| 车险 | 车辆损失和责任 | 法定必备（至少交强险） |
| 房屋/租房保险 | 房屋财产保护 | 租房者也建议 |

### 5.2 保险规划优先级

| 优先级 | 保险种类 |
|--------|----------|
| 1 | 社会医疗保险（基本医保） |
| 2 | 百万医疗险（补充医保不覆盖的部分） |
| 3 | 重疾险（确诊一次性赔付） |
| 4 | 意外险 |
| 5 | 定期寿险（有家庭依赖者） |

## 六、税务基础

### 6.1 个人所得税税率

| 应纳税所得额（年） | 税率 | 速算扣除数 |
|-------------------|------|------------|
| ≤36,000元 | 3% | 0 |
| 36,001-144,000元 | 10% | 2,520 |
| 144,001-300,000元 | 20% | 16,920 |
| 300,001-420,000元 | 25% | 31,920 |
| 420,001-660,000元 | 30% | 52,920 |
| 660,001-960,000元 | 35% | 85,920 |
| >960,000元 | 45% | 181,920 |

### 6.2 常见节税方式

| 方式 | 说明 |
|------|------|
| 专项附加扣除 | 子女教育、继续教育、大病医疗、房贷利息/房租、赡养老人 |
| 企业年金 | 单位补充养老金 |
| 个人养老金 | 每年最高12,000元税前扣除 |
| 商业健康险 | 部分可以税前扣除 |

## 七、退休账户

| 账户类型 | 特点 | 税收优惠 |
|----------|------|----------|
| 企业年金 | 企业缴纳+个人缴纳 | 缴费税前扣除 |
| 个人养老金 | 自主缴纳 | 缴费税前扣除，领取时缴税 |
| 401(k)（美国） | 企业匹配 | 税前缴费，提取时缴税 |
| Roth IRA（美国） | 税后缴费 | 缴稅后，提取免税 |

## 八、理财规划分阶段

### 8.1 20-30岁

| 目标 | 策略 |
|------|------|
| 建立紧急备用金 | 3个月支出×货币基金 |
| 开始投资 | 定投指数基金 |
| 控制债务 | 避免高利率消费贷 |
| 提升收入 | 投资自身技能 |

### 8.2 30-40岁

| 目标 | 策略 |
|------|------|
| 扩大紧急备用金 | 6个月支出 |
| 增加投资比例 | 50-70%股票+30-50%债券 |
| 购房准备 | 首付规划 |
| 保险配置 | 重疾+医疗+寿险 |

### 8.3 40-55岁

| 目标 | 策略 |
|------|------|
| 子女教育基金 | 教育储蓄/基金定投 |
| 养老规划 | 加大退休账户投入 |
| 优化资产配置 | 降低股票比例 |
| 考虑提前还贷 | 比较贷款利率和投资收益 |

### 8.4 55岁+

| 目标 | 策略 |
|------|------|
| 保本为主 | 债券+货币基金为主 |
| 退休收入 | 提取策略（4%法则） |
| 遗产规划 | 遗嘱+信托 |
| 医疗费用 | 长期护理保险 |

## 九、常见理财错误

| 错误 | 后果 | 避免方法 |
|------|------|----------|
| 没有紧急备用金 | 需要用高利率借贷 | 先存够3-6个月支出 |
| 过度消费 | 月光甚至负债 | 坚持预算，减少不必要开支 |
| 投资All in | 风险集中 | 分散投资 |
| 追涨杀跌 | 高买低卖 | 定投+长期持有 |
| 忽视保险 | 大额医疗可能致贫 | 配置基础保险 |
| 不设财务目标 | 缺乏方向和动力 | 设SMART财务目标 |
| 听信内部消息 | 容易亏钱 | 研究和学习，不做投机 |

## 十、财务独立（FIRE）基础

| 概念 | 说明 |
|------|------|
| FIRE | Financial Independence, Retire Early |
| 4%法则 | 退休后每年提取本金的4%，可持续30年 |
| 储蓄率 | 收入中储蓄的比例，决定达到FIRE的速度 |
| 被动收入 | 不需要工作就能获得的收入（股息/房租/版税） |
| 退休目标金额 | 年支出÷4%（如年支出20万→需要500万） |
