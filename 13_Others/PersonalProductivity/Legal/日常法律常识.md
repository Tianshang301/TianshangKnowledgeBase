# 日常法律常识

## 概述

法律知识是公民维护自身权益的重要工具。了解基本法律常识，能够帮助我们在日常生活中识别法律风险、保护合法权益。本章系统介绍合同法基础、劳动法知识、消费者权益保护和民事纠纷处理等内容。

## 合同法基础

### 合同的成立

合同是民事主体之间设立、变更、终止民事法律关系的协议。

| 合同要素 | 说明 | 示例 |
|---------|------|------|
| 要约 | 希望与他人订立合同的意思表示 | 报价单、标价 |
| 承诺 | 同意要约的意思表示 | 确认订单 |
| 主体 | 合同当事人 | 自然人或法人 |
| 标的 | 合同权利义务指向的对象 | 货物、服务 |
| 内容 | 双方权利义务约定 | 数量、价格、时间 |

### 合同的形式

| 形式 | 特点 | 法律效力 | 适用场景 |
|-----|------|---------|---------|
| 书面合同 | 有据可查 | 最强 | 重要交易 |
| 口头合同 | 方便快捷 | 有法律效力 | 日常小交易 |
| 电子合同 | 数字形式 | 等同书面 | 网购、在线服务 |

### 合同的主要条款

- 当事人信息
- 标的物
- 数量和质量
- 价款或报酬
- 履行期限和方式
- 违约责任
- 争议解决方式

### 合同的无效和撤销

| 情形 | 法律后果 |
|-----|---------|
| 欺诈 | 可撤销 |
| 胁迫 | 可撤销 |
| 重大误解 | 可撤销 |
| 违反法律强制性规定 | 无效 |
| 损害公共利益 | 无效 |
| 无民事行为能力人 | 无效 |

## 劳动法知识

### 劳动合同

#### 合同类型

| 类型 | 期限 | 适用场景 |
|-----|------|---------|
| 固定期限 | 有明确起止时间 | 多数情况 |
| 无固定期限 | 无终止时间 | 长期雇佣 |
| 以完成一定工作为期限 | 项目完成后终止 | 项目制工作 |

#### 合同必备条款

- 用人单位信息
- 劳动者信息
- 合同期限
- 工作内容和地点
- 工作时间和休息休假
- 劳动报酬
- 社会保险
- 劳动保护

### 工资和工时

#### 最低工资

- 各地标准不同，由省级政府制定
- 不包括加班费、特殊津贴
- 每年调整一次

#### 加班工资

| 加班类型 | 工资计算 |
|---------|---------|
| 工作日加班 | 150%日工资 |
| 休息日加班（不补休） | 200%日工资 |
| 法定节假日加班 | 300%日工资 |

#### 年休假

| 工作年限 | 年休假天数 |
|---------|-----------|
| 1-10年 | 5天 |
| 10-20年 | 10天 |
| 20年以上 | 15天 |

### 劳动关系解除

#### 劳动者解除

- 提前30天书面通知
- 试用期提前3天通知
- 用人单位有过错可随时解除

#### 用人单位解除

| 情形 | 条件 | 补偿 |
|-----|------|------|
| 劳动者过错 | 严重违纪、失职 | 无补偿 |
| 无过错解除 | 不胜任、裁员 | 支付经济补偿 |
| 违法解除 | 无正当理由 | 双倍补偿 |

### 社保与公积金

| 险种 | 缴费比例（大致） | 用途 |
|-----|----------------|------|
| 养老保险 | 单位16%，个人8% | 退休养老 |
| 医疗保险 | 单位8%，个人2% | 医疗报销 |
| 失业保险 | 单位0.5-2%，个人0.5% | 失业期间保障 |
| 工伤保险 | 单位0.2-1.9% | 工伤保障 |
| 生育保险 | 单位0.5-1% | 生育保障 |
| 住房公积金 | 单位5-12%，个人5-12% | 住房相关 |

## 消费者权益保护

### 消费者权利

| 权利 | 内容 |
|-----|------|
| 安全权 | 人身和财产安全 |
| 知情权 | 真实全面的信息 |
| 选择权 | 自主选择商品或服务 |
| 公平交易权 | 质量保障、价格合理 |
| 索赔权 | 受到损害要求赔偿 |
| 结社权 | 依法成立消费者组织 |
| 受教育权 | 获得消费知识 |

### 网购特别保护

- **七日无理由退货**：网购商品7天内可无理由退货
- **例外商品**：生鲜、定制商品、在线数字商品等除外
- **退货运费**：无理由退货运费由消费者承担

### 消费维权途径

1. 与经营者协商和解
2. 请求消费者协会调解
3. 向行政部门投诉（12315）
4. 根据仲裁协议提请仲裁
5. 向法院提起诉讼

## 民事侵权

### 侵权责任构成

| 要件 | 说明 |
|-----|------|
| 违法行为 | 违反了法律规定的义务 |
| 损害事实 | 造成了实际损害 |
| 因果关系 | 违法行为与损害之间有因果联系 |
| 过错 | 行为人有过错 |

### 常见侵权类型

| 类型 | 说明 | 赔偿范围 |
|-----|------|---------|
| 人身损害 | 交通事故、伤害 | 医疗费、误工费、残疾赔偿金 |
| 名誉损害 | 诽谤、侮辱 | 精神损害赔偿 |
| 财产损害 | 损坏他人财物 | 实际损失 |
| 产品责任 | 产品缺陷 | 人身和财产损失 |

### 诉讼时效

- 一般诉讼时效：3年
- 人身损害赔偿：1年（特殊情形）
- 从知道或应当知道权利受损之日起计算

## 房屋租赁

### 租赁合同要点

| 条款 | 注意事项 |
|-----|---------|
| 租金 | 明确金额、支付方式、调整机制 |
| 押金 | 金额、退还条件 |
| 维修责任 | 明确各方维修范围 |
| 转租 | 是否允许转租 |
| 提前解约 | 违约责任和通知期限 |

### 租客权利

- 要求房东提供安全的房屋
- 正常使用房屋不受干涉
- 优先购买权
- 合同到期后优先续租权

### 房东权利

- 按时收取租金
- 要求合理使用房屋
- 合同到期收回房屋
- 收取押金作为履约保障

## 婚姻家庭

### 结婚

- 法定婚龄：男22周岁，女20周岁
- 禁止结婚情形：直系血亲、三代以内旁系血亲
- 需双方自愿，到婚姻登记机关登记

### 离婚

| 方式 | 条件 | 程序 |
|-----|------|------|
| 协议离婚 | 双方自愿，子女财产协商 | 民政局登记 |
| 诉讼离婚 | 感情破裂 | 法院判决 |

### 财产分割

- 夫妻共同财产：婚后共同所得
- 个人财产：婚前财产、个人专用物品
- 共同债务：为共同生活所负的债务
