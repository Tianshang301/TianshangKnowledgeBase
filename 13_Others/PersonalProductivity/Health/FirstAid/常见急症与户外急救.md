# 常见急症与户外急救

## 概述

常见急症和户外急救知识对于每个人都是重要的生活技能。在遭遇突发疾病或户外意外时，正确的急救措施能够挽救生命、减轻伤害。本章系统介绍常见急症的识别与处理、户外急救技能和应急准备。

## 急救的基本原则

### 急救黄金时间

- **心脏骤停**：4分钟内实施 CPR 存活率最高
- **严重出血**：数分钟内可致命
- **中毒**：30分钟内处理效果最好
- **烧伤**：立即冷处理

### 急救优先级

#### 检伤分类

| 优先级 | 颜色 | 伤情描述 | 处理策略 |
|-------|------|---------|---------|
| 一级 | 红色 | 危及生命 | 立即抢救 |
| 二级 | 黄色 | 严重但不危及生命 | 尽快处理 |
| 三级 | 绿色 | 轻伤 | 可等待处理 |
| 四级 | 黑色 | 无法救治 | 最后处理 |

### 现场评估

- 判断现场是否安全
- 确定受伤人数
- 呼叫专业救援
- 分配急救资源
- 持续监控伤情变化

## 常见内科急症

### 心脏病发作

#### 识别

| 典型症状 | 非典型症状 |
|---------|-----------|
| 胸痛压迫感 | 下颌痛 |
| 左臂放射痛 | 背部疼痛 |
| 呼吸困难 | 上腹部不适 |
| 大汗淋漓 | 极度疲劳 |
| 恶心呕吐 | 晕厥 |

#### 处理

- 让患者立即停止活动
- 半坐卧位，放松衣物
- 如有医嘱，给予硝酸甘油
- 立即呼叫急救
- 准备好阿司匹林（如有医嘱）

### 中风

#### FAST 识别法

| 字母 | 含义 | 检查内容 |
|-----|------|---------|
| F | Face（面部） | 微笑是否嘴角歪斜 |
| A | Arm（手臂） | 平举是否单侧无力 |
| S | Speech（语言） | 说话是否口齿不清 |
| T | Time（时间） | 立即呼叫急救 |

#### 处理

- 让患者平卧，头部稍高
- 保持呼吸道通畅
- 记录症状开始时间
- 不要给患者进食或饮水
- 不要随意服用药物

### 糖尿病急症

#### 低血糖

| 症状 | 处理 |
|-----|------|
| 头晕、出汗、心慌 | 清醒者可口服糖水 |
| 意识模糊 | 立即呼叫急救 |
| 昏迷 | 不要经口喂食 |

#### 高血糖

- 口渴、多尿、乏力
- 呼吸有烂苹果味
- 意识逐渐模糊
- 立即就医

### 癫痫发作

- 保护头部，移开危险物品
- 记录发作时间
- 不要按压肢体
- 发作后侧卧位
- 持续超过5分钟呼叫急救

## 户外急救

### 环境相关急症

#### 失温（低体温症）

| 阶段 | 体温 | 症状 | 处理 |
|-----|------|------|------|
| 轻度 | 35-32℃ | 寒战、手脚麻木 | 保暖、热饮 |
| 中度 | 32-28℃ | 意识模糊、停止寒战 | 主动复温 |
| 重度 | <28℃ | 昏迷、呼吸微弱 | 轻柔运送，持续保暖 |

#### 热伤害

| 类型 | 症状 | 处理 |
|-----|------|------|
| 热痉挛 | 肌肉痉挛 | 补充电解质，停止活动 |
| 热衰竭 | 头晕、虚弱、大量出汗 | 到阴凉处，补充水分 |
| 热射病 | 体温>40℃、意识障碍 | 紧急降温，立即送医 |

#### 高原反应

| 程度 | 症状 | 处理 |
|-----|------|------|
| 轻度 | 头痛、失眠 | 休息、适应、不可继续攀高 |
| 中度 | 恶心、呕吐 | 吸氧、下撤 |
| 重度 | 意识改变、呼吸困难 | 立即下撤，就医 |

### 动物伤害

#### 蛇咬伤

- 保持冷静，减少活动
- 记住蛇的特征
- 去除首饰和紧身物品
- 不要切开伤口或吸出毒液
- 尽快就医

#### 虫咬/蜇伤

| 昆虫 | 处理 |
|-----|------|
| 蜜蜂 | 刮除毒刺，冷敷 |
| 蚊子 | 止痒药膏 |
| 蜱虫 | 用镊子夹出，不要挤压 |
| 蝎子 | 冷敷，就医 |

### 户外创伤

#### 扭伤

- RICE 原则：
  - R（休息）：停止活动
  - I（冰敷）：每次15-20分钟
  - C（压迫）：弹性绷带包扎
  - E（抬高）：抬高受伤部位

#### 骨折固定

- 使用自然材料（树枝、登山杖）做夹板
- 固定骨折上下关节
- 检查远端血液循环
- 尽量避免移动

## 水上急救

### 溺水

| 阶段 | 处理 |
|-----|------|
| 水中救援 | 使用浮具，避免直接接触 |
| 意识清醒 | 保暖，观察是否有呼吸困难 |
| 无意识有呼吸 | 侧卧位，清除口鼻异物 |
| 无意识无呼吸 | 立即 CPR，先做人工呼吸 |

### 潜水相关

- **减压病**：关节疼痛、皮疹、麻木。立即就医，高压氧治疗
- **肺气压伤**：胸痛、呼吸困难、咳血。紧急就医

## 应急准备

### 户外装备

| 类别 | 物品 | 用途 |
|-----|------|------|
| 急救包 | 敷料、绷带、消毒剂 | 处理伤口 |
| 求救装备 | 哨子、镜子、GPS | 求救定位 |
| 保暖装备 | 应急毯、火种 | 保暖 |
| 饮水装备 | 净水片、水壶 | 饮水安全 |
| 工具 | 多功能刀具、头灯 | 生存工具 |

### 出行前准备

- 了解目的地天气和地形
- 告知他人行程计划
- 检查装备完整
- 学习当地急救联系方式
- 携带必要的个人药品

### 求救信号

- 哨声：三短是三长是求救信号
- 火光：三堆火呈三角形
- 镜子：反射阳光
- 地面标志：SOS
- 手机：拨打紧急电话

## 急救误区

| 错误做法 | 正确做法 |
|---------|---------|
| 溺水后倒挂控水 | 立即 CPR |
| 蛇咬伤用嘴吸毒 | 保持不动，就医 |
| 心脏骤停先人工呼吸 | 先胸外按压 |
| 晕倒就掐人中 | 检查呼吸和脉搏 |
| 伤口用酒精直接冲洗 | 生理盐水或清水 |
