# 急救常识

## 概述

急救是在专业医疗人员到达之前对伤病者进行的紧急救助。掌握基本的急救知识和技能，可以在关键时刻挽救生命、减少伤害。本章系统介绍初步评估、常见急症处理、创伤急救和心肺复苏等内容。

## 初步评估

### 现场安全评估

在进行急救前，必须先确保现场安全：

- 检查环境是否存在危险（交通、火灾、气体泄漏等）
- 确保施救者和伤者的安全
- 如危险无法排除，应在安全距离外等待专业救援
- 做好个人防护（戴手套、口罩）

### 意识判断

1. 轻拍伤者肩膀，大声呼唤
2. 观察是否有反应
3. 无反应立即呼叫急救
4. 同时检查呼吸和脉搏

### 生命体征检查

| 生命体征 | 检查方法 | 正常范围 |
|---------|---------|---------|
| 意识 | 呼唤和反应 | 清醒、对答 |
| 呼吸 | 看胸廓起伏、听呼吸声 | 12-20次/分 |
| 脉搏 | 触摸颈动脉或桡动脉 | 60-100次/分 |
| 体温 | 体温计或手背感受 | 36.3-37.2℃ |
| 血压 | 血压计测量 | <120/80mmHg |

## 心肺复苏

### 成人CPR步骤

1. **确认安全**：确保现场安全
2. **判断意识**：轻拍重唤
3. **呼叫急救**：拨打120
4. **胸外按压**：
   - 按压位置：两乳头连线中点
   - 深度：5-6厘米
   - 频率：100-120次/分
   - 按压与放松比：1:1

5. **开放气道**：仰头抬颏法
6. **人工呼吸**：
   - 捏住鼻子，口对口吹气
   - 每次吹气1秒
   - 看到胸廓隆起
   - 按压与通气比：30:2

### AED使用

自动体外除颤器（AED）是心脏骤停抢救的重要设备。

| 步骤 | 操作 | 说明 |
|-----|------|------|
| 1 | 打开AED电源 | 语音提示启动 |
| 2 | 贴电极片 | 按图示位置粘贴 |
| 3 | 分析心律 | 不要接触伤者 |
| 4 | 建议除颤 | 确保无人接触 |
| 5 | 按下电击按钮 | 按照提示操作 |
| 6 | 继续CPR | 立即恢复按压 |

## 创伤急救

### 出血控制

#### 直接压迫止血法

- 用干净纱布或布料直接按压伤口
- 持续用力按压至少10分钟
- 如血液渗透，在上方加新的纱布
- 不要移除被血液浸透的纱布

#### 止血带使用

| 适用部位 | 上止血带位置 | 注意事项 |
|---------|------------|---------|
| 手臂 | 上臂中上1/3 | 记录上止血带时间 |
| 腿部 | 大腿中上1/3 | 每隔30-60分钟松开一次 |
| 手指 | 手指根部 | 不超过15分钟 |

### 骨折处理

#### 骨折判断

| 症状 | 说明 |
|-----|------|
| 疼痛 | 局部剧烈疼痛 |
| 肿胀 | 受伤部位肿胀 |
| 畸形 | 肢体外观异常 |
| 功能障碍 | 无法正常活动 |
| 骨擦音 | 移动时有摩擦感 |

#### 固定方法

- 使用夹板（木板、杂志、硬纸板）
- 固定骨折上下两个关节
- 绑扎不要过紧，保持血液循环
- 检查远端指/趾端颜色和温度

### 烧伤处理

| 烧伤程度 | 表现 | 处理方法 |
|---------|------|---------|
| 一度 | 红肿、疼痛 | 冷水冲洗10-15分钟 |
| 二度 | 水泡、剧痛 | 冷水冲洗，不挑破水泡 |
| 三度 | 焦黑、无痛 | 覆盖保护，立即就医 |

#### 急救原则

- 冲：冷水冲洗降温
- 脱：去除覆盖的衣物
- 泡：冷水浸泡
- 盖：无菌纱布覆盖
- 送：送医治疗

## 常见急症

### 中暑

| 症状 | 处理方法 |
|-----|---------|
| 头晕、恶心、大量出汗 | 移至阴凉通风处 |
| 体温升高、皮肤潮红 | 冷敷、补充水分 |
| 意识模糊、抽搐 | 立即就医 |
| 热射病（体温>40℃） | 紧急送医 |

### 窒息

海姆立克急救法（Heimlich Maneuver）：

- 站在患者背后
- 一手握拳放在肚脐上方
- 另一手抓住拳头，向内向上用力
- 反复操作直到异物排出

### 心脏急症

| 症状 | 处理方法 |
|-----|---------|
| 胸痛、胸闷 | 立即停止活动，坐下休息 |
| 放射痛（左臂、下颌） | 含服硝酸甘油（如有医嘱） |
| 呼吸困难 | 半坐卧位 |
| 大汗淋漓 | 呼叫急救 |

### 癫痫发作

- 保护患者，移开周围危险物品
- 不要强行按住患者
- 不要往嘴里塞东西
- 记录发作时间
- 发作后侧卧位，保持呼吸道通畅

### 过敏反应

| 严重程度 | 症状 | 处理 |
|---------|------|------|
| 轻度 | 皮疹、瘙痒 | 口服抗组胺药 |
| 中度 | 面部肿胀、呼吸困难 | 立即就医 |
| 重度 | 过敏性休克 | 注射肾上腺素 |

## 中毒急救

### 立即措施

1. 确定中毒物质
2. 拨打急救电话
3. 保留毒物样本
4. 按照急救中心指导处理
5. 不要盲目催吐

### 常见中毒

| 中毒类型 | 症状 | 急救措施 |
|---------|------|---------|
| 食物中毒 | 呕吐、腹泻 | 补充水分，就医 |
| 药物过量 | 意识改变 | 保留药瓶，就医 |
| 一氧化碳中毒 | 头晕、昏迷 | 立即到通风处，吸氧 |
| 酒精中毒 | 呕吐、昏迷 | 侧卧位，防止误吸 |

## 急救物品准备

### 家庭急救包

| 类别 | 物品 | 数量 |
|-----|------|------|
| 包扎用品 | 创可贴、纱布、绷带 | 各若干 |
| 消毒用品 | 碘伏、酒精棉片 | 各一盒 |
| 工具 | 剪刀、镊子、手套 | 各一 |
| 药品 | 止痛药、抗过敏药 | 适量 |
| 其他 | 急救手册、口罩 | 各一 |

### 急救包维护

- 每月检查有效期
- 用完及时补充
- 放在固定位置
- 家庭成员都知晓位置

## 急救误区

| 错误做法 | 正确做法 | 原因 |
|---------|---------|------|
| 烫伤涂牙膏 | 冷水冲洗 | 牙膏阻碍散热 |
| 扭伤热敷 | 冷敷 | 热敷加重肿胀 |
| 流鼻血仰头 | 身体前倾，捏鼻翼 | 仰头导致血液倒流 |
| 伤口涂紫药水 | 碘伏消毒 | 紫药水影响观察 |
| 鱼刺喝醋 | 去医院取出 | 醋不能软化鱼刺 |
