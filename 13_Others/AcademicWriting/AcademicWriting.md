# 学术写作

## 一、学术写作特征

### 1.1 核心特征

| 特征 | 说明 | 示例 |
|------|------|------|
| **正式语气** | 避免口语化表达、缩略词 | "It is evident that" → 而非 "It's clear" |
| **精确性** | 使用准确的术语和数据 | "A 15% increase" → 而非 "a big increase" |
| **客观性** | 避免主观评价和情感语言 | "The results indicate..." → 而非 "I think..." |
| **谨慎性 (Hedging)** | 使用缓和语言表达结论 | "This suggests that..." → 而非 "This proves that..." |
| **引用规范** | 标注文献来源，避免抄袭 | 使用 APA/MLA/Chicago 等格式 |

### 1.2 常见 hedging 表达

| 类型 | 表达 |
|------|------|
| 情态动词 | may, might, could, would |
| 频率副词 | often, sometimes, typically |
| 程度副词 | somewhat, relatively, to some extent |
| 认知动词 | suggest, indicate, appear, seem |
| 限定从句 | It is possible that... / It could be argued that... |

## 二、写作流程

### 2.1 五步流程

| 阶段 | 目标 | 关键活动 |
|------|------|----------|
| **规划 (Planning)** | 明确主题、目标读者、结构 | 头脑风暴、大纲、研究问题 |
| **起草 (Drafting)** | 将想法转化为文字 | 关注内容而非完美，写完初稿 |
| **修改 (Revising)** | 优化内容结构和逻辑 | 调整论点、增加证据、重组段落 |
| **编辑 (Editing)** | 润色语言和格式 | 检查语法、词汇、标点、引用格式 |
| **校对 (Proofreading)** | 最终检查细节错误 | 逐句通读，关注拼写和格式 |

### 2.2 大纲示例

```
标题
├── 引言（背景→研究问题→目的→结构预览）
├── 文献综述（主题1→主题2→研究空白）
├── 方法论（设计→样本→工具→分析）
├── 结果（按研究问题组织）
├── 讨论（解释结果→对比文献→局限）
└── 结论（总结→贡献→未来方向）
```

## 三、段落结构

### 3.1 段落要素

| 要素 | 作用 | 位置 |
|------|------|------|
| **主题句 (Topic Sentence)** | 概括段落核心观点 | 段落开头 |
| **支撑句 (Supporting Sentences)** | 提供证据、例子、数据 | 主题句之后 |
| **总结句 (Concluding Sentence)** | 总结段落，过渡到下一段 | 段落末尾 |

### 3.2 连贯与衔接 (Coherence & Cohesion)

**衔接手段：**

| 类型 | 示例 |
|------|------|
| **过渡词** | however, therefore, moreover, consequently |
| **指代** | this finding, these results, the former |
| **重复关键词** | 重复核心术语保持主题一致 |
| **平行结构** | 使用相同的语法结构表达并列内容 |

## 四、论证结构

### 4.1 图尔敏模型 (Toulmin Model)

| 要素 | 说明 | 示例 |
|------|------|----------|
| **主张 (Claim)** | 要证明的观点 | "社交媒体对青少年心理健康有负面影响" |
| **证据 (Evidence)** | 支持主张的事实/数据 | "研究表明每天使用3小时以上的青少年抑郁率高出30%" |
| **理由 (Warrant)** | 连接证据和主张的逻辑 | "长时间接触社交比较会降低自尊心" |
| **反驳 (Counterargument)** | 反对观点 | "有人认为社交媒体也能提供社交支持" |
| **回应 (Rebuttal)** | 对反驳的回应 | "但社交支持效应远小于负面比较效应" |

### 4.2 论证质量评估

| 标准 | 问题 |
|------|------|
| 相关性 | 证据是否直接支持主张？ |
| 充分性 | 证据是否足够？ |
| 可信度 | 证据来源是否可靠？ |
| 时效性 | 数据是否过时？ |
| 客观性 | 是否考虑了反方观点？ |

## 五、改写与总结

### 5.1 改写技巧 (Paraphrasing)

| 技巧 | 原句 | 改写后 |
|------|------|--------|
| **同义词替换** | "The study examined..." | "The research investigated..." |
| **语法重构** | "Although X is common, Y is rare." | "Y is rare despite the prevalence of X." |
| **词性转换** | "The analysis was conducted..." | "The analysis of data involved..." |
| **合并拆分** | 合并两个简单句 | 使用从句合并为一个复合句 |

### 5.2 总结技巧 (Summarizing)

- 识别原文的主旨和关键论点
- 省略细节、例子和重复内容
- 用自己的语言重新组织
- 保持原文的逻辑结构
- 始终标注原始出处

### 5.3 避免抄袭

| 做法 | 说明 |
|------|------|
| 正确引用 | 直接引用用引号+页码 |
| 适当改写 | 不能只是换词，要重构句子 |
| 记录来源 | 记笔记时同步记录文献信息 |
| 使用管理工具 | Zotero / EndNote / Mendeley |

## 六、学术词汇

### 6.1 正式替代词

| 非正式 | 正式 |
|--------|------|
| use | utilize, employ |
| do | perform, conduct, implement |
| show | demonstrate, illustrate, indicate |
| get | obtain, acquire, attain |
| look at | examine, analyze, investigate |
| make | produce, generate, construct |
| change | modify, alter, transform |
| help | facilitate, assist, enable |
| big | substantial, significant, considerable |
| find | discover, identify, reveal |

### 6.2 学术常用句式

| 功能 | 句式 |
|------|------|
| 定义 | "X refers to..." / "X is defined as..." |
| 分类 | "X can be categorized into..." / "There are three types of X..." |
| 对比 | "Whereas X..., Y..." / "In contrast to X, Y..." |
| 因果 | "X leads to Y..." / "Y results from X..." |
| 让步 | "Although X..., it is important to note that..." |

## 七、常见语法问题

### 7.1 冠词用法

| 规则 | 示例 |
|------|------|
| 可数名词单数前加冠词 | "The study found..." / "A significant factor..." |
| 泛指用不定冠词 | "A researcher must consider..." |
| 特指用定冠词 | "The methodology used in this study..." |
| 抽象名词通常不加冠词 | "Knowledge is power." / "Research is essential." |

### 7.2 主谓一致

- 主语和谓语在人称和数上保持一致
- 集合名词（team, group, committee）：英式英语可单可复，美式通常用单数
- 插入语不改变主谓一致性："The results, along with the analysis, **show**..."
- 不定代词（each, everyone, nobody）后接单数动词

### 7.3 时态使用

| 时态 | 使用场景 | 示例 |
|------|----------|------|
| **一般现在时** | 普遍事实、当前理论 | "Evolution occurs through natural selection." |
| **一般过去时** | 已完成的特定研究 | "Smith (2020) found that..." |
| **现在完成时** | 近期研究、持续影响 | "Several studies have examined..." |
| **现在完成进行时** | 仍在进行的研究 | "Researchers have been investigating..." |
| **将来时** | 未来研究方向 | "Future research will explore..." |

## 八、学术写作体裁

### 8.1 主要体裁对比

| 体裁 | 长度 | 目的 | 结构特点 |
|------|------|------|----------|
| **Essay** | 1,000-5,000字 | 论证一个观点 | 引言→主体→结论 |
| **Research Paper** | 5,000-10,000字 | 报告原创研究 | IMRaD（引言→方法→结果→讨论） |
| **Literature Review** | 3,000-8,000字 | 综合已有研究 | 主题组织 → 研究空白 |
| **Thesis/Dissertation** | 20,000-80,000字 | 展示原创学术贡献 | 多章节，完整研究 |
| **Abstract** | 150-250字 | 概括全文 | 背景→目的→方法→结果→结论 |
| **Conference Paper** | 3,000-6,000字 | 会议交流 | 类似研究论文，更聚焦 |

### 8.2 IMRaD 结构详解

| 部分 | 内容 | 常见问题 |
|------|------|----------|
| **Introduction** | 背景→研究空白→研究问题/假设→目的 | 背景过长，研究问题不明确 |
| **Method** | 设计→样本→工具→程序→分析 | 细节不足，无法复现 |
| **Results** | 客观呈现发现 | 不要在此讨论或解释 |
| **Discussion** | 解释结果→对比文献→→局限→意义 | 过度推断，忽略局限性 |

## 九、文献综述写作

### 9.1 综合 vs 总结

| 对比项 | 总结 | 综合 |
|--------|------|------|
| 结构 | 逐篇介绍研究 | 按主题/观点组织 |
| 分析 | 复述研究发现 | 比较、对比、批判 |
| 价值 | 展示知悉文献 | 构建新见解 |
| 逻辑 | "Smith found... then Jones found..." | "关于X有两种观点：A支持...而B反对..." |

### 9.2 主题组织方法

1. **时间线组织**：按研究发展历史组织
2. **主题分类**：按不同研究方向/主题组织
3. **方法论分类**：按研究方法区分（定量vs定性）
4. **理论框架分类**：按不同理论视角组织

### 9.3 识别研究空白

- 矛盾的研究发现
- 未被研究的人群/情境
- 过时的研究需要更新
- 方法论局限需要改进
- 理论框架需要检验

## 十一、学术诚信与引用

### 11.1 引用规范

| 格式 | 文内引用 | 参考文献 |
|------|----------|----------|
| **APA 7th** | (Smith, 2020, p. 15) | Smith, J. (2020). *Title*. Publisher. |
| **MLA 9th** | (Smith 15) | Smith, John. *Title*. Publisher, 2020. |
| **Chicago** | Smith 2020, 15 | Smith, John. *Title*. Publisher, 2020. |

### 11.2 学术不端类型

| 类型 | 定义 |
|------|------|
| 抄袭 (Plagiarism) | 未经注明使用他人成果 |
| 捏造 (Fabrication) | 编造数据或结果 |
| 篡改 (Falsification) | 修改或选择性报告数据 |
| 一稿多投 | 同一稿件同时投多个期刊 |
| 不当署名 | 未贡献者署名或贡献者未署名 |

## 十二、写作发表

### 12.1 投稿信 (Cover Letter)

**结构：**
1. 编辑姓名和期刊名称
2. 论文标题和投稿类型
3. 研究重要性和创新点（2-3句）
4. 声明：未一稿多投、所有作者同意
5. 推荐审稿人（可选）
6. 联系方式

### 12.2 回应审稿人

| 原则 | 做法 |
|------|------|
| 尊重所有意见 | 即使不同意也要礼貌回复 |
| 逐条回复 | 编号列出每一条意见及回应 |
| 区分处理 | 接受的修改说明做法；不接受的解释理由 |
| 标注变更 | 指出修改在文中的位置（行号/页码） |
| 及时回复 | 通常在2-4周内完成修改 |

### 12.3 常见退稿原因

- 研究问题不新颖（最重要原因）
- 方法论有缺陷
- 分析不充分
- 结论缺乏数据支持
- 语言/写作质量差
- 不符合期刊范围
