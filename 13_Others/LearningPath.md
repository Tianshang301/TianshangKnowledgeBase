# 生活技能学习路线图

## 一、紧急优先级 (Immediate)

### 1. 个人财务管理
**目标：** 不再月光，建立财务安全网

**步骤：**
1. **记账 3 个月**：了解自己的开支模式
   - 工具：MoneyWiz、YNAB、鲨鱼记账、Excel
2. **建立预算**：50/30/20 法则
   - 50% 必要开支（房租、食物、交通）
   - 30% 想要（娱乐、旅行、购物）
   - 20% 储蓄与投资
3. **建立应急基金**：3-6 个月生活费
4. **还清高息债务**：信用卡、花呗 > 消费贷 > 房贷
5. **入门投资**：指数基金定投（沪深300、标普500）

**书籍推荐：**
- 《富爸爸穷爸爸》- Robert Kiyosaki
- 《小狗钱钱》- 博多·舍费尔（理财启蒙）
- 《穷查理宝典》- Charlie Munger

### 2. 急救技能
**必须掌握：**
- **心肺复苏 (CPR)**：胸外按压 100-120 次/分，深度 5-6cm
- **海姆立克急救法**：气道异物阻塞的急救
- **止血**：直接压迫法、止血带使用
- **烧伤处理**：冷水冲洗 15-20 分钟
- **扭伤 RICE 原则**：Rest（休息）、Ice（冰敷）、Compression（加压）、Elevation（抬高）

**资源：**
- 红十字会急救培训课程
- YouTube：红会官方急救视频

### 3. 烹饪基础
**必备技能：**
- 煮饭（掌握水量）、煮面（不粘连）
- 炒青菜、煎鸡蛋、煮汤
- 调味基础：盐、酱油、醋、糖的比例感
- 刀工：切丝、切片、切丁

**入门菜谱：**
- 番茄炒蛋、可乐鸡翅、清炒时蔬、紫菜蛋花汤
- 一荤一素一汤 = 日常够用

**每周备餐 (Meal Prep)：**
- 周末花 2 小时准备好一周的食材
- 节省平时做饭时间

### 4. 时间管理

**核心方法：**
- **番茄工作法**：25 分钟专注 + 5 分钟休息
- **GTD (Getting Things Done)**：收集 → 整理 → 执行 → 回顾
- **艾森豪威尔矩阵**：重要/紧急四象限
- **2 分钟法则**：2 分钟内能做完的事立刻做

**工具：**
- Todoist / Microsoft To Do（任务管理）
- Google Calendar（日程安排）
- Notion（综合管理）

## 二、职业发展 (Career)

### 1. 简历写作

**原则：**
- **一页原则**：除非有 10+ 年经验
- **STAR 法则**：Situation, Task, Action, Result
- **量化成果**：用数字说话（"提升效率 30%" > "提升效率"）
- **针对岗位定制**：不要一份简历投所有公司

**模板结构：**
```
姓名 | 联系方式
教育背景
工作经历（用 STAR 描述）
项目经验
技能清单
```

### 2. 面试准备

**行为面试 (Behavioral)：**
- 准备 5-6 个 STAR 故事
- 常见问题：最大的失败、冲突处理、领导力经历

**技术面试：**
- 刷 LeetCode（软件工程师）
- 系统设计（高级职位）
- 案例分析（咨询/产品经理）

**薪资谈判：**
- 不要先出价
- 做市场调研（Glassdoor、Levels.fyi）
- 谈总包：底薪 + 奖金 + 股票 + 福利

### 3. 沟通与表达能力

- **金字塔原理**：先说结论，再列论据
- **非暴力沟通 (NVC)**：观察 → 感受 → 需要 → 请求
- **公共演讲**：TED 演讲准备方法
- **书面沟通**：简洁、清晰、有逻辑

## 三、长期技能 (Long-term)

### 1. 投资理财进阶

| 投资品 | 风险 | 适合人群 |
|--------|------|---------|
| 指数基金 | 低-中 | 初学者 |
| 个股 | 中-高 | 有研究能力者 |
| 债券 | 低 | 保守型 |
| 房地产 | 中 | 有资金者 |
| 加密货币 | 极高 | 高风险承受者 |

**定投策略：**
- 每月固定金额买入指数基金
- 市场跌时多买，涨时少买
- 长期持有（5 年以上）

### 2. 法律基础知识

**核心领域：**
- **劳动法**：合同、试用期、加班、赔偿、仲裁
- **租房法**：押金、合同、维修责任
- **消费者权益**：退货、维权、投诉渠道
- **知识产权**：版权、商标、专利基础

**资源：**
- 当地法律援助中心
- 《民法典》通俗解读

### 3. 家庭维护技能

- **电路基础**：换灯泡、排查跳闸、换开关
- **水管基础**：通下水道、换水龙头垫圈
- **墙面修补**：钉孔填充、裂缝修补
- **家具组装**：看懂说明书、使用电钻

### 4. 健康管理

- **睡眠**：7-9 小时、规律作息、睡前不玩手机
- **运动**：每周 150 分钟中等强度有氧 + 力量训练
- **饮食**：均衡营养、减少糖和加工食品
- **心理**：冥想/正念、定期与朋友交流
