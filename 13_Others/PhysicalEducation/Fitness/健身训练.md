# 健身训练

## 概述

健身训练是改善身体形态、增强体能、促进健康的重要途径。科学的健身方法能够帮助训练者高效达成目标，同时避免运动损伤。本章系统介绍训练原则、训练计划设计、常见训练方法和安全注意事项。

## 训练原则

### 渐进超负荷

身体只有在承受超过习惯的负荷时才会产生适应。逐步增加训练强度是持续进步的关键。

### 专项性

训练效果具有专项性，训练内容应与目标一致。想增加力量就做大重量训练，想增肌就做中等重量训练。

### 可逆性

停止训练后，训练效果会逐渐消退。维持训练成果需要持续的训练刺激。

### 个体差异

每个人对训练的反应不同，训练计划需要根据个人情况调整。

## 训练计划设计

### 目标设定

| 目标 | 训练重点 | 典型训练方案 |
|-----|---------|------------|
| 增肌 | 肌肉肥大 | 6-12次，3-4组 |
| 力量 | 最大力量 | 1-5次，3-5组 |
| 耐力 | 肌肉耐力 | 15-25次，2-3组 |
| 减脂 | 热量消耗 | 循环训练+有氧 |

### 训练频率

| 训练水平 | 每周天数 | 每次时长 |
|---------|---------|---------|
| 初级（<3个月） | 2-3天 | 45-60分钟 |
| 中级（3-12个月） | 3-4天 | 60-75分钟 |
| 高级（>1年） | 4-6天 | 60-90分钟 |

### 分化方案

#### 全身训练

适合初学者，每次训练所有主要肌群。

| 训练日 | 动作安排 |
|-------|---------|
| 周一 | 深蹲+卧推+划船+推举+硬拉 |
| 周三 | 同周一（增加重量） |
| 周五 | 同周一（增加重量） |

#### 上下肢分化

| 训练日 | 内容 |
|-------|------|
| 周一 | 上肢（推+拉） |
| 周二 | 下肢+核心 |
| 周四 | 上肢（推+拉） |
| 周五 | 下肢+核心 |

#### 推拉腿分化

| 训练日 | 内容 |
|-------|------|
| 周一 | 推（胸、肩、三头） |
| 周二 | 拉（背、二头） |
| 周三 | 腿（股四、腘绳、臀） |
| 周五 | 推 |
| 周六 | 拉 |
| 周日 | 腿 |

## 基本动作

### 复合动作

复合动作是多关节参与的动作，效率最高。

| 动作 | 主要肌群 | 训练建议 |
|-----|---------|---------|
| 深蹲 | 股四、臀大肌 | 每周1-2次 |
| 硬拉 | 全身 | 每周1次 |
| 卧推 | 胸、三头 | 每周2次 |
| 划船 | 背、二头 | 每周2次 |
| 推举 | 肩、三头 | 每周1-2次 |
| 引体向上 | 背、二头 | 每周2次 |

### 孤立动作

孤立动作针对单个肌群，用于补充和塑形。

| 动作 | 目标肌群 | 组数×次数 |
|-----|---------|----------|
| 二头弯举 | 肱二头肌 | 3×10-15 |
| 三头下压 | 肱三头肌 | 3×10-15 |
| 腿弯举 | 腘绳肌 | 3×12-15 |
| 提踵 | 小腿 | 3×15-20 |
| 侧平举 | 三角肌中束 | 3×12-15 |

## 训练方法

### 增肌训练

#### 训练参数

- 次数范围：6-12次
- 组数：每个动作3-4组
- 组间休息：60-90秒
- 每周每个肌群：10-20组

#### 高级技术

| 技术 | 方法 | 效果 |
|-----|------|------|
| 递减组 | 逐组减重连续做 | 增加代谢压力 |
| 超级组 | 两个动作连续做 | 节省时间 |
| 暂停组 | 每组最后加停顿 | 增加张力时间 |
| 强制次数 | 辅助完成最后1-2次 | 突破极限 |
| 离心训练 | 控制离心放慢 | 增加肌肉损伤 |

### 力量训练

#### 训练参数

- 次数范围：1-5次
- 组数：3-5组
- 组间休息：3-5分钟
- 重在动作质量

### 减脂训练

#### 训练策略

| 策略 | 方法 | 说明 |
|-----|------|------|
| 力量训练 | 保持肌肉 | 维持代谢率 |
| 有氧训练 | 增加消耗 | 每周3-5次 |
| HIIT | 高效燃脂 | 每周2-3次 |
| 日常活动 | 增加NEAT | 多走路、爬楼梯 |

## 热身与冷却

### 热身

#### 一般热身

- 5-10分钟低强度有氧
- 提高心率和体温
- 增加关节活动度

#### 专项热身

- 目标肌群动态拉伸
- 轻重量预热组
- 激活目标肌群
- 熟悉动作模式

### 冷却

- 5-10分钟低强度活动
- 静态拉伸主要肌群
- 泡沫轴放松
- 补充营养和水分

## 训练记录

### 记录内容

| 项目 | 记录方法 | 作用 |
|-----|---------|------|
| 动作 | 名称+变式 | 确认技术 |
| 重量 | 实际使用重量 | 渐进参考 |
| 次数 | 完成的次数 | 评估强度 |
| 组数 | 完成的组数 | 容量统计 |
| 感受 | RPE或RIR | 调整负荷 |

### 渐进方法

- 下次训练增加2.5-5kg重量
- 或增加1-2次/组
- 或增加1组/动作
- 或减少休息时间

## 安全与预防

### 动作技术安全

| 动作 | 安全要点 |
|-----|---------|
| 深蹲 | 膝盖对脚尖，背部挺直 |
| 硬拉 | 杠铃贴近身体，不弓背 |
| 卧推 | 肩胛骨收紧，控制下落 |
| 推举 | 核心收紧，不后仰 |
| 划船 | 保持背部紧张 |

### 常见损伤预防

- 充分热身
- 循序渐进
- 注意技术
- 合理休息
- 均衡发展

### 训练平台期突破

| 方法 | 说明 |
|-----|------|
| 调整训练计划 | 改变分化、动作顺序 |
| 增加训练强度 | 提高重量或密度 |
| 改变训练方法 | 尝试新技术 |
| 检查恢复情况 | 营养、睡眠 |
| 适当减量 | 减量后冲击新强度 |
