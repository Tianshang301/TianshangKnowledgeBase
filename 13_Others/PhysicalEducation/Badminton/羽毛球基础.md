# 羽毛球基础

## 一、握拍技术

### 1.1 基本握拍

| 握拍方式 | 适用 | 要点 |
|----------|------|------|
| 正手握拍（Forehand Grip） | 正手高远球、杀球、抽球 | 虎口对准拍柄窄面，拇指食指呈V形，掌心留空 |
| 反手握拍（Backhand Grip） | 反手高远球、反手抽球 | 拇指顶在拍柄宽面上，食指稍向前靠 |
| 网前握拍 | 搓球、勾球、放网 | 握在拍柄前端，拇指和食指轻夹 |
| 钳式握拍 | 平抽球 | 拇指和食指像钳子夹住拍柄 |

## 二、基本击球技术

### 2.1 后场技术

| 技术 | 英文 | 用途 | 要点 |
|------|------|------|------|
| 高远球 | Clear | 将球打到对方底线 | 侧身引拍，肘先于手，手腕内旋发力，击球点在最高点 |
| 平高球 | Offensive Clear | 快速平高到对方底线 | 击球点靠前，手腕发力快 |
| 吊球 | Drop Shot | 假装高远球，轻切球托落网前 | 动作与高远球一致，拍面斜切球托减速 |
| 杀球 | Smash | 全力进攻扣杀 | 击球点更靠前，手腕闪动发力，拍面下压 |
| 点杀 | Half Smash | 快速小幅下压 | 动作小，手腕发力，落点尖锐 |
| 跳杀 | Jump Smash | 全力扣杀，起跳增加高度 | 起跳后在空中完成击球，落地后快速回位 |

### 2.2 中场技术

| 技术 | 英文 | 用途 | 要点 |
|------|------|------|------|
| 平抽球 | Drive | 中场快速对抽 | 手腕小幅度发力，拍面控制高度和方向 |
| 挡网 | Block | 防守放网 | 卸力缓冲，球贴网下落 |
| 接杀 | Defense | 接对方杀球 | 重心低，举拍快，反手握拍接球 |

### 2.3 网前技术

| 技术 | 英文 | 要点 |
|------|------|------|
| 搓球 | Net Shot | 拍面斜切球托底部，使球旋转过网 |
| 勾对角 | Cross Court Net | 手腕翻转，拍面斜向切球，将对角线网前 |
| 推球 | Push | 快速推向对方后场，动作小，手腕发力 |
| 挑球 | Lift | 将网前低球挑到对方后场底线 |
| 扑球 | Net Kill | 高点下压扑杀，对方网前高球时使用 |
| 放网 | Net Drop | 轻挡过网，球贴网下落 |

## 三、步法

### 3.1 基本步法

| 步法 | 英文 | 使用场景 | 动作描述 |
|------|------|----------|----------|
| 分腿垫步 | Split Step | 对手击球瞬间起动 | 双脚同时离地小跳，落地后立刻向球的方向移动 |
| 交叉步 | Cross Step | 后退/横向长距离移动 | 左脚从右脚前交叉后退，右脚交叉前进 |
| 并步/滑步 | Chasse Step | 横向短距离 | 左（右）脚向右（左）脚并拢后跨出 |
| 跨步 | Lunge | 上网接球 | 同侧脚向目标方向跨大步，脚跟先着地 |
| 后退步 | Back Step | 退后场接球 | 转身后交叉步后退，保持身体面向球 |
| 垫步 | Adjustment Step | 调整击球位置 | 小碎步微调，找到最佳击球点 |

### 3.2 步法模式

- **后退步法**：分腿垫步+转身+交叉步2-3步+跨步击球
- **前进步法**：分腿垫步+并步/交叉步+跨步+搓/挑
- **两点移动**：从中心到两点往返（正手网前+反手后场等）
- **米字步法**：从中心到8个方向（前左中右，后左中右，两侧）

## 四、场上位置与策略

### 4.1 单打

| 要素 | 内容 |
|------|------|
| 中心位置 | 场地正中央稍靠后 |
| 基本策略 | 四方球拉开对手，寻找机会突击 |
| 进攻 | 杀球+上网封网，连续下压 |
| 防守 | 防守到对方进攻弱侧，挑球到后场争取时间 |

### 4.2 双打

| 要素 | 内容 |
|------|------|
| 进攻站位 | 前后站位：一人网前封网，一人后场进攻 |
| 防守站位 | 左右站位/平行站位：各守半场 |
| 轮转 | 根据球的位置前后⇔左右切换 |
| 核心策略 | 发接发前三拍争抢，多打结合部 |

## 五、计分与规则

| 规则项 | 内容 |
|--------|------|
| 计分 | 每球得分制（Rally Point），21分一局 |
| 局数 | 三局两胜 |
| 平分 | 20:20需领先2分，最多30分（30:29结束） |
| 发球规则 | 偶数分右半区发球，奇数分左半区发球 |
| 合法发球 | 击球点低于腰部，拍框指向下，双脚不移动 |
| 单打场地 | 13.4m×5.18m |
| 双打场地 | 13.4m×6.10m |
| 网高 | 网柱处1.55m，中间1.524m |

## 六、器材选择

| 参数 | 说明 | 建议 |
|------|------|------|
| 球拍重量（U） | 数字越大越轻，4U≈80-84g | 力量大选3U/4U，双打选4U/5U |
| 平衡点 | 头重=进攻，头轻=防守，均衡=全面 | 进攻选手选头重 |
| 中杆硬度 | 硬=弹性好需发力，软=省力 | 高手选硬，初学者选软 |
| 磅数（lbs） | 线张力，越高控制越好 | 初学者22-24，中级24-27，高手27+ |
| 球速 | 76慢速~78快速 | 夏季选76，冬季选78 |

## 七、战术进阶

### 7.1 单打战术

| 战术 | 方法 | 适用对手 |
|------|------|----------|
| 四方球 | 打四个角消耗对手 | 体能较差者 |
| 压后场突击 | 反复底线高远球+突击 | 步伐慢者 |
| 拉吊结合 | 拉吊+突然杀球 | 防守型 |
| 控网抢攻 | 网前控制→创造杀球机会 | 移动慢者 |
| 压反手 | 持续攻击反手位 | 反手弱者 |

### 7.2 双打战术

| 战术 | 方法 | 说明 |
|------|------|------|
| 前后轮转 | 一人网前封网+一人后场进攻 | 进攻状态 |
| 平挡防守 | 两人各守半场 | 防守状态 |
| 发接发抢攻 | 发球后第3拍直接进攻 | 前3拍是双打关键 |
| 打中间结合部 | 打两人中间位置 | 制造混乱 |
| 追身球 | 快速打向对方身体 | 减少反应时间 |

## 八、体能训练

| 训练项目 | 方法 | 目的 |
|----------|------|------|
| 折返跑 | 半场折返跑×10组 | 提升横向移动速度 |
| 跳绳 | 双摇100次×3组 | 提升脚踝力量和协调性 |
| 弓步跳 | 每侧10次×3组 | 提升上网爆发力 |
| 核心 | 平板支撑1分钟×3组 | 稳定身体 |
| 手腕力量 | 小哑铃绕腕 | 提升手腕爆发力 |

## 九、比赛心理

| 场景 | 策略 |
|------|------|
| 关键分（20平） | 专注每一分，不要想结果 |
| 连续失误 | 暂停节奏，深呼吸，重新集中 |
| 对手状态好 | 改变节奏，打不同路线 |
| 自己状态差 | 减少失误，打安全球，等机会 |
| 裁判误判 | 不要纠结，专注下一分 |

## 十、常见技术错误

| 技术 | 错误 | 修正 |
|------|------|------|
| 高远球 | 击球点太低 | 手臂完全伸直时击球 |
| 杀球 | 拍面不正 | 手腕内旋到位 |
| 网前 | 手腕太硬 | 放松手腕，手指发力 |
| 步法 | 启动太慢 | 分腿垫步+压低重心 |
| 反手 | 拇指位置不对 | 拇指顶在宽面上 |
