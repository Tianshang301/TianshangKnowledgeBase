# 瑜伽基础

## 概述

瑜伽是一种结合身体、呼吸和心灵的古老练习方式，起源于古印度。现代瑜伽已成为全球流行的健身和身心修行方式。本章系统介绍瑜伽历史与哲学、基础呼吸技巧、经典体式、练习原则和健康益处。

## 瑜伽历史与哲学

### 什么是瑜伽

瑜伽源自梵文词根yuj，意为连接或合一，指身、心、灵的统合。瑜伽通过体式、呼吸控制和冥想等方法，帮助练习者达到身心平衡和自我认知。

### 瑜伽的发展

| 时期 | 里程碑 | 贡献 |
|-----|-------|------|
| 远古 | 印度河流域文明 | 最早的瑜伽形象 |
| 古典 | 瑜伽经 | 系统化瑜伽理论 |
| 中世纪 | 哈他瑜伽 | 体式和净化法 |
| 近代 | 传入西方 | 全球普及 |
| 现代 | 多种流派 | 适应不同需求 |

### 瑜伽哲学核心

#### 瑜伽经的教导

- 瑜伽是控制心念的波动
- 通过持续练习和不执着达到稳定
- 八支是修行的完整路径

#### 非暴力（Ahimsa）

- 对他人非暴力
- 对自己非暴力（不强迫身体）
- 对所有生命尊重

## 呼吸技巧

### 基本呼吸原则

- 鼻呼吸（非嘴呼吸）
- 深长均匀
- 呼吸引导动作
- 保持呼吸流畅

### 基础呼吸练习

#### 腹式呼吸

| 要素 | 说明 |
|-----|------|
| 方法 | 吸气时腹部鼓起，呼气时腹部收回 |
| 作用 | 深度放松，增加肺活量 |
| 练习 | 仰卧，手放腹部感受起伏 |

#### 胸式呼吸

- 吸气时胸腔扩张
- 呼气时胸腔收缩
- 腹部保持平稳
- 适用于需要能量提升的练习

#### 完全呼吸

结合腹式呼吸和胸式呼吸：

1. 吸气：先腹部鼓起到胸部扩张
2. 呼气：先胸部收缩到腹部收回
3. 循环：连续流畅

### 调息法

#### 经络调息

| 步骤 | 操作 |
|-----|------|
| 1 | 右手大拇指关闭右鼻孔 |
| 2 | 左鼻孔吸气 |
| 3 | 无名指关闭左鼻孔 |
| 4 | 右鼻孔呼气 |
| 5 | 右鼻孔吸气 |
| 6 | 大拇指关闭右鼻孔 |
| 7 | 左鼻孔呼气 |

#### 圣光调息

- 快速有力的呼气
- 自然的被动吸气
- 腹部主动收缩和放松
- 从慢速开始，逐步加快

## 基础体式

### 站立体式

#### 山式

所有站立体式的基础：

- 双脚并拢或与髋同宽
- 体重均匀分布在双脚
- 大腿肌肉收紧
- 腹部微收，胸腔展开
- 肩膀放松下沉
- 手臂自然垂于身体两侧

#### 前屈式

- 从髋部向前折叠
- 膝盖可微屈
- 双手触地或抓脚踝
- 放松头部和颈部

#### 战士式系列

| 体式 | 功效 | 要点 |
|-----|------|------|
| 战士一 | 强化腿部和核心 | 前腿90度，后腿伸直 |
| 战士二 | 打开髋部和胸腔 | 前腿90度，手臂与腿平行 |
| 战士三 | 平衡和力量 | 单腿站立，身体水平 |

### 跪姿体式

#### 猫牛式

- 四足跪姿
- 吸气：塌腰抬头（牛式）
- 呼气：拱背低头（猫式）
- 配合呼吸，活动脊柱

#### 婴儿式

休息体式，在练习中随时可做：

- 跪姿，臀部坐于脚跟
- 身体前屈，额头触地
- 手臂前伸或放体侧
- 轻柔呼吸，放松全身

### 俯卧体式

#### 眼镜蛇式

- 俯卧，双手在肩膀下方
- 吸气：抬起头、胸
- 手肘微屈，肩膀下沉
- 保持耻骨贴地

#### 蝗虫式

- 俯卧，手臂在身体两侧
- 吸气：同时抬起头部、胸部和双腿
- 手臂向后伸展
- 保持呼吸，收紧背部

#### 上犬式

- 俯卧，双手在肩膀下方
- 手臂伸直，髋部离地
- 脚背贴地
- 胸腔打开，头部后仰（颈部不适可不后仰）

### 仰卧体式

#### 桥式

| 步骤 | 操作 |
|-----|------|
| 1 | 仰卧，膝盖弯曲，脚掌踩地 |
| 2 | 吸气，臀部向上抬起 |
| 3 | 双手在身体下方交握 |
| 4 | 保持肩膀和头部贴地 |
| 5 | 呼气，慢慢放下 |

#### 仰卧扭脊

- 仰卧，膝盖弯向一侧
- 手臂向两侧展开
- 头部转向反方向
- 保持肩部贴地

#### 挺尸式

最后的放松体式：

- 仰卧，双手双脚自然分开
- 全身完全放松
- 呼吸自然
- 保持意识清醒
- 停留5-10分钟

### 倒置体式

#### 下犬式

过渡和休息体式：

- 四足跪姿，手与肩同宽
- 脚趾踩地，臀部上提
- 伸直双腿，脚跟踩地
- 头部在手臂之间
- 保持脊柱拉长

#### 犁式

- 仰卧，双腿上举过头
- 脚趾触地
- 手支撑背部
- 保护颈部

## 练习原则

### 练习注意事项

| 原则 | 说明 |
|-----|------|
| 空腹练习 | 饭后至少2小时再练习 |
| 不强迫 | 尊重身体极限 |
| 配合呼吸 | 每个动作与呼吸协调 |
| 专注当下 | 关注身体感受 |
| 规律练习 | 比单次时长更重要 |

### 练习序列

#### 晨间序列（15分钟）

1. 呼吸练习（2分钟）
2. 猫牛式（1分钟）
3. 下犬式（2分钟）
4. 战士一（2分钟/侧）
5. 三角式（2分钟/侧）
6. 前屈式（2分钟）
7. 婴儿式（2分钟）
8. 挺尸式（2分钟）

#### 晚间放松序列（20分钟）

1. 腹式呼吸（3分钟）
2. 仰卧扭脊（3分钟/侧）
3. 桥式（3分钟）
4. 腿靠墙（5分钟）
5. 挺尸式（5分钟）
