# 瑜伽流派与体式精讲

## 概述

瑜伽起源于5000多年前的古印度，梵语词根yuj意为连接或合一，指身、心、灵的统合。经过数千年的发展，瑜伽形成了多种流派，每种流派有其独特的理念和方法。本章系统介绍瑜伽的起源与发展、主要流派和经典体式精讲。

## 瑜伽的起源与发展

### 历史脉络

| 时期 | 发展 | 代表作/人物 |
|-----|------|-----------|
| 前2500年 | 印度河流域文明 | 印章上的坐姿形象 |
| 前1500年 | 吠陀时期 | 吠陀经典 |
| 前500年 | 奥义书时期 | 奥义书 |
| 前200年 | 经典确立 | 瑜伽经（帕坦伽利） |
| 500-1500年 | 密教瑜伽 | 哈他瑜伽发展 |
| 1500-1900年 | 近代发展 | 各种流派形成 |
| 1900年至今 | 全球传播 | 现代瑜伽 |

### 瑜伽的八支

帕坦伽利在瑜伽经中提出了瑜伽的八支，是瑜伽修行的完整路径：

| 分支 | 含义 | 实践内容 |
|-----|------|---------|
| Yama | 持戒 | 非暴力、诚实、不偷盗 |
| Niyama | 精进 | 纯净、知足、自律 |
| Asana | 体式 | 身体姿势 |
| Pranayama | 呼吸控制 | 呼吸练习 |
| Pratyahara | 感官收摄 | 内省 |
| Dharana | 专注 | 集中注意力 |
| Dhyana | 冥想 | 深度冥想 |
| Samadhi | 三摩地 | 最终解脱 |

## 主要流派

### 哈他瑜伽

哈他瑜伽是大多数现代瑜伽流派的基础，强调体式和呼吸的练习。

| 特点 | 说明 |
|-----|------|
| 节奏 | 缓慢、温和 |
| 适合人群 | 初学者、所有级别 |
| 练习内容 | 体式、呼吸、放松 |
| 核心理念 | 平衡阴阳能量 |

### 阿斯汤加瑜伽

阿斯汤加是一种动态、力量型的瑜伽流派，体式按固定序列练习。

- 六个序列，难度递增
- 每个动作与呼吸配合（Vinyasa）
- 注重力量、柔韧和耐力
- 适合有基础的练习者

### 流瑜伽

流瑜伽强调动作与呼吸的流畅连接。

| 特点 | 说明 |
|-----|------|
| 节奏 | 中等至快速 |
| 衔接 | 动作之间流畅连接 |
| 变化 | 序列可灵活编排 |
| 强度 | 中等至高 |

### 艾扬格瑜伽

艾扬格瑜伽强调体式的精准和正位，大量使用辅助工具。

- 精确对齐身体各部位
- 使用砖、带、毯子等工具
- 适合康复和伤后恢复
- 注重细节和持久保持

### 阴瑜伽

阴瑜伽是慢节奏的瑜伽风格，体式保持时间较长，针对深层结缔组织。

| 特点 | 说明 |
|-----|------|
| 保持时间 | 3-5分钟甚至更长 |
| 目标组织 | 关节、韧带、筋膜 |
| 练习状态 | 肌肉放松，靠重力 |
| 适合人群 | 所有级别，尤其适合紧张者 |

### 热瑜伽

在高温环境（约40℃）中练习固定的26个体式和2个呼吸练习。

- 高温促进排毒和柔韧性
- 固定序列，结构明确
- 体力消耗大
- 需要适当补充水分

### 昆达里尼瑜伽

昆达里尼瑜伽结合体式、呼吸、唱诵和冥想，旨在唤醒生命能量。

- 快速动态体式
- 呼吸练习（调息）
- 唱诵和冥想
- 注重能量觉醒

## 经典体式精讲

### 站立体式

#### 山式

| 要素 | 要求 |
|-----|------|
| 梵文名 | Tadasana |
| 功效 | 改善体态，增强平衡 |
| 要点 | 双脚并拢，大腿收紧，脊柱延展 |

#### 战士一式

- 前腿弯曲呈90度
- 后腿伸直，脚跟压地
- 手臂上举，掌心相对
- 髋部摆正朝前

#### 三角式

| 要素 | 要求 |
|-----|------|
| 梵文名 | Utthita Trikonasana |
| 功效 | 拉伸侧腰，改善消化 |
| 要点 | 前腿伸直，身体侧屈，手臂垂直 |

### 坐立体式

#### 简易坐

- 盘腿而坐
- 脊柱挺直
- 双手放在膝盖上
- 用于冥想和呼吸练习

#### 英雄坐

- 双膝跪地，臀部坐于双脚之间
- 脊柱挺直
- 双手放于大腿
- 有助于消化

#### 坐姿前屈

| 要素 | 要求 |
|-----|------|
| 梵文名 | Paschimottanasana |
| 功效 | 拉伸后腿和背部 |
| 要点 | 从髋部前屈，脊柱拉长 |

### 前屈体式

#### 站立前屈

- 从髋部折叠身体
- 双腿伸直或微屈
- 双手触地或抓脚踝
- 放松头部和颈部

#### 加强侧伸展

- 前后腿分开
- 从前腿一侧前屈
- 双手在脚两侧或背后合掌
- 拉伸后腿和脊柱

### 后弯体式

#### 眼镜蛇式

| 要素 | 要求 |
|-----|------|
| 梵文名 | Bhujangasana |
| 功效 | 强健背部，打开胸腔 |
| 要点 | 手肘微屈，肩膀下沉，胸腔上提 |

#### 上犬式

- 俯卧，双手在肩下
- 手臂伸直，髋部离地
- 胸腔打开，头部后仰
- 脚背压地

#### 桥式

仰卧，双脚分开与髋同宽，膝盖弯曲，脚掌踩地，臀部向上抬起。

### 扭转体式

#### 坐姿脊柱扭转

- 坐姿，一条腿弯曲跨过另一条腿
- 身体向弯曲腿一侧扭转
- 一手在身后支撑
- 一手抵住膝盖

#### 仰卧脊柱扭转

- 仰卧，膝盖弯向一侧
- 手臂向两侧展开
- 头部转向反方向
- 放松脊柱

### 倒立体式

#### 下犬式

| 要素 | 要求 |
|-----|------|
| 梵文名 | Adho Mukha Svanasana |
| 功效 | 全身拉伸，恢复精力 |
| 要点 | 手脚压地，臀部上提，脊柱拉长 |

#### 肩倒立

- 仰卧，双腿上举
- 手支撑背部
- 身体垂直于地面
- 保护颈部

#### 头倒立

- 前臂撑地，手抱头
- 头顶触地
- 身体垂直上举
- 需在专业指导下练习

### 修复体式

#### 婴儿式

- 跪姿，臀部坐于脚后跟
- 身体前屈，额头触地
- 手臂前伸或放在身体两侧
- 休息和放松

#### 鱼式

- 仰卧，手肘支撑
- 胸腔上提，头顶触地
- 打开喉咙和胸腔

#### 挺尸式

| 要素 | 要求 |
|-----|------|
| 梵文名 | Savasana |
| 功效 | 深度放松 |
| 要点 | 仰卧，全身放松，意识清醒 |

## 体式练习安全

### 练习原则

- 尊重身体，不强迫
- 从基础体式开始
- 配合呼吸
- 保持专注
- 循序渐进

### 常见注意事项

- 饭后2小时再练习
- 有伤病时咨询医生
- 保持空腹或轻食
- 穿着舒适的衣服
- 使用瑜伽垫
