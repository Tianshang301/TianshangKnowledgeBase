# 篮球基础

## 概述

篮球是一项广受欢迎的球类运动，融合了速度、力量、技巧和团队合作。掌握篮球的基本技术是享受这项运动和进一步提升的基础。本章系统介绍基本技术、战术基础、训练方法和比赛规则等内容。

## 基本技术

### 运球

运球是控制球权、创造进攻机会的基础技术。

#### 基本运球

| 运球类型 | 方法 | 用途 |
|---------|------|------|
| 高运球 | 手在腰部以上运球 | 快速推进 |
| 低运球 | 手在膝盖以下运球 | 贴身防守时保护球 |
| 变向运球 | 突然改变方向 | 过掉防守者 |
| 背后运球 | 球从身后换手 | 保护球的同时变向 |
| 胯下运球 | 球从胯下换手 | 低重心变向 |

#### 运球要点

- 用手指而非手掌控球
- 保持低重心
- 抬头观察场上局势
- 非运球手保护球
- 控制运球节奏

### 传球

传球是团队进攻的核心，好的传球能创造得分机会。

| 传球方式 | 适用情况 | 要点 |
|---------|---------|------|
| 胸前传球 | 近距离安全传球 | 双手推球 |
| 击地传球 | 穿越防守 | 着地点在防守者身后 |
| 头上传球 | 越过防守 | 适用于高个子 |
| 背后传球 | 隐蔽传球 | 适合近距离 |
| 长传 | 快攻转换 | 力度和准确度 |

#### 传球原则

- 传球及时，不延误战机
- 传球准确，到接球人舒服的位置
- 传球隐蔽，不给防守预判机会
- 传球有力，快速到位

### 投篮

投篮是得分的最终手段，是篮球技术中最重要的环节。

#### 投篮姿势

1. 双脚与肩同宽，膝盖微屈
2. 投篮手托球底部，辅助手在侧面
3. 眼睛瞄准篮筐前沿
4. 从腿部发力，经身体传到手臂
5. 手腕柔和拨球，球后旋

#### 投篮类型

| 类型 | 距离 | 使用场景 |
|-----|------|---------|
| 罚球 | 4.5米 | 犯规后的罚球 |
| 跳投 | 中远距离 | 跳起出手 |
| 上篮 | 近距离 | 突破到篮下 |
| 勾手 | 近距离 | 内线得分 |
| 三分球 | 三分线外 | 远距离得分 |

### 上篮

#### 三步上篮

1. 第一步：接球或收球，迈出右脚（右手上篮）
2. 第二步：左脚起跳
3. 第三步：空中出手

#### 上篮变化

| 变化 | 说明 | 适用 |
|-----|------|------|
| 高手上篮 | 高举球出手 | 位置开放 |
| 低手上篮 | 低手上挑 | 躲避封盖 |
| 反手上篮 | 从篮板另一侧出手 | 穿过篮下 |
| 拉杆上篮 | 空中收球再出手 | 躲避封盖 |

### 篮板球

#### 卡位技术

- 防守篮板：挡住对手，占据有利位置
- 进攻篮板：绕过防守，冲抢位置

#### 抢板要领

- 判断球的落点
- 占据有利位置
- 双手高举抢球
- 落地后保护球

### 防守基本技术

#### 防守姿势

- 双脚比肩宽，膝盖弯曲
- 背部挺直
- 一手干扰传球
- 手臂张开增加防守面积

#### 滑步

- 向两侧移动时使用
- 脚步不交叉
- 保持低重心
- 快速调整位置

## 基本规则

### 比赛规则概要

| 规则 | 说明 |
|-----|------|
| 比赛时长 | 国际篮球：4节×10分钟；NBA：4节×12分钟 |
| 球队人数 | 每队5人在场，最多12人报名 |
| 得分 | 三分线内2分，三分线外3分，罚球1分 |
| 进攻时间 | 24秒内必须出手 |
| 走步 | 持球时中轴脚移动 |
| 两次运球 | 运球后停止又再运球 |

### 常见犯规

| 犯规类型 | 说明 | 判罚 |
|---------|------|------|
| 打手 | 防守时打到进攻者手臂 | 罚球或边线球 |
| 阻挡 | 防守者未站好位置阻挡 | 防守犯规 |
| 带球撞人 | 进攻者撞到已站好位的防守者 | 进攻犯规 |
| 违体犯规 | 危险动作 | 两罚一掷 |
| 技术犯规 | 不当行为 | 一罚一掷 |

## 战术基础

### 个人战术

| 战术 | 应用 | 要点 |
|-----|------|------|
| 无球跑动 | 创造接球机会 | 变速、变向 |
| 掩护 | 为队友制造机会 | 选定位置、站稳 |
| 突破分球 | 吸引防守找空位 | 阅读防守 |
| 假动作 | 迷惑防守者 | 逼真快速 |

### 基础配合

- **传切配合**：传球后切入篮下
- **挡拆配合**：掩护后拆开
- **突分配合**：突破后分球给空位
- **策应配合**：内线接球后分球

## 训练方法

### 基础训练计划

| 训练项目 | 频率 | 时长 |
|---------|------|------|
| 运球练习 | 每天 | 20分钟 |
| 投篮练习 | 每天 | 30分钟 |
| 传球练习 | 每周3次 | 15分钟 |
| 防守练习 | 每周3次 | 20分钟 |
| 体能训练 | 每周2次 | 30分钟 |

### 投篮训练方法

- 定点投篮：从不同位置投篮
- 移动投篮：接球后立即出手
- 对抗投篮：在防守中投篮
- 疲劳投篮：模拟比赛状态

### 球感训练

- 手指拨球
- 绕身体传球
- 背后接球
- 胯下运球
- 墙壁传球练习

## 体能训练

| 要素 | 训练方法 | 目的 |
|-----|---------|------|
| 速度 | 短距离冲刺 | 快攻和防守 |
| 弹跳 | 跳绳、深蹲跳 | 抢板和投篮 |
| 耐力 | 折返跑、间歇跑 | 全场奔跑 |
| 力量 | 核心和下肢 | 对抗能力 |
| 灵敏 | 变向跑、反应训练 | 快速反应 |

## 常见错误

| 技术 | 常见错误 | 纠正方法 |
|-----|---------|---------|
| 运球 | 低头运球、拍球太响 | 抬头练习、放松手腕 |
| 投篮 | 手肘外展、发力不连贯 | 对墙练习、腿部发力 |
| 传球 | 力量不足、时机不当 | 对墙练习、观察时机 |
| 防守 | 重心太高、脚步慢 | 保持低姿、滑步练习 |
| 上篮 | 步法乱、起跳点不对 | 分步练习、找准起步点 |
