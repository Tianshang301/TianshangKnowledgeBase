# 游泳技术

## 概述

游泳是一项全身运动，对心肺功能和身体素质的提升有显著效果。掌握正确的游泳技术是享受游泳乐趣、提高运动效率的基础。本章系统介绍四种竞技泳姿的基本技术、呼吸方法、转身技巧和训练方法。

## 四种竞技泳姿

### 自由泳

自由泳是速度最快、最常用的泳姿，也称为爬泳。

#### 身体姿势

- 身体接近水平
- 肩部带动身体旋转
- 头部稳定，目视池底前方

#### 手臂动作

| 阶段 | 动作描述 | 要点 |
|-----|---------|------|
| 入水 | 手指先入水，在肩延长线 | 手臂伸直 |
| 抓水 | 手腕微弯，手掌朝后 | 高肘 |
| 划水 | 肘部弯曲，手臂向后划 | 逐渐加速 |
| 推水 | 手臂推至大腿旁 | 快速有力 |
| 移臂 | 肘部先出水，手臂前移 | 放松 |

#### 腿部动作

- 大腿带动小腿
- 上下交替，幅度约30-40cm
- 频率适中，与划臂配合

#### 呼吸

- 一侧转头呼吸
- 转头时耳朵贴水面
- 水下呼气，水上吸气
- 建议双侧呼吸

### 蛙泳

蛙泳是模仿青蛙游泳的姿势，技术细节较多。

#### 手臂动作

| 阶段 | 动作 | 要点 |
|-----|------|------|
| 外划 | 双手向外划水 | 肘部保持高位 |
| 内划 | 手臂向内夹水 | 加速划水 |
| 前伸 | 手臂前伸恢复 | 手掌并拢 |

#### 腿部动作

1. 收腿：脚跟向臀部收回
2. 翻脚：脚掌向外翻转
3. 蹬腿：向两侧后方蹬水
4. 夹水：双腿并拢伸直

#### 配合节奏

一次划臂 + 一次蹬腿 + 一次呼吸 + 滑行

| 阶段 | 动作 | 呼吸 |
|-----|------|------|
| 划臂 | 外划+内划 | 抬头吸气 |
| 收腿 | 脚跟靠近臀部 | 憋气 |
| 蹬腿 | 蹬夹腿 | 呼气 |
| 滑行 | 身体伸直 | 呼气 |

### 仰泳

仰泳是唯一面部朝上的泳姿，呼吸不受限制。

#### 身体姿势

- 仰卧水面
- 耳朵在水平面以下
- 髋部靠近水面
- 头部稳定

#### 手臂动作

- 入水：小指先入水
- 抓水：手腕弯曲
- 划水：弯臂向后划水
- 推水：推至大腿旁
- 移臂：拇指先出水面

#### 腿部动作

- 上下交替打水
- 幅度比自由泳稍大
- 膝盖不露出水面

### 蝶泳

蝶泳是技术难度最高的泳姿，需要良好的力量和协调性。

#### 身体波浪

- 胸部下压
- 髋部上提
- 腿部下压
- 全身波浪运动

#### 手臂动作

- 入水：双手同时入水
- 外划：向外抓水
- 内划：加速向内向大腿划水
- 推水：向后推至大腿
- 移臂：从两侧向前摆臂

#### 呼吸

- 推水时抬头吸气
- 移臂时低头入水
- 每两次划臂一次呼吸

## 转身技术

### 自由泳转身

#### 前滚翻转身

1. 接近池壁时加速
2. 低头含胸，做前滚翻
3. 双脚触壁
4. 蹬壁滑行
5. 打水出水

#### 挥手转身

- 一手触壁
- 侧身转体
- 双脚触壁
- 蹬壁滑行

### 蛙泳转身

1. 双手同时触壁
2. 屈膝转向
3. 双脚触壁
4. 蹬壁滑行
5. 可做一次水下长划臂

### 仰泳转身

1. 接近池壁时身体转成俯卧
2. 做前滚翻
3. 双脚触壁
4. 仰卧蹬壁
5. 滑行出水

## 呼吸技术

### 呼吸原则

- 规律呼吸，不要憋气
- 水下呼气，水上吸气
- 呼吸与动作协调

### 各泳姿呼吸

| 泳姿 | 呼吸方式 | 呼吸频率 |
|-----|---------|---------|
| 自由泳 | 侧转头呼吸 | 每2-3次划臂 |
| 蛙泳 | 划臂时抬头 | 每次划臂 |
| 仰泳 | 自由呼吸 | 持续 |
| 蝶泳 | 推水时抬头 | 每2次划臂 |

## 训练方法

### 基础训练

| 训练项目 | 内容 | 作用 |
|---------|------|------|
| 打水训练 | 扶板打水 | 腿部力量和技术 |
| 划臂训练 | 夹板划臂 | 手臂力量和技术 |
| 呼吸训练 | 打水+呼吸 | 呼吸配合 |
| 分解训练 | 分动作练习 | 技术纠正 |
| 完整配合 | 全部动作 | 整体协调 |

### 训练计划

#### 初学者计划

- 适应水性：水中行走、呼吸练习（10分钟）
- 浮体练习：漂浮、站立（10分钟）
- 打水练习：扶板打水（10分钟）
- 划臂练习：夹板划臂（10分钟）
- 完整配合：短距离游（10分钟）

#### 进阶训练

| 训练内容 | 距离/时间 | 组数 |
|---------|----------|------|
| 热身 | 200m | 1组 |
| 技术训练 | 50m | 6组 |
| 主训练 | 100m | 8组 |
| 速度训练 | 25m | 10组 |
| 冷却 | 200m | 1组 |

### 常见错误纠正

| 泳姿 | 错误 | 纠正方法 |
|-----|------|---------|
| 自由泳 | 身体左右晃动 | 核心收紧，身体旋转带动 |
| 自由泳 | 呼吸时抬头过高 | 单侧转头，耳朵贴水面 |
| 蛙泳 | 蹬腿无效 | 翻脚做好，蹬夹加速 |
| 蛙泳 | 划臂过宽 | 肘部靠近身体 |
| 仰泳 | 臀部下沉 | 核心收紧，头部位置稳定 |
| 蝶泳 | 手臂入水太近 | 肩膀放松，手臂前伸 |

## 安全与卫生

### 游泳安全

- 不单独游泳
- 了解泳池水深
- 不做危险动作
- 注意体力分配
- 不饱食后游泳

### 泳池卫生

- 下水前淋浴
- 戴游泳帽
- 不随地吐痰
- 出水后及时擦干

## 装备选择

| 装备 | 用途 | 选择建议 |
|-----|------|---------|
| 泳镜 | 保护眼睛 | 防雾、贴合 |
| 泳帽 | 保护头发 | 硅胶材质 |
| 泳衣/裤 | 减少阻力 | 合身专业 |
| 打水板 | 腿部训练 | 浮力适中 |
| 夹板 | 手臂训练 | 大小合适 |
| 脚蹼 | 增强打水 | 短脚蹼 |
