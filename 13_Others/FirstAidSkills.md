# 急救技能

急救是在专业医疗人员到达前，对伤病者进行的紧急救护。掌握正确的急救技能可以在关键时刻挽救生命。每个人都应该学习基本的急救知识，因为意外随时可能发生在我们身边。

## CPR心肺复苏（Cardiopulmonary Resuscitation）

CPR是最基础也是最重要的急救技能，用于心脏骤停患者，通过人工方式维持血液循环和氧气供应。

### 心脏骤停的识别

- **无反应**：拍打双肩、大声呼唤无反应
- **无呼吸**：观察胸部无起伏，无正常呼吸（濒死喘息不算正常呼吸）
- **无脉搏**：专业人员可检查颈动脉（非专业人员可省略此步骤）

### CPR操作流程

```
1. 确认安全环境
2. 拍打呼唤，判断意识
3. 拨打120急救电话
4. 取AED（如有）
5. 开始胸外按压
6. 开放气道
7. 人工呼吸
8. 持续循环直到专业人员到达
```

### 胸外按压要点

| 要素 | 标准 |
|------|------|
| 按压位置 | 胸骨下半段（两乳头连线中点） |
| 按压深度 | 成人5-6厘米，儿童约5厘米，婴儿约4厘米 |
| 按压频率 | 100-120次/分钟 |
| 回弹要求 | 每次按压后胸廓完全回弹 |
| 按压通气比 | 30:2（30次按压后2次人工呼吸） |

### 人工呼吸

1. **仰头抬颏法**：一手按住额头，另一手抬起下巴，打开气道
2. **捏鼻吹气**：捏住鼻子，口对口密封吹气
3. **观察胸廓**：吹气时观察胸部是否隆起
4. **每次吹气1秒**：避免过度通气

### 特殊人群CPR

- **儿童（1-8岁）**：单手或双手按压，深度约5厘米
- **婴儿（<1岁）**：双指按压或双手环抱按压，深度约4厘米
- **孕妇**：按压位置稍高，考虑左侧倾斜体位

## AED使用（Automated External Defibrillator）

AED是自动体外除颤器，能够自动分析心律并在需要时电击除颤。AED设计为普通人也能安全使用。

### AED使用步骤

```
1. 打开AED电源
2. 按照图示贴好电极片
   - 右锁骨下方
   - 左腋下（心尖部）
3. AED自动分析心律
4. 如提示"建议电击"，确认无人接触患者
5. 按下电击按钮
6. 电击后立即继续CPR
7. 按照AED语音提示继续操作
```

### AED注意事项

- **电极片位置**：右锁骨下 + 左腋下，不能贴在湿皮肤上
- **金属首饰**：电极片不能贴在金属首饰或起搏器上
- **水环境**：将患者移出水面后再使用AED
- **儿童AED**：8岁以下使用儿童电极片或儿童模式
- **分析心律时**：不要触碰患者，确保无人接触

### AED获取途径

- 机场、火车站、地铁站等公共场所
- 学校、体育馆、商场
- 大型企业、政府机关
- 社区卫生服务中心
- 使用地图APP搜索"AED"查找最近设备

## 创伤处理（Trauma Care）

创伤是日常生活中最常见的急救情况，正确的处理可以减少感染、控制出血、促进愈合。

### 出血分类

| 类型 | 特征 | 危险程度 |
|------|------|---------|
| 动脉出血 | 鲜红色，喷射状 | 极高 |
| 静脉出血 | 暗红色，持续流出 | 高 |
| 毛细血管出血 | 渗血 | 低 |

### 止血方法

**直接压迫法**
1. 用干净的纱布或布料直接压住伤口
2. 持续施加压力至少10分钟
3. 如血液浸透，在上方再加一层，不要移除第一层
4. 抬高受伤部位（在没有骨折的情况下）

**止血带使用**
- 适用于四肢大出血且直接压迫无效时
- 绑在伤口上方5-10厘米处
- 记录上止血带的时间
- 每1-2小时松开一次（如有条件）
- 止血带是救命措施，不要犹豫使用

### 伤口处理

1. **清洁**：用清水或生理盐水冲洗伤口
2. **消毒**：使用碘伏或酒精消毒伤口周围
3. **覆盖**：使用无菌纱布覆盖
4. **固定**：用绷带或胶带固定

### 特殊创伤处理

- **穿透伤**：不要拔出异物，在周围固定
- **断肢**：用干净布包裹断肢，放入塑料袋，外层冰敷
- **烧烫伤**：冷水冲洗至少20分钟，不要涂抹牙膏等
- **头部受伤**：保持安静，观察意识变化

## 中毒急救（Poisoning First Aid）

中毒可通过口服、吸入、皮肤接触等途径发生，正确的急救措施能挽救生命。

### 口服中毒

- **意识清醒**：了解摄入物质和时间，拨打急救电话
- **不要催吐**：除非专业人员指示，不要自行催吐
- **腐蚀性物质**：不要催吐，给予少量清水或牛奶稀释
- **保留包装**：保留中毒物质的包装供医疗人员参考

### 吸入中毒

1. 将患者转移到空气新鲜处
2. 松开衣领，保持呼吸道通畅
3. 如无呼吸，立即进行CPR
4. 注意自身安全，避免在封闭空间内二次中毒

### 皮肤接触中毒

1. 脱去被污染的衣物
2. 用大量清水冲洗受污染皮肤至少15分钟
3. 不要用热水冲洗
4. 如有化学灼伤，持续冲洗并就医

### 常见中毒物质

| 物质 | 症状 | 急救要点 |
|------|------|---------|
| 一氧化碳 | 头痛、头晕、恶心 | 转移到通风处，吸氧 |
| 酒精 | 意识模糊、呕吐 | 侧卧位，防止窒息 |
| 药物过量 | 嗜睡、呼吸抑制 | 保持呼吸道通畅 |
| 食物中毒 | 呕吐、腹泻 | 补液，保留食物样本 |

## 海姆立克急救法（Heimlich Maneuver）

海姆立克急救法用于解除气道异物梗阻（FBAO），是应对窒息的关键技能。

### 气道梗阻的识别

- **完全梗阻**：无法说话、无法咳嗽、无法呼吸，双手抓住喉咙
- **部分梗阻**：能咳嗽但困难，呼吸有喘鸣音
- **面色发紫**：缺氧的表现

### 成人海姆立克法

```
1. 站在患者身后
2. 双臂环绕患者腰部
3. 一手握拳，拳眼对准肚脐上方两横指处
4. 另一手握住拳头
5. 快速向内向上冲击腹部
6. 重复直到异物排出或患者失去意识
```

### 婴儿气道梗阻急救

1. 将婴儿面朝下放在前臂上，头低脚高
2. 用手掌根部在两肩胛骨之间拍背5次
3. 翻转婴儿面朝上，用两指在胸骨下半部按压5次
4. 交替进行拍背和胸部按压
5. 如婴儿失去意识，立即开始CPR

### 自救方法

如果独自一人发生气道梗阻：
- 将腹部抵住椅背或桌角
- 快速向内向上冲击
- 重复直到异物排出

### 特殊人群

- **孕妇**：改为胸部按压，位置在胸骨中段
- **肥胖者**：同样使用胸部按压
- **意识丧失者**：立即开始CPR，每次打开气道时检查口腔有无异物

## 急救准备

### 家庭急救箱

1. 各种尺寸的创可贴
2. 无菌纱布和绷带
3. 医用胶带
4. 碘伏消毒液
5. 止痛退烧药
6. 抗过敏药
7. 止泻药
8. 体温计
9. 剪刀和镊子
10. 一次性手套
11. CPR面罩
12. 急救手册

### 急救知识更新

- 定期参加红十字会等机构的急救培训
- 每2年复训一次，更新急救证书
- 关注最新的急救指南更新
- 在家庭和工作场所进行急救演练

## 实践建议

1. 参加正规的急救培训课程并取得证书
2. 在家中、车内、办公室都备有急救包
3. 了解所在地区的急救电话和最近医院位置
4. 定期检查和补充急救箱内的物品
5. 教育家人和朋友基本的急救知识
