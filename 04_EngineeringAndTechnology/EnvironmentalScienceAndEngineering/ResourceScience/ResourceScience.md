# 资源科学

## 学科定义
资源科学是研究自然资源的形成、分布、评价、开发、利用与保护的科学，是自然科学、社会科学和工程技术交叉的综合性学科。其核心目标是实现资源的可持续利用，支撑人类社会经济发展。

## 核心内容

### 一、自然资源分类

#### 1.1 分类体系

| 分类依据 | 类别 | 典型实例 |
|:---|:---|:---|
| 再生性 | 可再生资源 | 太阳能、风能、生物质能 |
| | 非可再生资源 | 化石燃料、金属矿产 |
| 来源 | 生物资源 | 森林、渔业、野生生物 |
| | 非生物资源 | 矿物、水、土地 |
| 存在形式 | 耗竭性资源 | 石油、天然气、煤炭 |
| | 非耗竭性资源 | 太阳能、潮汐能 |
| 用途 | 能源资源 | 煤、石油、铀 |
| | 原料资源 | 铁矿石、铜矿、磷矿 |

**关键概念**：
- **存量（Stock）**：资源在地壳中的总存在量
- **流量（Flow）**：单位时间内的资源更新量（可再生资源）
- **资源基础（Resource Base）**：地壳中天然存在的全部资源量
- **储量（Reserve）**：在当前技术经济条件下可经济开采的部分

#### 1.2 可再生 vs 非可再生

| 特性 | 可再生资源 | 非可再生资源 |
|:---|:---|:---|
| 更新时间 | 人类时间尺度（年–百年） | 地质时间尺度（百万年） |
| 可持续利用 | 合理管理可永续 | 最终耗竭不可逆 |
| 典型例子 | 太阳能、木材、渔业 | 石油、铜矿、稀土 |
| 管理关键 | 最大可持续产量（MSY） | 延长生命周期、循环利用 |

### 二、资源评价方法

#### 2.1 储量估算

**McKelvey 框图**：从地质确定性（x轴）和经济效益（y轴）两个维度对资源进行分类：

| 经济可行性 | 已验证 | 推测存在 |
|:---|:---|:---|
| 经济可采 | **储量（Reserves）** | 推测储量 |
| 次经济 | 次经济资源 | 推测次经济资源 |
| 非经济 | 非经济资源 | 资源基础 |

**USGS 资源分类**：
- **已探明储量（Proved Reserves）**：地质信息充分，经济可采
- **概算储量（Probable Reserves）**：有一定地质依据，开采可能性高
- **可能储量（Possible Reserves）**：地质依据有限，尚不确定

**储量/产量比（R/P Ratio）**：

$$R/P = \frac{\text{剩余可采储量}}{\text{年产量}}$$

表示在当前产量水平下资源的可开采年限。全球石油R/P约50年，天然气约55年，煤炭约130年。

### 三、矿产资源经济学

#### 3.1 Hubbert 峰值理论
M.K. Hubbert（1956）提出，不可再生资源的产量随时间呈钟形曲线，峰值过后产量不可逆转地下降。

**逻辑斯蒂增长模型**：

$$P(t) = \frac{Q}{1 + ae^{-bt}}$$

$$Q(t) = \int_0^t P(\tau)d\tau$$

其中 $P(t)$ 为年产量，$Q$ 为最终可采资源量。

Hubbert成功预测了美国本土48州石油产量在1970年达到峰值。

#### 3.2 资源经济租金
- **稀缺租金**：资源稀缺性带来的超额利润
- **差异化租金**：优质资源（高品位、易开采）的额外价值
- **Hotelling 规则**：非可再生资源的净价格增长率应等于贴现率

$$\frac{\dot{p}}{p} = r$$

其中 $p$ 为资源价格，$r$ 为贴现率。

**能源投资回报率**（EROI）：
$$EROI = \frac{\text{产出能量}}{\text{投入能量}}$$

### 四、水资源

#### 4.1 水文循环与水资源分布
全球水资源总量约 13.86 亿 km³，其中：
- 海洋水：96.5%
- 淡水：2.5%（其中冰盖/冰川占68.7%，地下水占30.1%，地表淡水占1.2%）
- 可供人类直接利用的淡水资源不足总水量的 0.3%

#### 4.2 地表水与地下水

**地表水**：河流、湖泊、水库，受气候和季节影响显著。

**地下水**：
- 潜水层（无压含水层）：距地表最近，易受污染
- 承压含水层：介于隔水层之间，水质较好
- 地下水超采导致地面沉降、海水入侵

**水质量指标**：

| 指标 | 定义 | 标准限值（饮用水） |
|:---|:---|:---:|
| pH | 酸碱度 | 6.5–8.5 |
| TDS | 总溶解固体 | <1000 mg/L |
| BOD | 生化需氧量 | <3 mg/L |
| COD | 化学需氧量 | <10 mg/L |
| 硬度 | Ca²⁺+Mg²⁺含量 | <450 mg/L |
| 浊度 | 悬浮物 | <1 NTU |

#### 4.3 虚拟水贸易
虚拟水（Virtual Water）指生产商品和服务过程中消耗的水资源量。虚拟水贸易通过商品进出口实现了水资源在全球范围内的再分配：

- 生产1kg小麦：约 1,300 L 虚拟水
- 生产1kg牛肉：约 15,000 L 虚拟水
- 生产1件棉T恤：约 2,700 L 虚拟水

**水足迹计算**：
$$WF = WF_{blue} + WF_{green} + WF_{grey}$$

其中 $WF_{blue}$ 为蓝水（地表/地下水消耗），$WF_{green}$ 为绿水（降水消耗），$WF_{grey}$ 为灰水（稀释污染物）。

### 五、森林资源

#### 5.1 森林资源现状
全球森林面积约 40.6 亿公顷（FAO 2020），占陆地面积的 31%。每年净减少约 470 万公顷（2000–2010年），主要源于热带地区 deforestation。

**森林功能**：
- 碳汇：全球森林储存约 662 Gt 碳
- 生物多样性：承载地球80%的陆地生物多样性
- 水源涵养：调节水文循环，防止水土流失
- 木材供给：年采伐量约 30 亿 m³

#### 5.2 可持续森林管理
**最大可持续产量（Maximum Sustainable Yield, MSY）**：

$$MSY = \frac{rK}{4}$$

其中 $r$ 为增长率，$K$ 为环境承载力。

**REDD+ 机制**（联合国）：通过减少毁林和森林退化造成的碳排放，为发展中国家提供资金激励。

### 六、能源资源

#### 6.1 化石燃料

**煤炭**：
- 形成：古植物遗体经泥炭化→褐煤→烟煤→无烟煤
- 全球储量：约 1.07 万亿吨（R/P ~130年）
- 主要产区：中国、美国、印度、澳大利亚

**石油**：
- 形成：古海洋生物有机质在还原环境下经热成熟作用生成
- 全球储量：约 1.7 万亿桶（R/P ~50年）
- 主要产区：中东、北美、俄罗斯

**天然气**：
- 形成：与石油类似，但干酪根成熟度更高
- 全球储量：约 190 万亿 m³（R/P ~55年）
- 非常规天然气：页岩气（中美为主）、致密气、煤层气

#### 6.2 化石燃料形成条件对比

| 资源 | 原始有机物 | 成藏温度 | 埋藏时间 | 储层类型 |
|:---|:---|:---:|:---:|:---|
| 煤 | 陆生植物 | 100–150°C | 百万年 | 煤层 |
| 石油 | 浮游生物/藻类 | 60–120°C | 数百万–亿年 | 砂岩/碳酸盐岩 |
| 天然气 | 浮游生物/藻类 | >120°C | 更长时间 | 致密砂岩/页岩 |

### 七、可再生资源

#### 7.1 各类可再生资源特征

| 类型 | 能量来源 | 全球装机（2023） | 容量因子 | 主要局限 |
|:---|:---|:---:|:---:|:---|
| 太阳能 | 太阳辐射 | ~1,200 GW | 10–25% | 间歇性、占地 |
| 风能 | 大气运动 | ~900 GW | 20–45% | 间歇性、噪声 |
| 水能 | 水的势能 | ~1,400 GW | 30–60% | 生态影响、移民 |
| 地热能 | 地球内部热 | ~16 GW | 70–95% | 地域限制 |
| 生物质能 | 生物质化学能 | ~150 GW | 40–80% | 土地利用竞争 |

#### 7.2 可再生资源的间歇性挑战
可再生能源的核心挑战是供需不匹配。解决方案包括：
- **储能**：锂电池、抽水蓄能、压缩空气储能
- **智能电网**：需求侧响应、预测调度
- **多能互补**：风水互补、光热互补

### 八、关键材料与城市采矿

#### 8.1 关键材料（Critical Materials）
关键材料是对经济至关重要的、但供应风险高的材料。

| 材料 | 主要用途 | 主产国占比 | 回收率 |
|:---|:---|:---:|:---:|
| 稀土元素 | 永磁体、催化剂 | 中国 ~60% | <5% |
| 锂 | 电池 | 澳大利亚 ~50% | <1% |
| 钴 | 电池、高温合金 | 刚果（金） ~70% | ~30% |
| 铂族金属 | 催化剂、传感器 | 南非 ~70% | ~50% |
| 铟 | 透明导电膜 | 中国 ~80% | <1% |
| 镓 | 半导体 | 中国 ~95% | <1% |

#### 8.2 城市采矿
从电子废弃物、废旧电池、报废产品中回收有价金属的操作，称为城市采矿。

**城市矿产的经济性**：
- 1吨废旧手机含金量约 300g（是金矿品位的 30–40倍）
- 1吨废旧锂离子电池含锂约 3–8 kg、钴 50–150 kg

### 九、资源效率与循环经济

#### 9.1 资源效率指标

**材料流分析（Material Flow Analysis, MFA）**：追踪特定物质在经济社会中的流动路径，量化资源输入、存量和输出。

**全生命周期评价（Life Cycle Assessment, LCA）**：ISO 14040标准框架：
1. 目标和范围定义
2. 生命周期清单分析（LCI）
3. 生命周期影响评价（LCIA）
4. 结果解释

**常用影响评价指标**：
- 全球变暖潜值（GWP，kg CO₂-eq）
- 累积能源需求（CED，MJ）
- 水足迹（m³）
- 资源耗竭潜力（ADP）

#### 9.2 3R原则

| 原则 | 含义 | 实施方式 |
|:---|:---|:---|
| Reduce（减量化） | 减少资源消耗和废物产生 | 轻量化设计、提高效率 |
| Reuse（再利用） | 产品多次使用 | 共享经济、维修翻新 |
| Recycle（循环利用） | 废料回收再生 | 垃圾分类、再生材料 |

**循环经济模式框图**：Linear（开采-制造-废弃）→ Circular（闭环材料流）

### 十、可持续资源管理

#### 10.1 资源治理框架
- **强可持续性**：自然资本不可替代，维持自然资本存量不变
- **弱可持续性**：自然资本与人为资本可替代，总资本存量不减少
- **行星边界（Planetary Boundaries）**：Rockström等提出9个地球系统边界，其中生物多样性、氮循环、气候变化已被越界

#### 10.2 资源战略与政策
- **战略资源储备**：应对地缘政治风险和供应中断
- **资源诅咒理论**：资源丰富国家反而经济增长缓慢的悖论
- **ESG投资**：环境（E）、社会（S）、治理（G）因素纳入资源投资决策
- **碳中和与资源转型**：从化石燃料向可再生能源和关键材料的结构性转变
