# 机械设计

## 定义
机械设计是根据使用要求，对机械的工作原理、结构、运动方式、力和能量的传递方式、各零部件的材料和形状尺寸等进行构思、分析和计算的工程技术学科。

## 研究对象
- 通用机械零件（齿轮、轴、轴承、螺纹连接等）
- 机构分析与综合
- 机械系统总体设计
- 可靠性设计与优化设计

## 核心内容

### 机械零件设计准则

**强度准则**：
- 静强度：$\sigma \leq [\sigma] = \frac{\sigma_{lim}}{S}$
- 疲劳强度：考虑应力循环特性，利用疲劳极限图
- $\sigma_{-1}$（对称循环疲劳极限）、$\sigma_0$（脉动循环疲劳极限）

**S-N曲线**（疲劳寿命曲线）：
$$\sigma^m N = C$$

其中 $\sigma$ 为应力幅，$N$ 为循环次数，$m$ 为曲线斜率参数，$C$ 为材料常数。

**疲劳极限**：$\sigma_{-1}$ — 无限寿命下的最大应力（钢 $N>10^7$ 次）

**Goodman修正**（平均应力影响）：
$$\frac{\sigma_a}{\sigma_{-1}} + \frac{\sigma_m}{\sigma_b} = 1$$

其中 $\sigma_a$ 为应力幅，$\sigma_m$ 为平均应力，$\sigma_b$ 为抗拉强度。

**刚度准则**：$y \leq [y]$，挠度不超过许用值

**耐磨性准则**：$p \leq [p]$（压强）、$pv \leq [pv]$（ pv 值）

**稳定性准则**：受压细长杆件满足压杆稳定条件

### 齿轮传动设计

**渐开线齿轮基本参数**：模数 $m$、齿数 $z$、压力角 $\alpha = 20°$

**几何尺寸**：
- 分度圆直径：$d = mz$
- 齿顶高：$h_a = h_a^* m = m$
- 齿根高：$h_f = (h_a^* + c^*)m = 1.25m$
- 中心距：$a = \frac{m(z_1+z_2)}{2}$

**传动比**：
$$i = \frac{n_1}{n_2} = \frac{z_2}{z_1} = \frac{d_2}{d_1}$$

其中 $n_1,n_2$ 为输入输出转速，$z_1,z_2$ 为齿数。

**齿轮传动类型对比**：

| 类型 | 传动比范围 | 效率 | 特点 | 应用 |
|------|-----------|------|------|------|
| 直齿圆柱齿轮 | $3\sim5$ | 0.96~0.99 | 结构简单，有冲击 | 低速传动 |
| 斜齿圆柱齿轮 | $3\sim6$ | 0.96~0.99 | 运转平稳，承载高 | 高速传动 |
| 锥齿轮 | $2\sim4$ | 0.94~0.97 | 相交轴传动 | 差速器 |
| 蜗轮蜗杆 | $8\sim80$ | 0.40~0.90 | 大减速比，自锁 | 起重设备 |
| 行星齿轮 | $3\sim12$ | 0.97~0.99 | 紧凑，功率分流 | 汽车变速器 |

**齿面接触疲劳强度**：

$$\sigma_H = Z_H Z_E Z_\varepsilon \sqrt{\frac{2KT_1}{bd_1^2}\cdot\frac{u+1}{u}} \leq [\sigma_H]$$

其中 $Z_H$ 为节点区域系数，$Z_E$ 为弹性影响系数，$u$ 为齿数比。

**齿根弯曲疲劳强度**：

$$\sigma_F = \frac{2KT_1}{bd_1 m} Y_{Fa} Y_{Sa} Y_\varepsilon \leq [\sigma_F]$$

其中 $Y_{Fa}$ 为齿形系数，$Y_{Sa}$ 为应力修正系数。

### 轴的设计

**轴的分类**：心轴（只受弯矩）、传动轴（只受扭矩）、转轴（弯扭组合）

**强度计算**：
- 弯扭组合：$\sigma_{eq} = \sqrt{\sigma_b^2 + 4\tau^2} \leq [\sigma]$
- 第三强度理论：$\sigma_{r3} = \sqrt{\sigma^2+4\tau^2}$
- 第四强度理论：$\sigma_{r4} = \sqrt{\sigma^2+3\tau^2}$

**轴的结构设计**：
- 阶梯轴便于装配定位
- 轴肩、轴环、套筒、圆螺母定位
- 键槽、花键连接传递扭矩

### 轴承选型

**滚动轴承**：
- 基本额定动载荷 $C$：$L_{10} = (C/P)^p$（球轴承 $p=3$，滚子轴承 $p=10/3$）
- 基本额定静载荷 $C_0$
- 当量动载荷：$P = f_p(XF_r + YF_a)$
- 寿命计算：$L_h = \frac{10^6}{60n}\left(\frac{C}{P}\right)^p$

**滑动轴承**：
- 液体动压润滑：雷诺方程
- 液体静压润滑：外部供油
- 设计参数：$p = \frac{F}{Bd}$，$pv = \frac{F}{BL}$

**滚动轴承类型对比**：

| 类型 | 承受载荷 | 轴向限位 | 调心性 | 极限转速 | 应用 |
|------|---------|---------|-------|---------|------|
| 深沟球轴承 | 径向+双向轴向 | 双向 | 差 | 高 | 电机、减速器 |
| 角接触球轴承 | 径向+单向轴向 | 单向 | 差 | 高 | 机床主轴 |
| 圆柱滚子轴承 | 大径向 | 少 | 差 | 较高 | 大功率齿轮箱 |
| 圆锥滚子轴承 | 径向+单向轴向 | 单向 | 差 | 中 | 车轮轮毂 |
| 调心滚子轴承 | 大径向+轴向 | 双向 | 好 | 低 | 矿山机械 |
| 推力球轴承 | 单向轴向 | — | 差 | 低 | 立式转轴 |

### 螺纹连接设计

**螺纹类型**：三角形螺纹（连接）、梯形螺纹（传动）、锯齿形螺纹（单向传力）

**受力分析**：
- 承受轴向载荷：$F_a = F_{pre} + \Delta F$（被连接件分流）
- 被连接件刚度系数 $C_b = \frac{k_b}{k_b+k_m}$

**强度条件**：
- 拉伸强度：$\sigma = \frac{1.3F}{\frac{\pi}{4}d_1^2} \leq [\sigma]$
- 疲劳强度：需校核应力幅

### 带传动设计

**带传动基本公式**：
- 传动比：$i = n_1/n_2 = d_2/d_1$
- 带速：$v = \frac{\pi d_1 n_1}{60 \times 1000}$ (m/s)

**欧拉公式**（带传动极限拉力）：
$$\frac{F_1}{F_2} = e^{\mu\alpha}$$

其中 $F_1,F_2$ 为紧边/松边拉力，$\mu$ 为摩擦系数，$\alpha$ 为小轮包角。

**有效拉力**：$F_e = F_1 - F_2 = \frac{1000P}{v}$（$P$ 为功率kW）

**V带传动设计功率**：$P_{ca} = K_A P$

### 键连接设计

**平键强度条件**：
$$\sigma_p = \frac{2T}{dkl} \leq [\sigma_p]$$

其中 $T$ 为传递扭矩，$d$ 为轴径，$k$ 为键高，$l$ 为键的工作长度。

**花键连接**：
- 齿面挤压强度：$\sigma_p = \frac{2T}{\psi z h l d_m} \leq [\sigma_p]$
- $z$ 为齿数，$h$ 为齿工作高度，$d_m$ 为平均直径

### 联轴器与离合器

**联轴器**：
- 刚性联轴器：凸缘联轴器
- 弹性联轴器：弹性柱销联轴器、膜片联轴器
- 选型依据：扭矩、转速、对中性

**离合器**：
- 牙嵌式离合器
- 摩擦离合器（干式/湿式）
- 电磁离合器

### 弹簧设计

**圆柱螺旋压缩弹簧**：
- 刚度：$k = \frac{Gd^4}{8D^3n}$
- 切应力：$\tau = K\frac{8FD}{\pi d^3}$
- 变形：$f = \frac{8FD^3n}{Gd^4}$

**弹簧材料**：碳素弹簧钢丝（65Mn）、合金弹簧钢（60Si2Mn）、不锈钢丝
