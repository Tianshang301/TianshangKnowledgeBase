# 电力系统

## 定义
电力系统是由发电厂、变电所、输配电线路和用电设备组成的整体，实现电能的生产、传输、分配和使用。

## 研究对象
- 电力系统的组成与运行
- 电力系统稳定性分析
- 电力系统保护与控制
- 智能电网与新能源并网

## 核心内容

### 电力系统组成

**发电**：将一次能源转换为电能
- 火力发电（煤、气、油）：约占60%
- 水力发电：约占16%
- 核能发电：约占10%
- 风力/太阳能发电：快速增长

**输电**：高压输电减少线路损耗
- 输电电压等级：110kV、220kV、330kV、500kV、750kV、1000kV
- 特高压：1000kV交流、±800kV直流

**变电**：升压/降压变压器
- 发电厂升压：10~20kV → 110~500kV
- 用户降压：110kV → 10kV → 380V/220V

**配电**：将电能分配到各用户
- 高压配电：10kV、35kV
- 低压配电：380V/220V

**用电**：各类电力负荷

### 电力系统稳定性

**功角稳定性**：
- 静态稳定：系统受到小扰动后能恢复同步运行
- 暂态稳定：系统受到大扰动后能恢复同步运行
- 功角特性：$P = \frac{E'V}{X_d'}\sin\delta$

**电压稳定性**：
- 负荷节点电压维持能力
- P-V曲线、Q-V曲线分析
- 电压崩溃现象

**频率稳定性**：
- 有功功率平衡：$\Delta P = P_G - P_L$
- 调频特性：一次调频（调速器）、二次调频（调频器）
- 频率偏差：$\Delta f = -\frac{\Delta P}{K}$（$K$ 为系统频率调节效应系数）

### 变压器原理与应用

**工作原理**：电磁感应定律

$$\frac{V_1}{V_2} = \frac{N_1}{N_2}, \quad \frac{I_1}{I_2} = \frac{N_2}{N_1}$$

**等效电路**：
- 理想变压器 + 励磁阻抗 + 漏阻抗
- 标幺值表示简化计算

**效率**：

$$\eta = \frac{P_2}{P_2 + P_{Cu} + P_{Fe}} \times 100\%$$

铜损：$P_{Cu} = I^2R$，铁损：$P_{Fe} = \frac{V^2}{R_m}$

### 电力系统保护（继电保护）

**保护原则**：
- 选择性：只切除故障元件
- 速动性：快速切除故障
- 灵敏性：能反映保护范围内的故障
- 可靠性：不拒动、不误动

**主要保护**：
- 线路保护：距离保护、差动保护、零序保护
- 变压器保护：瓦斯保护、差动保护、过流保护
- 母线保护：母线差动保护

### 智能电网

**特征**：
- 自愈能力：故障自动检测、隔离、恢复
- 用户参与：需求响应、分布式发电
- 安全性：抵御网络攻击和自然灾害
- 兼容性：接纳多种发电形式

**关键技术**：
- 高级量测体系（AMI）
- 高级配电自动化
- 储能技术
- 信息通信技术（ICT）

### 新能源并网技术

**光伏并网**：
- 并网逆变器控制（电流型/电压型）
- 最大功率点跟踪（MPPT）
- 低电压穿越（LVRT）

**风电并网**：
- 双馈异步发电机（DFIG）
- 永磁直驱发电机（PMSG）
- 无功功率补偿

**挑战**：
- 间歇性与波动性
- 调峰调频压力
- 电能质量（谐波、闪变）
