# PLC 编程

## 定义
PLC（可编程逻辑控制器）是一种专为工业环境设计的数字运算操作电子系统，通过编程实现逻辑控制、顺序控制、定时计数、算术运算等功能。

## 研究对象
- PLC 的工作原理与硬件结构
- PLC 编程语言与指令系统
- PLC 控制系统设计方法
- PLC 在工业自动化中的应用

## 核心内容

### PLC 工作原理

**扫描周期**：
1. 输入采样：读取所有输入端子状态，存入输入映像区
2. 程序执行：从左到右、从上到下执行用户程序
3. 输出刷新：将输出映像区数据送到输出端子
4. 通信处理：与编程器、上位机通信
5. 自诊断：检查程序错误、看门狗

**扫描时间**：一般1~100ms，取决于程序长度和 CPU 性能

### 梯形图编程

**基本符号**：
- 常开触点：`-| |-$
- 常闭触点：`-|/|-$
- 线圈输出：`-( )-$
- 定时器：`T`
- 计数器：`C`

**编程规则**：
- 每一行从左母线开始，以线圈结束
- 触点可任意串并联
- 线圈只能并联，不能串联
- 同一编号线圈只能使用一次

**基本电路**：
- 自锁电路（起保停）
- 互锁电路
- 顺序启动电路
- 闪光电路

### 功能块图

**FBD 特点**：
- 类似数字电路图
- 功能块：AND、OR、NOT、定时器、计数器
- 输入在左，输出在右
- 适合复杂逻辑和运算

### 西门子 S7系列与三菱 FX 系列对比

| 特性 | 西门子 S7-1200/1500 | 三菱 FX5U/FX3U |
|------|-------------------|---------------|
| 编程软件 | TIA Portal | GX Works |
| 编程语言 | LAD/FBD/SCL | LAD/FBD/ST |
| 通信接口 | PROFINET/PROFIBUS | 以太网/CC-Link |
| 速度 | 快 | 中等 |
| 扩展性 | 强（分布式 I/O） | 一般 |
| 价格 | 较高 | 中等 |
| 应用场景 | 大中型项目 | 中小型项目 |

### 常用指令

**定时器指令**：
- TON（接通延时）：输入接通后延时输出
- TOF（断开延时）：输入断开后延时输出
- TP（脉冲定时器）：产生固定宽度脉冲

**计数器指令**：
- CTU（加计数）：输入上升沿计数
- CTD（减计数）：输入上升沿减计数
- CTUD（加减计数）：双向计数

**数据传送指令**：
- MOV：数据传送
- MVB：字节传送
- MOVW：字传送

**比较指令**：
- =（等于）
- <>（不等于）
- <、>、<=、>=

**运算指令**：
- ADD：加法
- SUB：减法
- MUL：乘法
- DIV：除法

### PLC 在工业自动化中的应用

**顺序控制**：
- 包装生产线
- 装配线
- 物料分拣

**过程控制**：
- 温度控制
- 压力控制
- 液位控制

**运动控制**：
- 位置控制
- 速度控制
- 同步控制

**通信联网**：
- PLC 联网（PROFINET、EtherNet/IP）
- 与上位机通信（OPC UA）
- 远程监控
