# 测量理论

## 定义
测量理论是研究测量原理、测量方法、测量误差和数据处理的学科。是仪器科学与技术的理论基础。

## 研究对象
- 测量误差的分类与处理
- 测量不确定度评定
- 测量仪器的特性指标
- 测量数据处理方法

## 核心内容

### 测量误差

**误差定义**：测量误差 = 测量值 - 真值

$$\Delta x = x - x_0$$

**绝对误差**：$\Delta x = x - x_0$

**相对误差**：$\delta = \frac{\Delta x}{x_0} \times 100\%$

**系统误差**：
- 定义：在同一条件下多次测量，误差的大小和符号固定不变或按一定规律变化
- 来源：仪器误差、方法误差、环境误差、人员误差
- 消除方法：校准、补偿、修正

**随机误差**：
- 定义：在同一条件下多次测量，误差的大小和符号以不可预知的方式变化
- 特征：服从统计规律
- 处理方法：多次测量取平均值

**粗大误差（过失误差）**：
- 定义：明显偏离真值的误差
- 判别准则：3σ准则（莱伊达准则）、格拉布斯准则、狄克逊准则
- 处理：剔除

### 不确定度评定

**测量不确定度**：表征合理地赋予被测量之值的分散性参数。

**A 类不确定度**（统计方法）：

$$u_A = \sqrt{\frac{\sum_{i=1}^{n}(x_i-\bar{x})^2}{n(n-1)}}$$

**B 类不确定度**（非统计方法）：

$$u_B = \frac{a}{k}$$

$a$ 为被测量可能变化的半区间，$k$ 为包含因子。

**合成不确定度**：

$$u_c = \sqrt{u_A^2 + u_B^2}$$

**扩展不确定度**：

$$U = k \cdot u_c$$

一般取 $k=2$（置信概率约95%）。

### 测量仪器特性

**灵敏度**：

$$S = \frac{\Delta y}{\Delta x}$$

输出量变化与输入量变化之比。

**分辨率**：仪器能检测到的最小输入变化量。

**线性度**：校准曲线与拟合直线的最大偏差。

**重复性**：同一条件下多次测量结果的一致程度。

**再现性（重复性）**：不同条件下测量结果的一致程度。

**迟滞**：输入量由小到大和由大到小过程中，同一输入量对应输出量的差异。

**稳定性**：仪器保持其性能特性随时间不变的能力。

**漂移**：仪器输出量随时间的缓慢变化。

### 数据处理方法

**有效数字运算**：
- 加减法：以精度最低的数为准
- 乘除法：以有效数字最少的数为准
- 四舍六入五成双

**最小二乘法**：

$$\min \sum_{i=1}^{n} [y_i - f(x_i)]^2$$

线性拟合：$y = a + bx$

$$b = \frac{n\sum x_iy_i - \sum x_i\sum y_i}{n\sum x_i^2 - (\sum x_i)^2}$$

**回归分析**：建立变量间的经验公式

**曲线拟合**：用数学函数逼近实验数据

### 计量基准与标准

**国际单位制（SI）基本单位**：
- 长度：米（m）
- 质量：千克（kg）
- 时间：秒（s）
- 电流：安培（A）
- 热力学温度：开尔文（K）
- 物质的量：摩尔（mol）
- 发光强度：坎德拉（cd）

**量值传递体系**：
- 国际基准 → 国家基准 → 工作基准 → 工作计量器具
- 检定与校准

**溯源性**：测量结果通过不间断的比较链与国家或国际基准联系。
