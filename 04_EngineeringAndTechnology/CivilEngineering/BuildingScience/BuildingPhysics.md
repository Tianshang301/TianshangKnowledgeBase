# 建筑物理

## 定义
建筑物理是研究建筑环境中的热、声、光等物理现象及其控制方法的学科。是建筑节能和环境舒适性设计的理论基础。

## 研究对象
- 建筑热工学
- 建筑声学
- 建筑光学
- 建筑节能设计

## 核心内容

### 建筑热工学

**传热方式**：
- 热传导：$q = -\lambda\frac{dT}{dx}$
- 热对流：$q = h(T_s - T_f)$
- 热辐射：$q = \varepsilon\sigma(T_1^4 - T_2^4)$

**围护结构传热系数**：

$$K = \frac{1}{R_i + \sum\frac{\delta_j}{\lambda_j} + R_e}$$

$R_i$、$R_e$ 为内外表面换热阻，$\delta_j/\lambda_j$ 为各层材料热阻。

**结露判断**：

$$T_s > T_{dp}$$

$T_s$ 为内表面温度，$T_{dp}$ 为空气露点温度。

### 建筑声学

**吸声**：
- 吸声系数：$\alpha = \frac{E_a}{E_i}$
- 多孔材料：中高频吸声好
- 共振吸声结构：低频吸声

**隔声**：
- 质量定律：$TL = 20\log(mf) - 42$ dB
- 双层墙：中间空气层提高隔声量
- 吻合效应：避免板材共振

**混响时间**：

$$T_{60} = \frac{0.161V}{A}$$

$V$ 为房间体积，$A$ 为总吸声量。

### 建筑光学

**天然采光**：
- 采光系数：$C = \frac{E_i}{E_w} \times 100\%$
- 侧面采光：窗地面积比1:4~1:6
- 顶部采光：天窗形式选择

**人工照明**：
- 照度：$E = \Phi/A$（lux）
- 色温：影响视觉舒适度
- 显色指数：$R_a > 80$（一般要求）

### 建筑节能设计

**节能标准**：
- 体形系数：$S = A_0/V$（不宜过大）
- 窗墙面积比：限制范围
- 围护结构热工性能要求

**节能技术**：
- 外墙外保温
- 中空玻璃窗
- 遮阳系统
- 自然通风
