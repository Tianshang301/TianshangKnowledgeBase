# 金属增材制造

金属增材制造（Metal Additive Manufacturing）是使用金属材料逐层构建三维物体的先进制造技术，在航空航天、医疗、汽车等领域有广泛应用。这项技术正在改变传统制造业的面貌，实现复杂结构的高效制造。

## 金属增材制造技术分类

### 选区激光熔化SLM (Selective Laser Melting)

SLM使用高功率激光完全熔化金属粉末，形成全致密金属零件。这是目前应用最广泛的金属增材制造技术。

#### SLM工作原理

1. **铺粉过程**
   - 刮刀在基板上铺设一层金属粉末
   - 粉层厚度：20-100μm
   - 粉末均匀性控制
   - 基板预热

2. **激光扫描**
   - 激光按照截面轮廓扫描粉末层
   - 激光功率：200-1000W
   - 扫描速度：0.5-10m/s
   - 光斑直径：50-200μm

3. **粉末熔化**
   - 激光能量使粉末完全熔化
   - 熔池温度：1500-3000°C
   - 熔池尺寸：100-500μm
   - 熔化深度：30-100μm

4. **快速凝固**
   - 熔池快速冷却凝固
   - 冷却速率：10^3-10^6 K/s
   - 晶粒细化
   - 残余应力产生

5. **逐层构建**
   - 基板下降一个层高
   - 重复铺粉和扫描过程
   - 直到完成整个零件
   - 后处理和热处理

#### SLM技术特点

1. **材料范围广**
   - 钛合金：Ti6Al4V、Ti6Al4V ELI
   - 镍基合金：Inconel 718、Inconel 625
   - 不锈钢：316L、17-4PH
   - 铝合金：AlSi10Mg、AlSi7Mg
   - 钴铬合金：CoCrMo
   - 工具钢：H13、M2

2. **精度高**
   - 层厚：20-100μm
   - 尺寸精度：±0.1mm
   - 表面粗糙度：Ra 5-15μm
   - 细节表现好

3. **致密度高**
   - 可达99%以上
   - 力学性能优异
   - 接近锻造材料
   - 各向异性小

4. **复杂结构**
   - 可制造传统工艺无法实现的复杂几何
   - 内部流道
   - 点阵结构
   - 拓扑优化结构

#### SLM工艺参数

1. **激光功率 (Laser Power)**
   - 影响熔化深度和效率
   - 功率不足：未熔合缺陷
   - 功率过高：飞溅、球化
   - 需要与扫描速度匹配

2. **扫描速度 (Scan Speed)**
   - 影响能量输入和质量
   - 速度过快：未熔合
   - 速度过慢：过度熔化
   - 需要优化平衡

3. **扫描间距 (Hatch Spacing)**
   - 影响层间结合和致密度
   - 间距过大：孔隙
   - 间距过小：效率低
   - 通常为光斑直径的0.6-0.8倍

4. **层厚 (Layer Thickness)**
   - 影响精度和表面质量
   - 层厚小：精度高，速度慢
   - 层厚大：精度低，速度快
   - 通常20-100μm

#### SLM设备系统

1. **激光系统**
   - 光纤激光器
   - 波长：1064nm
   - 功率：200-1000W
   - 光束质量好

2. **光学系统**
   - 扫描振镜
   - 聚焦透镜
   - 光束整形
   - 实时控制

3. **粉末系统**
   - 铺粉机构
   - 粉末回收
   - 惰性气体保护
   - 氧含量控制

4. **控制系统**
   - 运动控制
   - 过程监控
   - 数据采集
   - 质量控制

### 电子束熔化EBM (Electron Beam Melting)

EBM使用高能电子束在真空环境中熔化金属粉末，特别适合活性金属和高温合金。

#### EBM工作原理

1. **真空环境**
   - 工作腔抽真空至10^-3 mbar
   - 避免材料氧化
   - 提高电子束效率
   - 适合活性金属

2. **粉末预热**
   - 电子束预热粉末至高温
   - 温度：600-1100°C
   - 减少热应力
   - 提高粉末流动性

3. **电子束扫描**
   - 电子束按照轮廓熔化粉末
   - 电子束功率：3-10kW
   - 扫描速度快
   - 能量密度高

4. **逐层构建**
   - 重复铺粉和熔化过程
   - 零件在高温环境中构建
   - 冷却速率相对较低
   - 残余应力小

#### EBM技术特点

1. **真空环境**
   - 避免材料氧化
   - 适合活性金属
   - 纯净度高
   - 质量好

2. **高预热温度**
   - 减少残余应力
   - 降低开裂风险
   - 改善力学性能
   - 适合难焊材料

3. **高生产效率**
   - 电子束扫描速度快
   - 预热减少后续热处理
   - 整体效率高
   - 适合批量生产

4. **材料利用率高**
   - 未熔化粉末可回收使用
   - 成本效益好
   - 环保
   - 资源节约

#### EBM应用领域

1. **航空航天**
   - 钛合金零件
   - 高温合金零件
   - 轻量化结构
   - 复杂内部结构

2. **医疗植入物**
   - 钛合金植入物
   - 多孔结构
   - 个性化定制
   - 生物相容性好

3. **汽车工业**
   - 高性能零件
   - 轻量化设计
   - 原型制作
   - 小批量生产

### 定向能量沉积DED (Directed Energy Deposition)

DED使用激光、电子束或电弧作为热源，熔化同步供给的金属材料，适用于大型零件和修复。

#### DED技术类型

1. **激光定向能量沉积 (Laser DED)**
   - 使用激光作为热源
   - 精度高，热影响区小
   - 适用于精密修复和涂层
   - 代表技术：Laser Engineered Net Shaping (LENS)

2. **电子束定向能量沉积 (Electron Beam DED)**
   - 使用电子束作为热源
   - 真空环境，材料质量好
   - 适用于大型零件制造
   - 代表技术：Electron Beam Additive Manufacturing (EBAM)

3. **电弧定向能量沉积 (Wire Arc DED)**
   - 使用电弧作为热源
   - 成本低，沉积速率高
   - 适用于大型结构件
   - 代表技术：Wire Arc Additive Manufacturing (WAAM)

#### DED技术特点

1. **材料灵活性**
   - 可使用粉末或丝材
   - 材料范围广
   - 成本相对较低
   - 材料利用率高

2. **修复能力**
   - 可修复损坏的金属零件
   - 精密修复
   - 延长零件寿命
   - 成本效益好

3. **多材料加工**
   - 可实现梯度材料
   - 多材料制造
   - 功能梯度结构
   - 创新设计

4. **大尺寸制造**
   - 不受粉末床尺寸限制
   - 大型零件制造
   - 建筑规模制造
   - 工业应用

#### DED应用场景

1. **大型结构件**
   - 航空航天结构件
   - 船舶零件
   - 建筑结构
   - 桥梁部件

2. **修复应用**
   - 航空发动机叶片
   - 模具修复
   - 磨损零件修复
   - 价值恢复

3. **涂层应用**
   - 耐磨涂层
   - 耐腐蚀涂层
   - 热障涂层
   - 功能涂层

4. **梯度材料**
   - 多材料结构
   - 功能梯度材料
   - 性能优化
   - 创新设计

## 缺陷检测 (Defect Detection)

金属增材制造过程中可能产生各种缺陷，需要有效的检测方法确保质量。

### 常见缺陷类型

#### 孔隙 (Porosity)

1. **气孔**
   - 气体残留形成
   - 球形孔隙
   - 尺寸：10-100μm
   - 分布随机

2. **未熔合**
   - 能量不足导致
   - 不规则形状
   - 尺寸：50-500μm
   - 层间或道间

3. **裂纹**
   - 热应力引起
   - 线性缺陷
   - 尺寸：10-1000μm
   - 影响性能

#### 裂纹 (Cracks)

1. **热裂纹**
   - 凝固过程中产生
   - 晶界开裂
   - 温度范围：固相线附近
   - 材料敏感

2. **冷裂纹**
   - 冷却过程中产生
   - 残余应力导致
   - 温度范围：室温附近
   - 氢致裂纹

3. **液化裂纹**
   - 晶界液化导致
   - 热影响区
   - 温度范围：固液两相区
   - 成分偏析

#### 残余应力 (Residual Stress)

1. **热应力**
   - 温度梯度引起
   - 热胀冷缩不均
   - 影响尺寸精度
   - 可能导致开裂

2. **相变应力**
   - 组织转变产生
   - 体积变化
   - 影响性能
   - 需要热处理

3. **影响**
   - 尺寸精度
   - 力学性能
   - 疲劳寿命
   - 耐腐蚀性

#### 表面质量 (Surface Quality)

1. **球化现象**
   - 粉末未完全熔化
   - 表面粗糙
   - 影响精度
   - 需要后处理

2. **层间缺陷**
   - 层间结合不良
   - 影响强度
   - 影响密封性
   - 需要优化工艺

3. **支撑痕迹**
   - 支撑结构残留
   - 表面损伤
   - 需要后处理
   - 设计优化

### 检测方法

#### 无损检测 (Non-destructive Testing, NDT)

1. **X射线检测**
   - 检测内部缺陷
   - 二维成像
   - 分辨率高
   - 应用广泛

2. **CT扫描**
   - 三维缺陷分析
   - 完整内部结构
   - 高分辨率
   - 数据分析复杂

3. **超声波检测**
   - 检测裂纹和孔隙
   - 深度检测
   - 实时检测
   - 成本较低

4. **渗透检测**
   - 表面裂纹检测
   - 操作简单
   - 成本低
   - 灵敏度高

#### 在线监测 (In-situ Monitoring)

1. **熔池监测**
   - 监控熔化过程
   - 温度分布
   - 熔池尺寸
   - 质量预测

2. **温度场监测**
   - 检测温度分布
   - 热梯度分析
   - 残余应力预测
   - 工艺优化

3. **高速摄像**
   - 观察粉末行为
   - 熔池动态
   - 飞溅现象
   - 过程理解

4. **声发射监测**
   - 检测裂纹产生
   - 实时监测
   - 缺陷预警
   - 质量控制

#### 力学性能测试 (Mechanical Testing)

1. **拉伸测试**
   - 评估强度和塑性
   - 标准试样
   - 应力应变曲线
   - 性能验证

2. **疲劳测试**
   - 评估耐久性
   - 循环载荷
   - 寿命预测
   - 可靠性评估

3. **硬度测试**
   - 评估局部性能
   - 显微硬度
   - 硬度分布
   - 质量评估

4. **冲击测试**
   - 评估韧性
   - 冲击载荷
   - 断口分析
   - 安全评估
