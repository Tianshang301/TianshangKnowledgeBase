# 水力学

## 定义
水力学是研究液体（主要是水）的平衡和运动规律及其与固体边界相互作用的学科。是水利工程的技术基础。

## 研究对象
- 水静力学
- 水动力学基本方程
- 明渠水流
- 堰流与闸孔出流
- 渗流理论

## 核心内容

### 水静力学

**静水压强**：$p = p_0 + \gamma h$

**平面壁静水总压力**：

$$P = p_c A = \gamma h_c A$$

$h_c$ 为形心淹没深度，$A$ 为受压面积。

**压力中心**：

$$y_p = y_c + \frac{I_c}{y_c A}$$

### 水动力学基本方程

**连续性方程**：$Q = v_1 A_1 = v_2 A_2$

**能量方程**（伯努利方程）：

$$z_1 + \frac{p_1}{\gamma} + \frac{v_1^2}{2g} = z_2 + \frac{p_2}{\gamma} + \frac{v_2^2}{2g} + h_w$$

**动量方程**：

$$\rho Q(v_2 - v_1) = \sum F$$

**断面比能**（特定能量）：
$$E = h + \frac{v^2}{2g} = h + \frac{Q^2}{2gA^2}$$

**临界水深**：$\frac{Q^2}{g} = \frac{A_c^3}{B_c}$（矩形断面：$h_c = \sqrt[3]{\frac{q^2}{g}}$）

### 明渠水流

**均匀流**：$v = C\sqrt{Ri}$（谢才公式）

水力半径：$R = A/\chi$（$A$ 过水面积，$\chi$ 湿周）

谢才系数：$C = \frac{1}{n}R^{1/6}$（曼宁公式）

**弗劳德数**：$Fr = \frac{v}{\sqrt{gh}}$

- $Fr < 1$：缓流
- $Fr = 1$：临界流
- $Fr > 1$：急流

**明渠水流流态对比**：

| 流态 | $Fr$ | 水深 $h$ | 波速传播 | 典型现象 |
|------|------|---------|---------|---------|
| 缓流 | $Fr < 1$ | $h > h_c$ | 波能向上游传播 | 缓坡均匀流 |
| 临界流 | $Fr = 1$ | $h = h_c$ | 波速等于流速 | 流态分界 |
| 急流 | $Fr > 1$ | $h < h_c$ | 波不能向上游传播 | 急坡水流、跌水 |

**水跃方程**：
$$h_2 = \frac{h_1}{2}\left(\sqrt{1+8Fr_1^2}-1\right)$$

其中 $h_1$ 为跃前水深，$h_2$ 为跃后水深，$Fr_1$ 为跃前弗劳德数。

**水跃能量损失**：$\Delta E = \frac{(h_2 - h_1)^3}{4h_1h_2}$

### 堰流与闸孔出流

**堰流公式**：$Q = m b\sqrt{2g}H_0^{3/2}$

$m$ 为流量系数，取决于堰型。

**闸孔出流**：$Q = \mu b e\sqrt{2gH_0}$

$\mu$ 为流量系数，$e$ 为闸门开度。

### 渗流理论

**达西定律**：$v = kJ$

$J$ 为水力梯度，$k$ 为渗透系数。

**渗透流量**：$Q = kAJ$

**不同堰型流量系数对比**：

| 堰型 | 堰顶形状 | 流量系数 $m$ | 适用范围 |
|------|---------|-------------|---------|
| 薄壁堰 | 锐缘 | $m = 0.407 + 0.0533H/\delta$ | 实验室量测 |
| 实用堰（WES） | 曲线型 | $m \approx 0.49\sim0.52$ | 溢洪道 |
| 宽顶堰 | 水平顶 | $m \approx 0.32\sim0.38$ | 涵洞进口 |

**有压管道恒定流**：
$$h_f = \lambda \frac{L}{d}\frac{v^2}{2g}$$

其中 $\lambda$ 为沿程阻力系数（达西-魏斯巴赫公式），$L$ 为管长，$d$ 为管径。

### 水泵功率

**有效功率（水功率）**：
$$P_w = \rho g Q H$$

**水泵轴功率**：
$$P = \frac{\rho g Q H}{\eta}$$

其中 $\rho$ 为水的密度，$g$ 为重力加速度，$Q$ 为流量，$H$ 为扬程，$\eta$ 为水泵效率。

**比转速**：
$$n_s = \frac{n\sqrt{Q}}{H^{3/4}}$$

$n_s$ 用于水泵选型：低比转速对应离心泵，高比转速对应轴流泵。

**沿程阻力系数**：
- 层流：$\lambda = 64/Re$
- 湍流光滑区：$\lambda = 0.3164/Re^{1/4}$（Blasius公式）
- 湍流粗糙区：$\dfrac{1}{\sqrt{\lambda}} = -2\lg\left(\dfrac{\Delta}{3.7d} + \dfrac{2.51}{Re\sqrt{\lambda}}\right)$（Colebrook公式）
