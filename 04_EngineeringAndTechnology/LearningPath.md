# 学习路径：工程与技术科学

## 基础阶段
1. **数学基础**：微积分、线性代数、概率统计
2. **物理基础**：力学、电磁学、热力学
3. **工程力学**：静力学、动力学、材料力学
4. **工程数学**：数值方法、运筹学

## 专业方向选择
| 方向 | 核心课程 |
|------|----------|
| 机械工程 | 机械设计、制造技术、控制工程 |
| 电气工程 | 电路分析、电机学、电力系统 |
| 电子信息 | 信号处理、通信原理、微电子 |
| 土木工程 | 结构力学、岩土工程、施工技术 |
| 化学工程 | 单元操作、反应工程、工艺设计 |

## 推荐资料
- MIT OCW 工程类课程
- Engineering Toolbox 参考手册
- 各专业经典教材（如《机械设计手册》）
