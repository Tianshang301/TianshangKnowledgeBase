# 油藏工程

## 定义
油藏工程是研究油气藏开发理论和方法的学科，包括油藏评价、开发方案设计和动态分析。

## 核心内容

### 油藏驱动类型

**弹性驱动**：依靠油藏和流体的弹性能量

**水驱**：边水或底水推进

**溶解气驱**：溶解气析出膨胀驱动

**气顶驱动**：气顶膨胀驱动

### 油藏工程计算

**物质平衡方程**（Schilthuis方程）：

$$N_p B_o + W_p B_w - W_i B_w = N B_{oi}\left[\frac{(B_o - B_{oi}) + (R_s - R_{si})B_g}{B_{oi}} + \frac{m B_{oi}}{B_{gi}}(B_g - B_{gi})\right]$$

**产量递减规律**：
- 指数递减：$q = q_i e^{-D_i t}$
- 双曲递减：$q = q_i(1+bD_i t)^{-1/b}$
- 调和递减：$q = q_i(1+D_i t)^{-1}$

### 开发方案设计

**井网形式**：
- 正方形井网
- 三角形井网
- 不规则井网

**注水开发**：
- 注水量：$Q_w = Q_o \frac{B_o}{B_w}\frac{\mu_w}{\mu_o}$
- 注采比：注入量与采出量之比

### 提高采收率

**化学驱**：
- 聚合物驱：增加水相粘度
- 表面活性剂驱：降低界面张力
- 三元复合驱：碱+表面活性剂+聚合物

**热力驱**：
- 蒸汽吞吐
- 蒸汽驱
- 火烧油层

**气驱**：
- CO₂驱
- 氮气驱
- 烟道气驱
