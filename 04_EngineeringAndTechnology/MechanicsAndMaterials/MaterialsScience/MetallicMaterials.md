# 金属材料

## 定义
金属材料是以金属元素为基础、由金属元素或以金属元素为主构成的具有金属特性的材料的统称。包括纯金属、合金和金属基复合材料。

## 研究对象
- 金属及合金的晶体结构与微观组织
- 金属材料的力学性能与物理性能
- 金属材料的热处理工艺与相变规律
- 金属材料的腐蚀与防护

## 核心内容

### 晶体结构

**体心立方（BCC）**：
- 代表金属：α-Fe、Cr、W、Mo、V
- 配位数：8
- 原子半径 $r = \frac{\sqrt{3}a}{4}$，$a$ 为晶格常数
- 致密度：0.68
- 滑移系：{110}⟨111⟩（12个）

**面心立方（FCC）**：
- 代表金属：γ-Fe、Cu、Al、Ni、Au、Ag
- 配位数：12
- 原子半径 $r = \frac{\sqrt{2}a}{4}$
- 致密度：0.74
- 滑移系：{111}⟨110⟩（12个），塑性最好

**密排六方（HCP）**：
- 代表金属：Mg、Zn、Ti（α-Ti）、Co
- 配位数：12
- 致密度：0.74
- c/a 比值影响滑移：理想值1.633

### 铁碳相图（详细）

**关键温度点**：
- A₁线（727°C）：共析转变温度
- A₃线（912°C）：纯铁同素异构转变温度
- Acm 线：Fe₃C 在γ-Fe 中的溶解度曲线

**共析反应**：$\gamma(0.77\%C) \xrightarrow{727°C} \alpha(0.0218\%C) + Fe_3C(6.69\%C)$

**共晶反应**：$L(4.3\%C) \xrightarrow{1148°C} \gamma(2.11\%C) + Fe_3C(6.69\%C)$

**组织组成物**：
- 铁素体（α-Fe）：碳溶解度极低（室温约0.008%）
- 奥氏体（γ-Fe）：最大溶碳量2.11%（1148°C）
- 渗碳体（Fe₃C）：硬度高（≈800HV），脆性大
- 珠光体：铁素体+渗碳体层片状组织
- 莱氏体：奥氏体+渗碳体（高温）

**碳钢分类**：
- 低碳钢：C < 0.25%
- 中碳钢：0.25% ≤ C ≤ 0.6%
- 高碳钢：C > 0.6%

### 热处理

**退火**：
- 完全退火：加热到 Ac₃以上30~50°C，随炉冷却
- 去应力退火：500~650°C 保温后缓冷
- 目的：降低硬度、消除内应力、细化晶粒

**正火**：
- 加热到 Ac₃（亚共析钢）或 Acm（过共析钢）以上30~50°C，空冷
- 比退火冷却速度快，组织更细，强度更高

**淬火**：
- 加热到临界温度以上，快速冷却（水淬、油淬、盐浴淬火）
- 获得马氏体组织，大幅提高硬度
- 淬透性：钢在淬火时获得马氏体层深度的能力

**回火**：
- 低温回火（150~250°C）：马氏体回火，保持高硬度
- 中温回火（350~500°C）：回火屈氏体，高弹性
- 高温回火（500~650°C）：回火索氏体，良好的综合力学性能
- **调质处理** = 淬火 + 高温回火

### 合金钢分类

**合金结构钢**：
- 调质钢：40Cr、35CrMo、42CrMo
- 渗碳钢：20CrMnTi、20CrNi2Mo
- 弹簧钢：60Si2Mn、50CrV

**合金工具钢**：
- 低合金工具钢：9SiCr、CrWMn
- 高速钢：W18Cr4V、W6Mo5Cr4V2（红硬性）

**特殊钢**：
- 不锈钢：304（18Cr-8Ni）、316L
- 耐热钢：Cr25Ni20
- 耐磨钢：Mn13（高锰钢）

### 铝合金

**变形铝合金**：
- 2xxx 系（Al-Cu）：高强度，如2024-T3
- 6xxx 系（Al-Mg-Si）：中等强度，良好耐蚀性，如6061-T6
- 7xxx 系（Al-Zn-Mg-Cu）：超高强度，如7075-T6

**铸造铝合金**：
- Al-Si 系：ZL101、ZL102（流动性好）
- Al-Cu 系：ZL201（强度高）

### 钛合金

**α型钛合金**：TA7（Ti-5Al-2.5Sn），高温性能好
**β型钛合金**：TB6（Ti-10V-2Fe-3Al），强度高，可热处理
**α+β型钛合金**：TC4（Ti-6Al-4V），应用最广泛

### 材料性能指标

**强度**：
- 抗拉强度 $\sigma_b$：材料断裂前最大应力
- 屈服强度 $\sigma_s$：产生0.2%残余变形时的应力
- $\sigma = \frac{F}{A_0}$

**硬度**：
- 布氏硬度（HB）：$HB = \frac{2F}{\pi D(D-\sqrt{D^2-d^2})}$
- 洛氏硬度（HR）：HRC、HRB
- 维氏硬度（HV）：$HV = \frac{2F\sin(136°/2)}{d^2}$

**韧性**：
- 冲击韧性：$a_k = \frac{A_k}{A}$（夏比冲击试验）
- 断裂韧性：$K_{IC}$

**疲劳**：
- S-N 曲线：$\sigma_a = \sigma_f'(2N_f)^b$
- 疲劳极限：碳钢约 $\sigma_{-1} \approx 0.5\sigma_b$

- 汽车工业（车身、发动机零部件）

- 建筑工程（钢结构、钢筋）
- 机械制造（齿轮、轴承、模具）
- 电子信息（铜合金引线框架、铝散热器）
- 能源装备（核电站管道、风力发电塔架）
