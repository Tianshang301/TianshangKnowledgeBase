# 计量经济学与金融学

## 一、计量经济学基础

### 1.1 经典线性回归

**简单线性回归**：$Y_i = \beta_0 + \beta_1 X_i + \varepsilon_i$。**多元线性回归**：$Y = X\beta + \varepsilon$。

**OLS估计量**：$\hat{\beta} = (X'X)^{-1}X'y$，通过最小化残差平方和 $RSS = (Y-X\hat{\beta})'(Y-X\hat{\beta})$ 得到。

**高斯-马尔可夫定理**：在经典假定下，OLS是最佳线性无偏估计量（BLUE）。

| 假定 | 含义 | 违反后果 |
|------|------|----------|
| 线性性 | 模型关于参数线性 | 设定偏误 |
| 外生性 | $E(\varepsilon\|X) = 0$ | 遗漏变量偏误 |
| 球形扰动 | $Var(\varepsilon\|X) = \sigma^2 I$ | 标准误错误 |
| 满秩 | $rank(X) = K$ | 无法求逆 |

### 1.2 拟合优度与假设检验

$R^2 = 1-\frac{SSR}{SST} = \frac{SSE}{SST}$，$\bar{R}^2 = 1-\frac{SSR/(n-k)}{SST/(n-1)}$

**t检验**：$t = \frac{\hat{\beta}_j}{SE(\hat{\beta}_j)} \sim t_{n-k}$，检验单个参数显著性。

**F检验**：$F = \frac{(SSR_r-SSR_{ur})/q}{SSR_{ur}/(n-k)} \sim F_{q,n-k}$，检验联合假设。

**AIC** $= n\ln(SSR/n) + 2k$，**BIC** $= n\ln(SSR/n) + k\ln n$

### 1.3 模型诊断

| 问题 | 检测方法 | 处理方式 |
|------|----------|----------|
| 异方差 | Breusch-Pagan检验、White检验 | 稳健标准误（HC）、WLS |
| 自相关 | Durbin-Watson检验、BG检验 | GLS、Newey-West HAC标准误 |
| 多重共线性 | VIF > 10、条件数 > 30 | 岭回归、变量选择、主成分分析 |
| 设定偏误 | RESET检验 | 增加非线性项、交互项、遗漏变量 |

**稳健标准误**：当异方差或自相关存在时，OLS系数估计仍无偏，但标准误有偏。使用异方差稳健标准误（White标准误）或HAC标准误（Newey-West）可得到正确的统计推断。

## 二、时间序列分析

### 2.1 平稳性与单位根

**弱平稳**：均值、方差恒定，协方差仅依赖滞后长度而非时间。

**ADF检验**：$\Delta Y_t = \alpha + \beta t + \gamma Y_{t-1} + \sum\delta_i \Delta Y_{t-i} + \varepsilon_t$，$H_0:\gamma=0$（存在单位根）。经济变量通常 $I(1)$。

### 2.2 ARIMA模型

| 模型 | 形式 | ACF特征 | PACF特征 |
|------|------|---------|----------|
| AR(p) | $Y_t = c + \sum_{i=1}^p \phi_i Y_{t-i} + \varepsilon_t$ | 拖尾 | $p$阶截尾 |
| MA(q) | $Y_t = \mu + \varepsilon_t + \sum_{j=1}^q \theta_j \varepsilon_{t-j}$ | $q$阶截尾 | 拖尾 |
| ARIMA(p,d,q) | 差分 $d$次后拟合ARMA | 拖尾 | 拖尾 |

**Box-Jenkins方法**：① 模型识别（通过ACF和PACF确定p,d,q阶数）；② 参数估计（OLS或最大似然）；③ 诊断检验（残差是否为白噪声）；④ 预测（最小均方误差预测）。

**信息准则选择滞后阶数**：AIC $= T\ln(SSR) + 2k$，BIC $= T\ln(SSR) + k\ln T$。AIC倾向于选择更灵活的模型，BIC对参数惩罚更严厉。

### 2.3 协整与ECM

若 $X_t, Y_t \sim I(1)$ 且 $Y_t - \theta X_t \sim I(0)$，则协整。

**误差修正模型**：$\Delta Y_t = \alpha + \beta \Delta X_t + \gamma(Y_{t-1}-\theta X_{t-1}) + \varepsilon_t$，$\gamma<0$ 反映向长期均衡的调整速度。

### 2.4 波动率模型

**ARCH**：$\sigma_t^2 = \omega + \alpha \varepsilon_{t-1}^2$。**GARCH(1,1)**：$\sigma_t^2 = \omega + \alpha \varepsilon_{t-1}^2 + \beta \sigma_{t-1}^2$，$\alpha+\beta<1$ 保证平稳性。

$\alpha$反映新信息对波动率的影响，$\beta$反映波动率的持续性，$\alpha+\beta$接近1表示波动冲击高度持久（集成GARCH）。GARCH模型广泛应用于金融风险度量（VaR计算）、资产定价和投资组合优化。扩展模型包括EGARCH（允许杠杆效应——好消息和坏消息对波动率影响不对称）、TGARCH、GARCH-M（波动率进入均值方程）等。

## 三、面板数据与因果推断

### 3.1 面板数据模型

$Y_{it} = \alpha + \beta X_{it} + \mu_i + \lambda_t + \varepsilon_{it}$

| 模型 | $\mu_i$与$X$的关系 | 估计方法 |
|------|-------------------|----------|
| 混合OLS | 无异质性 | OLS |
| 固定效应 | $\mu_i$与$X_{it}$相关 | 组内去均值 |
| 随机效应 | $\mu_i$与$X_{it}$不相关 | GLS |

**Hausman检验**：$H = (\hat{\beta}_{FE}-\hat{\beta}_{RE})'[Var(\hat{\beta}_{FE})-Var(\hat{\beta}_{RE})]^{-1}(\hat{\beta}_{FE}-\hat{\beta}_{RE}) \sim \chi^2_K$，拒绝则用固定效应。

### 3.2 因果推断

**内生性来源**：遗漏变量、测量误差、反向因果。

**工具变量（2SLS）**：第一阶段 $X = Z\gamma + v$，第二阶段 $Y = \hat{X}\beta + \varepsilon$。需满足 $Cov(Z,X)\neq 0$ 和 $Cov(Z,\varepsilon)=0$。

**双重差分（DiD）**：$\tau = (\bar{Y}_{T,Post}-\bar{Y}_{T,Pre}) - (\bar{Y}_{C,Post}-\bar{Y}_{C,Pre})$，关键假设：共同趋势。

**断点回归（RDD）**：$\tau = \lim_{x\to c^+}E[Y\|X=x] - \lim_{x\to c^-}E[Y\|X=x]$

**倾向得分匹配（PSM）**：基于可观测特征估计处理概率（倾向得分），匹配处理组与对照组后比较结果差异。常用匹配方法：最近邻匹配、核匹配、卡尺匹配。

**常见偏误来源**：选择偏误（非随机分组）、遗漏变量偏误（不可观测因素导致内生性）、测量误差（测度不准确导致衰减偏误或放大偏误）、样本选择偏误（非随机缺失导致样本不具代表性）、联立性偏误（变量相互决定导致反向因果）。研究者应结合研究设计和敏感性分析评估偏误方向和大小。

**面板数据的进一步主题**：**动态面板**引入被解释变量的滞后项作为解释变量，$Y_{it} = \alpha + \rho Y_{i,t-1} + \beta X_{it} + \mu_i + \varepsilon_{it}$。由于$\mu_i$与$Y_{i,t-1}$相关，OLS和固定效应均有偏。**差分GMM**（Arellano-Bond估计量）使用滞后水平值作为差分方程的工具变量。**系统GMM**（Blundell-Bond）结合差分方程和水平方程，提高效率。

**短面板**（N大T小）和**长面板**（T大N小）采用不同估计策略。面板数据同时包含横截面和时间维度，能控制个体异质性、提供更多信息和变异性、更好地研究动态调整过程，在经济学实证研究中应用极为广泛。

**空间计量经济学**：当个体之间存在空间依赖或空间溢出效应时，传统计量方法不再适用。**空间自回归模型（SAR）** 引入空间滞后项：$Y = \rho WY + X\beta + \varepsilon$。**空间误差模型（SEM）** 假设误差项存在空间相关。空间权重矩阵$W$反映个体之间的空间邻近关系（邻接矩阵或距离矩阵）。**莫兰检验（Moran's I）** 用于检验空间自相关。空间计量在区域经济学、城市经济学、环境经济学等领域有广泛应用。

**有限因变量模型**：当因变量是离散的或受限时，OLS不再适用。**二元选择模型**（Probit、Logit）：$P(Y=1\|X) = F(X\beta)$，Probit使用标准正态累积分布函数，Logit使用逻辑分布函数。**多值选择模型**：多项Logit（无序）、有序Probit/Logit（有序）。**截断和归并模型**：Tobit模型处理因变量在某一阈值处被归并（如消费支出在零点处归并）。**计数数据模型**：泊松回归、负二项回归处理取非负整数的因变量（如专利数、就诊次数）。这些模型扩展了计量经济学的应用范围。

## 四、货币银行学

### 4.1 货币层次

M0（流通现金）→ M1（M0+活期存款）→ M2（M1+定期存款+储蓄存款）

### 4.2 货币创造

存款乘数 $D = \frac{1}{r}R$，货币乘数 $m = \frac{1+c}{r+c+e}$，基础货币 $B = C+R$。$c$为现金漏出率，$r$为准备金率，$e$为超额准备金率。

### 4.3 商业银行

**三大业务**：负债业务（吸收存款）、资产业务（发放贷款）、中间业务（支付结算、理财、托管等表外业务）。

**巴塞尔协议演进**：巴塞尔I（1988）引入最低资本要求，将信用风险加权资产与资本挂钩。巴塞尔II（2004）三大支柱：最低资本要求、监管审查、市场纪律。巴塞尔III（2010后）提高资本质量要求，引入流动性覆盖率和净稳定融资比率。核心一级资本充足率 $\geq 4.5\%$，总资本充足率 $\geq 8\%$，另设逆周期资本缓冲和系统重要性银行附加资本。

## 五、国际金融

### 5.1 汇率

**直接标价法**（1外币=若干本币）与**间接标价法**。

**购买力平价（PPP）**：绝对 $E = P/P^*$，相对 $\Delta E = \pi - \pi^*$。

**利率平价**：$F = S \times \frac{1+i}{1+i^*}$

| 汇率制度 | 特点 | 代表 |
|----------|------|------|
| 固定汇率 | 央行干预维持窄区间 | 香港 |
| 浮动汇率 | 市场供求决定 | 美、欧、日 |
| 管理浮动 | 有管理的浮动 | 中国 |

### 5.2 国际收支与蒙代尔-弗莱明

**经常账户**（货物、服务、收入、转移）+ **资本账户** + **金融账户**。**J曲线**：贬值后贸易收支先恶化后改善。

**蒙代尔-弗莱明**：浮动汇率下货币政策有效、财政政策弱（本币升值挤出出口）；固定汇率下财政政策有效、货币政策弱。三元悖论（不可能三角）指出：一国不可能同时实现资本自由流动、固定汇率和独立货币政策，三者只能择其二。

## 六、资本市场

### 6.1 股票

**戈登增长模型**：$P_0 = \frac{D_1}{r-g}$。**有效市场假说（EMH）**：弱式（历史信息）、半强式（公开信息）、强式（所有信息）。行为金融挑战：过度反应、动量效应等。

### 6.2 债券

$P = \sum_{t=1}^T \frac{C}{(1+i)^t} + \frac{F}{(1+i)^T}$，到期收益率（YTM）使价格等于现值。**久期** $D = \frac{\sum t\cdot CF_t/(1+y)^t}{P}$。

### 6.3 CAPM

$E(R_i) = R_f + \beta_i(E(R_m)-R_f)$，$\beta_i = \frac{Cov(R_i,R_m)}{Var(R_m)}$。$\beta>1$进攻型、$\beta<1$防御型。

## 七、国际贸易

**比较优势**（李嘉图）：专业生产比较成本更低的产品，贸易互利。

**H-O理论**：出口密集使用丰裕要素的产品。**里昂惕夫悖论**：美国出口劳动密集型多于资本密集型，与H-O理论预测相反。

**新贸易理论**（克鲁格曼）：规模经济和产品差异化导致产业内贸易。

| 政策工具 | 福利影响 |
|----------|----------|
| 关税 | 消费者受损 > 政府+生产者获利 |
| 配额 | 消费者受损，配额租金 |
| 补贴 | 财政负担，报复风险 |

## 八、发展经济学

**刘易斯二元模型**：农业剩余劳动力向工业转移。**刘易斯拐点**：劳动力吸收完毕，工资上升。中国约2004-2010年到达。

**贫困陷阱**（纳克斯恶性循环）：低收入→低储蓄→低资本→低生产率→低收入。**大推进**：多产业同时投资突破均衡。

**森的能力方法**：发展扩展人的实质自由，贫困是可行能力的剥夺。

**制度经济学**：科斯定理（零交易成本时产权不影响效率），诺斯（制度决定增长，路径依赖），阿西莫格鲁（包容性制度促进长期增长）。

**发展指标**：人类发展指数（HDI）综合收入、教育和健康三个维度。千年发展目标（MDGs）和可持续发展目标（SDGs）为全球发展提供框架。减贫成就显著，但全球不平等仍然严峻。发展援助的有效性、援助依赖和所有权问题备受关注。

**公司金融**：**MM定理**（Modigliani-Miller）：在无摩擦市场中，公司价值与资本结构无关。考虑税收后，债务融资具有税盾优势，但过度债务带来财务困境成本。**资本结构权衡理论**在税盾收益与财务困境成本之间权衡。

**投资组合理论**（马科维茨）：投资者通过分散化降低非系统性风险。有效前沿（Efficient Frontier）是给定风险下收益最大的投资组合集合。**夏普比率**：$SR = \frac{E(R_p)-R_f}{\sigma_p}$，衡量风险调整后的超额收益。**Fama-French三因子模型**扩展CAPM，加入规模因子（SMB）和价值因子（HML）。**套利定价理论（APT）**：资产收益率由多个宏观因子线性决定，无需假设市场组合有效。**优序融资理论**（Myers & Majluf）：公司融资顺序为内部融资→债务→股权，源于信息不对称导致的逆向选择成本。**代理理论**：股东与管理层之间的利益冲突（股权代理成本），股东与债权人之间的利益冲突（债务代理成本）。

**行为经济学基础**：与传统理性人假设不同，行为经济学关注心理因素对经济决策的影响。**前景理论**（卡尼曼、特沃斯基）用价值函数替代效用函数，损失厌恶和参照点依赖是关键特征。**启发式与偏见**：代表性、可得性、锚定效应。**助推（Nudge）**：通过改变选择架构引导更好决策，在储蓄、健康、环保领域广泛应用。塞勒获2017年诺贝尔经济学奖。

**金融衍生品**：**期货**是约定在未来以特定价格买卖资产的标准化合约，用于套期保值和价格发现，逐日盯市结算。

**期权**赋予买方在未来以特定价格买卖资产的权利而非义务。看涨期权（Call）赋予买入权，看跌期权（Put）赋予卖出权。**BS公式**：$C = S_0 N(d_1) - Ke^{-rT}N(d_2)$，$d_1 = \frac{\ln(S_0/K)+(r+\sigma^2/2)T}{\sigma\sqrt{T}}$，$d_2 = d_1 - \sigma\sqrt{T}$。

**信用违约互换（CDS）** 为债券提供违约保险，在2008年金融危机中因缺乏监管而加剧了系统性风险。

**风险管理**：市场风险（VaR、预期亏空）、信用风险（违约概率PD、违约损失率LGD）、操作风险（内部欺诈、系统故障）、流动性风险。风险管理是金融机构的核心职能。

**大数据与计量经济学**：大数据时代为计量经济学带来新机遇和挑战。**高维数据分析**：Lasso（最小绝对收缩和选择算子）、岭回归、弹性网等正则化方法处理变量数多于样本数的问题。**文本分析**：将非结构化文本数据转化为可用于回归的特征变量（情感分析、主题模型）。**机器学习方法**：随机森林、梯度提升、神经网络用于预测而非因果推断。**因果森林**（Athey & Imbens）将机器学习引入因果推断，在异质性处理效应估计方面有重要应用。**交叉验证**用于模型选择和超参数调优，避免过拟合。大数据方法与传统计量经济学的融合是当代计量经济学发展的重要方向。
