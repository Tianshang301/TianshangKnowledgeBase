# 宏观经济学

## 一、国民收入核算

### 1.1 GDP 定义

**GDP**：一定时期内一国境内生产的最终产品和服务市场价值总和。四个要点：一定时期（流量）、一国境内（地域概念）、最终产品（避免重复）、市场价值（货币衡量）。

| 核算方法 | 公式 | 说明 |
|------|------|------|
| 支出法 | $GDP = C + I + G + NX$ | 消费+投资+政府购买+净出口 |
| 收入法 | $GDP = W+R+\pi+T_n+Dep$ | 工资+租金+利润+间接税+折旧 |
| 生产法 | $GDP = \sum$ 各产业增加值 | 总产出减中间投入 |

**支出法分解**：$C$（耐用品+非耐用品+服务），$I$（固定资产投资+存货+住宅），$G$（政府消费+投资，不含转移支付），$NX$（出口-进口）。

### 1.2 名义与实际 GDP

**名义 GDP**以当期价格计算，**实际 GDP**以基期价格计算。

**GDP 平减指数** $= \frac{名义 GDP}{实际 GDP} \times 100$。

**CPI**：$CPI = \frac{\sum P_t Q_0}{\sum P_0 Q_0} \times 100$，固定一篮子消费品和服务。

**通胀率**：$\pi_t = \frac{CPI_t - CPI_{t-1}}{CPI_{t-1}} \times 100\%$

### 1.3 其他国民收入指标

**GNP** = GDP + NFP（国外净要素收入）；**NNP** = GNP - 折旧；**NI** = NNP - 间接税 + 补贴；**PI** = NI - 公司税 - 留存收益 - 社保 + 转移支付；**DPI** = PI - 个人所得税。

## 二、通货膨胀

### 2.1 类型与成因

| 类型 | 成因 | 特征 |
|------|------|------|
| 需求拉动型 | 总需求超过总供给 | 过多货币追逐过少商品 |
| 成本推动型 | 工资、原材料成本上升 | 滞胀风险（产出降、价格升） |
| 结构性通胀 | 经济结构变动 | 局部价格上升 |
| 惯性通胀 | 适应性预期 | 自我维持 |

**工资-价格螺旋**：物价上升→工人要求加薪→生产成本上升→物价进一步上升。预期通胀率的上升使短期菲利普斯曲线右移。

**消费者物价指数（CPI）的局限性**：替代偏差（消费者转向相对便宜商品，CPI 高估生活成本）、新商品偏差（未及时纳入新产品）、质量变化偏差（未充分反映质量提升）。

### 2.2 通胀效应

**费雪效应**：$i = r + \pi^e$（名义利率 = 实际利率 + 预期通胀率）。实际利率：$r = i - \pi$。

通胀有利于债务人（固定利率债务贬值）、不利于债权人；有利于浮动收入者、不利于固定收入者。

**社会成本**：鞋底成本（减少持币）、菜单成本（频繁调价）、税收扭曲（名义收入增加导致税级爬升）、财富再分配、不确定性增加。

### 2.3 菲利普斯曲线

**短期**：$\pi = \pi^e - \beta(u-u^*) + \varepsilon$，通胀与失业存在替代关系。

**长期**：垂直（自然率假说），失业率回归自然失业率，与通胀无关。

## 三、失业理论

### 3.1 失业类型

| 类型 | 成因 | 政策应对 |
|------|------|----------|
| 摩擦性失业 | 转换工作、信息不对称 | 完善就业信息服务 |
| 结构性失业 | 技能与岗位不匹配 | 职业培训、教育调整 |
| 周期性失业 | 经济衰退总需求不足 | 扩张性宏观政策 |
| 季节性失业 | 气候、假期 | 多元化安排 |

### 3.2 奥肯定律

$\frac{Y-Y^*}{Y^*} = -\alpha(u-u^*)$，$\alpha \approx 2$。失业率每高于自然率1个百分点，实际 GDP 约低于潜在产出2个百分点。

### 3.3 自然失业率

自然失业率 = 摩擦性失业 + 结构性失业。决定因素：劳动人口结构、失业保险制度、技术进步、劳动力市场灵活性、工会力量。

**NAIRU**：不加速通胀的失业率，是政策制定参考指标。

## 四、总需求与总供给

### 4.1 总需求曲线

$Y = C + I + G + NX$，向右下方倾斜。

**三大效应**：财富效应（$P\downarrow \to$ 实际财富$\uparrow \to C\uparrow$）、利率效应（$P\downarrow \to$ 利率$\downarrow \to I\uparrow$）、汇率效应（$P\downarrow \to$ 本币贬值$\to NX\uparrow$）。

### 4.2 总供给曲线

**SRAS**：向右上方倾斜（工资黏性、价格黏性、信息不完全）。**LRAS**：垂直，取决于生产要素和技术，与价格无关。

### 4.3 AD-AS 模型

| 情景 | 短期产出 | 短期价格 | 长期产出 |
|------|----------|----------|----------|
| 需求扩张 | 增加 | 上升 | 不变 |
| 需求萎缩 | 减少 | 下降 | 不变 |
| 不利供给冲击 | 减少 | 上升（滞胀） | 减少 |

## 五、凯恩斯经济学

### 5.1 凯恩斯交叉

消费函数 $C = a + bY_d$，$b = MPC$（边际消费倾向），$MPS = 1 - MPC$。均衡 $Y = C + I + G$。

### 5.2 乘数效应

$\Delta Y = \frac{1}{1-MPC}\Delta I$，$\Delta Y = \frac{1}{1-MPC}\Delta G$，$\Delta Y = -\frac{MPC}{1-MPC}\Delta T$

| $MPC$ | 支出乘数 | 税收乘数 |
|-------|----------|----------|
| 0.5 | 2 | -1 |
| 0.75 | 4 | -3 |
| 0.8 | 5 | -4 |
| 0.9 | 10 | -9 |

### 5.3 IS-LM 模型

**IS 曲线**（产品市场均衡）：$Y = C(Y-T) + I(r) + G$，右下倾斜。财政扩张（$G\uparrow$ 或 $T\downarrow$）→ IS 右移。IS 斜率取决于投资对利率的敏感度和乘数大小。

**LM 曲线**（货币市场均衡）：$\frac{M}{P} = L(Y, r) = kY - hr$，右上倾斜。货币扩张（$M\uparrow$）→ LM 右移。LM 斜率取决于货币需求的利率弹性$h$和收入弹性$k$。

IS 与 LM 交点决定均衡利率和均衡产出。**挤出效应**：财政扩张使利率上升，抑制私人投资。挤出效应大小取决于 LM 斜率（越陡挤出越大）和 IS 斜率。**流动性陷阱**：利率极低时货币需求无限大，LM 水平，货币政策失效，财政政策最有效。

## 六、货币政策

### 6.1 货币政策工具

| 工具 | 操作 | 效果 |
|------|------|------|
| 法定准备金率 | 提高→货币乘数下降 | 强力但粗放 |
| 再贴现率 | 提高→银行借贷成本上升 | 温和被动 |
| 公开市场操作 | 卖出债券→回笼货币 | 灵活精准 |

**货币乘数**：$m = \frac{1+c}{r+c+e}$。基础货币 $B = C + R$，货币供给 $M = m \times B$。

### 6.2 泰勒规则

$i = r^* + \pi + 0.5(\pi-\pi^*) + 0.5\frac{Y-Y^*}{Y^*}$。通胀高于目标或产出高于潜在时提高利率。

### 6.3 传导机制

利率渠道 → 信贷渠道（银行借贷） → 资产价格渠道（股票房产） → 汇率渠道 → 预期渠道

## 七、财政政策

### 7.1 工具

扩张性：增支、减税、增加转移支付。紧缩性：减支、增税、减少转移支付。

### 7.2 自动稳定器

**累进所得税**（繁荣自动增税抑制过热，衰退自动减税缓冲），**失业救济金**（衰退自动增加维持消费）。**优点**：无需干预、无时滞。**局限**：力度有限。

**相机抉择政策**：针对性强但存在认识时滞、决策时滞、执行时滞。

### 7.3 赤字与公债

**结构性赤字**（经济在潜在产出时的赤字）和**周期性赤字**（经济波动导致）。公债可持续条件：$g > r$。

**李嘉图等价**（巴罗）：理性消费者预期未来加税偿还债务，减税不增加消费。因流动性约束、非理性等不完全成立。

## 八、经济增长

### 8.1 索洛模型

$Y = AK^\alpha L^{1-\alpha}$，人均 $y = Ak^\alpha$。$\dot{k} = sf(k) - (n+\delta)k$。

稳态条件：$sf(k^*) = (n+\delta)k^*$。**黄金律**：$f'(k_{gold}) = n+\delta$，最大化人均消费。

**索洛剩余**（TFP）：$\frac{\Delta A}{A} = \frac{\Delta Y}{Y} - \alpha\frac{\Delta K}{K} - (1-\alpha)\frac{\Delta L}{L}$

### 8.2 内生增长

**AK 模型** $Y = AK$，$g = sA - \delta$。**罗默模型**：知识非竞争性，R&D 驱动技术进步。**卢卡斯模型**：人力资本 $Y = AK^\alpha (hL)^{1-\alpha}$ 产生正外部性。

### 8.3 增长核算

$g_Y = g_A + \alpha g_K + (1-\alpha)g_L$。增长来源：资本深化、劳动增长、全要素生产率（TFP）。

## 九、经济周期

| 阶段 | 产出 | 就业 | 物价 |
|------|------|------|------|
| 繁荣 | 高于潜在 | 充分就业 | 上升 |
| 衰退 | 下降 | 失业上升 | 可能下降 |
| 萧条 | 严重低于潜在 | 大量失业 | 通缩风险 |
| 复苏 | 回升 | 改善 | 稳定 |

**乘数-加速数模型**：投资加速效应（产出变化引起投资倍数变化）放大乘数效应，形成内生周期。投资取决于产出变化而非产出水平，导致经济波动的自我加强。**实际经济周期理论（RBC）**：技术冲击是经济波动的主要根源，货币中性，市场始终出清。**凯恩斯周期理论**：总需求波动特别是投资的不稳定性是经济周期的主因。**政治经济周期理论**：政府为赢得选举在经济政策上操纵，选举前扩张选举后紧缩，导致有规律的经济波动。

### 9.3 宏观政策面临的挑战

**政策时滞**：认识时滞（识别问题需要时间）、决策时滞（制定政策需要时间）、执行时滞（政策生效需要时间）。**理性预期**：公众预期会抵消政策效果，只有未预期到的政策才有效。**时间不一致性**：政策制定者事先承诺的最优方案在实施时有偏离的激励，可信度至关重要。**零利率下限**：名义利率不能低于零，常规货币政策在严重衰退中受限，需使用量化宽松（QE）、前瞻指引、负利率等非常规工具。**量化宽松**通过购买长期国债和抵押贷款支持证券压低长期利率，改善信贷条件。**前瞻指引**通过沟通未来政策路径影响市场预期。这些非常规工具在2008年全球金融危机和2020年新冠疫情后被各国央行广泛采用。

**宏观审慎政策**：与微观审慎关注单个金融机构不同，宏观审慎关注系统性金融风险。工具包括逆周期资本缓冲、贷款价值比（LTV）限制、系统性重要金融机构附加资本要求。宏观审慎政策旨在防范金融体系顺周期性累积的系统性风险，维护金融稳定。

### 9.4 开放经济的宏观政策协调

在全球化背景下，各国宏观经济政策相互溢出。**蒙代尔-弗莱明模型**扩展至大国情形。**政策协调**：各国同时采取协调一致的政策比各自为政能获得更好的全球产出和通胀组合。但协调面临承诺和执行难题。**二十国集团（G20）** 是国际经济政策协调的主要平台。**国际货币基金组织（IMF）** 提供危机援助和政策监督。全球金融安全网的完善、国际储备货币体系改革、跨境资本流动管理等是国际宏观政策协调中的长期议题。
