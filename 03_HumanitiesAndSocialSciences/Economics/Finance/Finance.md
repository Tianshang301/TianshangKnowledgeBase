# 金融学

## 一、货币时间价值

### 1.1 现值与终值

**终值**：$FV = PV(1+r)^n$。**现值**：$PV = FV/(1+r)^n$。折现是金融中最核心的概念——"今天的1元钱比明天的1元钱更值钱"。

**多期现金流**：$PV = \sum_{t=1}^n \frac{CF_t}{(1+r)^t}$

**年金**：等额定期现金流。普通年金现值：$PV = PMT \times \frac{1-(1+r)^{-n}}{r}$。永续年金（英国统一公债）：$PV = PMT/r$。增长永续年金（戈登模型）：$PV = PMT/(r-g)$。

### 1.2 投资项目评价

| 方法 | 公式 | 决策规则 | 优点 | 缺点 |
|------|------|----------|------|------|
| **净现值（NPV）** | $\sum CF_t/(1+r)^t$ | NPV>0接受 | 直接度量价值创造 | 依赖折现率 |
| **内部收益率（IRR）** | NPV=0的解$r$ | IRR>资本成本 | 直观收益率 | 非常规现金流多解 |
| **回收期** | 累计CF转正时间 | 短于目标年限 | 简单 | 忽略时间价值和后期CF |
| **盈利指数（PI）** | PV/初始投资 | PI>1 | 资本配置参考 | 规模效应问题 |
| **经济增加值（EVA）** | 税后利润-资本成本 | EVA>0 | 联系会计与金融 | 计算复杂 |

**NPV与IRR冲突**：互斥项目规模不同或现金流时间模式不同时，NPV准则优先。修正IRR（MIRR）可解决部分问题。

## 二、资产定价

### 2.1 马科维茨组合理论

**均值-方差模型**：投资者在给定风险（方差）下追求最大期望收益。**有效前沿**（efficient frontier）是给定风险下最大收益的组合集。**最小方差组合**（MVP）是有效前沿的左端点。

**组合风险公式**：$\sigma_p^2 = w_1^2\sigma_1^2 + w_2^2\sigma_2^2 + 2w_1w_2\sigma_1\sigma_2\rho_{12}$

分散化降低风险的程度取决于相关系数$\rho$：$\rho=1$无分散化效果，$\rho=-1$可完全对冲。现实中$\rho$多在0-0.5之间。

### 2.2 CAPM

**资本资产定价模型**（Sharpe, 1964；Lintner, 1965）：$E(R_i) = R_f + \beta_i(E(R_m) - R_f)$

$\beta_i = \frac{Cov(R_i,R_m)}{Var(R_m)}$——衡量系统性风险。

| $\beta$ | 含义 | 例子 |
|---------|------|------|
| $\beta=1$ | 与市场同步波动 | 市场指数基金 |
| $\beta>1$ | 比市场波动大（高系统性风险） | 科技股、券商股 |
| $0<\beta<1$ | 比市场波动小（低系统性风险） | 公用事业、消费必需品 |
| $\beta=0$ | 无系统性风险 | 国债 |
| $\beta<0$ | 与市场反向（极少见） | 黄金在某些时期 |

**证券市场线（SML）**：横轴$\beta$，纵轴期望收益，斜率=市场风险溢价$E(R_m)-R_f$。

### 2.3 有效市场假说

法玛（Fama, 1970）：**弱式**（历史价格无效）、**半强式**（公开信息无效，事件研究检验）、**强式**（所有信息无效，不成立）。

**市场异象**：规模效应（小公司超额收益）、价值效应（低市净率超额收益）、动量效应（短期趋势延续）、反转效应（长期均值回归）。行为金融认为异象来自投资者非理性。

## 三、公司金融

### 3.1 MM定理

Modigliani-Miller定理（1958, 1963）是公司金融基石。

**无税MM定理**：$V_L = V_U$（公司价值与资本结构无关）；$r_E = r_A + \frac{D}{E}(r_A - r_D)$（权益成本随杠杆上升）。

**有税MM定理**：$V_L = V_U + Tc \times D$（税盾价值）；$r_E = r_A + \frac{D}{E}(1-Tc)(r_A - r_D)$。

**拓展**：权衡理论（税盾 vs 财务困境成本）、优序融资理论（内部→债务→股权）、代理理论（股东-经理-债权人冲突）。

### 3.2 资本预算

**加权平均资本成本（WACC）**：$WACC = \frac{E}{V}r_E + \frac{D}{V}r_D(1-Tc)$。WACC作为折现率评估项目。

**调整现值法（APV）**：$APV = NPV_{全权益} + NPV_{融资效应}$。将经营决策和融资决策分开，更灵活。

## 四、衍生品

### 4.1 远期与期货

**远期合约**：OTC定制化，双方约定未来以特定价格买卖资产。**期货**：交易所标准化，每日盯市结算。

**期货定价**（持有成本模型）：$F = S_0 e^{(r-y)T}$，其中$y$为便利收益/红利。

**套期保值比率**：$h = \frac{Cov(\Delta S,\Delta F)}{Var(\Delta F)}$。最优最小方差避险比率。

### 4.2 期权

**基本类型**：看涨（Call）——买入权；看跌（Put）——卖出权。**行权方式**：欧式（到期日行权）、美式（到期前任何时间）。

**Black-Scholes公式**（1973）：

$C = S_0 N(d_1) - Ke^{-rT}N(d_2)$，$P = Ke^{-rT}N(-d_2) - S_0 N(-d_1)$

其中$d_1 = \frac{\ln(S_0/K) + (r+\sigma^2/2)T}{\sigma\sqrt{T}}$，$d_2 = d_1 - \sigma\sqrt{T}$

**希腊字母**：$\Delta$（价格敏感性）、$\Gamma$（Delta对标的的敏感性）、$\Theta$（时间衰减）、$Vega$（波动率敏感性）、$\rho$（利率敏感性）。

### 4.3 期权策略

| 策略 | 构成 | 适用 |
|------|------|------|
| 保护性看跌 | 股票+看跌 | 对冲下行风险 |
| 备兑开仓 | 股票+卖出看涨 | 获取权利金，牺牲上行 |
| 跨式 | 同时买入看涨+看跌 | 预期大幅波动 |
| 蝶式 | 多个期权组合 | 预期窄幅波动 |

## 五、风险管理

### 5.1 风险类型

| 类型 | 来源 | 管理工具 |
|------|------|----------|
| 市场风险 | 利率、汇率、股价、商品价格波动 | 衍生品对冲、VaR、压力测试 |
| 信用风险 | 交易对手违约 | 抵押、CDS、评级、信用分析 |
| 流动性风险 | 无法及时以公价成交 | 流动性储备、流动性覆盖率(LCR) |
| 操作风险 | 流程、系统、人员失误 | 内控、保险、BCP |
| 系统性风险 | 整个金融体系危机 | 宏观审慎监管 |

### 5.2 在险价值（VaR）

$P(\Delta P \leq -VaR) = 1-\alpha$。95%置信水平、日VaR=100万意味着有5%概率日损失超100万。

**方法**：参数法（方差-协方差，假设正态分布）、历史模拟法（过去收益分布）、蒙特卡洛模拟（随机模拟路径）。

**局限性**：不满足次可加性（分散化不降低VaR）、不刻画尾部极端损失（尾部风险）。**CVaR/Expected Shortfall**作为补充。

## 六、行为金融

挑战传统金融的理性人假设：

| 偏差 | 描述 | 市场影响 |
|------|------|----------|
| 过度自信 | 高估判断精确性 | 频繁交易→低收益（Odean实证） |
| 损失厌恶 | 损失痛苦>等量收益快乐(约2倍) | 持有亏损股票过早卖出盈利股 |
| 锚定效应 | 过分依赖初始参考点 | 对历史价格过度关注 |
| 羊群效应 | 跟随他人行为 | 追涨杀跌，放大波动 |
| 心理账户 | 分类管理不同资金 | 对工资和奖金不同态度 |
| 可得性启发 | 易想起的事件概率更高 | 近期灾难后过度避险 |

**前景理论**（Kahneman-Tversky, 1979）：价值函数在收益域凸（风险厌恶）、损失域凹（风险寻求），且损失域更陡峭（损失厌恶）。
