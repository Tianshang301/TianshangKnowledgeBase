# 多元统计分析方法

## 一、多元统计分析概述

多元统计分析是研究多个随机变量之间相互依赖关系的统计方法体系。当研究对象的特征需用多个变量共同描述时，单变量分析方法无法揭示变量之间的复杂关联结构，需要借助多元统计方法进行综合分析。

多元统计分析的核心问题包括：多元数据的描述和可视化、总体参数的估计与假设检验、变量降维与特征提取、分类与判别、变量间相互关系的建模等。现代多元分析的应用涵盖生物统计、金融分析、市场研究、心理测量、经济计量等广泛领域。

## 二、多元正态分布与统计推断

多元正态分布是多元统计分析的理论基础。$p$维随机向量$\mathbf{X} = (X_1, X_2, \ldots, X_p)'$服从多元正态分布的概率密度函数为：

$$
f(\mathbf{x}) = \frac{1}{(2\pi)^{p/2} |\mathbf{\Sigma}|^{1/2}} \exp\left[-\frac{1}{2} (\mathbf{x} - \boldsymbol{\mu})' \mathbf{\Sigma}^{-1} (\mathbf{x} - \boldsymbol{\mu})\right]
$$

其中$\boldsymbol{\mu}$为均值向量，$\mathbf{\Sigma}$为协方差矩阵。

多元正态分布的重要性质包括：线性变换保持正态性、边缘分布为正态分布、条件分布为正态分布。当$p=2$时，二元正态分布的密度曲面呈钟形，等高线为椭圆。协方差$\sigma_{12}$决定了椭圆的方向——相关程度越高，椭圆越狭长。

多元均值向量的检验使用霍特林$T^2$统计量：

$$
T^2 = n(\bar{\mathbf{x}} - \boldsymbol{\mu}_0)' \mathbf{S}^{-1} (\bar{\mathbf{x}} - \boldsymbol{\mu}_0)
$$

其中$\mathbf{S}$为样本协方差矩阵。$T^2$统计量与$F$分布的关系为：$\frac{n-p}{p(n-1)} T^2 \sim F_{p, n-p}$。

## 三、主成分分析

主成分分析（PCA）是最常用的降维方法，由皮尔逊（Pearson, 1901）提出、霍特林（Hotelling, 1933）发展。PCA 通过线性变换将原始变量转换为一组互不相关的新变量（主成分），且主成分按方差大小排序，使得前$k$个主成分尽可能多地保留原始数据的信息。

主成分的求解归结为协方差矩阵（或相关矩阵）的特征值分解：

$$
\mathbf{\Sigma} \mathbf{v}_i = \lambda_i \mathbf{v}_i
$$

其中$\lambda_1 \geq \lambda_2 \geq \cdots \geq \lambda_p \geq 0$为特征值，$\mathbf{v}_i$为对应的特征向量。第$i$个主成分为$Y_i = \mathbf{v}_i' \mathbf{X}$，其方差为$\lambda_i$。第$i$个主成分的方差贡献率为$\lambda_i / \sum_{j=1}^p \lambda_j$。

主成分个数通常根据累计算贡献率（如达到80%-90%）、特征值大于1的 Kaiser 准则或碎石图（Scree Plot）来选定。PCA 的应用领域包括：高维数据可视化、多重共线性处理、特征压缩和噪声过滤。

## 四、因子分析

因子分析（Factor Analysis）探索观测变量背后的潜在结构（公共因子）。与 PCA 不同，因子分析将变量分解为公共因子和特殊因子两部分：

$$
\mathbf{X} = \boldsymbol{\mu} + \mathbf{LF} + \boldsymbol{\epsilon}
$$

其中$\mathbf{X}$为$p$维观测变量，$\mathbf{L}$为$p \times m$的因子载荷矩阵，$\mathbf{F}$为$m$维公共因子向量（$m < p$），$\boldsymbol{\epsilon}$为特殊因子向量。

因子载荷$a_{ij}$表示变量$i$在因子$j$上的载荷，其平方和（共性方差）反映了变量被公共因子解释的程度。因子旋转（如最大方差旋转 Varimax）通过旋转因子载荷矩阵使载荷结构更加简单、易于解释。

因子分析与 PCA 的关键区别在于：PCA 是对变量的线性组合进行方差最大化旋转，因子分析则假设观测变量受潜在因子的驱动。在实际应用中，因子分析更适合于量表开发、心理构念测量和指标综合。

## 五、聚类分析

聚类分析是将研究对象按相似性分成若干类别的无监督学习方法。其主要步骤包括：选择距离/相似度度量、选择聚类算法、确定聚类数量、解释聚类结果。

常用的距离度量包括：（一）欧氏距离：$d(\mathbf{x}_i, \mathbf{x}_j) = \sqrt{\sum_{k=1}^p (x_{ik} - x_{jk})^2}$；（二）曼哈顿距离：$d(\mathbf{x}_i, \mathbf{x}_j) = \sum_{k=1}^p |x_{ik} - x_{jk}|$；（三）马氏距离：$d(\mathbf{x}_i, \mathbf{x}_j) = \sqrt{(\mathbf{x}_i - \mathbf{x}_j)' \mathbf{S}^{-1} (\mathbf{x}_i - \mathbf{x}_j)}$。

聚类方法可分为两类：

（一）层次聚类法：通过逐次合并（凝聚法）或分裂（分裂法）建立聚类树状图。类间距离的度量方法包括最短距离法、最长距离法、类平均法和瓦尔德法（离差平方和法）。

（二）划分聚类法：指定聚类数量后通过迭代优化将样本分配到不同类别。以 K-means 算法最为经典，通过迭代更新聚类中心最小化类内平方和：

$$
\min \sum_{k=1}^K \sum_{i \in C_k} \|\mathbf{x}_i - \boldsymbol{\mu}_k\|^2
$$

| 方法 | 优点 | 缺点 |
|------|------|------|
| 层次聚类 | 无需预指定 K，可视化树状图 | 计算复杂度高，不可逆 |
| K-means | 高效，适合大样本 | 需预指定 K，对初始值敏感 |
| DBSCAN | 可识别任意形状，抗噪声 | 对密度参数敏感 |

## 六、判别分析

判别分析是根据已知分类的样本构建判别规则，用于对新样本进行分类的有监督方法。线性判别分析（LDA，Fisher, 1936）通过寻找投影方向使类间离散度与类内离散度的比值最大化。

Fisher 线性判别函数的基本思想是最大化以下准则：

$$
J(\mathbf{w}) = \frac{\mathbf{w}' \mathbf{S}_B \mathbf{w}}{\mathbf{w}' \mathbf{S}_W \mathbf{w}}
$$

其中$\mathbf{S}_B$为类间离散矩阵，$\mathbf{S}_W$为类内离散矩阵。最优投影方向$\mathbf{w}^*$为$\mathbf{S}_W^{-1} \mathbf{S}_B$的最大特征值对应的特征向量。

以贝叶斯判别为基础的二次判别分析（QDA）假设各类的协方差矩阵不同，判别边界为二次曲面。在实际应用中，选择 LDA 还是 QDA 需根据各类协方差矩阵的异质性和样本量决定。

## 七、对应分析与多维尺度分析

对应分析（Correspondence Analysis）用于分析两个或多个分类变量之间的关联关系，将列联表的行和列映射到低维空间，揭示变量类别之间的对应模式。对应分析是主成分分析在分类数据上的推广。

多维尺度分析（MDS）基于对象之间的不相似性矩阵，在低维空间中重建对象间的距离关系。MDS 广泛应用于消费者感知图分析、品牌定位研究和社会网络分析。

多元分析方法的适用场景总结如下：

| 方法 | 数据类型 | 分析目标 | 是否有监督 |
|------|---------|---------|-----------|
| PCA | 连续变量 | 降维 | 无 |
| 因子分析 | 连续变量 | 潜在结构 | 无 |
| 聚类分析 | 连续/分类 | 分类 | 无 |
| 判别分析 | 连续变量 | 分类预测 | 有 |
| 对应分析 | 分类变量 | 关联可视化 | 无 |
| MDS | 距离矩阵 | 空间映射 | 无 |

## 八、多元分析软件与实务

常用的多元统计分析软件包括：R（功能最全面，免费开源）、Python（scikit-learn、statsmodels 等库）、SPSS（操作简便，适合教学）、Stata（计量经济学导向）、SAS（企业级商业分析）和 MATLAB（数值计算优势）。

多元分析的实际应用需要注意：变量选择和预处理（异常值处理、缺失值处理、变量标准化）、模型假设的验证（正态性、独立性、方差齐性）、过拟合的控制（交叉验证、正则化）和结果的合理解释（统计显著性与实际显著性的区分）。

## 关键概念辨析
概念定义的清晰度直接影响研究的质量。
以下是该领域中若干容易混淆的概念。

**概念一与概念二的区分**：
前者侧重于外在的形式特征。
后者关注内在的运作机制。
两者在实际分析中往往需要结合使用。

**微观与宏观层面的联系**：
微观现象是宏观结构的基础。
宏观结构又约束微观行为。
理解两者的相互作用是社会分析的核心。

**静态分析与动态分析**：
静态分析关注某一时点的截面特征。
动态分析关注过程和变化的轨迹。
两种视角互补而非替代。
