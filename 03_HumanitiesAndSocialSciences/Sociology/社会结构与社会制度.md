# 社会结构与社会制度

## 一、社会分层

### 1.1 分层的基本维度

社会分层是社会中稀缺资源和有价值物品不平等分配的结构化体系。
凡是存在持续性社会不平等的群体中，就存在社会分层。
分层不仅是个体差异的结果，更是社会制度安排的系统产物。

构成社会分层的核心维度包括四个方面：

| 维度 | 定义 | 测量指标 | 理论重心 |
|------|------|---------|---------|
| 财富 | 经济资源和资产的占有 | 收入、房产、股票、储蓄 | 马克思的阶级分析 |
| 权力 | 影响他人行为和决策的能力 | 职位等级、决策参与度 | 韦伯的权威分析 |
| 声望 | 社会尊重和荣誉的评价 | 职业声望评分 | 布迪厄的象征资本 |
| 教育 | 知识和技能的系统获取 | 学历层次、资格证书 | 布迪厄的文化资本 |

四个维度之间相互关联但不完全重叠。
教育可以转化为财富和声望，财富可以购买教育机会。
但在不同社会和不同历史时期，四种资源的转换程度存在较大差异。

### 1.2 分层体系的类型

人类社会历史上存在四种主要的社会分层体系：

| 类型 | 特征 | 社会流动 | 合法性基础 | 典型社会 |
|------|------|---------|-----------|---------|
| 奴隶制 | 人作为财产被占有 | 基本无流动 | 种族/法律 | 古希腊、美国南方 |
| 种姓制 | 出身决定终身地位 | 极低 | 宗教纯洁观 | 印度传统社会 |
| 等级制 | 法律赋予不同特权 | 低 | 血统/封赠 | 中世纪欧洲 |
| 阶级制 | 经济地位决定分层 | 较高 | 个人成就/市场 | 现代资本主义社会 |

当代社会通常呈现混合特征。
阶级制中仍然残留种姓制和等级制的因素，如种族性不流动。

### 1.3 社会流动

社会流动是指个体或群体在社会分层体系中的位置变化。
一个社会的开放程度主要体现在社会流动率的高低上。

四种基本的社会流动类型：

| 类型 | 定义 | 研究关注点 |
|------|------|-----------|
| 代内流动 | 个体一生中社会地位的变化 | 职业生涯发展、失业与贫困化 |
| 代际流动 | 子女相对于父母的社会地位变化 | 阶层固化、机会均等 |
| 垂直流动 | 在社会等级中向上或向下的移动 | 社会晋升与下行流动 |
| 水平流动 | 同层级内的地理或职业变动 | 职业变迁与人口迁移 |

### 1.4 社会流动的促进与障碍

| 促进因素 | 障碍因素 |
|----------|---------|
| 普及而平等的教育体系 | 教育资源分配不均 |
| 经济持续增长提供优质岗位 | 结构性失业和经济转型 |
| 反歧视法律与积极行动政策 | 制度性歧视与隐含偏见 |
| 广泛的社会资本网络 | 阶层固化与社会排斥 |

布劳-邓肯的地位获得模型是社会流动研究的经典范本。
通过路径分析发现教育是连接家庭背景与职业地位的关键中介变量。

## 二、阶级理论

### 2.1 马克思的阶级理论

马克思以生产资料占有关系作为定义阶级的根本标准。

在资本主义社会中两大基本阶级及其关系：

| 阶级 | 生产资料 | 核心利益 | 历史命运 |
|------|---------|---------|---------|
| 资产阶级 | 占有工厂、机器、土地 | 榨取剩余价值 | 被无产阶级推翻 |
| 无产阶级 | 仅拥有自身劳动力 | 提高工资和解放 | 成为革命主体 |

除了这两大阶级外，还存在小资产阶级。
小资产阶级拥有少量生产资料，但在资本积累过程中不断向两端分化。

### 2.2 韦伯的多元阶级理论

韦伯以"市场处境"定义阶级，即个体在商品市场和劳动力市场中的谈判机会。

韦伯区分了三种阶级类型：

| 类型 | 决定因素 | 特征 |
|------|---------|------|
| 财产阶级 | 财产差异 | 基本生活机会不同 |
| 职业阶级 | 市场技能 | 职业分层和收入差异 |
| 社会阶级 | 代际流动模式 | 社会整体阶级结构 |

韦伯不认为阶级会自动产生集体行动。
从客观的阶级处境到主观的阶级意识再到实际的阶级行动，需要一系列条件的配合。

### 2.3 布迪厄的资本理论

布迪厄扩展了马克思的资本概念，提出了多维资本理论：

| 资本类型 | 含义 | 转换方式 |
|----------|------|---------|
| 经济资本 | 可直接转换为货币的财富 | 购买教育、进入精英圈子 |
| 文化资本 | 知识、品味、教育证书 | 通过教育获得经济回报 |
| 社会资本 | 社会网络和关系资源 | 促进职业发展和信息获取 |
| 象征资本 | 被合法化的资本形式 | 将经济资本转化为声望 |

文化资本的三种状态：

| 状态 | 表现形式 |
|------|---------|
| 具身状态 | 知识、品味、言谈举止 |
| 客观状态 | 书籍、艺术品、教育证书 |
| 制度化状态 | 学历、资格证书 |

布迪厄的《区隔》一书展示了阶级如何通过日常生活中的品味被持续再生产。

### 2.4 当代阶级框架

| 理论家 | 核心观点 | 分类标准 |
|--------|---------|---------|
| 赖特 | 多重剥削位置 | 生产资料、组织资产、技能/证书 |
| 戈德索普 | 以雇佣关系为核心 | 就业关系类型（服务关系 vs 劳动契约） |

戈德索普的七大阶级分类：

| 阶级 | 典型职业 |
|------|---------|
| I-高级专业人员/管理人员 | 医生、律师、企业高管 |
| II-低级专业人员/管理人员 | 教师、护士、中层经理 |
| III-常规白领 | 文员、销售员 |
| IV-小资产阶级 | 小店主、自雇者 |
| V-技术蓝领 | 电工、水管工 |
| VI-半技术蓝领 | 流水线工人 |
| VII-非技术蓝领 | 清洁工、搬运工 |

### 2.5 中产阶级的兴起与萎缩

二战后中产阶级大规模扩张，被视为社会稳定的"缓冲带"。
然而从1970年代开始，中产阶级面临日益严峻的挑战：

| 阶段 | 特征 |
|------|------|
| 扩张期（1945-1970） | 白领岗位增加，收入持续增长 |
| 挤压期（1970-2000） | 收入增长停滞，住房成本激增 |
| 极化期（2000至今） | 高技能高薪和低技能低薪岗位同时增加，中间岗位萎缩 |

## 三、社会不平等

### 3.1 收入不平等

基尼系数是衡量收入不平等最广泛使用的指标：

| 基尼系数 | 不平等程度 | 典型国家 |
|---------|-----------|---------|
| < 0.25 | 高度平等 | 瑞典、挪威 |
| 0.25-0.35 | 相对平等 | 德国、法国 |
| 0.35-0.45 | 中度不平等 | 美国、中国 |
| > 0.45 | 高度不平等 | 南非、巴西 |

### 3.2 性别不平等

| 现象 | 表现 | 原因 |
|------|------|------|
| 职业隔离 | 女性集中在教育、护理等低薪行业 | 性别社会化、刻板印象 |
| 工资差距 | 同等工作女性收入低于男性 | 直接歧视、职业中断 |
| 玻璃天花板 | 女性难以晋升到高层管理岗位 | 无意识偏见、网络排斥 |
| 第二班 | 职场女性承担更多家务和育儿 | 传统性别角色分工未变 |

### 3.3 种族/民族不平等

| 概念 | 含义 | 举例 |
|------|------|------|
| 结构性歧视 | 制度在结果上造成种族差异 | 住房贷款的种族差异审批 |
| 制度性种族主义 | 组织常规中嵌入的种族不平等 | 学校纪律处分的种族差异 |
| 隐含偏见 | 无意识的刻板印象影响判断 | 简历筛选中的种族偏好 |

### 3.4 教育不平等

布迪厄的文化再生产理论认为学校再生产阶级不平等：

| 机制 | 过程 |
|------|------|
| 文化资本传递 | 上层阶级子女拥有更多符合学校标准的文化资本 |
| 教师期望 | 教师评价偏向中产阶级的文化实践 |
| 合法化叙事 | 阶级特权转化为"个人能力"的天然差异 |

## 四、社会制度

### 4.1 家庭制度

家庭是社会化的首要场所：

| 功能 | 具体内容 |
|------|---------|
| 社会化 | 将文化规范、价值观传递给下一代 |
| 情感支持 | 提供亲密关系和安全依恋 |
| 经济合作 | 资源共担、消费共享 |

当代家庭变迁趋势：

| 趋势 | 表现 | 原因 |
|------|------|------|
| 核心家庭化 | 从扩展家庭到核心家庭 | 工业化、城市化 |
| 离婚率上升 | 离婚数量持续增加 | 法律放宽、女性经济独立 |
| 家庭形式多元化 | 单亲、丁克、再组、同性家庭 | 个体化价值观 |

### 4.2 宗教制度

| 理论传统 | 核心观点 |
|----------|---------|
| 涂尔干 | 宗教是集体的自我崇拜——神圣与世俗的分立维系社会整合 |
| 韦伯 | 新教伦理的天职观念和禁欲主义催生了资本主义精神 |
| 世俗化命题 | 现代化导致宗教权威衰微 |
| 反世俗化 | 宗教在全球范围内仍然活跃 |

宗教类型：

| 类型 | 特征 |
|------|------|
| 教会 | 大规模，接受并融入社会 |
| 教派 | 中规模，与主流社会关系复杂 |
| 宗派 | 小规模，排斥或脱离主流社会 |
| 膜拜团体 | 小规模，创新性宗教形式 |

### 4.3 教育制度

| 功能主义视角 | 冲突论视角 |
|-------------|-----------|
| 知识传递与社会化 | 文化再生产——维护阶级不平等 |
| 社会筛选与人才选拔 | 文凭通胀——教育证书贬值 |
| 促进社会流动 | 标签效应——教师期望自我实现 |

鲍尔斯和金蒂斯的"对应原则"认为教育的结构对应资本主义生产结构。
等级制、外在激励、碎片化任务在教育中再现了劳动场所的权力关系。

### 4.4 社会制度综合比较

| 制度 | 核心功能 | 主要变迁趋势 |
|------|---------|-------------|
| 家庭 | 社会化、情感支持、经济合作 | 多元化和去制度化 |
| 教育 | 知识传递、社会筛选、文化再生产 | 扩张与文凭通胀 |
| 宗教 | 提供意义、社会整合、伦理规范 | 世俗化与宗教复兴 |
| 政府 | 社会秩序、公共物品、利益分配 | 福利国家到新自由主义 |
