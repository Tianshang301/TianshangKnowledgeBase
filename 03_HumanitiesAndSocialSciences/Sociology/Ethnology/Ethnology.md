# 民族社会学

## 一、基本概念

### 1.1 民族与族群

| 概念 | 核心特征 | 社会学视角 |
|------|---------|-----------|
| 民族（ethnic group） | 共享文化、语言、历史、祖先认同的群体 | 社会建构的范畴 |
| 种族（race） | 基于身体特征的群体分类 | 社会建构而非生物本质 |
| 民族国家（nation-state） | 单一民族为主体的国家 | 现代性的产物 |

**原生论 vs 工具论**

| 视角 | 核心观点 | 代表 |
|------|---------|------|
| 原生论 | 族群归属是基于血缘、语言等原生纽带 | Geertz |
| 工具论 | 族群认同是利益竞争中被动员的资源 | Barth |
| 建构论 | 族群是历史和社会建构的产物 | Anderson |

### 1.2 族群边界

Barth（1969）提出族群研究的核心不是"族群的文化内涵"，而是"族群边界"——维持群体区分的社会过程。族群边界通过社会互动得以维持和强化，文化内容可以变化而边界可以延续。

Anderson 提出"想象的共同体"概念：民族是被想象出来的政治共同体——尽管成员之间互不相识，但他们共享一种"共属"的想象。印刷资本主义和国家教育体系是民族想象的关键媒介。

## 二、同化与多元

### 2.1 同化理论

**经典同化理论**（Gordon, 1964）

| 同化维度 | 内容 |
|---------|------|
| 文化同化 | 语言、习俗、价值观的改变 |
| 结构同化 | 进入主流社会的制度和社交网络 |
| 婚姻同化 | 族际通婚 |
| 身份同化 | 认同转向主流社会 |
| 态度同化 | 消除偏见 |
| 行为同化 | 无歧视对待 |
| 公民同化 | 价值冲突的消除 |

**分段同化**（Portes & Zhou, 1993）

| 同化路径 | 特征 | 例 |
|---------|------|-----|
| 向上同化 | 融入中产阶级主流 | 欧洲移民后代 |
| 向下的同化 | 融入底层或亚文化 | 部分非裔移民 |
| 选择性同化 | 保留族群文化+经济成功 | 部分亚裔移民 |

### 2.2 多元文化主义

多元文化主义承认并尊重文化多样性，主张少数群体有权维持自己的文化身份。政策工具包括双语教育、文化敏感的服务、向少数群体开放公共机构等。

多元文化主义的批评来自两面：保守派认为它削弱了国家认同和社会凝聚；激进派认为它掩盖了结构性的不平等。

## 三、偏见与歧视

### 3.1 偏见

| 偏见层次 | 表现 | 测量 |
|---------|------|------|
| 外显偏见 | 有意识、可报告的态度 | 量表（现代种族主义量表） |
| 内隐偏见 | 无意识的自动联想 | IAT（内隐联想测验） |
| 微歧视 | 日常的、微妙的冒犯 | 定性研究 |

**偏见的社会根源**：群体冲突理论（资源竞争导致偏见）、社会身份理论（内群体偏向和外群体贬低）、社会化（家庭和文化的学习）。

### 3.2 歧视

| 类型 | 定义 | 例 |
|------|------|-----|
| 个人歧视 | 基于族群身份的个别行为 | 房东拒绝租房给少数族裔 |
| 制度歧视 | 制度和政策造成的不平等对待 | 红线政策、种族隔离法 |
| 结构性歧视 | 多重制度交织的系统性排斥 | 教育→就业→居住→健康的恶性循环 |

## 四、移民

### 4.1 国际移民趋势

| 时期 | 主要流向 | 推动因素 |
|------|---------|---------|
| 19世纪 | 欧洲→美洲 | 工业化、人口压力 |
| 二战后 | 前殖民地→宗主国 | 劳动力短缺 |
| 1965年后 | 全球多样化 | 全球化、政策变革 |
| 当代 | 南→北、南→南 | 经济差距、冲突、气候变化 |

### 4.2 移民的跨国主义

跨国主义视角关注移民同时参与两个或多个社会的跨界实践。移民保持与原籍国的经济（汇款、投资）、社会（家庭、社区）和政治（选举、政策影响）联系。

汇款是发展经济学的重要议题——全球移民汇款总额远超过官方发展援助（ODA）。

## 五、族群冲突

### 5.1 冲突类型

| 类型 | 特征 | 例 |
|------|------|-----|
| 种族灭绝 | 系统性地消灭一个群体 | 卢旺达（1994） |
| 种族清洗 | 以暴力驱赶族群 | 波黑战争 |
| 种族骚乱 | 族群间的暴力事件 | 印尼排华（1998） |
| 低烈度冲突 | 持续紧张与零星暴力 | 缅甸罗兴亚人 |

### 5.2 冲突的解释

| 理论 | 核心观点 |
|------|---------|
| 原初仇恨 | 历史宿怨导致当下冲突 |
| 精英操纵 | 政治精英利用族群动员谋取权力 |
| 资源竞争 | 稀缺资源争夺恶化族群关系 |
| 安全困境 | 一群体自卫行为被对方视为威胁 |
| 失败国家 | 国家能力弱导致族群冲突失控 |

### 5.3 冲突管理

**协和制民主**（Consociationalism, Lijphart）

| 原则 | 内容 | 例 |
|------|------|-----|
| 大联合政府 | 所有族群政党分享行政权 | 瑞士、比利时 |
| 相互否决 | 少数群体有权否决对其不利的决策 | 北爱尔兰 |
| 比例原则 | 行政、立法、资源按族群比例分配 | 黎巴嫩（历史） |
| 群体自治 | 族群在文化事务上的自治权 | 比利时语言区 |

## 六、中国的民族

### 6.1 民族分类

中国政府正式识别并确认56个民族。汉族约占91.5%，55个少数民族约占8.5%。少数民族人口最多的有壮族、回族、满族、维吾尔族、苗族等。

识别标准（1950s）：斯大林提出的民族定义——共同语言、共同地域、共同经济生活、共同心理素质。

### 6.2 民族区域自治

中国在少数民族聚居地区实行民族区域自治制度。自治区（5个）、自治州（30个）、自治县（120个）构成了多层次的自治体系。

| 层级 | 例 |
|------|-----|
| 自治区 | 新疆、西藏、内蒙古、广西、宁夏 |
| 自治州 | 延边朝鲜族自治州、凉山彝族自治州 |
| 自治县 | 峨山彝族自治县、丽江纳西族自治县（历史） |

### 6.3 当代民族议题

| 议题 | 讨论 |
|------|------|
| 民族认同与国家认同 | 中华民族多元一体格局（费孝通） |
| 民族平等与优惠政策 | 高考加分、生育政策差异 |
| 文化保护与发展 | 双语教育、非物质文化遗产 |
| 反分裂与国家安全 | 边疆治理、去极端化 |

## 七、美国的种族关系

| 时期 | 关键事件 | 制度特征 |
|------|---------|---------|
| 奴隶制（1619-1865） | 南北战争、解放宣言 | 奴隶制度 |
| 吉姆·克劳（1876-1965） | 民权运动、布朗案 | 法律上的种族隔离 |
| 后民权时代（1965-） | 平权行动、黑命贵运动 | 法律平等但事实不平等 |

美国种族不平等在收入、财富、教育、医疗、司法等领域仍然显著。

## 八、民族研究的理论与方法

### 8.1 研究伦理

民族研究涉及敏感的族群议题，研究伦理尤为重要：

| 原则 | 含义 | 实践 |
|------|------|------|
| 知情同意 | 充分告知研究对象研究目的和风险 | 使用研究对象能理解的语言 |
| 互惠 | 研究应使被研究群体受益 | 研究成果回馈社区 |
| 去殖民化 | 反思研究中的权力关系 | 让社区参与研究设计 |
| 文化胜任 | 研究者的文化敏感性和知识 | 学习研究对象的文化 |

### 8.2 测量问题

族群研究的方法论挑战：

| 挑战 | 内容 |
|------|------|
| 分类的稳定性 | 族群身份可以随情境变化 |
| 自我认同 vs 外部认定 | 官方分类与个人认同可能不一致 |
| 混合身份 | 多族群背景人口的测量复杂性 |
| 比较的跨文化有效性 | 同一概念在不同文化中的含义差异 |

### 8.3 当代争论

| 议题 | 争论焦点 |
|------|---------|
| 族群统计 | 收集族群数据有助于识别不平等还是加强了种族化分类？ |
| 肤色歧视 | 在同一族群内部，肤色深浅是否影响社会地位？ |
| 肯定性行动 | 基于族群的优惠政策是否公平？ |
| 标识政治 | 族群身份成为政治动员的基础是解放还是限制？ |

## 九、全球视角的民族问题

### 9.1 欧洲的移民与整合

欧洲面临的主要民族议题包括：穆斯林移民的文化整合、极右翼政党崛起和多元文化主义的危机。不同国家的整合模式各异——法国的共和同化模式、德国的族群文化模式和英国的多元文化模式。

| 整合模式 | 特征 | 挑战 |
|---------|------|------|
| 法国同化模式 | 强调公民身份、弱化族群差异 | 对文化差异的压制 |
| 德国族群模式 | 以血缘为基础的公民权 | 排斥长期居住的非德裔 |
| 英国多元模式 | 承认群体差异的公共角色 | 社区隔离和极端化 |

### 9.2 原住民权利

全球原住民运动要求承认土地权利、文化自治和政治代表权。联合国《原住民权利宣言》（2007）确立了原住民的自决权、土地权和文化权。

### 9.3 种族灭绝的预防

种族灭绝是族群冲突的最极端形式。预防机制包括：

| 预警机制 | 预防措施 |
|---------|---------|
| 人权监测 | 联合国调查、国际刑事法院 |
| 和平干预 | 外交斡旋、经济制裁 |
| 保护责任（R2P） | 国际社会的保护责任 |
| 过渡司法 | 真相委员会、刑事追诉 |
