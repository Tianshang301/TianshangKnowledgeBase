# 人口学

## 一、基本测量指标

### 1.1 人口规模与结构

| 指标 | 公式 | 含义 |
|------|------|------|
| 粗出生率（CBR） | （年出生人口 / 年中总人口）× 1000 | 每千人年出生数 |
| 粗死亡率（CDR） | （年死亡人口 / 年中总人口）× 1000 | 每千人年死亡数 |
| 自然增长率（RNI） | CBR - CDR | 人口自然增加速度 |
| 总和生育率（TFR） | 各年龄别生育率之和 | 妇女一生平均生育数 |
| 出生时预期寿命 | 生命表计算 | 平均可活年数 |
| 婴儿死亡率（IMR） | （婴儿死亡数 / 活产数）× 1000 | 不满1岁婴儿死亡比例 |

### 1.2 人口金字塔

人口金字塔是反映人口年龄和性别结构的可视化工具。三种基本类型：

| 类型 | 形状 | 特征 | 例 |
|------|------|------|-----|
| 扩张型 | 底部宽、顶部窄 | 高出生率、高死亡率 | 尼日尔 |
| 稳定型 | 各层接近 | 低出生率、低死亡率 | 瑞典 |
| 收缩型 | 底部窄、中间宽 | 出生率下降、老龄化 | 日本、中国 |

### 1.3 生命表

生命表是描述同批人死亡过程的统计模型：

| 符号 | 含义 |
|------|------|
| $l_x$ | 活到x岁的人数 |
| $d_x$ | x岁到x+1岁的死亡人数 |
| $q_x$ | x岁到x+1岁的死亡概率 |
| $L_x$ | x岁到x+1岁的人年数 |
| $e_x$ | x岁时的预期寿命 |

## 二、人口转变理论

### 2.1 经典人口转变模型

人口转变理论描述从高出生率高死亡率向低出生率低死亡率的转变过程：

| 阶段 | 出生率 | 死亡率 | 自然增长 | 代表国家 |
|------|--------|--------|---------|---------|
| 第一阶段（前工业） | 高 | 高 | 低/稳定 | 前现代社会 |
| 第二阶段（早期工业化） | 高 | 快速下降 | 快速增长 | 18-19世纪英国 |
| 第三阶段（成熟工业） | 开始下降 | 低 | 缓慢增长 | 20世纪日本 |
| 第四阶段（后工业） | 低 | 低 | 低/零增长 | 现代德国 |
| 第五阶段（超低生育） | 极低 | 低 | 负增长 | 韩国、日本 |

### 2.2 第二次人口转变

van de Kaa和Lesthaeghe提出"第二次人口转变"概念，特征包括：生育率降至更替水平以下、婚姻推迟、同居增加、非婚生育上升、离婚率上升。驱动因素包括文化变迁（个人主义、自主性增强）和经济结构变化（女性教育提升、劳动力参与增加）。

## 三、生育理论

### 3.1 经济学解释

| 理论 | 核心观点 | 关键概念 |
|------|---------|---------|
| 孩子需求理论（Leibenstein） | 孩子提供效用和成本，家庭平衡两者 | 孩子的效用（消费、劳动、保障）vs 成本（直接、机会） |
| 数量-质量权衡（Becker） | 家庭在孩子的数量和每孩"质量"间权衡 | 教育投入增加导致生育数量下降 |
| 生育决策的经济模型 | 收入效应和替代效应影响生育 | 女性工资上升→机会成本上升→生育下降 |

### 3.2 社会学解释

| 理论 | 核心观点 |
|------|---------|
| 扩散理论 | 生育控制观念和技术的传播 |
| 制度理论 | 社会制度（教育、劳动力市场）塑造生育决策 |
| 性别革命理论 | 性别平等的进展与生育率的关系 |
| 风险规避理论 | 现代社会的个体化进程增加不确定性 |

## 四、人口迁移

### 4.1 迁移理论

**推拉理论**（Lee, 1966）

| 因素类型 | 内容 |
|---------|------|
| 推力 | 迁出地的消极因素（失业、贫困、冲突） |
| 拉力 | 迁入地的积极因素（就业、教育、自由） |
| 障碍因素 | 迁移成本、法律限制、距离、家庭约束 |
| 个人因素 | 年龄、教育、技能、风险偏好 |

**重力模型**

$$
M_{ij} = k \frac{P_i^\alpha P_j^\beta}{D_{ij}^\gamma}
$$

其中 $M_{ij}$ 为从 $i$ 地到 $j$ 地的迁移量，$P_i$ 和 $P_j$ 为两地人口，$D_{ij}$ 为两地的距离，$k$、$\alpha$、$\beta$、$\gamma$ 为参数。

### 4.2 迁移类型

| 类型 | 特征 | 例 |
|------|------|-----|
| 内部迁移 | 国内地区间流动 | 农村→城市 |
| 国际迁移 | 跨国流动 | 墨西哥→美国 |
| 循环迁移 | 定期往返 | 季节工 |
| 永久迁移 | 定居迁入地 | 移民 |
| 强迫迁移 | 非自愿流动 | 难民 |

## 五、人口老龄化

### 5.1 老龄化指标

| 指标 | 定义 | 国际警戒线 |
|------|------|-----------|
| 老年人口比例 | 65岁+人口 / 总人口 | >7%（老龄化社会）>14%（老龄社会）>21%（超老龄社会） |
| 老年抚养比 | 65岁+人口 / 15-64岁人口 | 反映劳动力抚养负担 |
| 总抚养比 | (0-14岁 + 65岁+) / 15-64岁人口 | 总抚养负担 |
| 中位年龄 | 年龄分布的中间值 | 越高表示越老化 |

### 5.2 老龄化的挑战

| 领域 | 挑战 | 应对策略 |
|------|------|---------|
| 劳动力 | 劳动年龄人口减少 | 延迟退休、提高生产率 |
| 社会保障 | 养老金不可持续 | 降低替代率、增加缴费 |
| 医疗 | 慢性病和长期护理需求增加 | 发展老年医学和护理 |
| 代际关系 | 代际资源分配冲突 | 代际公平、社会团结 |

## 六、人口政策

### 6.1 政策类型

| 方向 | 目标 | 工具 | 例 |
|------|------|------|-----|
| 鼓励生育 | 提高生育率 | 现金补贴、育儿假、托儿服务 | 法国、瑞典 |
| 限制生育 | 降低生育率 | 计划生育、避孕服务 | 中国（历史） |
| 管理迁移 | 调控人口流入/流出 | 签证制度、积分制 | 加拿大、新加坡 |

### 6.2 中国的生育政策演变

| 时期 | 政策 | 效果 |
|------|------|------|
| 1970s | 晚、稀、少 | 生育率开始下降 |
| 1980-2013 | 独生子女政策 | 生育率降至更替水平以下 |
| 2013-2015 | 单独二孩 | 短期小幅度回升 |
| 2016-2021 | 全面二孩 | 低于预期 |
| 2021- | 三孩政策+配套措施 | — |

## 七、城市化

| 阶段 | 特征 | 例 |
|------|------|-----|
| 初期 | 农村人口向城市缓慢迁移 | 工业革命前 |
| 加速期 | 工业化推动城市快速发展 | 20世纪的发展中国家 |
| 成熟期 | 城市化率超过70%，速度放缓 | 发达国家 |
| 逆城市化 | 人口从城市中心向外围扩散 | 美国1970s |
| 再城市化 | 城市中心复兴 | 伦敦、纽约 |

## 八、人口与可持续发展

### 8.1 人口与资源环境

人口增长与资源环境的关系经历了从马尔萨斯悲观论到技术乐观论的演变：

| 理论 | 核心观点 | 代表 |
|------|---------|------|
| 马尔萨斯陷阱 | 人口指数增长 > 食物线性增长 → 危机 | Malthus |
| 新马尔萨斯主义 | 资源有限性约束人口增长 | Ehrlich《人口炸弹》 |
| 技术乐观主义 | 技术进步突破资源约束 | Boserup, Simon |
| 可持续人口 | 在生态承载力范围内的人口规模 | 联合国可持续发展目标 |

### 8.2 人口红利

"人口红利"指劳动年龄人口占比较高、抚养比较低的时期，为经济增长创造的有利条件。东亚四小龙的经济增长在很大程度上受益于人口红利。

人口红利的三个阶段：

| 阶段 | 特征 | 政策应对 |
|------|------|---------|
| 初期 | 少儿抚养比下降 | 投资教育和健康 |
| 高峰 | 劳动年龄人口比例最大 | 创造就业机会 |
| 后期 | 老年抚养比上升 | 养老金改革、延迟退休 |

### 8.3 全球人口前景

联合国《世界人口展望》预测全球人口将在2080年代达到约104亿的峰值后开始下降。不同区域的人口趋势差异显著：非洲持续增长，欧洲和东亚加速老龄化，南亚和拉美处于人口转变的中后期。

## 九、中国人口问题

### 9.1 人口转变的历史

中国在20世纪下半叶经历了快速的人口转变。1949年后死亡率急剧下降，而出生率直到1970年代才显著下降。独生子女政策加速了生育率的下降，但也带来了性别比例失衡、老龄化加速等意外后果。

### 9.2 当前挑战

| 问题 | 数据 | 影响 |
|------|------|------|
| 低生育率 | TFR约1.2（2023） | 劳动力萎缩、老龄化加速 |
| 性别失衡 | 出生性别比约111（正常103-107） | 婚姻挤压、人口拐卖 |
| 老龄化 | 65岁+人口占比约15% | 社保压力、护理需求增加 |
| 人口流动 | 流动人口约3.8亿 | 城乡分割、留守儿童 |

### 9.3 应对政策

| 政策方向 | 措施 |
|---------|------|
| 鼓励生育 | 生育津贴、延长产假、发展托幼 |
| 延迟退休 | 渐进式延迟法定退休年龄 |
| 人才政策 | 放宽城市落户限制、吸引海外人才 |
| 养老改革 | 多层次养老保险体系、长期护理保险 |
