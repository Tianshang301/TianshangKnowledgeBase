# 知识产权法学

## 一、知识产权概述

### 1.1 知识产权的概念与特征
- **定义**：权利人对其智力成果享有的专有权利
- **特征**：

| 特征 | 含义 |
|------|------|
| 无形性 | 客体为非物质形态的智力成果 |
| 专有性 | 权利人独占使用，他人未经许可不得使用 |
| 地域性 | 仅在授予国或地区有效 |
| 时间性 | 有法定保护期限，期满进入公共领域 |

### 1.2 知识产权的分类

| 分类 | 内容 |
|------|------|
| 工业产权 | 专利权、商标权、地理标志 |
| 著作权及邻接权 | 文学艺术科学作品、表演者权等 |
| 其他 | 商业秘密、集成电路布图设计、植物新品种 |

### 1.3 知识产权法体系
- **国内法**：《著作权法》《专利法》《商标法》《反不正当竞争法》
- **国际公约**：《巴黎公约》《伯尔尼公约》TRIPS协定

## 二、著作权法

### 2.1 受保护的作品
- **定义**：文学、艺术和科学领域内具有独创性并能以一定形式复制的智力成果

| 作品类型 | 示例 | 法条 |
|----------|------|------|
| 文字作品 | 小说、论文、诗歌 | 《著作权法》第3条第1项 |
| 口述作品 | 演讲、授课、法庭辩论 | 第3条第2项 |
| 音乐/戏剧/曲艺/舞蹈作品 | 歌曲、剧本、相声、编舞 | 第3条第3项 |
| 美术/建筑作品 | 绘画、雕塑、建筑物 | 第3条第4项 |
| 摄影作品 | 照片、摄影集 | 第3条第5项 |
| 视听作品 | 电影、电视剧、短视频 | 第3条第6项 |
| 工程设计图/产品设计图/地图/示意图 | 建筑图纸、地图 | 第3条第7项 |
| 计算机软件 | 程序代码、文档 | 第3条第8项 |
| 民间文学艺术作品 | 民歌、传说 | 第3条第9项 |

### 2.2 著作权内容

| 权利类型 | 具体权利 | 保护期限 |
|----------|----------|----------|
| 人身权 | 发表权、署名权、修改权、保护作品完整权 | 署名/修改/完整权：永久 |
| 财产权 | 复制权、发行权、出租权、展览权、表演权、放映权、广播权、信息网络传播权、摄制权、改编权、翻译权、汇编权 | 作者生前+死后50年 |

### 2.3 著作权主体
- **作者**：创作作品的自然人（法人可视为作者）
- **合作作品**：著作权由合作作者共同享有
- **职务作品**：一般归作者，单位有权优先使用；特殊职务作品（工程设计等）单位可享有著作权
- **委托作品**：可约定归属，无约定归受托人

### 2.4 著作权的限制
- **合理使用**（12种情形）：
  - 个人学习/研究
  - 介绍/评论适当引用
  - 新闻报道引用
  - 媒体转载政经宗教时事文章
  - 课堂教学/科研翻译
  - 国家机关公务使用
  - 免费表演
  - 图书馆/档案馆复制
  - 公共场所艺术品临摹
  - 改成盲文出版
  - 无障碍格式版
  - 其他
- **法定许可**：
  - 教科书编写使用
  - 报刊转载（作者声明不得转载除外）
  - 录音制品制作
  - 广播电台电视台播放

### 2.5 邻接权
| 权利主体 | 权利内容 |
|----------|----------|
| 表演者 | 表明身份、保护形象、许可直播/录制 |
| 录音录像制作者 | 复制、发行、出租、信息网络传播 |
| 广播电台电视台 | 转播、录制、复制 |
| 出版者 | 版式设计专有权 |

### 2.6 著作权保护
- **侵权责任**：停止侵害、消除影响、赔礼道歉、赔偿损失
- **赔偿标准**：实际损失→侵权获利→法定赔偿（500-500万元）
- **技术措施与权利管理信息保护**

## 三、专利法

### 3.1 专利类型

| 类型 | 保护对象 | 保护期限 | 审批方式 |
|------|----------|----------|----------|
| 发明 | 产品/方法的技术方案 | 20年 | 实质审查 |
| 实用新型 | 产品的形状/构造方案 | 10年 | 初步审查+评价报告 |
| 外观设计 | 产品的形状/图案/色彩设计 | 15年 | 初步审查 |

### 3.2 专利授权条件
- **新颖性**：不属于现有技术，无在先申请
- **创造性**：
  - 发明：突出的实质性特点和显著进步
  - 实用新型：实质性特点和进步
  - 外观设计：明显区别
- **实用性**：能够制造或使用，产生积极效果

### 3.3 不授予专利的情形
- 科学发现
- 智力活动的规则和方法
- 疾病的诊断和治疗方法
- 动物和植物品种（生产方法可授权）
- 原子核变换方法获得的物质
- 违反法律、社会公德或妨害公共利益的发明创造

### 3.4 专利申请与审查
- **申请原则**：先申请原则（以申请日为准）
- **文件**：请求书、说明书、权利要求书、摘要
- **审查程序**：受理→初步审查→公布→实质审查（发明）→授权/驳回
- **优先权**：外国优先权12个月/6个月，本国优先权

### 3.5 专利权的内容
- 禁止他人未经许可实施专利
- 许可权（独占许可、排他许可、普通许可）
- 转让权（书面合同+登记）
- 标记权

### 3.6 专利侵权
- **判定标准**：全面覆盖原则（落入权利要求保护范围）
- **等同原则**：等同特征落入范围
- **抗辩**：现有技术抗辩、非生产经营目的、权利用尽、先用权、临时过境

## 四、商标法

### 4.1 商标类型

| 分类标准 | 类型 |
|----------|------|
| 构成要素 | 文字商标、图形商标、字母商标、数字商标、三维标志商标、颜色组合商标、声音商标 |
| 使用对象 | 商品商标、服务商标 |
| 主体特征 | 集体商标、证明商标 |
| 知名度 | 普通商标、驰名商标 |

### 4.2 商标注册条件
- **绝对理由**（不得注册）：
  - 缺乏显著特征
  - 仅有通用名称/图形/型号
  - 直接表示商品质量/原料/功能
  - 三维标志仅为商品自身性质
- **相对理由**（不得注册）：
  - 与在先商标相同或近似于相同或类似商品
  - 侵犯他人在先权利
  - 以不正当手段抢注他人已经使用并有一定影响的商标

### 4.3 商标注册程序
- 申请→形式审查→实质审查→公告（3个月异议期）→授权
- 保护期限：10年（可无限续展，每次10年）

### 4.4 驰名商标特别保护
- **认定**：个案认定、被动保护
- **保护范围**：跨类保护（在不相同/不类似商品上复制/模仿/翻译）
- **证据**：相关公众知晓程度、使用持续时间、宣传情况

### 4.5 商标侵权
- **侵权行为**：同种+同标、同种+近似、类似+同标、类似+近似
- **混淆可能性**：相关公众的一般注意
- **侵权责任**：行政处罚、民事赔偿、刑事责任

## 五、商业秘密

| 要件 | 含义 |
|------|------|
| 秘密性 | 不为公众所知悉 |
| 价值性 | 具有商业价值 |
| 保密性 | 权利人采取保密措施 |

- **侵权行为**：以不正当手段获取、披露、使用
- **保护方式**：保密协议、竞业限制、侵权诉讼

## 六、国际知识产权保护

| 公约 | 主要内容 |
|------|----------|
| 《巴黎公约》 | 工业产权保护，国民待遇+优先权+独立保护 |
| 《伯尔尼公约》 | 著作权保护，自动保护+国民待遇+最低保护期限 |
| TRIPS协定 | WTO框架，最惠国待遇+执法要求+争端解决 |
| 《专利合作条约》(PCT) | 专利国际申请统一程序 |
| 《商标国际注册马德里协定》 | 商标国际注册体系 |
| 《世界知识产权组织(WIPO)》 | 知识产权国际协调 |

## 七、知识产权侵权的法律救济

### 7.1 民事救济
| 救济方式 | 适用情形 | 说明 |
|----------|----------|------|
| 停止侵害 | 侵权行为仍在继续 | 永久禁令/临时禁令 |
| 消除影响 | 侵权行为造成不良影响 | 公开声明 |
| 赔礼道歉 | 侵害人身权 | 公开道歉 |
| 赔偿损失 | 造成经济损失 | 实际损失→侵权获利→法定赔偿 |
| 销毁侵权物品 | 防止进一步侵权 | 销毁侵权复制品及制造工具 |

### 7.2 行政保护
- **著作权**：版权行政管理部门可责令停止侵权、没收违法所得、罚款
- **专利**：专利管理部门可责令停止侵权、调解赔偿
- **商标**：市场监督管理部门可没收销毁侵权商品、罚款
- **海关保护**：知识产权海关边境保护（主动/被动扣留）

### 7.3 刑事保护
- **侵犯著作权罪**：以营利为目的，未经许可复制发行、出版、制作出售假冒他人署名美术作品
- **假冒专利罪**：假冒他人专利，情节严重
- **假冒注册商标罪**：在同种商品上使用相同商标
- **销售假冒注册商标商品罪**：明知是假冒注册商标商品而销售
- **非法制造/销售非法制造注册商标标识罪**
- **侵犯商业秘密罪**：以盗窃/利诱/胁迫等不正当手段获取商业秘密

### 7.4 知识产权诉讼特殊规则
- **管辖**：知识产权法院（北京、上海、广州、海南）及各地知识产权法庭
- **举证**：举证责任倒置（方法发明专利侵权）、证据保全
- **诉前禁令**：权利人有证据证明侵权正在发生或即将发生，可在起诉前申请禁止令
- **惩罚性赔偿**：故意侵权情节严重的，可适用1-5倍惩罚性赔偿

## 八、知识产权与其他法律的关系

### 8.1 反不正当竞争法保护
- 对未注册的知名商品特有名称/包装/装潢的保护
- 对未注册驰名商标的保护
- 禁止仿冒行为
- 禁止虚假宣传

### 8.2 知识产权与民法典
- 知识产权为民事权利的一种
- 民法典总则第123条明确知识产权为民事客体
- 知识产权许可合同适用合同法规则
- 知识产权侵权适用侵权责任规则

## 九、知识产权战略与政策

### 9.1 国家知识产权战略
- 2008年《国家知识产权战略纲要》
- 2021年《知识产权强国建设纲要（2021-2035年）》
- 从知识产权大国向知识产权强国转变

### 9.2 知识产权运营
- **知识产权转让**：所有权转移
- **知识产权许可**：使用权授权（独占/排他/普通/交叉许可）
- **知识产权质押融资**：以知识产权出质获得贷款
- **知识产权证券化**：以未来许可收益为基础资产发行证券

## 十、典型知识产权案例

### 10.1 著作权案例
- **琼瑶诉于正案**：剧本《梅花烙》著作权侵权，认定实质性相似+接触
- **谷歌图书案**：谷歌扫描图书构成合理使用（美国法）

### 10.2 专利案例
- **华为诉三星案**：标准必要专利侵权+FRAND许可条件
- **西电捷通诉索尼案**：方法专利侵权，举证责任分配

### 10.3 商标案例
- **王老吉与加多宝案**：商标许可合同到期后包装装潢归属
- **新百伦商标案**：外国企业中文商标保护
