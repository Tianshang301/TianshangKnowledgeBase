# 国际法学

## 一、国际法基础理论

### 1.1 国际法的定义与特征
- **定义**：主要调整国家之间关系的法律规范的总和
- **特征**：
  - 主体主要是国家（而非个人）
  - 由各国协议制定（无中央立法机关）
  - 依靠国家自愿遵守（无强制执行机关）
  - 制裁方式：报复、反报、集体制裁

### 1.2 国际法主体

| 主体 | 地位 | 说明 |
|------|------|------|
| 国家 | 完全主体 | 具备主权，享有完整权利义务 |
| 国际组织 | 派生主体 | 在职能范围内享有权利义务（联合国、WTO） |
| 个人 | 有限主体 | 国际人权法、国际刑法中的部分权利/义务 |
| 民族解放运动 | 过渡主体 | 争取独立的民族（巴勒斯坦等） |

### 1.3 国家要素
- **定居的居民**：固定的人口
- **确定的领土**：有边界的土地
- **政府**：对内统治、对外交往
- **主权**：独立处理内外事务的最高权力

### 1.4 国际法的渊源

| 渊源 | 定义 | 依据 |
|------|------|------|
| 国际条约 | 国家间书面协议 | 《国际法院规约》第38条第1项 |
| 国际习惯 | 国家实践+法律确信（opinio juris） | 第38条第2项 |
| 一般法律原则 | 各国法律体系共有的原则 | 第38条第3项 |
| 司法判例 | 国际法院等判决作为辅助资料 | 第38条第4项 |
| 公法学家学说 | 各国权威学者的著述 | 第38条第5项 |

### 1.5 国际法与国内法的关系

| 理论 | 主张 |
|------|------|
| 一元论（Monism） | 国际法与国内法属于同一法律体系，国际法优先 |
| 二元论（Dualism） | 国际法与国内法分属不同体系，需转化适用 |

- **中国实践**：条约优先适用（民商事领域），声明保留条款除外；习惯国际法无明确规定

## 二、国家

### 2.1 国家承认
- **宣告说**（Declaratory Theory）：承认仅宣告既成事实
- **构成说**（Constitutive Theory）：承认构成国家资格
- **方式**：明示承认（声明/照会）、默示承认（建交/缔约）

### 2.2 国家继承
- **条约继承**：人身条约不继承，边界条约继承
- **财产继承**：不动产随领土转移，动产按公平比例
- **债务继承**：国家债务继承，恶债不继承

### 2.3 国家管辖权

| 管辖权类型 | 依据 |
|------------|------|
| 属地管辖 | 领土内人/物/事 |
| 属人管辖 | 本国国民（不论何处） |
| 保护性管辖 | 危害国家安全的行为 |
| 普遍管辖 | 国际罪行（海盗、灭绝种族、奴隶制） |

### 2.4 国家豁免
- **绝对豁免论**：国家一切行为均豁免
- **限制豁免论**：主权行为豁免，商业行为不豁免
- **放弃豁免**：明示放弃或提起反诉

### 2.5 国家责任
- **归因**：国家机关行为、经授权行使权力要素的行为、逾越权限的行为
- **解除不当性**：同意、自卫、反措施、不可抗力、危难、紧急状态
- **法律后果**：停止不法行为 + 保证不重复 + 赔偿（恢复原状/补偿/满足）

## 三、条约法

### 3.1 条约的定义与分类
- **定义**：国家间所缔结的以国际法为准的国际书面协定
- **名称**：条约、公约、协定、议定书、宪章、盟约、规约

### 3.2 条约的缔结

| 阶段 | 内容 |
|------|------|
| 谈判 | 代表全权证书 |
| 认证 | 草签、签署 |
| 批准 | 国内程序确认 |
| 加入 | 未签署国成为当事国 |
| 保留 | 排除或更改条约某些条款的效力 |

### 3.3 条约的生效与适用
- **生效条件**：规定日期、全部签署、一定数量批准
- **适用原则**：条约必须遵守（pacta sunt servanda）
- **条约解释**：《维也纳条约法公约》第31-33条：
  - 第31条：善意解释，上下文+目的宗旨
  - 第32条：补充解释资料（准备工作和缔约情况）
  - 第33条：不同文本的作准解释

### 3.4 条约的无效、终止与暂停

| 情形 | 类型 |
|------|------|
| 无效 | 违反国内法基本权限、错误、欺诈、贿赂、强迫、违反强行法（jus cogens） |
| 终止 | 期满、履行完毕、解除条件成就、全体同意、违约、情势变迁 |
| 暂停 | 当事人同意、违约、情势变迁 |

#### 强行法（jus cogens）
- 国际社会全体接受并公认不许损抑的基本规范
- **例子**：禁止侵略、禁止种族灭绝、禁止奴隶制、禁止种族歧视、禁止酷刑

## 四、国际争端解决

### 4.1 和平解决争端原则
- 《联合国宪章》第2条第3款：以和平方法解决争端
- **不干涉内政原则**：不得干涉本质上属于国家国内管辖的事项

### 4.2 外交手段

| 方式 | 特点 |
|------|------|
| 谈判/协商 | 直接对话，最常用 |
| 斡旋 | 第三方促成谈判但不直接参与 |
| 调停 | 第三方提出建议并参与协商 |
| 调查 | 调查委员会查明事实 |
| 和解 | 和解委员会提出解决方案 |

### 4.3 法律手段

| 机构 | 管辖权 | 适用法律 |
|------|--------|----------|
| 国际法院（ICJ） | 自愿/协定/强制管辖 | 条约+习惯+一般原则+判例 |
| 常设仲裁法院（PCA） | 双方同意提交仲裁 | 当事人约定 |
| 国际海洋法法庭（ITLOS） | UNCLOS争端 | 海洋法 |
| WTO争端解决机构 | 强制管辖（贸易争端） | WTO协定 |
| 国际刑事法院（ICC） | 灭绝种族/危害人类/战争/侵略罪 | 《罗马规约》 |

## 五、海洋法（UNCLOS）

| 海域 | 范围 | 沿海国权利 | 他国权利 |
|------|------|------------|----------|
| 内水 | 基线向陆一侧 | 完全主权 | 无 |
| 领海 | 基线至12海里 | 主权（除无害通过） | 无害通过权 |
| 毗连区 | 12-24海里 | 海关/财政/移民/卫生管制 | 航行自由 |
| 专属经济区（EEZ） | 12-200海里 | 自然资源勘探开发 | 航行飞越自由+铺设电缆 |
| 大陆架 | 200海里或自然延伸 | 自然资源主权 | 铺设电缆+管道 |
| 公海 | 专属经济区以外 | 无主权 | 公海六大自由 |
| 国际海底区域 | 国家管辖范围以外 | — | 人类共同继承财产 |

## 六、国际人道法（武装冲突法）

### 6.1 基本原则
- **区分原则**：区分平民与战斗员、民用与军事目标
- **比例原则**：军事利益与附带损害成比例
- **人道原则**：禁止引起过分伤害的手段
- **马顿斯条款**：无明文规定时人道原则保护

### 6.2 日内瓦四公约（1949）

| 公约 | 保护对象 |
|------|----------|
| 第一公约 | 战地武装部队伤者病者 |
| 第二公约 | 海上武装部队伤者病者遇船难者 |
| 第三公约 | 战俘 |
| 第四公约 | 战时平民 |

### 6.3 禁止的作战手段与方法
- 禁止使用化学武器、生物武器
- 禁止滥杀滥伤武器（地雷、集束炸弹）
- 禁止背信弃义行为
- 禁止攻击文化财产

## 七、国际人权法

### 7.1 国际人权宪章
| 文件 | 通过年份 | 主要内容 |
|------|----------|----------|
| 《世界人权宣言》 | 1948 | 公民政治权利+经济社会文化权利（联合国大会决议） |
| 《公民权利和政治权利国际公约》 | 1966 | 生命权、禁止酷刑、公正审判、言论宗教自由等 |
| 《经济、社会及文化权利国际公约》 | 1966 | 工作权、社会保障、教育权、健康权等 |

### 7.2 区域性人权保护
- **欧洲人权公约**（ECHR）——欧洲人权法院
- **美洲人权公约**——美洲人权法院
- **非洲人权与民族权宪章**——非洲人权与民族权委员会

### 7.3 国际人道法与人权法的关系
- 人道法适用于武装冲突，人权法平时/战时均适用
- 人道法保护不区分国籍，人权法调整国家与个人关系
- 两者互补：人道法细化战时人权保护

## 八、国际环境法

### 8.1 基本原则
- 国家环境主权与不损害他国环境原则
- 可持续发展原则
- 预防原则
- 污染者负担原则

## 九、国际刑法

### 9.1 国际刑事法院（ICC）
- 2002年根据《罗马规约》成立，常设于海牙
- **管辖罪行**：
  - 灭绝种族罪（genocide）
  - 危害人类罪（crimes against humanity）
  - 战争罪（war crimes）
  - 侵略罪（crime of aggression）
- **补充性原则**：仅在国家不愿或不能真正起诉时行使管辖权

### 9.2 国际刑事审判的历史发展

| 机构 | 时期 | 管辖 |
|------|------|------|
| 纽伦堡国际军事法庭 | 1945-1946 | 纳粹德国战犯 |
| 东京国际军事法庭 | 1946-1948 | 日本战犯 |
| 前南国际刑事法庭（ICTY） | 1993-2017 | 前南斯拉夫战争罪行 |
| 卢旺达国际刑事法庭（ICTR） | 1994-2015 | 卢旺达种族灭绝 |
| 国际刑事法院（ICC） | 2002- | 常设 |

## 十、外交与领事关系法

### 10.1 外交关系
- **《维也纳外交关系公约》（1961）**
- 外交特权与豁免：
  - 使馆馆舍不得侵犯
  - 外交代表人身不得侵犯
  - 刑事管辖绝对豁免
  - 民事行政管辖有限豁免
  - 免纳捐税、关税

### 10.2 领事关系
- **《维也纳领事关系公约》（1963）**
- 领事特权与豁免弱于外交特权
- 领事职务：保护本国侨民、签证护照、公证认证

## 十一、国际法上的个人

### 11.1 国籍
- 国籍的取得：出生（血统主义/出生地主义/混合主义）+ 归化
- 国籍冲突：双重/多重国籍（积极冲突）、无国籍（消极冲突）
- 中国不承认双重国籍

### 11.2 外国人的法律地位
- 国民待遇（一般民商事领域）
- 最惠国待遇（经贸领域）
- 差别待遇（基于互惠）
- 外交保护：国籍连续原则+用尽当地救济

### 11.3 引渡与庇护
- **引渡**：一国应他国请求移交犯罪嫌疑人/已判刑人
- **原则**：双重犯罪原则、特定性原则、政治犯不引渡
- **庇护**：国家给予因政治原因受追捕的外国人以保护
- **不推回原则**：不得将难民推回其生命或自由受威胁的国家
