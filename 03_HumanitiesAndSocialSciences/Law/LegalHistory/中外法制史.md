# 中外法制史

## 一、中国法制史

### 1.1 夏商西周时期

#### 夏商
- **夏**：禹刑（中国最早的法律，源于夏禹）
  - 神权法思想：天命、天罚
  - 罪名：昏（恶而掠美）、墨（贪以败官）、贼（杀人不忌）
- **商**：汤刑
  - 继续神权法：€天命玄鸟，王权神授
  - 以酷刑著称：炮烙、醢、脯

#### 西周
- **以德配天，明德慎罚**：
  - 以德配天：统治者有德才能承受天命
  - 明德慎罚：提倡德教，慎用刑罚
- **礼刑并用**：
  - 礼（正面规范）：亲亲、尊尊
  - 刑（反面制裁）：九刑→吕刑
- **刑罚原则**：三赦（幼弱/老耄/蠢愚）、三宥（过失/不知/遗忘）、罪刑不相及

### 1.2 秦汉时期

#### 秦（法家主导）
- **指导思想**：商鞅变法——以法治国、重刑轻罪
  - 商鞅改法为律
  - 连坐制度、告奸制度
- **秦律内容**（睡虎地秦墓竹简）：
  - 重视农业（田律、厩苑律）
  - 严密的官制（置吏律、效律）
  - 重刑主义：什伍连坐、族刑

#### 汉（儒法合流）
- **汉承秦制**：萧何制定《九章律》（在秦六律基础上加户/兴/厩三章）
- **儒家化过程**：
  - 德主刑辅：董仲舒"大德小刑"
  - 春秋决狱：以儒家经义审判（原心定罪）
  - 亲亲得相首匿：亲属间容隐犯罪
  - 恤刑原则：老人幼童减免刑罚

### 1.3 唐宋时期

#### 唐（中华法系巅峰）
- **《唐律疏议》**（永徽律疏）：中华法系最辉煌的代表

| 项目 | 内容 |
|------|------|
| 结构 | 12篇502条 |
| 篇目 | 名例、卫禁、职制、户婚、厩库、擅兴、贼盗、斗讼、诈伪、杂律、捕亡、断狱 |
| 五刑 | 笞（10-50）、杖（60-100）、徒（1-3年）、流（2000-3000里）、死（绞/斩） |
| 十恶 | 谋反、谋大逆、谋叛、恶逆、不道、大不敬、不孝、不睦、不义、内乱 |
| 八议 | 议亲、议故、议贤、议能、议功、议贵、议勤、议宾 |

- **特点**：礼法合一、科条简要、刑罚适中
- **影响**：东亚各国（日本/朝鲜/越南）继受，形成中华法系

#### 宋
- 《宋刑统》：第一部刊版印刷的法典
- 编敕：皇帝诏敕上升为法律
- 刑罚变化：折杖法（笞杖徒流折为杖刑）

### 1.4 明清时期

#### 明
- **《大明律》**：7篇30卷460条（名例+吏户礼兵刑工六部结构）
- **明大诰**：朱元璋亲自编纂，重典治吏
- **廷杖制度**：皇帝在朝廷上杖打大臣

#### 清
- **《大清律例》**：7篇436条（律+例结构）
- **秋审制度**：死刑复核——情实、缓决、可矜、可疑、留养承祀
- **文字狱**：以言论定罪的文化高压

### 1.5 近代法制转型

#### 清末修律（1902-1911）
- **沈家本、伍廷芳**主持修律
- **主要成果**：
  - 废除凌迟、枭首、戮尸等酷刑
  - 《大清现行刑律》（1910）
  - 《大清新刑律》（1911首部近代刑法典）
  - 《大清民律草案》
  - 《大清商律草案》
  - 《法院编制法》
- **意义**：打破诸法合体，引进西方近代法律体系

#### 民国时期
- **南京临时政府（1912）**：临时约法（人民主权）
- **北洋政府**：援用清末法律
- **南京国民政府（1927-1949）**：
  - 六法全书：宪法、民法、刑法、民事诉讼法、刑事诉讼法、行政法
  - 以大陆法系为蓝本

#### 人民共和国时期
- **1949-1978**：废除 KMT 六法全书、苏联法律移植、文革法制破坏
- **1978-2010**：
  - 1979年《刑法》《刑事诉讼法》
  - 1982年宪法
  - 1986年《民法通则》
  - 1997年修订《刑法》（罪刑法定）
  - 1999年"依法治国"入宪
  - 2004年"人权保障"入宪
- **2010至今**：中国特色社会主义法律体系形成
  - 2020年《中华人民共和国民法典》通过（1260条）

## 二、外国法制史

### 2.1 罗马法

| 时期 | 事件 | 内容 |
|------|------|------|
| 王政/共和早期 | 十二表法（450BC） | 罗马第一部成文法，平民斗争的成果 |
| 古典时期 | 万民法（jus gentium） | 适用于外邦人的法律体系 |
| 帝国时期 | 法学繁荣 | 五大法学家（盖尤斯、保罗、乌尔比安、帕比尼安、莫德斯丁） |
| 东罗马帝国 | 查士丁尼法典编纂（528-534） | 《查士丁尼法典》《学说汇纂》《法学阶梯》《新律》→《国法大全》 |

- **罗马法体系**：人法（身份+家庭）、物法（物权+继承+债）、诉讼法
- **影响**：欧洲大陆继受→大陆法系的渊源

### 2.2 大陆法系（Civil Law）

| 法典 | 年份 | 特点 |
|------|------|------|
| 《法国民法典》（拿破仑法典） | 1804 | 三编制（人/财产及所有权/取得财产方法）、个人主义、自由平等 |
| 《德国民法典》（BGB） | 1900 | 五编制（总则/债/物权/亲属/继承）、潘德克顿体系、抽象严谨 |

- **特征**：成文法为主、法典化、演绎推理、职权主义诉讼模式

### 2.3 英美法系（Common Law）

| 时期 | 发展 | 内容 |
|------|------|------|
| 盎格鲁-撒克逊时期 | 日耳曼习惯法 | 地方习惯、神明裁判 |
| 诺曼征服后（1066） | 普通法的形成 | 国王巡回法院→统一判例→普通法 |
| 14世纪 | 衡平法的兴起 | 大法官法庭→补充普通法不足 |
| 近代 | 法律改革 | 1873-1875司法改革融合普通法与衡平法 |

- **特征**：
  - 遵循先例（stare decisis）——判例法核心
  - 对抗制（adversarial system）——当事人主导
  - 法律与衡平并存
- **分布**：英国、美国、加拿大、澳大利亚、印度等
- **英美区别**：
  - 美国有联邦制（联邦+州双重法律体系）
  - 美国有成文宪法+司法审查

### 2.4 两大法系比较

| 比较项 | 大陆法系 | 英美法系 |
|--------|----------|----------|
| 法律渊源 | 成文法为主 | 判例法为主 |
| 法官角色 | 适用法律（纠问制） | 中立裁判（对抗制） |
| 推理方式 | 演绎推理（从一般到个别） | 归纳推理（从个别到一般） |
| 法律分类 | 公法/私法 | 普通法/衡平法 |
| 法典化 | 高度重视法典 | 不强调法典化 |

## 三、世界主要法律传统

### 3.1 中华法系
- **形成**：以唐律为核心，中国古代法律的东亚影响圈
- **分布**：中国、日本、朝鲜、越南、琉球
- **特征**：
  - 礼法合一：道德与法律高度融合
  - 以刑为主：刑事规范占据主导
  - 诸法合体：实体程序不分，刑民不分
  - 家族本位：宗法伦理贯穿法典
  - 行政兼理司法：地方行政长官兼理审判
- **衰落**：清末修律后逐渐解体，被大陆法系取代

### 3.2 伊斯兰法系
- **渊源**：古兰经（Quran）、圣训（Sunna）、类比（Qiyas）、公议（Ijma）
- **沙里亚法**（Sharia）：伊斯兰教法，涵盖宗教、道德、法律
- **适用范围**：婚姻家庭继承仍适用伊斯兰法，刑民领域多西化
- **分布**：沙特阿拉伯、伊朗、巴基斯坦、印度尼西亚等

### 3.3 印度法系
- **渊源**：《摩奴法典》——古代印度宗教法律汇编
- 种姓制度（婆罗门/刹帝利/吠舍/首陀罗）
- **近代转型**：英国殖民后引入英美法系
- **现状**：印度属英美法系，但属人法（婚姻继承）保留宗教色彩

## 五、中国法律史专题

### 5.1 古代司法制度
- **西周**：司法官（司寇），诉讼（狱/讼）
- **秦汉**：廷尉（中央司法长官），春秋决狱（援引经义）
- **唐代**：大理寺（审判）+刑部（复核）+御史台（监察），三司推事
- **宋代**：鞫谳分司（审/判分离）
- **明清**：刑部（审判）+大理寺（复核）+都察院（监察），三法司会审
- **会审制度**：九卿会审、秋审、朝审、热审

### 5.2 古代法律职业
- **讼师**：民间法律工作者，不受官方认可
- **刑名幕友**：地方官聘用的法律顾问，实际掌握审判
- **胥吏**：衙门中办理具体法律事务的吏员

### 5.3 法律思想的演变

| 时期 | 主导思想 | 代表人物 |
|------|----------|----------|
| 西周 | 以德配天 | 周公 |
| 春秋战国 | 百家争鸣（儒/法/道/墨） | 孔子、商鞅、韩非 |
| 秦 | 法家（以法治国） | 李斯 |
| 汉至清 | 儒家化（德主刑辅） | 董仲舒、长孙无忌 |
| 清末民初 | 西学东渐 | 沈家本、伍廷芳 |
| 新中国 | 社会主义法治 | — |

## 六、外国法律史专题

### 6.1 西方法律传统的形成
- **11-13世纪教皇革命**：教会法与世俗法分离
- **波伦那大学**：罗马法复兴（伊尔内留斯注释法学派）
- **教会法**（Canon Law）：婚姻、继承、诉讼制度影响深远
- **商法**（Lex Mercatoria）：中世纪商人习惯法，现代商法渊源

### 6.2 普通法的形成与发展
- 亨利二世（1154-1189）：司法改革，建立巡回审判制度
- 令状制度（writ system）：无令状即无救济
- **爱德华一世**（1272-1307）：英国法律黄金时代
- **柯克（Coke）**：普通法至上，对抗王权
- **布莱克斯通（Blackstone）**：《英国法释义》——普通法体系化

### 6.3 美国的法律发展
- 殖民地时期：英国普通法的适用
- 独立后：继受英国法但发展本国特色
- **马歇尔法院**（1801-1835）：司法审查、联邦至上
- 19世纪：法律形式主义（Langdell 案例教学法）
- 20世纪：法律现实主义（Holmes"法律的生命是经验"）

### 6.4 德国法的发展
- 1495年：帝国最高法院建立，罗马法正式继受
- 自然法法典化运动：普鲁士一般邦法（ALR, 1794）、奥地利一般民法典（ABGB, 1811）
- **历史法学派**（萨维尼）：法律是民族精神的体现
- **潘德克顿法学**：温德沙伊德等人对罗马法的体系化整理
- BGB 的制定：1871年统一后开始编纂，1900年生效，代表德国法学的最高成就
