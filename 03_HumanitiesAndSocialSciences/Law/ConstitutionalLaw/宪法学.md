# 宪法学

## 一、宪法基本原理

### 1.1 宪法的概念与特征
- **宪法**：国家的根本法，具有最高法律效力
- **特征**：内容根本性（规定国家根本制度）、效力最高性（法律位阶最高）、制定修改程序严格性

### 1.2 宪法的基本原则

| 原则 | 含义 | 制度体现 |
|------|------|----------|
| 人民主权 | 国家权力属于人民 | 人民代表大会制度 |
| 基本人权 | 尊重和保障人权 | 公民基本权利体系 |
| 法治 | 依法治国，法律至上 | 宪法第五条 |
| 权力制约 | 权力相互监督制约 | 人大监督、司法监督 |

### 1.3 宪法的分类

| 分类标准 | 类型 | 示例 |
|----------|------|------|
| 是否有统一法典 | 成文宪法 / 不成文宪法 | 中国（成文）/ 英国（不成文） |
| 修改程序 | 刚性宪法 / 柔性宪法 | 中国（刚性）/ 英国（柔性） |
| 制定主体 | 钦定宪法 / 民定宪法 / 协定宪法 | 日本1889（钦定）/ 美国（民定） |

### 1.4 宪法渊源
- **宪法典**：根本法主源
- **宪法性法律**：组织法、选举法、立法法等
- **宪法惯例**：无明文规定但实际运行的政治习惯
- **宪法解释**：全国人大常委会的解释
- **国际条约**：人权公约等

## 二、中国宪法发展史

### 2.1 1949年《共同纲领》
- 临时宪法性质，确立新民主主义国家制度
- 规定政权机关、军事制度、经济政策、文教政策

### 2.2 1954年宪法
- 新中国第一部社会主义类型宪法
- 确立人民代表大会制度、国家机构体系
- 规定公民基本权利与义务

### 2.3 1975年宪法与1978年宪法
- 1975年：文革产物，大量删减公民权利
- 1978年：拨乱反正过渡，仍保留部分文革痕迹

### 2.4 1982年宪法（现行宪法）
- 继承1954年宪法传统，总结历史经验
- 序言+四章（总纲、公民权利义务、国家机构、国旗国歌国徽首都）
- 共143条

### 2.5 宪法修正案

| 修正案 | 年份 | 主要内容 |
|--------|------|----------|
| 第1-2条修正 | 1988 | 允许私营经济、土地使用权转让 |
| 第3-11条修正 | 1993 | 社会主义初级阶段、改革开放、市场经济 |
| 第12-17条修正 | 1999 | 依法治国、法治国家、社会主义经济制度 |
| 第18-31条修正 | 2004 | 人权保障、私有财产保护、征地补偿、三个代表 |
| 第32-52条修正 | 2018 | 科学发展观、习近平新时代中国特色社会主义思想、监察委、宪法宣誓 |

## 三、中国宪法的基本制度

### 3.1 国体（国家性质）
- **人民民主专政**：对人民民主，对敌人专政
- 工人阶级领导的、以工农联盟为基础的社会主义国家
- **爱国统一战线**：全体社会主义劳动者+建设者+拥护社会主义爱国者+拥护祖国统一爱国者

### 3.2 政体（政权组织形式）
- **人民代表大会制度**：根本政治制度
- 民主集中制原则：民主基础上的集中，集中指导下的民主

### 3.3 国家结构形式
- **单一制国家**：中央统一领导地方
- 民族区域自治制度：自治区、自治州、自治县
- 特别行政区制度：香港、澳门（一国两制，高度自治）

### 3.4 基本经济制度
- 公有制为主体、多种所有制经济共同发展
- 按劳分配为主体、多种分配方式并存
- 社会主义市场经济体制

## 四、公民基本权利与义务

### 4.1 基本权利体系

| 类别 | 具体权利 | 宪法条文 |
|------|----------|----------|
| 平等权 | 法律面前一律平等 | 第33条 |
| 政治权利 | 选举权、被选举权 | 第34条 |
| 表达自由 | 言论、出版、集会、结社、游行、示威 | 第35条 |
| 宗教信仰 | 信仰宗教自由 | 第36条 |
| 人身自由 | 人身自由、人格尊严、住宅不受侵犯 | 第37-39条 |
| 通信自由 | 通信秘密受法律保护 | 第40条 |
| 监督权 | 批评建议、申诉控告检举 | 第41条 |
| 社会经济权 | 劳动权、休息权、社会保障 | 第42-45条 |
| 文化教育权 | 受教育权、科研文艺创作自由 | 第46-47条 |
| 特定主体 | 妇女、儿童、老人、华侨、残障人士 | 第48-50条 |

### 4.2 基本义务
- 维护国家统一和民族团结
- 遵守宪法和法律，保守国家秘密
- 维护祖国安全、荣誉和利益
- 依法服兵役和参加民兵组织
- 依法纳税
- 受教育和劳动的义务

## 五、国家机构

### 5.1 全国人民代表大会及其常委会
- **地位**：最高国家权力机关
- **职权**：修改宪法、制定基本法律、选举/决定/罢免国家领导人、审查批准国民经济计划、批准省/自治区/直辖市建制

### 5.2 国家主席
- **地位**：国家元首
- **产生**：全国人大选举
- **职权**：公布法律、任免国务院人员、发布特赦令/戒严令、进行国事活动

### 5.3 国务院
- **地位**：最高国家行政机关（中央人民政府）
- **组成**：总理、副总理、国务委员、各部部长、各委员会主任、审计长、秘书长
- **职权**：行政法规制定权、行政管理权、经济/社会/文化管理权

### 5.4 中央军事委员会
- **地位**：最高军事领导机关
- **主席**：由全国人大选举产生

### 5.5 国家监察委员会
- **地位**：最高监察机关（2018年宪法修正案增设）
- **职权**：对所有行使公权力的公职人员进行监察

### 5.6 人民法院
- **体系**：最高人民法院→高级人民法院→中级人民法院→基层人民法院
- **专门法院**：军事法院、海事法院、知识产权法院
- **原则**：依法独立行使审判权、公开审判、辩护制度、人民陪审员制度

### 5.7 人民检察院
- **地位**：国家法律监督机关
- **体系**：最高人民检察院→地方各级人民检察院
- **职权**：侦查监督、审查逮捕、审查起诉、提起公诉、审判监督、执行监督

## 六、宪法监督制度

### 6.1 合宪性审查
- **主体**：全国人大及其常委会
- **对象**：行政法规、地方性法规、司法解释等规范性文件
- **方式**：事前审查（批准制）+ 事后审查（备案审查制）

### 6.2 备案审查制度
- 行政法规、地方性法规、司法解释报全国人大常委会备案
- 审查标准：合宪性、合法性、适当性

### 6.3 宪法解释
- **主体**：全国人大常委会
- **方式**：抽象解释（对宪法条文的统一解释）
- **效力**：与宪法具有同等效力

## 七、外国宪法比较

### 7.1 美国宪法

| 项目 | 内容 |
|------|------|
| 制定 | 1787年费城制宪会议 |
| 特点 | 世界上第一部成文宪法 |
| 联邦制 | 联邦与州分权 |
| 三权分立 | 立法（国会）、行政（总统）、司法（法院） |
| 权利法案 | 前10条修正案，保障言论、宗教、持枪等权利 |
| 司法审查 | Marbury v. Madison (1803) 确立联邦最高法院违宪审查权 |

### 7.2 德国基本法（Grundgesetz）
- 1949年制定，原为临时性质，1990年两德统一后成为正式宪法
- **特色制度**：
  - 宪法法院（Bundesverfassungsgericht）——专门宪法审查机构
  - 社会国原则（Sozialstaatsprinzip）
  - 防御性民主（wehrhafte Demokratie）——禁止反宪政党
  - 人性尊严（Menschenwürde）——第1条不可修改核心

### 7.3 英国宪法
- 不成文宪法：宪法性法律+宪法惯例+宪法判例
- 议会主权（Parliamentary Sovereignty）——议会立法不受限制
- 法治原则（Rule of Law）——戴雪（Dicey）三要素

### 7.4 日本宪法
- 现行宪法1946年颁布，1947年实施
- 和平宪法：第9条放弃战争，不保持战争力量
- 天皇为日本国象征（无政治权力）
- 国会两院制（众议院+参议院）

## 八、宪法与部门法的关系

### 8.1 宪法与行政法
- 行政法以宪法为根本依据
- 依法行政原则源于宪法法治原则
- 行政组织法、行政许可法、行政处罚法均以宪法授权为基础

### 8.2 宪法与刑法
- 刑法基本原则（罪刑法定）源于宪法精神
- 公民人身自由保护是宪法在刑法中的体现
- 死刑复核制度体现宪法生命权保障

### 8.3 宪法与民法
- 民法平等原则源于宪法平等权
- 私有财产保护：宪法第13条→物权法→民法典
- 人格权保护与宪法人格尊严

### 8.4 宪法与诉讼法
- 宪法规定的公民诉讼权保障
- 公开审判制度源于宪法第125条
- 辩护制度保障与宪法人身自由

## 九、宪法学重要概念辨析

### 9.1 宪法与宪政
- **宪法**：静态的根本法规范
- **宪政**：动态的宪法实施过程
- 宪政核心：有限政府、保障人权

### 9.2 宪法规范与宪法效力
- **宪法规范**：具有最高法律效力的规范
- **直接效力**：宪法可否在司法中直接适用（中国：不能直接援引）
- **间接效力**：通过普通法律间接实现宪法价值

### 9.3 宪法修改与宪法解释
- **修宪**：改变宪法条文（刚性程序）
- **释宪**：阐明宪法条文的含义（无需修改条文）
- **实质修改**：通过解释实质上改变原意

### 9.4 宪法序言的法律效力
- 宪法序言是否具有法律效力存在争议
- 中国通说：宪法序言与正文具有同等法律效力
- 历史叙述部分虽不直接设定规范，但具有指导意义

## 十、中国宪法实施机制

### 10.1 宪法宣誓制度
- 2018年宪法修正案正式确立
- 各级人大选举或决定任命的国家工作人员就职时公开宣誓
- 誓词：忠于宪法，维护宪法权威，履行法定职责

### 10.2 宪法日
- 12月4日为国家宪法日（1982年宪法通过日）
- 全国开展宪法宣传教育活动

### 10.3 宪法教育
- 宪法纳入国民教育体系
- 中小学法治教育中的宪法内容
- 领导干部宪法学习培训制度

### 10.4 宪法案例
- **齐玉苓案（1999）**：宪法受教育权司法化尝试（最高院批复后于2008年废止）
- **孙志刚案（2003）**：引发收容遣送制度废除，推动违宪审查讨论
- **宪法在裁判中的适用**：中国法院不直接援引宪法作为裁判依据，但可在说理部分引用宪法精神
- **备案审查案例**：全国人大常委会每年审查备案规范性文件，纠正合宪性/合法性问题
- **合宪性审查展望**：加强宪法监督能力建设，完善备案审查衔接联动机制
