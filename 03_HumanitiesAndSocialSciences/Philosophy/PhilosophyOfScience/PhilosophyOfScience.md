# 科学哲学

## 一、逻辑实证主义

### 1.1 维也纳学派

逻辑实证主义（维也纳学派，1920年代）主张科学哲学的核心任务是分析科学语言和科学方法。他们接受了休谟的"二分法"：分析命题（逻辑、数学真理）和综合命题（经验事实）构成一切有意义的陈述。

| 命题类型 | 真值依据 | 例 |
|---------|---------|-----|
| 分析命题 | 逻辑形式本身 | "所有单身汉都是未婚的" |
| 综合命题 | 经验验证 | "窗外在下雨" |
| 无意义命题 | 无法验证 | 形而上学陈述 |

### 1.2 可证实性原则

意义的可证实性原则：一个陈述是有意义的，当且仅当它在原则上可以被经验证实。这一原则将形而上学、伦理学、美学等领域的命题排除在科学之外。

| 版本 | 内容 | 问题 |
|------|------|------|
| 强的可证实性 | 可被观察语句演绎推出 | 过于严格，排除全称命题 |
| 弱的可证实性 | 可被经验证据部分确证 | 过于宽松 |
| 可检验性 | 可推导出可观察的检验命题 | 仍需明确标准 |

### 1.3 观察与理论

逻辑实证主义严格区分观察语言和理论语言：观察语言直接记录可观察事实，是理论中立的。理论语言通过对应规则与观察语言相连。这一二分法后来受到历史主义学派的强烈批评。

## 二、波普尔的证伪主义

### 2.1 分界问题

波普尔认为科学和非科学的分界标准不是可证实性，而是**可证伪性**。一个理论是科学的，当且仅当它原则上可以被经验证据证伪。

| 条件 | 科学（可证伪） | 非科学（不可证伪） |
|------|---------------|-----------------|
| 理论 | "所有天鹅都是白色的" | "明天要么下雨要么不下雨" |
| 特征 | 禁止某些情况发生 | 不禁止任何情况 |
| 候选 | 相对论、进化论 | 占星术、精神分析 |

### 2.2 猜想与反驳

科学的发展模式：$P_1 \to TT \to EE \to P_2$

| 阶段 | 含义 |
|------|------|
| $P_1$ | 初始问题 |
| $TT$ | 试探性理论（猜想） |
| $EE$ | 排除错误（反驳） |
| $P_2$ | 新问题 |

科学进步是不断提出大胆猜想和严格反驳的过程。波普尔强调，科学知识不是累积增长的，而是通过不断推翻旧理论而进步的。

### 2.3 逼真性

波普尔提出"逼真性"（verisimilitude）概念：虽然科学无法获得真理，但可以不断逼近真理。一个理论的逼真度等于它的真内容减去假内容。

## 三、库恩的范式革命

### 3.1 范式与科学共同体

库恩在《科学革命的结构》中提出"范式"概念：范式是科学共同体共同接受的理论框架、方法信念和价值标准。范式决定了什么问题值得研究、什么方法可接受、什么算作解答。

### 3.2 科学发展的动态模式

| 阶段 | 特征 | 活动 |
|------|------|------|
| 前科学 | 多种范式竞争，无统一框架 | 争论基本问题 |
| 常规科学 | 范式确立，共同体形成 | 解谜活动 |
| 危机 | 反常积累，范式失效 | 探索调整 |
| 革命 | 新旧范式竞争，更替 | 范式转换 |
| 新常规科学 | 新范式成为新标准 | 新的解谜 |

### 3.3 不可通约性

库恩提出新旧范式之间具有"不可通约性"（incommensurability）：不同范式的支持者在不同的世界中从事科学活动，他们对术语的定义、问题的设定和评价的标准各不相同。

| 不可通约的类型 | 含义 |
|--------------|------|
| 方法论 | 不同的解题方法和评价标准 |
| 感知 | 不同的观察方式和知觉经验 |
| 语义 | 相同的术语有不同的意义 |

### 3.4 对库恩的批评

| 批评者 | 观点 |
|--------|------|
| Popper | 库恩将科学理性降低为社会心理学 |
| Lakatos | 常规科学不是"解谜"而是"教条主义" |
| Laudan | 不可通约性被夸大了 |

## 四、拉卡托斯的科学研究纲领

### 4.1 纲领的结构

拉卡托斯在《证伪与科学研究纲领方法论》中提出，科学的基本单元不是单个理论，而是研究纲领。研究纲领由硬核和保护带构成：

| 部分 | 特征 | 例：牛顿纲领 |
|------|------|-------------|
| 硬核 | 不可反驳的基本假设 | 万有引力定律和运动三定律 |
| 保护带 | 可调整的辅助假说 | 行星轨道参数计算 |
| 正面启发法 | 指导发展的积极策略 | 用引力解释更多现象 |
| 反面启发法 | 禁止反驳硬核的规则 | 不质疑引力存在 |

### 4.2 进步与退化

| 纲领状态 | 特征 | 例 |
|---------|------|-----|
| 进步纲领 | 预言新颖事实，被部分证实 | 牛顿纲领（17-18世纪） |
| 退化纲领 | 仅事后解释，无新颖预言 | 牛顿纲领（19世纪末） |

科学史是进步纲领取代退化纲领的历史。

## 五、费耶阿本德的无政府主义

### 5.1 怎么都行

费耶阿本德在《反对方法》中主张科学方法论的无政府主义，提出唯一不违反科学进步的原则是"怎么都行"（anything goes）。不存在普遍的、不变的科学方法。

| 方法论神话 | 费耶阿本德的批判 |
|-----------|----------------|
| 唯一方法 | 任何方法论规则都有反例 |
| 理性重构 | 科学史远比理性模型复杂 |
| 科学优越性 | 科学只是众多知识传统之一 |

### 5.2 科学与国家分离

费耶阿本德主张科学与国家分离，类似政教分离。他反对科学在教育和公共政策中的垄断地位，主张给予非科学传统（如中医、巫术、占星术）与科学平等的发展机会。

## 六、科学实在论 vs 反实在论

### 6.1 核心争论

| 立场 | 对不可观察实体的态度 | 对科学理论的态度 |
|------|---------------------|-----------------|
| 科学实在论 | 理论实体真实存在 | 成熟理论近似为真 |
| 工具主义 | 理论实体只是有用的虚构 | 理论只是预测工具 |
| 经验构造论 | 经验适当性为准，不承诺存在 | 理论是对经验的建构 |
| 结构实在论 | 理论的结构关系真实 | 知识的是结构而非实体 |

### 6.2 无奇迹论证

实在论的"无奇迹论证"（Putnam）：科学如此成功，如果理论实体不存在，科学的成功就是奇迹。

### 6.3 悲观元归纳

反实在论的"悲观元归纳"（Laudan）：历史上大多数成功的科学理论后来被证明是错的，因此现在的理论也很可能被推翻。

## 七、贝叶斯主义

### 7.1 贝叶斯定理

贝叶斯主义将科学推理视为概率的更新过程。贝叶斯定理描述了给定证据 $E$ 时假设 $H$ 的后验概率：

$$
P(H|E) = \frac{P(E|H)P(H)}{P(E)}
$$

| 符号 | 含义 |
|------|------|
| $P(H)$ | 先验概率——初始信念 |
| $P(E|H)$ | 似然度——假设预测证据的力度 |
| $P(H|E)$ | 后验概率——更新后的信念 |
| $P(E)$ | 证据的无条件概率 |

### 7.2 确认理论

贝叶斯定理提供了确认的逻辑：如果 $P(H|E) > P(H)$，则 $E$ 确认 $H$。确认程度：

$$
\text{确认度} = P(H|E) - P(H)
$$

贝叶斯主义为科学推理提供了统一的框架，解决了归纳问题、理论选择和证据权重等经典难题。

## 八、科学知识社会学

### 8.1 强纲领

Bloor 提出科学知识社会学（SSK）的"强纲领"——科学知识本身（包括数学和逻辑）应当作为社会学分析的对象。强纲领的四条信条：

| 信条 | 内容 |
|------|------|
| 因果性 | 科学知识产生的原因应当被解释 |
| 公正性 | 对真和假、合理和不合理的信念做对称分析 |
| 对称性 | 用同一类型的原因解释真假信念 |
| 反身性 | 上述原则也适用于 SSK 自身 |

### 8.2 实验室研究

拉图尔和伍尔加在《实验室生活》中通过人类学方法研究科学实验室。他们揭示科学事实不是被"发现"的，而是在实验室中被"建构"的——通过设备、记录、讨论、论文审稿等一系列实践，从混沌的观察中创造出确定的科学事实。

### 8.3 科学论战

科学论战的研究揭示了科学共识形成的社会过程。柯林斯（Collins）对引力波探测的案例研究表明，实验的可重复性本身就是社会协商的产物——什么是"相同的实验"需要科学共同体来判定。

| 阶段 | 特征 |
|------|------|
| 实验阶段 | 早期结果自相矛盾 |
| 协商阶段 | 各学派争论方法标准 |
| 封闭阶段 | 科学共同体达成共识 |

### 8.4 科学主义与反科学主义

科学主义的争议：科学是否是唯一有效的知识形式？科学是否应当在公共决策中享有特权地位？反科学主义批判科学在当代社会中的过度权威化，主张科学不是价值中立的，且其他知识形式（本地知识、传统知识）具有不可替代的价值。

## 九、科学实在论与反实在论的前沿

### 9.1 实体实在论

Hacking 提出"实体实在论"（entity realism）——相信实验科学中的理论实体（如电子、基因）是真实存在的，因为这些实体被用来干预自然并产生可观察的效应。"如果你可以喷洒它们，它们就是真实的。"

### 9.2 结构实在论

Worrall 提出"结构实在论"——理论实体可能随着理论更替而改变，但理论的数学结构保留了下来。例如，菲涅尔的波动光学到麦克斯韦电磁理论，实体（以太）被抛弃了，但数学结构得到了保留。

### 9.3 模型与表征

当代科学哲学越来越关注科学模型在科学实践中的作用。模型的类型：

| 模型类型 | 特征 | 例 |
|---------|------|-----|
| 尺度模型 | 物理缩比 | 风洞中的飞机模型 |
| 类比模型 | 基于类比 | 弹球模型的气体分子 |
| 理想化模型 | 简化假设 | 无摩擦平面 |
| 现象学模型 | 拟合数据的经验规律 | 洛伦茨吸引子 |
| 数学模型 | 数学方程描述 | 种群增长的逻辑斯蒂方程 |

### 9.4 价值与科学

科学中的价值问题：认知价值（简单性、普遍性、精确性）与道德社会价值在科学推理中的角色。科学是否应当（或能否）价值中立？

| 立场 | 观点 |
|------|------|
| 价值中立理想 | 科学只关注事实，不应受价值影响 |
| 价值不可避免 | 价值渗透于问题选择、方法偏好、证据解读 |
| 价值有认知作用 | 价值可以合理引导证据解读 |
