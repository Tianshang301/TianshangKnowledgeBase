# 西方哲学史

## 一、古希腊哲学（约前6世纪-前3世纪）

### 1.1 前苏格拉底哲学

古希腊最早的哲学家探索世界的本原（arche），从神话思维转向理性思维。

| 学派/哲学家 | 本原/核心概念 | 主要思想 | 名言 |
|-------------|---------------|----------|------|
| **米利都学派** | | | |
| 泰勒斯（Thales） | **水** | 水是万物的本原；第一个哲学家 | "万物充满神灵" |
| 阿那克西曼德 | **无限定者**（apeiron） | 无定形的本源，生成万物 | |
| 阿那克西美尼 | **气** | 气通过稀散和凝聚生成万物 | |
| **毕达哥拉斯学派** | **数** | 数是万物的本原；灵魂转世；音乐和谐 | "万物皆数" |
| **赫拉克利特** | **火**与**逻各斯**（Logos） | 万物流变，对立统一，逻各斯是宇宙的规律 | "人不能两次踏进同一条河流" |
| **巴门尼德** | **存在**（Being） | 存在是唯一、不变、永恒的；思维与存在同一 | "存在者存在，非存在者不存在" |
| **芝诺悖论** | 否定运动的逻辑论证 | 阿基里斯追不上乌龟、飞矢不动 | 揭示有限与无限的矛盾 |
| **德谟克利特** | **原子**（atom）与**虚空** | 万物由不可分割的原子和虚空构成；机械论宇宙观 | "一切都由必然性产生" |

**核心转变**：从神话解释（Mythos）到理性解释（Logos）——哲学从此诞生。

### 1.2 苏格拉底（Socrates，前469-前399）

苏格拉底将哲学从天上拉回人间，专注于伦理问题。

**苏格拉底方法（精神助产术）**：
1. **诘问（Elenchus）**：通过不断提问揭露对方的无知和矛盾
2. **归纳**：从个别事例中归纳出普遍定义
3. **助产**：帮助对方"生出"真理——真理在人的内心，而非灌输进去

**核心思想**：
- **认识你自己**（德尔斐神谕）——哲学的第一要务是认识自己
- **美德即知识**：道德行为基于知识——无人自愿作恶，作恶源于无知
- **灵魂关怀**：最重要的不是身体或财富，而是灵魂的完善
- **坦然赴死**：因"败坏青年"和"不信城邦之神"被判处死刑——"我去死，你们去活，谁的去处好，唯有神知道"

### 1.3 柏拉图（Plato，前427-前347）

柏拉图是苏格拉底的学生，西方哲学史上最重要的人物之一。

**理念论（Theory of Forms/Ideas）**：

| 两个世界 | 特征 | 实例 |
|----------|------|------|
| **理念世界** | 永恒、不变、完美、真实 | "圆"的理念——完美的圆 |
| **感官世界** | 变化、生成、不完美、虚幻 | 现实中的圆——近似的圆 |

**洞穴比喻**——认识论的核心隐喻：
- 囚徒面壁，只能看见墙壁上的影子（感官世界）
- 转身走出洞穴，看见真实事物和太阳（理念世界和善的理念）
- 返回洞穴告诉其他人，却被视为疯子——哲学家与大众的隔阂

**灵魂三分与城邦三分**：

| 灵魂部分 | 德性 | 对应城邦阶层 |
|----------|------|-------------|
| **理性** | **智慧** | **统治者（哲学王）** |
| **激情** | **勇敢** | **护卫者（军人）** |
| **欲望** | **节制** | **生产者（平民）** |
| **整体正义** | **各尽其职，互不干涉** | |

**《理想国》**：政治哲学经典——正义是"各做各的事"，哲学王是最理想的统治者。

**回忆说**：学习即灵魂对前世知识的回忆——"一切研究不过是回忆而已"

### 1.4 亚里士多德（Aristotle，前384-前322）

"吾爱吾师，吾更爱真理"——亚里士多德批判并超越了柏拉图。

**形而上学（Metaphysics）**——"第一哲学"：

| 概念 | 含义 |
|------|------|
| **实体（Substance）** | 独立存在的个别事物——"这一个"（如这匹马、这个人） |
| **形式（Form）** | 事物的本质、定义 |
| **质料（Matter）** | 构成事物的材料 |
| **现实（Actuality）** | 已经实现的、完全的存在 |
| **潜能（Potentiality）** | 潜在的、尚未实现的存在 |

**四因说**——解释事物存在和变化的四种原因：

| 原因 | 解释 | 示例（一座雕像） |
|------|------|------------------|
| **质料因** | 由什么构成 | 大理石 |
| **形式因** | 是什么 | 雕像的形状和本质 |
| **动力因** | 由什么造成 | 雕刻家的技艺 |
| **目的因** | 为了什么 | 装饰/纪念 |

**逻辑学**：
- **三段论（Syllogism）**——形式逻辑的奠基
  - 大前提：所有人都会死
  - 小前提：苏格拉底是人
  - 结论：苏格拉底会死
- 提出了概念、判断、推理的基本逻辑框架

**伦理学**：
- **德性伦理学**——"幸福（Eudaimonia）是最高善"
- **中道（Golden Mean）**：德性是两个极端之间的中道
  - 勇敢是怯懦（不足）与鲁莽（过度）的中道
  - 慷慨是吝啬与浪费的中道
- "美德即中道"

**政治学**：
- "人是政治的动物"——只有在城邦中才能实现人的本性
- 政体分类：君主制→暴君制；贵族制→寡头制；共和制→民主制

## 二、中世纪哲学（约5-15世纪）

### 2.1 教父哲学

**奥古斯丁（Augustine，354-430）**：

| 思想 | 内容 |
|------|------|
| **上帝之城 vs 世俗之城** | 两种爱造就两座城——爱上帝至极者组成上帝之城，爱自己至极者组成世俗之城 |
| **时间论** | 时间是心灵的延伸——过去是记忆，现在是直观，未来是期待 |
| **自由意志** | 恶不是实体，而是善的缺乏——自由意志的误用导致堕落 |
| **恩典论** | 人无法靠自身得救，必须依靠上帝的恩典 |
| **《忏悔录》** | 第一部自传体哲学著作——"我的灵魂在不安中寻找你" |

### 2.2 经院哲学

**托马斯·阿奎那（Thomas Aquinas，1225-1274）**：

**五大证明（Five Ways）**——理性证明上帝存在：

| 论证 | 起点 | 逻辑 |
|------|------|------|
| **第一推动者** | 事物运动需要推动 | 追溯至第一个不动的推动者 |
| **第一因** | 结果必有原因 | 追溯至第一个无因之因 |
| **可能性与必然性** | 事物生灭，可有可无 | 必然存在的存在者使其他存在成为可能 |
| **完善性等级** | 事物有美善真程度差异 | 最完善的存在者为标准 |
| **目的论** | 无理智事物有目的 | 存在一个智慧的存在安排万物 |

**信仰与理性**：信仰超越理性但不违背理性——"哲学是神学的婢女"

**奥卡姆的威廉（William of Ockham，约1287-1347）**：
- **奥卡姆剃刀**："如无必要，勿增实体"（Entities should not be multiplied beyond necessity）
- 最简单的解释往往是最好的
- 唯名论：共相只是名称，个别事物才是真实存在

## 三、近代哲学（17-18世纪）

### 3.1 理性主义（Rationalism）

**笛卡尔（Descartes，1596-1650）——近代哲学之父**：

| 思想 | 内容 | 突破 |
|------|------|------|
| **方法论的怀疑** | 怀疑一切可以怀疑的东西 | 以理性为判准，打破经院哲学权威 |
| **"我思故我在"**（Cogito ergo sum） | 唯一不可怀疑的事实——我在思考 | 找到哲学的第一确定出发点 |
| **心物二元论** | 心灵（思维实体）与物质（广延实体）相互独立 | 身心问题成为哲学核心问题 |
| **天赋观念** | 有些观念（如数学公理、上帝观念）是先天固有的 | 反对经验主义白板说 |

**斯宾诺莎（Spinoza，1632-1677）**：
- **泛神论**：上帝即自然（Deus sive Natura）——唯一实体
- **身心平行论**：思想和广延是同一实体的两个不可通约的属性
- **《伦理学》**：用几何学方法论证——"自由是对必然的认识"

**莱布尼茨（Leibniz，1646-1716）**：
- **单子论**：世界由无数不可分割的精神性单子构成——"单子没有窗户"
- **预定和谐**：上帝预先安排所有单子的协调一致
- **最好世界**："我们生活在一切可能世界中最好的世界"

### 3.2 经验主义（Empiricism）

| 哲学家 | 核心主张 | 名句/概念 |
|--------|----------|-----------|
| **洛克（Locke）** | 白板说（Tabula Rasa）——心灵最初如白纸 | 第一性质（广延、运动）vs 第二性质（颜色、声音） |
| **贝克莱（Berkeley）** | "存在即被感知"（Esse est percipi） | 物质实体不存在，只有心灵和观念——主观唯心主义 |
| **休谟（Hume）** | 一切知识源于印象和观念 | 因果关系是习惯性联想而非理性必然 |

**休谟的怀疑论**：
- **印象与观念**：印象（直接感觉）→观念（记忆和想象）——一切知识源于感觉经验
- **因果问题**：我们看不到"因果"，只看到"先后"和"恒常联结"——因果关系只是心理习惯
- **归纳问题**：从有限事例推出普遍规律没有逻辑必然性
- **对康德的唤醒**："休谟把我从独断论的睡梦中唤醒"

### 3.3 康德（Kant，1724-1804）——哥白尼式革命

**哥白尼式革命**：不是认识符合对象，而是对象符合认识的形式。

| 概念 | 含义 |
|------|------|
| **先天综合判断** | 既增加知识（综合）又具有普遍必然性（先天）——数学和自然科学的基础 |
| **现象（Phenomena）** | 事物向我们的感官和知性呈现的样子 |
| **物自体（Noumena）** | 事物自身的本来面目——不可认识 |
| **知性十二范畴** | 时空（感性直观形式）+ 因果等12范畴（知性概念）= 经验成为可能 |
| **先验统觉** | "我思"伴随一切表象——意识的统一性 |

**三大批判**：

| 批判 | 领域 | 核心问题 | 主要发现 |
|------|------|----------|----------|
| **《纯粹理性批判》** | 认识论 | 我能知道什么？ | 人为自然立法——知性为自然立规律 |
| **《实践理性批判》** | 伦理学 | 我应当做什么？ | 人为自己立法——道德律令在人心 |
| **《判断力批判》** | 美学 | 我可以希望什么？ | 人为美立法——审美无利害 |

**绝对命令（Categorical Imperative）**：
- 普遍化原则："要只按照你同时认为也能成为普遍规律的准则去行动"
- 人是目的："你要如此行动，即无论对你自己还是对他人，始终把人当作目的，而不仅仅是手段"

## 四、19世纪哲学

### 4.1 德国唯心主义

**黑格尔（Hegel，1770-1831）**——绝对精神与辩证法

| 核心概念 | 含义 |
|----------|------|
| **绝对精神** | 宇宙的终极实在，自我实现、自我认识的过程 |
| **辩证法** | 正题→反题→合题——通过矛盾推动发展 |
| **《精神现象学》** | 意识从感性确定性到绝对知识的发展历程 |
| **"现实的就是合理的"** | 现实不等于现存——合乎理性的现实才是真正的现实 |

### 4.2 叔本华（Schopenhauer，1788-1860）

- **作为意志和表象的世界**：世界有两面——表象（现象）和意志（本质）
- **意志**：盲目的、永不满足的生命冲动——人生的本质是痛苦
- **解脱之道**：审美观照（暂时摆脱意志）、同情（超越个体化原理）、禁欲（彻底否定意志）

### 4.3 尼采（Nietzsche，1844-1900）

| 概念 | 含义 | 出处 |
|------|------|------|
| **"上帝死了"** | 传统基督教道德已失去约束力 | 《快乐的科学》 |
| **权力意志（Will to Power）** | 生命本质是自我超越、自我扩张 | 《善恶的彼岸》 |
| **超人（Übermensch）** | 超越传统善恶、创造新价值的人 | 《查拉图斯特拉如是说》 |
| **主人道德 vs 奴隶道德** | 主人肯定自身，奴隶否定生命 | 《论道德的谱系》 |
| **永恒轮回** | "假如恶魔在某一天闯入你的孤独……" | 《快乐的科学》 |
| **重估一切价值** | 批判传统价值，创造新价值 | 《偶像的黄昏》 |

## 五、现代与当代哲学

### 5.1 存在主义

| 哲学家 | 核心思想 |
|--------|----------|
| **海德格尔（Heidegger）** | 《存在与时间》——Dasein（此在）的生存论分析，"向死而生" |
| **萨特（Sartre）** | "存在先于本质"——人自由地创造自己；"他者即地狱" |
| **加缪（Camus）** | 荒谬——西西弗推石上山，在无意义中创造意义 |
| **克尔凯郭尔（Kierkegaard）** | 个体的主观真理——信仰的跳跃（先驱） |

### 5.2 分析哲学

| 哲学家 | 贡献 |
|--------|------|
| **弗雷格（Frege）** | 现代逻辑奠基，概念文字 |
| **罗素（Russell）** | 摹状词理论，逻辑原子主义 |
| **维特根斯坦（早期）** | 《逻辑哲学论》——语言是现实的图像 |
| **维特根斯坦（后期）** | 《哲学研究》——语言游戏，意义即用法 |
| **逻辑实证主义** | 可证实原则——形而上学命题无认知意义 |

### 5.3 现象学

| 哲学家 | 贡献 |
|--------|------|
| **胡塞尔（Husserl）** | "回到事物本身"——意向性（意识总是关于某物的意识） |
| **梅洛-庞蒂（Merleau-Ponty）** | 身体现象学——知觉的首要性 |

### 5.4 后现代主义

| 哲学家 | 核心思想 |
|--------|----------|
| **福柯（Foucault）** | 知识即权力——规训社会，《词与物》《规训与惩罚》 |
| **德里达（Derrida）** | 解构主义——文本没有固定意义，意义是延异的 |
| **利奥塔（Lyotard）** | 宏大叙事的终结——知识合法性的危机 |
| **鲍德里亚（Baudrillard）** | 拟像与超真实——真实被符号所取代 |

### 5.5 西方哲学史总结

| 时期 | 核心问题 | 主要成就 |
|------|----------|----------|
| 古希腊 | 本原、存在、知识、德性 | 理性思维、逻辑学、伦理学奠基 |
| 中世纪 | 上帝、信仰与理性 | 哲学与神学的融合，唯名论与实在论之争 |
| 近代 | 认识论、主体性 | 科学方法论、理性主义与经验主义、康德批判哲学 |
| 19世纪 | 意志、历史、价值 | 辩证法、意志哲学、价值重估 |
| 20世纪 | 语言、存在、权力 | 分析哲学、现象学、存在主义、后现代主义 |

### 5.6 西方哲学的核心问题演变

| 时期 | 核心问题 | 典型回答 | 代表人物 |
|------|----------|----------|----------|
| **本体论**（古希腊） | 世界的本原是什么？ | 水、数、原子、理念、实体 | 泰勒斯→德谟克利特→柏拉图→亚里士多德 |
| **神正论**（中世纪） | 上帝是否存在？ | 五大证明、本体论证明 | 奥古斯丁、阿奎那、安瑟尔谟 |
| **认识论**（近代） | 我们能知道什么？ | 理性主义（天赋观念）、经验主义（白板说） | 笛卡尔、洛克、休谟、康德 |
| **语言转向**（20世纪） | 语言如何与世界相关？ | 图像论、语言游戏 | 维特根斯坦（早/晚期） |
| **存在论**（20世纪） | 什么是"存在"？ | 此在的生存论分析 | 海德格尔、萨特 |

### 5.7 重要哲学术语对照

| 术语（中文） | 术语（英文） | 核心含义 |
|-------------|-------------|----------|
| 本体论 | Ontology | 研究"存在"本身的哲学分支 |
| 认识论 | Epistemology | 研究知识的来源、范围和限度的哲学分支 |
| 伦理学 | Ethics | 研究道德善恶的哲学分支 |
| 形而上学 | Metaphysics | 超越经验的研究终极实在的哲学 |
| 辩证法 | Dialectic | 通过矛盾对立的统一达到真理的方法 |
| 先验 | Transcendental | 经验成为可能的先天条件（康德） |
| 超验 | Transcendent | 超越经验范围的（如上帝、物自体） |
| 主体性 | Subjectivity | 人作为认识和行为主体的地位 |
| 意向性 | Intentionality | 意识总是"关于"某物的意识（胡塞尔） |
| 此在 | Dasein | 人的存在方式——"在世界之中存在"（海德格尔） |

### 5.8 西方哲学学习建议

1. **断代纵览**：先通读哲学史（罗素《西方哲学史》或梯利《西方哲学史》）
2. **原著精读**：选读核心原著——柏拉图《理想国》、笛卡尔《第一哲学沉思集》
3. **问题导向**：选择一个核心问题（如自由意志、心身问题）追踪不同时代的回答
4. **概念辨析**：西方哲学的关键在于概念的精确界定——厘清术语是理解的基础
5. **批判思维**：不是被动接受，而是与哲学家对话——"哲学不是占有真理，而是追求真理"
