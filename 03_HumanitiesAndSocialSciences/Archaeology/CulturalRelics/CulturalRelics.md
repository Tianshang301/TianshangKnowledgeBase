# 文物学 (Cultural Relics Studies)

## 一、文物分类体系

### 1.1 基本分类

| 分类维度 | 类别 | 说明 |
|---------|------|------|
| **可移动性** | 可移动文物 | 器物、书画、文献等可搬运对象 |
| | 不可移动文物 | 古建筑、遗址、石窟等固定遗存 |
| **材质属性** | 有机质文物 | 竹木漆器、纺织品、皮革、骨角器 |
| | 无机质文物 | 陶瓷、金属、玉石、玻璃、石器 |
| **时代** | 史前文物 | 旧石器时代至青铜时代 |
| | 历史时期文物 | 夏商周至明清 |
| | 近现代文物 | 1840年以后 |

### 1.2 中国文物定级标准

| 级别 | 标准 |
|------|------|
| 一级文物 | 具有特别重要历史、艺术、科学价值的代表性文物 |
| 二级文物 | 具有重要历史、艺术、科学价值的文物 |
| 三级文物 | 具有一定历史、艺术、科学价值的文物 |
| 一般文物 | 有一定价值但不足以定级的文物 |

## 二、陶瓷器

### 2.1 陶器与瓷器

| 特征 | 陶器 | 瓷器 |
|------|------|------|
| 原料 | 普通黏土 | 高岭土（瓷土） |
| 烧成温度 | 800-1100℃ | 1200-1400℃ |
| 吸水率 | >3% | <0.5% |
| 胎体 | 不透明，断面粗糙 | 半透明，断面致密 |
| 釉 | 无釉或低温釉 | 高温玻璃质釉 |

### 2.2 中国名窑体系

| 窑系 | 时代 | 特点 | 代表釉色 |
|------|------|------|---------|
| 汝窑 | 北宋 | 官窑御用，存世极少 | 天青釉 |
| 官窑 | 北宋/南宋 | 官方烧造，紫口铁足 | 粉青、灰青 |
| 哥窑 | 宋 | 金丝铁线开片 | 米黄、粉青 |
| 定窑 | 唐—元 | 白瓷为主，刻花印花 | 白釉 |
| 钧窑 | 宋—元 | 窑变釉，入窑一色出窑万彩 | 天蓝、玫瑰紫 |
| 龙泉窑 | 宋—明 | 青瓷顶峰，梅子青、粉青 | 青釉 |
| 景德镇窑 | 元—今 | 瓷都，青花、粉彩、珐琅彩 | 白釉为底 |

### 2.3 陶瓷断代方法

**传统方法**：
- 型制学：器型演变规律（如瓶口由宽变窄）
- 纹饰学：题材、风格、技法变化
- 款识学：官窑年号款、民窑吉语款

**科技方法**：
- 热释光（TL）：测定最后一次受热年代
- XRF 成分分析：胎釉元素组成比对
- 岩相学：原料产地溯源

## 三、青铜器

### 3.1 铸造工艺

**块范法**（Piece-mold Casting）：中国青铜时代主流铸造技术。以陶泥制模，翻制外范，制作内芯，组合范芯后浇铸铜液。

**失蜡法**（Lost-wax Casting）：以蜡制模，外敷泥范，加热脱蜡后形成空腔，再浇铸铜液。适用于复杂镂空器物。

### 3.2 青铜器类型与功能

| 类型 | 功能 | 代表器物 |
|------|------|---------|
| 礼器 | 祭祀、宴飨 | 鼎、簋、尊、爵 |
| 乐器 | 礼乐演奏 | 钟、铎、铙 |
| 兵器 | 战争 | 戈、矛、剑、镞 |
| 工具 | 生产 | 斧、锛、凿、刀 |
| 车马器 | 车马装饰 | 轭、辖、衔、当卢 |

### 3.3 铭文与锈蚀

**铭文研究**：
- 商代：族徽、人名，字数少
- 西周：长篇记事，史料价值极高
- 春秋战国：地域风格分化

**锈蚀分析**：
- 有害锈：粉状锈（碱式氯化铜），会蔓延破坏器物
- 无害锈：铜绿（碱式碳酸铜），稳定保护层
- 分析手段：XRD 鉴定锈蚀物相，SEM 观察微观结构

## 四、玉器

### 4.1 玉材分类

| 种类 | 矿物学 | 特征 | 主要产地 |
|------|--------|------|---------|
| 和田玉（软玉） | 透闪石—阳起石 | 温润细腻，韧性高 | 新疆和田 |
| 翡翠（硬玉） | 钠铝辉石 | 玻璃光泽，颜色丰富 | 缅甸 |
| 岫岩玉 | 蛇纹石 | 硬度低，透明度好 | 辽宁岫岩 |
| 独山玉 | 黝帘石化斜长岩 | 多色混杂 | 河南南阳 |

### 4.2 史前玉文化

**红山文化**（约6500-5000年前）：
- 特征器物：C 形龙、猪龙、勾云形器
- 工艺：片切割、线切割、钻孔、打磨
- 功能：祭祀、身份象征

**良渚文化**（约5300-4300年前）：
- 特征器物：玉琮、玉璧、玉钺
- 工艺：微雕神人兽面纹，切割精度极高
- 社会意义：宗教权力与军事权力的物化

### 4.3 玉器鉴定

- 材质鉴定：折射率、密度、红外光谱
- 工艺鉴定：钻孔方式（桯钻、管钻）、切割痕迹
- 沁色分析：埋藏环境导致的色变，需区分天然沁与人工做旧

## 五、书画与纺织品

### 5.1 书画鉴定

**材料分析**：

| 材料 | 鉴定要点 |
|------|---------|
| 纸 | 纤维种类（麻、楮、檀、竹）、帘纹、厚度 |
| 绢 | 经纬密度、丝线粗细、织造方式 |
| 墨 | 烟炱种类（松烟、油烟）、胶料成分 |
| 颜料 | 天然矿物 vs 合成颜料（出现年代） |

**鉴定方法**：
- 风格鉴定：笔墨特征、构图习惯、款识书法
- 文献考证：著录、题跋、印章流传
- 科技检测：碳14（纸绢）、拉曼光谱（颜料）、X 射线（底层草稿）

### 5.2 纺织品

**织造结构**：
- 平纹：经纬一上一下交织，最基础
- 斜纹：经纬二上一下或三上一下，质地柔软
- 缎纹：经纬交织点分散，表面光滑
- 提花：多色经纬按图案提沉

**染料分析**：天然染料——靛蓝、茜草红、栀子黄、紫草紫；分析手段——高效液相色谱（HPLC）、质谱（MS）

## 六、文物保护

### 6.1 保护原则

| 原则 | 内涵 |
|------|------|
| 最小干预 | 仅在必要时进行处理，保留原始信息 |
| 可逆性 | 修复材料可安全去除，不损害原物 |
| 可识别性 | 修复部分可与原物区分 |
| 完整性 | 保护文物整体及其环境 |

### 6.2 预防性保护

**环境控制**：
- 温度：20±2℃（有机质），波动<5℃/日
- 相对湿度：40±5%（金属、陶瓷），50-55%（有机质）
- 光照：50勒克斯（有机质），<200勒克斯（无机质）
- 污染物：SO₂、NO₂、O₃、颗粒物控制

### 6.3 修复技术

| 文物类型 | 常见问题 | 修复方法 |
|---------|---------|---------|
| 青铜器 | 有害锈、断裂 | 去氯、缓蚀、粘接、补配 |
| 陶瓷器 | 破碎、缺损 | 清洗、粘接、补配、作色 |
| 书画 | 虫蛀、霉变、断裂 | 清洗、加固、全色、装裱 |
| 纺织品 | 腐朽、褪色 | 清洗、加固、背衬、存储 |
| 漆木器 | 变形、干裂 | 脱水、定形、加固 |

## 七、文化遗产法律与国际框架

### 7.1 中国文物保护法

**核心制度**：
- 文物保护单位：全国重点文物保护单位（国家级）、省级、市县级
- 考古发掘许可：须经国务院文物行政部门批准
- 文物出境：一级文物禁止出境，二三级文物限制出境
- 文物收藏：国有博物馆、民间收藏、考古发掘品

### 7.2 世界遗产体系

| 公约 | 年份 | 核心内容 |
|------|------|---------|
| 世界遗产公约 | 1972 | 保护具有突出普遍价值的文化和自然遗产 |
| 非物质文化遗产公约 | 2003 | 保护口头传统、表演艺术、社会实践 |
| 水下文化遗产保护公约 | 2001 | 保护水下考古遗存，优先原址保护 |
