# 现代汉语

## 一、语音

### 1.1 声韵调系统

普通话有22个声母（含零声母）、39个韵母、4个声调。**声调采用五度标记法**：阴平(55)、阳平(35)、上声(214)、去声(51)。

**声韵配合规律**：

| 声母类别 | 开口呼 | 齐齿呼 | 合口呼 | 撮口呼 |
|----------|--------|--------|--------|--------|
| b p m | + | + | +（限于u） | - |
| f | +（限于o/ou） | - | +（限于u） | - |
| d t | + | + | + | - |
| n l | + | + | + | + |
| g k h | + | - | + | - |
| j q x | - | + | - | + |
| zh ch sh r | + | - | + | - |
| z c s | + | - | + | - |
| 零声母 | + | + | + | + |

### 1.2 变调

**上声变调**：
- 上声+上声→阳平(35)+上声(214)：如"你好" nǐhǎo→níhǎo，"很好" hěnhǎo→hénhǎo
- 上声+非上声→半上(211)+非上声："好书" hǎo shū，"好人" hǎo rén，"好事" hǎo shì

**"一""不"变调**：

| 原字 | 后接声调 | 变调 | 例词 |
|------|----------|------|------|
| 一 | 去声 | 阳平35 | 一个yí gè |
| 一 | 非去声（阴/阳/上） | 去声51 | 一天yì tiān |
| 一 | 嵌在词语中间 | 轻声 | 看一看 |
| 不 | 去声 | 阳平35 | 不对bú duì |
| 不 | 非去声 | 去声51 | 不好bù hǎo |
| 不 | 嵌在词语中间 | 轻声 | 差不多 |

**轻声**：音节失去原调，读得又轻又短。语法作用：区分词义和词性——"东西"dōngxi（事物）vs dōngxī（方向）；"地道"dìdao（纯正）vs dìdào（地下通道）。

**儿化**：韵母卷舌化。区别词义（画/画儿）、表示细小/喜爱（花儿、小孩儿）、改变词性（盖/盖儿）。

### 1.3 语调与重音

**句调**：陈述句降调↘、疑问句（是非问）升调↗、特殊疑问句降调↘、祈使句降调↘、感叹句降调或高亢。

**重音类型**：语法重音（谓语>主语）、逻辑重音（对比强调）、情感重音（突出情绪）。

## 二、词汇

### 2.1 词汇构成

| 来源 | 比例近似 | 例词 | 说明 |
|------|----------|------|------|
| 古语词 | 核心基本词 | 人、山、水、大、小、日、月 | 传承词汇，稳定不变 |
| 方言词 | 补充 | 搞（西南官话）、买单（粤语）、忽悠（东北话） | 从方言进入普通话 |
| 外来词 | 文化交流 | 咖啡、幽默、因特网、GDP | 音译/意译/字母词 |
| 新造词 | 社会发展 | 上网、刷屏、内卷、躺平、元宇宙 | 反映时代变化 |

### 2.2 外来词类型

| 类型 | 说明 | 例词 |
|------|------|------|
| 音译 | 模拟外语发音 | 沙发、巧克力、酷、秀、模特 |
| 意译 | 翻译词义 | 计算机、电话、软件、篮球 |
| 音译兼意译 | 音义兼顾 | 可口可乐、幽默、基因、托福 |
| 音译+意译语素 | 部分音译部分意译 | 啤酒、卡片、芭蕾舞、高尔夫球 |
| 字母词 | 直接使用外文字母 | GDP、NBA、WiFi、AI、PPT |
| 日源回归 | 日语创造的汉字词 | 经济、社会、哲学、干部、手续 |

### 2.3 新词语

**信息技术类**：上网、网站、电商、微信、微博、区块链、元宇宙。**社交生活类**：宅、萌、吐槽、给力、点赞、内卷、躺平、破防。**经济类**：月光族、房奴、共享经济、斜杠青年。**网络流行语**：YYDS、绝绝子、EMO。

**新词衍生规律**：双音节占主导（约70%），缩略语增多（高知=高级知识分子、疾控=疾病控制），词缀化趋势（-化、-性、-主义），外来词持续增加。

## 三、语法

### 3.1 词类

| 词类 | 定义 | 例词 | 语法功能 |
|------|------|------|----------|
| 名词 | 表示人、事物、时间、地点 | 学生、书、北京、昨天 | 作主宾语，可受数量短语修饰 |
| 动词 | 表示动作、状态 | 走、看、是、有、喜欢 | 作谓语，可带宾语，可加"了/着/过" |
| 形容词 | 表示性质、状态 | 好、大、红、快乐 | 作谓语/定语，可加"很" |
| 数词 | 表示数目 | 一、百、第一、几 | 与量词组合修饰名词 |
| 量词 | 计算单位 | 个、条、次、遍、双 | 与数词组合，分为名量词和动量词 |
| 代词 | 代替其他词 | 我、这、谁、什么 | 所指明确，活用产生修辞效果 |
| 副词 | 修饰谓词 | 很、都、不、已经、正在 | 作状语，一般不修饰名词 |
| 介词 | 引介体词成分 | 在、从、对、把、被 | 与名词构成介词短语作状语/补语 |
| 连词 | 连接词/短语/句子 | 和、而且、因为、但是 | 连接作用，不充当句子成分 |
| 助词 | 附着性语法成分 | 的、地、得、了、着、过 | 附着性强，独立性弱 |

### 3.2 特殊句式

**把字句**：主语+把+宾语+动词+补语。"我把作业写完了"——将宾语提到动词前，强调对宾语的处理。动词不能是光杆形式。

**被字句**：受事+被+施事+动词。"作业被他写完了"——表达被动。传统上多表不如意，中性用法渐增。

**连动句**：同一主语连续发出两个动作。"我去商店买东西""他站起来开门"。

**兼语句**：前一动词的宾语是后一动词的主语。"老师让我们回答问题""他派小王去办事"。

**存现句**：表示存在、出现或消失。"桌子上放着一本书""村里来了一个人""昨天走了一批客人"。

**是字句**：表示判断、强调或存在。"他是学生""他是昨天来的""到处都是人"。

### 3.3 句类

| 句类 | 功能 | 语气词 | 例句 |
|------|------|--------|------|
| 陈述句 | 陈述事实 | 的、了、嘛、呢、啊 | "他来了。" |
| 疑问句 | 提出问题 | 吗、呢、吧 | "他来了吗？""谁来？" |
| 祈使句 | 请求/命令 | 吧、啊 | "你快过来吧！" |
| 感叹句 | 强烈情感 | 啊、啦 | "太好了！" |

**疑问句类型**：是非问（用"吗"）、特指问（用疑问词）、选择问（A还是B）、正反问（V不V）。

### 3.4 复句

**联合复句**：并列（一边……一边）、承接（先……然后）、递进（不但……而且）、选择（或者……或者）。**偏正复句**：因果（因为……所以）、转折（虽然……但是）、条件（只有……才）、假设（如果……就）、目的（为了……）、让步（即使……也）。

**关联词语**是复句的标志，作用在于明确分句间的逻辑关系。

## 四、语用与修辞

### 4.1 合作原则

格赖斯（Grice, 1975）提出：**量准则**（信息充分必要）、**质准则**（真实）、**关联准则**（相关）、**方式准则**（清晰）。违背合作原则产生**会话含义**（implicature），如隐喻、反讽。

### 4.2 礼貌原则

利奇（Leech, 1983）提出六准则：**得体**（少让他人受损）、**慷慨**（少让自己得益）、**赞誉**（多赞誉他人）、**谦逊**（多贬自己）、**一致**（减少分歧）、**同情**（增加同情）。

汉语尤其注重谦逊原则，如"拙作""寒舍""贵姓""赐教"等敬辞谦辞体系丰富。

### 4.3 修辞手法

汉语修辞充分利用汉字音形义特点：对偶（音节对称工整）、顶真（首尾蝉联）、回文（正反可读——"上海自来水来自海上"）、双关（谐音双关"道是无晴却有晴"）。

## 五、规范化与推广

### 5.1 普通话与方言

普通话：以北京语音为标准音、以北方方言为基础方言、以典范的现代白话文著作为语法规范。**国家通用语言文字法**（2000）确立普通话法律地位。推广普通话同时保护方言——方言是地域文化载体。

### 5.2 汉字规范

简化字（《简化字总表》）为规范汉字。**汉语拼音方案**（1958）是汉字注音工具和国际交流辅助手段。规范书写：笔画规范、笔顺规范、字形规范、标点规范。

## 六、汉英对比

| 维度 | 汉语特点 | 英语特点 |
|------|----------|----------|
| 语音 | 声调语言——4个声调区别意义 | 重音语言——词重音和句重音 |
| 文字 | 意音文字，形音义三位一体 | 拼音文字，字母记录语音 |
| 词汇 | 双音节为主，无形态变化 | 多音节为主，屈折变化（-ed/-s/-ing） |
| 语法 | 主题突出，重意合（隐性连接） | 主语突出，重形合（连接词显性） |
| 语序 | SVO，定语必在中心语前 | SVO，定语可在前后 |
| 量词 | 有丰富量词（一张、一头、一把） | 有冠词无量词，可数/不可数 |
| 体范畴 | "了/着/过"表体，无时态 | 时态丰富（过去/现在/将来） |
