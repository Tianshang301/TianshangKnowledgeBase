# 语言知识与古诗文

## 一、文言文基础知识

### 1.1 常见文言虚词

| 虚词 | 主要用法 | 含义 | 示例 |
|------|----------|------|------|
| **之** | 代词/助词"的"/宾语前置标志/取消独立性 | "他/她/它"；"的"；无实义 | "叫之"（他）；"赤壁之游"（的）；"何陋之有"（宾前标志）；"师道之不传也久矣"（取消独立性） |
| **其** | 代词/语气副词/连词 | "他的/它的"；"难道/大概/还是" | "其真无马邪"（难道）；"其真不知马也"（大概）；"其业有不精"（如果） |
| **而** | 连词 | 并列/转折/承接/递进/修饰 | "学而时习之"（并列）；"人不知而不愠"（转折）；"拔剑切而啖之"（承接） |
| **以** | 介词/连词 | "用/凭借/因为/来" | "以刀劈狼首"（用）；"以其境过清"（因为）；"以光先帝遗德"（来） |
| **于** | 介词 | "在/从/向/对于/比" | "青出于蓝"（从）；"苛政猛于虎"（比）；"告知于帝"（向） |
| **乃** | 副词/代词 | "于是/才/竟然/是/你" | "乃大惊"（于是）；"乃不知有汉"（竟然）；"是乃仁术也"（是） |
| **为** | 介词/动词 | "被/替/为了/做/成为" | "温故而知新可以为师矣"（成为）；"为天下笑"（被）；"为人谋而不忠乎"（替） |
| **之、其、而、以、于、乃、为** | **高频虚词七字诀** | 掌握这七个虚词可解决80%的文言虚词问题 | |

### 1.2 古今异义实词

很多文言实词在古代和现代的意义发生了显著变化，下面是高频词：

| 实词 | 古义 | 今义 | 示例 |
|------|------|------|------|
| **走** | 跑 | 步行 | "走送之"（跑着送还）；"兔走触株"（跑） |
| **去** | 离开 | 前往 | "去国怀乡"（离开都城）；"我从去年辞帝京"（过去的） |
| **汤** | 热水 | 菜汤 | "媵人持汤沃灌"（热水）；"赴汤蹈火"（沸水） |
| **坐** | 因为/犯罪 | 坐下 | "停车坐爱枫林晚"（因为）；"坐法去官"（因……犯罪） |
| **或** | 有的人/有时 | 或者（选择） | "或以为死，或以为亡"（有的人）；"或遇其叱咄"（有时） |
| **信** | 信任/的确/信使 | 书信/相信 | "信芳"（的确）；"信臣精卒"（忠诚的） |
| **爱** | 吝惜/舍不得 | 喜爱 | "齐国虽褊小，吾何爱一牛"（吝惜）；"百姓皆以王为爱也"（吝惜） |
| **池** | 护城河 | 池塘 | "城非不高也，池非不深也"（护城河） |
| **卑鄙** | 身份低微 | 品行恶劣 | "先帝不以臣卑鄙"（地位低微见识浅陋） |
| **交通** | 交错相通 | 交通运输 | "阡陌交通，鸡犬相闻"（交错相通） |
| **妻子** | 妻子和子女 | 妻子（配偶） | "率妻子邑人来此绝境"（妻子和儿女） |
| **无论** | 不必说 | 不管 | "乃不知有汉，无论魏晋"（更不必说） |

### 1.3 特殊句式

**宾语前置**：
- 疑问句：疑问代词（谁、何、安）作宾语时前置——"大王来何操？"（操何）
- 否定句：代词宾语前置——"不患人之不己知"（不知己）
- "是/之"标志：——"何陋之有"（有何陋）；"唯命是从"（从命）
- 介宾前置：——"何以战"（以何）

**状语后置（介词结构后置）**：
- "于"字结构——"战于长勺"（于长勺战）；"青取之于蓝而青于蓝"
- "以"字结构——"饰以篆文山龟鸟兽之形"（以篆文……之形饰）

**定语后置**：
- "之……者"结构——"马之千里者"（千里马）
- 中心词+"之"+定语——"居庙堂之高则忧其民"（高之庙堂）
- 中心词+"者"——"求人可使报秦者"（可使报秦之人）

**被动句**：
- "为"字句——"为天下笑"（被天下人耻笑）
- "见"字句——"见欺于王"（被王欺骗）
- "被"字句——"被戮于前"
- 无标志被动——"王之蔽甚矣"（被蒙蔽）

**判断句**：
- "……者，……也"——"陈胜者，阳城人也"
- "……，……也"——"此诚危急存亡之秋也"
- "乃/即/则/皆/非"——"当立者乃公子扶苏"

**省略句**：
- 省略主语——"（村人）见渔人，乃大惊"
- 省略谓语——"一鼓作气，再（鼓）而衰"
- 省略宾语——"尉剑挺，广起，夺（剑）而杀尉"
- 省略介词——"将军战（于）河北"

### 1.4 词类活用

| 活用类型 | 说明 | 示例 |
|----------|------|------|
| **名词作动词** | 名→动 | "沛公军霸上"（驻军）；"左右欲刃相如"（杀） |
| **名词作状语** | 名→状语 | "吾得兄事之"（像对待兄长一样）；"箕畚运于渤海之尾"（用箕畚） |
| **形容词作名词** | 形→名 | "小学而大遗"（小的问题）；"温故而知新"（旧知识） |
| **形容词作动词** | 形→动 | "亲贤臣，远小人"（亲近/远离） |
| **使动用法** | 让…… | "项伯杀人臣活之"（使之活）；"必先苦其心志"（使其痛苦） |
| **意动用法** | 认为…… | "孔子登东山而小鲁"（认为鲁国小）；"渔人甚异之"（认为奇怪） |

## 二、唐宋诗词名篇鉴赏

### 2.1 盛唐气象

**王维——空灵山水**（诗中有画，画中有诗）

| 诗作 | 名句 | 赏析要点 |
|------|------|----------|
| 《山居秋暝》 | **"空山新雨后，天气晚来秋。明月松间照，清泉石上流"** | 以动衬静，色光交融——雨后空山、松间明月、石上清泉，宛如一幅水墨画 |

| 《蜀道难》 | "噫吁嚱，危乎高哉！蜀道之难，难于上青天" | 极度夸张，想象飞腾 |
| 《静夜思》 | "举头望明月，低头思故乡" | 最朴素的语言写最深沉的情感，明白如话却千古传诵 |
| 《望庐山瀑布》 | "飞流直下三千尺，疑是银河落九天" | 极度夸张中见奇特想象 |

| 诗作 | 名句 | 赏析要点 |
|------|------|----------|
| 《茅屋为秋风所破歌》 | **"安得广厦千万间，大庇天下寒士俱欢颜"** | 推己及人，由个人苦难想到天下苍生——仁者之心 |

| 《望岳》 | "会当凌绝顶，一览众山小" | 青年杜甫的豪情壮志，登泰山而小天下 |

### 2.4 宋词双峰

**豪放派——苏轼与辛弃疾**

**苏轼**（1037-1101）：
- 《念奴娇·赤壁怀古》：**"大江东去，浪淘尽，千古风流人物"**——开篇壮阔，"人生如梦，一尊还酹江月"——旷达中有悲慨
- 《水调歌头》：**"但愿人长久，千里共婵娟"**——手足之情升华为普世祝福
- 《定风波》：**"一蓑烟雨任平生"**——"也无风雨也无晴"——面对困境的超然态度
- 《江城子·密州出猎》："会挽雕弓如满月，西北望，射天狼"——豪情壮志
- 《江城子·乙卯正月二十日夜记梦》："十年生死两茫茫，不思量，自难忘"——婉约深情

**辛弃疾**（1140-1207）：
- 《永遇乐·京口北固亭怀古》：**"想当年金戈铁马，气吞万里如虎"**——壮怀激烈
- 《青玉案·元夕》：**"众里寻他千百度，蓦然回首，那人却在灯火阑珊处"**——王国维喻为治学第三境界
- 《破阵子》："醉里挑灯看剑，梦回吹角连营"——报国无门的悲愤
- 《丑奴儿》："而今识尽愁滋味，欲说还休。欲说还休，却道天凉好个秋"——愁到极处反无言

## 三、阅读与鉴赏方法

### 3.1 小说阅读

**小说三要素**：

| 要素 | 内涵 | 分析角度 |
|------|------|----------|
| **人物** | 作品中塑造的人物形象 | 外貌/语言/动作/心理/神态描写→人物性格；直接描写与间接描写 |
| **情节** | 事件的发展过程 | 开端→发展→高潮→结局；线索（单线/双线/明暗线） |
| **环境** | 人物活动的背景 | 社会环境（时代背景、人际关系）；自然环境（氛围烘托、象征） |
| **主题** | 作品的核心思想 | 通过人物命运、情节冲突、环境描写揭示 |

**高频题型**：
- 分析人物形象——"从哪些方面刻画人物？"→ 手法分析+性格概括+作用
- 情节作用——"某段情节有何作用？"→ 内容上（塑造人物/推动情节）+ 结构上（铺垫/照应/转折）
- 环境描写分析——"环境描写的作用？"→ 交代背景/烘托气氛/推动情节/暗示主题

### 3.2 散文阅读

**核心特征**：形散神不散
- **形散**：取材广泛，结构自由，手法多样——古今中外、天文地理均可入文
- **神聚**：始终围绕一个核心——情感、哲思或主题

**散文分类**：

| 类型 | 特点 | 名篇示例 |
|------|------|----------|
| **叙事散文** | 以人事为核心，以小见大 | 朱自清《背影》——父亲爬月台买橘子的细节 |
| **抒情散文** | 以情感为主线，情景交融 | 朱自清《荷塘月色》——"这几天心里颇不宁静" |
| **哲理散文** | 以思辨为核心，融理于情 | 史铁生《我与地坛》——生命的思考 |
| **写景散文** | 以景物为对象，融情于景 | 老舍《济南的冬天》 |

**鉴赏要点**：
1. 品味语言——修辞运用、句式特点、用词精妙
2. 把握线索——时间线/空间线/情感线/事物线
3. 体悟情感——直接抒情与间接抒情（借景抒情/托物言志）

### 3.3 论述文阅读

**论述文三要素**：

| 要素 | 定义 | 识别方法 |
|------|------|----------|
| **论点** | 作者的核心观点 | 通常在开头（中心论点）或每段首句（分论点） |
| **论据** | 支撑论点的材料 | 事实论据（事例/数据）和道理论据（名言/原理） |
| **论证** | 用论据证明论点的过程 | 举例论证/对比论证/引用论证/比喻论证/因果论证 |

**论证方法辨析**：

| 方法 | 特点 | 标志 |
|------|------|------|
| **举例论证** | 具体事例证明 | "比如""例如""如" |
| **道理论证** | 名人名言/公认原理 | "古人云""据统计""科学研究表明" |
| **对比论证** | 正反对照 | "相反""然而""与之相反" |
| **比喻论证** | 形象比喻说明抽象 | "如同""像""犹如" |
| **因果论证** | 分析因果 | "之所以……是因为……""因此" |
