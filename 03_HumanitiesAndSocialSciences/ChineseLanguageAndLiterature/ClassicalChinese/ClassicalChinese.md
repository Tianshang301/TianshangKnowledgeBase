# 古代汉语

## 一、文言虚词

### 1.1 核心虚词

古代汉语虚词数量虽少但使用频率极高，掌握虚词是读懂古文的关键。清代刘淇《助字辨略》、王引之《经传释词》为虚词研究的经典著作。

| 虚词 | 词性 | 核心功能 | 用法举例 | 现代翻译 |
|------|------|----------|----------|----------|
| **之** | 代词/助词 | 代词/助词"的"/取消独立性/宾语前置标志 | "呼而与之" | 他 |
| | | | "赤壁之游" | 的 |
| | | | "孤之有孔明犹鱼之有水也" | 取消独立性（不译） |
| | | | "何陋之有" | 宾语前置标志（不译） |
| **其** | 代词/副词 | 代词（他的/它的）/语气词（难道/大概/还是） | "吾知其难也" | 它 |
| | | | "其真无马邪" | 难道（反问） |
| | | | "其恕乎" | 大概（推测） |
| | | | "汝其勿悲" | 还是（祈使） |
| **者** | 助词 | ……的人/物/事；判断句提顿 | "读书者" | ……的人 |
| | | | "陈胜者，阳城人也" | 提顿标志 |
| **也** | 语气词 | 判断/肯定/疑问/感叹 | "此诚危急存亡之秋也" | 判断语气 |
| | | | "其可怪也欤" | 感叹语气 |
| **而** | 连词 | 并列/承接/转折/修饰/因果 | "学而时习之" | 并且（并列） |
| | | | "人不知而不愠" | 却（转折） |
| | | | "吾尝终日而思矣" | 地（修饰） |
| | | | "因人之力而敝之" | 却/反而 |
| **以** | 介词/连词 | 用/凭借/因为/用来/而 | "以刀劈狼首" | 用 |
| | | | "以其境过清" | 因为 |
| | | | "以塞忠谏之路" | 以致（结果） |
| **于** | 介词 | 在/从/向/对/比/被 | "青出于蓝" | 从 |
| | | | "苛政猛于虎" | 比 |
| | | | "战于长勺" | 在 |
| | | | "困于酒" | 被（被动） |

### 1.2 虚词辨析要点

"之"在"取消句子独立性"用法中位于主谓之间（"孤之有孔明"），不译。"其"作语气副词时，根据语境可表推测（大概）、反问（难道）或祈使（还是）。"者"字短语相当于现代汉语的"的字结构"。"而"表因果关系时相当于"因而"。"以"表目的时译作"用来"，表结果时译作"以致"。注意"以"后的宾语可省略（"必以分人"=以之分人）。

## 二、实词与词义

### 2.1 古今异义

| 古词 | 古义 | 今义 | 例证 |
|------|------|------|------|
| 走 | 跑 | 步行 | "走送之"（跑着送还） |
| 去 | 离开 | 前往 | "去国怀乡"（离开国都） |
| 汤 | 热水 | 菜汤 | "媵人持汤沃灌"（热水） |
| 坐 | 因为 | 坐下 | "停车坐爱枫林晚"（因为） |
| 爱 | 吝惜 | 喜爱 | "齐国虽褊小，吾何爱一牛"（吝惜） |
| 恨 | 遗憾 | 仇恨 | "未尝不叹息痛恨于桓灵也"（遗憾） |
| 涕 | 眼泪 | 鼻涕 | "涕泗滂沱"（眼泪） |
| 信 | 的确 | 相信/信件 | "信芳"（的确美好） |
| 池 | 护城河 | 池塘 | "城非不高也，池非不深也" |
| 臭 | 气味（读xiù） | 难闻气味 | "其臭如兰"（气味） |

### 2.2 词类活用

| 活用类型 | 说明 | 例句 | 翻译 |
|----------|------|------|------|
| **名词→动词** | 名词用作谓语 | "沛公军霸上" | 驻军 |
| | | "左右欲刃相如" | 用刀杀 |
| **形容词→动词** | 形容词带宾语 | "亲贤臣远小人" | 亲近、疏远 |
| **使动用法** | 使宾语怎么样 | "活我" | 使我活（救活我） |
| | | "宁许以负秦曲" | 使秦承担理亏 |
| **意动用法** | 认为宾语怎么样 | "孔子登山而小鲁" | 认为鲁国小 |
| | | "吾妻之美我者" | 认为我美 |
| **名词→状语** | 名词修饰动词 | "犬坐于前" | 像狗一样 |
| | | "箕畚运于渤海之尾" | 用箕畚 |
| **为动用法** | 为宾语而做 | "死国可乎" | 为国而死 |

## 三、特殊句式

### 3.1 判断句

| 形式 | 例句 | 说明 |
|------|------|------|
| ……者，……也 | "陈胜者，阳城人也" | 最典型格式 |
| ……，……也 | "此诚危急存亡之秋也" | 省"者" |
| 乃/即/则/非 | "是乃仁术也""此则岳阳楼之大观也" | 副词表判断 |
| 是（系词） | "问今是何世" | 汉以后才发展 |

### 3.2 被动句

| 形式 | 例句 | 说明 |
|------|------|------|
| 于+施事 | "困于酒" | 被…… |
| 见+动词 | "信而见疑" | 被 |
| 见+动词+于+施事 | "见欺于王" | 被（完整式） |
| 为+施事+动词 | "为天下笑" | 被…… |
| 为+所+动词 | "不者若属皆且为所虏" | 被 |
| 无标志（概念被动） | "蔓草犹不可除" | 凭语义理解 |

### 3.3 倒装句

| 类型 | 条件 | 例句 | 正常语序 |
|------|------|------|----------|
| **宾语前置** | 否定句代词宾语 | "时不我待" | 时不待我 |
| | 疑问代词宾语 | "吾谁欺" | 吾欺谁 |
| | 之/是标志 | "何陋之有""唯利是图" | 有何陋/唯图利 |
| **状语后置** | 介宾短语后置 | "战于长勺" | 于长勺战 |
| **定语后置** | 之……者 | "马之千里者" | 千里马 |

## 四、音韵与格律

### 4.1 平仄

中古汉语有平（阴平、阳平）、上、去、入四声。平仄规则：平声为"平"，上去入为"仄"。

**近体诗平仄格式**：五言基本句式——"仄仄平平仄，平平仄仄平。平平平仄仄，仄仄仄平平"。七言在五言前加两字（"平"前加"仄仄"，"仄"前加"平平"）。

**拗救**：如果某字该平而用仄（拗），须在适当位置用该仄而平（救）。如"野火烧不尽"四连仄，下句"春风吹又生"首字用平相救。

### 4.2 对仗

律诗中间两联必须对仗。要求：词性相对、结构相同、平仄相反、避免重复字。

| 对仗类型 | 要求 | 示例 |
|----------|------|------|
| 工对 | 同类词相对（天文对天文） | "明月松间照，清泉石上流" |
| 宽对 | 邻近词类相对 | "文章千古事，得失寸心知" |
| 借对 | 借义或借音 | "酒债寻常行处有，人生七十古来稀"（寻=八尺→对七十） |

### 4.3 押韵

近体诗一般押平声韵，一韵到底，不换韵。首句可押可不押（七言首句多押韵，五言首句多不押）。用韵以《平水韵》106韵为准，现代人作诗常用《中华新韵》。

## 五、训诂与校勘

### 5.1 训诂方法

| 方法 | 说明 | 示例 |
|------|------|------|
| **形训** | 通过字形分析字义 | "日"象太阳之形；"武"止戈为武 |
| **音训** | 通过读音推求字源 | "政者，正也"；"仁者，人也" |
| **义训** | 用已知词义解释未知词义 | "元，始也"；"及，到也" |
| **互文见义** | 根据对文推求词义 | "雷霆不作，风雨不兴"中"作"即"兴" |
| **因声求义** | 突破字形的音义关系 | 利用通假字找到本字 |

### 5.2 校勘方法

陈垣《校勘学释例》总结四法：**对校法**（据善本对照异同）、**本校法**（本书前后互证，如用相同句式）、**他校法**（引他书的引文校正）、**理校法**（据文理、声韵、史实推断）。

### 5.3 古书注释术语

| 术语 | 含义 | 示例 |
|------|------|------|
| 传 | 传述经义 | 《春秋左氏传》 |
| 注 | 解释原文 | 《诗经》毛亨传、郑玄笺 |
| 疏 | 解释注文 | 《十三经注疏》 |
| 笺 | 对传注的补充说明 | 郑玄《毛诗笺》 |
| 正义 | 对注疏的进一步解释 | 孔颖达《五经正义》 |
| 曰/为/谓之 | 释义用术语 | "沂，水名也" |

## 七、阅读策略

1. **通读了解大意**：不必逐字逐句翻译，先把握整体意思
2. **标注疑难字词**：圈出不认识的实词和虚词
3. **利用注释字典**：查阅《古汉语常用字字典》《王力古汉语字典》
4. **分析语法结构**：确定句子成分，判断特殊句式
5. **反复诵读**：古汉语讲究语感，熟读自然理解——"读书百遍，其义自见"
6. **参看翻译**：对照现代译文检查理解是否正确
7. **积累背诵**：熟读成诵，培养文言语感，积累实词虚词用法

## 关键概念辨析
概念定义的清晰度直接影响研究的质量。
以下是该领域中若干容易混淆的概念。

**概念一与概念二的区分**：
前者侧重于外在的形式特征。
后者关注内在的运作机制。
两者在实际分析中往往需要结合使用。

**微观与宏观层面的联系**：
微观现象是宏观结构的基础。
宏观结构又约束微观行为。
理解两者的相互作用是社会分析的核心。

**静态分析与动态分析**：
静态分析关注某一时点的截面特征。
动态分析关注过程和变化的轨迹。
两种视角互补而非替代。
