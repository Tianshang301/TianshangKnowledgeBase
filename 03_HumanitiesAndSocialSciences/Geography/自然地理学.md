# 自然地理学

## 一、气候系统

### 1.1 气候系统的组成

气候系统由五大圈层相互作用构成：大气圈（atmosphere）、水圈（hydrosphere）、岩石圈（lithosphere）、生物圈（biosphere）和冰冻圈（cryosphere）。各圈层之间通过能量交换、物质循环和动量传递相互耦合，形成复杂的气候系统。

| 圈层 | 主要成分 | 气候作用 |
|------|---------|---------|
| 大气圈 | N₂, O₂, CO₂, H₂O | 温室效应、大气环流 |
| 水圈 | 海洋、湖泊、河流 | 储存热量、水汽来源 |
| 岩石圈 | 陆地表面、山脉 | 地形强迫、反照率 |
| 生物圈 | 植被、动物、微生物 | 碳循环、蒸腾作用 |
| 冰冻圈 | 冰盖、冰川、海冰 | 高反照率、冷源 |

### 1.2 气候带

根据气温和降水的差异，全球可划分为以下气候带：

| 气候带 | 气候类型 | 特征 |
|--------|---------|------|
| 热带 | 赤道雨林气候 | 全年高温多雨，年均温>25°C |
| 热带 | 热带草原气候 | 干湿季分明 |
| 热带 | 热带沙漠气候 | 全年干旱少雨 |
| 温带 | 地中海气候 | 夏季炎热干燥，冬季温和多雨 |
| 温带 | 温带海洋性气候 | 全年温和湿润 |
| 温带 | 温带大陆性气候 | 冬冷夏热，降水较少 |
| 温带 | 亚热带季风气候 | 夏季高温多雨，冬季温和少雨 |
| 寒带 | 极地苔原气候 | 夏季短暂，冬季严寒 |
| 寒带 | 冰原气候 | 全年严寒，冰雪覆盖 |
| 高山 | 高山气候 | 垂直分异明显 |

### 1.3 大气环流

大气环流是地球表面热量分布不均和地转偏向力共同作用的结果。三圈环流模型将每半球划分为三个环流圈：

- **Hadley环流**（低纬环流，0°—30°）：赤道上升气流→高空气流向极地→副热带下沉→地面信风
- **Ferrel环流**（中纬环流，30°—60°）：副热带上升→高空气流向极地→极地锋面下沉→地面西风
- **Polar环流**（高纬环流，60°—90°）：极地上升→高空气流向赤道→副极地下沉→地面极地东风

地面风带：信风带（0°—30°，东北信风/东南信风）、西风带（30°—60°）、极地东风带（60°—90°）。

### 1.4 气候因子

- **纬度**：决定太阳辐射总量的基本因子
- **海陆分布**：影响降水和气温年较差（大陆性vs海洋性）
- **洋流**：暖流增温增湿（湾流、黑潮），寒流降温减湿（秘鲁寒流、本格拉寒流）
- **地形**：迎风坡多地形雨，背风坡雨影区
- **大气环流**：输送热量和水汽

## 二、地貌学

### 2.1 内力作用与板块构造

板块构造理论将岩石圈划分为六大板块：太平洋板块、亚欧板块、非洲板块、美洲板块、印度-澳大利亚板块和南极板块。板块边界类型：

| 边界类型 | 运动方式 | 地貌 | 实例 |
|---------|---------|------|------|
| 离散型 | 板块分离 | 洋中脊、裂谷 | 大西洋中脊、东非裂谷 |
| 汇聚型 | 板块碰撞/俯冲 | 山脉、海沟、岛弧 | 喜马拉雅山、马里亚纳海沟 |
| 转换型 | 水平错动 | 断层 | 圣安德烈斯断层 |

**火山**：分布在板块边界和热点地区。环太平洋火山带（"火环"）是全球最主要的火山地震带。

**地震**：由板块运动积累的应力突然释放引起。震级（magnitude）表示释放能量大小，烈度（intensity）表示地表破坏程度。

### 2.2 外力作用地貌

**流水地貌**：
- 上游：V形谷、瀑布、急流（以侵蚀为主）
- 中游：曲流、河漫滩（以搬运为主）
- 下游：三角洲、冲积扇（以堆积为主）
- 特殊：冲积扇（山前）、三角洲（河口）

**冰川地貌**：
- 侵蚀：U形谷、角峰（horn）、冰斗（cirque）、刃脊（arête）
- 堆积：冰碛（moraine）、鼓丘（drumlin）

**风成地貌**：
- 侵蚀：雅丹地貌（yardang）、风蚀蘑菇
- 堆积：沙丘（新月形沙丘、抛物线沙丘）、黄土堆积

**喀斯特地貌**：
- 地表：溶沟、石芽、峰林、天坑
- 地下：溶洞、钟乳石、石笋、石柱、地下河

### 2.3 岩石循环

火成岩（岩浆冷却）→风化剥蚀→沉积物→成岩作用→沉积岩→变质作用→变质岩→重熔→火成岩。三大岩类在自然界中不断循环转化。

## 三、水文学

### 3.1 水循环

全球水循环过程：蒸发（海洋/陆地）→水汽输送→凝结→降水（大气降水）→地表径流→下渗→地下径流→回归海洋。水循环的驱动能量来自太阳辐射和地球重力。

### 3.2 河流系统

河流的三个水文阶段：
| 阶段 | 主导作用 | 地形特征 |
|------|---------|---------|
| 上游 | 侵蚀 | V形谷、急流、瀑布 |
| 中游 | 搬运 | 曲流、河漫滩 |
| 下游 | 堆积 | 三角洲、河口 |

### 3.3 地下水

- **饱水带**：地下水充填于孔隙之中
- **潜水面**：饱水带的上界面
- **含水层**（aquifer）：透水性好的岩层
- **隔水层**（aquitard）：透水性差的岩层
- **潜水**：第一个隔水层之上的地下水，无压
- **承压水**：两个隔水层之间的地下水，有压

## 四、土壤地理学

### 4.1 土壤剖面

典型土壤剖面自上而下：
| 层次 | 名称 | 特征 |
|------|------|------|
| O层 | 有机质层 | 枯枝落叶、腐殖质 |
| A层 | 淋溶层 | 表土，富含有机质 |
| E层 | 过渡层 | 硅酸盐淋失，颜色浅 |
| B层 | 淀积层 | 黏粒、铁锰胶膜聚集 |
| C层 | 母质层 | 风化碎屑物 |
| R层 | 基岩 | 未风化岩石 |

### 4.2 主要土壤类型

- **淋溶土**（Alfisol）：湿润气候下，黏粒向下淋溶
- **钙积土**（Mollisol）：半干旱草原，钙积层
- **荒漠土**（Aridisol）：干旱区，有机质少
- **冰沼土**（Gelisol）：寒带，永冻层
- **氧化土**（Oxisol）：热带，深度风化

## 五、生物地理学

### 5.1 主要生物群落

热带雨林→热带草原→温带森林→温带草原→泰加林（针叶林）→冻原，沿纬度梯度呈现带状分布。

### 5.2 生物多样性梯度

生物多样性从赤道向两极递减：热带雨林物种最丰富（生物多样性热点），极地物种最贫乏。

### 5.3 岛屿生物地理学

MacArthur & Wilson（1967）提出岛屿生物地理学理论：岛屿物种数由迁入率和灭绝率的动态平衡决定。**面积效应**：岛屿面积越大，物种数越多。**距离效应**：岛屿距大陆越近，物种数越多。公式：$S = cA^z$，其中$S$为物种数，$A$为面积，$c$和$z$为常数。

## 六、气候变化

### 6.1 自然气候变化

- **米兰科维奇周期**：地球轨道参数（偏心率、地轴倾斜、岁差）的变化导致太阳辐射量周期性变化，驱动冰期—间冰期循环
- **太阳活动**：太阳黑子周期（11年、22年）影响地球气候
- **火山活动**：大规模火山喷发将火山灰和气溶胶送入平流层，遮蔽太阳辐射，导致全球降温

### 6.2 人为气候变化

工业革命以来，人类活动（化石燃料燃烧、土地利用变化、工业过程）导致大气温室气体浓度显著上升。CO$_2$浓度从工业革命前的280ppm上升到2023年的420ppm以上。全球平均气温已上升约1.2°C。气候变化的影响包括：海平面上升、极端天气事件增多、冰川退缩、生态系统退化。

### 6.3 气候反馈机制

| 反馈类型 | 过程 | 效应 |
|---------|------|------|
| 冰反照率正反馈 | 冰雪融化→反照率下降→吸收更多热量→进一步融化 | 加速增温 |
| 水汽正反馈 | 温度升高→大气持水能力增强→温室效应增强 | 加速增温 |
| 云反馈 | 云量变化影响辐射收支 | 可正可负 |
| 碳循环反馈 | 温度升高→土壤有机质分解加速→CO$_2$释放增加 | 加速增温 |

## 七、地球系统科学

### 7.1 地球系统的圈层相互作用

地球系统科学将地球视为一个由大气圈、水圈、岩石圈、生物圈和冰冻圈组成的复杂系统。各圈层通过以下过程相互作用：
- **生物地球化学循环**：碳循环、氮循环、磷循环、硫循环
- **能量收支**：太阳辐射输入—反射—吸收—再辐射的平衡
- **全球对流层-平流层交换**：影响大气化学成分和气候

### 7.2 碳循环

碳循环是地球系统中最关键的生物地球化学循环之一。主要碳库包括：

| 碳库 | 碳储量（GtC） | 周转时间 |
|------|-------------|---------|
| 大气 | 约870 | 数年 |
| 陆地植被 | 约450 | 数十年 |
| 土壤有机质 | 约1500 | 数十年至数百年 |
| 海洋 | 约38000 | 百年至千年 |
| 化石燃料 | 约5000 | 不可再生 |

### 7.3 人类世

"人类世"（Anthropocene）是一个提议中的地质年代概念，指人类活动成为影响地球系统的主导因素的时代。人类世的标志性特征包括：核试验残留物、塑料污染、CO$_2$浓度升高、物种灭绝加速、地表改变（城市扩张、农业用地）。目前人类世尚未被国际地层委员会正式批准为地质年代单位。

## 八、自然灾害地理

### 8.1 主要自然灾害类型

| 类型 | 成因 | 分布特征 |
|------|------|---------|
| 地震 | 板块运动、断层活动 | 板块边界带（环太平洋带、地中海-喜马拉雅带） |
| 火山喷发 | 岩浆活动 | 板块边界和热点 |
| 台风/飓风 | 热带海洋上的低压系统 | 热带和副热带沿海 |
| 洪涝 | 强降水、河流泛滥 | 季风区、沿海低地 |
| 干旱 | 降水持续偏少 | 半干旱和干旱区 |
| 滑坡/泥石流 | 地形陡峭+强降水 | 山区 |

### 8.2 自然灾害风险评估

灾害风险 = 致灾因子危险性（hazard）× 暴露度（exposure）× 脆弱性（vulnerability）。降低灾害风险的途径包括：灾害监测预警系统建设、工程防护措施（堤坝、加固）、土地利用规划、防灾减灾教育和社区应急能力建设。

### 8.3 地理学在防灾减灾中的应用

地理学为防灾减灾提供关键支撑：利用GIS进行灾害风险评估与制图、利用遥感进行灾情快速评估、利用空间分析确定避难所选址、利用数字高程模型（DEM）进行洪水淹没模拟。地理信息系统在灾害管理的四个阶段（减灾、备灾、响应、恢复）中均发挥重要作用。
