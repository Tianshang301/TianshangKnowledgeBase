# 世界通史

## 一、古代文明

### 1.1 两河流域

幼发拉底河与底格里斯河之间的新月沃地——文明的摇篮。

| 文明 | 时期 | 成就 |
|------|------|------|
| 苏美尔 | 前3500—前2000 | 楔形文字、城市国家、轮子、六十进制 |
| 阿卡德 | 前2334—前2154 | 萨尔贡一世建第一个帝国 |
| 古巴比伦 | 前1894—前1595 | 汉谟拉比法典282条 |
| 亚述 | 前1365—前609 | 军事帝国、尼尼微图书馆2.2万泥板 |
| 新巴比伦 | 前626—前539 | 空中花园、巴别塔 |

《吉尔伽美什史诗》——最早文学巨著。

### 1.2 古埃及

尼罗河的赠礼。约前3100年美尼斯统一。

**古王国**（约前2686—前2181）：胡夫金字塔（原高146.6米、230万块2.5吨石块）、狮身人面像。法老神王合一。木乃伊。

**中王国**（约前2055—前1650）：阿蒙神崇拜。**新王国**（约前1550—前1069）：图特摩斯三世远征、拉美西斯二世卡迭石战役（前1274）、阿肯那顿宗教改革。图坦卡蒙墓（1922年卡特发现黄金面具）。象形文字（圣书体→僧侣体→世俗体）。1822年商博良破译罗塞塔石碑。

### 1.3 古印度

印度河流域文明（前2600—前1900）：哈拉帕摩亨佐-达罗网格状街道排水系统。印章文字未破译。

吠陀时代（前1500—前600）：雅利安人种姓制度——婆罗门刹帝利吠舍首陀罗不可接触者。婆罗门教梵天毗湿奴湿婆。列国：佛教——释迦牟尼四谛八正道十二因缘。耆那教。孔雀王朝——阿育王弘扬佛教。

### 1.4 古希腊

爱琴文明：米诺斯（克诺索斯迷宫）、迈锡尼（狮子门）。特洛伊战争。

城邦时代：斯巴达（军事寡头）vs 雅典（民主）。梭伦→克里斯提尼（陶片放逐法）→伯里克利。希波战争（马拉松温泉关萨拉米斯）。伯罗奔尼撒战争。

文化：苏格拉底柏拉图亚里士多德。三大悲剧。希罗多德修昔底德。欧几里得阿基米德。帕特农。奥运会（前776）。

亚历山大东征希腊化时代。

### 1.5 古罗马

**共和国**（前509—前27）：元老院执政官保民官。十二铜表法（前451）。布匿战争灭迦太基。凯撒渡卢比孔河被刺。屋大维亚克兴海战。

**帝国**（前27—476）：奥古斯都元首制。Pax Romana 五贤帝。基督教：耶稣→保罗→米兰敕令（313）→国教（380）。395年东西分裂。476年西罗马亡。

| 成就 | 内容 |
|------|------|
| 罗马法 | 十二铜表→万民法→自然法→查士丁尼《民法大全》 |
| 工程 | 引水道11条、道路8万公里、斗兽场、万神殿穹顶43米 |
| 文学 | 维吉尔李维塔西佗奥维德西塞罗。拉丁语→罗曼语族 |

## 二、中世纪（约476—1500）

### 2.1 西欧封建

封君封臣采邑制。庄园经济三圃制。教皇与皇帝之争（卡诺莎1077）。十字军东征1096—1291八次。黑死病人口减1/3。百年战争1337—1453圣女贞德。阿奎那经院哲学。

### 2.2 拜占庭

东罗马千年。查士丁尼（527—565）民法大全圣索菲亚大教堂。1054年东西教会大分裂。1204年十字军攻陷。1453年奥斯曼穆罕默德二世攻陷君士坦丁堡。

### 2.3 阿拉伯帝国

穆罕默德610年创伊斯兰教。622年徙志纪元。四大哈里发扩张。什叶派 vs 逊尼派。倭马亚（大马士革）→阿拔斯（巴格达）。

伊斯兰黄金时代（750—1258）：百年翻译运动。贡献——阿拉伯数字（印度起源）、代数学花拉子密（algorithm 词源）、伊本西那《医典》（500年教材）、伊本路世德评亚里士多德、《一千零一夜》。1258年蒙古灭巴格达。

### 2.4 文艺复兴

意大利佛罗伦萨美第奇赞助。人文主义以人为中心。但丁《神曲》、彼特拉克十四行诗、薄伽丘《十日谈》。达芬奇（蒙娜丽莎最后的晚餐）、米开朗基罗（大卫像西斯廷天顶画）、拉斐尔（圣母像雅典学院）。马基雅维利《君主论》。

传播：莎士比亚（英）、塞万提斯（西）堂吉诃德、拉伯雷（法）、伊拉斯谟（尼德兰）、丢勒（德）。

## 三、近代时期（1500—1900）

### 3.1 地理大发现

| 航海家 | 时间 | 成就 |
|--------|------|------|
| 迪亚士 | 1488 | 绕过好望角 |
| 哥伦布 | 1492 | 发现美洲（西班牙资助） |
| 达·伽马 | 1498 | 到达印度卡利卡特 |
| 麦哲伦 | 1519—1522 | 首次环球航行 |
| 库克 | 1768—1779 | 太平洋探险 |

葡西率先殖民。荷兰东印度公司1602第一家股份公司。三角奴隶贸易——非洲数千万损失。

### 3.2 宗教改革

路德1517《九十五条论纲》——因信称义圣经唯一权威。路德宗（德国北欧）。加尔文预定论（瑞士法国苏格兰）。英国国教亨利八世1534《至尊法案》。三十年战争《威斯特伐利亚和约》主权国家。

### 3.3 科学革命

哥白尼日心说（1543）。布鲁诺被烧（1600）。第谷→开普勒三定律。伽利略审判。牛顿《原理》1687——万有引力三大定律微积分。

### 3.4 启蒙运动

| 思想家 | 核心观点 |
|--------|---------|
| 洛克 | 天赋人权社会契约分权 |
| 伏尔泰 | 宗教宽容开明君主 |
| 孟德斯鸠 | 三权分立 |
| 卢梭 | 人民主权社会契约 |
| 狄德罗 | 百科全书28卷 |
| 康德 | 敢于运用理性 |

亚当·斯密《国富论》（1776）——看不见的手。

### 3.5 资产阶级革命

**英国**（1640—1688）：查理一世处死→克伦威尔→光荣革命→权利法案君主立宪。**美国**（1775—1783）：独立宣言（1776杰斐逊起草）→萨拉托加→1787宪法三权分立联邦制。**法国**（1789—1799）：攻占巴士底狱→人权宣言→路易十六断头→雅各宾恐怖→拿破仑民法典。

### 3.6 工业革命

**第一次**（1760—1840英国）：瓦特蒸汽机1769、史蒂芬森火车1825。工厂城市化。**第二次**（1870—1900德美）：电力（爱迪生西门子）、内燃机（奔驰汽车1886）、福特流水线。化学钢铁垄断。

## 四、当代时期（1900至今）

### 4.1 一战（1914—1918）

萨拉热窝事件。凡尔登索姆河绞肉机。俄国十月革命。1918停战。1000万死亡。《凡尔赛和约》割地赔款。国际联盟。

### 4.2 两次大战之间

大萧条1929黑色星期四。罗斯福新政。希特勒1933纳粹。墨索里尼法西斯。日本军国主义。西班牙内战（1936—1939）。

### 4.3 二战（1939—1945）

闪电战→法国沦陷→不列颠空战→巴巴罗萨1941珍珠港美参战→斯大林格勒转折→诺曼底登陆→原子弹广岛长崎。6000万死亡。联合国成立。

### 4.4 冷战（1947—1991）

铁幕杜鲁门主义马歇尔计划。北约 vs 华约。朝鲜战争古巴导弹危机越南战争。柏林墙（1961—1989）。核竞赛太空竞赛（加加林1961阿姆斯特朗1969）。第三世界不结盟运动。1989东欧剧变1991苏联解体。

### 4.5 全球化当代

欧盟欧元。互联网。9·11反恐。2008金融危机。气候变化。新冠。多极化（中美欧俄印）。俄乌冲突（2022—）。

## 五、总结

人类文明从分散走向整体。历史有进步也有悲剧。文明交流则繁荣封闭则落后。全球挑战需要命运共同体。

## 关键概念辨析
概念定义的清晰度直接影响研究的质量。
以下是该领域中若干容易混淆的概念。

**概念一与概念二的区分**：
前者侧重于外在的形式特征。
后者关注内在的运作机制。
两者在实际分析中往往需要结合使用。

**微观与宏观层面的联系**：
微观现象是宏观结构的基础。
宏观结构又约束微观行为。
理解两者的相互作用是社会分析的核心。

**静态分析与动态分析**：
静态分析关注某一时点的截面特征。
动态分析关注过程和变化的轨迹。
两种视角互补而非替代。
