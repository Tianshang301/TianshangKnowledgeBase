# 国际关系理论

## 概述

国际关系理论是研究国家之间互动规律和国际体系运行逻辑的系统性知识框架。主要理论流派包括现实主义、自由主义、建构主义、马克思主义/国际政治经济学和英国学派。

---

## 1. 现实主义（Realism）

### 1.1 古典现实主义

汉斯·摩根索（Hans Morgenthau）1948年出版《国家间政治》，提出**政治现实主义六原则**：

| 原则 | 内容 |
|------|------|
| 第一原则 | 政治受源于人性的客观法则支配 |
| 第二原则 | 以权力界定的利益是政治的核心概念 |
| 第三原则 | 权力界定的利益范畴是普遍适用的 |
| 第四原则 | 普遍道德原则不能抽象应用于国家行为 |
| 第五原则 | 特定民族的道德抱负不等于普世道德法则 |
| 第六原则 | 政治现实主义坚持政治领域的自主性 |

核心假设：**人性本恶**，国家追求权力是自然倾向，**国家利益以权力定义**。

### 1.2 新现实主义（结构现实主义）

肯尼思·沃尔兹（Kenneth Waltz）1979年出版《国际政治理论》，将理论重点从人性转向国际体系结构。

国际体系的三要素：
- **秩序原则**：无政府状态（缺乏中央权威）
- **单元功能分化**：在无政府体系中，所有国家功能相似（自助）
- **能力分布**：体系中的权力分配决定结构类型

**极性与稳定性**：

| 体系类型 | 例子 | 稳定性评估 |
|---------|------|-----------|
| 单极 | 冷战后美国主导 | 最不稳定？ |
| 两极 | 冷战美苏 | 最稳定（清晰对抗、盟友锁定） |
| 多极 | 二战前欧洲 | 最不稳定（误判风险高） |

### 1.3 进攻性现实主义

约翰·米尔斯海默（John Mearsheimer）2001年出版《大国政治的悲剧》。

核心命题：
- 国际体系无政府性
- 大国拥有进攻性军事能力
- 他国意图无法确定
- **生存是首要目标**
- 大国是理性行为体

结论：**大国追求地区霸权**，美国是唯一的地区霸权，其他大国无法达到同等地位。大国间永久冲突的"悲剧"源于体系结构而非个别国家恶意。

### 1.4 防御性现实主义

斯蒂芬·范埃弗拉（Van Evera）、斯蒂芬·沃尔特（Stephen Walt）等人发展。

核心观点：
- 多数结构性因素倾向于**维持现状**
- 国家通常安全足够，不需要扩张
- **安全困境**是防御性现实主义的分析核心

**威胁制衡理论**（Walt, 1987）：国家制衡的是**威胁**而非仅仅权力。威胁 = 实力 × 地理邻近性 × 进攻能力 × 侵略意图。

### 1.5 现实主义关键概念

| 概念 | 提出者 | 含义 |
|------|--------|------|
| 安全困境 | Herz（1950） | 一国增强安全的行为无意中导致他国不安全，引发螺旋升级 |
| 内部制衡 | 国家自身 | 增强军备、经济增长、内部动员 |
| 外部制衡 | 国家通过联盟 | 结盟对抗共同威胁 |
| 硬制衡 | 军事手段 | 军备竞赛、军事联盟 |
| 软制衡 | 非军事手段 | 外交协调、国际制度制约 |
| 追随（Bandwagoning） | — | 弱者加入强者阵营 |
| 推卸责任（Buck-passing） | — | 将制衡责任推给他国 |
| 连锁（Chain-ganging） | Waltz | 盟国被拖入不必要的战争（如WWI机制） |

## 2. 自由主义（Liberalism）

### 2.1 民主和平论

源自康德（1795）《永久和平论》，认为**两个民主国家之间极少发生战争**。

解释机制：
- **制度约束**：民主制度的分权制衡、立法监督增加战争决策成本
- **规范/文化约束**：民主国家将国内冲突解决的妥协规范外化
- **公众监督/受众成本**：选民惩罚失败的战争决策者

争议：民主国家与非民主国家同样好战（"民主战争"如科索沃、利比亚）。

### 2.2 商业自由主义与共和自由主义

- **商业自由主义**：贸易相互依赖提高战争成本，促进和平
- **共和自由主义**：代议制政府减少好战倾向

### 2.3 新自由制度主义

罗伯特·基欧汉（Robert Keohane）1984年出版《霸权之后》。

国际制度/机制的功能：

| 功能 | 解释 |
|------|------|
| 降低交易成本 | 提供信息、规则和谈判框架 |
| 议题关联 | 将不同领域问题联系起来，扩大合作空间 |
| 提高声誉成本 | 违约损害国家声誉，影响未来合作 |
| 信息提供 | 减少信息不对称，克服"囚徒困境" |
| 监督执行 | 监督协议遵守情况 |

### 2.4 复合相互依赖

基奥恩与奈（Keohane & Nye, 1977）提出，三个特征：

| 特征 | 含义 |
|------|------|
| 多渠道联系 | 政府间、跨政府、跨国非政府联系 |
| 议题间无等级 | 军事安全不始终优先于经济、环境等议题 |
| 军事力量作用下降 | 在发达民主国家间，武力不再是有效政策工具 |

## 3. 建构主义（Constructivism）

### 3.1 亚历山大·温特的核心论点

温特（Alexander Wendt）1992年发表《无政府状态是国家造就的》，挑战现实主义关于无政府性质的假定。

**三种无政府文化**：

| 文化类型 | 角色身份 | 逻辑 | 例子 |
|---------|---------|------|------|
| 霍布斯文化 | 敌人 | 相互消灭 | 三十年战争 |
| 洛克文化 | 对手 | 竞争但承认生存权 | 现代主权体系 |
| 康德文化 | 朋友 | 非暴力解决冲突+共同安全 | 北约内部关系 |

**身份与利益的社会建构**：国家的身份（如"大国""民主国家""朋友"）不是给定的，而是在互动中建构的。利益源于身份。

### 3.2 规范理论

芬尼莫尔与辛金克（Finnemore & Sikkink）提出**规范生命周期**：

| 阶段 | 特征 | 行为体 |
|------|------|--------|
| 兴起（Emergence） | 规范倡导者推动新规范 | 规范企业家（NGOs、个人） |
| 普及（Cascade） | 规范被广泛接受 | 各国、国际组织 |
| 内化（Internalization） | 规范视为理所当然 | 专业群体 |

**规范类型**：
- **规制性规范**：规定行为标准（如禁止使用化学武器）
- **构成性规范**：定义行为体身份（如主权规范定义"国家"）

### 3.3 认知共同体

哈斯（Peter Haas）提出，认知共同体（Epistemic Communities）是由具有公认专业知识和能力的专业人士组成的网络，帮助国家界定利益和制定政策。例如：气候变化科学家的IPCC影响全球气候政策。

### 3.4 安全化理论

哥本哈根学派（Buzan, Waever, de Wilde）1998年提出。
- **安全化（Securitization）**：一种言语行为（speech act），通过将某问题标定为"存在性威胁"，使之超越正常政治，进入紧急政治领域
- **安全化过程**：安全化行为体 → 言语行为 → 受众接受 → 采取紧急措施
- **非安全化**：将问题从安全议程移回正常政治

## 4. 马克思主义与国际政治经济学（IPE）

### 4.1 世界体系理论

伊曼纽尔·沃勒斯坦（Immanuel Wallerstein）提出：

| 层级 | 特征 | 例示 |
|------|------|------|
| 核心 | 高技术、高利润生产、资本密集 | 美国、西欧、日本 |
| 半边缘 | 核心与边缘之间的缓冲带 | 中国、印度、巴西 |
| 边缘 | 原材料出口、低工资、依附性强 | 撒哈拉以南非洲 |

**不平等交换**：剩余价值通过不等的贸易条件从边缘向核心转移。**长周期理论**：资本主义世界体系经历扩张-停滞的长周期（康德拉季耶夫周期）。

### 4.2 依赖理论

安德烈·冈德·弗兰克（Andre Gunder Frank）：**不发达的发展**（Development of Underdevelopment）。

核心论点：欠发达不是原始状态或传统社会的结果，而是被资本主义世界体系"制造"的。卫星国（边缘）与中心（核心）的联系越强，卫星国越不发达。

### 4.3 新葛兰西主义

罗伯特·考克斯（Robert Cox）提出：**理论总是为某人、某种目的服务**。

关键概念：
- **社会力量**：物质能力、思想和制度的复合体
- **霸权**：不仅仅是国家间支配，而是**同意+强制**的组合
- **历史集团**：社会力量的特定历史组合，形成稳定统治秩序

## 5. 英国学派（English School）

### 5.1 国际社会与国际体系的区分

| 概念 | 含义 | 条件 |
|------|------|------|
| 国际体系 | 国家间有足够的互动影响彼此行为 | 有外交、贸易、战争等联系 |
| 国际社会 | 国家间不仅互动，而且共享规则和制度 | 有共同利益、规则和制度 |

### 5.2 主要制度

英国学派认定的**主要制度**包括：

| 制度 | 作用 |
|------|------|
| 主权 | 国家独立与领土完整的相互承认 |
| 外交 | 国家间沟通与谈判的规范化渠道 |
| 均势 | 防止任何一国获得霸权 |
| 国际法 | 约束国家行为的法律框架 |
| 大国管理 | 大国的特殊责任与特权 |
| 战争 | 作为国际社会认可的合法（正义战争） |
| 民族主义 | 民族自决原则 |

英国学派代表人物：赫德利·布尔（Hedley Bull）《无政府社会》（1977）。

## 6. 理论比较与总结

| 维度 | 现实主义 | 自由主义 | 建构主义 | 马克思主义 |
|------|---------|---------|---------|-----------|
| 核心行为体 | 国家 | 国家+非国家 | 国家+观念 | 阶级+国家 |
| 人性假设 | 性恶 | 性善或中性 | 可塑 | 经济决定 |
| 国际政治本质 | 权力斗争 | 合作可能 | 观念建构 | 阶级冲突 |
| 变革的可能性 | 低 | 中等 | 高 | 革命性 |
| 对制度的看法 | 权力工具 | 促进合作 | 建构身份 | 统治阶级工具 |

## 7. 中国学者与国际关系理论

中国国际关系学界在吸收西方理论的基础上，提出了一些具有中国特色的视角：
- **关系理论**（秦亚青）：以"关系"为核心概念的东方国际关系理论
- **道义现实主义**（阎学通）：将道义因素引入现实主义分析框架
- **天下体系**（赵汀阳）：以"天下"为单位的政治秩序理想
