# 公共行政学

## 一、传统公共行政

### 1.1 政治-行政二分法

Wilson在《行政学研究》（1887）中提出政治与行政的二分：政治是政策的制定，行政是政策的执行。行政应当在政治之外独立运行，追求效率和专业化。

| 领域 | 功能 | 主体 |
|------|------|------|
| 政治 | 制定政策、表达民意 | 政治家、议会 |
| 行政 | 执行政策、公共管理 | 官僚、行政机构 |

Goodnow在《政治与行政》中进一步发展了这一区分，认为行政应当从政治干预中分离出来以获得效率。这一二分法奠定了公共行政学作为独立学科的基础。

### 1.2 韦伯的官僚制

韦伯的"理想型"官僚制是现代行政管理的基本组织形式，具有以下特征：

| 特征 | 含义 |
|------|------|
| 等级制 | 明确的金字塔式层级结构 |
| 分工 | 专业化的职能划分 |
| 非人格化 | 按规则行事，不掺杂个人情感 |
| 书面文件 | 一切公务以文件形式记录 |
| 专业培训 | 官员具备专业知识 |
| 职业化 | 官员以公务员为职业，有固定薪俸 |

韦伯认为官僚制是现代社会中最理性、最有效的组织形式，但也警惕其可能导致"铁笼"——将人禁锢在规则和等级中。

### 1.3 泰勒的科学管理

Taylor在《科学管理原理》（1911）中将科学方法引入管理，核心原则包括：

| 原则 | 内容 |
|------|------|
| 科学而非经验 | 用科学方法研究每一个工作要素 |
| 挑选与培训 | 科学地挑选工人并进行培训 |
| 合作而非个人主义 | 管理者和工人密切合作 |
| 职能分工 | 管理者和工人各司其职 |

科学管理对公共行政的影响深远——推动了预算制度、绩效评估和效率标准在政府中的应用，但也因忽视人的社会需求而受到批评。

## 二、新公共管理

### 2.1 新公共管理运动

Osborne & Gaebler在《重塑政府》（1992）中提出新公共管理（NPM）的核心理念——用企业家精神改造政府：

| 传统行政 | 新公共管理 |
|---------|-----------|
| 强调规则 | 强调结果 |
| 官僚控制 | 市场竞争 |
| 集权 | 分权 |
| 政府生产 | 政府购买 |
| 效率优先 | 顾客满意 |
| 稳定预算 | 绩效预算 |

### 2.2 NPM的核心机制

| 机制 | 内容 | 例 |
|------|------|-----|
| 私有化 | 国有企业转让给私人 | 英国电信私有化 |
| 合同外包 | 政府购买服务 | 垃圾清运外包 |
| 内部市场 | 公共服务引入市场机制 | NHS内部市场 |
| 绩效管理 | 指标考核、结果导向 | 绩效工资 |
| 顾客导向 | 将公民视为顾客 | 一站式服务 |

### 2.3 对NPM的批评

| 批评 | 论证 |
|------|------|
| 公共性丧失 | 公民被降格为顾客 |
| 碎片化 | 合同外包导致政府协调能力下降 |
| 问责困难 | 外包服务中谁对问题负责？ |
| 公平问题 | 市场化服务忽视弱势群体 |

## 三、新公共服务

### 3.1 核心主张

Denhardt在《新公共服务》（2003）中提出基于公民权、社区和民主理论的替代框架：

> 公共行政官员在管理公共组织和执行公共政策时应该集中于承担为公民服务和向公民放权的职责。

| 原则 | 核心内容 |
|------|---------|
| 服务于公民而非顾客 | 关注公民的共同利益 |
| 追求公共利益 | 公共利益是目标而非副产品 |
| 战略思考民主行动 | 政策制定需要民主参与 |
| 服务于人而非生产率 | 尊重人的价值和尊严 |
| 重视公民权胜过企业家精神 | 官员是公共利益的受托人 |

### 3.2 治理理论

"治理"（governance）超越"政府"（government）概念。治理强调多元主体的网络化协作，包括政府、市场、社会组织、公民等。Rhodes提出治理是"没有政府的治理"——自组织网络在公共事务中发挥核心作用。

## 四、公共预算

### 4.1 预算类型

| 类型 | 特征 | 优点 | 缺点 |
|------|------|------|------|
| 增量预算 | 在上年基础上调整 | 稳定、易操作 | 缺乏效率考量 |
| 零基预算 | 每年从零开始论证 | 资源优化 | 耗时、行政成本高 |
| 绩效预算 | 按产出和结果分配 | 效率导向 | 难以量化 |
| 项目预算（PPBS） | 按项目目标分配 | 全局规划 | 技术要求高 |

### 4.2 预算原则

| 原则 | 含义 |
|------|------|
| 全面性 | 全部收支纳入预算 |
| 公开性 | 预算过程应当透明 |
| 年度性 | 预算按年度编制 |
| 约束性 | 预算具有法律效力 |
| 统一性 | 统一的预算体系 |

## 五、电子政务与行政伦理

### 5.1 电子政务

| 阶段 | 特征 | 例 |
|------|------|-----|
| 信息发布 | 政府网站公布信息 | 政策文件网上公开 |
| 单向互动 | 提供下载服务 | 表格下载 |
| 双向互动 | 在线办理业务 | 网上报税 |
| 网络治理 | 全面数字化协同 | 数字政府平台 |

电子政务的核心理念是提升效率、透明度和公民参与。但需要警惕数字鸿沟问题——老年人、低收入群体可能被排除在数字化服务之外。

### 5.2 行政伦理

| 伦理议题 | 内容 |
|---------|------|
| 利益冲突 | 公务员个人利益与公共利益的冲突 |
| 腐败 | 滥用公共权力谋取私利 |
| 举报（whistleblowing） | 内部人员揭露组织不当行为 |
| 透明度 | 决策公开、信息自由 |
| 责任 | 行政人员的法律、政治和道德责任 |

举报人保护是行政伦理的重要议题，涉及个人道德勇气与组织忠诚之间的张力。

### 5.3 伦理治理框架

| 层次 | 机制 |
|------|------|
| 法律 | 反腐败法、信息公开法 |
| 制度 | 伦理委员会、审计监督 |
| 职业 | 公务员行为准则、职业培训 |
| 个人 | 领导示范、伦理自觉 |

## 六、公共行政改革与比较

### 6.1 国家治理现代化

国家治理体系和治理能力现代化是当代中国公共行政的核心议题。改革方向包括：政府职能转变（从全能政府向服务型政府转变）、放管服改革、数字政府建设、绩效管理体系建设。

| 改革领域 | 措施 | 目标 |
|---------|------|------|
| 职能转变 | 简政放权、负面清单管理 | 市场在资源配置中起决定性作用 |
| 数字政府 | 一网通办、数据共享 | 提升政府服务效率 |
| 绩效管理 | 目标责任制、绩效考核 | 提高政府执行力 |
| 事业单位改革 | 分类改革、去行政化 | 提高公共服务质量 |

### 6.2 比较公共行政

| 行政模式 | 特征 | 例 |
|---------|------|-----|
| 英美模式 | 新公共管理、市场化改革 | 英国、美国、新西兰 |
| 欧洲大陆模式 | 法律主义、科层制传统 | 德国、法国 |
| 东亚模式 | 发展型国家、精英官僚制 | 日本、韩国、新加坡 |
| 转型模式 | 从计划走向市场 | 中国、越南 |

### 6.3 公共行政的伦理框架

行政伦理在当代公共行政中日益受到重视。伦理治理需要多层次机制：

| 层面 | 机制 | 功能 |
|------|------|------|
| 预防 | 伦理培训、行为准则 | 提前规范行为 |
| 监督 | 审计、信息公开、媒体监督 | 及时发现不当行为 |
| 惩戒 | 纪律处分、法律制裁 | 对违规行为形成威慑 |

### 6.4 跨部门协作

公共问题的复杂性要求政府部门之间、政府与市场和社会组织之间的协作。协作治理强调多元主体的平等参与和共识决策。协作失败的原因包括：部门利益冲突、沟通不畅、资源分配争议和信任缺失。

## 七、公共行政的当代挑战

### 7.1 公共行政的合法性危机

现代公共行政面临合法性危机：政府被期望做更多（更高的公民期望），同时被信任得更少（下降的政府信任度）。这一"期望-信任"张力是公共行政的核心挑战。

| 挑战 | 表现 |
|------|------|
| 财政压力 | 税收增长赶不上公共服务需求增长 |
| 复杂性 | 跨界问题（气候变化、公共卫生）超出单个部门能力 |
| 信息革命 | 社交媒体加速了信息传播和问题放大 |
| 不平等 | 公共服务的均等化面临挑战 |

### 7.2 敏捷治理

敏捷治理借鉴了软件开发的敏捷方法，强调快速迭代、用户参与和灵活应变。与传统的"计划-执行"模式不同，敏捷治理通过小规模试点、快速反馈和持续调整来应对不确定性。

### 7.3 公共价值

Moore提出"公共价值"（public value）框架——公共管理者不仅是规则执行者，更是公共价值的创造者。公共价值三角模型：

| 维角 | 内容 |
|------|------|
| 价值 | 对公民而言什么是值得做的？ |
| 授权与环境 | 如何获得合法性和政治支持？ |
| 操作能力 | 组织是否有能力交付？ |
