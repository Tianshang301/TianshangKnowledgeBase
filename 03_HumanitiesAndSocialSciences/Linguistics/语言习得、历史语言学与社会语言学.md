# 语言习得、历史语言学与社会语言学

## 1. 第一语言习得（First Language Acquisition）

### 1.1 关键理论流派

| 理论学派 | 代表人物 | 核心主张 | 关键概念 |
|---------|---------|---------|---------|
| **行为主义** | Skinner (1957) | 语言通过"刺激-反应-强化"学会 | 模仿、操作条件反射 |
| **先天论/先天主义** | Chomsky (1959) | 人类有先天的语言能力 | LAD（Language Acquisition Device） |
| **交互作用论** | Piaget, Vygotsky | 先天能力+环境输入+认知发展共同作用 | 认知发展先行于语言 |
| **使用基模语言习得** | Tomasello (2003) | 从具体的"语块"逐步抽象出规则 | 语项学习、图式化 |

### 1.2 关键期假说

**Lenneberg（1967）** 提出**关键期假说**（Critical Period Hypothesis）：

> 语言习得存在一个关键的生物学窗口期——从大约 2 岁到青春期（约 12 岁）。在此之后，大脑的可塑性下降，语言习得变得困难且不完整。

**支持证据**：
- **Genie 案例**（1970）：13 岁前被隔离的儿童，此后无法完全习得语言
- **聋人手语**（Newport, 1990）：晚期接触 ASL 的美国手语学习者语法表现不如早期接触者
- **第二语言年龄效应**：成年后学习外语几乎无法达到母语者的语法判断能力

**争议**：Johnson & Newport (1989) 支持"关键期"，但后续研究显示可能是"敏感期"（Sensitive Period）而非严格的"关键期"。

### 1.3 儿童语言发展阶段

| 阶段 | 年龄 | 特征 | 示例 |
|------|------|------|------|
| **前语言期** | 0-12 个月 | 哭→咕咕声→咿呀语 | baba, dada（普遍性） |
| **单词期** | 12-18 个月 | 单词语义过度延伸/缩小 | "狗"指所有四足动物 |
| **双词期** | 18-24 个月 | 电报式言语 | "妈妈抱""狗狗走" |
| **电报语期** | 24-30 个月 | 句子变长，语法词缺失 | "爸爸看书" |
| **后期** | 3-5 岁 | 语法趋于成熟，复杂句出现 | "因为……所以……" |

**形态句法习得顺序**（Brown, 1973）：
英语中-ing > 复数 -s > 不规则过去时 > 所有格 's > 冠词 > 规则过去时 -ed > 第三人称单数 -s

### 1.4 输入假说（Input Hypothesis）

**Krashen（1982, 1985）** 提出五大假说：

| 假说 | 内容 |
|------|------|
| **习得-学习假说** | 无意识"习得"与有意识"学习"是两种不同过程 |
| **监控假说** | 有意识的知识只起编辑（监控）作用 |
| **自然顺序假说** | 语法结构以可预测的顺序习得 |
| **输入假说** | $i+1$：略高于学习者当前水平的可理解输入是习得的必要条件 |
| **情感过滤假说** | 焦虑→过滤升高→输入受阻→习得减少 |

**"i+1"公式**：
$$i = \text{当前水平},\quad i+1 = \text{下一阶段水平}$$
可理解输入 = 包含 $i+1$ 的语言输入。

**评价**：输入假说的"可理解输入是充分必要条件"受到质疑——Swain 提出输出假说。

## 2. 第二语言习得（Second Language Acquisition, SLA）

### 2.1 核心概念

| 概念 | 提出者 | 说明 |
|------|--------|------|
| **母语迁移** | Lado (1957) | 母语结构对二语学习的影响：正迁移（相似→促进）、负迁移（差异→干扰） |
| **中介语** | Selinker (1972) | 学习者自成一体的语言系统，具有系统性、动态性和过渡性 |
| **石化**（Fossilization） | Selinker | 中介语中某些错误永久固定，无法消除 |
| **输出假说** | Swain (1985) | 输出促使学习者注意语言形式、检验假设、反思元语言知识 |
| **互动假说** | Long (1981, 1996) | 意义协商（negotiation for meaning）促进习得 |

**对比分析假说**（Contrastive Analysis Hypothesis, CAH）：
- 强式：可通过对比母语和目的语预测所有学习难点
- 弱式：仅分析已知错误
- 局限：并非所有差异都造成困难（过度预测），也非所有相似都容易

### 2.2 教学法演变

| 教学法 | 时期 | 核心理念 | 课堂活动 |
|--------|------|---------|---------|
| **语法翻译法** | 19 世纪 | 语法规则+翻译+词汇记忆 | 翻译句子、背诵规则 |
| **听说法** | 1940s-1960s | 结构主义+行为主义 | 句型操练（drills） |
| **交际教学法** | 1970s- | 交际能力（Hymes） | 角色扮演、信息差活动 |
| **任务型教学法** | 1980s- | 用真实任务组织教学 | 解决问题、项目式学习 |

**交际能力**（Communicative Competence, Canale & Swain, 1980）：
- 语法能力：语言形式的掌握
- 社会语言能力：语境中恰当使用语言
- 语篇能力：连贯与衔接
- 策略能力：克服沟通障碍

### 2.3 个体差异因素

| 因素 | 对 SLA 的影响 |
|------|-------------|
| 年龄 | 关键期效应：儿童更易达到母语水平 |
| 语言学能 | 语音编码、语法敏感性、归纳能力 |
| 动机 | 融入型（认同目的语文化）vs 工具型（实用目的） |
| 人格特质 | 外向者更多交流机会，内向者更准确？ |
| 工作记忆 | 加工容量影响句法加工和学习效率 |

## 3. 历史语言学（Historical Linguistics）

### 3.1 语音变化（Sound Change）

**格林定律**（Grimm's Law, 1822）：日耳曼语系相对于原始印欧语的辅音系统性移位。

| PIE → 日耳曼语 | 规则 | 示例（拉丁语→英语） |
|---------------|------|-------------------|
| $p \rightarrow f$ | 清塞音→清擦音 | piscis → fish |
| $t \rightarrow \theta$ |  | tres → three |
| $k \rightarrow h$ |  | centum → hundred |
| $b \rightarrow p$ | 浊塞音→清塞音 | cannabis → hemp |
| $d \rightarrow t$ |  | duo → two |
| $g \rightarrow k$ |  | genus → kin |
| $bh \rightarrow b$ | 送气浊塞音→浊塞音 | bhrāter → brother |

**Verner 定律**（Verner, 1877）：解释了格林定律的例外——非重读音节后的清擦音变为浊擦音/浊塞音。

**元音大推移**（Great Vowel Shift, 15-17 世纪）：英语元音系统的系统性上移。

| 中古英语 | 现代英语 | 示例 |
|---------|---------|------|
| /iː/ → | /aɪ/ | five /fiːv/ → /faɪv/ |
| /eː/ → | /iː/ | feet /feːt/ → /fiːt/ |
| /aː/ → | /eɪ/ | name /naːmə/ → /neɪm/ |
| /uː/ → | /aʊ/ | house /huːs/ → /haʊs/ |

### 3.2 语法化（Grammaticalization）

实义词在特定语境中逐渐获得语法功能的历时过程。

**典型的语法化路径**：
$$实义词 \rightarrow 语法词 \rightarrow 附着语素 \rightarrow 屈折语素$$

**示例**：
- 汉语"将"：动词（"将军"→率领）→ 将来时标记（"将要"）
- 英语"will"：动词（"to wish/want"）→ 将来时助动词
- 英语"be going to"：空间移动 → 未来意图 → 单纯未来时标记

**语法化的特征**（Hopper & Traugott, 2003）：
- **去语义化**：语义缩蚀
- **去范畴化**：丧失原有词类特征
- **语音缩减**：语音形式弱化

### 3.3 语义变化（Semantic Change）

| 类型 | 说明 | 示例 |
|------|------|------|
| **词义扩大**（Broadening） | 词义范围扩大 | dog（古英"特定犬种"→所有犬） |
| **词义缩小**（Narrowing） | 词义范围缩小 | meat（古英"食物"→肉） |
| **词义转移**（Shift） | 词义从一个领域到另一领域 | nice（古法"愚蠢的/无知的"→好的） |
| **词义变贬**（Pejoration） | 词义负面化 | silly（古英"幸福的/神圣的"→傻的） |
| **词义变褒**（Amelioration） | 词义正面化 | knight（古英"仆人"→骑士） |

### 3.4 比较法与历史重建

**历史比较法**（Comparative Method）：
1. 辨认同源词（cognates）
2. 建立规则对应关系
3. 重建祖语（proto-language）形式

**例：原始印欧语（PIE）重建**：

| 语言 | "父亲" | "母亲" | "三" | "名字" |
|------|--------|--------|------|--------|
| 拉丁语 | pater | māter | trēs | nōmen |
| 希腊语 | patēr | mētēr | treis | onoma |
| 梵语 | pitṛ | mātṛ | tri | nāma |
| 古英语 | fæder | mōdor | þrī | nama |
| **PIE** | ***ph₂tēr** | ***méh₂tēr** | ***tréyes** | ***h₁nómn̥** |

### 3.5 语言谱系分类

| 语系 | 主要分支 | 代表性语言 |
|------|---------|-----------|
| **印欧语系** | 日耳曼、罗曼、斯拉夫、印度-伊朗、凯尔特、希腊 | 英/德/法/俄/印地/波斯 |
| **汉藏语系** | 汉语、藏缅语族 | 汉语、藏语、缅甸语 |
| **阿尔泰语系（有争议）** | 突厥、蒙古、通古斯 | 土耳其语、蒙古语 |
| **尼日尔-刚果语系** | Bantu 语族 | 斯瓦希里语、祖鲁语 |
| **亚非语系（Hamito-Semitic）** | 闪语族、库什语族 | 阿拉伯语、希伯来语 |
| **南岛语系** | 马来-波利尼西亚 | 马来语、菲律宾语 |

## 4. 社会语言学（Sociolinguistics）

### 4.1 方言与语言变异

**地域方言**：语言的地理变体。

**汉语十大方言**：官话（约 70% 人口）、粤语、吴语、闽语、客家话、湘语、赣语、晋语、徽语、平话。

**等语线**（Isogloss）：地图上表示某一语言特征分布边界的线。多条等语线重合构成方言边界。

**社会方言**：与社会阶层、年龄、性别、民族等相关联的语言变异。

**Labov 纽约百货公司调查（1966）**：
- 研究对象：三家不同等级百货公司中 /r/ 发音的分布
- 发现：社会阶层越高，/r/ 发音率越高（Saks 最高，Macy's 居中，Klein's 最低）
- 意义：首次系统证明语言变异与**社会阶层**的系统性关联

### 4.2 语域与语码

| 概念 | 说明 | 示例 |
|------|------|------|
| **语域（Register）** | 根据情境选择的不同语言风格 | 正式："I wish to enquire whether…"vs 非正式："Can I ask…" |
| **语码转换**（Code-switching） | 双语者在一段话语中切换语言/方言 | "我今天有个 meeting，超 busy" |
| **双言制**（Diglossia, Ferguson） | 同一语言的高低变体分工使用 | 阿拉伯语：Fusha（高：正式）vs Ammiyya（低：日常） |

**语码转换的功能**（Blom & Gumperz, 1972）：
- 情境性转换：听话人改变时切换
- 隐喻性转换：表达身份认同或态度

### 4.3 语言政策与规划

| 类型 | 说明 | 示例 |
|------|------|------|
| **地位规划** | 决定语言的官方地位 | 汉语为中国的官方语言 |
| **本体规划** | 语言的标准化与现代化 | 普通话的语音/词汇规范化 |
| **习得规划** | 语言教育政策 | 新加坡的双语教育政策 |
| **语言复兴** | 拯救濒危语言 | 希伯来语复兴（最成功案例） |

## 5. 认知语言学（Cognitive Linguistics）

### 5.1 范畴化（Categorization）

**经典范畴观**（Aristotle）：范畴有明确的边界和充要条件。

**原型理论**（Prototype Theory, Rosch, 1973, 1975）：
- 范畴并非离散的，而是有**中心成员**（原型）和**边缘成员**
- "鸟"的原型：麻雀 > 鸽子 > 鸡 > 企鹅 > 鸵鸟
- 家族相似性（Wittgenstein）：成员间共享部分而非全部特征

**基本层次范畴**（Basic Level Categories, Rosch）：
- 上位层次：动物（概括、抽象）
- **基本层次**：狗（认知上最凸显、最常用、最早习得）
- 下位层次：柯基犬（更具体、更少区分性）

### 5.2 概念隐喻（Conceptual Metaphor）

**Lakoff & Johnson（1980）**《我们赖以生存的隐喻》：

> 隐喻不仅是语言现象，更是**思维方式**。我们的概念系统本质上是隐喻性的。

**概念隐喻公式**：$目标域 \text{ IS } 源域$

| 概念隐喻 | 语言表达 |
|---------|---------|
| 争论是战争 | 你的论点**不堪一击**；我**攻击**了你的立场 |
| 时间是金钱 | 别**浪费**时间；他**花**了三小时 |
| 人生是旅行 | 他**走**了很长的路；人生**旅途** |
| 情感是温度 | 她**热情**洋溢；**冷淡**的回应 |

**隐喻的认知功能**：通过具体的源域（身体经验、空间关系）理解抽象的目标域。

### 5.3 构式语法（Construction Grammar）

**Goldberg（1995, 2006）**：语法结构本身是有意义的。

**构式**（Construction）的定义：
> 任何形式-意义的配对，其某些形式或意义特征不能从其组成部分完全预测。

**构式示例**：
- **双宾构式**（Subj V Obj1 Obj2）："He gave her a book"——构式义：传递
- **动结构式**（Subj V Obj Xcomp）："He painted the door red"——构式义：致使状态变化
- **way 构式**："He shoved his way through the crowd"——构式义：费力前进

**构式语法的优势**：
- 无需"中心语"和"补足语"的严格区分
- 能处理习语和不规则结构
- 契合语言习得中的"语块"学习模式

### 5.4 意象图式（Image Schemas）

基于身体经验的、前概念性的认知结构，构成概念系统的基础。

| 图式 | 基于身体经验 | 映射到抽象域 |
|------|-------------|-------------|
| **容器** | 身体作为容器（进/出） | 范畴作为容器（"在范畴内/外"） |
| **路径** | 从 A 到 B 的移动 | 目的作为目的地（"到达目标"） |
| **连接** | 脐带、手拉手 | 社会关系作为连接（"建立联系"） |
| **上下** | 直立（上）vs 躺下（下） | 好为上、坏为下（"心情高涨"） |
| **平衡** | 身体平衡 | 权利平衡、权衡利弊 |
