# 语义学、语用学与语言类型学

## 1. 语义学（Semantics）

语义学研究语言的意义，包括词义、句义及意义组合的规律。

### 1.1 词义关系（Lexical Relations）

| 关系类型 | 说明 | 示例 |
|---------|------|------|
| **同义**（Synonymy） | 意义相同或相近 | big ~ large；快乐 ~ 高兴 |
| **反义**（Antonymy） | 意义相反 | 互补反义：alive/dead |
| |  | 等级反义：hot/cold（可比较） |
| |  | 方向反义：enter/exit |
| |  | 关系反义：buy/sell；teacher/student |
| **上下义**（Hyponymy） | 包含关系 | dog 是 animal 的下义词 |
| |  | rose 是 flower 的下义词 |
| **部分整体**（Meronymy） | 部分-整体关系 | finger 是 hand 的组成部分 |
| |  | page 是 book 的组成部分 |
| **同音/多义** |  | 同音（bank¹银行/bank²河岸） |
| |  | 多义（head：头/领导/顶端） |

**语义场**（Semantic Field）：同一概念域中词项构成的有组织的词汇集合。如颜色词、亲属词系统在不同语言中有不同切分方式。

### 1.2 语义角色（Semantic Roles / Thematic Relations）

语义角色描述名词短语在事件中承担的"角色"：

| 角色 | 说明 | 示例 |
|------|------|------|
| **施事**（Agent） | 动作的有意识执行者 | 张三\_Agent 打了李四 |
| **受事**（Patient/Theme） | 承受动作或状态变化 | 张三打了李四\_Patient |
| **工具**（Instrument） | 用来完成动作的事物 | 他用刀\_Instrument 切菜 |
| **地点**（Location） | 动作发生的地点 | 他在厨房\_Location 做饭 |
| **来源**（Source） | 移动的起点 | 他从北京\_Source 出发 |
| **目标**（Goal） | 移动的终点 | 他到了上海\_Goal |
| **经验者**（Experiencer） | 感知/心理状态的主体 | 他\_Experiencer 喜欢音乐 |
| **受益者**（Beneficiary） | 动作的受益方 | 他为妈妈\_Beneficiary 买了礼物 |

**语义角色与句法主语的不对称性**：
- 主动句：施事作主语（张三\_Agt 打了李四）
- 被动句：受事作主语（李四\_Pat 被张三打了）

### 1.3 谓词逻辑（Predicate Logic）

语义学使用形式逻辑工具精确刻画句子的真值条件。

**一阶谓词逻辑**基本要素：
- 个体常量（$a, b, c$）：John, Mary
- 谓词（$P, Q, R$）：$Dog(x)$, $Love(x,y)$
- 量词：$\forall$（全称量词），$\exists$（存在量词）
- 逻辑联结词：$\land, \lor, \rightarrow, \neg$

**示例**：
- "每条狗都是哺乳动物"：$\forall x (Dog(x) \rightarrow Mammal(x))$
- "有人爱张三"：$\exists x (Person(x) \land Love(x, Zhangsan))$
- "所有人都会死"：$\forall x (Human(x) \rightarrow Mortal(x))$

**λ-演算**（Montague 语义学）：用于处理量词辖域、动词词组组合等问题。
- $\lambda x[P(x)]$：将 $x$ 映射到 $P(x)$ 的函数
- $\lambda x[Love(j, x)]$：被约翰爱的人的集合

### 1.4 组合语义学（Compositional Semantics）

**Frege 原则**（Frege's Principle of Compositionality）："一个复合表达式的整体意义是其组成部分的意义及其组合方式的函数。"

**形式化表述**：
$$
\llbracket \alpha \circ \beta \rrbracket = f(\llbracket \alpha \rrbracket, \llbracket \beta \rrbracket, \circ)
$$

其中 $\llbracket X \rrbracket$ 表示 $X$ 的语义值。

**应用示例**：
- "张三爱李四" = $Love(Zhangsan, Lisi)$
- $[[张三]]] [爱李四]_{VP}]_S$ → 谓词 $Love$ 应用于个体 $Zhangsan$ 和 $Lisi$
- 组合过程：$Love(Lisi)$ 作为属性，$Zhangsan$ 作为主语，得到 $Love(Zhangsan, Lisi)$

**挑战**：习语（"kick the bucket"≠桶+踢）、语境依赖、隐喻等问题对严格的组合性提出挑战。

### 1.5 真值条件语义学（Truth-Conditional Semantics）

**核心主张**：知道一个句子的意义，就是知道它在什么条件下为真。

- "雪是白的"为真当且仅当雪确实是白的
- $\llbracket 雪是白的 \rrbracket = 1$ iff $White(snow)$

**可能世界语义学**（Possible Worlds Semantics, Kripke）：
- 命题 = 所有使该命题为真的可能世界的集合
- 必然真 = 在所有可能世界中为真
- 可能真 = 在至少一个可能世界中为真

| 概念 | 符号 | 说明 |
|------|------|------|
| 必然 | $\Box P$ | 在所有可能世界中 $P$ 真 |
| 可能 | $\Diamond P$ | 在某个可能世界中 $P$ 真 |

## 2. 语用学（Pragmatics）

语用学研究语境中语言的使用和理解——"说话人意义"而非"句子意义"。

### 2.1 言语行为理论（Speech Act Theory）

**Austin（1962）** 的突破性贡献：说话本身就是在做事。

**三分框架**：
| 层次 | 说明 | 示例（"你能把门关上吗？"） |
|------|------|--------------------------|
| **言内行为**（Locutionary Act） | 说出有意义的句子 | 发出了"你能把门关上吗"的声音 |
| **言外行为**（Illocutionary Act） | 说话时的意图和功能 | 请求对方关门（而非询问能力） |
| **言后行为**（Perlocutionary Act） | 话语产生的效果 | 实际导致对方关上了门 |

**显性施为句**（Explicit Performatives）："我承诺……""我警告你……""我宣布……"

**Searle（1969, 1979）** 的言语行为分类：

| 类别 | 言外之意 | 心理状态 | 示例 |
|------|---------|---------|------|
| **断言类**（Assertives） | 说话人相信某事为真 | 信念 | 我断言、陈述、声称 |
| **指令类**（Directives） | 让听话人做某事 | 愿望 | 请求、命令、建议 |
| **承诺类**（Commissives） | 承诺做某事 | 意图 | 承诺、保证、威胁 |
| **表达类**（Expressives） | 表达心理状态 | 情感 | 道歉、感谢、祝贺 |
| **宣告类**（Declarations） | 改变现实状态 | — | 宣战、解雇、命名 |

**间接言语行为**（Indirect Speech Act）：表面形式与言外之力不匹配。
- "你能把盐递给我吗？"——表面是问句，实际是请求
- "It's cold in here."——表面是陈述，实际是请求关窗

### 2.2 合作原则与含义（Cooperative Principle & Implicature）

**Grice（1975）** 提出，会话参与者通常遵循**合作原则**——使自己的话语符合当前交流的共同目的。

**四条准则（Maxims）**：

| 准则 | 说明 | 违反示例 |
|------|------|---------|
| **量**（Quantity） | 提供适量信息（不多不少） | 问时间答"现在是白天" |
| **质**（Quality） | 不说假话或缺乏证据的话 | 明知假却说"地球是平的" |
| **关系**（Relation） | 说话相关 | 答非所问 |
| **方式**（Manner） | 清晰、有序、避免晦涩和歧义 | "你用那个东西弄那个" |

**会话含义**（Conversational Implicature）：说话人故意违反某个准则以传达额外意义。

**示例**：
A: "我们去看电影吧？"
B: "我明天有考试。"
- B 表面上违反了"关系"准则
- **含义**：B 拒绝了这个邀请（因为需要复习）

**含义的特征**：
- **可取消性**：可在后面加上"不过我还是可以去"
- **不可分离性**：与语义内容相关
- **可推导性**：基于准则和语境可推导

### 2.3 礼貌理论（Politeness Theory）

**Leech（1983）** 的礼貌原则——将礼貌视为交际中的调节机制。

**Brown & Levinson（1987）** 的**面子理论（Face Theory）**：

**面子** = 每个社会成员希望为自己维护的公共自我形象。
- **积极面子**：希望被认可、接纳、赞赏
- **消极面子**：希望自主、不被干涉

**面子威胁行为**（Face-Threatening Act, FTA）：本质上威胁面子的言语行为。

**礼貌策略**（以威胁强弱排序）：

| 策略 | 说明 | 示例 |
|------|------|------|
| 公开直接 | 不加修饰 | "关门！" |
| 积极礼貌 | 表达关注和亲近 | "亲爱的，能帮我关下门吗？" |
| 消极礼貌 | 尊重对方自主权 | "不知是否方便关一下门？" |
| 间接暗示 | 言外之意 | "门开着呢……" |
| 不实施 FTA | 完全不说 | 自己去关门 |

**文化差异**：积极礼貌文化（中国、日本等集体主义社会）vs 消极礼貌文化（英美等个人主义社会）。

### 2.4 预设（Presupposition）

**预设**：说话人假设的背景信息，在句子中不必明说。

**预设触发语**（Presupposition Triggers）：

| 类型 | 触发语 | 示例 | 预设 |
|------|--------|------|------|
| 定指描述 | 冠词/专名 | "法国的国王是秃头" | 法国有国王 |
| 事实动词 | 知道、后悔 | "张三后悔没去" | 张三没去 |
| 蕴含动词 | 停止、继续 | "他停止抽烟了" | 他之前抽烟 |
| 分裂句 | 是……的 | "是李四偷了钱" | 有人偷了钱 |
| 疑问句 | — | "你怎么又迟到了？" | 你之前迟到过 |

**预设的投影问题**（Projection Problem）：预设如何在复合句中"幸存"？
- 在否定句中：预设通常保留（"法国国王不是秃头"→仍有国王）
- 在条件句中：预设可能被取消

### 2.5 指示词（Deixis）

**指示**是指语言的"索引"功能——词汇的指称依赖于语境。

| 指示类型 | 说明 | 示例 |
|---------|------|------|
| **人称指示** | 话语角色 | 我、你、他、我们 |
| **时间指示** | 时间锚定 | 现在、昨天、明年、那时 |
| **地点指示** | 空间锚定 | 这里、那里、左边、上面 |
| **语篇指示** | 文本内部指代 | 如上所述、下文中、前例 |
| **社交指示** | 社会关系 | 您、你；先生、老师 |

**指示中心的转移**（Deictic Projection）："我现在就到你那边去"——定向以听话人为参考点。

**人称指示与社会等级**：在许多语言中（如汉语、韩语、日语、泰语），人称代词的选择由社会关系和礼貌程度决定（"您"vs"你"）。

## 3. 语言类型学（Linguistic Typology）

语言类型学研究和比较世界语言的共性与差异。

### 3.1 基本语序类型

| 语序 | 说明 | 代表性语言 | 例句 |
|------|------|-----------|------|
| **SVO** | 主-动-宾 | 英语、汉语、法语、越南语 | "猫吃了鱼" |
| **SOV** | 主-宾-动 | 日语、韩语、土耳其语、印地语 | "猫鱼吃了" |
| **VSO** | 动-主-宾 | 爱尔兰语、阿拉伯语、毛利语 | "吃了猫鱼" |
| **VOS** | 动-宾-主 | 马达加斯加语、Fijian | "吃了鱼猫" |
| **OVS** | 宾-动-主 | Hixkaryana（极少） | "鱼吃了猫" |
| **OSV** | 宾-主-动 | Xavante（极少） | "鱼猫吃了" |

**SOV 语言的分布**：SOV 是世界上最常见的语序类型（约 45% 的语言）。

### 3.2 Greenberg 的语言普遍性

**Joseph Greenberg（1963）** 提出 45 条语言普遍性，分为**绝对普遍性**（无例外）和**倾向性普遍性**（统计显著）。

**典型普遍性**：

| 编号 | 内容 |
|------|------|
| 普遍性 1 | 主语在宾语之前 | 在陈述句中，主语优于宾语 |
| 普遍性 3 | VSO 语言使用前置词 | SVO 也使用前置词，SOV 使用后置词 |
| 普遍性 4 | SOV 语言的后置词 | SOV 语言绝大多数为后置词（日语：本\_ACC を 読む） |
| 普遍性 17 | 修饰语与名词 | 形容词在名词前（SVO）或后（VSO/SOV） |
| 普遍性 25 | 代词性宾语 | 代词宾语通常先于名词宾语 |

**蕴含普遍性**（Implicational Universals）的公式：
> 如果语言有特征 $P$，则它也有特征 $Q$（$P \supset Q$）

例：如果一种语言是 VSO 语序，则它使用前置词（VSO ⊃ Prepositions）。

### 3.3 形态类型

| 类型 | 描述 | 特征 | 代表性语言 |
|------|------|------|-----------|
| **分析语（孤立语）** | 每个词素独立成词 | 无词形变化，语序和虚词表达语法关系 | 汉语、越南语 |
| **屈折语（融合语）** | 一个词缀含多个语法信息 | 词根与词缀高度融合，形态变化丰富 | 俄语、拉丁语、阿拉伯语 |
| **黏着语** | 词缀逐个按顺序附着 | 每个词缀表达单一语法意义，边界清晰 | 土耳其语、斯瓦希里语 |
| **多式综合语** | 一个词相当于一个句子 | 大量词缀包含主语、宾语等信息 | 因纽特语、Mohawk语 |

**黏着语示例——土耳其语**：
$$ev\text{-}ler\text{-}imiz\text{-}de \quad \text{"在我们的房子里"}$$
$$house\text{-}PL\text{-}1PL\text{-}LOC$$

**屈折语示例——俄语**：
$$\text{книг-а（书\_NOM）} \neq \text{книг-у（书\_ACC）}$$

**分析语示例——汉语**：
"我昨天看书"——没有时态标记，时间信息由词汇"昨天"表达。

### 3.4 类型学中的其他重要分类

**格标记系统**（Case-marking Systems）：

| 系统 | 说明 | 示例 |
|------|------|------|
| **主格-宾格** | 及物动词主语=不及物动词主语≠及物动词宾语 | 英语、汉语、日语 |
| **能格-绝对格** | 及物动词宾语=不及物动词主语≠及物动词主语 | 巴斯克语、格鲁吉亚语 |

**声调语言**：
| 语言 | 声调数量 | 说明 |
|------|---------|------|
| 汉语普通话 | 4 个 | 阴平、阳平、上声、去声 |
| 粤语 | 6-9 个 | 更复杂的声调系统 |
| 越南语 | 6 个 | 声调+喉塞 |
| 约鲁巴语 | 3 个 | 高、中、低 |

**量词语言**（Classifier Languages）：
- 汉语：一个人、两头牛、三张纸、四本书
- 日语：一人（ひとり）、二匹（にひき）
- 泰语、越南语、韩语也有丰富的量词系统

### 3.5 语言的普遍性探索

**语言类型学的近期发展**：
- **Dryer（1992）** 大规模样本统计验证语序相关
- **WALS**（*World Atlas of Language Structures*）：世界 2679 种语言的结构数据库
- **Hawkins（2004）** 的效率理论——语序偏好由加工效率驱动

**语言功能-类型学**：Greenberg 传统与功能语言学的结合，强调语言结构受交际功能和使用频率的塑造。
