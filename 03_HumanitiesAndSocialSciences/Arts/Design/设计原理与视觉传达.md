# 设计原理与视觉传达

## 一、设计元素 (Elements of Design)

### 1.1 线条 (Line)

线条是点的运动轨迹，是最基础的设计元素。

| 类型 | 心理效应 | 应用场景 |
|------|----------|----------|
| 水平线 (Horizontal) | 稳定、平静、开阔 | 风景、基础线 |
| 垂直线 (Vertical) | 力量、崇高、尊严 | 建筑、柱子 |
| 斜线 (Diagonal) | 运动、动感、不安 | 动态构图 |
| 曲线 (Curved) | 优雅、柔和、自然 | 人体、自然形态 |
| 折线 (Zigzag) | 紧张、节奏、能量 | 现代设计 |
| 虚线 (Dashed) | 方向暗示、轻盈 | 引导线、边界 |

### 1.2 形状 (Shape)

| 类型 | 特征 | 联想 |
|------|------|------|
| 几何形 (Geometric) | 精确、规则、理性 | 正方形：稳定；圆形：完整；三角形：动感 |
| 有机形 (Organic) | 自然、流动、不规则 | 树叶、云朵、生物形态 |
| 抽象形 (Abstract) | 简化的自然形态 | 符号、标志 |

### 1.3 色彩 (Color)

**色相环与色彩关系：**

| 色彩关系 | 定义 | 示例 |
|----------|------|------|
| 互补色 (Complementary) | 色相环上相对的颜色 | 红-绿、蓝-橙、黄-紫 |
| 类似色 (Analogous) | 色相环上相邻的颜色 | 蓝-蓝绿-绿 |
| 三色系 (Triadic) | 色相环上三等分的颜色 | 红-黄-蓝 |
| 分裂互补 (Split-Complementary) | 基色加互补色两侧的邻色 | 红-黄绿-蓝绿 |
| 单色系 (Monochromatic) | 同一色相不同明度/饱和度 | 浅蓝-中蓝-深蓝 |

**色彩三要素：**

| 要素 | 英文 | 含义 |
|------|------|------|
| 色相 | Hue | 色彩的名称，如红、蓝、黄 |
| 饱和度 | Saturation | 色彩的纯度或鲜度 |
| 明度 | Brightness / Value | 色彩的明暗程度 |

**色彩心理学：**

| 颜色 | 常见联想 | 心理效应 |
|------|----------|----------|
| 红色 | 激情、危险、能量 | 提高心率，引起注意 |
| 蓝色 | 冷静、信任、专业 | 降低血压，产生安全感 |
| 黄色 | 快乐、乐观、警告 | 刺激神经，引起注意 |
| 绿色 | 自然、健康、成长 | 放松、平静 |
| 紫色 | 奢华、神秘、创意 | 高贵、沉思 |
| 橙色 | 活力、温暖、友好 | 食欲、热情 |
| 黑色 | 权威、优雅、死亡 | 重量感、正式 |
| 白色 | 纯洁、干净、简约 | 开阔、空灵 |

### 1.4 质感 (Texture)

| 类型 | 定义 | 示例 |
|------|------|------|
| 触觉质感 (Tactile) | 实际可触摸的表面特征 | 粗糙、光滑、柔软、坚硬 |
| 视觉质感 (Visual) | 视觉上模拟的质感 | 照片中的木纹、布纹 |

### 1.5 空间 (Space)

| 概念 | 内容 |
|------|------|
| 正空间 (Positive Space) | 图形或主体占据的空间 |
| 负空间 (Negative Space) | 背景或空白空间 |
| 图底关系 (Figure-Ground) | 图形与背景之间的感知关系 |
| 留白 (White Space) | 中国美学中的"计白当黑" |
| 深度 (Depth) | 通过重叠、透视、阴影创造三维感 |

### 1.6 形态 (Form)

| 维度 | 特征 | 示例 |
|------|------|------|
| 二维 (2D) | 平面中的形状 | 绘画、标志 |
| 三维 (3D) | 具有体积和深度 | 雕塑、产品 |
| 真实形态 (Real) | 实际存在的三维物体 | 建筑、家具 |
| 幻觉形态 (Implied) | 通过技法创造的立体感 | 透视画法 |

## 二、设计原则 (Principles of Design)

### 2.1 平衡 (Balance)

| 类型 | 定义 | 特征 |
|------|------|------|
| 对称平衡 (Symmetrical) | 中轴线两侧完全相同 | 正式、稳定、古典 |
| 非对称平衡 (Asymmetrical) | 通过不同元素达成平衡 | 动态、有趣、现代 |
| 放射平衡 (Radial) | 从中心向外辐射 | 聚集、聚焦 |

### 2.2 对比 (Contrast)

对比通过元素间的差异创造视觉张力：

| 对比类型 | 示例 |
|----------|------|
| 大小对比 | 大-小 |
| 色彩对比 | 互补色、冷暖对比 |
| 形状对比 | 几何-有机 |
| 质感对比 | 光滑-粗糙 |
| 方向对比 | 水平-垂直 |
| 明度对比 | 黑-白 |

### 2.3 层级 (Hierarchy)

视觉层级引导观众按特定顺序阅读内容，通过以下方式建立：

- 尺寸：主要元素更大
- 色彩：高对比色彩吸引注意力
- 位置：上方和左侧通常获得优先关注
- 文字粗细：粗体标题、常规正文
- 留白：围绕重要元素留出空白

### 2.4 重复与韵律 (Repetition & Rhythm)

| 韵律类型 | 描述 |
|----------|------|
| 规则韵律 (Regular) | 等间距的重复 |
| 渐变韵律 (Gradated) | 逐渐变化大小或间距 |
| 流动韵律 (Flowing) | 自然、曲线的重复 |
| 交替韵律 (Alternating) | 两个或多个元素的交替 |

### 2.5 统一 (Unity)

格式塔心理学（Gestalt Psychology）原理：

| 原理 | 英文 | 内容 |
|------|------|------|
| 接近性 | Proximity | 接近的元素被视为一组 |
| 相似性 | Similarity | 相似的元素被视为一组 |
| 连续性 | Continuation | 视觉倾向跟随最平滑的路径 |
| 闭合性 | Closure | 倾向于将不完整的图形视为完整 |
| 对称性 | Symmetry | 对称形式被视为整体 |
| 图底 | Figure-Ground | 区分主体和背景 |

### 2.6 比例 (Proportion)

**黄金分割 (Golden Ratio)：**

- φ = 1.6180339887……
- 在自然、建筑和艺术中广泛存在
- 帕特农神庙、金字塔、《蒙娜丽莎》的比例
- 斐波那契数列近似黄金比例

其他经典比例体系：
- 人体比例（维特鲁威人）
- 模度（Le Corbusier的Modulor）
- 日本传统空间的榻榻米模数

## 三、平面设计 (Graphic Design)

### 3.1 字体设计 (Typography)

| 分类 | 英文 | 特征 | 示例 |
|------|------|------|------|
| 有衬线 | Serif | 笔画末端有装饰线，古典、正式 | Times New Roman, Garamond |
| 无衬线 | Sans-Serif | 无装饰线，简洁、现代 | Helvetica, Arial, Futura |
| 手写体 | Script | 模仿手写，优雅、个人化 | 草书风格字体 |
| 展示体 | Display | 装饰性强，适用于标题 | 各种创意字体 |
| 等宽体 | Monospace | 每个字符宽度相等 | Courier, Consolas |

**字体基本术语：**

| 术语 | 英文 | 定义 |
|------|------|------|
| 字距 | Kerning | 特定字符间的间距调整 |
| 行距 | Leading | 行与行之间的垂直间距 |
| 字距调整 | Tracking | 整体字符间距的调整 |
| 字号 | Font Size | 以点（pt）为单位 |
| 字重 | Font Weight | 粗细：Light, Regular, Bold |

### 3.2 布局 (Layout)

| 布局工具 | 内容 |
|----------|------|
| 网格系统 (Grid Systems) | 将页面分为规则的行和列 |
| 三分法 (Rule of Thirds) | 将画面分为9等份 |
| 留白 (White Space) | 空白区域的重要性 |
| 对齐 (Alignment) | 左对齐、居中、右对齐、齐行 |
| 对称 (Symmetry) | 中轴线对齐 |

### 3.3 品牌识别 (Brand Identity)

品牌识别系统包括：

1. **标志 (Logo)**：品牌的核心视觉符号
2. **色彩系统 (Color Palette)**：品牌专用色彩
3. **字体系统 (Typography)**：品牌专用字体
4. **辅助图形 (Patterns)**：品牌装饰元素
5. **应用规范 (Applications)**：名片、信纸、包装、数字媒体

## 四、设计思维 (Design Thinking)

### 4.1 设计思维五阶段

| 阶段 | 英文 | 活动 | 方法 |
|------|------|------|------|
| 共情 | Empathize | 了解用户需求 | 用户访谈、观察、移情映射 |
| 定义 | Define | 明确问题 | 问题陈述、POV（观点） |
| 构思 | Ideate | 产生创意 | 头脑风暴、思维导图、SCAMPER |
| 原型 | Prototype | 制作模型 | 纸质原型、低保真模型 |
| 测试 | Test | 验证方案 | 用户测试、A/B测试 |

### 4.2 设计伦理

1. **通用设计 (Universal Design)**：为所有人设计，包括残障人士
2. **可持续设计 (Sustainable Design)**：环保材料、可回收、减少浪费
3. **社会责任设计 (Socially Responsible Design)**：解决社会问题
4. **包容性设计 (Inclusive Design)**：考虑多样性
5. **伦理广告 (Ethical Advertising)**：诚实的视觉传达
