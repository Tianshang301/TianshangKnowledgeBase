# 绘画与雕塑

## 一、绘画媒介 (Painting Mediums)

### 1.1 油画 (Oil Painting)

油画使用植物油（亚麻仁油、核桃油等）调和的颜料，是最重要的绘画媒介之一。

| 特点 | 说明 |
|------|------|
| 慢干性 | 干燥时间长，便于混合颜色和反复修改 |
| 色彩丰富 | 色彩饱和度高，色域宽广 |
| 层次感 | 可薄涂可厚堆，能创造丰富的肌理 |
| 耐久性 | 干燥后形成坚固的有弹性的膜层 |

**油画技法：**

| 技法 | 英文 | 描述 | 应用 |
|------|------|------|------|
| 薄涂/透明叠色 | Glazing | 透明薄层叠加，创造微妙色彩 | 伦勃朗的深层光影 |
| 厚涂 | Impasto | 厚重颜料堆叠，突出笔触肌理 | 梵·高的表现力 |
| 直接画法 | Alla Prima | 一次完成，直接用混色作画 | 印象派的户外写生 |
| 刮涂 | Scumbling | 不透明薄层干扫在底层上 | 创造雾状效果 |
| 晕涂 | Sfumato | 色彩过渡柔和，轮廓模糊 | 达·芬奇的"没有线条" |

**西方油画大师：**

| 画家 | 时期 | 贡献 | 代表作 |
|------|------|------|--------|
| 扬·凡·艾克 | 早期尼德兰 | 油画技术的完善者 | 《阿尔诺芬尼夫妇像》 |
| 达·芬奇 | 文艺复兴盛期 | 晕涂法，解剖学 | 《蒙娜丽莎》 |
| 提香 | 威尼斯画派 | 色彩的解放 | 《乌尔比诺的维纳斯》 |
| 伦勃朗 | 巴洛克 | 光影大师，自画像 | 《夜巡》 |
| 维米尔 | 荷兰黄金时代 | 室内光线 | 《倒牛奶的女仆》 |
| 德拉克洛瓦 | 浪漫主义 | 色彩与激情 | 《自由引导人民》 |
| 莫奈 | 印象主义 | 外光与色彩 | 《睡莲》系列 |
| 塞尚 | 后印象派 | 现代艺术之父 | 《圣维克多山》 |
| 毕加索 | 立体主义 | 打破透视 | 《亚威农少女》 |

### 1.2 水彩 (Watercolor)

水彩以水为溶剂，颜料透明，以白纸的白色作为亮部。

| 技法 | 英文 | 描述 |
|------|------|------|
| 湿画法 | Wet-on-Wet | 湿纸上着色，颜色自然渗化 |
| 干画法 | Wet-on-Dry | 干纸上着色，边缘清晰 |
| 洗涤法 | Wash | 大面积均匀着色：平涂、渐变 |
| 干刷法 | Dry Brush | 少水干笔，制造粗糙肌理 |
| 留白 | Reserve | 保留纸面白色作为高光 |
| 吸除 | Lift-off | 用纸巾吸去颜料创造亮部 |

著名水彩画家：透纳（J.M.W. Turner，风景水彩）、萨金特（John Singer Sargent，肖像水彩）。

### 1.3 丙烯 (Acrylic)

丙烯颜料使用丙烯酸树脂作为粘合剂，是20世纪发明的现代颜料。

| 优点 | 缺点 |
|------|------|
| 快干（几分钟到几小时） | 干后颜色略微变深 |
| 干后防水、耐久 | 快干限制混色时间 |
| 可用在任何表面（纸、布、木、墙） | 干后不可逆 |
| 可稀释为水彩效果，也可厚涂为油画效果 | 调色盘需保持湿润 |

### 1.4 蛋彩 (Tempera)

蛋彩以鸡蛋黄或全蛋为粘合剂，混合颜料粉和水，是文艺复兴前的主要绘画媒介。

- 干燥速度快
- 色彩明亮、持久
- 画面细腻、有光泽
- 代表：波提切利《春》《维纳斯的诞生》

### 1.5 湿壁画 (Fresco)

湿壁画是在刚抹好的湿石灰泥上用颜料水溶液绘制的技法，分为两种：

| 类型 | 英文 | 做法 | 特点 |
|------|------|------|------|
| 真湿壁画 | Buon Fresco | 颜料与湿润的灰泥结合 | 极其耐久，色彩融入墙壁 |
| 干壁画 | Fresco Secco | 在干灰泥上绘画 | 较易剥落 |

**米开朗基罗的西斯廷教堂天顶画 (1508-1512)：**

- 面积约500平方米
- 包含343个人物
- 九幅《创世纪》场景，如《创造亚当》
- 历时四年完成，对米开朗基罗的身体造成巨大负担

### 1.6 中国水墨画 (Chinese Ink Painting)

| 分类 | 特点 | 代表人物 |
|------|------|----------|
| 工笔 | 精细勾勒，设色严谨 | 顾恺之、张萱 |
| 写意 | 笔墨酣畅，不求形似 | 徐渭、八大山人 |
| 兼工带写 | 工笔与写意结合 | 齐白石、潘天寿 |

## 二、绘画技法 (Painting Techniques)

### 2.1 晕涂法 (Sfumato)

- 意大利语"烟雾化"之意
- 色彩和明暗之间的渐进过渡，没有清晰的轮廓线
- 达·芬奇是晕涂法的集大成者
- 《蒙娜丽莎》的微笑正是得益于晕涂法的朦胧效果

### 2.2 明暗对照法 (Chiaroscuro)

- 意大利语"明-暗"之意
- 强烈的明暗对比，创造戏剧性的三维效果
- **卡拉瓦乔 (Caravaggio)**：将明暗对照法推向极致，使用戏剧性顶光
- **伦勃朗 (Rembrandt)**：善于用光塑造人物内心世界

### 2.3 暗调主义 (Tenebrism)

- 明暗对照法的极端形式
- 画面大部分处于黑暗，只有少部分被强光照亮
- 卡拉瓦乔的代表性风格

### 2.4 厚涂法 (Impasto)

- 颜料厚重堆叠，笔触清晰可见
- **梵·高 (Van Gogh)**：使用厚涂法表现情感和精神状态
- 安塞姆·基弗 (Anselm Kiefer)：厚涂材料的综合运用

### 2.5 点彩法 (Pointillism)

- 使用小色点而非传统笔触混色
- 乔治·修拉 (Georges Seurat)：创始人，《大碗岛星期天的下午》
- 基于光学混合原理——颜色在观众眼中混合

### 2.6 透视法 (Perspective)

| 类型 | 描述 | 代表 |
|------|------|------|
| 线性透视 | 平行线汇聚于消失点 | 文艺复兴绘画 |
| 大气透视 | 远处物体色彩变淡、轮廓模糊 | 达·芬奇 |
| 短缩法 | 物体沿纵深方向缩短 | 曼特尼亚《哀悼基督》 |
| 多点透视 | 多个视角的拼合 | 毕加索的立体主义 |

## 三、雕塑 (Sculpture)

### 3.1 雕刻——减法

从块状材料中去除多余部分，显露内在形态。

**大理石雕刻——米开朗基罗：**

米开朗基罗认为雕塑就是"把被囚禁在大理石中的形象解放出来"。

其杰作《大卫》（1501-1504）：
- 高5.17米，由整块卡拉拉大理石雕刻而成
- 精确的人体解剖学知识
- 动态的姿势——对立式平衡（Contrapposto）
- 象征着佛罗伦萨共和国的自由精神

**木雕：** 在中世纪哥特式教堂、中国佛教造像中广泛使用。

### 3.2 塑造——加法

通过添加柔软材料（黏土、蜡）逐步构建形体。

**奥古斯特·罗丹 (Auguste Rodin, 1840-1917)：**

| 作品 | 年份 | 特点 |
|------|------|------|
| 《思想者》 | 1880-1882 | 冥想中的力量，全身肌肉紧张 |
| 《地狱之门》 | 1880-1917 | 受但丁《神曲》启发，含200多个人物 |
| 《吻》 | 1882 | 大理石雕刻，激情与永恒 |
| 《加莱义民》 | 1889 | 英雄与牺牲，六个不同心理状态的义民 |

### 3.3 铸造——复制

通过模具复制雕塑，最常用的是失蜡法（Lost-wax Casting / Cire Perdue）。

**失蜡法工艺流程：**
1. 制作黏土原型
2. 翻制石膏模具
3. 在模具内涂蜡层
4. 用耐火材料包裹模具
5. 加热使蜡熔化流出
6. 向空腔注入熔化的金属（青铜等）
7. 冷却后敲去外层，抛光修整

### 3.4 现成物与装配 (Readymade & Assemblage)

**马塞尔·杜尚 (Marcel Duchamp, 1887-1968)：**

- 1917年《泉》（小便池，签名后作为艺术品展出）
- 挑战"什么是艺术"的根本问题
- 颠覆了传统艺术对技巧和美的要求

**毕加索的构造雕塑：** 用日常物品组装，《公牛头》（自行车座和车把）。

### 3.5 装置艺术 (Installation Art)

| 特征 | 说明 |
|------|------|
| 空间性 | 利用和改造整个空间 |
| 沉浸感 | 观众被包围在作品中 |
| 多媒介 | 综合使用声音、影像、光线、物体 |
| 互动性 | 观众参与成为作品的一部分 |

**草间弥生 (Yayoi Kusama) 的无限镜屋 (Infinity Mirror Rooms)：**

- 使用镜子、灯光和重复图案创造无限空间感
- 观众沉浸其中，体验自我消融的感觉
- 波点与无限网是她标志性的视觉语言
