# 群体过程与人际关系

## 1. 群体极化（Group Polarization）

群体极化指群体讨论使个体成员的平均初始倾向变得更加极端的现象。这一现象挑战了"群体讨论导致温和化"的传统假设。

### 1.1 从风险偏移到群体极化

最初的研究关注**风险偏移**（Risky Shift）——群体讨论使成员做出更大胆的决策。后来发现不仅是风险，任何方向的初始倾向都可能被极化。

| 初始倾向（讨论前） | 讨论后效应 | 现实案例 |
|-----------------|-----------|---------|
| 略微偏向风险 | 更加冒险 | 投资小组做出更激进的资产配置 |
| 略微偏向保守 | 更加谨慎 | 医疗委员会采用更保守的治疗方案 |
| 略微偏向某群体 | 更强烈的偏见 | 种族态度在群体讨论后更极端 |
| 对某人略有喜欢 | 更喜欢 | 陪审团讨论后对被告的态度更加明确 |

### 1.2 两种机理解释

| 机制 | 核心假设 | 实验证据 |
|------|---------|---------|
| 信息影响模型 | 讨论提供新的、偏向初始方向的信息和论点 | 成员在讨论中听到最初未想到的支持性论点 |
| 社会比较模型 | 个体比较自己与群体的位置后向"更优"方向调整 | 成员希望自己比群体平均"更正确" |

## 2. 群体思维（Groupthink）

Irving Janis（1972, 1982）在分析了珍珠港事件、猪湾入侵（Bay of Pigs Invasion）、越战升级等重大外交决策失误后，提出了群体思维理论。

### 2.1 群体思维的前提条件

| 条件类别 | 具体因素 |
|---------|---------|
| 群体结构因素 | 高群体凝聚力、群体隔离、缺乏系统性决策程序、指导型领导 |
| 情境因素 | 高强度外部压力（威胁/时间压力）、近期失败、道德困境 |

### 2.2 八种症状表现

| 症状 | 定义 | 历史案例 |
|------|------|---------|
| 无懈可击错觉 | 群体过度乐观，低估风险 | 珍珠港美军指挥官认为日军不可能进攻 |
| 集体合理化 | 群体忽视/歪曲警示信息 | 猪湾入侵前 CIA 忽视登陆困难预警 |
| 道德信念 | 群体未质疑自身决策的道德性 | 尼克松团队在水门事件中的决策 |
| 对反对者的刻板印象 | 将对方视为邪恶/愚蠢/无能 | 越战决策中对北越的刻板认知 |
| 直接压力 | 对持不同意见者施压 | "你为什么不支持团队的判断？" |
| 自我审查 | 成员压抑自己的疑虑 | "可能是我自己想多了" |
| 一致错觉 | 沉默被误解为全体同意 | 无人发言 = 全体同意 |
| 心理守护 | 阻止反对信息进入群体 | 挑战者号发射前工程师被阻止发声 |

### 2.3 预防策略

Janis 提出以下预防措施来减少群体思维的风险：

- 领导者在讨论中保持中立，不预先表达偏好
- 指定一位成员担任"魔鬼代言人"（Devil's Advocate）
- 将群体分为独立小组分别讨论
- 邀请外部专家参与并鼓励批评
- 每个成员在最终决策前进行一次"第二次机会"会议
- 在讨论初期允许匿名提交意见

## 3. 去个性化（Deindividuation）

### 3.1 Le Bon 的群体心理理论

Gustave Le Bon（1895）在《乌合之众》中首次描述了群体心理现象：
- 个体在群体中失去独立思考能力
- 匿名性和群体规模导致责任消失
- "情绪传染"使群体行为趋于极端

### 3.2 Zimbardo 的斯坦福监狱实验

Philip Zimbardo（1971）的实验最戏剧化地展示了情境和角色对行为的塑造力：

| 实验要素 | 描述 |
|---------|------|
| 被试招募 | 75 名应征者 → 24 名心理健康的男性大学生 |
| 随机分配 | 狱警和囚犯（角色随机分配） |
| 狱警行为 | 迅速内化角色，施加侮辱和虐待 |
| 囚犯反应 | 部分心理崩溃，被动服从 |
| 终止 | 原定 14 天 → 第 6 天被迫终止 |
| 伦理后果 | 引发心理学研究伦理的重大改革 |

**Zimbardo 的解释**：匿名制服 + 角色规范 + 责任分散 + 群体认同感丧失 → 去个性化行为。

### 3.3 去个性化的现代解释

| 心理过程 | 解释 | 减少去个性化的方式 |
|---------|------|-----------------|
| 匿名性 | 降低评估焦虑和个人责任感 | 使用姓名而非编号 |
| 责任分散 | "大家都在这样做" | 明确个人责任 |
| 自我意识减弱 | 注意从内部转向外部 | 提供镜子、摄像机等自省提示 |
| 群体规模 | 越大越易去个性化 | 小群体分组 |

## 4. 人际吸引

### 4.1 接近效应（Proximity）

Festinger、Schachter 和 Back（1950）对麻省理工学院西威斯汀豪斯宿舍的研究：

- 同楼层住户成为朋友的概率远高于不同楼层
- 相邻房间（近距离）成为好友的比例约为相隔较远房间的 3-4 倍
- **功能性距离**比物理距离更重要：楼梯口、信箱等场所的日常接触更为关键

| 空间距离 | 友谊概率 |
|---------|---------|
| 相邻房间 | ~40% |
| 隔一个房间 | ~22% |
| 走廊尽头 | ~10% |
| 不同楼层 | <5% |

### 4.2 相似性

| 相似维度 | 吸引效应 | 理论机制 |
|---------|---------|---------|
| 态度 | 最强的单一预测因素 | 认知一致性理论 |
| 价值观 | 长期关系的核心预测因素 | 共享世界观提供安全感 |
| 人格特质 | 结果不一致——有些研究支持匹配，有些支持互补 | 情境依赖 |
| 人口特征 | 年龄、教育、收入、宗教、种族 | 社会结构和熟悉性 |

**匹配假说（Matching Hypothesis）**：人们倾向于与相似吸引力水平的人建立浪漫关系。

### 4.3 曝光效应

Zajonc（1968）提出单纯接触即可增加好感——不需要互动、奖励或有意识觉察：

- **必要条件**：初始态度必须中性或正性（负性刺激反复曝光 → 更不喜欢）
- **跨物种验证**：人类、非人灵长类、鸟类、鼠类均有曝光效应
- **无意识曝光**：以极短时间呈现（无法有意识觉察）仍可增加偏好

### 4.4 物理吸引力

**"美即好"刻板印象（Halo Effect）**：有吸引力的人被认为更善良、聪明、社交能力强、心理健康。

| 吸引力特征 | 跨文化普遍性 | 进化解释 |
|-----------|------------|---------|
| 面部对称性 | 高 | 表明基因健康、无发育异常 |
| 平均化 | 高 | 平均脸 = 基因多样性 = 健康 |
| 腰臀比（女性 0.7） | 较高 | 表明生育力和健康 |
| 肩腰比（男性 1.4） | 较高 | 表明力量和健康 |
| 年龄信号（年轻） | 较高 | 与生育能力相关 |

## 5. 亲密关系

### 5.1 Sternberg 的爱情三角理论

Robert Sternberg（1986, 2006）提出爱情由三个基本成分组成，不同组合形成七种爱情：

| 成分 | 定义 | 发展轨迹 |
|------|------|---------|
| 激情（Passion） | 强烈生理吸引、兴奋、渴望结合 | 早期迅速上升 → 随时间自然下降 |
| 亲密（Intimacy） | 亲近、连结、分享、理解、支持 | 逐步增长，可在不同阶段维持不同水平 |
| 承诺（Commitment） | 决定爱一个人并维持关系的意愿 | 逐步增长并趋于稳定 |

**七种爱情类型矩阵**：

| 爱情类型 | 激情 | 亲密 | 承诺 |
|---------|:----:|:----:|:----:|
| 喜欢（Liking） | ✘ | ✔ | ✘ |
| 迷恋（Infatuated Love） | ✔ | ✘ | ✘ |
| 空洞之爱（Empty Love） | ✘ | ✘ | ✔ |
| 浪漫之爱（Romantic Love） | ✔ | ✔ | ✘ |
| 伴侣之爱（Companionate Love） | ✘ | ✔ | ✔ |
| 愚昧之爱（Fatuous Love） | ✔ | ✘ | ✔ |
| 完美之爱（Consummate Love） | ✔ | ✔ | ✔ |

### 5.2 Lee 的爱情颜色理论

John Alan Lee（1973）提出了六种爱情类型，类比于颜色混合：

| 类型 | 特征 | 混合成分 |
|------|------|---------|
| Eros（激情爱） | 强烈外表吸引，快速深度投入 | 基本色 |
| Ludus（游戏爱） | 不承诺、多处留情、游戏态度 | 基本色 |
| Storge（友谊爱） | 从友谊发展，温和稳定，共同兴趣 | 基本色 |
| Pragma（实用爱） | 理性条件匹配，考虑社会适合度 | Ludus + Storge |
| Agape（无私爱） | 无私奉献，不求回报，以对方幸福为优先 | Eros + Storge |
| Mania（占有爱） | 强烈嫉妒、情绪依赖、占有欲 | Eros + Ludus |

### 5.3 Gottman 的婚姻预测模型

John Gottman 在"爱情实验室"中通过 20+ 年纵向研究识别了预测离婚的关键指标：

**启示录四骑士（The Four Horsemen）**：

| 沟通模式 | 描述 | 出现顺序 | 婚姻健康评分 |
|---------|------|---------|-------------|
| 批评 | 攻击人格而非行为："你总是这么自私" | 第一步 | -1 |
| 轻蔑 | 嘲讽、辱骂、翻白眼、居高临下 | 第二步 | -4（最强预测因子） |
| 防御 | 拒绝负责、反向指责："又不是我的错！" | 第三步 | -2 |
| 筑墙 | 退出互动、沉默以对 | 第四步 | -3 |

**5:1 积极/消极比例**：成功婚姻的平均比例为 5.6:1，离婚预备组的比例为 0.8:1（积极少于消极）。

## 6. 利他行为

### 6.1 旁观者效应（Bystander Effect）

Darley 和 Latané（1968）受 1964 年 Kitty Genovese 谋杀案（38 名目击者无人报警）启发，实验证明：

| 条件 | 帮助率 | 反应时间中位数 |
|------|--------|-------------|
| 独自一人 | 85% | 约 30 秒 |
| 四人一组（三人为同盟者保持冷静） | 31% | 约 120 秒 |

### 6.2 紧急干预的五步决策模型

1. **注意**：注意到事件发生（可能被分心）
2. **解释**：判断是否为紧急情况——**多元无知**（Pluralistic Ignorance）导致不作为，因为每个人都在观察他人的反应，表现出平静以显得冷静
3. **责任**：判断自己是否有责任——**责任扩散**（Diffusion of Responsibility）：旁观者越多，个人责任感越小
4. **知道**：知道如何帮助——能力/知识
5. **行动**：实际实施帮助——社会评价恐惧（担心帮错）

### 6.3 Batson 的共情-利他假说

Daniel Batson（1991, 2011）区分了帮助的两种动机：

| 动机 | 引发条件 | 目标 | 行为特征 | 逃离可能性 |
|------|---------|------|---------|-----------|
| 共情关怀 | 感知他人需要 + 对他人的共情 | 减轻他人痛苦 | 真正的利他（帮到底） | 容易逃离也帮助 |
| 个人痛苦 | 感知他人需要 + 自己的不安 | 减轻自己的不安 | 工具性（帮完就走） | 容易逃离就不帮助 |

## 7. 攻击性

### 7.1 攻击的分类

| 维度 | 类型 | 区别 | 示例 |
|------|------|------|------|
| 动机 | 敌意性攻击 | 目标：伤害他人 | 愤怒报复 |
|  | 工具性攻击 | 目标：获取其他利益 | 抢劫、战争 |
| 形式 | 直接攻击 | 面对面 | 暴力、辱骂 |
|  | 间接攻击 | 不面对面 | 散布谣言、关系排斥 |

### 7.2 攻击的主要理论

| 理论 | 核心主张 | 重要实验/证据 |
|------|---------|-------------|
| 社会学习理论（Bandura） | 攻击通过观察和强化习得 | 波波玩偶实验——儿童模仿成人攻击行为 |
| 挫折-攻击假说（Dollard） | 挫折必然引发攻击 | 修正后：挫折→愤怒（可能攻击），但非必然 |
| 修正模型（Berkowitz） | 消极情感 + 攻击线索 | 挫折增量的愤怒 + 环境中攻击相关线索 |
| 一般攻击模型（Anderson） | 人×情境交互 | 个体因素（特质、信念）× 情境因素（线索、激惹） |

### 7.3 影响攻击的因素

| 情境因素 | 效应 | 应用 |
|---------|------|------|
| 高温 | 温度↑ → 攻击↑（热假说） | 夏季暴力犯罪率升高 |
| 酒精 | 降低抑制、缩小注意范围 | 多数暴力犯罪中涉案方有饮酒 |
| 暴力媒体 | 元分析显示暴露→攻击思想/行为↑ | 游戏分级制度 |
| 匿名性 | 去个性化→攻击↑ | 网络暴力 |
| 文化 | 荣誉文化中受辱后攻击↑ | 美国南方 vs 北方的实验 |
| 武器效应 | 武器本身即攻击线索 | Berkowitz 的枪支启动实验 |
