# 临床心理学

## 一、心理评估

### 1.1 临床访谈

临床访谈是心理评估的核心，通过面谈收集病史、症状、人格特征和功能水平等信息。访谈分为结构化、半结构化和非结构化三种形式。

| 类型 | 特征 | 优点 | 缺点 |
|------|------|------|------|
| 结构化访谈 | 标准化问题顺序和措辞 | 信度高，适合研究 | 灵活性低 |
| 半结构化访谈 | 既有框架又有探索空间 | 平衡信度和深度 | 培训要求高 |
| 非结构化访谈 | 开放式探索 | 深度挖掘 | 信度低 |

### 1.2 心理测验

**智力测验**

| 测验 | 适用年龄 | 结构 | 主要用途 |
|------|---------|------|---------|
| WAIS-IV（韦氏成人智力量表） | 16-90岁 | 言语理解、知觉推理、工作记忆、加工速度 | 智力评估、学习障碍诊断 |
| WISC-V（儿童版） | 6-16岁 | 类似 WAIS 结构 | 儿童认知能力评估 |

**人格测验**

| 测验 | 类型 | 题目 | 临床应用 |
|------|------|------|---------|
| MMPI-2（明尼苏达多相人格量表） | 自陈量表 | 567题，效度量表+临床量表 | 精神病理学评估 |
| Rorschach（罗夏墨迹测验） | 投射测验 | 10张墨迹图 | 人格结构与动力 |

**抑郁评估**

Beck 抑郁量表（BDI-II）是使用最广泛的抑郁自评工具。21个项目涵盖情感、认知和躯体症状，总分0-63分：

| 严重程度 | 总分范围 |
|---------|---------|
| 最低 | 0-13 |
| 轻度 | 14-19 |
| 中度 | 20-28 |
| 重度 | 29-63 |

## 二、主要治疗取向

### 2.1 心理动力学

心理动力学传统源自弗洛伊德，强调无意识冲突、早期经验和防御机制在心理障碍中的作用。

| 代表 | 核心概念 | 治疗技术 |
|------|---------|---------|
| Freud | 无意识、本我/自我/超我、防御机制 | 自由联想、梦的解析 |
| Jung | 集体无意识、原型 | 积极想象 |
| Klein | 客体关系、投射性认同 | 游戏治疗 |
| Kohut | 自体心理学、自恋 | 共情性回应 |
| Winnicott | 过渡客体、足够好的母亲 | 抱持性环境 |

### 2.2 认知行为疗法

**Beck 的认知疗法**

Beck 提出情绪障碍的认知模型：功能失调的信念（核心信念）→ 负性自动化思维 → 情绪和行为症状。抑郁症的"认知三联"——对自我、世界和未来的负性看法。

| 治疗技术 | 内容 |
|---------|------|
| 认知重建 | 识别并挑战负性自动化思维 |
| 行为实验 | 用行为验证负性信念 |
| 行为激活 | 增加活动水平和正性体验 |
| 苏格拉底式提问 | 引导来访者自己发现不合理信念 |

**REBT（理性情绪行为疗法）**

Ellis 的 REBT 强调情绪困扰源于非理性信念。A-B-C 模型：A（激活事件）→ B（信念）→ C（情绪和行为后果）。治疗目标是识别和驳斥非理性信念。

### 2.3 人本主义疗法

Rogers 的以人为中心疗法提出三个核心条件：

| 条件 | 含义 | 治疗作用 |
|------|------|---------|
| 共情 | 准确理解来访者的内在参考框架 | 建立信任 |
| 无条件积极关注 | 不评价地接纳来访者 | 减少防御 |
| 一致 | 治疗师真实、真诚 | 促进自我探索 |

### 2.4 系统家庭治疗

家庭系统理论将问题重新定义为家庭互动的产物而非个体的病理。主要流派：

| 流派 | 代表 | 核心概念 |
|------|------|---------|
| 结构式 | Minuchin | 边界、子系统的分化 |
| 策略式 | Haley | 症状的沟通功能、悖论干预 |
| 鲍文式 | Bowen | 分化、三角化、多代传递 |
| 叙事式 | White | 问题外化、重新叙事 |

### 2.5 治疗取向比较

| 维度 | 心理动力 | CBT | 人本主义 | 系统家庭 |
|------|---------|-----|---------|---------|
| 焦点 | 过去-现在 | 现在-未来 | 此时此地 | 互动模式 |
| 时间 | 长期 | 短期-中期 | 中期 | 短期-中期 |
| 治疗关系 | 移情分析 | 合作 | 核心条件 | 中立-卷入 |
| 改变机制 | 领悟 | 认知重构 | 自我实现 | 模式改变 |

## 三、心理障碍

### 3.1 抑郁症

| 诊断标准（DSM-5） | 说明 |
|------------------|------|
| 主要症状 | 情绪低落或兴趣丧失（至少一项） |
| 附加症状 | 体重变化、睡眠问题、疲劳、无价值感、注意力下降、自杀意念 |
| 持续时间 | 至少两周 |
| 功能损害 | 社交、职业或其他重要领域功能受损 |

### 3.2 焦虑障碍

| 类型 | 核心特征 | 患病率 |
|------|---------|--------|
| 广泛性焦虑障碍 | 过度担忧，难以控制 | 约3% |
| 惊恐障碍 | 突发性强烈恐惧，伴躯体症状 | 约2-3% |
| 社交焦虑障碍 | 害怕社交场合被评价 | 约7% |
| 特定恐惧症 | 对特定物体或情境的恐惧 | 约8-12% |

### 3.3 PTSD

创伤后应激障碍的核心症状群：

| 症状群 | 表现 |
|--------|------|
| 侵入性症状 | 闪回、噩梦、闯入性记忆 |
| 回避 | 回避与创伤相关的人、地、事 |
| 认知和情绪负性改变 | 负性信念、情感麻木 |
| 警觉与反应改变 | 过度警觉、易怒、睡眠障碍 |

### 3.4 人格障碍

| 簇 | 障碍 | 核心特征 |
|----|------|---------|
| A 簇（怪异） | 偏执型、分裂样、分裂型 | 社会孤立、认知扭曲 |
| B 簇（戏剧化） | 反社会型、边缘型、表演型、自恋型 | 情绪不稳定、冲动 |
| C 簇（焦虑） | 回避型、依赖型、强迫型 | 恐惧、焦虑、完美主义 |

## 四、循证实践与伦理

### 4.1 循证实践

循证实践（EBP）是"将最佳研究证据与临床专长和来访者特征相结合"的实践模式。

| 组成部分 | 内容 |
|---------|------|
| 研究证据 | RCT、元分析、系统综述 |
| 临床专长 | 诊断技能、治疗经验、文化胜任力 |
| 来访者特征 | 价值观、偏好、文化背景 |

### 4.2 伦理原则

| 原则 | 应用 |
|------|------|
| 知情同意 | 来访者了解治疗的性质、风险和替代方案 |
| 保密 | 保护来访者隐私，例外情况（自杀、他杀、儿童虐待） |
| 边界 | 避免双重关系、性关系绝对禁止 |
| 胜任力 | 在专业能力范围内实践 |
| 多元文化 | 尊重来访者的文化背景和身份 |

## 五、特殊人群与常见议题

### 5.1 儿童与青少年

| 障碍类型 | 核心特征 | 循证治疗 |
|---------|---------|---------|
| 注意缺陷/多动障碍（ADHD） | 注意力不集中、冲动、多动 | 行为干预+药物治疗 |
| 自闭症谱系障碍（ASD） | 社交障碍、刻板行为 | 应用行为分析、社交技能训练 |
| 品行障碍 | 持续性违反规范的行为 | 多系统治疗、父母管理训练 |
| 青少年抑郁 | 情绪低落、易怒、退缩 | 认知行为疗法、家庭治疗 |

### 5.2 物质使用障碍

物质使用障碍（成瘾）涉及大脑奖赏回路的长期改变。常见物质包括酒精、阿片类、兴奋剂和大麻。

| 干预阶段 | 目标 | 方法 |
|---------|------|------|
| 脱毒 | 安全戒除生理依赖 | 医学监测、替代药物 |
| 康复 | 预防复发、重建生活 | CBT、动机访谈、12步项目 |
| 维持 | 长期保持戒断 | 互助团体、环境干预 |

### 5.3 自杀预防

自杀是严重的公共卫生问题。风险因素包括精神障碍（特别是抑郁症和双相情感障碍）、既往自杀未遂、家族史、社会孤立和重大生活危机。

| 干预层次 | 目标人群 | 措施 |
|---------|---------|------|
| 普遍预防 | 全体人群 | 减少自杀手段可得性、媒体负责任报道 |
| 选择性预防 | 风险人群 | 筛查、心理健康教育 |
| 指示性预防 | 高危个体 | 危机热线、安全计划、治疗 |

## 六、文化与临床实践

### 6.1 文化对心理病理的影响

不同文化对心理障碍的理解和表达方式存在差异。

| 文化因素 | 影响 | 例 |
|---------|------|-----|
| 症状表达的文化差异 | 不同文化中障碍的呈现方式不同 | 东亚多躯体化表达 |
| 病因解释 | 对心理问题的归因不同 | 心理归因 vs 神秘归因 |
| 求助行为 | 寻求帮助的方式和途径 | 先看中医还是看心理师 |

### 6.2 文化适应治疗

文化适应治疗要求治疗师尊重来访者的文化背景并调整治疗方案。具体方法包括：使用来访者的母语或常用语、考虑文化中的家庭结构和角色、整合本土的治疗方法和信仰体系。

### 6.3 多元文化胜任力

| 维度 | 内容 |
|------|------|
| 意识 | 意识到自身的文化偏见和局限 |
| 知识 | 了解不同文化群体的历史、价值观和治疗需求 |
| 技能 | 能够运用文化适应的治疗干预 |
