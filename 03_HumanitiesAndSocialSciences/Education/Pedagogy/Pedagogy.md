# 教学论

## 一、教学作为专业

### 1.1 教师知识基础

舒尔曼（Lee Shulman）提出教师知识分类框架：

| 知识类型 | 英文缩写 | 内涵 | 示例 |
|---------|---------|------|------|
| 学科内容知识 | CK | 所教学科的事实、概念、原理 | 数学教师的代数知识 |
| 一般教学法知识 | PK | 超越学科的通用教学策略 | 课堂管理、提问技巧 |
| 学科教学知识 | PCK | 特定学科内容的教学表征 | 用图形解释分数概念 |
| 课程知识 | CurK | 课程材料与资源 | 教材、课程标准 |
| 学习者知识 | LK | 学生特点与学习规律 | 学生常见错误、认知发展 |
| 教育情境知识 | CxK | 学校、社区、文化背景 | 学区政策、社区资源 |
| 教育目标知识 | GK | 教育目的与价值观 | 培养目标、核心素养 |

### 1.2 学科教学知识（PCK）

PCK是区分学科专家与教学专家的关键：

$$\text{有效教学} = \text{学科知识} \times \text{教学知识}$$

PCK的核心要素：
- **学生理解**：学生对该主题的典型理解水平与常见错误
- **教学表征**：该主题最有力的教学比喻、解释、例子
- **课程知识**：该主题在课程中的位置与前后联系

## 二、教学策略

### 2.1 直接教学

直接教学（Direct Instruction）特征：

| 阶段 | 教师行为 | 学生行为 |
|------|---------|---------|
| 定向与回顾 | 明确学习目标，复习先备知识 | 激活已有知识 |
| 呈现新内容 | 清晰讲解，示范，举例 | 注意、理解 |
| 引导练习 | 提供练习机会，即时反馈 | 尝试应用 |
| 独立练习 | 布置作业，监控 | 独立完成作业 |
| 总结与扩展 | 总结要点，联系新情境 | 巩固与迁移 |

### 2.2 探究式学习

探究式学习（Inquiry-Based Learning）层次：

| 层次 | 学生自主程度 | 教师角色 |
|------|------------|---------|
| 结构化探究 | 低 | 提供问题与程序 |
| 引导式探究 | 中 | 提供问题，学生设计程序 |
| 开放式探究 | 高 | 学生自主提出问题与程序 |

科学探究基本环节：

$$\text{提出问题} \rightarrow \text{作出假设} \rightarrow \text{设计实验} \rightarrow \text{收集证据} \rightarrow \text{得出结论} \rightarrow \text{交流反思}$$

### 2.3 合作学习

合作学习（Cooperative Learning）核心要素（约翰逊兄弟）：

1. **积极互赖**（Positive Interdependence）：目标、资源、角色互赖
2. **个人责任**（Individual Accountability）：每个成员承担个人责任
3. **促进性互动**（Promotive Interaction）：相互帮助、共享资源
4. **社交技能**（Social Skills）：沟通、决策、冲突解决
5. **小组自评**（Group Processing）：反思小组合作过程

常用合作学习结构：

| 结构 | 操作方式 | 适用场景 |
|------|---------|---------|
| 思考-配对-分享 | 独立思考→两人讨论→全班分享 | 讨论开放性问题 |
| 拼图法 | 专家组学习→原组互教 | 复杂内容分解学习 |
| 小组调查 | 小组选择课题→分工研究→汇报 | 项目式学习 |
| 编号共学 | 每人负责一部分→互相测验 | 事实性知识学习 |

### 2.4 基于问题的学习（PBL）

PBL流程：

1. **呈现问题**：提供复杂、真实、结构不良的问题情境
2. **问题分析**：小组讨论，识别已知与未知
3. **自主学习**：学生针对知识缺口进行独立学习
4. **方案提出**：小组整合知识，提出解决方案
5. **反思评价**：评估学习过程与解决方案

### 2.5 案例教学

案例教学（Case-Based Learning）要素：

- **真实情境**：基于真实事件或高度仿真的情境
- **决策点**：需要分析、判断、决策的节点
- **多元视角**：可从多角度分析
- **无标准答案**：培养批判性思维

### 2.6 项目式学习（PBL）

项目式学习黄金标准（Buck Institute for Education）：

| 要素 | 内涵 |
|------|------|
| 挑战性问题 | 驱动项目的核心问题 |
| 持续探究 | 学生经历较长周期的深入探究 |
| 真实性 | 项目与现实世界相关 |
| 学生自主 | 学生有发言权和选择权 |
| 反思 | 学生反思学习过程与成果 |
| 评价与修正 | 通过反馈改进作品 |
| 公开展示 | 向真实受众展示成果 |

## 三、课堂管理

### 3.1 库宁的"警觉"（With-it-ness）

雅各布·库宁（Jacob Kounin）提出有效课堂管理的关键：

- **警觉**（With-it-ness）：教师对学生行为的全面感知
- **重叠**（Overlapping）：同时处理多项事务
- **活动流畅**（Momentum）：保持教学活动的连贯性
- **小组警觉**（Group Alerting）：让所有学生保持参与

### 3.2 德雷克斯的逻辑后果

鲁道夫·德雷克斯（Rudolf Dreikurs）提出：

| 行为目的 | 学生信念 | 教师应对 |
|---------|---------|---------|
| 寻求关注 | 只有被关注才有归属感 | 忽视不当行为，关注积极行为 |
| 争夺权力 | 只有我说了算才有归属感 | 退出权力争夺，提供选择 |
| 报复 | 只有伤害他人才有归属感 | 避免报复循环，建立信任 |
| 自暴自弃 | 只有放弃才有归属感 | 小步调成功体验，不放弃 |

### 3.3 积极行为支持（PBS）

三层干预模型：

| 层级 | 干预对象 | 策略 |
|------|---------|------|
| 第一层：普遍预防 | 全体学生 | 明确期望、教学社交技能、积极强化 |
| 第二层：针对性干预 | 风险学生 | 小组辅导、检查-连接系统 |
| 第三层：强化干预 | 高危学生 | 功能行为评估、个别化行为计划 |

## 四、促进学习的评价

### 4.1 形成性评价

形成性评价（Assessment for Learning）策略：

| 策略 | 操作 | 效果 |
|------|------|------|
| 有效反馈 | 具体、及时、可操作 | 缩小当前水平与目标差距 |
| 自我评价 | 学生对照标准评估自己 | 培养元认知能力 |
| 同伴评价 | 学生相互评价 | 发展批判性思维 |
| 提问技术 | 高质量问题、等待时间 | 促进深度思考 |

反馈的质量标准（哈蒂和廷珀利）：

$$\text{有效反馈} = \text{反馈目标} + \text{反馈内容} + \text{反馈方式} + \text{反馈时机}$$

### 4.2 总结性评价

| 类型 | 特征 | 示例 |
|------|------|------|
| 纸笔测验 | 标准化、易评分 | 期末考试 |
| 表现性评价 | 真实任务、过程可见 | 实验操作、口头报告 |
| 档案袋评价 | 长期积累、多元证据 | 学习档案袋 |
| 项目评价 | 综合成果、真实应用 | 研究报告、作品展示 |

### 4.3 评分规则（Rubric）

评分规则三要素：

1. **评价维度**：评价的具体方面（如内容、组织、语言）
2. **表现水平**：从低到高的等级描述（如优秀、良好、合格、不合格）
3. **具体描述**：每个维度、每个水平的具体行为描述

### 4.4 标准本位评分

标准本位评分（Standards-Based Grading）特征：

- **基于标准**：评分直接对应学习目标
- **分离评价**：学习成果与学习行为（如作业完成、出勤）分开评价
- **允许重试**：学生可多次尝试直至掌握
- **趋势评分**：关注学习进步趋势而非平均分

## 五、教学设计

### 5.1 教学目标设定

SMART原则：

| 字母 | 含义 | 示例 |
|------|------|------|
| S | Specific（具体） | 不说"理解"，说"能解释..." |
| M | Measurable（可测） | 可观察、可评估 |
| A | Achievable（可达） | 符合学生发展水平 |
| R | Relevant（相关） | 与课程标准、学生需求相关 |
| T | Time-bound（限时） | 明确完成时间 |

布鲁姆目标分类与动词：

| 层次 | 认知过程 | 行为动词 |
|------|---------|---------|
| 记忆 | 识别、回忆 | 列举、识别、复述 |
| 理解 | 解释、举例、分类、总结、推断、比较、说明 | 解释、概括、区分 |
| 应用 | 执行、实施 | 运用、计算、演示 |
| 分析 | 区分、组织、归因 | 分析、比较、解构 |
| 评价 | 判断、批判 | 评价、辩护、批判 |
| 创造 | 产生、计划、建构 | 设计、创作、提出 |

### 5.2 教学序列设计

有效教学序列（Gagne的九大教学事件）：

| 事件 | 教师行为 | 对应心理过程 |
|------|---------|------------|
| 1. 引起注意 | 提问、展示、变化刺激 | 警觉 |
| 2. 告知目标 | 明确学习目标 | 期望 |
| 3. 刺激回忆先备知识 | 复习、提问 | 提取长时记忆 |
| 4. 呈现刺激材料 | 讲解、示范 | 选择性知觉 |
| 5. 提供学习指导 | 提示、支架 | 编码 |
| 6. 引出行为 | 提问、练习 | 反应 |
| 7. 提供反馈 | 确认、纠正 | 强化 |
| 8. 评价行为 | 测验、观察 | 提取与概括 |
| 9. 促进保持与迁移 | 复习、变式练习 | 提取与迁移 |

### 5.3 差异化教学

差异化教学维度（Tomlinson）：

$$\text{差异化} = f(\text{内容}, \text{过程}, \text{产品}) \times \text{学习者特征}$$

| 维度 | 差异化策略 |
|------|-----------|
| 内容 | 不同难度材料、分层任务、兴趣选择 |
| 过程 | 不同学习活动、不同分组方式、不同节奏 |
| 产品 | 不同输出形式、不同评价标准 |

学习者特征考虑：
- **准备度**：学生当前知识与技能水平
- **兴趣**：学生的学习动机与偏好
- **学习档案**：学生的学习风格与优势

## 六、师生互动

### 6.1 提问技术

布鲁姆分类法与提问层次：

| 层次 | 问题类型 | 示例 |
|------|---------|------|
| 记忆 | 回忆性问题 | "什么是...？" |
| 理解 | 解释性问题 | "你能用自己的话解释...？" |
| 应用 | 应用性问题 | "如何在...情境中运用...？" |
| 分析 | 分析性问题 | "...与...有什么区别？" |
| 评价 | 评价性问题 | "你同意...观点吗？为什么？" |
| 创造 | 创造性问题 | "你能提出一个...方案吗？" |

等待时间（Wait Time）效应：

- **等待时间I**：提问后等待学生回答的时间（建议3-5秒）
- **等待时间II**：学生回答后教师再回应的时间（建议3-5秒）

延长等待时间的效果：
- 回答长度增加
- 推理证据增加
- 学生提问增加
- 参与学生增加

### 6.2 苏格拉底方法

苏格拉底式提问类型：

| 类型 | 目的 | 示例 |
|------|------|------|
| 澄清问题 | 澄清概念 | "你说的...是什么意思？" |
| 假设问题 | 检验假设 | "你的假设是什么？" |
| 证据问题 | 要求证据 | "你有什么证据支持...？" |
| 视角问题 | 考虑多元视角 | "...会怎么看这个问题？" |
| 影响问题 | 探讨后果 | "如果...会发生什么？" |
| 元问题 | 反思问题本身 | "这个问题为什么重要？" |

## 七、反思性实践

### 7.1 舍恩的反思模型

唐纳德·舍恩（Donald Schön）提出：

**行动中反思**（Reflection-in-Action）：
- 在教学过程中即时调整
- 基于情境的直觉性判断
- "与情境对话"

**对行动反思**（Reflection-on-Action）：
- 教学结束后的回顾与分析
- 系统性的经验总结
- 为未来行动提供指导

### 7.2 行动研究

行动研究循环（Kemmis & McTaggart）：

$$\text{计划} \rightarrow \text{行动} \rightarrow \text{观察} \rightarrow \text{反思} \rightarrow \text{再计划}$$

行动研究特征：
- **实践导向**：解决真实教学问题
- **参与性**：教师作为研究者
- **循环性**：持续改进的螺旋
- **公开性**：成果分享与验证

## 八、教学效能研究

### 8.1 增值模型

增值模型（Value-Added Models, VAM）：

$$\text{增值} = \text{学生实际成绩} - \text{预测成绩（基于背景与先备知识）}$$

- **优势**：控制学生背景差异，更公平评价教师贡献
- **争议**：统计方法复杂、测量误差、非随机分班

### 8.2 哈蒂的可见学习

约翰·哈蒂（John Hattie）元分析研究：

**效应量（Effect Size）解释**：

| 效应量 | 影响程度 | 示例 |
|--------|---------|------|
| d > 0.4 | 可见的积极影响 | 反馈、形成性评价、教师可信度 |
| d ≈ 0.4 | 基准效应量（一年学习进步） |  |
| d < 0.4 | 低于平均效果 | 班级规模、假期 |

**高效应量教学策略**（d > 0.6）：

| 策略 | 效应量 | 内涵 |
|------|--------|------|
| 教师集体效能 | 1.57 | 教师团队对学生能力的共同信念 |
| 自我报告成绩 | 1.33 | 学生对自己学习的主观评估 |
| 形成性评价 | 0.9 | 为学习而评价 |
| 反馈 | 0.7 | 关于学习进展的信息 |
| 师生关系 | 0.72 | 积极的师生互动 |
| 元认知策略 | 0.69 | 对思维的思维 |
| 目标与目标共识 | 0.56 | 清晰的学习目标 |
| 直接教学 | 0.59 | 系统、明确的教学 |
| 同伴辅导 | 0.55 | 学生相互辅导 |
| 合作学习 | 0.4 | 小组协作学习 |

**可见学习的核心信息**：

$$\text{可见学习} = \text{教师可见学生的学} + \text{学生可见教师的教}$$

教师应成为：
- **激活者**：激活学生作为学习者的主体性
- **评价者**：持续收集学习证据
- **反馈提供者**：提供及时、具体、可操作的反馈
- **适应性专家**：根据学生反应灵活调整教学
