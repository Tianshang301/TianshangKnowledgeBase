# 语言学习

## 一、第二语言习得理论

### 1.1 理论发展脉络

第二语言习得（Second Language Acquisition, SLA）作为独立学科始于20世纪60年代末。其发展经历了从行为主义到认知主义再到社会文化理论的范式转换。

| 时期 | 主导理论 | 核心假设 |
|------|---------|---------|
| 1950s—1960s | 行为主义（对比分析假说） | L1习惯干扰L2学习 |
| 1970s | 先天论（UG理论） | 语言习得机制（LAD） |
| 1980s | 监控模型（Krashen） | 可理解输入、情感过滤 |
| 1990s | 交互理论 | 交互修正促进习得 |
| 2000s至今 | 社会文化理论 | 学习是社会参与的过程 |

### 1.2 普遍语法（UG）理论

乔姆斯基（Chomsky）的普遍语法理论应用于SLA，核心问题：UG在二语习得中是否仍然可及？

$$\text{UG Accessibility} = f(\text{age}, \text{L1-L2 parameter setting}, \text{input})$$

| 假设 | 观点 | 证据 |
|------|------|------|
| 完全可及 | UG完整作用于L2 | 学习者能习得L1中不存在的参数 |
| 部分可及 | 仅通过L1间接可及 | L2表现与L1参数设置相关 |
| 不可及 | UG对L2无效 | 成人L2学习者达不到母语者水平 |

### 1.3 交互假设

迈克尔·朗（Michael Long）提出交互假设（Interaction Hypothesis）：

$$\text{Acquisition} \leftarrow \text{Comprehensible Input} + \text{Interactional Modification} + \text{Negotiation of Meaning}$$

意思协商（Negotiation of Meaning）过程——当交流出现障碍时，学习者通过澄清请求、确认检查、理解检查等手段修正交互，从而促进语言习得。交互假设强调"可理解输入"与"可理解输出"的辩证关系。

### 1.4 输出假设

斯温（Merrill Swain）提出输出假设（Output Hypothesis），认为输出起三个作用：

- **注意/触发功能**：输出使学习者注意到自己语言能力的不足
- **假设检验功能**：输出是学习者检验目标语言假设的方式
- **元语言反思功能**：输出促使学习者反思语言形式

### 1.5 注意假说

施密特（Richard Schmidt）的注意假说（Noticing Hypothesis）：

$$\text{Input} \xrightarrow{\text{注意（Noticing）}} \text{Intake} \xrightarrow{\text{整合}} \text{Acquisition}$$

没有注意就没有习得。但注意是否充分条件？争议：潜意识学习是否可能？

## 二、学习策略

### 2.1 策略分类（Oxford, 1990）

| 类别 | 子类 | 举例 |
|------|------|------|
| 记忆策略 | 联想、图像、关键词法 | 用关键词"ambulance→俺不能死"记单词 |
| 认知策略 | 推理、分类、笔记、概括 | 归纳语法规则、做思维导图 |
| 补偿策略 | 猜测、手势、迂回表达 | 用"the thing you cut paper with"代替scissors |
| 元认知策略 | 计划、监控、评估 | 制定学习计划、自我测试 |
| 情感策略 | 焦虑控制、自我鼓励 | 用积极自我对话减轻焦虑 |
| 社交策略 | 合作、提问、共情 | 找语伴练习对话 |

### 2.2 元认知策略

元认知（Metacognition）是"对认知的认知"，在语言学习中起关键作用：

$$\text{自我调节学习} = \text{元认知知识} + \text{元认知调控}$$

元认知知识的三个维度：**个人知识**（了解自己的学习风格）、**任务知识**（理解任务要求）、**策略知识**（知道何时使用何种策略）。

### 2.3 学习策略训练

策略训练的有效性研究表明，显性教学比隐性教学更有效。CALLA（Cognitive Academic Language Learning Approach）模型将策略训练融入学科内容教学。

## 三、克拉申的监控模型

### 3.1 五大假说

斯蒂芬·克拉申（Stephen Krashen）在20世纪70—80年代提出监控模型（Monitor Model）：

| 假说 | 核心内容 |
|------|---------|
| 习得-学习假说 | 潜意识习得 vs 有意识学习，二者功能不同 |
| 自然顺序假说 | 语法结构按可预测顺序习得 |
| 监控假说 | 学习仅作为编辑器（monitor）修正输出，需满足三个条件：时间、形式关注、规则掌握 |
| 输入假说 | i+1——输入应略高于当前水平 |
| 情感过滤假说 | 焦虑、动机、自信影响输入能否转化为习得 |

### 3.2 i+1假说

克拉申的i+1假说：

$$i + 1 = \text{当前水平} + \text{略高难度}$$

可理解输入是习得的必要且充分条件（强假设）。批评者认为i+1无法精确操作化。交互主义者补充：输入的"可理解性"本身是交互建构的，而非预先给定的。

### 3.3 情感过滤

情感过滤（Affective Filter）是阻止输入进入语言习得机制（LAD）的心理屏障：

$$\text{Acquisition} = \text{Input} \times (\text{Low Affective Filter})$$

| 情感变量 | 高过滤 | 低过滤 |
|---------|--------|--------|
| 焦虑 | 高焦虑阻碍习得 | 放松有利于习得 |
| 动机 | 工具型/融合型动机强则过滤低 | 缺乏动机过滤高 |
| 自信 | 自信者过滤低 | 自卑者过滤高 |

## 四、语言学习动机

### 4.1 动机类型

| 类型 | 定义 | 例子 |
|------|------|------|
| 融合型动机 | 认同目标语文化/群体 | 想融入英语国家社会 |
| 工具型动机 | 实用目的 | 通过考试、找工作 |
| 内在动机 | 学习本身带来的愉悦 | 喜欢学习语言 |
| 外在动机 | 外部奖惩驱动 | 父母要求、学分要求 |

### 4.2 Dörnyei的二语动机自我系统

佐尔坦·东耶（Zoltán Dörnyei）提出二语动机自我系统（L2 Motivational Self System）：

$$\text{L2 Motivation} = \text{Ideal L2 Self} + \text{Ought-to L2 Self} + \text{L2 Learning Experience}$$

- **理想二语自我**：学习者希望成为的会说二语的人（最有预测力）
- **应该二语自我**：社会期待学习者具备的二语能力
- **二语学习体验**：学习情境（教师、教材、同伴）的影响

### 4.3 动机的衰减与维持

动机不是稳定特质而是动态过程。动机衰减（demotivation）的原因包括：教师因素（权威型教学）、失败体验、教材枯燥、缺乏自主性。动机维持策略：目标设定、自我激励、学习共同体支持。

## 五、语言教学方法

### 5.1 教学方法演变

| 方法 | 时期 | 核心理念 | 教学活动 |
|------|------|---------|---------|
| 语法翻译法 | 1800s— | 翻译、语法规则 | 翻译句子、背单词 |
| 听说法 | 1940s—1960s | 句型操练、习惯养成 | 模仿、重复、替换 |
| 直接法 | 1900s | 直接用目标语教学 | 实物教学、动作演示 |
| 交际法（CLT） | 1970s— | 语言作为交际工具 | 角色扮演、信息差活动 |
| 任务型（TBLT） | 1980s— | 用任务驱动学习 | 计划旅行、解决问题 |
| 内容型（CBI） | 1990s— | 语言与内容整合 | 用英语学历史、科学 |

### 5.2 交际语言教学（CLT）

交际语言教学（Communicative Language Teaching）的核心原则：

- **交际能力**（Canale & Swain）：语法能力、社会语言能力、话语能力、策略能力
- **意义优先**：真实交际活动为中心
- **流畅与准确**：流畅优先于准确，误差被视为自然过程
- **学习者中心**：教师作为协助者

海姆斯（Dell Hymes）提出交际能力概念——语言能力不仅是语法正确，更是在何时、何地、对谁、如何说话的得体性。

### 5.3 任务型语言教学（TBLT）

任务型教学以"任务"为教学基本单位：

$$\text{任务} = \text{真实} \lor \text{教学性} \land \text{意义优先} \land \text{有交际目标}$$

任务型教学的三个阶段：**任务前**（准备主题与语言）、**任务环**（执行任务、计划报告）、**语言聚焦**（分析语言形式、练习）。Willis（1996）的TFI（Task-based Framework for Instruction）模型是经典操作框架。

## 六、语言能力标准

### 6.1 欧洲共同语言参考框架（CEFR）

CEFR（Common European Framework of Reference for Languages）是欧洲乃至全球最广泛采用的语言能力标准：

| 等级 | 描述 | 能力特征 |
|------|------|---------|
| A1 | 入门级 | 理解并使用日常表达、简单句子 |
| A2 | 基础级 | 理解常用句子，在简单场景中交流 |
| B1 | 进阶级 | 能处理旅行中的常见情况，简单描述经历 |
| B2 | 高阶级 | 能理解复杂文本，与母语者流畅交流 |
| C1 | 流利级 | 能理解长而难的文本，自如表达 |
| C2 | 精通级 | 几乎无障碍理解所见所闻，精确表达细微语义 |

### 6.2 CEFR的行动导向方法

CEFR以"行动导向"（action-oriented）为核心理念：语言使用者是社会行动者，在特定情境中完成交际任务。语言能力包括：语言能力（词汇、语法、语音）、社会语言能力（得体性）、语用能力（功能、话语组织）。

## 七、计算机辅助语言学习

### 7.1 CALL的发展阶段

| 阶段 | 时期 | 技术 | 教学法 |
|------|------|------|--------|
| 行为主义CALL | 1960s—1970s | 大型机 | 操练与练习 |
| 交际CALL | 1980s—1990s | 个人电脑 | 交际法、发现学习 |
| 整合CALL | 2000s至今 | 互联网、多媒体 | 任务型、内容型 |
| 移动学习 | 2010s至今 | 智能手机、APP | 随时随地学习 |

### 7.2 MALL（移动辅助语言学习）

移动学习的特点：个性化、情境化、微学习（micro-learning）、即时反馈。典型应用：Duolingo利用游戏化机制保持动机；Anki利用间隔重复（Spaced Repetition）优化记忆效率。

$$\text{记忆保留率} = e^{-\frac{t}{s}}$$

其中 $$s$$ 为记忆强度，间隔重复通过在记忆即将消退时重新激活，逐步增加 $$s$$。

### 7.3 语料库在语言学习中的应用

数据驱动学习（Data-Driven Learning, DDL）——学习者通过检索语料库（COCA、BNC）观察真实语言使用规律。Concordance（索引行）展示词汇的搭配模式与语法规则，促进发现式学习。

## 八、个体差异

### 8.1 年龄因素

关键期假说（Critical Period Hypothesis）：青春期后语言习得能力下降。但争议：是"关键期"还是"敏感期"？成人学习者可能在词汇、语法分析方面有优势。

### 8.2 语言学能

卡洛尔（John Carroll）的语言学能（Language Aptitude）四成分：语音编码能力、语法敏感性、归纳能力、机械记忆能力。现代学能测试（MLAT, LLAMA）预测成功率。

### 8.3 学习风格

场独立/场依赖、视觉/听觉/动觉型、分析型/整体型。匹配理论认为教学策略与学习风格匹配时效果最佳。
