# 泰语基础（Basic Thai Language）

泰语（ภาษาไทย, Phasa Thai）是泰国的官方语言，属于壮侗语系（Kra-Dai）。泰语是一种声调语言，拥有独特的辅音、元音系统和复杂的声调规则。本指南涵盖字母、声调、语法、词汇和日常会话。

## 一、字母与声调

### 1.1 基础语言特性

| 特性 | 说明 |
|------|------|
| 语系 | 壮侗语系（Kra-Dai） |
| 书写系统 | 泰文（Saw Thai），起源于高棉文 |
| 辅音 | 44 个（实际使用 42 个） |
| 元音 | 32 个（长短对立） |
| 声调 | 5 个（中平、低降、降调、高平、升调） |
| 语法 | SVO，无时态变化，无冠词 |

### 1.4 5 个声调

| 声调 | 名称 | 音高 | 符号 | 例词 (用 ข) | 含义 |
|:----:|------|:----:|:----:|:-----------:|:----:|
| **1 中平** | สามัญ (samun) | /33/ | 无 | ขา /kʰǎː/ | 腿 |
| **2 低降** | เอก (ek) | /21/ | ่ | ข่า /kʰàː/ | 高良姜 |
| **3 降调** | โท (tho) | /41/ | ้ | ข้า /kʰâː/ | 仆人/我 |
| **4 高平** | ตรี (tri) | /45/ | ๊ | ข๊า /kʰáː/ | （语气词） |
| **5 升调** | จัตวา (jattawa) | /14/ | ๋ | ข๋า /kʰǎː/ | （语气词） |

> 声调规则取决于：辅音类别（中/高/低）+ 元音长短 + 声调符号 + 音节类型（活/死）

### 1.5 发音规则

| 规则 | 说明 |
|------|------|
| **送气 vs 不送气** | ก /k/ vs ข /kʰ/, ต /t/ vs ถ /tʰ/ |
| **尾音** | 8 种：ก /k̚/, ด /t̚/, บ /p̚/, ง /ŋ/, น /n/, ม /m/, ย /j/, ว /w/ |
| **无重音** | 每个音节轻重相似，靠声调辨义 |

## 二、词序

### 2.1 SVO 语序

**ผม รัก คุณ** (我 爱 你)

**修饰语在中心词后**：รถ **สีแดง** (车 红色), นักเรียน **คนนั้น** (学生 那位)

### 2.2 基本句子结构

- **主语 + 谓语 + 宾语**：ฉัน กิน ข้าว (我 吃 饭)
- **主语 + 谓语 + 补语**：เขา เป็น หมอ (他 是 医生)
- **主语 + 形容词**：บ้าน หลังนี้ ใหญ่ (房子 这栋 大)

## 三、量词

泰语是量词语言，数词不能直接修饰名词，必须使用量词。

| 量词 | 用途 | 示例 |
|------|------|------|
| **คน** | 人 | นักเรียน **สามคน** (三个学生) |
| **ตัว** | 动物/某些物品 | แมว **สองตัว** (两只猫) |
| **เล่ม** | 书/本 | หนังสือ **ห้าเล่ม** (五本书) |
| **คัน** | 车/伞 | รถ **หนึ่งคัน** (一辆车) |
| **ใบ** | 扁平物/水果/盘 | จาน **สองใบ** (两个盘子) |
| **อัน** | 小物/件/工具 | ปากกา **สามอัน** (三支笔) |
| **ลูก** | 圆物/球/水果 | แอปเปิ้ล **สี่ลูก** (四个苹果) |
| **หลัง** | 房子/建筑 | บ้าน **สองหลัง** (两栋房子) |
| **แก้ว** | 玻璃杯 | น้ำ **หนึ่งแก้ว** (一杯水) |
| **แท่ง** | 条状物 | ดินสอ **สองแท่ง** (两支铅笔) |

> 量词必须：数词 + 量词 + 名词 或 名词 + 数词 + 量词

## 四、人称代词

| 代词 | 含义 | 性别/场合 |
|------|------|---------|
| **ผม** | 我 | 男性，礼貌 |
| **ดิฉัน / ฉัน** | 我 | 女性，礼貌/普通 |
| **หนู** | 我 | 年轻女性/女性对长辈 |
| **กระผม** | 我 | 男性，极正式 |
| **คุณ** | 你 | 中性，礼貌 |
| **เธอ** | 你/她 | 非正式 |
| **เขา** | 他/她 | 中性 |
| **มัน** | 它/他/她 | 非正式（可带贬义） |
| **เรา** | 我们 | 中性 |
| **พวกเขา** | 他们 | 复数 |

## 五、动词（无变形）

泰语没有时态、人称变形，通过时间词表示：

| 结构 | 含义 | 示例 |
|------|------|------|
| V | 现在/一般 | ผม **ไป** (我去) |
| **กำลัง** + V | 正在 | ผม **กำลังไป** (我正要去) |
| **จะ** + V | 将要 | ผม **จะไป** (我将去) |
| V + **แล้ว** | 已经 | ผม **ไปแล้ว** (我已经去了) |
| **เมื่อวานนี้** + V | 昨天 | **เมื่อวานนี้** ผมไป (昨天我去) |
| **พรุ่งนี้** + V | 明天 | **พรุ่งนี้** ผมไป (明天我去) |

## 六、否定与疑问

### 6.1 否定

| 结构 | 示例 |
|------|------|
| **ไม่** + V | **ไม่**ไป (不去) |
| **ยังไม่** + V | **ยังไม่**ไป (还没去) |
| **ไม่เคย** + V | **ไม่เคย**ไป (没去过) |

### 6.2 疑问

| 结构 | 示例 |
|------|------|
| V + **ไหม**？ | ไป**ไหม**？(去吗？) |
| V + **หรือเปล่า**？ | ไป**หรือเปล่า**？(去不去？) |
| **ใช่**...**ไหม**？ | **ใช่**เขา**ไหม**？(是他吗？) |

### 6.3 疑问词

| 词 | 含义 |
|----|------|
| **ใคร** | 谁 |
| **อะไร** | 什么 |
| **ที่ไหน** | 哪里 |
| **เมื่อไหร่** | 何时 |
| **ทำไม** | 为什么 |
| **อย่างไร** | 怎样 |
| **เท่าไหร่** | 多少钱 |
| **กี่** | 几个 |

## 七、形容词

- **位置**：在名词后 — บ้าน **หลังนั้น** **ใหญ่** (房子 那栋 大)
- **比较级**：**กว่า** (更) — **ใหญ่กว่า** (更大)
- **最高级**：**ที่สุด** (最) — **ใหญ่ที่สุด** (最大)
- **等量**：**เท่ากัน** (一样)

## 八、敬语（句末）

所有句子必须加句末敬语：

| 性别 | 肯定/陈述 | 疑问 |
|:----:|----------|------|
| 男 | **ครับ** (kráp) | **ครับ** (kráp) |
| 女 | **ค่ะ** (khâ) | **คะ** (khá) |

示例：สวัสดี**ครับ** (你好，男性), ขอบคุณ**คะ** (谢谢，女性疑问)
