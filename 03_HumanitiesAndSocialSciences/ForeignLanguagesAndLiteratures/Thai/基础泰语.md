### 5个声调

| 声调 | 名称 | 音高 | 符号 | 例词 (用 ก) |
|------|------|------|------|-----------|
| **1 中平** | สามัญ | /33/ | 无 | กา /kaː/ (乌鸦) |
| **2 低降** | เอก | /21/ | ่ | ข่า /kʰàː/ (高良姜) |
| **3 降调** | โท | /41/ | ้ | ข้า /kʰâː/ (仆人/我) |
| **4 高平** | ตรี | /45/ | ๊ | ก้า /káː/ |
| **5 升调** | จัตวา | /14/ | ๋ | ขา /kʰǎː/ (腿) |

> 声调规则取决于：辅音类别(中/高/低) + 元音长短 + 声调符号 + 音节类型(活/死)

### 发音规则

| 规则 | 说明 |
|------|------|
| **送气 vs 不送气** | ป /p/ vs พ /pʰ/, ต /t/ vs ท /tʰ/ |
| **尾音** | 仅8种：ก /k̚/, ด /t̚/, บ /p̚/, ง /ŋ/, น /n/, ม /m/, ย /j/, ว /w/ |
| **无重音** | 每个音节轻重相似，靠声调辨义 |

## 2. 词序

### SVO 语序

**ผมรักคุณ** (我 爱 你)

**修饰语在中心词后**：รถ **สีแดง** (车 红色), นักเรียน **คนนั้น** (学生 那位)

## 3. 量词

| 量词 | 用途 | 示例 |
|------|------|------|
| **คน** | 人 | นักเรียน **สามคน** (三个学生) |
| **ตัว** | 动物/某些物品 | แมว **สองตัว** (两只猫) |
| **เล่ม** | 书/本 | หนังสือ **ห้าเล่ม** (五本书) |
| **คัน** | 车/伞 | รถ **หนึ่งคัน** (一辆车) |
| **ใบ** | 扁平物/水果/碗 | จาน **สองใบ** (两个盘子) |
| **อัน** | 小物/笔/工具 | ปากกา **สามอัน** (三支笔) |
| **ลูก** | 圆物/球/水果 | แอปเปิ้ล **สี่ลูก** (四个苹果) |
| **หลัง** | 房子/建筑 | บ้าน **สองหลัง** (两栋房子) |
| **แก้ว** | 玻璃杯 | น้ำ **หนึ่งแก้ว** (一杯水) |
| **แท่ง** | 条状物 | ดินสอ **สองแท่ง** (两支铅笔) |

> 量词必须有：数词 + 量词 + 名词 或 名词 + 数词 + 量词

## 5. 动词 (无变形)

泰语没有时态/人称变形，通过时间词表示：

| 结构 | 含义 | 示例 |
|------|------|------|
| V | 现在/一般 | ฉัน **ไป** (我去) |
| **กำลัง** + V | 正在 | ฉัน **กำลังไป** (我正在去) |
| **จะ** + V | 将要 | ฉัน **จะไป** (我將去) |
| V + **แล้ว** | 已经 | ฉัน **ไปแล้ว** (我已经去了) |
| **เมื่อวาน** + V | 昨天 | **เมื่อวาน** ฉันไป (昨天我去了) |
| **พรุ่งนี้** + V | 明天 | **พรุ่งนี้** ฉันไป (明天我去) |

## 6. 否定与疑问

### 否定

| 结构 | 示例 |
|------|------|
| **ไม่** + V | **ไม่**ไป (不去) |
| **ยังไม่** + V | **ยังไม่**ไป (还没去) |
| **ไม่เคย** + V | **ไม่เคย**ไป (没去过) |

### 疑问

| 结构 | 示例 |
|------|------|
| V + **ไหม**? | ไป**ไหม**? (去吗？) |
| V + **หรือเปล่า**? | ไป**หรือเปล่า**? (去不去？) |
| **ใช่...ไหม**? | **ใช่**เขา**ไหม**? (是他吗？) |

### 疑问词

| 词 | 含义 |
|----|------|
| **ใคร** | 谁 |
| **อะไร** | 什么 |
| **ที่ไหน** | 哪里 |
| **เมื่อไหร่** | 何时 |
| **ทำไม** | 为什么 |
| **อย่างไร** | 怎样 |
| **เท่าไหร่** | 多少钱 |
| **กี่** | 几个 |

## 7. 形容词

- 位置：在名词后 — บ้าน **หลัง** **ใหญ่** (房子 大的)
- 比较级：**กว่า** (更) — ใหญ่**กว่า** (更大)
- 最高级：**ที่สุด** (最) — ใหญ่**ที่สุด** (最大)
- 等量：**เท่ากัน** (一样)

## 8. 敬语 (句末)

所有句子必须加句末敬语：

| 性别 | 肯定/陈述 | 疑问 |
|------|----------|------|
| 男 | **ครับ** (kráp) | **ครับ** (kráp) |
| 女 | **ค่ะ** (khâ) | **คะ** (khá) |

示例：สวัสดี**ครับ** (你好，男性), ขอบคุณ**คะ** (谢谢，女性疑问)
