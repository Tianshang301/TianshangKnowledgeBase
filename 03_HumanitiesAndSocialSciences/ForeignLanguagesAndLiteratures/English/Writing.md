# 英语写作指南

## 文章结构 (Essay Structure)

### 引言 (Introduction)

```
┌─────────────────────────────────────┐
│  Hook (钩子 - 吸引读者)              │
│  - 提问: Have you ever wondered...?  │
│  - 数据: According to recent studies │
│  - 引用: As Einstein once said...    │
├─────────────────────────────────────┤
│  Background (背景信息)               │
│  - 提供必要的上下文                  │
│  - 定义关键术语                      │
├─────────────────────────────────────┤
│  Thesis Statement (论点)             │
│  - 文章核心观点                      │
│  - 包含主要论点预览                  │
└─────────────────────────────────────┘
```

**示例引言**:
> Have you ever wondered why some students excel while others struggle? Education researchers have long debated the role of class size in academic performance. **This essay argues that smaller class sizes significantly improve student outcomes by enabling personalized instruction, increasing student engagement, and reducing disciplinary issues.**

### 主体段落 (Body Paragraphs)

**PEEL 结构**:

| 要素 | 说明 | 示例 |
|------|------|------|
| P (Point) | 主题句 | Firstly, smaller classes allow for personalized instruction. |
| E (Evidence) | 论据 | A 2023 study found that students in classes of 15 or fewer scored 20% higher on standardized tests. |
| E (Explanation) | 解释 | This is because teachers can identify individual weaknesses and adapt their teaching methods accordingly. |
| L (Link) | 连接 | Therefore, personalized instruction is a key mechanism through which small class sizes boost achievement. |

### 结论 (Conclusion)

```
┌─────────────────────────────────────┐
│  Restate Thesis (重述论点)           │
│  - 换种说法重申论文观点              │
├─────────────────────────────────────┤
│  Summarize Key Points (总结要点)     │
│  - 简要回顾主体论点                  │
├─────────────────────────────────────┤
│  Final Thought (总结)               │
│  - 展望未来 / 提出建议 / 发人深省    │
└─────────────────────────────────────┘
```

## 文章类型

### 议论文 (Argumentative/Persuasive)

- **目的**: 说服读者接受某一立场
- **结构**: 引言 + 论点1 + 论点2 + 反驳 + 结论
- **语言**: 强烈主张 (must, should, undoubtedly)

### 说明文 (Expository)

- **目的**: 解释或说明一个主题
- **结构**: 引言 + 分点说明 + 结论
- **语言**: 客观中立 (can be defined as, consists of)

### 描述文 (Descriptive)

- **目的**: 生动描述人、物、场景
- **技巧**: 感官细节 (sight, sound, smell, touch, taste)
- **语言**: 比喻、形容词、副词

### 记叙文 (Narrative)

- **目的**: 讲述一个故事
- **结构**: 开端 → 发展 → 高潮 → 结局
- **语言**: 第一人称、过去时、对话

## 衔接与连贯 (Coherence & Cohesion)

### 过渡词 (Transition Words)

| 功能 | 过渡词 |
|------|--------|
| 添加信息 | furthermore, moreover, in addition, also, besides |
| 对比/转折 | however, nevertheless, on the other hand, conversely, although, whereas |
| 因果 | therefore, consequently, as a result, thus, hence, because of |
| 顺序 | firstly, secondly, finally, next, subsequently, meanwhile |
| 举例 | for example, for instance, such as, namely, in particular |
| 总结 | in conclusion, to summarize, in summary, overall |
| 让步 | although, even though, despite, in spite of |

### 指代 (Referencing)

``` 
The government introduced new policies. **These** were designed to...
**Such** measures have proven effective.
**This** approach differs from the previous one.
```

### 替代 (Substitution)

``` 
Some people prefer cities. Others prefer the countryside. (用 "Others" 替代 "Other people")
I like classical music, and my brother does too. (用 "does" 替代 "likes")
```

## 学术写作风格

### 正式语言

| 非正式 | 正式 |
|--------|------|
| a lot of | a significant number of |
| kids | children |
| stuff | materials / matters |
| get better | improve |
| go up | increase |
| look at | examine |
| show | demonstrate / illustrate |
| a bit | slightly / somewhat |
| really | extremely / highly |

### 模糊限制语 (Hedging)

学术写作需谨慎，避免绝对化表述：

```
不清: Climate change is destroying the planet.
准确: Climate change may be contributing to environmental degradation.

不清: This proves that...
准确: This suggests that... / This provides evidence that...
```

### 引用 (Citation)

**APA 格式**:
```
文内： (Author, Year) 或 Author (Year) stated that...
参考文献： Author, A. A. (Year). Title. Publisher.
```

**MLA 格式**:
```
文内： (Author Page) 或 Author claims... (Page)
参考文献： Author, Name. Title. Publisher, Year.
```

## 常见语法错误

### 主谓一致 (Subject-Verb Agreement)

```
错误: The list of items are on the table.
正确: The list of items is on the table.

错误: Each of the students have a book.
正确: Each of the students has a book.

错误: Neither the teacher nor the students was present.
正确: Neither the teacher nor the students were present. (就近原则)
```

### 逗号拼接 (Comma Splice)

```
错误: I went to the store, I bought some milk.
正确: I went to the store, and I bought some milk.
正确: I went to the store. I bought some milk.
正确: I went to the store; I bought some milk.
```

### 悬垂修饰语 (Dangling Modifier)

```
错误: Walking to school, the rain started.
正确: Walking to school, I felt the rain start.
正确: As I was walking to school, the rain started.

错误: To improve your writing, practice is necessary.
正确: To improve your writing, you need to practice.
```

### 代词指代不清

```
模糊: When John talked to Tom, he was angry. (谁是 "he"?)
清晰: When John talked to Tom, John was angry.
清晰: When John talked to Tom, Tom was angry.
```

## 电子邮件写作

### 正式邮件结构

```
Subject: Inquiry Regarding Course Registration

Dear Professor Smith,

I hope this email finds you well. My name is Li Wei, and I am a sophomore in your Introduction to Economics class.

I am writing to inquire about the possibility of registering for your advanced seminar next semester, as I have completed all prerequisites.

[正文内容]

Thank you for your time and consideration. I look forward to your reply.

Best regards,
Li Wei
Student ID: 2024001
```

### 常用邮件短语

| 功能 | 正式 | 非正式 |
|------|------|--------|
| 开头 | I hope this email finds you well | Hi / Hello |
| 目的 | I am writing to... | Just wanted to... |
| 请求 | I would be grateful if you could... | Could you please... |
| 附件 | Please find attached... | I've attached... |
| 截止 | I look forward to hearing from you | Let me know |
| 签名 | Best regards, / Sincerely, | Cheers, / Thanks, |

## 修改与润色清单 (Revision Checklist)

- [ ] 是否有明确的中心论点 (Thesis)?
- [ ] 每个段落是否有主题句 (Topic sentence)?
- [ ] 论点是否有充分的证据支持?
- [ ] 文章结构是否逻辑连贯?
- [ ] 过渡词使用是否恰当?
- [ ] 是否存在语法错误?
- [ ] 词汇是否准确多样?
- [ ] 是否有拼写/标点错误?
- [ ] 是否符合指定的格式要求?
