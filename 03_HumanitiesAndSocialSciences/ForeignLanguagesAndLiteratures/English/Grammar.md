# 英语语法详解

## 词性 (Parts of Speech)

| 词性 | 英文 | 功能 | 示例 |
|------|------|------|------|
| 名词 | Noun | 表示人、事物、概念 | dog, China, happiness |
| 动词 | Verb | 表示动作或状态 | run, eat, be |
| 形容词 | Adjective | 修饰名词 | beautiful, tall, happy |
| 副词 | Adverb | 修饰动词/形容词/其他副词 | quickly, very, well |
| 代词 | Pronoun | 代替名词 | I, you, he, they, it |
| 介词 | Preposition | 表示关系 | in, on, at, for |
| 连词 | Conjunction | 连接词/句 | and, but, because |
| 感叹词 | Interjection | 表达情感 | Oh!, Wow!, Ouch! |
| 冠词 | Article | 限定名词 | a, an, the |

## 时态 (Tenses)

### 12 种时态总览

| 时态 | 结构 | 示例 | 用法 |
|------|------|------|------|
| 一般现在 | V / V-s | I eat. | 习惯、事实、常理 |
| 现在进行 | am/is/are + V-ing | I am eating. | 正在发生 |
| 现在完成 | have/has + V-p.p. | I have eaten. | 过去到现在的经历/影响 |
| 现在完成进行 | have/has + been + V-ing | I have been eating. | 持续到现在的动作 |
| 一般过去 | V-ed | I ate. | 过去完成 |
| 过去进行 | was/were + V-ing | I was eating. | 过去某一刻正在发生 |
| 过去完成 | had + V-p.p. | I had eaten. | 过去的过去 |
| 过去完成进行 | had been + V-ing | I had been eating. | 持续到过去某时刻 |
| 一般将来 | will + V | I will eat. | 将会发生 |
| 将来进行 | will be + V-ing | I will be eating. | 将来某刻正在发生 |
| 将来完成 | will have + V-p.p. | I will have eaten. | 将来某刻已完成 |
| 将来完成进行 | will have been + V-ing | I will have been eating. | 持续到将来某刻 |

### 时态选择要点

- **一般现在时**: 客观真理 (Water boils at 100°C)、习惯 (I run every day)
- **现在完成时**: 经历 (I have been to Japan)、结果 (He has broken his leg)
- **现在完成进行时**: 强调持续 (She has been studying for 3 hours)
- **过去完成时**: 必须有另一个过去动作参照 (When I arrived, he had left)
- **一般将来时**: will + V (临时决定), going to (计划)

## 被动语态 (Passive Voice)

### 结构

```
be + 过去分词 (by + 宾语)
```

| 时态 | 主动 | 被动 |
|------|------|------|
| 一般现在 | write | is/are written |
| 一般过去 | wrote | was/were written |
| 一般将来 | will write | will be written |
| 现在进行 | is writing | is being written |
| 过去进行 | was writing | was being written |
| 现在完成 | has written | has been written |
| 过去完成 | had written | had been written |
| 情态动词 | can write | can be written |

### 何时使用被动

- 动作执行者不重要或未知: My bike was stolen.
- 强调动作承受者: The president was elected.
- 正式/学术写作: It can be concluded that...

### 双宾语动词的被动

```
主动: Someone gave me a gift.
被动1: I was given a gift (by someone).
被动2: A gift was given to me (by someone).
```

## 条件句 (Conditionals)

| 类型 | if 从句 | 主句 | 示例 |
|------|---------|------|------|
| 零条件 (事实) | 一般现在 | 一般现在 | If you heat ice, it melts. |
| 第一条件 (可能) | 一般现在 | will + V | If it rains, I will stay home. |
| 第二条件 (虚拟) | 一般过去 | would + V | If I were rich, I would travel. |
| 第三条件 (过去虚拟) | 过去完成 | would have + V.p.p. | If I had known, I would have come. |
| 混合条件 | 过去完成 | would + V | If I had studied, I would be a doctor now. |

### 其他条件表达

```
unless = if not: I won't go unless you come.
as long as: You can stay as long as you behave.
provided (that): Provided it doesn't rain, we'll go.
in case: Take an umbrella in case it rains.
even if: I'll do it even if it's hard.
```

## 间接引语 (Reported Speech)

### 时态回调 (Backshift)

| 直接引语 | 间接引语 |
|---------|---------|
| 一般现在 → | 一般过去 |
| 现在进行 → | 过去进行 |
| 现在完成 → | 过去完成 |
| 一般过去 → | 过去完成 |
| will → | would |
| can → | could |
| may → | might |
| must → | had to |

### 示例

```
直接: "I am tired," she said.
间接: She said she was tired.

直接: "I will call you," he said.
间接: He said he would call me.

直接: "Have you seen this movie?" she asked.
间接: She asked if I had seen that movie.

直接: "Go away!" he shouted.
间接: He told me to go away.
```

### 不回调的情况

- 真理: He said that water boils at 100°C.
- 仍成立: She said that she lives in Beijing. (现在仍住在北京)

## 关系从句 (Relative Clauses)

### 限定性 vs 非限定性

| 类型 | 用法 | 逗号 | 关系代词 |
|------|------|------|---------|
| 限定性 | 必须信息，不可省略 | 无 | who/which/that |
| 非限定性 | 附加说明，可省略 | 有 | who/which (不能用 that) |

### 关系代词选择

| 先行词 | 主语 | 宾语 | 所属 |
|--------|------|------|------|
| 人 | who/that | who(m)/that | whose |
| 物 | which/that | which/that | whose/of which |
| 时间 | when | when | - |
| 地点 | where | where | - |
| 原因 | why | why | - |

### 示例

```
限定性: The man who lives next door is a doctor.
非限定性: My brother, who lives in Shanghai, is a doctor.

关系代词省略: The book (that) I read was good. (宾语可省)
关系代词保留: The book that I read was good. (主语不可省)
```

## 动名词与不定式

### 动词 + 动名词 (V-ing)

```
enjoy, finish, avoid, suggest, mind, practice, miss, consider
```

### 动词 + 不定式 (to V)

```
want, hope, expect, decide, plan, promise, agree, refuse
```

### 两者皆可但意义不同

| 动词 | V-ing | to V |
|------|-------|------|
| stop | 停止正在做的事 | 停下来去做 |
| remember | 记得做过 | 记得要去做 |
| forget | 忘记做过 | 忘记要去做 |
| try | 尝试做 (试验) | 努力做 (尽力) |
| regret | 后悔做过 | 遗憾地告知 |

### go + 动名词

```
go shopping, go swimming, go running, go fishing
```

## 情态动词 (Modal Verbs)

| 情态动词 | 用法 | 过去式 |
|---------|------|--------|
| can | 能力、许可 | could |
| could | 能力 (过去)、礼貌请求 | - |
| may | 许可、可能性 (正式) | might |
| might | 可能性 (较小) | - |
| must | 必须 (主观) | had to |
| have to | 必须 (客观) | had to |
| shall | 提议、征询意见 (英式) | should |
| should | 建议、应该 | should have |
| would | 意愿、习惯、虚拟 | - |
| ought to | 应该 (正式) | ought to have |
| need | 需要 | needed to |
| dare | 敢于 | dared to |
| used to | 过去习惯 | - |

### 程度确定性

```
100%: must / can't (一定/不可能)
 90%: should / ought to (很可能)
 50%: may / might / could (可能)
  0%: can't (不可能)
```

### 情态动词 + have + V.p.p.

```
must have done: 过去一定做了 (推测)
may/might have done: 过去可能做了
could have done: 原本可以做 (但没做)
should have done: 本应该做 (但没做)
needn't have done: 本不必做 (却做了)
```
