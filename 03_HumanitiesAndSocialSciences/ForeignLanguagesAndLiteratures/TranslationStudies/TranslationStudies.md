# 翻译学

## 一、翻译学的学科定位

### 1.1 翻译学的发展

翻译学（Translation Studies）作为独立学科始于20世纪后半叶。霍姆斯（James Holmes）在1972年发表的《翻译学的名与实》中提出翻译学学科结构框架，被视为学科成立的标志。

$$\text{翻译学} = \text{纯翻译学} + \text{应用翻译学}$$

| 分支 | 子领域 | 研究对象 |
|------|--------|---------|
| 理论翻译学 | 通论、媒介论、层级论 | 翻译的普遍规律 |
| 描写翻译学 | 产品导向、过程导向、功能导向 | 实际翻译现象 |
| 应用翻译学 | 教学、测试、批评、工具 | 实践应用 |

### 1.2 翻译学的跨学科性质

| 相关学科 | 贡献 |
|---------|------|
| 语言学 | 语义对等、语用分析、语篇分析 |
| 文学研究 | 文学翻译的审美与文化层面 |
| 文化研究 | 翻译中的权力关系和身份政治 |
| 认知科学 | 翻译过程与译者心理机制 |
| 计算机科学 | 机器翻译与CAT工具 |

## 二、翻译理论

### 2.1 奈达的功能对等

尤金·奈达（Eugene Nida）借鉴乔姆斯基的转换生成语法，提出功能对等（Functional Equivalence）：

$$\text{翻译} = \text{源语表层结构} \rightarrow \text{深层结构（核心句）} \rightarrow \text{译语表层结构}$$

| 类型 | 定义 | 侧重点 |
|------|------|--------|
| 形式对等 | 逐字对应目标语形式 | 源语结构 |
| 动态/功能对等 | 译语读者反应≈源语读者反应 | 交际效果 |

奈达以"最贴切的自然对等"（closest natural equivalent）定义翻译目标——"自然"指向译语习惯，"对等"指向信息保真，"最贴切"指向最大限度地忠实。

### 2.2 目的论（Skopos Theory）

汉斯·弗米尔（Hans Vermeer）提出目的论，视翻译为有目的的行为：

$$\text{Skopos Rule: 翻译策略由译文目的决定}$$

| 原则 | 内容 |
|------|------|
| 目的原则 | 翻译行为的目的决定翻译策略 |
| 连贯原则（语内连贯） | 译文必须对目标受众有意义 |
| 忠实原则（语际连贯） | 译文与源文之间保持忠实关系 |

功能学派代表人物还包括赖斯（Katharina Reiss）——文本类型理论；诺德（Christiane Nord）——"功能加忠诚"（function plus loyalty），在功能主义与伦理义务之间取得平衡。

### 2.3 描写翻译学（DTS）

吉迪恩·图里（Gideon Toury）提出描写翻译学（Descriptive Translation Studies），将翻译研究从"应该怎样翻译"的规范视角转向"实际怎样翻译"的描写视角：

$$\text{翻译规范（Translation Norms）} = f(\text{社会文化语境})$$

| 规范类型 | 阶段 | 功能 |
|---------|------|------|
| 预备规范 | 翻译行为前 | 决定选择什么文本翻译 |
| 初始规范 | 翻译决策时 | 偏向源语（adequacy）vs 偏向译语（acceptability） |
| 操作规范 | 翻译过程中 | 决定具体翻译策略（增删、分段、改写） |

### 2.4 其他重要理论

| 理论 | 提出者 | 核心观点 |
|------|--------|---------|
| 多元系统论 | 埃文-佐哈尔 | 翻译文学在目标文化文学系统中的位置（中心/边缘） |
| 操纵学派 | 勒菲弗尔 | 翻译是对原文的改写，受意识形态与诗学支配 |
| 后殖民翻译理论 | 尼兰贾娜、斯皮瓦克 | 翻译是殖民权力的工具或抵抗的空间 |
| 女性主义翻译 | 西蒙、冯·弗洛托 | 翻译中的性别政治，女性主义翻译策略 |

## 三、翻译类型与策略

### 3.1 直译与意译

直译（Literal Translation）与意译（Free Translation）是翻译策略的基本二元对立：

| 维度 | 直译 | 意译 |
|------|------|------|
| 取向 | 源语导向 | 目的语导向 |
| 形式 | 保留原文语法结构 | 重写为自然的译语形式 |
| 适用 | 法律合同、科技文献 | 文学、广告、歌词 |
| 风险 | 生硬翻译腔 | 过度归化失原意 |

### 3.2 归化与异化

劳伦斯·韦努蒂（Lawrence Venuti）提出归化（Domestication）与异化（Foreignization）：

$$\text{归化} = \text{使译文流畅自然，抹去异域痕迹}$$
$$\text{异化} = \text{保留异域特征，抵抗目标文化主流价值观}$$

韦努蒂从后殖民视角批判英语世界的"隐形翻译"（invisibility of the translator），倡导异化策略以抵抗英美文化霸权。归化与异化的选择不仅是语言策略，更是政治立场。

### 3.3 翻译方法谱系

| 方法 | 特征 | 应用 |
|------|------|------|
| 逐词翻译 | 单词对应 | 计算机辅助翻译 |
| 字面翻译 | 语法结构调整 | 技术文档 |
| 忠实翻译 | 再现原文意图 | 学术文本 |
| 语义翻译 | 兼顾源语审美 | 文学翻译 |
| 交际翻译 | 读者理解优先 | 新闻翻译 |
| 地道翻译 | 完全归化 | 广告、儿童文学 |
| 编译 | 选择性重组 | 摘要翻译 |

## 四、口译

### 4.1 口译类型

| 类型 | 操作方式 | 特点 | 应用场景 |
|------|---------|------|---------|
| 交替传译（CI） | 讲者说完，译员翻译 | 准确度高，费时 | 商务会谈、小型会议 |
| 同声传译（SI） | 讲者讲的同时译 | 实时性，高认知负荷 | 国际会议（联合国、欧盟） |
| 耳语传译 | 低声为个别听众翻译 | 不干扰会议 | 小范围 |
| 联络/陪同口译 | 双向对话翻译 | 灵活、现场性强 | 外交接待、商务陪同 |

### 4.2 口译过程模型

吉尔（Daniel Gile）的认知负荷模型（Effort Models）：

$$\text{SI} = L + M + P + C$$
$$\text{CI} = L + N + (\text{Note-taking}) + M + P$$

其中 $L$ = 听力与分析，$M$ = 短期记忆，$P$ = 产出，$C$ = 协调，$N$ = 笔记。

吉尔模型说明口译的认知资源是有限的，各项任务之间竞争注意力。当总需求超过可用容量时，口译质量下降。

### 4.3 口译技巧

| 技巧 | 内容 |
|------|------|
| 预测 | 根据语境预判讲者接下来的话 |
| 断句 | 将长句切分为可处理的单位 |
| 释义 | 用同义表达替代 |
| 概括 | 压缩冗余信息 |
| 延迟反应 | 当遇到难点时推迟翻译等待更多语境 |

## 五、机器翻译

### 5.1 机器翻译的发展

| 阶段 | 技术 | 代表 | 特征 |
|------|------|------|------|
| 规则机器翻译（RBMT） | 语法规则+词典 | SYSTRAN | 依赖专家规则，质量有限 |
| 统计机器翻译（SMT） | 平行语料统计模型 | Google Translate（早期） | 基于短语翻译概率 |
| 神经机器翻译（NMT） | 深度学习、注意力机制 | Google NMT（2016），DeepL | 端到端，流畅度高 |

$$P(target|source) = \prod_{i=1}^{n} P(t_i|source, t_1...t_{i-1})$$

NMT的核心架构——编码器-解码器（Encoder-Decoder）加上注意力机制（Attention Mechanism），使模型在翻译每个词时能"关注"源句的不同部分。

### 5.2 机器翻译的局限

| 局限 | 表现 |
|------|------|
| 语境理解不足 | 多义词、指代消解错误 |
| 文化负载词 | 成语、典故、幽默的翻译失真 |
| 长句处理 | 复杂句法结构下的流畅度下降 |
| 语体识别 | 正式/非正式语体转换不当 |
| 创造性翻译 | 文学隐喻、双关语的损失 |

### 5.3 译后编辑（Post-Editing）

机器翻译+人工编辑（MTPE, Machine Translation Post-Editing）成为行业标准工作流程。译后编辑分为：

- **轻编辑**（Light PE）：修正明显错误，保证可理解
- **全编辑**（Full PE）：达到人工翻译质量水平

| 工具 | 开发者 | 核心功能 | 特点 |

### 6.2 翻译记忆与术语管理

翻译记忆（Translation Memory, TM）的工作原理：

$$\text{Match} = f(\text{Source Segment}, \text{TM Database}) \rightarrow \text{Target Segment}$$

| 匹配类型 | 定义 | 折扣率 |
|---------|------|--------|
| 完全匹配（100%） | 源句完全一致 | 50%—70% |
| 模糊匹配（>75%） | 源句部分一致 | 30%—50% |
| 术语匹配 | 术语库命中 | 固定折扣 |

### 6.3 翻译质量管理

| 工具/方法 | 功能 |
|-----------|------|
| QA Checker | 自动检查数字、格式、术语一致性 |
| 双语审校（Bilingual Review） | 对照原文检查准确性 |
| 单语审校（Monolingual Review） | 只读译文检查流畅度 |
| LQA（语言质量评估） | 按错误类型分级打分 |

## 七、翻译质量评估（TQA）

### 7.1 BLEU评分

BLEU（Bilingual Evaluation Understudy）是机器翻译最广泛使用的自动评估指标：

$$\text{BLEU} = \text{BP} \times \exp\left(\sum_{n=1}^{N} w_n \log p_n\right)$$

其中 $p_n$ 为n-gram精确率，BP为简短惩罚因子。BLEU的优点是快捷、客观，缺点是不考虑语义和语法。

### 7.2 人工评估标准

| 评估维度 | 内容 |
|---------|------|
| 准确性 | 信息是否完整准确传递 |
| 流畅度 | 译文是否自然通顺 |
| 术语一致性 | 专业术语是否统一 |
| 语体适当性 | 译文的正式程度是否合适 |
| 文化适配 | 文化元素是否恰当转换 |
