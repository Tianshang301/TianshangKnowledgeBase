# 法语基础

## 1. 字母与发音

### 字母表 (26个拉丁字母)

Aa Bb Cc Dd Ee Ff Gg Hh Ii Jj Kk Ll Mm Nn Oo Pp Qq Rr Ss Tt Uu Vv Ww Xx Yy Zz

### 重音符号

| 符号 | 名称 | 作用 | 示例 |
|------|------|------|------|
| é | accent aigu | 闭音 /e/ | été, café |
| è | accent grave | 开音 /ɛ/ | mère, très |
| ê | accent circonflexe | /ɛ/，常表示省去的 s | tête (←teste) |
| ë | tréma | 与前一字母分开发音 | Noël, naïve |
| ç | cédille | s 音 | français, garçon |

### 元音

| 音素 | 字母 | 示例 |
|------|------|------|
| /a/ | a | chat /ʃa/ |
| /e/ | é, -er, -ez | été /ete/, parler /paʁle/ |
| /ɛ/ | è, ê, ei, ai | mère /mɛʁ/, neige /nɛʒ/ |
| /i/ | i, y | lit /li/ |
| /o/ | o, ô, au, eau | mot /mo/, beau /bo/ |
| /u/ | ou | tout /tu/ |
| /y/ | u | tu /ty/ |
| /ø/ | eu, œu 在开音节 | feu /fø/ |
| /œ/ | eu, œu 在闭音节 | fleur /flœʁ/ |
| /ə/ | e 在单音节或词中 | le /lə/, regarder /ʁəɡaʁde/ |

### 鼻化元音

| 音素 | 拼写 | 示例 |
|------|------|------|
| /ɑ̃/ | an, am, en, em | enfant /ɑ̃fɑ̃/ |
| /ɛ̃/ | in, im, ain, aim, ein, yn | vin /vɛ̃/ |
| /ɔ̃/ | on, om | bon /bɔ̃/ |
| /œ̃/ | un, um | brun /bʁœ̃/ |

### 辅音

| 音素 | 说明 | 示例 |
|------|------|------|
| /ʁ/ | 小舌颤音/擦音 | paris /paʁi/, rouge /ʁuʒ/ |
| /ɲ/ | 腭鼻音 gn | montagne /mɔ̃taɲ/ |
| /ʃ/ | ch | chat /ʃa/ |
| /ʒ/ | j, g 在 e/i/y 前 | jour /ʒuʁ/ |

### 联诵与连音

- **联诵 (Liaison)**：词末不发音辅音 + 后词元音 → 发音
  - les amis → /lez‿ami/
  - nous avons → /nuz‿avɔ̃/
  - très important → /tʁɛz‿ɛ̃pɔʁtɑ̃/
- **连音 (Enchaînement)**：词末发音辅音 + 后词元音 → 连续读
  - elle aime → /ɛl‿ɛm/

## 2. 名词

### 性 (Genre)

| 类型 | 示例 |
|------|------|
| 阳性 | le livre, le chat, le soleil |
| 阴性 | la table, la chaise, la lune |
| 阳性后缀 | -age, -ment, -eau, -isme |
| 阴性后缀 | -tion, -sion, -té, -eur, -ance |
| 职业 | acteur/actrice, serveur/serveuse |

### 数 (Nombre)

| 规则 | 单数→复数 |
|------|----------|
| 一般加 -s | livre → livres |
| 以 -s/-x/-z 结尾 | cours → cours |
| 以 -au/-eau/-eu 结尾加 -x | château → châteaux |
| 以 -al 结尾变 -aux | cheval → chevaux |
| 以 -ail 结尾变 -aux (部分) | travail → travaux |

## 3. 冠词

| 类型 | 阳性 | 阴性 | 复数 |
|------|------|------|------|
| 定冠词 | le / l' | la / l' | les |
| 不定冠词 | un | une | des |
| 部分冠词 | du / de l' | de la / de l' | des |

**用法**：
- 定冠词：特指 — **Le** livre est intéressant.
- 不定冠词：泛指 — J'ai **un** chat.
- 部分冠词：不可数 — Je bois **du** café. / Je mange **de la** viande.

## 4. 形容词

### 性数配合

| 阳性 | 阴性 | 复数 |
|------|------|------|
| petit | petite | petits |
| grand | grande | grands |
| bon | bonne | bons |
| beau | belle | beaux |
| vieux | vieille | vieux |

### 位置 — BAGS 规则

BAGS 形容词放在名词前：
- **B**eauty: beau, joli
- **A**ge: jeune, vieux, nouveau
- **G**oodness: bon, mauvais
- **S**ize: grand, petit, gros

其余形容词放在名词后：un livre **intéressant**

## 5. 动词

### 三组动词

**第一组 -er (规则)**：parler

| 人称 | 现在时 |
|------|--------|
| je | parle |
| tu | parles |
| il/elle/on | parle |
| nous | parlons |
| vous | parlez |
| ils/elles | parlent |

**第二组 -ir (规则)**：finir

| 人称 | 现在时 |
|------|--------|
| je | finis |
| tu | finis |
| il/elle | finit |
| nous | finissons |
| vous | finissez |
| ils/elles | finissent |

**第三组 -re/-ir/-oir (不规则)**：prendre, sortir, vouloir

| 人称 | prendre | sortir | vouloir |
|------|---------|--------|---------|
| je | prends | sors | veux |
| tu | prends | sors | veux |
| il/elle | prend | sort | veut |
| nous | prenons | sortons | voulons |
| vous | prenez | sortez | voulez |
| ils/elles | prennent | sortent | veulent |

### 助动词：être 与 avoir

| 人称 | être | avoir |
|------|------|-------|
| je | suis | ai |
| tu | es | as |
| il/elle | est | a |
| nous | sommes | avons |
| vous | êtes | avez |
| ils/elles | sont | ont |

### 核心时态

| 时态 | 构成 | 示例 |
|------|------|------|
| **Passé composé** | avoir/être + 过去分词 | j'ai parlé, je suis allé |
| **Imparfait** | 词根 + -ais/-ais/-ait/-ions/-iez/-aient | je parlais |
| **Futur proche** | aller + 不定式 | je vais parler |
| **Futur simple** | 不定式 + -ai/-as/-a/-ons/-ez/-ont | je parlerai |
| **Conditionnel** | 不定式 + -ais (同 imparfait 词尾) | je voudrais |

### 否定

| 结构 | 示例 |
|------|------|
| ne...pas | Je **ne** parle **pas**. |
| ne...jamais | Je **ne** fume **jamais**. |
| ne...rien | Je **ne** vois **rien**. |
| ne...personne | Je **ne** vois **personne**. |
| ne...plus | Il **n'**est **plus** là. |

### 疑问句

| 方式 | 示例 |
|------|------|
| 语调上升 | Tu parles français ? |
| est-ce que | **Est-ce que** tu parles français ? |
| 主谓倒装 | Parles-**tu** français ? |
