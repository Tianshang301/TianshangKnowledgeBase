# 拉丁语基础

## 1. 字母与发音

### 字母表 (24个古典字母)

A B C D E F G H I K L M N O P Q R S T V X Y Z
a b c d e f g h i k l m n o p q r s t v x y z

（J, U, W 是后来添加的）

### 古典 vs 教会发音

| 字母/组合 | 古典发音 | 教会发音 | 古典例词 |
|----------|---------|---------|---------|
| c | /k/ 永远硬 | /k/ 或 /tʃ/ (e/i 前) | Caesar /ˈkajsar/ |
| g | /g/ 永远硬 | /g/ 或 /dʒ/ (e/i 前) | gens /gens/ |
| v | /w/ 像英语 w | /v/ | via /ˈwia/ |
| ae | /aj/ 像 eye | /ɛ/ | caelum /ˈkajlum/ |
| oe | /oj/ 像 boy | /ɛ/ | poena /ˈpojna/ |
| ch | /kʰ/ 送气 | /k/ | pulcher /ˈpulkʰer/ |
| ph | /pʰ/ 送气 | /f/ | philosophia |
| th | /tʰ/ 送气 | /t/ | theatrum |
| ti (元音前) | /ti/ | /tsi/ | gratia (古典 /ˈgratia/) |

### 重音规则

| 音节数 | 规则 | 示例 |
|--------|------|------|
| 双音节 | 重音在倒数第二 | **á**mor, **ví**ta |
| 多音节 | 倒数第二为长音则重读，否则倒数第三 | fe**ré**bat, **pó**pulus |

## 2. 名词 — 5种变位法

### 第一变位法 (-a, -ae) 多阴性

| 格 | 单数 | 复数 |
|---|------|------|
| **Nom.** | puell**a** (女孩) | puell**ae** |
| **Gen.** | puell**ae** | puell**ārum** |
| **Dat.** | puell**ae** | puell**īs** |
| **Acc.** | puell**am** | puell**ās** |
| **Abl.** | puell**ā** | puell**īs** |
| **Voc.** | puell**a** | puell**ae** |

### 第二变位法 (-us/-um, -ī) 多阳/中性

| 格 | 阳单 | 阳复 | 中单 | 中复 |
|---|------|------|------|------|
| **Nom.** | domin**us** | domin**ī** | bell**um** | bell**a** |
| **Gen.** | domin**ī** | domin**ōrum** | bell**ī** | bell**ōrum** |
| **Dat.** | domin**ō** | domin**īs** | bell**ō** | bell**īs** |
| **Acc.** | domin**um** | domin**ōs** | bell**um** | bell**a** |
| **Abl.** | domin**ō** | domin**īs** | bell**ō** | bell**īs** |

### 第三变位法 (各种结尾)

| 格 | 单数 | 复数 |
|---|------|------|
| **Nom.** | rēx (国王) | rēg**ēs** |
| **Gen.** | rēg**is** | rēg**um** |
| **Dat.** | rēg**ī** | rēg**ibus** |
| **Acc.** | rēg**em** | rēg**ēs** |
| **Abl.** | rēg**e** | rēg**ibus** |

### 第四变位法 (-us, -ūs)

| **Acc.** | fruct**um** | fruct**ūs** |

### 第五变位法 (-ēs, -ēī)

## 3. 形容词

### 第一/二类形容词 (bonus, -a, -um)

| 格 | 阳性 | 阴性 | 中性 |
|---|------|------|------|
| **Nom.** | bon**us** | bon**a** | bon**um** |
| **Gen.** | bon**ī** | bon**ae** | bon**ī** |
| **Acc.** | bon**um** | bon**am** | bon**um** |
| **Abl.** | bon**ō** | bon**ā** | bon**ō** |

### 第三类形容词 (ingens)

单数：ingens (Nom), ingent**is** (Gen), ingent**ī** (Dat), ingent**em** (Acc), ingent**ī** (Abl)
复数：ingent**ēs** (Nom阳阴), ingent**ia** (中), ingent**ium** (Gen)

### 比较级/最高级

| 原级 | 比较级 | 最高级 |
|------|--------|--------|
| altus (高) | alt**ior** (更高) | alt**issimus** (最高) |
| pulcher (美) | pulchr**ior** | pulcher**rimus** |
| facilis (易) | facil**ior** | facil**limus** |
| bonus (好) | **melior** (更好) | **optimus** (最好) |
| malus (坏) | **peior** (更坏) | **pessimus** (最坏) |
| magnus (大) | **maior** (更大) | **maximus** (最大) |
| parvus (小) | **minor** (更小) | **minimus** (最小) |

## 4. 动词 — 4种变位法

### 现在时主动语态

| 人称 | 1st amāre (爱) | 2nd monēre (警告) | 3rd regere (统治) | 4th audīre (听) |
|------|--------------|-----------------|-----------------|----------------|
| **1sg** | am**ō** | mone**ō** | reg**ō** | audi**ō** |
| **2sg** | amā**s** | monē**s** | regi**s** | audī**s** |
| **3sg** | ama**t** | mone**t** | regi**t** | audi**t** |
| **1pl** | amā**mus** | monē**mus** | regi**mus** | audī**mus** |
| **2pl** | amā**tis** | monē**tis** | regi**tis** | audī**tis** |
| **3pl** | ama**nt** | mone**nt** | regu**nt** | audi**unt** |

### 核心不规则动词

| 人称 | sum (是) | possum (能够) | volō (想要) |
|------|---------|-------------|------------|
| **1sg** | **s**um | **pos**sum | **vol**ō |
| **2sg** | **es** | **pot**es | **vīs** |
| **3sg** | **est** | **pot**est | **vult** |
| **1pl** | **s**umus | **pos**sumus | **vol**umus |
| **2pl** | **es**tis | **pot**estis | **vul**tis |
| **3pl** | **s**unt | **pos**sunt | **vol**unt |

### 六时态系统 (以 amō 为例)

| 时态 | 词干 | 示例 (1sg) |
|------|------|-----------|
| **Praesens** (现在) | 现在词干 | am**ō** |
| **Imperfectum** (过去未完成) | 现在词干 + -bā- | amā**bam** |
| **Futūrum I** (将来) | 现在词干 + -b- | amā**bō** |
| **Perfectum** (完成) | 完成词干 | amāv**ī** |
| **Plūsquamperfectum** (过去完成) | 完成词干 + -erā- | amāv**eram** |
| **Futūrum Exactum** (将来完成) | 完成词干 + -eri- | amāv**erō** |

### 被动语态

用主动词尾替换为被动词尾：
- -or/-ris/-tur/-mur/-minī/-ntur
- am**or** (我被爱), amā**ris** (你被爱), amā**tur** (他被爱)

## 5. 句法特征

| 特征 | 示例 |
|------|------|
| **SOV 语序** | Caesar **Galliam vīcit.** (凯撒 高卢 征服了) |
| **间接引语 (AcI)** | Dico **eum venīre.** (我说他来) |
| **夺格独立主格** | **Urbe captā,** milites recessērunt. (城市被攻陷后，士兵撤退了) |
| **目的从句** | Venit **ut videat.** (他来为了看) |
| **条件句** | Si id faciās, poenam patiāris. (如果你做那个，你会受罚) |
