# 西班牙语基础

## 1. 字母与发音

### 字母表 (27个字母)

Aa Bb Cc Dd Ee Ff Gg Hh Ii Jj Kk Ll Mm Nn Ññ Oo Pp Qq Rr Ss Tt Uu Vv Ww Xx Yy Zz

### 元音 (5个纯元音)

| 音素 | 示例 |
|------|------|
| /a/ | casa /ˈkasa/, amar /aˈmaɾ/ |
| /e/ | café /kaˈfe/, mesa /ˈmesa/ |
| /i/ | sí /si/, vida /ˈbida/ |
| /o/ | sol /sol/, como /ˈkomo/ |
| /u/ | luna /ˈluna/, tú /tu/ |

### 辅音

| 字母 | 发音 | 规则 | 示例 |
|------|------|------|------|
| c | /k/ 或 /θ/ | a/o/u 前 /k/；e/i 前 /θ/(西) /s/(美) | casa, cena |
| g | /g/ 或 /x/ | a/o/u 前 /g/；e/i 前 /x/ | gato, gente |
| h | 不发音 | 始终无声 | hora /ˈoɾa/ |
| j | /x/ | 喉擦音 | jamón /xaˈmon/ |
| ll | /ʎ/ 或 /ʝ/ | 根据方言 | llamar /ʎaˈmaɾ/ |
| ñ | /ɲ/ | 腭鼻音 | año /ˈaɲo/ |
| r | /ɾ/ 或 /r/ | 词首/rr/为颤音；词中单 r 为闪音 | pero /ˈpeɾo/, perro /ˈpero/ |
| qu | /k/ | 只在 e,i 前 | queso, aquí |
| y | /ʝ/ 或 /ʃ/ | 元音前/词尾 | yo, ley |
| z | /θ/ 或 /s/ | 西班牙 /θ/，拉美 /s/ | zapato |

### 重音规则

| 条件 | 重音位置 | 示例 |
|------|---------|------|
| 以 n/s/元音结尾 | 倒数第二音节 | casa /ˈkasa/, hablan /ˈaβlan/ |
| 以其他辅音结尾 | 最后音节 | hablar /aˈβlaɾ/, ciudad /θjuˈðad/ |
| 违反规则 | 加重音符号 | café, árbol, lápiz |

## 2. 名词与冠词

### 名词阴阳性

| 词尾 | 示例 |
|------|------|
| 阳性 -o, -ma, -l, -r, -n | el libro, el problema, el árbol |
| 阴性 -a, -ción, -sión, -dad, -tad | la casa, la canción, la ciudad |

**例外**：el día (m.), la mano (f.), el agua (f. 但用 el)

### 冠词

| 冠词 | 阳性单 | 阴性单 | 阳性复 | 阴性复 |
|------|--------|--------|--------|--------|
| 定冠词 | el | la | los | las |
| 不定冠词 | un | una | unos | unas |

## 3. 形容词

- **性数配合**：un chico **alto** / una chica **alta** / chicos **altos** / chicas **altas**
- **位置**：一般放在名词后 — un libro **interesante**
- **短形式**：某些形容词在阳性单数名词前省略 -o
  - **bueno** → **buen** día
  - **malo** → **mal** momento
  - **grande** → **gran** hombre

## 4. 动词

### 三组变位 (现在时)

| 人称 | -ar (hablar) | -er (comer) | -ir (vivir) |
|------|-------------|------------|------------|
| yo | hablo | como | vivo |
| tú | hablas | comes | vives |
| él/ella/Ud. | habla | come | vive |
| nosotros | hablamos | comemos | vivimos |
| vosotros | habláis | coméis | vivís |
| ellos/ellas/Uds. | hablan | comen | viven |

### 核心不规则动词

| 人称 | ser | estar | ir | tener |
|------|-----|-------|-----|-------|
| yo | soy | estoy | voy | tengo |
| tú | eres | estás | vas | tienes |
| él/ella | es | está | va | tiene |
| nosotros | somos | estamos | vamos | tenemos |
| vosotros | sois | estáis | vais | tenéis |
| ellos | son | están | van | tienen |

### 核心时态

| 时态 | 构成 | 示例 |
|------|------|------|
| **Presente** | 词根+词尾 | hablo |
| **Pretérito** | -é/-aste/-ó/-amos/-asteis/-aron | hablé |
| **Imperfecto** | -aba/-abas/-aba/-ábamos/-abais/-aban | hablaba |
| **Futuro** | 不定式 + -é/-ás/-á/-emos/-éis/-án | hablaré |
| **Condicional** | 不定式 + -ía/-ías/-ía/-íamos/-íais/-ían | hablaría |
| **Subjuntivo** | 从 que 引导的特殊变位 | que hable |
| **Perfecto** | haber + 过去分词 | he hablado |

### ser vs estar

| ser (永久) | estar (状态/位置) |
|------------|-----------------|
| Soy profesor. (我是老师) | Estoy cansado. (我累了) |
| Es alto. (他很高) | Está en casa. (他在家) |
| Son las dos. (两点) | Está bien. (没问题) |

### por vs para

| por | para |
|-----|------|
| 原因/动机：Lo hago **por** ti. (为了你) | 目的：Estudio **para** aprender. (为了学) |
| 经过/通过：Camino **por** el parque. | 接受者：Es **para** ti. (给你) |
| 交换：**Por** favor | 期限：**Para** mañana. (明天前) |

### pronombres (代词)

- **主语**：yo, tú, él/ella/Ud., nosotros, vosotros, ellos/ellas/Uds.
- **直接宾语**：me, te, lo/la, nos, os, los/las
- **间接宾语**：me, te, le, nos, os, les

**注意**：le/lo 混淆（leísmo）— 西班牙部分地区用 le 替代 lo 作阳性直接宾语

### gustar 结构

- Me **gusta** el libro. (主谓倒装：书让我喜欢)
- Me **gustan** los libros.
- 动词按后面名词的数变化

## 5. 否定与疑问

| 否定 | 示例 |
|------|------|
| No + 动词 | **No** hablo español. |
| Nunca / Jamás | **Nunca** como carne. |
| Nada / Nadie | **Nada** sé. / No veo a **nadie**. |

| 疑问词 | 含义 |
|--------|------|
| ¿Qué? | 什么 |
| ¿Quién? | 谁 |
| ¿Dónde? | 哪里 |
| ¿Cuándo? | 何时 |
| ¿Por qué? | 为什么 |
| ¿Cómo? | 怎样 |
| ¿Cuánto? | 多少 |

## 6. 常用表达

| 场景 | 西班牙语 |
|------|----------|
| 你好 | ¡Hola! / Buenos días / Buenas tardes |
| 再见 | Adiós / Hasta luego |
| 谢谢 | Gracias / Muchas gracias |
| 不客气 | De nada |
| 对不起 | Perdón / Lo siento |
| 我叫... | Me llamo... |
| 你叫什么？ | ¿Cómo te llamas? |
| 你好吗？ | ¿Cómo estás? |
| 很好，谢谢 | Muy bien, gracias |
| 多少钱？ | ¿Cuánto cuesta? |
| 我不明白 | No entiendo |
| 请再说一遍 | ¿Cómo? / ¿Perdón? |
| 干杯！ | ¡Salud! |
