### 收音 (받침)

7种代表音：**ㄱ, ㄴ, ㄷ, ㄹ, ㅁ, ㅂ, ㅇ**（27个字母归入7种发音）

### 发音规则

| 规则 | 说明 | 示例 |
|------|------|------|
| **连音化** | 收音+后词元音→移到后词 | 음악을 → [으마글] |
| **鼻音化** | ㄱ→ㅇ, ㄷ→ㄴ, ㅂ→ㅁ 在鼻音前 | 국물 → [궁물] |
| **送气化** | ㄱ/ㄷ/ㅂ/ㅈ + ㅎ → 送气音 | 좋다 → [조타] |

## 2. 助词

| 助词 | 功能 | 示例 |
|------|------|------|
| **은/는** | 主题标记 | 저**는** 학생입니다. |
| **이/가** | 主语标记 | 이것**이** 책입니다. |
| **을/를** | 宾语标记 | 책**을** 읽습니다. |
| **의** | 所有格 | 친구**의** 책 |
| **에** | 时间/地点/方向 | 학교**에** 갑니다. |
| **에서** | 动作场所/起点 | 집**에서** 공부합니다. |
| **으로/로** | 方向/工具 | 버스**로** 갑니다. |
| **와/과** | 和, 与 | 친구**와** 이야기합니다. |
| **보다** | 比 | 나**보다** 키가 커요. |

## 3. 语序 SOV

**저는** (主题) **책을** (宾语) **읽습니다** (动词谓语).

## 4. 动词/形容词变形

### 现在时 (해요体)

| 词干 | 规则 | 示例 |
|------|------|------|
| ㅏ/ㅗ 结尾 | + -아요 | 가다 → 가**요**, 보다 → 봐**요** |
| 其他元音结尾 | + -어요 | 먹다 → 먹**어요**, 읽다 → 읽**어요** |
| 하다 | → 해요 | 공부하다 → 공부**해요** |

### 过去时

词干 + -았/었어요：가다 → 갔어요, 먹다 → 먹었어요

### 将来时

- **-겠어요**：의지 — 가겠어요
- **-(을) 거예요**：추측 — 갈 거예요

### 进行时

-고 있다：먹고 있어요 (正在吃)

### 否定

| 形式 | 示例 |
|------|------|
| **안** + V | **안** 가요 (不去) |
| **못** + V | **못** 가요 (去不了) |
| **-지 않다** | 가**지 않아요** (不去) |

### 敬语 (시)

动词词干 + **시** + 敬语词尾：가다 → 가**시**다 → 가**세**요 (请走)

### 命令/请求

| 形式 | 示例 |
|------|------|
| -(으)세요 | 가**세요**, 먹**으세요** |
| -(으)십시오 | 가**십시오** (极正式) |

### 提议

| 形式 | 示例 |
|------|------|
| -자 (半语) | 가**자**! |
| -아/어요 (해요) | 가**요**! |
| -(으)ㅂ시다 (正式) | 갑**시다**! |

## 5. 连接词尾

| 词尾 | 含义 | 示例 |
|------|------|------|
| **-고** | 和/并且 | 가**고** 먹어요 (去然后吃) |
| **-지만** | 但是 | 가**지만** (但是去) |
| **-아/어서** | 因为 | 배가 고파**서** 먹어요 (饿了所以吃) |
| **-(으)면** | 如果 | 비가 오**면** (如果下雨) |
| **-(으)려고** | 为了 | 공부하**려고** (为了学习) |
