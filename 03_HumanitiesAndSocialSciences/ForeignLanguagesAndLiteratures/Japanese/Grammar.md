# 日语语法详解

## 句子结构 (SOV)

日语是 **主-宾-谓 (SOV)** 语言，与英语的 **主-谓-宾 (SVO)** 不同。

```
私は リンゴを 食べる。
Watashi wa ringo o taberu.
I       apple   eat.
(我吃苹果。)
```

## 助词 (Particles)

助词标记句子成分的语法功能。

| 助词 | 功能 | 示例 |
|------|------|------|
| は (wa) | 主题标记 | 私は学生です。 |
| が (ga) | 主语标记 | 猫が好きです。 |
| を (o) | 宾语标记 | 本を読む。 |
| に (ni) | 目标/时间/方向 | 学校に行く。 |
| で (de) | 手段/地点 | 電車で行く。 |
| へ (e) | 方向 | 東京へ行く。 |
| と (to) | 并列/共同 | 友達と話す。 |
| から (kara) | 起点/原因 | 中国から来た。 |
| まで (made) | 终点 | 家まで走る。 |
| より (yori) | 比较 | 猫より犬が好き。 |
| の (no) | 所属/修饰 | 私の本。 |

### は vs が

| は | が |
|----|----|
| 主题标记 ("至于...") | 主语标记 |
| 已知信息 | 新信息 |
| 对比 | 客观描述 |
| 私は学生です。 | 雨が降っている。 |
| 象は鼻が長い。 | 誰が来た？ 山田さんが来た。 |

## 动词分类

### グループ分け

| グループ | 特征 | 例 |
|---------|------|----|
| グループ1 (五段) | 结尾为 -u 段 (く、す、つ、ぬ、ぶ、む、る、ぐ、う) | 書く、話す、待つ、死ぬ、遊ぶ、読む、作る、泳ぐ、買う |
| グループ2 (一段) | 结尾为 -eru 或 -iru | 食べる、見る、起きる、寝る |
| グループ3 (不规则) | する、来る | 勉強する、来る |

### ます形 (丁寧体)

| グループ | 变化规则 | 例 |
|---------|---------|----|
| 五段 | 词尾 u → i + ます | 書く→書きます、読む→読みます |
| 一段 | 去る + ます | 食べる→食べます、見る→見ます |
| 来る | 来る→来ます (きます) | |
| する | する→します | 勉強する→勉強します |

### て形 (连接形)

| グループ | 规则 | 例 |
|---------|------|----|
| 五段 (く→いて) | く→いて | 書く→書いて |
| 五段 (ぐ→いで) | ぐ→いで | 泳ぐ→泳いで |
| 五段 (す→して) | す→して | 話す→話して |
| 五段 (う/つ/る→って) | →って | 買う→買って、待つ→待って、作る→作って |
| 五段 (ぬ/ぶ/む→んで) | →んで | 死ぬ→死んで、遊ぶ→遊んで、読む→読んで |
| 一段 | 去る + て | 食べる→食べて |
| 来る | 来る→来て (きて) | |
| する | する→して | 勉強する→勉強して |

### た形 (过去形)

与て形规则相同，て→た、で→だ：

```
書いて → 書いた
読んで → 読んだ
食べて → 食べた
して → した
来て → 来た (きた)
```

### ない形 (否定形)

| グループ | 规则 | 例 |
|---------|------|----|
| 五段 | 词尾 u → a + ない | 書く→書かない |
| 一段 | 去る + ない | 食べる→食べない |
| 来る | 来る→来ない (こない) | |
| する | する→しない | 勉強する→勉強しない |

## 形容词

### い形容词

| 形 | 肯定 | 否定 |
|----|------|------|
| 现在 | 高い | 高くない |
| 过去 | 高かった | 高くなかった |
| て形 | 高くて | - |
| 副词 | 高く | - |

### な形容词

| 形 | 肯定 | 否定 |
|----|------|------|
| 现在 | 静かだ/静かな | 静かではない/じゃない |
| 过去 | 静かだった | 静かではなかった |
| て形 | 静かで | - |
| 副词 | 静かに | - |

## て形用法

| 用法 | 结构 | 示例 |
|------|------|------|
| 请求 | てください | 待ってください。 |
| 进行 | ている | 本を読んでいる。 |
| 允许 | てもいい | 入ってもいいですか。 |
| 禁止 | てはいけない | ここで遊んではいけない。 |
| 顺序 | てから | 食べてから出かける。 |
| 原因 | て | 雨が降って、行けなかった。 |
| 并列 | て | 安くて美味しい。 |

## 简体 vs 敬体

| 时态 | 敬体 (です/ます) | 简体 (だ/である) |
|------|-----------------|-----------------|
| 现在肯定 | 食べます | 食べる |
| 现在否定 | 食べません | 食べない |
| 过去肯定 | 食べました | 食べた |
| 过去否定 | 食べませんでした | 食べなかった |
| 名词现在 | 学生です | 学生だ |
| 名词否定 | 学生ではありません | 学生ではない |
| い形现在 | 高いです | 高い |
| い形过去 | 高かったです | 高かった |

## 动词活用汇总

| 形 | 書く (五段) | 食べる (一段) | 来る | する |
|----|-----------|-------------|------|------|
| 辞書形 | 書く | 食べる | 来る | する |
| ます形 | 書きます | 食べます | 来ます | します |
| て形 | 書いて | 食べて | 来て | して |
| た形 | 書いた | 食べた | 来た | した |
| ない形 | 書かない | 食べない | 来ない | しない |
| 可能形 | 書ける | 食べられる | 来られる | できる |
| 条件形 (ば) | 書けば | 食べれば | 来れば | すれば |
| 意向形 | 書こう | 食べよう | 来よう | しよう |
| 命令形 | 書け | 食べろ | 来い | しろ |
| 禁止形 | 書くな | 食べるな | 来るな | するな |

## 条件形

| 条件 | 用法 | 示例 |
|------|------|------|
| と | 必然结果、自然现象 | 春になると、花が咲く。 |
| たら | 假设已发生 (后接主观) | 日本に行ったら、富士山を見たい。 |
| ば | 一般假设 | 安ければ、買います。 |
| なら | 对方提到的话题 | 行くなら、傘を持って行きなさい。 |

## 授受表达

| 动词 | 方向 | 示例 |
|------|------|------|
| あげる | 我给 / 别人给别人 | 友達に本をあげる。 |
| くれる | 别人给我 | 母が本をくれた。 |
| もらう | 我从别人得到 | 友達に本をもらった。 |

| 変わる | 変える |
| 終わる | 終える |

| 並ぶ | 並べる |
