# 新闻学基础

## 一、新闻基本概念

### 1.1 新闻定义

陆定一（1943）："新闻是新近发生的事实的报道。"

| 要素 | 含义 |
|------|------|
| 新近 | 时间性——新闻是即刻的 |
| 发生 | 真实性——有客观事实基础 |
| 事实 | 本源——事实是新闻前提 |
| 报道 | 传播——是经过选择加工的信息 |

徐宝璜（1919）："多数阅者所注意之最近事实也。"西方：狗咬人不是新闻人咬狗是新闻（反常性）。

### 1.2 新闻价值

| 要素 | 英文 | 说明 |
|------|------|------|
| 时新性 | Timeliness | 最新发生的事件 |
| 接近性 | Proximity | 地理或心理上接近 |
| 显著性 | Prominence | 名人名地名事 |
| 冲突性 | Conflict | 矛盾争议战争竞争 |
| 人情味 | Human Interest | 情感共鸣趣味性 |
| 重要性 | Impact | 对公众影响程度 |
| 新奇性 | Novelty | 罕见反常事件 |

要素越多、强度越大，新闻价值越大。

### 1.3 新闻客观性

新闻应如实报道事实，事实与意见分离。历史：便士报→美联社（1848）→1923年《新闻规约》。要求：平衡报道、多方核实、避免情绪语言。完全客观不可能但作为职业理想值得追求。

### 1.4 新闻真实性

真实是新闻生命。三个层次：①具体事实真实（5W1H 准确）②总体真实（反映全局）③本质真实（揭示内在联系）。虚假新闻：故意捏造失实报道片面报道。核实力第一道防线。

## 二、新闻采访

### 2.1 新闻来源

| 类型 | 特点 | 举例 |
|------|------|------|
| 政府 | 权威常规 | 新闻发布会公报 |
| 机构 | 专业 | 企业高校医院 |
| 个人 | 直接生动 | 目击者当事人专家 |
| 社媒 | 即时碎片 | 微博朋友圈短视频 |
| 文献 | 可验证 | 档案统计年报 |

信源保护：重大公共利益可匿名不滥用。

### 2.2 采访准备

五步：①明确目的②研究背景③制订提纲④准备设备⑤预约沟通。

### 2.3 提问技巧

| 类型 | 特点 | 例子 |
|------|------|------|
| 开放式 | 自由发挥 | "您对此事看法？" |
| 闭合式 | 确认信息 | "您是否同意？" |
| 追问式 | 深入细节 | "具体说说当时？" |
| 假设式 | 探测立场 | "如果政策调整？" |
| 反问式 | 检验观点 | "这解释有无漏洞？" |
| 渐进式 | 由浅入深 | 轻松切入敏感 |

STAR 法则：Situation→Task→Action→Result。

### 2.4 采访记录

录音须征得同意。笔记速记关键词时间线引语。观察记录：环境表情肢体语言。即时整理（24小时记忆衰退）。

### 2.5 新闻发布会

提前到场有利位置。提问简短明确。避免炫耀自表。尊重同行。

## 三、新闻写作

### 3.1 倒金字塔结构

最重要信息最前面，依次递减。导语5W1H。

| 导语类型 | 特点 | 例子 |
|---------|------|------|
| 直叙式 | 陈述核心事实 | "昨下午 X 桥坍塌" |
| 描述式 | 场景描写 | "烟雾中有人呼救" |
| 引语式 | 引语开头 | "'我以为活不成了'" |
| 悬念式 | 留悬念 | "一张照片揭开秘密" |

### 3.2 新闻分类

**按方式**：消息（500-800字）→通讯→特写→评论→深度报道。**按题材**：时政经济社会文化体育科技。

### 3.3 硬新闻 vs 软新闻

| | 硬新闻 | 软新闻 |
|--|--------|--------|
| 定义 | 重要严肃即时 | 轻松有趣 |
| 功能 | 告知信息 | 娱乐好奇 |
| 时效 | 非常高 | 宽松 |
| 写作 | 倒金字塔 | 叙事故事化 |

### 3.4 消息规范

标题实题为主15字内。第三人称。直接引语。段落短小。简讯200-300消息500-800通讯1500-3000字。

## 四、新闻编辑

### 4.1 标题制作

| 类型 | 特点 | 例子 |
|------|------|------|
| 实题 | 直陈事实 | "中美签署经贸协议" |
| 虚题 | 渲染评价 | "春风拂面暖人心" |
| 复合题 | 虚实结合 | "一带一路花开全球" |

标准：准确简练生动易读。避免标题党。

### 4.2 版面设计

左上角黄金位置。模块化。字距行距留白。色彩稳重。

### 4.3 照片选择

新闻性→冲击力→真实性（禁摆拍 PS）→情感性→技术性。

## 五、新闻伦理

### 5.1 核心原则

| 原则 | 要求 |
|------|------|
| 准确 | 事实引语背景准确 |
| 公正 | 平衡报道多方发声 |
| 隐私 | 尊重个人隐私 |
| 透明 | 披露信源冲突 |
| 责任 | 及时更正道歉 |

### 5.2 伦理困境

隐私权 vs 新闻价值。未成年人匿名。隐性采访限重大公共利益且其他方法无效。拒绝有偿新闻。避免利益冲突。抵制抄袭洗稿。灾难采访避免二次伤害。

### 5.3 算法时代伦理

信息茧房回音室效应。假新闻比真相快6倍（MIT 研究）。Deepfake 挑战眼见为实。事实核查需完善。

## 六、新闻专业主义

### 6.1 专业主义核心

服务公众→客观报道→新闻自治→伦理准则。挑战：商业流量导向、政治压力、技术变革（AI 算法）。

### 6.2 全媒体记者

纸媒→文字+摄影+短视频+数据可视化+社交运营。跨界协作。终身学习。

## 七、总结

新闻学理论与实践并重。从业者须掌握采写编技能，坚守真实客观公正底线。在技术革命时代坚持新闻核心价值——以专业发现事实、以责任传递信息、以独立服务公众。

## 关键概念辨析
概念定义的清晰度直接影响研究的质量。
以下是该领域中若干容易混淆的概念。

**概念一与概念二的区分**：
前者侧重于外在的形式特征。
后者关注内在的运作机制。
两者在实际分析中往往需要结合使用。

**微观与宏观层面的联系**：
微观现象是宏观结构的基础。
宏观结构又约束微观行为。
理解两者的相互作用是社会分析的核心。

**静态分析与动态分析**：
静态分析关注某一时点的截面特征。
动态分析关注过程和变化的轨迹。
两种视角互补而非替代。
