# 新媒体与舆论学

## 一、新媒体基本概念

### 1.1 新媒体定义

基于数字技术网络技术，通过互联网无线通信网提供信息服务的传播形态。与传统媒体（报纸广播电视）相对。

### 1.2 核心特征

| 特征 | 说明 |
|------|------|
| 数字化 | 二进制编码 |
| 交互性 | 用户平台双向互动 |
| 超文本性 | 超链接非线性 |
| 网络化 | 节点网络结构 |
| 移动化 | 智能手机随时获取 |
| 即时性 | 实时发布接收 |
| 多媒体 | 文图音视频AR/VR |
| 个性化 | 算法推荐 |
| UGC | 用户生产内容 |

### 1.3 发展历程

Web 1.0（1990s—2000）门户编辑中心。Web 2.0（2000—2010）社媒UGC。Web 3.0（2010至今）智能算法AI。

## 二、主要平台

### 2.1 微博

中国Twitter。裂变传播（关注转发）。热搜实时热点。大V影响巨大。问题：水军网络暴力。

### 2.2 微信

腾讯月活超13亿。朋友圈（强关系）公众号视频号小程序微信支付。私密闭环信任传播。谣言顽固。

### 2.3 抖音/TikTok

字节跳动。算法推荐（协同过滤内容推荐）。15秒沉浸式。去中心化。兴趣电商直播。

### 2.4 B站

Z世代弹幕社区。中长视频（5-30分）。答题入站。UP主生态。知识区增长快。

### 2.5 平台对比

| 平台 | 形态 | 社交 | 内容 | 用户 |
|------|------|------|------|------|
| 微博 | 短内容 | 弱关系 | 热点 | 全年龄 |
| 微信 | 社交+服务 | 强关系 | 生活新闻 | 全年龄 |
| 抖音 | 短视频 | 算法 | 娱乐电商 | 年轻扩张 |
| B站 | 中长视频 | 兴趣 | 深度创意 | Z世代 |

## 三、关键业态

### 3.1 短视频

15秒—5分。碎片化消费。MCN孵化变现。知识类增长快。短视频新闻（央视抖音1.5亿粉）。

### 3.2 直播

电商（李佳琦薇娅——2020双十一李佳琦112亿GMV）。知识（得到）。秀场打赏。直播助农。

### 3.3 媒体融合

传统+新媒体。人民日报中央厨房一次采集多元分发。县级融媒体2000+。新华社AI主播。深度融合从相加到相融。

### 3.4 数据新闻

数据可视化D3.js ECharts Flourish。交互式新闻。《卫报》伊拉克日志、财新数字说、澎湃美数课。需数据素养。

## 四、舆论学基础

### 4.1 舆论定义与特征

舆论是公众关于公共事务意见态度的集合。

| 特征 | 说明 |
|------|------|
| 公众性 | 公共事务 |
| 集合性 | 多数人意见 |
| 动态性 | 随事件变化 |
| 影响力 | 影响政策 |

李普曼《公众舆论》（1922）。**拟态环境**——媒介建构的"脑中世界"非客观镜像，经选择加工。**刻板印象**——认知经济化。

### 4.2 议程设置

McCombs & Shaw（1972）：媒介议题排序影响公众排序。

数字时代新路径：传统→公众（仍有效）；社媒→传统（热搜倒逼）；公众→公众（水平）；NAS网络议程。

### 4.3 框架理论

恩特曼（1993）四功能：定义问题→诊断原因→道德判断→解决方案。框架类型：策略议题冲突人情味。框架竞争多方争夺定义权。

### 4.4 网络舆论形成

| 阶段 | 特征 |
|------|------|
| 触发 | 事件引爆 |
| 扩散 | 裂变传播 |
| 聚合 | 主流形成 |
| 博弈 | 多方回应 |
| 消退 | 新热点或解决 |

主体：网民意见领袖媒体机构政府企业。算法为隐形议程建筑师。

## 五、网络舆情管理与危机传播

### 5.1 舆情监测

关键词→语义→情感（正负中）→趋势→影响力分析。工具：人民网舆情清博微热点。指标：声量情感敏感度爆发速度。

### 5.2 SCCT危机传播

Coombs（2007）：

| 危机类型 | 归因 | 策略 |
|---------|------|------|
| 受害者型 | 无责 | 否认→减责 |
| 意外型 | 无意 | 辩解→减责 |
| 可防范型 | 有意疏忽 | 道歉→全责 |

最佳：第一时间→坦诚→担责→行动→持续。完全道歉vs形式道歉。

### 5.3 谣言

Allport公式：谣言=重要性×模糊性。修正：焦虑×不确定性×涉入度。社媒特征：裂变难查情绪驱动回音室Deepfake。辟谣：Snopes PolitifFact 中国互联网联合辟谣平台。困境：辟谣慢于谣言。

## 六、新媒体舆论新特征

### 6.1 信息茧房

桑斯坦（2001）：算法只推同质信息形成封闭空间。回音室效应观点强化极化。过滤气泡（Pariser, 2011）算法决定可见内容。

### 6.2 后真相

2016牛津词：情感和信念比客观事实更能影响民意。逆火效应越纠越强。Deepfake动摇眼见为实。

### 6.3 数字行动主义

冰桶挑战（2014 ALS募2.2亿） #MeToo（2017全球）。特征：去中心化快速低门槛。批评：Slacktivism转发≠真正参与。

### 6.4 算法治理

中国2022《算法推荐管理规定》要求透明可解释。欧盟DSA要求评估风险。平台审核与言论自由的张力。

## 七、总结

新媒体重塑传播生态。"人人有麦克风"赋予个体表达权力。但信息过载算法操控后真相使舆论场更复杂。从李普曼拟态环境到算法环境，信息批判素养和公共理性是应对虚假极化的根本。

## 关键概念辨析
概念定义的清晰度直接影响研究的质量。
以下是该领域中若干容易混淆的概念。

**概念一与概念二的区分**：
前者侧重于外在的形式特征。
后者关注内在的运作机制。
两者在实际分析中往往需要结合使用。

**微观与宏观层面的联系**：
微观现象是宏观结构的基础。
宏观结构又约束微观行为。
理解两者的相互作用是社会分析的核心。

**静态分析与动态分析**：
静态分析关注某一时点的截面特征。
动态分析关注过程和变化的轨迹。
两种视角互补而非替代。
