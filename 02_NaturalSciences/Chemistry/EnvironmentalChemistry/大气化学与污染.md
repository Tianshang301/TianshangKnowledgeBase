# 大气化学与污染详解

## 一、大气组成与结构

### 大气组成

| 成分 | 体积分数 | 作用 |
|------|---------|------|
| 氮气 ($\text{N}_2$) | 78.08% | 稀释氧气，参与固氮循环 |
| 氧气 ($\text{O}_2$) | 20.95% | 呼吸和燃烧 |
| 氩气 (Ar) | 0.93% | 惰性 |
| 二氧化碳 ($\text{CO}_2$) | 0.041% (420 ppm) | 温室气体 |
| 氖、氦、氪、氙 | 微量 | 惰性 |
| 水蒸气 ($\text{H}_2\text{O}$) | 0-4% | 温室气体，参与云形成 |
| 臭氧 ($\text{O}_3$) | 10 ppm (平流层) | 吸收紫外线 |

### 大气分层

| 层名 | 高度范围 | 温度变化 | 特征 |
|------|---------|---------|------|
| 对流层 | 0-12 km | 降温 (6.5°C/km) | 天气现象，垂直混合强 |
| 平流层 | 12-50 km | 升温 | 臭氧层，水平气流 |
| 中间层 | 50-85 km | 降温 | 最低温度 ~ -90°C |
| 热层 | 85-500 km | 升温 | 吸收太阳 X 射线 |
| 逸散层 | > 500 km | 等温 | 轻气体逃逸 |

## 二、光化学烟雾

### 形成条件

光化学烟雾的形成需要三个条件：氮氧化物 ($\text{NO}_x$) + 挥发性有机物 (VOCs) + 强阳光。

### 化学反应机制

1. $\text{NO}_2 \xrightarrow{h\nu} \text{NO} + \text{O}$
2. $\text{O} + \text{O}_2 \to \text{O}_3$
3. $\text{O}_3 + \text{NO} \to \text{NO}_2 + \text{O}_2$（正常情况下臭氧被消耗）
4. VOCs 与 $\text{OH}^\cdot$ 反应生成过氧自由基 $\text{RO}_2^\cdot$
5. $\text{RO}_2^\cdot + \text{NO} \to \text{RO}^\cdot + \text{NO}_2$（不消耗臭氧，反而将 NO 转化为 NO₂）
6. 臭氧不断累积

### 危害

- $\text{O}_3$：刺激呼吸道，损害肺功能
- $\text{PAN}$（过氧乙酰硝酸酯）：强烈刺激眼睛，损害植物
- 降低能见度
- 损害橡胶和织物

## 三、酸雨

### 定义

pH < 5.6 的降水（未污染雨水因 $\text{CO}_2$ 溶解呈弱酸性，pH ≈ 5.6）。

### 成因

二氧化硫 ($\text{SO}_2$) 和氮氧化物 ($\text{NO}_x$) 在大气中转化为硫酸和硝酸。

**$\text{SO}_2$ 氧化途径：**

气相氧化：$\text{SO}_2 + \text{OH}^\cdot \to \text{HSO}_3^\cdot \xrightarrow{\text{O}_2} \text{SO}_3 \xrightarrow{\text{H}_2\text{O}} \text{H}_2\text{SO}_4$

液相氧化：$\text{SO}_2$ 溶解于云滴，被 $\text{H}_2\text{O}_2$、$\text{O}_3$、$\text{O}_2$（$\text{Fe}^{3+}$、$\text{Mn}^{2+}$ 催化）氧化。

**$\text{NO}_x$ 转化：** $\text{NO}_2 + \text{OH}^\cdot \to \text{HNO}_3$

### 危害

- 酸化土壤和水体，释放 $\text{Al}^{3+}$ 等有毒金属离子
- 损害森林（如德国黑森林）
- 腐蚀建筑（大理石 $\text{CaCO}_3$ 被酸侵蚀）
- 损害文化遗产

## 四、臭氧层破坏

### 臭氧层

平流层中 $\text{O}_3$ 浓度最大的区域（15-35 km）。$\text{O}_3$ 吸收 97-99% 的太阳 UV-B（280-315 nm）和 UV-C（< 280 nm）辐射。

### 臭氧损耗机制

**催化循环（以 Cl 为例）：**

$$\text{Cl} + \text{O}_3 \to \text{ClO} + \text{O}_2$$
$$\text{ClO} + \text{O} \to \text{Cl} + \text{O}_2$$
$$\text{净反应：} \text{O}_3 + \text{O} \to 2\text{O}_2$$

一个 Cl 原子可以破坏 $10^5$ 个 $\text{O}_3$ 分子。

### 氟氯碳化物 (CFCs)

CFCs（$\text{CCl}_2\text{F}_2$、$\text{CCl}_3\text{F}$ 等）在平流层中被紫外光解产生 Cl 原子：

$$\text{CFCl}_3 \xrightarrow{h\nu} \text{CFCl}_2^\cdot + \text{Cl}^\cdot$$

### 南极臭氧洞

南极春季（9-10月）臭氧浓度急剧下降。原因：极地平流层云 (PSC) 表面发生非均相反应释放 Cl₂，太阳光解 Cl₂ 产生大量 Cl 原子。

### 《蒙特利尔议定书》(1987)

限制并逐步淘汰 CFCs 的生产和消费。被认为是最成功的国际环境协议之一，臭氧层预计在 2060-2070 年恢复。

## 五、温室效应与全球变暖

### 温室气体

温室气体吸收地表发射的红外辐射，使地表温度维持在约 15°C（无温室效应时为 -18°C）。

**主要温室气体及其全球变暖潜能值 (GWP)：**

| 气体 | 来源 | GWP (100年) | 浓度增幅 |
|------|------|------------|---------|
| $\text{CO}_2$ | 化石燃料燃烧 | 1 | 280 → 420 ppm |
| $\text{CH}_4$ | 农业、天然气泄漏 | 28 | 0.7 → 1.9 ppm |
| $\text{N}_2\text{O}$ | 化肥、工业 | 265 | 270 → 330 ppb |
| $\text{CFCs}$ | 制冷剂 | 数千-上万 | 人为排放 |

### 全球变暖

自工业革命以来全球平均气温上升约 1.2°C。主要影响：

- 冰川退缩、海平面上升（每年 3.3 mm）
- 极端天气事件增加（热浪、暴雨、干旱）
- 生态系统变化（物种迁徙、珊瑚白化）
- 粮食安全威胁

### IPCC 与《巴黎协定》

目标：将本世纪末全球变暖控制在 1.5-2°C 以内。途径：减少化石燃料使用、提高能效、发展可再生能源、碳捕集与封存 (CCS)。

## 六、大气颗粒物 (PM)

### 分类

- **PM10：** 可吸入颗粒物（直径 ≤ 10 μm）
- **PM2.5：** 细颗粒物（直径 ≤ 2.5 μm），可进入肺泡
- **PM0.1：** 超细颗粒物（直径 ≤ 0.1 μm），可进入血液

### 来源

- **一次颗粒物：** 直接排放（扬尘、烟尘、海盐、花粉）
- **二次颗粒物：** 气态前体物转化（$\text{SO}_2$ → 硫酸盐，$\text{NO}_x$ → 硝酸盐，VOCs → 有机气溶胶）

### 健康影响

- PM2.5 可深入肺部，引起炎症
- 长期暴露增加心血管疾病、肺癌风险
- 全球每年约 700 万人死于空气污染

## 七、大气污染控制

### 污染物减排技术

| 污染物 | 控制技术 | 效率 |
|--------|---------|------|
| $\text{SO}_2$ | 石灰石-石膏脱硫 (FGD) | 90-99% |
| $\text{NO}_x$ | 选择性催化还原 (SCR) | 80-95% |
| 颗粒物 | 静电除尘器 (ESP)、袋式除尘器 | 99%+ |
| VOCs | 催化燃烧、活性炭吸附 | 90-99% |
| $\text{CO}_2$ | 碳捕集与封存 (CCS) | 85-95% |
