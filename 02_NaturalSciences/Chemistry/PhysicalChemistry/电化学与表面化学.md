# 电化学与表面化学详解

## 一、电化学基础

### 原电池

原电池将化学能转化为电能。由两个半电池（电极 + 电解质）组成。

**丹尼尔电池：**

$$(-) \ \text{Zn} | \text{Zn}^{2+} || \text{Cu}^{2+} | \text{Cu} \ (+)$$

阳极（氧化）：$\text{Zn} \to \text{Zn}^{2+} + 2e^-$
阴极（还原）：$\text{Cu}^{2+} + 2e^- \to \text{Cu}$
总反应：$\text{Zn} + \text{Cu}^{2+} \to \text{Zn}^{2+} + \text{Cu}$

### 标准电极电势

标准氢电极（SHE）定义为 $\text{Pt} | \text{H}_2(1\text{atm}) | \text{H}^+(1\text{M})$，$E^\circ = 0$ V。

**常见标准电极电势：**

| 电极反应 | $E^\circ$ (V) |
|---------|-------------|
| $\text{Li}^+ + e^- \to \text{Li}$ | $-3.04$ |
| $\text{Zn}^{2+} + 2e^- \to \text{Zn}$ | $-0.76$ |
| $\text{H}^+ + e^- \to \frac{1}{2}\text{H}_2$ | $0$ |
| $\text{Cu}^{2+} + 2e^- \to \text{Cu}$ | $+0.34$ |
| $\text{Cl}_2 + 2e^- \to 2\text{Cl}^-$ | $+1.36$ |
| $\text{F}_2 + 2e^- \to 2\text{F}^-$ | $+2.87$ |

### 能斯特方程

$$E = E^\circ - \frac{RT}{nF} \ln Q = E^\circ - \frac{0.0592}{n} \lg Q \quad (25^\circ\text{C})$$

**示例：** $\text{Zn}^{2+}/\text{Zn}$ 电极，$E = E^\circ - \frac{0.0592}{2} \lg \frac{1}{[\text{Zn}^{2+}]}$

### 电池电动势与热力学

$$\Delta_r G = -nFE$$
$$\Delta_r G^\circ = -nFE^\circ$$
$$\ln K = \frac{nFE^\circ}{RT}$$

## 二、电解池

### 电解原理

电解池将电能转化为化学能。阳极发生氧化，阴极发生还原。

**法拉第定律：**
$$m = \frac{MIt}{nF}$$

- $m$：析出物质的质量
- $M$：摩尔质量
- $I$：电流
- $t$：时间
- $F = 96485 \ \text{C/mol}$：法拉第常数

### 分解电压与超电势

**分解电压：** 使电解持续进行所需的最小外加电压。

**超电势：** 实际电解电压与理论可逆电动势之差。气体的超电势较大（特别是 $H_2$ 在 $Hg$ 电极上）。

### 电解的应用

- **电镀：** 在基体表面沉积金属层
- **电解精炼：** 粗金属为阳极，纯金属为阴极
- **电解水：** $2\text{H}_2\text{O} \to 2\text{H}_2 + \text{O}_2$
- **氯碱工业：** $2\text{NaCl} + 2\text{H}_2\text{O} \to \text{Cl}_2 + \text{H}_2 + 2\text{NaOH}$

## 三、电化学分析

### 电位分析

测量电极电位以测定离子浓度。使用离子选择性电极（ISE）。

**pH 计：** 玻璃电极对 $\text{H}^+$ 敏感。

### 伏安法

测量电流随电位变化的曲线。

**循环伏安法 (CV)：** 三角波电位扫描，获取氧化还原峰信息。用于研究电极反应机理、测定扩散系数。

### 电导分析

测量溶液的电导率。电导率随离子浓度增大而增大。

## 四、表面化学

### 表面张力

液体表面分子受到向内的净力，使表面倾向于收缩。

$$\gamma = \frac{F}{L}$$

单位：N/m。水的表面张力 ($72.8 \ \text{mN/m}$ at $20^\circ\text{C}$) 高于大多数有机液体。

### 表面活性剂

同时含有亲水基和疏水基的两亲分子：

- **临界胶束浓度 (CMC)：** 开始形成胶束的浓度
- **胶束：** 疏水端向内、亲水端向外的聚集体

### 吸附

**物理吸附：** 范德华力，可逆，可形成多层，吸附热小。
**化学吸附：** 化学键力，不可逆，单分子层，吸附热大。

**Langmuir 吸附等温式：** $\theta = \frac{KP}{1 + KP}$

**Freundlich 吸附等温式：** $\theta = KP^{1/n}$

**BET 吸附等温式：** 多分子层吸附：

$$\frac{P}{V(P_0 - P)} = \frac{1}{V_m C} + \frac{C-1}{V_m C} \cdot \frac{P}{P_0}$$

## 五、胶体与界面

### 胶体分散系

胶体粒子直径在 1-100 nm 之间。可分为：
- **溶胶：** 固体分散在液体中
- **凝胶：** 固体网络结构充满液体
- **乳状液：** 液体分散在另一种不相溶的液体中
- **气溶胶：** 液体或固体分散在气体中

### 胶体性质

- **丁达尔效应：** 胶体粒子散射光（Tyndall 效应）
- **布朗运动：** 胶体粒子受介质分子碰撞随机运动
- **电泳：** 带电胶粒在电场中定向移动

### DLVO 理论

胶体稳定性由范德华吸引力和双电层排斥力的竞争决定：

$$V_{\text{total}} = V_{\text{attraction}} + V_{\text{repulsion}}$$

- 排斥势垒足够高 → 胶体稳定
- 加入电解质 → 双电层压缩 → 势垒降低 → 聚沉

### 聚沉

**电解质聚沉：** 电解质中和胶粒电荷导致聚集。舒尔策-哈迪规则：反离子价数越高，聚沉能力越强。

## 六、催化作用

### 均相催化

催化剂与反应物同一相：

- **酸催化：** 酯的水解、糖的转化
- **碱催化：** Aldol 缩合、Cannizzaro 反应
- **过渡金属配合物催化：** 氢化、羰基化

### 多相催化

催化反应在固体表面进行：

**步骤：**
1. 反应物扩散到催化剂表面
2. 反应物吸附在表面
3. 表面反应
4. 产物脱附
5. 产物扩散离开

**重要工业催化剂：**

| 催化剂 | 反应 | 应用 |
|--------|------|------|
| Fe/Al₂O₃ | $\text{N}_2 + 3\text{H}_2 \to 2\text{NH}_3$ | Haber 法合成氨 |
| Pt/Rh | $4\text{NO} + 4\text{NH}_3 + \text{O}_2 \to 4\text{N}_2 + 6\text{H}_2\text{O}$ | 汽车尾气催化转化 |
| V₂O₅ | $2\text{SO}_2 + \text{O}_2 \to 2\text{SO}_3$ | 硫酸生产 |
| Zeolites | 烃类裂化 | 石油化工 |

### 酶催化

酶是生物催化剂，具有高效性、高度专一性。酶催化机理包括：

- **锁钥模型：** 底物与酶活性中心精确匹配
- **诱导契合模型：** 底物结合诱导酶构象变化
