# 配位化学与晶体场理论详解

## 一、配位化合物基础

### 基本概念

配合物由中心原子（通常是过渡金属离子）和围绕它的配体组成。

**中心原子：** 提供空轨道的金属离子或原子。
**配体：** 提供孤对电子的分子或阴离子。
**配位数：** 直接与中心原子配位的配位原子数目。
**螯合物：** 多齿配体与中心原子形成的环状配合物。

### 命名规则

1. 阴离子配体在前，中性分子配体在后
2. 配体名称前加希腊数字表示数量（二、三、四...）
3. 中心原子后加括号标明氧化数
4. 配阴离子用"酸"字结尾

**示例：** $[\text{Co(NH}_3)_6]\text{Cl}_3$ — 三氯化六氨合钴(III)
$K_4[\text{Fe(CN)}_6]$ — 六氰合铁(II)酸钾

## 二、配位几何

### 常见几何构型

| 配位数 | 几何构型 | 示例 |
|--------|---------|------|
| 2 | 直线形 | $[\text{Ag(NH}_3)_2]^+$ |
| 4 | 四面体 | $[\text{Zn(NH}_3)_4]^{2+}$ |
| 4 | 平面四边形 | $[\text{PtCl}_4]^{2-}$ |
| 6 | 八面体 | $[\text{Fe(CN)}_6]^{3-}$ |

### 异构现象

- **几何异构：** 顺式-反式异构（平面四边形、八面体配合物）
- **光学异构：** 对映异构体（如 $[\text{Co(en)}_3]^{3+}$ 的 Δ-Λ 构型）
- **电离异构：** 配离子与外界离子交换（如 $[\text{Co(NH}_3)_5\text{Br}]\text{SO}_4$ 与 $[\text{Co(NH}_3)_5\text{SO}_4]\text{Br}$）
- **水合异构：** 水分子在配位内界和外界分布不同

## 三、晶体场理论

### d 轨道分裂

在八面体场中，五个 $d$ 轨道分裂为两组：

- $e_g$ 轨道（$d_{z^2}, d_{x^2-y^2}$）：能量升高，指向配体方向
- $t_{2g}$ 轨道（$d_{xy}, d_{xz}, d_{yz}$）：能量降低，指向配体之间

分裂能 $\Delta_0$（或 $10D_q$）的大小取决于：
- 中心金属的电荷和周期数
- 配体场的强弱
- 光谱化学序：$\text{I}^- < \text{Br}^- < \text{Cl}^- < \text{F}^- < \text{OH}^- < \text{H}_2\text{O} < \text{NH}_3 < \text{en} < \text{NO}_2^- < \text{CN}^- < \text{CO}$

### 高自旋与低自旋

配合物的电子排布由分裂能 $\Delta$ 和电子成对能 $P$ 的相对大小决定：

- **高自旋（$\Delta < P$）：** 电子优先分占不同轨道，自旋平行
- **低自旋（$\Delta > P$）：** 电子优先成对填充低能轨道

**示例：** $[\text{Fe(H}_2\text{O)}_6]^{2+}$（弱场，高自旋，$\mu = 4.9 \,\mu_B$）vs $[\text{Fe(CN)}_6]^{4-}$（强场，低自旋，$\mu = 0 \,\mu_B$）

### 晶体场稳定化能 (CFSE)

$$\text{CFSE} = (-0.4n_{t_{2g}} + 0.6n_{e_g})\Delta_0$$

其中 $n_{t_{2g}}$ 和 $n_{e_g}$ 分别为 $t_{2g}$ 和 $e_g$ 轨道上的电子数。

### 四面体场

四面体场中 $d$ 轨道分裂为：
- $e$ 轨道（$d_{z^2}, d_{x^2-y^2}$）：能量降低
- $t_2$ 轨道（$d_{xy}, d_{xz}, d_{yz}$）：能量升高

$$\Delta_t = \frac{4}{9}\Delta_0$$

四面体配合物总是高自旋（分裂能小）。

## 四、配合物的颜色与磁性

### 颜色

配合物的颜色来源于 $d$-$d$ 跃迁（电子从 $t_{2g}$ 激发到 $e_g$）：

$$\Delta E = h\nu = \frac{hc}{\lambda}$$

吸收光的互补色就是配合物呈现的颜色。

**示例：** $[\text{Ti(H}_2\text{O)}_6]^{3+}$ 吸收蓝绿光（~490 nm），呈现紫红色。

### 电荷迁移光谱

配体到金属或金属到配体的电子转移产生的吸收，通常比 $d$-$d$ 跃迁强 $10^2-10^4$ 倍。

**示例：** $[\text{Fe(SCN)}]^{2+}$ 的深红色来自 $\text{SCN}^- \to \text{Fe}^{3+}$ 的电荷迁移。

### 磁性

配合物的磁矩由其未成对电子数决定：

$$\mu = \sqrt{n(n+2)} \,\mu_B$$

其中 $n$ 为未成对电子数，$\mu_B$ 为玻尔磁子。

## 五、配合物的稳定性

### 稳定常数

配合物逐级形成时的平衡常数：

$$M + L \rightleftharpoons ML \quad K_1 = \frac{[ML]}{[M][L]}$$
$$ML + L \rightleftharpoons ML_2 \quad K_2 = \frac{[ML_2]}{[ML][L]}$$

累积稳定常数：$\beta_n = K_1 K_2 \cdots K_n = \frac{[ML_n]}{[M][L]^n}$

### 螯合效应

多齿配体形成的螯合物比相似的单齿配体更稳定。原因：
- 多齿配体与中心离子结合的解离概率更小（需要同时断裂多个配位键）
- 熵增效应（一个多齿配体取代多个单齿配体，体系混乱度增大）

### 影响稳定性的因素

- 中心金属离子：电荷越高、半径越小，配合物越稳定
- 配体碱性强弱：一般碱性越强，配位能力越强
- 软硬酸碱原则：硬酸倾向与硬碱结合，软酸倾向与软碱结合

**Irving-Williams 序列：** 二价过渡金属离子配合物的稳定常数顺序：

$$\text{Mn}^{2+} < \text{Fe}^{2+} < \text{Co}^{2+} < \text{Ni}^{2+} < \text{Cu}^{2+} > \text{Zn}^{2+}$$

该顺序与离子半径减小和晶体场稳定化能（CFSE）变化一致，$\text{Cu}^{2+}$ 因 Jahn-Teller 效应额外稳定。

## 六、配合物的应用

| 应用 | 原理 | 实例 |
|------|------|------|
| 湿法冶金 | 配位溶解 | 氰化法提取金 |
| 电镀 | 配位控制沉积速率 | $[\text{Cu(CN)}_3]^{2-}$ 镀铜 |
| 分析化学 | 配位滴定 | EDTA 滴定金属离子 |
| 生物化学 | 金属酶活性中心 | 血红蛋白中的血红素 |
| 催化 | 过渡金属配合物催化 | Wilkinson 催化剂 |
| 医学 | 配合物药物 | 顺铂抗癌 |
