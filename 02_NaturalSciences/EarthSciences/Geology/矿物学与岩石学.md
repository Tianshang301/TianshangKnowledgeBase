# 矿物学与岩石学

## 一、矿物学基础

### 1.1 矿物的定义与特征

矿物是天然形成的、具有确定化学组成和有序原子排列的无机固体。矿物的五大基本特征：

1. **天然形成**：非人工合成
2. **无机固体**：非有机起源
3. **确定化学组成**：可在一定范围内变化，但有明确化学式
4. **晶体结构**：原子呈周期性有序排列
5. **均一性**：各部分物理化学性质一致

### 1.2 矿物的物理性质

**莫氏硬度**：由德国矿物学家 Friedrich Mohs 于 1812 年提出，采用十种标准矿物划分 1-10 级：

| 硬度 | 标准矿物 | 常见对照物 |
|------|----------|------------|
| 1 | 滑石 (Talc) | 指甲 (2.5) |
| 2 | 石膏 (Gypsum) | 铜币 (3) |
| 3 | 方解石 (Calcite) | 铁钉 (4) |
| 4 | 萤石 (Fluorite) | 玻璃 (5.5) |
| 5 | 磷灰石 (Apatite) | 钢刀 (6.5) |
| 6 | 正长石 (Orthoclase) | |
| 7 | 石英 (Quartz) | |
| 8 | 黄玉 (Topaz) | |
| 9 | 刚玉 (Corundum) | |
| 10 | 金刚石 (Diamond) | |

**解理与断口**：

- **解理**：矿物沿晶体内部特定方向裂开的性质，分为极完全、完全、中等、不完全
  - 云母：一组极完全解理
  - 方解石：三组完全解理（菱面体）
  - 长石：两组完全解理（近 90°）
- **断口**：矿物破裂面不规则，包括贝壳状（石英）、锯齿状（自然铜）、参差状等

**光泽**：矿物表面反光能力

| 光泽类型 | 描述 | 示例 |
|----------|------|------|
| 金属光泽 | 如金属表面 | 黄铁矿、方铅矿 |
| 半金属光泽 | 略暗淡 | 铬铁矿 |
| 金刚光泽 | 折射率高 | 金刚石、闪锌矿 |
| 玻璃光泽 | 最常见 | 石英、长石、方解石 |
| 油脂光泽 | 表面似涂油 | 石英断口 |
| 珍珠光泽 | 似珍珠 | 云母、滑石 |
| 丝绢光泽 | 纤维状 | 石棉、纤维石膏 |
| 蜡状光泽 | 似蜡 | 玉髓 |

**条痕**：矿物在无釉瓷板上划出粉末的颜色，比矿物本身颜色更稳定，是重要的鉴定特征：
- 赤铁矿：条痕樱红色（矿物可呈黑色）
- 黄铁矿：条痕绿黑色
- 黄金：条痕金黄色

**比重**（Specific Gravity）：矿物质量与同体积 4°C 水质量之比

| 比重范围 | 分类 | 示例 |
|----------|------|------|
| < 2.5 | 轻矿物 | 石墨 (2.2) |
| 2.5-4.0 | 中等 | 石英 (2.65)、长石 (2.6) |
| 4.0-5.0 | 重矿物 | 黄铁矿 (5.0) |
| > 5.0 | 极重矿物 | 方铅矿 (7.5)、自然金 (19.3) |

### 1.3 晶体系统

按照晶胞参数（$a$、$b$、$c$ 轴长及 $\alpha$、$\beta$、$\gamma$ 轴角）的差异，晶体分为七大晶系：

| 晶系 | 轴长关系 | 轴角关系 | 对称性 | 示例 |
|------|----------|----------|--------|------|
| 等轴晶系 | $a = b = c$ | $\alpha = \beta = \gamma = 90^\circ$ | 最高 | 岩盐、金刚石、石榴石 |
| 四方晶系 | $a = b \neq c$ | $\alpha = \beta = \gamma = 90^\circ$ | 高 | 锆石、金红石 |
| 正交晶系 | $a \neq b \neq c$ | $\alpha = \beta = \gamma = 90^\circ$ | 中等 | 黄玉、橄榄石 |
| 六方晶系 | $a = b \neq c$ | $\alpha = \beta = 90^\circ, \gamma = 120^\circ$ | 高 | 石英、方解石 |
| 三方晶系 | $a = b = c$ | $\alpha = \beta = \gamma \neq 90^\circ$ | 中 | 方解石（菱面体）、电气石 |
| 单斜晶系 | $a \neq b \neq c$ | $\alpha = \gamma = 90^\circ, \beta > 90^\circ$ | 低 | 石膏、正长石 |
| 三斜晶系 | $a \neq b \neq c$ | $\alpha \neq \beta \neq \gamma \neq 90^\circ$ | 最低 | 钠长石、蓝晶石 |

### 1.4 硅酸盐矿物分类

硅酸盐矿物是地壳中含量最丰富的矿物（占地壳质量 > 90%），其基本结构单元为硅氧四面体 $[\mathrm{SiO_4}]^{4-}$。按四面体连接方式分为：

**岛状硅酸盐**（孤立四面体）：
- $[\mathrm{SiO_4}]^{4-}$ 四面体孤立存在，由阳离子连接
- 代表矿物：橄榄石 $(\mathrm{Mg,Fe})_2\mathrm{SiO_4}$、锆石 $\mathrm{ZrSiO_4}$
- Si:O = 1:4

**环状硅酸盐**：
- 四面体连接成环，如三环 $[\mathrm{Si_3O_9}]^{6-}$、四环 $[\mathrm{Si_4O_{12}}]^{8-}$、六环 $[\mathrm{Si_6O_{18}}]^{12-}$
- 代表矿物：电气石、绿柱石 $\mathrm{Be_3Al_2(Si_6O_{18})}$
- Si:O = 1:3

**链状硅酸盐**：
- **单链**：$[\mathrm{SiO_3}]_{n}^{2n-}$，如辉石族（普通辉石、透辉石）
- **双链**：$[\mathrm{Si_4O_{11}}]_{n}^{6n-}$，如角闪石族（普通角闪石、阳起石）
- Si:O 分别为 1:3 和 4:11

**层状硅酸盐**：
- 四面体连接成无限二维层状 $[\mathrm{Si_2O_5}]_{n}^{2n-}$
- 代表矿物：云母族（黑云母、白云母）、黏土矿物（高岭石、蒙脱石）
- Si:O = 2:5
- 具有一组极完全解理

**架状硅酸盐**：
- 四面体连接成三维空间骨架，部分 Si 被 Al 取代
- 代表矿物：长石族（正长石 $\mathrm{KAlSi_3O_8}$、钠长石 $\mathrm{NaAlSi_3O_8}$）、石英 $\mathrm{SiO_2}$
- Si:O = 1:2

## 二、岩石循环

岩石循环描述了三大类岩石（火成岩、沉积岩、变质岩）之间相互转化的过程：

$$
\text{岩浆} \xrightarrow{\text{冷却结晶}} \text{火成岩} \xrightarrow{\text{风化侵蚀}} \text{沉积物} \xrightarrow{\text{压实胶结}} \text{沉积岩} \xrightarrow{\text{变质作用}} \text{变质岩} \xrightarrow{\text{熔融}} \text{岩浆}
$$

| 转变过程 | 驱动力 | 产物 |
|----------|--------|------|
| 冷却结晶 | 温度降低 | 火成岩 |
| 风化侵蚀 | 水、风、冰、重力 | 沉积物 |
| 压实胶结（成岩作用） | 上覆压力 + 化学沉淀 | 沉积岩 |
| 变质作用 | 高温 + 高压 | 变质岩 |
| 熔融 | 温度升高 + 减压 | 岩浆 |

## 三、火成岩

### 3.1 形成与分类

火成岩（岩浆岩）由岩浆冷却凝固形成。根据形成位置分为：

**侵入岩**：岩浆在地下深处缓慢冷却，结晶时间长，形成粗粒结构
- 产状：岩基、岩株、岩脉、岩床、岩盖

**喷出岩**：岩浆喷出地表快速冷却，结晶时间短，形成细粒或玻璃质结构
- 产状：熔岩流、火山锥、火山穹丘

### 3.2 SiO₂含量分类

| 类型 | SiO₂含量 | 代表侵入岩 | 代表喷出岩 | 主要矿物 |
|------|----------|------------|------------|----------|
| 酸性 | > 65% | 花岗岩 | 流纹岩 | 石英、钾长石、云母 |
| 中性 | 52-65% | 闪长岩 | 安山岩 | 斜长石、角闪石 |
| 基性 | 45-52% | 辉长岩 | 玄武岩 | 斜长石、辉石、橄榄石 |
| 超基性 | < 45% | 橄榄岩 | 科马提岩 | 橄榄石、辉石 |

岩浆黏度与 SiO₂ 含量正相关：

$$
\eta \propto \text{SiO}_2 \text{含量}
$$

酸性岩浆黏度大、气体不易逸出，易导致爆炸性喷发；基性岩浆黏度小，喷发相对温和。

### 3.3 鲍文反应系列

N. L. Bowen（1928 年）通过实验揭示岩浆结晶顺序：

**不连续系列**（暗色矿物）：
$$
\text{橄榄石} \rightarrow \text{辉石} \rightarrow \text{角闪石} \rightarrow \text{黑云母}
$$

**连续系列**（长石）：
$$
\text{钙长石} (\mathrm{Ca}) \rightarrow \text{拉长石} \rightarrow \text{中长石} \rightarrow \text{奥长石} \rightarrow \text{钠长石} (\mathrm{Na})
$$

两系列最终合并：
$$
\text{钾长石} \rightarrow \text{白云母} \rightarrow \text{石英}
$$

意义：解释了岩浆分异作用和矿物共生组合规律，温度越高结晶越早。

### 戈尔德ich 溶解序列

与鲍文反应系列相对应，矿物在地表风化条件下的稳定性顺序：

$$\text{橄榄石} < \text{辉石} < \text{角闪石} < \text{黑云母} < \text{钾长石} < \text{白云母} < \text{石英}$$

即结晶温度越高的矿物，在地表越不稳定（越易风化）。

## 四、沉积岩

### 4.1 形成过程

风化 → 侵蚀 → 搬运 → 沉积 → 压实 → 胶结 → 成岩

### 4.2 分类

**碎屑岩**（由岩石碎屑和矿物颗粒组成）：

| 类型 | 颗粒直径 | 特征 | 常见岩石 |
|------|----------|------|----------|
| 砾岩 | > 2 mm | 磨圆砾石 | 砾岩、角砾岩 |
| 砂岩 | 0.06-2 mm | 砂粒感 | 石英砂岩、长石砂岩 |
| 粉砂岩 | 0.004-0.06 mm | 粉状 | 黄土 |
| 页岩 | < 0.004 mm | 页理发育 | 泥页岩、黏土岩 |

**化学岩**（由化学沉淀形成）：

| 岩石 | 化学成分 | 形成环境 |
|------|----------|----------|
| 石灰岩 | $\mathrm{CaCO_3}$ | 浅海 |
| 白云岩 | $\mathrm{CaMg(CO_3)_2}$ | 潟湖、蒸发环境 |
| 盐岩 | $\mathrm{NaCl}$ | 蒸发盆地 |
| 燧石 | $\mathrm{SiO_2}$ | 化学或生物沉淀 |

**生物岩**（生物遗体堆积形成）：
- 煤：植物遗体堆积 → 泥炭 → 褐煤 → 烟煤 → 无烟煤
- 白垩：微生物（颗石藻）钙质壳体堆积
- 礁灰岩：珊瑚、藻类等造礁生物构建

沉积岩最显著的特征是**层理构造**，包括水平层理、交错层理、粒序层理等。

## 五、变质岩

### 5.1 变质作用类型

| 变质类型 | 主要因素 | 典型环境 |
|----------|----------|----------|
| 接触变质 | 高温（岩浆热） | 侵入体周围 |
| 区域变质 | 高温 + 高压 | 造山带 |
| 动力变质 | 定向压力 | 断裂带 |
| 埋藏变质 | 静压力 + 温度 | 沉积盆地深部 |

### 5.2 叶理与非叶理

**叶理化变质岩**：矿物定向排列，具片理或片麻理构造

| 变质程度 | 原岩（页岩） | 特征结构 |
|----------|--------------|----------|
| 低级 | 板岩 | 板理发育，隐晶质 |
| 中低级 | 千枚岩 | 丝绢光泽，显微褶皱 |
| 中级 | 片岩 | 片理发育，可见斑晶 |
| 高级 | 片麻岩 | 片麻构造，黑白相间条带 |

**非叶理化变质岩**：

| 岩石 | 原岩 | 特征 |
|------|------|------|
| 大理岩 | 石灰岩/白云岩 | 粒状变晶，遇盐酸起泡 |
| 石英岩 | 砂岩 | 致密坚硬，断口油脂光泽 |
| 角岩 | 泥质岩 | 致密块状，接触变质产物 |
| 麻粒岩 | 多种原岩 | 高温变质，不含含水矿物 |

### 5.3 变质程度与矿物指示

不同变质程度具有特征矿物组合：

| 变质带 | 特征矿物 | 变质条件 |
|--------|----------|----------|
| 绿泥石带 | 绿泥石、绢云母 | 低温低压 |
| 黑云母带 | 黑云母 | 中低温 |
| 石榴石带 | 铁铝榴石 | 中温中压 |
| 蓝晶石带 | 蓝晶石 | 中温高压 |
| 硅线石带 | 硅线石 | 高温 |

变质程度可用**变质相**表示：

$$
\text{变质相} = f(T, P)
$$

常见的变质相包括：沸石相 → 葡萄石-绿纤石相 → 绿片岩相 → 角闪岩相 → 麻粒岩相（温度递增），以及蓝片岩相和榴辉岩相（高压特殊相）。

## 六、矿物鉴定方法概述

| 方法 | 原理 | 适用对象 |
|------|------|----------|
| 肉眼鉴定 | 颜色、光泽、晶形、解理 | 手标本 |
| 条痕试验 | 粉末颜色 | 金属矿物 |
| 硬度测试 | 莫氏硬度计 | 所有矿物 |
| 盐酸反应 | $\mathrm{CaCO_3 + 2HCl \rightarrow CaCl_2 + CO_2 \uparrow + H_2O}$ | 碳酸盐矿物 |
| 显微镜鉴定 | 偏光显微镜下光性特征 | 薄片 |
| X 射线衍射 (XRD) | 晶体结构衍射 | 所有晶体 |
| 电子探针 (EPMA) | 电子束激发 X 射线 | 微区成分分析 |
