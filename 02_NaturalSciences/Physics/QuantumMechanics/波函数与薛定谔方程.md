# 波函数与薛定谔方程详解

## 一、波函数 (Wave Function)

### 波函数的定义

量子力学中用波函数 $\Psi(\mathbf{r}, t)$ 来描述微观粒子的状态。波函数是复值函数，包含了关于粒子状态的全部信息。

### 波函数的统计诠释（玻恩诠释）

波函数的模平方代表概率密度：

$$\rho(\mathbf{r}, t) = |\Psi(\mathbf{r}, t)|^2 = \Psi^*(\mathbf{r}, t) \Psi(\mathbf{r}, t)$$

$|\Psi(\mathbf{r}, t)|^2 d^3r$ 表示在时刻 $t$、体积元 $d^3r$ 内找到粒子的概率。

### 归一化条件

粒子在整个空间出现的概率为 1：

$$\int_{\text{all space}} |\Psi(\mathbf{r}, t)|^2 \, d^3r = 1$$

任意波函数都可以归一化：$\Psi_{\text{norm}} = \frac{\Psi}{\sqrt{\int |\Psi|^2 d^3r}}$

### 概率流密度

概率守恒的连续性方程：

$$\frac{\partial \rho}{\partial t} + \nabla \cdot \mathbf{j} = 0$$

概率流密度：

$$\mathbf{j} = \frac{\hbar}{2mi}(\Psi^* \nabla \Psi - \Psi \nabla \Psi^*)$$

### 态叠加原理

若 $\psi_1$ 和 $\psi_2$ 是体系可能的状态，则它们的线性叠加 $\psi = c_1 \psi_1 + c_2 \psi_2$ 也是体系可能的状态。

叠加态的概率密度不是简单相加：

$$|\psi|^2 = |c_1|^2|\psi_1|^2 + |c_2|^2|\psi_2|^2 + 2\text{Re}(c_1^* c_2 \psi_1^* \psi_2)$$

交叉项来源于量子干涉，是量子效应的核心特征。

## 二、薛定谔方程 (Schrödinger Equation)

### 含时薛定谔方程

$$i\hbar \frac{\partial}{\partial t} \Psi(\mathbf{r}, t) = \hat{H} \Psi(\mathbf{r}, t)$$

其中哈密顿算符 $\hat{H} = -\frac{\hbar^2}{2m} \nabla^2 + V(\mathbf{r}, t)$。

这是量子力学的基本方程，地位相当于经典力学中的牛顿第二定律。

### 定态薛定谔方程

当势能不显含时间 $V = V(\mathbf{r})$ 时，波函数可分离变量：

$$\Psi(\mathbf{r}, t) = \psi(\mathbf{r}) e^{-iEt/\hbar}$$

得到定态薛定谔方程：

$$-\frac{\hbar^2}{2m} \nabla^2 \psi(\mathbf{r}) + V(\mathbf{r}) \psi(\mathbf{r}) = E \psi(\mathbf{r})$$

或简写为：$\hat{H} \psi = E \psi$

### 波函数的边界条件

- 有限性：$\psi$ 必须有限
- 单值性：$\psi$ 必须是单值函数
- 连续性：$\psi$ 和 $\nabla\psi$ 连续（$V$ 无穷大突变时 $\nabla\psi$ 可不连续）

## 三、一维无限深方势阱

$$V(x) = \begin{cases} 0, & 0 < x < a \\ \infty, & \text{其他} \end{cases}$$

阱内定态方程：$-\frac{\hbar^2}{2m} \frac{d^2\psi}{dx^2} = E\psi$

通解：$\psi(x) = A\sin(kx) + B\cos(kx)$，其中 $k = \sqrt{2mE}/\hbar$

边界条件 $\psi(0) = \psi(a) = 0$ 给出：

$$\psi_n(x) = \sqrt{\frac{2}{a}} \sin\left(\frac{n\pi x}{a}\right), \quad n = 1, 2, 3, \dots$$

$$E_n = \frac{n^2 \pi^2 \hbar^2}{2ma^2}$$

特点：
- 能量量子化：$E_n \propto n^2$
- 零点能：$E_1 > 0$（基态能量不为零）
- 节点数：第 $n$ 个态有 $n-1$ 个节点

## 四、一维谐振子

$$V(x) = \frac{1}{2} m \omega^2 x^2$$

### 代数解法（产生湮灭算符）

定义升算符和降算符：

$$\hat{a} = \sqrt{\frac{m\omega}{2\hbar}} \left( \hat{x} + \frac{i}{m\omega} \hat{p} \right), \quad \hat{a}^\dagger = \sqrt{\frac{m\omega}{2\hbar}} \left( \hat{x} - \frac{i}{m\omega} \hat{p} \right)$$

对易关系：$[\hat{a}, \hat{a}^\dagger] = 1$

哈密顿量：$\hat{H} = \hbar\omega(\hat{a}^\dagger \hat{a} + \frac{1}{2})$

### 能量本征值和本征态

$$E_n = \left(n + \frac{1}{2}\right) \hbar\omega, \quad n = 0, 1, 2, \dots$$

基态：$\hat{a} \psi_0 = 0 \implies \psi_0(x) = \left(\frac{m\omega}{\pi\hbar}\right)^{1/4} e^{-m\omega x^2/(2\hbar)}$

激发态：$\psi_n = \frac{(\hat{a}^\dagger)^n}{\sqrt{n!}} \psi_0$

### 波函数解析形式

$$\psi_n(x) = \frac{1}{\sqrt{2^n n!}} \left(\frac{m\omega}{\pi\hbar}\right)^{1/4} H_n(\xi) e^{-\xi^2/2}$$

其中 $\xi = \sqrt{m\omega/\hbar}\, x$，$H_n(\xi)$ 为厄米多项式。

## 五、势垒穿透（隧道效应）

$$V(x) = \begin{cases} V_0, & 0 < x < a \\ 0, & \text{其他} \end{cases}$$

当 $E < V_0$ 时，经典力学禁止粒子通过势垒，但量子力学中粒子有一定概率穿透势垒。

透射系数：

$$T = \frac{1}{1 + \frac{V_0^2}{4E(V_0 - E)} \sinh^2(\kappa a)}$$

其中 $\kappa = \sqrt{2m(V_0 - E)}/\hbar$。

对于厚势垒 $(\kappa a \gg 1)$：

$$T \approx \frac{16E(V_0 - E)}{V_0^2} e^{-2\kappa a}$$

**应用：** $\alpha$ 衰变、扫描隧道显微镜 (STM)、隧道二极管

## 六、氢原子

### 中心势场中的薛定谔方程

对于氢原子，$V(r) = -\frac{e^2}{4\pi\varepsilon_0} \frac{1}{r}$

分离变量：$\psi(r, \theta, \phi) = R(r) Y_l^m(\theta, \phi)$

径向方程：

$$-\frac{\hbar^2}{2m} \frac{1}{r^2} \frac{d}{dr}\left(r^2 \frac{dR}{dr}\right) + \left[ V(r) + \frac{\hbar^2 l(l+1)}{2mr^2} \right] R = ER$$

### 能级和波函数

$$E_n = -\frac{me^4}{2\hbar^2 (4\pi\varepsilon_0)^2} \frac{1}{n^2} = -\frac{13.6\,\text{eV}}{n^2}, \quad n = 1, 2, 3, \dots$$

量子数：
- 主量子数 $n = 1, 2, 3, \dots$
- 角量子数 $l = 0, 1, 2, \dots, n-1$
- 磁量子数 $m = -l, -l+1, \dots, 0, \dots, l-1, l$

第 $n$ 能级的简并度：$\sum_{l=0}^{n-1} (2l+1) = n^2$

### 基态波函数

$$\psi_{100}(r, \theta, \phi) = \frac{1}{\sqrt{\pi a^3}} e^{-r/a}$$

其中 $a = \frac{4\pi\varepsilon_0 \hbar^2}{m e^2} \approx 0.529 \, \text{Å}$ 为玻尔半径。

## 七、多粒子体系的波函数

### 全同粒子的不可区分性

交换两个全同粒子不改变物理状态：

$$|\psi(\mathbf{r}_1, \mathbf{r}_2)|^2 = |\psi(\mathbf{r}_2, \mathbf{r}_1)|^2$$

### 交换对称性

$$\psi(\mathbf{r}_2, \mathbf{r}_1) = \pm \psi(\mathbf{r}_1, \mathbf{r}_2)$$

- **玻色子**（整数自旋）：对称波函数（$+$ 号）
- **费米子**（半整数自旋）：反对称波函数（$-$ 号）

### 泡利不相容原理

两个全同费米子不能占据完全相同的量子态。这一原理解释了：
- 原子中电子的壳层结构
- 元素周期表
- 金属中的电子简并压（白矮星、中子星的稳定性）
