# 半导体物理详解

## 一、半导体的基本性质

### 能带结构

半导体的能带结构特点是导带底 $E_C$ 与价带顶 $E_V$ 之间存在宽度为 $E_g$ 的禁带：

$$E_g = E_C - E_V$$

典型半导体的带隙（300 K）：

| 材料 | $E_g$ (eV) | 类型 |
|------|-----------|------|
| Ge | 0.67 | 间接带隙 |
| Si | 1.12 | 间接带隙 |
| GaAs | 1.43 | 直接带隙 |
| GaN | 3.39 | 直接带隙 |
| InP | 1.34 | 直接带隙 |
| SiC (4H) | 3.26 | 间接带隙 |

**直接带隙 vs 间接带隙：**
- 直接带隙：导带底与价带顶在同一 $\mathbf{k}$ 点，电子跃迁不需动量变化（发光效率高）
- 间接带隙：导带底与价带顶在不同 $\mathbf{k}$ 点，需要声子参与跃迁（发光效率低）

### 有效质量

晶体周期势场中电子的加速度与外力关系可等效为：

$$\frac{1}{m^*} = \frac{1}{\hbar^2} \frac{d^2 E}{dk^2}$$

有效质量反映了能带的曲率。导带底 $m^* > 0$，价带顶 $m^* < 0$（空穴）。

## 二、载流子与掺杂

### 本征半导体

纯净无缺陷的半导体中，电子和空穴成对产生（本征激发）：

$$n = p = n_i = \sqrt{N_c N_v} e^{-E_g/(2k_B T)}$$

其中 $N_c = 2\left( \frac{2\pi m_e^* k_B T}{h^2} \right)^{3/2}$，$N_v = 2\left( \frac{2\pi m_h^* k_B T}{h^2} \right)^{3/2}$

### N 型掺杂

掺入施主杂质（5 价元素，如 P、As、Sb 掺入 Si），在带隙中引入靠近导带的施主能级 $E_D$：

$$n \approx N_D, \quad p = \frac{n_i^2}{N_D}$$

其中 $N_D$ 为施主浓度。

### P 型掺杂

掺入受主杂质（3 价元素，如 B、Al、Ga 掺入 Si），在带隙中引入靠近价带的受主能级 $E_A$：

$$p \approx N_A, \quad n = \frac{n_i^2}{N_A}$$

其中 $N_A$ 为受主浓度。

### 费米能级

$$n = N_c F_{1/2}\left( \frac{E_F - E_C}{k_B T} \right), \quad p = N_v F_{1/2}\left( \frac{E_V - E_F}{k_B T} \right)**

非简并情况下（玻尔兹曼近似）：

$$n = N_c e^{-(E_C - E_F)/(k_B T)}, \quad p = N_v e^{-(E_F - E_V)/(k_B T)}$$

### 载流子输运

**漂移电流密度：** 在外电场 $E$ 作用下，载流子做定向运动形成漂移电流：

$$J = q(n\mu_n + p\mu_p)E$$

其中 $\mu_n$、$\mu_p$ 分别为电子和空穴的迁移率。电导率 $\sigma = q(n\mu_n + p\mu_p)$。

**扩散电流：** 由载流子浓度梯度引起的电流：

$$J_n = qD_n \frac{dn}{dx}, \quad J_p = -qD_p \frac{dp}{dx}$$

总电流密度为漂移与扩散之和。爱因斯坦关系联系了迁移率与扩散系数：

$$\frac{D_n}{\mu_n} = \frac{D_p}{\mu_p} = \frac{k_B T}{q}$$

**连续性方程：** 载流子浓度随时间的变化率等于净产生率减去净复合率加上流入率：

$$\frac{\partial n}{\partial t} = G_n - R_n + \frac{1}{q}\nabla\cdot\mathbf{J}_n$$

$$\frac{\partial p}{\partial t} = G_p - R_p - \frac{1}{q}\nabla\cdot\mathbf{J}_p$$

**泊松方程：** 静电势与电荷密度的关系：

$$\nabla^2 \phi = -\frac{\rho}{\varepsilon} = -\frac{q}{\varepsilon}(p - n + N_D^+ - N_A^-)$$

## 三、PN 结

### 内建电势

P 型和 N 型半导体接触时，载流子扩散形成空间电荷区和内建电场：

$$V_{bi} = \frac{k_B T}{q} \ln\left( \frac{N_A N_D}{n_i^2} \right)$$

### 耗尽层宽度

$$W = \sqrt{\frac{2\varepsilon}{q} \left( \frac{1}{N_A} + \frac{1}{N_D} \right) (V_{bi} - V)}$$

### 电流-电压特性

理想 PN 结的 I-V 特性（肖克利方程）：

$$I = I_s \left( e^{qV/(k_B T)} - 1 \right)$$

反向饱和电流：

$$I_s = A q n_i^2 \left( \frac{D_p}{L_p N_D} + \frac{D_n}{L_n N_A} \right)$$

其中 $D$ 为扩散系数，$L$ 为扩散长度。

### 击穿机制

- **齐纳击穿：** 强电场下隧穿效应（高掺杂、低电压）
- **雪崩击穿：** 碰撞电离倍增效应（低掺杂、高电压）

## 四、半导体器件

### 二极管

| 类型 | 原理 | 应用 |
|------|------|------|
| 整流二极管 | PN 结单向导电 | 电源整流 |
| 肖特基二极管 | 金属-半导体接触 | 高速开关 |
| 齐纳二极管 | 反向击穿 | 稳压 |
| 发光二极管 (LED) | 电子-空穴复合发光 | 照明、显示 |
| 激光二极管 (LD) | 受激辐射 | 光通信、光存储 |
| 光电二极管 | 光生载流子 | 光检测 |
| 太阳能电池 | 光伏效应 | 发电 |

### 双极型晶体管 (BJT)

NPN 或 PNP 三层结构。电流增益：

$$\beta = \frac{I_C}{I_B}, \quad \alpha = \frac{I_C}{I_E}$$

### 场效应晶体管 (FET)

**MOSFET (金属-氧化物-半导体场效应管)：**

阈值电压：

$$V_T = V_{FB} + 2\phi_F + \frac{\sqrt{2\varepsilon_s q N_A (2\phi_F)}}{C_{ox}}$$

线性区：$I_D = \mu_n C_{ox} \frac{W}{L} \left[ (V_{GS} - V_T) V_{DS} - \frac{1}{2} V_{DS}^2 \right]$

饱和区：$I_D = \frac{1}{2} \mu_n C_{ox} \frac{W}{L} (V_{GS} - V_T)^2$

### 集成电路

摩尔定律（已放缓）：集成电路上的晶体管数量约每两年翻一番。当前最先进工艺已进入 3 nm 节点。

## 五、半导体光学性质

### 光吸收

本征吸收边对应带隙能量：

$$\alpha(\hbar\omega) \propto (\hbar\omega - E_g)^{1/2}$$（直接带隙）

$$\alpha(\hbar\omega) \propto (\hbar\omega - E_g \pm \hbar\Omega)^2$$（间接带隙，$\hbar\Omega$ 为声子能量）

### 光致发光

光激发产生的电子-空穴对复合发出荧光。发光峰位反映带隙信息。

### 量子阱与异质结

两种不同带隙的半导体薄层交替生长形成量子阱结构。量子尺寸效应导致分立能级。

**量子阱激光器：** 阈值电流低、温度稳定性好、调制速率高。

## 六、新型半导体材料

### 宽禁带半导体

| 材料 | $E_g$ (eV) | 优势 | 应用 |
|------|-----------|------|------|
| SiC | 3.26 | 高导热、高击穿场强 | 功率器件 |
| GaN | 3.39 | 高电子迁移率 | LED、射频 |
| 金刚石 | 5.5 | 最高导热率 | 极端环境 |

### 二维材料

- **石墨烯：** 零带隙，超高迁移率
- **过渡金属硫族化合物 (TMD)：** MoS₂、WS₂ 等，带隙 1-2 eV
- **黑磷：** 带隙可调，0.3-2.0 eV

### 氧化物半导体

- ZnO、IGZO 等：透明导电薄膜
- 应用：平板显示驱动、透明电子学
