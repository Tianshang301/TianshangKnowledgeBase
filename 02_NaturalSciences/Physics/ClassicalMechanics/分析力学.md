# 分析力学详解

## 一、变分法基础

### 泛函与变分

泛函是将函数映射到实数的算子：$J[y] = \int_{x_1}^{x_2} F(x, y, y') \, dx$

泛函的变分 $\delta J$ 是函数 $y(x)$ 发生微小变化时 $J$ 的变化量的一阶近似。

### 欧拉-拉格朗日方程

泛函 $J[y] = \int_{x_1}^{x_2} F(x, y, y') \, dx$ 取极值的必要条件是：

$$\frac{\partial F}{\partial y} - \frac{d}{dx}\left(\frac{\partial F}{\partial y'}\right) = 0$$

**推导思路：** 设 $y(x) + \epsilon\eta(x)$ 为变分路径，其中 $\eta(x_1) = \eta(x_2) = 0$。令 $\left.\frac{dJ}{d\epsilon}\right|_{\epsilon=0} = 0$，分部积分后得到欧拉-拉格朗日方程。

**示例：最短路径问题** 求平面上两点间的最短路径。弧长泛函 $L[y] = \int_{x_1}^{x_2} \sqrt{1 + (y')^2} \, dx$，代入欧拉-拉格朗日方程得 $y'' = 0$，即直线。

## 二、拉格朗日力学 (Lagrangian Mechanics)

### 广义坐标

系统由 $N$ 个质点组成，若有 $k$ 个约束条件，则自由度为 $s = 3N - k$。用 $s$ 个独立参数 $q_1, q_2, \dots, q_s$ 描述系统的位形，称为广义坐标。

广义速度：$\dot{q}_i = dq_i/dt$

### 拉格朗日量

$$L(q, \dot{q}, t) = T - V$$

其中 $T$ 为系统动能，$V$ 为系统势能。

### 拉格朗日方程

$$\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{q}_i}\right) - \frac{\partial L}{\partial q_i} = 0, \quad i = 1, 2, \dots, s$$

**优点：**
- 形式与坐标选择无关
- 自动处理约束力（约束力不做功，不出现在方程中）
- 适用于任意广义坐标系

### 示例：单摆

取广义坐标 $q = \theta$，则：

$$T = \frac{1}{2}ml^2\dot{\theta}^2, \quad V = -mgl\cos\theta$$
$$L = \frac{1}{2}ml^2\dot{\theta}^2 + mgl\cos\theta$$

代入拉格朗日方程：

$$\frac{d}{dt}(ml^2\dot{\theta}) + mgl\sin\theta = 0 \implies \ddot{\theta} + \frac{g}{l}\sin\theta = 0$$

小角度近似 $\sin\theta \approx \theta$：$\ddot{\theta} + \frac{g}{l}\theta = 0$，简谐振动。

### 对称性与守恒量

**诺特定理：** 每个连续对称性对应一个守恒量。

| 对称性 | 守恒量 | 数学条件 |
|--------|--------|---------|
| 时间平移 | 能量 | $\partial L/\partial t = 0$ |
| 空间平移 | 动量 | $\partial L/\partial q_i = 0$ |
| 空间旋转 | 角动量 | $L$ 在旋转下不变 |

**循环坐标：** 若 $L$ 不显含某广义坐标 $q_j$，则 $\partial L/\partial q_j = 0$，对应的广义动量 $p_j = \partial L/\partial \dot{q}_j$ 守恒。

### 约束力与拉格朗日乘子法

对于非完整约束或需要求约束力的情况，使用拉格朗日乘子法：

$$L' = L + \sum_{j=1}^k \lambda_j f_j(q,t)$$

$$\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{q}_i}\right) - \frac{\partial L}{\partial q_i} = \sum_{j=1}^k \lambda_j \frac{\partial f_j}{\partial q_i}$$

其中 $\lambda_j$ 为拉格朗日乘子，$f_j(q,t)=0$ 为约束方程。

## 三、哈密顿力学 (Hamiltonian Mechanics)

### 勒让德变换

从拉格朗日量变换到哈密顿量：

$$H(q, p, t) = \sum_{i=1}^s p_i \dot{q}_i - L(q, \dot{q}, t)$$

其中广义动量定义为 $p_i = \partial L / \partial \dot{q}_i$。

### 哈密顿正则方程

$$\dot{q}_i = \frac{\partial H}{\partial p_i}, \quad \dot{p}_i = -\frac{\partial H}{\partial q_i}$$

这是 $2s$ 个一阶微分方程，而拉格朗日方程是 $s$ 个二阶微分方程。

### 示例：谐振子

$L = \frac{1}{2}m\dot{x}^2 - \frac{1}{2}kx^2$，$p = m\dot{x}$，则：

$$H = \frac{p^2}{2m} + \frac{1}{2}kx^2$$

正则方程：

$$\dot{x} = \frac{p}{m}, \quad \dot{p} = -kx$$

### 哈密顿量的物理意义

若 $L$ 不显含时间，且变换为自然系统（动能是广义速度的二次齐次函数），则：

$$H = T + V = E$$

即哈密顿量等于系统的总能量。

### 泊松括号

$$\{A, B\} = \sum_{i=1}^s \left(\frac{\partial A}{\partial q_i}\frac{\partial B}{\partial p_i} - \frac{\partial A}{\partial p_i}\frac{\partial B}{\partial q_i}\right)$$

**性质：**

- 反对称性：$\{A, B\} = -\{B, A\}$
- 雅可比恒等式：$\{A, \{B, C\}\} + \{B, \{C, A\}\} + \{C, \{A, B\}\} = 0$
- 基本泊松括号：$\{q_i, q_j\} = 0,\ \{p_i, p_j\} = 0,\ \{q_i, p_j\} = \delta_{ij}$

**运动方程：** $\dot{A} = \{A, H\} + \frac{\partial A}{\partial t}$

若 $\{A, H\} = 0$ 且 $\partial A/\partial t = 0$，则 $A$ 为守恒量。

### 正则变换

保持哈密顿方程形式不变的坐标变换 $(q, p) \to (Q, P)$。

**生成函数：** 正则变换由生成函数 $F$ 描述，四种形式：

- $F_1(q, Q, t)$：$p = \partial F_1/\partial q,\ P = -\partial F_1/\partial Q$
- $F_2(q, P, t)$：$p = \partial F_2/\partial q,\ Q = \partial F_2/\partial P$
- $F_3(p, Q, t)$：$q = -\partial F_3/\partial p,\ P = -\partial F_3/\partial Q$
- $F_4(p, P, t)$：$q = -\partial F_4/\partial p,\ Q = \partial F_4/\partial P$

## 四、哈密顿-雅可比理论

### 哈密顿-雅可比方程

寻找正则变换使新哈密顿量 $K = 0$，此时 $Q, P$ 均为常数。令生成函数 $S(q, P, t) = S(q, \alpha, t)$，则：

$$\frac{\partial S}{\partial t} + H\left(q, \frac{\partial S}{\partial q}, t\right) = 0$$

其中 $S$ 称为哈密顿主函数。

### 分离变量法

若哈密顿量不显含时间，可设 $S = W(q, \alpha) - Et$：

$$H\left(q, \frac{\partial W}{\partial q}\right) = E$$

$W$ 称为哈密顿特征函数，满足时间无关的哈密顿-雅可比方程。

### 作用量-角变量

对于周期运动，引入作用量变量：

$$J_i = \oint p_i \, dq_i$$

对应的角变量 $\theta_i$ 线性增长：$\dot{\theta}_i = \partial H/\partial J_i = \nu_i$（常数）

这种方法在微扰论和半经典近似中有重要应用。

## 五、分析力学与量子力学的联系

### 对应原理

经典力学是量子力学在 $\hbar \to 0$ 时的极限。泊松括号对应量子对易子：

$$[A, B] = i\hbar\,\{A, B\}$$

### 路径积分表述

费曼路径积分将量子跃迁振幅表示为所有经典路径的贡献之和：

$$\langle q_f | e^{-iHt/\hbar} | q_i \rangle = \int \mathcal{D}[q(t)] \, e^{iS[q(t)]/\hbar}$$

其中 $S[q(t)] = \int L \, dt$ 为经典作用量。
