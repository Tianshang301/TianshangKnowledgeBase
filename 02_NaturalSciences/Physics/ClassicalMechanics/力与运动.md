# 力与运动详解

## 一、运动学 (Kinematics)

### 基本概念

位置矢量 $\mathbf{r}(t)$ 描述质点在时刻 $t$ 的空间位置。速度与加速度定义为：

$$\mathbf{v}(t) = \frac{d\mathbf{r}}{dt}, \quad \mathbf{a}(t) = \frac{d\mathbf{v}}{dt} = \frac{d^2\mathbf{r}}{dt^2}$$

### 常见运动形式

**匀速直线运动：**

$$\mathbf{r}(t) = \mathbf{r}_0 + \mathbf{v}t$$

**匀加速直线运动：**

$$v = v_0 + at, \quad x = x_0 + v_0t + \tfrac{1}{2}at^2, \quad v^2 = v_0^2 + 2a(x - x_0)$$

**抛体运动：** 水平方向匀速，竖直方向匀加速

$$x = v_0\cos\theta \cdot t, \quad y = v_0\sin\theta \cdot t - \tfrac{1}{2}gt^2$$

**圆周运动：**

$$a_c = \frac{v^2}{r} = \omega^2 r, \quad \omega = \frac{d\theta}{dt}$$

### 参考系变换

**伽利略变换：** 在两个惯性系 $S$ 和 $S'$（$S'$ 以速度 $u$ 相对 $S$ 沿 $x$ 方向运动）之间：

$$x' = x - ut, \quad y' = y, \quad z' = z, \quad t' = t$$
$$v' = v - u, \quad a' = a$$

## 二、牛顿动力学 (Newtonian Dynamics)

### 牛顿三定律

**第一定律（惯性定律）：** 物体在不受外力或合外力为零时，保持静止或匀速直线运动状态。

**第二定律：** 物体所受合外力等于其质量与加速度的乘积：

$$\mathbf{F} = m\mathbf{a} = \frac{d\mathbf{p}}{dt}, \quad \mathbf{p} = m\mathbf{v}$$

**第三定律（作用与反作用）：** 两物体之间的作用力与反作用力大小相等、方向相反、沿同一直线：

$$\mathbf{F}_{12} = -\mathbf{F}_{21}$$

### 常见力

| 力 | 公式 | 说明 |
|----|------|------|
| 重力 | $\mathbf{F} = m\mathbf{g}$ | 在地表附近，$g \approx 9.8\text{m/s}^2$ |
| 万有引力 | $\mathbf{F} = -\frac{GMm}{r^2}\hat{\mathbf{r}}$ | 适用于任意两质点 |
| 弹簧弹力 | $\mathbf{F} = -k\mathbf{x}$ | 胡克定律，$k$ 为劲度系数 |
| 滑动摩擦力 | $f = \mu N$ | $\mu$ 为摩擦系数，$N$ 为正压力 |
| 静摩擦力 | $f_s \leq \mu_s N$ | 最大静摩擦力略大于滑动摩擦力 |

### 力的合成与分解

矢量求和：$\mathbf{F}_{\text{net}} = \sum_i \mathbf{F}_i$

## 三、动量与冲量

### 动量定理

冲量等于动量的变化：

$$\mathbf{I} = \int_{t_1}^{t_2} \mathbf{F} \, dt = \Delta \mathbf{p} = \mathbf{p}_2 - \mathbf{p}_1$$

### 动量守恒定律

若系统所受合外力为零，则系统的总动量守恒：

$$\mathbf{F}_{\text{ext}} = 0 \implies \sum_i \mathbf{p}_i = \text{常量}$$

### 碰撞

**弹性碰撞：** 动能守恒，动量守恒

$$m_1v_1 + m_2v_2 = m_1v_1' + m_2v_2'$$
$$\tfrac{1}{2}m_1v_1^2 + \tfrac{1}{2}m_2v_2^2 = \tfrac{1}{2}m_1v_1'^2 + \tfrac{1}{2}m_2v_2'^2$$

**完全非弹性碰撞：** 碰撞后粘合，动能损失最大

$$v' = \frac{m_1v_1 + m_2v_2}{m_1 + m_2}$$

## 四、功与能

### 功

力沿路径所做的功：

$$W = \int \mathbf{F} \cdot d\mathbf{s}$$

### 动能定理

合外力做的功等于物体动能的变化：

$$W_{\text{net}} = \Delta E_k = \tfrac{1}{2}mv_2^2 - \tfrac{1}{2}mv_1^2$$

### 保守力与势能

保守力做功与路径无关，可定义势能：

$$\mathbf{F} = -\nabla V$$

- 重力势能：$V = mgh$
- 弹性势能：$V = \tfrac{1}{2}kx^2$
- 万有引力势能：$V = -\dfrac{GMm}{r}$

### 机械能守恒

若系统中只有保守力做功，则机械能守恒：

$$E = E_k + V = \text{常量}$$

## 五、刚体力学

### 质心

$$\mathbf{r}_{\text{cm}} = \frac{\sum_i m_i \mathbf{r}_i}{\sum_i m_i}$$

### 转动惯量

$$I = \sum_i m_i r_i^2 = \int r^2 \, dm$$

**常见转动惯量：**

| 形状 | 转轴 | 转动惯量 |
|------|------|---------|
| 细圆环 | 过中心垂直环面 | $I = mR^2$ |
| 圆盘 | 过中心垂直盘面 | $I = \frac{1}{2}mR^2$ |
| 细杆 | 过中心垂直杆 | $I = \frac{1}{12}mL^2$ |
| 球壳 | 过球心 | $I = \frac{2}{3}mR^2$ |
| 球体 | 过球心 | $I = \frac{2}{5}mR^2$ |

### 转动动力学

**转动定律：** $\boldsymbol{\tau} = I\boldsymbol{\alpha} = \frac{d\mathbf{L}}{dt}$

**角动量：** $\mathbf{L} = I\boldsymbol{\omega}$，对质点 $\mathbf{L} = \mathbf{r} \times \mathbf{p}$

**角动量守恒：** 合外力矩为零时，$L = \text{常量}$

**转动动能：** $E_k = \frac{1}{2}I\omega^2$

**平行轴定理：** $I = I_{\text{cm}} + Md^2$

## 六、振动

### 简谐振动

$$x(t) = A\cos(\omega t + \varphi)$$

其中 $\omega = \sqrt{k/m}$，周期 $T = 2\pi/\omega = 2\pi\sqrt{m/k}$。

**能量：** $E = \frac{1}{2}kA^2$（动能与势能之和恒定）

### 阻尼振动

$$m\ddot{x} + c\dot{x} + kx = 0$$

- 欠阻尼：$x = Ae^{-\gamma t}\cos(\omega' t + \varphi)$，$\omega' = \sqrt{\omega_0^2 - \gamma^2}$
- 临界阻尼：$\gamma = \omega_0$，系统最快回到平衡
- 过阻尼：$\gamma > \omega_0$，缓慢回到平衡

### 受迫振动与共振

$$m\ddot{x} + c\dot{x} + kx = F_0\cos(\Omega t)$$

共振条件：驱动力频率 $\Omega$ 等于系统固有频率 $\omega_0 = \sqrt{k/m}$

## 七、波动

### 波动方程

$$\frac{\partial^2 y}{\partial t^2} = v^2 \frac{\partial^2 y}{\partial x^2}$$

通解：$y(x,t) = f(x - vt) + g(x + vt)$

### 简谐波

$$y(x,t) = A\cos(kx - \omega t + \varphi)$$

波数 $k = 2\pi/\lambda$，角频率 $\omega = 2\pi f$，波速 $v = \lambda f = \omega/k$

### 叠加原理与干涉

两列相干波叠加：

$$y = y_1 + y_2 = 2A\cos\left(\frac{\Delta\varphi}{2}\right)\cos(kx - \omega t + \frac{\varphi_1 + \varphi_2}{2})$$

- 相长干涉：$\Delta\varphi = 2n\pi$，振幅最大
- 相消干涉：$\Delta\varphi = (2n+1)\pi$，振幅最小

### 驻波

两列反向传播的相干波叠加形成驻波：

$$y = 2A\cos(kx)\cos(\omega t)$$

驻波条件（两端固定弦）：$L = n\frac{\lambda}{2}, \ n = 1,2,3,\dots$

### 多普勒效应

当波源与观察者有相对运动时，观察者接收到的频率为：

$$f' = f \frac{v \pm v_o}{v \mp v_s}$$

其中 $v$ 为波速，$v_o$ 为观察者速度，$v_s$ 为波源速度。分子取 $+$ 对应观察者靠近波源，分母取 $-$ 对应波源靠近观察者。

### 波的强度

波的强度（能流密度）与振幅的平方成正比：

$$I \propto A^2$$

对于平面简谐波，$I = \frac{1}{2}\rho v \omega^2 A^2$。
