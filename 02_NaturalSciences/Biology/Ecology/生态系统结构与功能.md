# 生态系统结构与功能

## 一、生态系统概述

**生态系统**（Ecosystem）是指在一定的空间内，生物群落与非生物环境之间通过物质循环、能量流动和信息传递而相互作用的统一整体。

$$生态系统 = 生物群落 + 非生物环境$$

### 1.1 生态系统的组成

生态系统由**生物组分**和**非生物组分**两大部分构成：

| 组分 | 类别 | 功能角色 | 典型生物 |
|------|------|----------|----------|
| 生产者 | 自养生物 | 将无机物转化为有机物，固定太阳能 | 绿色植物、藻类、光合细菌、化能合成细菌 |
| 消费者 | 异养生物 | 直接或间接以生产者为食 | 动物（草食、肉食、杂食、寄生） |
| 分解者 | 异养生物 | 将有机物分解为无机物，归还营养物质 | 细菌、真菌、某些原生动物 |
| 非生物组分 | 环境因子 | 提供生存条件与物质来源 | 阳光、温度、水、土壤、大气、pH 值 |

```text
生态系统结构示意:

          阳光 (能量输入)
              ↓
     ┌─────────────────┐
     │   生产者         │ ← CO₂ + H₂O + 无机盐
     │  (植物/藻类)     │
     └────────┬────────┘
              ↓  (被摄食)
     ┌─────────────────┐
     │   初级消费者     │
     │  (草食动物)      │
     └────────┬────────┘
              ↓  (被摄食)
     ┌─────────────────┐
     │   次级消费者     │
     │  (肉食动物)      │
     └────────┬────────┘
              ↓  (被摄食)
     ┌─────────────────┐
     │   三级消费者     │
     │  (顶级捕食者)    │
     └────────┬────────┘
              ↓  (死亡/排泄)
     ┌─────────────────┐
     │   分解者         │──── CO₂ + 无机物 → 生产者
     │  (细菌/真菌)     │
     └─────────────────┘
```

### 1.2 营养级与食物链

**营养级**（Trophic Level）是生物在食物链中所处的位置。能量沿食物链逐级传递，每经过一个营养级都会大量损失。

**食物链类型**：

| 类型 | 起点 | 示例 |
|------|------|------|
| 捕食食物链 | 绿色植物 | 草 → 蝗虫 → 青蛙 → 蛇 → 鹰 |
| 碎屑食物链 | 死亡有机物 | 落叶 → 蚯蚓 → 鸟类 |
| 寄生食物链 | 寄主生物 | 大树 → 槲寄生 → 真菌 |

在自然生态系统中，食物链相互交织形成**食物网**（Food Web），食物网越复杂，生态系统越稳定。

## 二、能量流动

### 2.1 能量流动的基本定律

能量在生态系统中的流动遵循**热力学定律**：

- **热力学第一定律**（能量守恒）：能量既不会凭空产生也不会消失，只能从一种形式转换为另一种形式
- **热力学第二定律**（熵增）：能量转化过程中总会有一部分能量以热能形式散失，系统的总熵增加

$$光合作用能量转换: 6CO_2 + 6H_2O \xrightarrow{光能} C_6H_{12}O_6 + 6O_2$$

$$呼吸作用能量释放: C_6H_{12}O_6 + 6O_2 \rightarrow 6CO_2 + 6H_2O + ATP$$

### 2.2 生态金字塔

生态系统中营养级之间的数量关系可用金字塔表示：

```text
能量金字塔 (单位: kJ·m⁻²·yr⁻¹):

     ┌──────┐
     │  0.1 │ ← 四级消费者
     ├──────┤
     │   1  │ ← 三级消费者
     ├──────┤
     │  10  │ ← 次级消费者
     ├──────┤
     │ 100  │ ← 初级消费者
     ├──────┤
     │ 1000 │ ← 生产者
     └──────┘
```

**三种生态金字塔对比**：

| 类型 | 形状 | 含义 | 备注 |
|------|------|------|------|
| 能量金字塔 | 总是正立 | 能量沿营养级递减 | 最稳定、最有代表性 |
| 生物量金字塔 | 通常正立，可倒立 | 各营养级生物总量 | 海洋中浮游动物 > 浮游植物时可倒立 |
| 数量金字塔 | 可倒立 | 各营养级个体数 | 森林中一棵树可养活大量昆虫 |

### 2.3 10% 法则与林德曼效率

**林德曼效率**（Lindeman Efficiency）由美国生态学家林德曼（R.L. Lindeman, 1942）提出，指出能量从一个营养级传递到下一营养级的效率通常在 **5%-20%** 之间，平均约 **10%**，称为"10% 法则"。

$$林德曼效率 = \frac{下一营养级的同化量}{上一营养级的同化量} \times 100\%$$

**能量分配**：每个营养级同化的能量有三个去向：

1. **呼吸消耗**：维持生命活动，转化为热能散失（约占 50%-80%）
2. **未被利用**：以粪便、残体等形式传递给分解者
3. **生长繁殖**：转化为生物量，可传递给下一营养级（仅 5%-20%）

**例题**：若生产者固定 10,000 kJ 能量，按 10% 传递效率，则四级消费者可获得多少能量？

$$10,000 \times (10\%)^{3} = 10,000 \times 0.001 = 10\ \text{kJ}$$

### 2.4 生态效率类型

| 效率类型 | 定义 | 公式 | 典型值 |
|----------|------|------|--------|
| 同化效率 | 同化量 / 摄入量 | $A_n / I_n$ | 植物 20-50%，动物 50-80% |
| 生长效率 | 生产量 / 同化量 | $P_n / A_n$ | 恒温动物 1-3%，变温动物 30-40% |
| 林德曼效率 | 同化量 / 上营养级同化量 | $A_{n+1} / A_n$ | 5-20% |
| 消费效率 | 摄食量 / 上营养级生产量 | $I_{n+1} / P_n$ | 25-75% |

## 三、物质循环

物质在生态系统中不是单向流动，而是**循环利用**的。生物所需的元素在生物与非生物环境之间反复循环，构成**生物地球化学循环**（Biogeochemical Cycle）。

### 3.1 碳循环

碳是构成有机物的基本元素，碳循环是全球最重要的物质循环之一。

**主要碳库**：

| 碳库 | 碳储量 (GtC) | 周转时间 |
|------|-------------|----------|
| 海洋 | ~38,000 | 数百年-数千年 |
| 陆地生态系统（土壤+植被） | ~3,000 | 数十年-数百年 |
| 大气 CO₂ | ~860 | 数年 |
| 化石燃料 | ~5,000-10,000 | 不可再生（地质时间尺度） |

**碳循环路径**：

```text
    ┌──────────────────────────────────────────────┐
    │                大气 CO₂ (约 420 ppm)          │
    │                   ↓     ↑                     │
    │  光合作用 (120 GtC/yr)  呼吸/分解 (120 GtC/yr)│
    │                   ↓     ↑                     │
    │  ┌──────────────┐  ┌───────────────┐          │
    │  │  陆地植被     │  │  土壤有机碳    │          │
    │  │  (450 GtC)   │→ │  (1,500 GtC)   │          │
    │  └──────────────┘  └───────────────┘          │
    │        ↓ 化石燃料燃烧 (+9 GtC/yr)              │
    │  ┌──────────────┐                              │
    │  │  化石燃料     │                             │
    │  │  (5,000 GtC)  │                             │
    │  └──────────────┘                              │
    │        ↕ 气体交换 (90 GtC/yr)                   │
    │  ┌────────────────┐                            │
    │  │   海洋          │                           │
    │  │   (38,000 GtC)  │                           │
    │  └────────────────┘                            │
    └──────────────────────────────────────────────┘
```

**人类影响**：工业革命以来，化石燃料燃烧和土地利用变化已向大气排放约 900 Gt CO₂，大气 CO₂ 浓度从 280 ppm 上升到约 420 ppm。

**海洋酸化**：海洋吸收了约 30% 的人为 CO₂，导致海水 pH 值下降约 0.1 个单位：

$$CO_2 + H_2O \rightleftharpoons H_2CO_3 \rightleftharpoons HCO_3^- + H^+ \rightleftharpoons CO_3^{2-} + 2H^+$$

$$pH = -\log_{10}[H^+]$$

增加的 H⁺ 离子降低碳酸钙饱和度，威胁珊瑚、贝类等钙化生物。

### 3.2 氮循环

氮是蛋白质和核酸的重要成分，大气中 N₂ 占 78%，但绝大多数生物无法直接利用。

**氮循环的关键过程**：

| 过程 | 反应 | 参与微生物/条件 |
|------|------|----------------|
| 固氮 | $N_2 \rightarrow NH_3$ | 固氮菌（根瘤菌、固氮螺菌）、蓝藻、闪电、工业合成 |
| 氨化 | 有机氮 $\rightarrow NH_4^+$ | 氨化细菌、真菌 |
| 硝化 | $NH_4^+ \rightarrow NO_2^- \rightarrow NO_3^-$ | 亚硝化菌（Nitrosomonas）、硝化菌（Nitrobacter） |
| 反硝化 | $NO_3^- \rightarrow NO_2^- \rightarrow N_2O \rightarrow N_2$ | 反硝化菌（Pseudomonas） |

```text
氮循环示意图:

         大气 N₂ (78%)
             │
     ┌───────┴───────┐
     │  固氮作用       │ ← 生物固氮 (200 Tg/yr)
     │                │    工业固氮 (120 Tg/yr)
     │                │    闪电固氮 (5 Tg/yr)
     └───────┬───────┘
             ↓
       NH₃ / NH₄⁺
       (氨/铵)
         │
         ↓ 硝化作用
       NO₂⁻ → NO₃⁻
       (亚硝酸盐 → 硝酸盐)
         │        ↘
         │     ┌─────────┐
         │     │ 植物吸收 │ → 氨基酸 → 蛋白质
         │     └─────────┘
         ↓              ↑
      反硝化作用       动物摄食/排泄
         │              ↑
         ↓              │
     N₂O → N₂ (返回大气)│
                      ┌─┴─────────┐
                      │ 死亡/排泄物 │
                      │  有机氮     │→ 氨化作用 → NH₄⁺
                      └───────────┘
```

**人类影响**：哈伯-博世法（Haber-Bosch process）合成的活性氮已与全球自然固氮总量相当，导致：

- 水体富营养化（藻华、缺氧死区）
- 土壤酸化
- 温室气体 N₂O 排放增加
- 生物多样性下降

### 3.3 水循环

水是生命的基础，水循环（Hydrological Cycle）由太阳能驱动，是物质循环的重要组成部分。

$$全球水平衡: 降水 = 蒸发 + 径流$$

| 通量 | 年总量 (km³/yr) | 百分比 |
|------|-----------------|--------|
| 海洋蒸发 | 502,800 | 87.5% |
| 海洋降水 | 458,000 | 79.7% |
| 陆地蒸发/蒸腾 | 74,200 | 12.9% |
| 陆地降水 | 119,000 | 20.7% |
| 径流（陆地→海洋） | 44,800 | 7.8% |

```text
全球水循环:
                  水蒸气输送 (40,000 km³/yr)
         ┌─────────────────────────────────────┐
         │                                     │
    ┌────┴─────┐                          ┌────┴─────┐
    │  海洋      │                          │  陆地      │
    │  蒸发      │→→→→→→→→→→→→→←←←←←←←←←│  降水      │
    │  502,800  │          ← 降水           │  119,000   │
    │            │      ← 458,000            │            │
    │  ← 降水    │                          │  ↓          │
    │  ← 458,000 │                          │  蒸腾/蒸发  │
    │            │                          │  74,200     │
    └────┬───────┘                          └────┬────────┘
         │ 蒸发                                  │
         └──←──←──←──←──←──←──←──←──←──←──←──←──┘
                       径流 44,800
```

**水循环的生态意义**：

1. 调节全球气候（通过潜热输送）
2. 运输和分配营养物质
3. 塑造地表形态（侵蚀、沉积）
4. 维持湿地、河流、湖泊等生态系统
5. 影响物种分布和群落结构

## 四、生态系统服务

**生态系统服务**（Ecosystem Services）是指人类从生态系统获得的惠益，分为四大类：

| 服务类型 | 定义 | 具体内容 | 示例 |
|----------|------|----------|------|
| 供给服务 | 直接提供的物质产品 | 食物、淡水、木材、纤维、药材、遗传资源 | 森林提供木材，农田产出粮食 |
| 调节服务 | 生态系统过程的调节效益 | 气候调节、洪水调控、水质净化、疾病调控、传粉 | 湿地净化污水，森林涵养水源 |
| 支持服务 | 维持其他服务的基础过程 | 养分循环、土壤形成、光合作用、水循环 | 土壤微生物分解有机质 |
| 文化服务 | 精神文化层面的惠益 | 休闲旅游、审美体验、教育科研、精神宗教 | 国家公园的生态旅游 |

### 4.1 生态系统服务的价值评估

**全球生态系统服务价值估算**（Costanza et al., 1997; 2014）：

| 生态系统类型 | 年价值 (美元/ha/yr, 2011年价格) |
|-------------|-------------------------------|
| 珊瑚礁 | ~350,000 |
| 海岸湿地 | ~200,000 |
| 内陆湿地 | ~25,000 |
| 热带森林 | ~5,000 |
| 温带森林 | ~3,000 |
| 草原 | ~3,000 |
| 农田 | ~5,000 |
| 海洋 | ~500 |

### 4.2 生态承载力与生态足迹

**生态足迹**（Ecological Footprint）衡量人类对生态系统资源的消费和对废弃物吸收所需的生物生产性面积。

$$生态足迹 = 食物足迹 + 能源足迹 + 居住足迹 + 水足迹 + 其他$$

$$生态赤字/盈余 = 生态承载力 - 生态足迹$$

全球目前正处于**生态赤字**状态，人类每年消耗约相当于 1.75 个地球的生物资源。

### 4.3 生态系统管理的原则

1. **整体性原则**：生态系统是一个有机整体，管理需考虑所有组分
2. **动态性原则**：生态系统不断变化，管理需适应动态过程
3. **阈值效应**：生态系统对外界压力有承受极限，超过阈值可能发生不可逆转变
4. **预防原则**：在不确定性高时，应优先采取保护措施
5. **适应性管理**：采用"学习-调整"的迭代管理方式

## 五、关键生态模型与公式汇总

$$净初级生产力: NPP = GPP - R$$

其中 GPP 为总初级生产力，R 为呼吸消耗

$$净生态系统生产力: NEP = NPP - R_h$$

其中 $R_h$ 为异养呼吸

$$生态系统稳定性 = \frac{抵抗性 + 恢复性}{2}$$

$$生态效率: \frac{A_{n+1}}{A_n} \times 100\%$$

$$全球碳平衡: \Delta C_{atm} = E_{fossil} + E_{land} - (S_{ocean} + S_{land})$$

> **核心思想总结**：能量单向流动、物质循环利用、生态系统服务的可持续管理是生态学理解和应用的关键。
