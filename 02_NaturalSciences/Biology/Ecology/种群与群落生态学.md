# 种群与群落生态学

## 一、种群生态学基础

**种群**（Population）是指在特定时间和空间内，同种个体的集合群。种群是进化的基本单位，也是生态学研究的重要层次。

### 1.1 种群的基本特征

| 特征 | 定义 | 意义 |
|------|------|------|
| 种群密度 | 单位面积/体积内的个体数 | 反映种群的大小 |
| 出生率 | 单位时间新增个体比例 | 决定种群增长潜力 |
| 死亡率 | 单位时间死亡个体比例 | 决定种群衰减速度 |
| 迁入率/迁出率 | 个体进出种群的比例 | 影响种群时空动态 |
| 年龄结构 | 不同年龄组个体比例 | 预测种群未来趋势 |
| 性比 | 雌雄个体比例 | 影响繁殖潜力 |
| 空间分布格局 | 均匀/随机/集群分布 | 反映种内关系与环境 |

**年龄结构类型**：

```text
增长型:     稳定型:     衰退型:
   △          △           △
  ███        ███         ███     ← 老年
 █████      █████       █████    ← 中年
███████    ███████     ███████   ← 幼年
 基部宽      各段均衡    基部窄
```

### 1.2 种群增长模型

**指数增长模型**（Malthus 模型）：在资源无限、无种内竞争的理想条件下，种群以恒定增长率 r 增长。

$$N_t = N_0 e^{rt}$$

其中 $N_t$ 为 t 时刻的种群数量，$N_0$ 为初始数量，r 为内禀增长率，t 为时间。

微分形式：

$$\frac{dN}{dt} = rN$$

```text
指数增长 (J型曲线):
 N
 ↑
 |             ↗
 |          ↗
 |       ↗
 |   ↗ ↗
 | ↗
 └──────────→ t
```

**逻辑斯蒂增长模型**（Verhulst 模型）：在有限资源下，种群增长受到环境容纳量 K 的限制。

$$\frac{dN}{dt} = rN\left(1 - \frac{N}{K}\right)$$

其中 K 为环境容纳量（Carrying Capacity），$(1 - N/K)$ 为剩余空间项。

积分形式：

$$N_t = \frac{K}{1 + \left(\frac{K - N_0}{N_0}\right)e^{-rt}}$$

```text
逻辑斯蒂增长 (S型曲线):
 N
 ↑
 K ──────────────── (渐近线)
 |              ↗
 |           ↗
 |        ↗
 |    ↗
 | ↗
 └────────────→ t
```

**两种模型对比**：

| 特征 | 指数增长 | 逻辑斯蒂增长 |
|------|----------|-------------|
| 前提条件 | 资源无限、无竞争 | 资源有限、有密度制约 |
| 曲线形状 | J 型 | S 型 |
| 增长率 | 恒定 (r) | 随密度递减 |
| 环境容纳量 | 无限制 | K 值的负反馈调节 |
| 自然界常见性 | 短暂（入侵初期、恢复期） | 长期稳定状态 |

### 1.3 生命表与存活曲线

**生命表**（Life Table）记录种群各年龄组的存活率和繁殖率，用于计算净增殖率 $R_0$、平均世代时间 T 和内禀增长率 r：

$$R_0 = \sum l_x m_x$$

$$r = \frac{\ln R_0}{T}$$

**三种存活曲线**（Deevey, 1947）：

```text
存活数 (log)
    ↑
    │ Ⅰ (凸型)──人类/大型哺乳动物
    │     ── Ⅱ (对角线型)──鸟类/啮齿动物
    │         ── Ⅲ (凹型)──鱼类/昆虫
    └─────────────────→ 年龄
```

| 类型 | 特点 | 幼年死亡 | 代表生物 |
|------|------|---------|---------|
| Ⅰ型（凸型） | 老年死亡率高 | 低 | 人类、大象、鲸 |
| Ⅱ型（对角线型） | 各年龄死亡率恒定 | 中等 | 鸟类、某些爬行动物 |
| Ⅲ型（凹型） | 幼年死亡率极高 | 非常高 | 鱼类、昆虫、牡蛎 |

## 二、生活史策略

### 2.1 r/K 选择理论

MacArthur & Wilson (1967) 提出 r/K 选择理论，描述了不同环境下的适应策略。

| 特征 | r-选择 | K-选择 |
|------|--------|--------|
| 环境 | 不稳定、不可预测 | 稳定、可预测 |
| 种群增长率 r | 高 | 低 |
| 体型 | 小 | 大 |
| 寿命 | 短（通常 <1 年） | 长（通常 >1 年） |
| 繁殖次数 | 一次性繁殖（单次） | 多次繁殖 |
| 后代数量 | 多 | 少 |
| 后代大小 | 小 | 大 |
| 亲代投入 | 少 | 多 |
| 竞争能力 | 弱（r 为选择优势） | 强（K 为选择优势） |
| 种群存活曲线 | Ⅲ型 | Ⅰ型 |
| 代表生物 | 昆虫、杂草、啮齿类 | 鲸、大象、人类 |

**r-K 连续体**：大多数生物介于纯 r 与纯 K 之间，形成连续谱系。

### 2.2 其他生活史策略

**Grime 三角形理论**（植物策略）：

```text
       竞争(C)
       /\
      /  \
     /    \
    /______\
   / 耐逆(S) \ 杂草(R)
```

- **竞争型**：资源丰富时，以竞争能力取胜
- **耐逆型**：资源匮乏、环境严酷时，以耐受能力生存
- **杂草型**：干扰频繁时，以快速繁殖和扩散占据空间

## 三、种间关系

### 3.1 种间关系分类

种间关系以对物种的效应（+ 正效应、- 负效应、0 无效应）分为以下类型：

| 关系类型 | 物种 A | 物种 B | 数学模型 | 经典示例 |
|----------|--------|--------|----------|----------|
| 互利共生 | + | + | 互利的 Lotka-Volterra | 菌根真菌 + 植物根系 |
| 捕食 | + | - | Lotka-Volterra 捕食模型 | 狼 + 驯鹿 |
| 竞争 | - | - | Lotka-Volterra 竞争模型 | 两种共存草食动物 |
| 寄生 | + | - | 寄主-寄生虫模型 | 蛔虫 + 人体 |
| 偏利共生 | + | 0 | — | 附生兰 + 树冠 |
| 偏害共生 | - | 0 | — | 胡桃树化感作用 |
| 中性 | 0 | 0 | — | 两种无关联昆虫 |

### 3.2 竞争与生态位

**竞争排斥原理**（Gause 原理）：两个物种不能长期占据同一生态位，竞争结果要么一方灭绝，要么生态位分化。

$$竞争系数: \alpha = \frac{物种B对物种A的抑制作用}{物种A对自身的抑制作用}$$

**Lotka-Volterra 竞争模型**：

$$\frac{dN_1}{dt} = r_1 N_1 \left(1 - \frac{N_1 + \alpha N_2}{K_1}\right)$$

$$\frac{dN_2}{dt} = r_2 N_2 \left(1 - \frac{N_2 + \beta N_1}{K_2}\right)$$

其中 $\alpha$ 为物种 2 对物种 1 的竞争系数，$\beta$ 为物种 1 对物种 2 的竞争系数。

**四种竞争结果**：

| 条件 | 结果 |
|------|------|
| $K_1 > K_2/\beta$ 且 $K_2 < K_1/\alpha$ | 物种 1 胜出，物种 2 灭绝 |
| $K_2 > K_1/\alpha$ 且 $K_1 < K_2/\beta$ | 物种 2 胜出，物种 1 灭绝 |
| $K_1 > K_2/\beta$ 且 $K_2 > K_1/\alpha$ | 不稳定共存（取决于初始条件） |
| $K_1 < K_2/\beta$ 且 $K_2 < K_1/\alpha$ | 稳定共存（生态位分化） |

**生态位分化**方式：
1. **空间分化**：在同一区域占据不同微生境
2. **时间分化**：活动时间错开（昼夜、季节）
3. **食物分化**：取食不同资源或同一资源不同部位
4. **形态分化**：如达尔文雀的喙形适应不同食物

### 3.3 捕食与 Lotka-Volterra 模型

**Lotka-Volterra 捕食模型**（1925-1926）：

$$\frac{dN}{dt} = rN - aNP \quad (\text{猎物})$$

$$\frac{dP}{dt} = caNP - dP \quad (\text{捕食者})$$

其中 N 为猎物密度，P 为捕食者密度，r 为猎物种群增长率，a 为捕食率，c 为转化效率，d 为捕食者死亡率。

**模型预测的种群振荡**：

```text
种群数量
    ↑
    │   猎物 ────────↗↘──────↗↘────  (实线)
    │   捕食者 ──↗↘────↗↘──────↗↘  (虚线)
    │
    └────────────────────────→ 时间
```

**功能反应**（Functional Response）：捕食者摄食率随猎物密度变化的三种类型：

| 类型 | 特点 | 图形 | 示例 |
|------|------|------|------|
| Ⅰ型 | 线性增加至饱和 | 平台形 | 滤食性浮游动物 |
| Ⅱ型 | 减速增加至饱和 | 凸曲线 | 多数捕食者（无脊椎动物） |
| Ⅲ型 | S 形增加至饱和 | S 曲线 | 脊椎动物捕食者（有学习行为） |

### 3.4 寄生与协同进化

**寄主-寄生虫动态模型**（Anderson-May 模型）：

$$\frac{dH}{dt} = rH - \alpha P$$

$$\frac{dP}{dt} = \beta HP - (\mu + \alpha)P - \frac{\alpha P^2}{H}$$

其中 H 为寄主密度，P 为寄生虫密度，$\alpha$ 为致病率，$\beta$ 为传播率。

**协同进化**（Coevolution）：两个或多个物种相互施加选择压力，导致双方共同进化。典型例子：
- 捕食者-猎物军备竞赛（速度 vs 敏捷性）
- 植物-草食动物协同进化（毒素 vs 解毒能力）
- 传粉者-花朵协同进化（花冠深度 vs 口器长度）

## 四、群落生态学

### 4.1 群落结构与特征

**生物群落**（Community）是指在特定区域内相互作用的多个种群的集合。

**群落的基本特征**：

1. **物种组成**：群落的物种名录（科、属、种构成）
2. **物种多样性**：物种丰富度和均匀度的综合度量
3. **优势种**：对群落结构和环境影响最大的物种
4. **空间结构**：垂直分层、水平格局、时间格局
5. **营养结构**：食物链、食物网构成的能量流动体系

**群落垂直结构**：

```text
森林群落垂直分层:

   乔木层 (≥5m)       │ 树冠、林冠
       │              │
   灌木层 (1-5m)      │ 下层木本
       │              │
   草本层 (<1m)       │ 地面草本植物
       │              │
   地被层 (地表)      │ 苔藓、地衣
       │              │
   地下层 (土壤)      │ 根系、土壤生物
```

### 4.2 物种多样性

**物种多样性**包括两个维度：
- **物种丰富度**（Species Richness, S）：群落中物种的数量
- **物种均匀度**（Species Evenness, E）：群落中各物种个体数的均匀程度

**Shannon-Wiener 多样性指数**（信息论基础）：

$$H' = -\sum_{i=1}^{S} p_i \ln p_i$$

其中 $p_i = n_i / N$ 为第 i 种个体数占总个体数的比例，S 为总物种数。

**Simpson 多样性指数**（概率基础）：

$$D = 1 - \sum_{i=1}^{S} p_i^2$$

**Pielou 均匀度指数**：

$$J = \frac{H'}{H_{\max}} = \frac{H'}{\ln S}$$

**多样性比较示例**：

| 群落 | 物种数 S | 个体分布 | Shannon H' | 说明 |
|------|---------|---------|-----------|------|
| A | 3 | 33, 33, 34 | 1.099 | 均匀分布 |
| B | 3 | 90, 5, 5 | 0.394 | 极不均匀 |
| C | 10 | 各 10 | 2.303 | 丰富+均匀 |
| D | 10 | 91, 1×9 | 0.435 | 丰富但极不均匀 |

### 4.3 影响物种多样性的因素

**纬度梯度**：

$$S_{lat} = S_0 e^{-k \cdot |lat|}$$

其中 $S_0$ 为赤道物种数，k 为衰减常数，lat 为纬度。

| 因素 | 趋势 | 机制 |
|------|------|------|
| 纬度 | 热带 > 温带 > 寒带 | 太阳辐射、气候稳定性、进化时间 |
| 海拔 | 低 > 高 | 温度、面积、生境异质性 |
| 栖息地异质性 | 复杂 > 简单 | 更多生态位、更多微生境 |
| 干扰 | 中度干扰最高 | 排除竞争排斥与极端干扰 |
| 面积（岛屿） | 大 > 小 | 更大生境、更低灭绝率 |

**岛屿生物地理学理论**（MacArthur & Wilson, 1967）：

$$S = c A^z$$

其中 S 为物种数，A 为面积，c 为常数，z 为斜率（通常 0.2-0.35）。

$$\frac{dS}{dt} = I - E$$

平衡时：$I = E$，即迁入率 = 灭绝率。

### 4.4 群落演替

**演替**（Succession）是指群落中物种组成和结构随时间发生有序、定向的变化过程。

**原生演替**：在无生命的基质上开始的演替（如火山熔岩、裸岩、冰川退缩地）。

```text
原生演替序列 (从裸岩到森林):

裸岩 → 地衣(先锋) → 苔藓 → 草本 → 灌木 → 先锋乔木 → 顶极森林

             每阶段约数十年至数百年
             ↑                 ↑
         风化土壤积累     土壤深厚、微气候稳定
```

**次生演替**：在已有土壤和生物种源的基质上发生的演替（如弃耕地、火烧迹地）。

```text
次生演替序列 (弃耕地):

弃耕地 → 一年生杂草 → 多年生草本 → 灌木丛 → 幼龄林 → 成熟林

             每阶段约数年至数十年
             ↑                 ↑
          土壤已存在      恢复速度较快
```

| 特征 | 原生演替 | 次生演替 |
|------|----------|----------|
| 起始条件 | 无土壤、无生物 | 有土壤、有种子库 |
| 先锋物种 | 地衣、苔藓、蓝藻 | 一年生草本、杂草 |
| 速度 | 慢（数百年至数千年） | 快（数十年至数百年） |
| 土壤发育 | 从零开始 | 从已有土壤开始 |
| 案例 | 火山熔岩、冰川退缩 | 弃耕地、火烧迹地 |

**演替的机制**：

1. **促进模型**：早期物种改变环境使其不再适合自己，但适合后期物种
2. **抑制模型**：早期物种抑制后期物种的定居
3. **忍耐模型**：后期物种更能忍耐早期物种造成的资源限制

**顶极群落**（Climax Community）：演替的最终稳定状态，由区域气候决定。顶极群落具有以下特征：
- 生物量与多样性达到最大
- 净生产力接近零（NEP ≈ 0）
- 能量输入与输出平衡
- 物种组成相对稳定

### 4.5 生态交错区与边缘效应

**生态交错区**（Ecotone）是两个或多个不同群落之间的过渡地带。

**边缘效应**（Edge Effect）：在生态交错区，因环境条件剧变导致物种组成和丰富度显著不同的现象。

| 指标 | 群落内部 | 生态交错区 |
|------|---------|-----------|
| 光照强度 | 低（林下） | 高 |
| 温度波动 | 小 | 大 |
| 湿度 | 高 | 中-低 |
| 风速 | 小 | 大 |
| 物种多样性 | 通常较低 | 通常较高（边缘种+内部种） |
| 入侵物种概率 | 低 | 高 |

**生态交错区的重要性**：

1. **生物多样性热点**：包含相邻群落的物种和特有边缘种
2. **生态通道**：作为物种迁移和扩散的廊道
3. **缓冲区**：减缓环境变化的冲击
4. **敏感指示区**：对全球变化反应最敏感
