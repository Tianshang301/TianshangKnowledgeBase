# 基因组学与生物信息学详解

## 一、基因组结构

### 原核基因组

- 单环状 DNA，密集排列（基因间隔小）
- 操纵子结构：多个基因共用一个启动子
- 可移动元件：质粒、转座子、插入序列

### 真核基因组

人类基因组（约 3.2 Gb）的特点：

| 特征 | 数据 |
|------|------|
| 基因总数 | ~20,000 |
| 编码序列比例 | ~1.5% |
| 外显子平均长度 | ~150 bp |
| 基因平均外显子数 | ~8 |
| 重复序列比例 | ~50% |
| 最长基因 | TTN（~100 kb 编码区） |

### 重复序列

- **散布重复：** LINE (17%), SINE (13%, 如 Alu), LTR 逆转座子 (8%)
- **串联重复：** 卫星 DNA、小卫星（VNTR）、微卫星 (STR)
- **转座子：** DNA 转座子 (3%)

## 二、基因组测序与组装

### 测序策略

**全基因组鸟枪法：** 基因组片段化 → 构建文库 → 测序 → 重叠群组装 → 支架连接。

**测序质量分数（Phred 分数）：**

$$Q = -10\log_{10}P$$

其中 $P$ 为碱基识别错误的概率。$Q = 30$ 对应准确率 99.9%。

**光学图谱 / 长读长测序：** 辅助组装解决重复区域和结构变异。

### 人类基因组计划 (HGP, 1990-2003)

国际协作项目，耗费 30 亿美元。测序结果：
- 发现基因数远少于预期（~100,000 → ~20,000）
- 人类个体间基因组差异约 0.1%（~3 百万 SNP）
- 约 80% 的基因组有功能（ENCODE 项目）

## 三、功能基因组学

### 转录组学

RNA-seq 分析全局基因表达模式。差异表达基因、聚类分析、富集分析 (GO/KEGG)。

### 表观基因组学

- **DNA 甲基化组：** 全基因组亚硫酸盐测序 (WGBS)
- **组蛋白修饰组：** ChIP-seq
- **染色质可及性：** ATAC-seq、DNase-seq
- **Hi-C：** 3D 染色质构象捕获

### 蛋白质组学

质谱鉴定蛋白质种类和定量。翻译后修饰分析。蛋白质-蛋白质相互作用网络。

### 代谢组学

小分子代谢物的全局分析（NMR、质谱）。

## 四、比较基因组学

### 进化保守性

- 编码序列保守性高（可作为功能注释依据）
- 非编码保守元件（ncRNA、调控序列）
- 系统发育足迹：跨物种序列比对识别保守区域

### 全基因组复制

在进化过程中发生的基因倍增事件：
- 脊椎动物：两次全基因组复制 (2R)
- 被子植物：多次全基因组复制（拟南芥至少 2 次）

### 分子钟

中性突变以近似恒定的速率积累。利用分子钟估计物种分歧时间。

## 五、生物信息学基础

### 序列比对

**成对比对：** Needleman-Wunsch（全局）、Smith-Waterman（局部）。

**BLAST：** 启发式快速搜索算法，将查询序列与数据库比对。

BLAST 的期望值（E-value）表示在随机情况下出现当前比对结果的期望次数：

$$E = Kmn e^{-\lambda S}$$

**多序列比对：** ClustalW, MUSCLE, MAFFT。

### 数据库

| 数据库 | 内容 | 网址 |
|--------|------|------|
| GenBank/NCBI | 核酸序列 | ncbi.nlm.nih.gov |
| UniProt | 蛋白质序列和功能 | uniprot.org |
| PDB | 蛋白质三维结构 | rcsb.org |
| Ensembl | 基因组注释 | ensembl.org |
| KEGG | 代谢通路 | genome.jp/kegg |
| OMIM | 人类遗传病 | omim.org |

### 基因组浏览器

UCSC Genome Browser、IGV（Integrative Genomics Viewer）可视化基因组数据。

## 六、系统发育分析

### 建树方法

- **距离法：** UPGMA、邻接法 (NJ)，计算快但信息利用率低
- **最大简约法 (MP)：** 最少突变步骤，适用于高度相似序列
- **最大似然法 (ML)：** 基于替换模型，计算精确但耗时
- **贝叶斯法 (BI)：** 马尔可夫链蒙特卡罗采样，后验概率

### 可靠性评估

Bootstrap 检验：对原始数据重采样重复建树，支持率 = 采样中出现相同分支的比例。

## 七、统计遗传学

### 连锁分析

利用家系中的遗传标记与致病基因的连锁关系进行定位。LOD 分数 > 3 表示显著连锁。

$$\text{LOD}(\theta) = \log_{10}\frac{L(\theta)}{L(\theta = 0.5)}$$

### GWAS

病例-对照研究设计，检测 SNP 与疾病/性状的关联。需要大样本（千至百万级）。结果以曼哈顿图展示。

### 多基因风险评分 (PRS)

$$PRS = \sum_{i=1}^n \beta_i \cdot SNP_i$$

综合多个微效位点的效应，预测个体疾病风险。

## 八、伦理与法律问题

- **基因隐私：** 谁有权访问个人基因信息？
- **歧视：** 保险公司和雇主是否可以使用基因信息？
- **基因增强：** 是否允许非治疗目的的基因编辑？
- **知情同意：** 基因检测前充分告知风险和局限
- **GINA（遗传信息非歧视法案，2008）：** 美国禁止基于遗传信息的健康和就业歧视
