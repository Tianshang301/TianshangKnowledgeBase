# 微生物生态与工业应用详解

## 一、微生物生态

### 微生物在生态系统中的作用

**分解者：** 细菌和真菌分解有机物，释放无机营养（C、N、P 循环）。

**生产者：** 蓝藻和光合细菌固定 $\text{CO}_2$ 产生有机物。

**共生关系：**
- **豆科植物-根瘤菌：** 固氮（$\text{N}_2 \to \text{NH}_3$）
- **反刍动物-瘤胃微生物：** 纤维素降解
- **珊瑚-虫黄藻：** 光合共生
- **人体肠道菌群：** 消化、维生素合成、免疫调节

### 极端环境微生物

| 类型 | 环境 | 适应机制 | 代表 |
|------|------|---------|------|
| 嗜热菌 | > 60°C | 热稳定酶、反向旋转 DNA 回旋酶 | Thermus aquaticus |
| 嗜冷菌 | < 15°C | 不饱和膜脂、抗冻蛋白 | Psychrobacter |
| 嗜盐菌 | > 3M NaCl | 钾离子积累相容溶质 | Haloferax |
| 嗜酸菌 | pH < 3 | 质子泵、耐酸膜 | Acidithiobacillus |
| 嗜碱菌 | pH > 9 | $\text{Na}^+$/$\text{H}^+$ 反转运 | Bacillus alcalophilus |
| 嗜压菌 | > 40 MPa | 不饱和膜脂、压力适应酶 | Shewanella |

## 二、人体微生物组

### 人体各部位微生物

**肠道菌群：** 约 $10^{14}$ 个细菌（1000+ 种），以厚壁菌门和拟杆菌门为主。

**微生物生长动力学（Monod 方程）：**

$$\mu = \mu_{\max} \frac{S}{K_s + S}$$

其中 $\mu$ 为比生长速率，$S$ 为限制性底物浓度，$K_s$ 为半饱和常数。

**功能：** 发酵不可消化膳食纤维（产短链脂肪酸，如丁酸）、合成维生素 K 和 B 族维生素、训练免疫系统。

**菌群失调与疾病：** 肥胖、炎症性肠病 (IBD)、过敏、糖尿病、抑郁症。

### 益生菌与粪菌移植

**益生菌：** 乳酸杆菌、双歧杆菌发酵制品改善肠道健康。

**粪菌移植 (FMT)：** 将健康供体的粪便微生物转移到患者肠道，治疗复发性艰难梭菌感染效果极好（> 90% 治愈率）。

## 三、工业微生物

### 发酵工程

**发酵类型：**
- **分批发酵：** 一次性投料，产物积累。

微生物在分批发酵中的群体增长遵循 Logistic 模型：

$$\frac{dN}{dt} = \mu_{\max} N \left(1 - \frac{N}{N_{\max}}\right)$$
- **连续发酵：** 不断补充培养基和收获产物，稳态运行。
- **补料分批发酵：** 逐步补加营养，克服底物抑制。

**发酵过程优化：** 参数（温度、pH、溶氧、搅拌速度）、培养基优化、菌种选育。

### 重要发酵产品

| 产品 | 微生物 | 应用 |
|------|--------|------|
| 乙醇 | 酿酒酵母 | 燃料、饮料 |
| 乳酸 | 乳酸菌 | 食品、可降解塑料 (PLA) |
| 谷氨酸钠（味精） | 谷氨酸棒状杆菌 | 调味品 |
| 柠檬酸 | 黑曲霉 | 食品酸味剂、清洗剂 |
| 青霉素 | 产黄青霉 | 抗生素 |
| 胰岛素 | 重组大肠杆菌/酵母 | 糖尿病治疗 |
| 维生素 C | 氧化葡糖杆菌 | 维生素 |
| 黄原胶 | 野油菜黄单胞菌 | 增稠剂（食品、石油） |

## 四、微生物在环境中的应用

### 污水处理

活性污泥法：曝气池（好氧微生物降解有机物）→ 二沉池（沉淀分离污泥）→ 出水。

**过程：** 有机物（BOD）$\xrightarrow{\text{微生物}}$ $\text{CO}_2 + \text{H}_2\text{O}$ + 新细胞。

BOD 的降解遵循一级反应动力学：

$$\frac{dS}{dt} = -kS$$
$$S_t = S_0 e^{-kt}$$

氮的去除：氨氧化细菌（$\text{NH}_3 \to \text{NO}_2^-$）→ 亚硝酸氧化菌（$\text{NO}_2^- \to \text{NO}_3^-$）→ 反硝化菌（$\text{NO}_3^- \to \text{N}_2$）。

### 生物修复

微生物降解有机污染物的过程。在石油泄漏场地，补充氮磷营养促进土著烃降解菌生长。

**降解途径：** 烷烃（$\omega$ 氧化）→ 醇 → 醛 → 脂肪酸 → TCA 循环。

## 五、微生物在农业中的应用

### 生物肥料

**根瘤菌剂：** 接种豆科植物种子，形成根瘤固氮。
**解磷菌：** 溶解土壤中难溶磷酸盐。
**菌根真菌 (AMF)：** 与植物根共生，促进磷吸收。

### 生物农药

**苏云金芽孢杆菌 (Bt)：** 产生 Cry 蛋白毒素，对昆虫幼虫有特异性毒杀作用。是转基因抗虫作物的基础。

**白僵菌 (Beauveria) 和绿僵菌 (Metarhizium)：** 昆虫病原真菌，生物防治。

## 六、微生物安全

### 生物安全等级

| 等级 | 病原体 | 防护 |
|------|--------|------|
| BSL-1 | 无害（大肠杆菌 K12） | 标准微生物操作 |
| BSL-2 | 中等风险（金黄色葡萄球菌） | 生物安全柜 |
| BSL-3 | 严重/致死（结核杆菌、SARS-CoV-2） | 负压实验室 |
| BSL-4 | 最危险（埃博拉、马尔堡病毒） | 最高防护、正压防护服 |

### 抗生素耐药性

WHO 将抗生素耐药性列为全球十大健康威胁之一。

**超级细菌：** MRSA（耐甲氧西林金黄色葡萄球菌）、CRE（耐碳青霉烯肠杆菌）。

**应对措施：** 合理使用抗生素、开发新抗生素、噬菌体疗法、抗菌肽、疫苗预防。
