# 逻辑学知识索引

## 学科概览

逻辑学是研究推理形式和思维规律的科学。逻辑学为各门科学提供方法论基础，是正确思维和有效论证的工具。

## 核心领域

### 演绎逻辑

#### 命题逻辑
- **命题**：具有真假值的陈述句
- **联结词**：否定（¬）、合取（∧）、析取（∨）、蕴含（→）、等价（↔）
- **真值表**：判断复合命题的真假
- **逻辑等值**：A↔B永真
- **推理规则**：假言推理、假言三段论、析取三段论

#### 谓词逻辑（一阶逻辑）
- **个体词**：表示对象
- **谓词**：表示对象的性质或关系
- **量词**：全称量词（∀）、存在量词（∃）
- **公式**：谓词逻辑表达式
- **解释与有效性**：在所有解释下为真

### 归纳逻辑

#### 贝叶斯推理
- **贝叶斯定理**：P(H|E) = P(E|H)P(H)/P(E)
- **先验概率**：P(H)
- **后验概率**：P(H|E)
- **似然度**：P(E|H)
- **应用**：医学诊断、垃圾邮件过滤

#### 简单枚举归纳
- **从个别到一般**：基于观察的概括
- **归纳强度**：证据支持结论的程度
- **归纳悖论**：乌鸦悖论、绿蓝悖论

### 数理逻辑

#### 命题演算与谓词演算
- **形式系统**：公理、推理规则
- **可靠性**：可证明的都是有效的
- **完全性**：有效的都是可证明的（哥德尔完全性定理）

#### 哥德尔不完备定理
- **第一不完备定理**：任何一致的、包含初等算术的形式系统都存在不可判定的命题
- **第二不完备定理**：这样的系统不能证明自身的一致性
- **意义**：形式系统的局限性

#### 集合论与模型论
- **集合论**：康托尔的无穷理论
- **模型论**：语言与结构的关系
- **递归论**：可计算性理论

### 方法论

#### 科学方法
- **假说-演绎法**：提出假说→演绎推论→实验验证
- **证伪主义**：可证伪性是科学的标准（波普尔）
- **范式理论**：科学革命的结构（库恩）

#### 论证理论
- **论证结构**：前提→结论
- **论证类型**：演绎论证、归纳论证、类比论证
- **谬误**：形式谬误、非形式谬误

## 逻辑学在计算机科学中的应用

### 形式化方法
- **程序验证**：证明程序正确性
- **形式规约**：精确描述系统行为
- **模型检验**：验证有限状态系统

### 人工智能
- **知识表示**：逻辑公式表示知识
- **自动推理**：定理证明、推理机
- **描述逻辑**：本体语言基础

### 数据库
- **关系代数**：数据库查询的逻辑基础
- **约束满足**：完整性约束

## 核心公式

| 公式 | 名称 | 说明 |
|------|------|------|
| P→Q ↔ ¬P∨Q | 蕴含等值 | 条件命题的逻辑等值 |
| ¬(P∧Q) ↔ ¬P∨¬Q | 德摩根律 | 否定合取/析取 |
| ∀xP(x) → P(a) | 全称消去 | 全称量词推理 |
| P(a) → ∃xP(x) | 存在引入 | 存在量词推理 |
