# 学习路径：农学

## 基础阶段
1. **生物学基础**：植物学、动物学、遗传学
2. **化学基础**：有机化学、生物化学
3. **土壤学基础**：土壤物理、化学、生物学
4. **农业气象学**：气候与农业生产关系

## 专业方向

| 方向 | 核心内容 | 进阶内容 |
|------|----------|----------|
| 作物学 | 育种、遗传、生理、保护 | 种子科学与技术、耕作学 |
| 园艺学 | 果树、蔬菜、观赏植物 | 设施园艺、采后技术、贮藏加工 |
| 植物保护 | 病理、昆虫、农药、杂草 | 植物检疫、生物防治 |
| 农业资源 | 土壤、化学、气象、土地管理、精准农业 | 农业生态学、农业政策与管理 |
| 畜牧学 | 动物遗传、营养、育种、饲料、健康 | 动物生理学、畜牧经济学 |
| 兽医学 | 病理、药理、外科、内科 | 兽医免疫学、兽医公共卫生 |
| 林学 | 森林生态、经理、造林、木材 | 森林培育学、森林经营规划 |
| 水产 | 养殖、渔业科学、海洋生物、渔业管理 | 水产营养与饲料、水产病害防治 |
| 食品科学 | 食品化学、微生物、加工 | 食品工程原理、食品营养与安全 |
