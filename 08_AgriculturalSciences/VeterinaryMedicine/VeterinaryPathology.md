# 兽医病理学

## 概述

兽医病理学是研究动物疾病发生、发展规律及其机制的科学，包括病因学、发病学、病理变化和转归。它借助大体解剖、组织学和分子生物学方法揭示疾病本质，为兽医临床诊断与防治提供理论依据。

## 疾病发生发展规律

### 疾病分期

**潜伏期：**
- 病原侵入机体到出现最初症状的阶段
- 传染病在此期具有隔离检疫意义
- 诊断困难，病原可在体内繁殖

**前驱期：**
- 疾病非特异性症状开始出现
- 表现精神沉郁、食欲减退、体温升高等
- 临床诊断窗口期

**明显期：**
- 疾病特征性症状完全显现
- 诊断最佳时期，实验室检查阳性率高
- 治疗的关键阶段

**转归期：**
- 痊愈：完全恢复或遗留后遗症
- 死亡：重要器官功能衰竭
- 慢性化：疾病转为慢性迁延过程

### 疾病分类

| 分类方法 | 类别 | 主要特点 |
|---------|------|---------|
| 按病因 | 感染性疾病 | 病原微生物或寄生虫引起 |
| | 非感染性疾病 | 理化因素、营养失衡等 |
| | 遗传性疾病 | 基因突变或染色体异常 |
| 按病程 | 急性病 | 起病急、病程短（数小时至数天） |
| | 亚急性病 | 病程介于急性和慢性之间 |
| | 慢性病 | 病程长（数月至数年） |

## 细胞损伤

### 变性

变性是细胞代谢障碍引起的形态学改变，属可逆性损伤。

**细胞肿胀：**
- 细胞体积增大，胞质淡染
- 线粒体和内质网肿胀
- 原因：缺氧、中毒、感染

**脂肪变性：**
- 胞质中出现脂滴，HE 染色呈空泡状
- 好发于肝脏、心脏和肾脏
- 原因：中毒、缺氧、营养缺乏

**玻璃样变性：**
- 细胞内或间质中出现均质透明蛋白物质
- 常见于血管壁、肾小球和瘢痕组织
- 机制：蛋白质沉积或变性

### 坏死

坏死是细胞或组织死亡的病理过程，属不可逆性损伤。

**坏死类型：**
- 凝固性坏死：蛋白质凝固，组织结构保留轮廓
- 液化性坏死：酶性溶解，组织液化
- 坏疽：继发腐败菌感染，分干性和湿性

**细胞核变化：**
- 核浓缩：核质浓缩，染色加深
- 核碎裂：核膜破裂，染色质碎块散落
- 核溶解：DNA 降解，核淡染消失

**坏死结局：**
- 溶解吸收：坏死物被水解酶分解
- 分离排出：坏死物脱落形成缺损
- 机化：肉芽组织长入取代坏死组织
- 包裹包裹：大块坏死灶周围形成纤维包裹

## 炎症

### 炎症基本病理变化

炎症是机体对损伤因子的防御反应，基本病理变化包括变质、渗出和增生。

$$
\text{炎症反应} = \text{血管反应} + \text{细胞反应} + \text{体液反应}
$$

### 炎症类型

**急性炎症：**
- 血管反应明显：血管扩张、通透性增加
- 渗出性变化：浆液、纤维素、脓性渗出
- 中性粒细胞浸润为主，发挥吞噬作用

**慢性炎症：**
- 持续时间长（数周至数月）
- 淋巴细胞和浆细胞浸润为主
- 增生性变化明显（纤维增生、肉芽肿形成）

### 炎症介质

| 介质类别 | 主要代表 | 生物效应 |
|---------|---------|---------|
| 血管活性胺 | 组胺、5-羟色胺 | 血管扩张、通透性增加 |
| 花生四烯酸代谢物 | 前列腺素、白三烯 | 促炎、趋化、致痛 |
| 细胞因子 | IL-1、TNF-α | 发热、急性期反应 |
| 补体系统 | C3a、C5a | 趋化、肥大细胞释放 |

### 炎症结局

- 痊愈：完全恢复组织结构和功能，或不完全恢复（瘢痕修复）
- 蔓延：局部蔓延至周围组织，或经血道、淋巴道扩散
- 慢性化：持续存在反复发作，转为慢性炎症

## 肿瘤病理

### 肿瘤分类

**良性肿瘤：**
- 生长缓慢，呈膨胀性生长
- 边界清楚，常有包膜
- 不转移，分化程度高

**恶性肿瘤：**
- 生长迅速，呈浸润性生长
- 边界不清，无完整包膜
- 可转移（淋巴道、血道、种植性转移）

### 肿瘤命名

**上皮组织肿瘤：**
- 良性：乳头状瘤、腺瘤
- 恶性：癌（鳞状细胞癌、腺癌、移行细胞癌）

**间叶组织肿瘤：**
- 良性：纤维瘤、脂肪瘤、平滑肌瘤
- 恶性：肉瘤（纤维肉瘤、骨肉瘤、脂肪肉瘤）

### 肿瘤诊断方法

- 大体病理检查：肿瘤大小、形状、质地、切面
- 组织学检查：HE 染色观察细胞异型性
- 细胞学检查：细针穿刺、脱落细胞学
- 免疫组化（IHC）：检测肿瘤标志物确定组织来源
- 分子病理诊断：基因突变检测、FISH 分析

$$
\text{肿瘤增殖指数} = \frac{\text{Ki-67阳性细胞数}}{\text{肿瘤细胞总数}} \times 100\%
$$

## 系统病理学

### 消化系统病变

**肝脏病变：**
- 肝炎：肝细胞变性、坏死、炎细胞浸润
- 肝硬化：纤维组织增生，假小叶形成
- 脂肪肝：肝细胞脂肪变性，脂质堆积

**胃肠病变：**
- 胃肠炎：黏膜充血水肿，炎细胞浸润
- 溃疡：黏膜缺损，深度达黏膜下层
- 出血：黏膜出血点或大片出血

### 呼吸系统病变

**肺脏病变：**
- 肺炎：肺泡腔内渗出物填充
- 肺水肿：肺泡间隔增宽，肺泡内浆液渗出
- 肺气肿：肺泡壁破裂，肺泡腔融合扩大

### 泌尿系统病变

**肾脏病变：**
- 肾炎：肾小球或肾间质炎症
- 肾病综合征：肾小球基底膜损伤，大量蛋白尿
- 肾功能衰竭：肾单位大量丧失，氮质血症

### 循环系统病变

**心脏病变：**
- 心肌炎：心肌纤维变性坏死，间质炎细胞浸润
- 心内膜炎：心瓣膜赘生物形成
- 心包炎：心包腔渗出积液

### 神经系统病变

**脑部病变：**
- 脑炎：神经细胞变性坏死，血管套形成
- 脑软化：脑组织液化坏死，空洞形成
- 脑水肿：脑组织含水量增加，脑回变平
